Einstellungen - Marktplatz


Mit den Marketplace-Einstellungen wird das Marketplace-Zertifikat zum Hochladen oder Kaufen von Assets eingerichtet. Das Zertifikatformular besteht aus den folgenden Abschnitten, wenn Sie bereits ein Zertifikat erstellt haben:

  • Zertifikatsdatei: In diesem Bereich geben Sie die *.pem Datei, die der Marktplatz für die Zertifizierung von Assets verwendet. Wenn Sie ein zuvor erstelltes Zertifikat haben, können Sie hier auf den Dateibrowser klicken und es dem Forum hinzufügen, aber wenn Sie kein Zertifikat haben und eines erstellen möchten, sollten Sie hier alle Informationen weglassen und stattdessen auf die Schaltfläche klicken Klicken Sie unten auf die Schaltfläche Neues Zertifikat erstellen.

  • Zertifikatskennwort: Hier müssen Sie das Zertifikatskennwort für die oben ausgewählte Datei eingeben. Das Passwort wird erstellt, wenn Sie das erste Mal erstellen *.pem Datei (siehe Erstellen eines neuen Zertifikats, unten) und wird irreversibel mit der erstellten Zertifikatsdatei verknüpft.

  • Check: Nachdem Sie die obigen Felder ausgefüllt haben, sollten Sie auf diese Schaltfläche klicken, um sicherzustellen, dass das Passwort und die Dateiauswahl korrekt sind und zusammen funktionieren.


Erstellen Sie ein neues Zertifikat


Wenn Sie noch keine Zertifikatsdatei für die Nutzung des Marketplace erstellt haben, klicken Sie am unteren Rand dieses Formulars auf den Button Create New Certificate. Wenn Sie das tun, wird ein neues Fenster geöffnet, in dem Sie aufgefordert werden, die folgenden Informationen einzugeben:

  • Publisher-Name: Dies ist der Name, der als Herausgebername für das Asset verwendet wird. Im Marketplace-Dashboard können Sie verschiedene Publisher einrichten, wenn Sie möchten, und jeder sollte seinen eigenen entsprechen *.pem Zertifikatsdatei.

  • Zertifikatspasswort: Hier müssen Sie ein eindeutiges Passwort für die zu erstellende Zertifikatsdatei angeben. Achten Sie darauf, mit dem, was Sie wählen, wie Sie es nicht später ändern können, und wenn Sie nicht das richtige Passwort für die Zertifikatsdatei in den Abschnitten haben über Sie werden nicht in der Lage alle vorhandenen Markt Vermögenswerte zu aktualisieren (noch laden neue).

Mit diesen Angaben können Sie dann auf die Schaltfläche Create Certificate klicken, um die *.pem Datei. Sie werden aufgefordert, einen Speicherort und einen Namen für die Datei anzugeben, und Sie sollten dies zur späteren Bezugnahme beachten. Sobald Sie dies getan haben, werden die neue Datei und das Passwort automatisch in die oben aufgeführten Formularfelder eingefügt und Sie können auf die Schaltfläche Prüfen klicken, um zu testen, ob das Zertifikat korrekt generiert wurde.

HINWEIS: Wir empfehlen, dass Sie einen Screenshot der tatsächlichen Einstellungen erstellen und eine Kopie der PEM-Datei erstellen. Beide sollten zusammen an einem sicheren Ort aufbewahrt werden, um sie später zu GameMaker Studio, oder falls Sie GameMaker Studio 2 zu einem späteren Zeitpunkt erneut installieren müssen. Sie können Assets, die bereits hochgeladen wurden, nur aktualisieren, wenn das aktualisierte Paket mit demselben Zertifikat signiert ist, mit dem das Original erstellt wurde.
