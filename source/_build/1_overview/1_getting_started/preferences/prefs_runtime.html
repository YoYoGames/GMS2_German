Einstellungen - Laufzeit-Feeds


In den Voreinstellungen für Laufzeit-Feeds werden die aktuell installierten Laufzeiten sowie die derzeit verwendete Laufzeit angezeigt und geändert. GameMaker Studio 2 können mehrere verschiedene Laufzeit-Tools gleichzeitig installiert sein. Sie können festlegen, welches von diesen Einstellungen verwendet werden soll. Idealerweise möchten Sie die aktuellste Laufzeit verwenden, aber manchmal gibt es ein Problem, oder Sie müssen aus Gründen der Vergleichbarkeit ein älteres verwenden. Sie können also die Laufzeitwerkzeuge von GameMaker Studio 2 ändern GameMaker Studio 2 wird hier dein Spiel erstellt.

Der erste Abschnitt dieser Einstellung bezieht sich darauf, wo Sie die Laufzeit-Tools abrufen, und Sie müssen eine URL zu einem RSS-Feed eingeben. Dieser Feed wird dann nach Updates der Laufzeitwerkzeuge durchsucht, die heruntergeladen und installiert werden können. In diesem Abschnitt wird standardmäßig die URL des Master-Laufzeit-Feeds angezeigt. Sie können jedoch auf die Schaltfläche Hinzufügen klicken, um einen neuen Laufzeit-Feed zu erstellen, in dem Sie den Namen und die URL für den RSS-Feed angeben. Klicken Sie anschließend auf Übernehmen, um GameMaker Studio 2 darauf zuzugreifen. Sie müssen das Fenster "Einstellungen" schließen und erneut öffnen, damit der neue Feed angezeigt wird.

HINWEIS: Wenn Sie sich für den Beta-Kanal entscheiden, werden die Beta-Kanal-Laufzeiten zusammen mit "Master" in einem "Beta-Feed" angezeigt, und Sie können jederzeit zwischen den beiden wechseln.


Nachdem ein Laufzeit-Feed erstellt wurde, wird er in diesem Abschnitt als Unterkategorie aufgeführt. Es sollte immer mindestens ein Feed aufgeführt sein (standardmäßig ist dies der Master-Feed). Mit den Voreinstellungen für den Laufzeit-Feed können Sie die verschiedenen Laufzeit-Tools auswählen, installieren und deinstallieren. Oben befinden sich die aktuell verwendeten Laufzeitwerkzeuge, das Erstellungsdatum und der Ort, an dem sie auf der Festplatte gespeichert sind. Unter diesen Details finden Sie eine Liste aller verfügbaren Laufzeiten, und Sie können eine beliebige aus dieser Liste auswählen, um sie in die "laufende" Laufzeit umzuwandeln, die zum Kompilieren Ihrer Projekte verwendet wird.

Um eine andere Laufzeit auszuwählen, suchen Sie sie einfach in der Liste und doppelklicken Sie dann darauf Wenn die Runtime nicht heruntergeladen und installiert wurde, werden Sie dazu aufgefordert. Nach der Installation werden Sie aufgefordert, die IDE zu schließen und erneut zu starten, damit die neue Runtime initialisiert werden kann.

Diese Laufzeitliste enthält alle verfügbaren Laufzeiten, ihre Versionsnummer, das Datum, an dem sie veröffentlicht wurden, und, wenn sie installiert sind, der Pfad zu dem sie installiert wurden. Auf der rechten Seite werden auch die verschiedenen Statussymbole angezeigt:

Dies markiert die aktuelle Laufzeit, die zur Erstellung Ihrer Projekte ausgewählt wurde.
Wenn Sie hierauf klicken, werden der Download und die Installation der ausgewählten Laufzeitumgebung gestartet (eine Statusleiste wird oben rechts in der IDE angezeigt).
Wenn Sie hierauf klicken, wird die ausgewählte Laufzeit von Ihrer Festplatte entfernt.

Sie können die Laufzeitliste jederzeit aus dem angegebenen RSS-Feed aktualisieren, indem Sie auf die Schaltfläche Aktualisieren klicken .
