Lizenzierung


Nachdem Sie GameMaker Studio 2 heruntergeladen und installiert haben, werden Sie bei der ersten Verwendung aufgefordert, Ihre Lizenzdetails einzugeben:

Ihre Lizenz ist an ein YoYo Konto (YYA) gebunden, und Sie müssen eine Lizenz erworben haben, um GameMaker Studio 2 verwenden zu können. Dieses Konto enthält alle Ihre Lizenzdetails für die YoYo Games-Produkte, die Sie besitzen, und wird auch für den YoYo Games Marketplace verwendet. Weitere Informationen zum YYA finden Sie auf der folgenden Seite:

  1. Ein YoYo Konto YoYo
WICHTIG! Wenn Sie eine Testlizenz für GameMaker Studio 2 und dann eine der anderen Lizenzen kaufen (Desktop, Mobile, UWP, Web oder Console), müssen Sie sich möglicherweise von der IDE abmelden und GameMaker Studio 2 anschließend neu starten erneut, um sicherzustellen, dass die neue Lizenz korrekt angewendet wird (dies sollte jedoch automatisch geschehen). Sie können sich vom Datei-Menü abmelden.


Proxy-Unterstützung

Wenn Sie für Ihre Installation von GameMaker Studio 2 eine Proxy-Unterstützung benötigen, können Sie diese einfach durch Klicken auf den Proxy-Server hinzufügen. Das folgende Fenster wird geöffnet:

Sie können jetzt Ihre Proxy-Details hinzufügen. Einige der oben gezeigten Felder sind optional und wenn Sie eines nicht benötigen, lassen Sie das Feld einfach leer. Bevor Sie fortfahren, sollten Sie jedoch Folgendes beachten: Wenn Sie in Ihren Windows-Internetoptionen einen Proxy festgelegt haben und kein Benutzername / Kennwort erforderlich ist, sollte es nur "wie es ist" und Sie werden wahrscheinlich keine Einstellungen vornehmen Proxy in GameMaker Studio 2.

Wenn für Ihren Proxy ein Benutzername und / oder ein Kennwort erforderlich ist, müssen Sie sie zu den Feldern im obigen Fenster hinzufügen. Beachten Sie, dass in diesen Fällen die Angaben zu Benutzername / Kennwort / Proxy immer erforderlich sind und dass der Proxy-IP auch der Port hinzugefügt werden muss, dh: xx.xx.xx.xx:port.

Wenn Sie die relevanten Informationen hinzugefügt haben, klicken Sie auf OK und melden Sie sich wie GameMaker Studio 2 bei GameMaker Studio 2 an.
