GameMaker Studio 2 Lizenzen


Es gibt verschiedene Lizenztypen, die Sie für GameMaker Studio 2 können. Die Auswahl hängt weitgehend davon ab, welches Publikum Sie ansprechen möchten und ob Sie nur Spaß daran haben möchten, Spiele zu entwickeln oder diese verkaufen möchten verteile sie auch. Im Folgenden finden Sie Details zu jedem Lizenztyp.

GameMaker Studio 2 wird zunächst mit einer Testlizenz geliefert, und dann können Sie je nach Bedarf verschiedene Upgrades erwerben. Die Trial-Lizenz unterliegt bestimmten Einschränkungen, erlaubt Ihnen jedoch, die wichtigsten Funktionen und Prototyp-Projekte zu testen. Das Wichtigste, das Sie mit der Testlizenz des Produkts beachten sollten, ist, dass Sie zwar keine Spiele auf Ihren Rechnern testen können, aber keine fertigen ausführbaren Pakete für eine Zielplattform erstellen können. Darüber hinaus sind Sie auch hinsichtlich der verfügbaren Ressourcen eingeschränkt:

  • Objekte - 15
  • Sprites - 20 (keine dynamische Ladung erlaubt, keine SWF- oder Spine-Sprites erlaubt)
  • Sounds - 10 (keine Audiopuffer erlaubt)
  • Fliesensets - 2
  • Skripte - 10
  • Pfade - 5 (keine dynamische Pfaderstellung erlaubt)
  • Timelines - 2 (keine dynamische Timeline-Erstellung erlaubt)
  • Schriftarten - 5 (keine dynamische Schriftherstellung erlaubt)
  • Räume - 5 (keine dynamische Raumerstellung erlaubt)
  • Shader - 0
  • Enthaltene Dateien - 0
  • Erweiterungen - 0
  • Konfigurationen - 0


Abgesehen von diesen Ressourcenbeschränkungen fehlen in der IDE noch einige andere Elemente:

  • Sie können keine Strukturseiten erstellen
  • Sie können keine Texturgruppen erstellen
  • Sie können keine Audiogruppen erstellen
  • Sie können keine vorhandenen Ressourcen über das Kontextmenü hinzufügen
  • Sie können die integrierte Versionskontrolle nicht verwenden
  • Sie können keine Assets vom Marktplatz herunterladen
  • Sie können keine alten Projekte aus früheren Versionen von GameMaker Studio importieren
  • Sie können YYZ-Pakete nicht importieren
  • Sie können den YoYo Games Splash Screen nicht ändern
  • Sie können die anonyme IDE-Analyse nicht deaktivieren

Mit der Creator- Version von GameMaker Studio 2 meisten Einschränkungen der Trial-Lizenz (siehe oben) entfernt, mit Ausnahme der Möglichkeit, den Begrüßungsbildschirm zu ändern, anonyme IDE-Statistiken zu deaktivieren und das YYC-Ziel zu verwenden. Einmal gekauft, können Sie Projekte auf eines der folgenden Ziele übersetzen:

  • Windows
  • Mac OS

Die verfügbare Zielplattform hängt von der Version der IDE ab, die Sie gekauft haben ( Ersteller für macOS oder Creator für Windows ), und verwendet die VM (einen Runner + interpretierten Code) zum Ausführen und Erstellen Ihrer Spiele. Informationen zur Einrichtung finden Sie im Helpdesk-Abschnitt zu erforderlichen SDKs oder im Abschnitt zu Desktop-Zielen.


Mit der Desktop- Version von GameMaker Studio 2 alle Einschränkungen der Testlizenz (siehe oben) entfernt und Sie können Projekte auf folgende Desktop-Ziele kompilieren:

  • Windows
  • Mac OS
  • Ubuntu

Alle Ziele haben zwei Kompilierungsoptionen. Die eine ist die VM, die einen Runner und interpretierten Code verwendet, und die andere ist der YYC ( YoYo Compiler), der nativen Code und eine kompilierte ausführbare Datei erzeugt. Beachten Sie, dass die YYC-Option eine zusätzliche Einrichtung erfordert. Weitere Informationen finden Sie im Abschnitt Erforderliche SDKs oder im Helpdesk-Abschnitt zu Desktop-Zielen.


Mit der Web- Version von GameMaker Studio 2 alle Einschränkungen der Testlizenz (siehe oben) entfernt und Sie können Projekte zum HTML5- Ziel kompilieren. Spiele werden mit JavaScript kompiliert. Weitere Informationen finden Sie im Helpdesk von YoYo Games: Einrichten für das Web (HTML5).


Mit der Amazon Fire Version von GameMaker Studio 2 alle Einschränkungen der Trial-Lizenz (siehe oben) entfernt und Sie können Projekte zum Amazon Fire Ziel kompilieren. Spiele werden mit dem Android SDK kompiliert und können entweder mit der VM (einem Runner mit interpretierten Code) oder dem YYC ( YoYo Compiler, der nativen Code erzeugt) kompiliert werden. Weitere Informationen zum Einrichten finden Sie im Abschnitt Erforderliche SDKs oder im Abschnitt Helpdesk der Amazon Fire Lizenz.


Mit der Mobile- Version von GameMaker Studio 2 alle Einschränkungen der Trial-Lizenz (siehe oben) entfernt und Sie können Projekte für folgende Ziele kompilieren:

  • Android
  • Amazon Fire
  • iOS

Alle Ziele haben zwei Kompilierungsoptionen. Die eine ist die VM, die einen Runner und interpretierten Code verwendet, und die andere ist der YYC ( YoYo Compiler), der nativen Code und eine kompilierte ausführbare Datei erzeugt. Beachten Sie, dass die YYC-Option eine zusätzliche Einrichtung erfordert. Weitere Informationen zum Einrichten finden Sie im Abschnitt zu den erforderlichen SDKs oder im Helpdesk-Abschnitt zu Mobile Targets.


Mit der UWP- Version von GameMaker Studio 2 alle Einschränkungen der Testlizenz (siehe oben) entfernt und Sie können Projekte als UWP-kompatible ausführbare Dateien kompilieren. Diese ausführbaren Dateien können so konfiguriert werden, dass sie auf einer von drei UWP-Plattformen ausgeführt werden:

  • Windows 10 Desktop
  • Windows 10 Mobil
  • Xbox Eins

Das UWP-Ziel verfügt über zwei Kompilierungsoptionen. Die eine ist die VM, die einen Runner und interpretierten Code verwendet, und die andere ist der YYC ( YoYo Compiler), der nativen Code und eine kompilierte ausführbare Datei erstellt. Beachten Sie, dass die YYC-Option eine zusätzliche Einrichtung erfordert. Weitere Informationen zum Einrichten finden Sie im Abschnitt Erforderliche SDKs oder im Helpdesk-Abschnitt zu UWP-Zielen.


Mit der Console- Version von GameMaker Studio 2 alle Einschränkungen der Trial-Lizenz (siehe oben) entfernt und Sie können Projekte für die folgenden Zielplattformen kompilieren

  • Xbox Eins
  • Playstation 4

Jedes der Konsolenziele verfügt über zwei Kompilierungsoptionen. Die eine ist die VM, die einen Runner und interpretierten Code verwendet, und die andere ist der YYC ( YoYo Compiler), der nativen Code und eine kompilierte ausführbare Datei erstellt. Beachten Sie, dass die YYC-Option eine zusätzliche Einrichtung erfordert. Weitere Informationen zum Einrichten finden Sie im Abschnitt zu den erforderlichen SDKs oder im Helpdesk-Abschnitt zu Konsolenzielen.


GameMaker Studio 2 kann auch in einer College-Umgebung verwendet werden. Für weitere Informationen kontaktieren Sie bitte YoYo Games über das YoYo Games Education Portal.


WICHTIG! Wenn Sie eine Testlizenz für GameMaker Studio 2 und dann eine der anderen Lizenzen kaufen (Ersteller, Desktop, Amazon Fire, Mobile, UWP, Web oder Konsole), müssen Sie sich von der IDE GameMaker Studio 2 und GameMaker Studio 2 neu starten GameMaker Studio 2. Wenn Sie das Programm neu starten und sich erneut anmelden, sollte die neue Lizenz korrekt angewendet werden. Sie können sich über das Dateimenü abmelden.
