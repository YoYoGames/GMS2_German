Installieren


GameMaker Studio 2 ist für die Verwendung auf Desktop-Computern mit den folgenden Mindestanforderungen konzipiert:

Für die Windows Plattform benötigen Sie die folgenden Grundeinstellungen:

  • 64-Bit-Intel-kompatible Dual-Core-CPU
  • 2 GB RAM
  • DX11-kompatible Grafikkarte
  • Microsoft 64bit Windows 7
  • Mindestens 3 GB freier Festplattenspeicher

Die empfohlenen Spezifikationen sind jedoch:

  • 64-Bit-Intel-kompatibler Quad-Core-Prozessor
  • 8 GB RAM
  • DX11-basierte Grafikkarte
  • Microsoft 64bit Windows 10
  • SSD-Festplatte

Für die Mac OS-Plattform benötigen Sie mindestens die folgenden Mindestanforderungen:

  • Mac OSX El Capitan oder höher
  • Ein Intel Dual Core i3-Prozessor
  • 4 GB RAM oder mehr
  • Xcode 7 oder höher

Die empfohlenen Spezifikationen sind jedoch:

  • Mac OS
  • Ein Intel Quad Core i5 Prozessor
  • 8 GB RAM
  • SSD-Festplatte

Sie können das GameMaker Studio 2 Installationsprogramm von der YoYo Games Website herunterladen. Nach der Installation und dem ersten Start von GameMaker Studio 2 werden Sie möglicherweise aufgefordert, ein Update durchzuführen, bevor Sie das Produkt verwenden können. Nachdem Updates installiert wurden, müssen Sie die Lizenzdetails für Ihr YoYo Games angeben (siehe hier ), bevor Sie das Produkt verwenden können. Das bedeutet, dass Sie bei der ersten Verwendung von GameMaker Studio 2 eine Internetverbindung benötigen. Nach der ersten Anmeldung sollten Sie das Produkt offline verwenden können, es ist jedoch in regelmäßigen Abständen erforderlich, dass Sie eine Verbindung herstellen, um das Anmeldekonto zu überprüfen.
