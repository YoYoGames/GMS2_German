Installieren


GameMaker Studio 2 wurde für Desktop-Computer mit den folgenden Mindestanforderungen entwickelt:

Für die Windows-Plattform müssen Sie die folgenden grundlegenden Einstellungen haben:

  • 64bit Intel kompatible Dual Core CPU
  • 2 GB RAM
  • DX11 kompatible Grafikkarte
  • Microsoft 64bit Windows 7
  • Mindestens 3 GB freier Festplattenspeicher

Die empfohlenen Spezifikationen sind jedoch:

  • 64bit Intel-kompatible Quad-Core-CPU
  • 8 GB RAM
  • DX11 basierte Grafikkarte
  • Microsoft 64bit Windows 10
  • SSD-Festplatte

Für die Mac OS-Plattform benötigen Sie mindestens die folgenden Mindestanforderungen:

  • Mac OSX El Capitan oder höher
  • Ein Intel Dual Core i3 Prozessor
  • 4 GB oder mehr RAM
  • Xcode 7 oder höher

Die empfohlenen Spezifikationen sind jedoch:

  • Mac OS
  • Ein Intel Quad Core i5-Prozessor
  • 8 GB RAM
  • SSD-Festplatte

Sie können das Installationsprogramm von GameMaker Studio 2 von der YoYo Games Website herunterladen. Nach der Installation und dem ersten Start von GameMaker Studio 2 werden Sie möglicherweise zur Aktualisierung aufgefordert, bevor Sie das Produkt verwenden können. Nachdem alle Updates installiert wurden, müssen Sie die Lizenzdetails für Ihr YoYo Games Konto angeben (siehe hier ), bevor Sie das Produkt verwenden können. Das bedeutet, dass Sie bei der ersten Verwendung von GameMaker Studio 2 eine Internetverbindung benötigen. Nach der ersten Anmeldung sollten Sie in der Lage sein, das Produkt offline zu verwenden. Es wird jedoch in regelmäßigen Abständen erforderlich sein, dass Sie eine Verbindung herstellen, um das Anmeldekonto zu überprüfen.
