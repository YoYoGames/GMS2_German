Compilerfehler


Wenn Sie Ihr Spiel erstellen, vergisst man sehr leicht, eine Variable zu deklarieren oder eine Zeichenkette falsch zu GameMaker Studio 2 usw. Diese Fehler werden von GameMaker Studio 2 aufgefangen, während das Spiel kompiliert oder ausgeführt wird und ein Fenster mit geöffnet wird Fehler gegeben sowie wo der Fehler gefunden wurde. Diese Information wird auch in der Compiler-Fehlerausgabe angezeigt, wie unten gezeigt:

Sie können auf eine der angezeigten Textzeilen klicken und Sie werden direkt zu dem Objekt oder Skript weitergeleitet, das den Fehler ausgelöst hat, und Sie können dann anhand der in der Nachricht enthaltenen Informationen genau feststellen, wo im Objekt oder Skript das Objekt liegt ein Fehler ist aufgetreten. Manchmal können diese Fehler jedoch ein wenig kryptisch erscheinen. Im Folgenden finden Sie eine vollständige Liste aller Fehler und eine kurze Erklärung ihrer Bedeutung:
