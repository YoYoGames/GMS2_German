Mikro-Webserver


Dies ist die primäre Funktion von kleinen Tools, um Dateien mit einigen der verfügbaren Zielplattformen für GameMaker Studio 2 zu verbinden und zu bedienen (wie zum Beispiel das HTML5-Ziel). Wenn Sie ein Spiel auf einem Ziel ausführen, für das der Micro Web Server erforderlich ist, wird ein neues Fenster geöffnet, sobald das Spiel auf dem ausgewählten Gerät geschaltet wird:

HINWEIS: Wenn Sie das erste Mal ein Spiel auf einem Ziel ausführen, für das der Micro Web Server erforderlich ist, erhalten Sie möglicherweise eine Firewall-Warnung. Sie sollten zulassen, dass der Webserver in diesem Fall über die Firewall kommuniziert.


Der Web-Server zeigt eine bestimmte Menge an Debug-Informationen zu den Dateien an, die geliefert wurden, und hat eine Schaltfläche, um diese unten zu löschen. Sie können die meisten anderen Informationen hier nicht bearbeiten (wie es über die Target Manager-Geräteeinstellungen festgelegt ist, aber Sie können den Wert für die Netzwerkgeschwindigkeit ändern. Wenn Sie einen anderen Wert als den Maximalwert einstellen, wird die Geschwindigkeit automatisch verringert und es Ihnen erlaubt um Spiele in einer Umgebung mit geringer Bandbreite zu testen.
