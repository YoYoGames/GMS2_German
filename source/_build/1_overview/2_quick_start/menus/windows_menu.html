Das Windows-Menü


In diesem Menü werden die allgemeinen angedockten IDE-Fenster geöffnet. Da Docks neu angeordnet werden können und die Inhalte als eigenständige Fenster entfernt oder sogar vollständig geschlossen werden können, ermöglicht es dieses Fenster, sie wieder zu öffnen, wenn Sie feststellen, dass Sie sie für irgendetwas benötigen. Die Fenster, die von hier aus geöffnet werden können, sind das Ressourcenfenster und alle verschiedenen Ausgabefenster (Ausgabe, Suche, SCM, Breakpoints und Compile / Syntax Errors).

Als Nächstes können Sie die Liste Zuletzt verwendete Windows öffnen. Außerdem können Sie das Fenster Umgebungsvariablen öffnen, in dem Sie die verschiedenen Pfade sehen können, auf die die verschiedenen Umgebungsvariablen in den Einstellungen verweisen. Diese Variablen können nicht bearbeitet werden (weder der Name noch der Pfad / die Einstellung), aber Sie können sie auswählen und kopieren / einfügen, um sie anderweitig zu verwenden.

Schließlich haben Sie die Möglichkeit, Tutorial Windows zu öffnen. Dadurch werden alle Fenster wieder geöffnet, die sich auf ein geöffnetes Tutorial-Projekt beziehen, jedoch nur, wenn es sich um ein Projekt handelt, das im Rahmen eines Tutorials erstellt wurde (andernfalls ist diese Option nicht verfügbar). Informationen zum Erstellen eigener Tutorial-Projekte finden Sie im Abschnitt zum Marketplace (da alle Tutorials von dort erstellt, gehostet und heruntergeladen werden.
