Das Dateimenü


Im Dateimenü können Sie Projekte ändern, speichern oder importieren sowie die GameMaker Studio 2 IDE-Einstellungen festlegen. Die verfügbaren Optionen sind:

  • Neues Projekt - Dies ist zum Erstellen eines neuen Projekts von Grund auf in GameMaker Studio 2. Es öffnet sich der Dialog "Speichern" und Sie werden aufgefordert, einen Speicherort für das neue Projekt auszuwählen. Nachdem Sie dem Projekt einen Namen und einen Speicherort zum Speichern gegeben haben, werden Sie in einen neuen, leeren Arbeitsbereich geleitet, um mit der Bearbeitung zu beginnen. Projektdateien werden mit dem gespeichert *.yyp Dateierweiterung.

  • Projekt öffnen - Dies öffnet den GameMaker Studio Explorer-Datei-Explorer für die Auswahl eines zuvor erstellten *.yyp Projektdatei zum Laden.

  • Projekt importieren - Wenn Sie ein Projekt mit der Software GameMaker Studio 2 1.4 von YoYo Games, können Sie diese Option wählen, um entweder die *.gmx oder der *.gmz Datei.

    HINWEIS: Um ein vorheriges GameMaker Projekt in GameMaker Studio 2 zu importieren, muss es mit der Version 1.4 erstellt worden sein, und selbst dann funktioniert es möglicherweise nicht direkt beim Import. Zwischen GameMaker Studio 2 und GameMaker Studio 2 1.4 unterscheiden sich sehr viele Dinge. GameMaker Studio 2 Ihre älteren Projekte möglicherweise etwas optimiert werden, damit sie funktionieren. Weitere Informationen finden Sie im Artikel Portierung eines GMS-Spiels auf GameMaker Studio 2 in der YoYo Games Knowledge Base und in dem Abschnitt des Handbuchs, der veraltete Funktionen und Kompatibilitätsskripte behandelt.
    WICHTIG! Importieren von alten Projekten und YYZ - Projekte ist mit der Testlizenz des Produkts nicht zur Verfügung.
  • Letzte Projekte - Mit dieser Option können Sie kürzlich verwendete Projekte anzeigen. Die Auswahl wird das wieder öffnen Startseite (sollten Sie haben es geschlossen) und Sie können ein kürzlich verwendetes Projekt aus der Liste auswählen dort gefunden.

  • Projekt speichern - Dadurch wird Ihr Spiel an dem Speicherort gespeichert, den Sie beim Erstellen des Projekts (oder an dem Ort, an dem Sie es geöffnet haben) definiert haben.

  • Projekt speichern unter - Wie oben, nur geben Sie den Speicherort und den neuen Namen des Projekts an.

  • Projekt exportieren - Dadurch wird das Projekt als einzelnes Projekt exportiert *.yyz Datei. Sehr nützlich zum Senden von Kopien an Personen oder zum Speichern von Sicherungen.
    WICHTIG! YYZ - Projekte exportieren ist nicht mit der Testlizenz des Produkts zur Verfügung.
  • Einstellungen - Wenn Sie diese Option wählen, gelangen Sie zu den Windows-Einstellungen, in denen Sie einstellen können, wie GameMaker Studio 2 aussieht und sich anfühlt.

  • Neue IDE - Wenn Sie dies auswählen, wird eine neue Instanz der IDE geöffnet, die Sie verwenden können (sie wird auf der Startseite geöffnet).

  • Mein Konto - Mit dieser Option wird Ihr Standard-Webbrowser geöffnet und Sie gelangen auf die Seite Ihres YoYo Kontos.

  • Abmelden - Wenn Sie dies auswählen, werden Sie von GameMaker Studio 2 GameMaker. Sie müssen sich erneut mit derselben Lizenz (oder einer anderen Lizenz) anmelden, um das Produkt später verwenden zu können.
    WICHTIG! Wenn Sie eine Testlizenz für GameMaker Studio 2 und dann eine der anderen Lizenzen (Desktop, Mobile, UWP, Web oder Konsole) kaufen, müssen Sie sich möglicherweise von der IDE abmelden und dann GameMaker Studio 2 neu starten und sich anmelden um sicherzustellen, dass die neue Lizenz korrekt angewendet wird (obwohl dies automatisch geschehen sollte).
  • Beenden - Beenden Sie GameMaker Studio 2 auf dem Desktop.
