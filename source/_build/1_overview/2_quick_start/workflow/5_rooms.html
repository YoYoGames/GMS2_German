Räume erstellen


Alle Spiele, die Sie in GameMaker Studio 2 benötigen mindestens einen Raum (aber es können viele, viele mehr), und ein Raum ist nur ein Platz, wo Sie Instanzen der Objekte platzieren, aus denen Ihr Spiel besteht, sowie Kacheln und andere grafische Ressourcen. Wenn Sie zuerst einen Raum in Ihrem Ressourcenbaum erstellen, wird Ihnen ein neues Fenster angezeigt, in dem Sie seine Eigenschaften bearbeiten können.

Standardmäßig werden die Raumwerkzeuge auf der linken Seite mit drei Hauptbereichen platziert:

  • Ebeneneditor: Alles in einem Raum wird auf Ebenen platziert, und es gibt verschiedene Ebenentypen, aus denen Sie auswählen können (mehr dazu später). Ebenen sind nach Tiefe sortiert und diese Tiefe definiert die Reihenfolge, in der die Ebeneninhalte beim Ausführen des Spiels auf dem Bildschirm wiedergegeben werden. Die Tiefenordnung ist von der höchsten zur niedrigsten, so dass je niedriger die Tiefe desto näher der "Kamera" und je höher die Tiefe desto weiter weg. Beispielsweise wird eine Ebene mit einer Tiefe von -300 über einer Ebene mit einer Tiefe von -100 und eine Ebene mit einer Tiefe von 1000 unter einer Ebene mit einer geringeren Tiefe dargestellt.

  • Layer-Eigenschaften: Jeder Layer, den Sie dem Raum hinzufügen, hat seine eigenen Eigenschaften, und diese Eigenschaften ändern sich abhängig vom Layer-Typ. In diesem Fenster können Sie diese Eigenschaften ändern und bearbeiten, wie die Ebene gerendert wird.

  • Zimmereigenschaften: Die Zimmer verfügen über eine Reihe von Eigenschaften zu, und diese können hier eingestellt werden. Dinge wie die Raumgröße, die aktiven Kameraansichts-Ports und ein paar andere Dinge können hier eingestellt werden.

Wir haben erwähnt, dass es verschiedene Layertypen gibt. Gehen wir also einfach darüber aus, was sie sind und wie sie beim Erstellen Ihrer Spiele verwendet werden können:

Hintergrundebene Die Hintergrundebene ist eine Ebene, die mit einer einzelnen Farbe oder einem einzelnen Bild gefüllt werden kann. Es kann innerhalb des Raums bewegt und positioniert werden, und Sie können mehrere Hintergrundebenen haben. Im Allgemeinen wird dies verwendet, wie der Name andeutet, um einen konstanten Hintergrund für die anderen Schichten im Raum zu erzeugen. Standardmäßig enthält ein neuer Raum immer eine Hintergrundebene, Sie können sie jedoch entfernen, wenn Sie sie nicht benötigen.
Instanzebene In der Instanzenebene platzieren Sie alle Instanzen der Objekte, die Sie für das Spiel benötigen. Um einer Ebene eine Instanz hinzuzufügen, klicken Sie einfach auf auf das Objekt in der Ressourcenstruktur und ziehen Sie es dann in den Arbeitsbereich des Raumeditors. Wenn Sie die Maustaste loslassen, wird dem Raum eine Instanz dieses Objekts hinzugefügt. Standardmäßig enthält ein neuer Raum immer einen Instanz-Layer, Sie können ihn jedoch entfernen, wenn Sie ihn nicht benötigen.
Tilemap-Ebene Nachdem Sie einen Kachelsatz erstellt haben, müssen Sie die Kacheln zu Ihrem Raum hinzufügen, indem Sie einen Kachelkarten-Layer erstellen. Ein Kachelkarten-Layer ist ein Layer, mit dem Sie Kacheln aus beliebigen der von Ihnen erstellten Kachel-Ressourcen hinzufügen können. Er wird automatisch so eingerichtet, dass er ein Gitter mit der Größe der Kachel-Zellen verwendet. Sie können nur eine einzelne Kachel zu jeder Gitterzelle hinzufügen. Wenn Sie also mehrere Kacheln benötigen, um denselben Platz einzunehmen, sollten Sie mehrere Kachelkarten-Layer verwenden.
Pfadschicht Die Pfadebene ist die einzige Ebene, die beim Ausführen des Spiels nicht gerendert wird. Diese Ebene ist eher eine "Komfortschicht", die Ihnen beim Erstellen oder Bearbeiten von Pfadressourcen hilft, da Sie damit Pfade innerhalb des tatsächlichen Raumbereichs hinzufügen oder bearbeiten können.
Asset-Ebene Manchmal möchten Sie einen schönen grafischen Effekt, möchten aber nicht den Overhead einer Instanz verwenden oder brauchen ihn nur zum Zeichnen. Sie können eine Kachelkarte verwenden, aber da Kachelkarten nur ein Raster und nur ein einzelnes Bild pro Rasterzelle verwenden, können sie ein wenig restriktiv sein. In diesem Fall würden Sie die Asset-Ebene verwenden. Die Asset-Ebene nimmt einfach eine Sprite-Ressource und zeichnet sie unter Verwendung der Parameter, die Sie beim Hinzufügen in den Raum festgelegt haben. Sprites können dieser Ebene auf die gleiche Weise hinzugefügt werden, wie Instanzen von Objekten zur Instanzebene hinzugefügt werden, dh: Sie klicken auf auf dem Sprite im Ressourcenbaum und ziehen Sie es dann in den Arbeitsbereich des Raumeditors.
