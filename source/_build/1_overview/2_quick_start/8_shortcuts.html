Tastenkombinationen


Es gibt eine große Anzahl von Tastaturkürzeln für GameMaker Studio 2, und obwohl die meisten in der IDE neben den Menüelementen, auf die sie sich beziehen, markiert sind, sind nicht alle für den Benutzer zu Beginn offensichtlich. Hier finden Sie eine vollständige Liste aller Tastenkombinationen für eine einfache Referenz. Im Allgemeinen fallen Verknüpfungen in die folgenden drei Kategorien:

  • Global - Globale Verknüpfungen sind diejenigen, die unabhängig von der GameMaker Studio 2 in der GameMaker Studio 2 IDE funktionieren, es sei denn, Ihr Fokus liegt auf einem bestimmten Fenster oder Gadget, das dieses überschreibt.

  • Fenster - Fensterverknüpfungen sind diejenigen, die für ein einzelnes Fenster oder eine Registerkarte spezifisch sind, es sei denn, eine Gadgetverknüpfung überschreibt sie.

  • Gadget - Gadget-Verknüpfungen sind sehr spezifisch und funktionieren nur, wenn Sie ein einzelnes Gadget verwenden (z. B. den Farbwähler).

Nachfolgend finden Sie die vollständige Liste der Verknüpfungen, die in Abschnitte mit Editoren unterteilt sind:
