array_copy


Beschreibung

Mit dieser Funktion können Sie alle oder einen Teil eines Arrays an beliebiger Stelle in ein anderes Array kopieren. Sie müssen sowohl das Quell- als auch das Zielarray (beide müssen zuvor erstellt worden sein) sowie eine Position innerhalb des Quellarrays zum Kopieren von und eine Position innerhalb des Zielarrays, in das kopiert werden soll, angeben. Schließlich müssen Sie die Länge des Arrays (oder die Länge des gewünschten Teils) angeben, die kopiert werden soll. Wenn die kopierten Daten die Länge des Zielarrays überschreiten, wird das Array erweitert, um die Daten zu akzeptieren.


Syntax:

array_copy(dest, dest_index, src, src_index, length);


Argumente:

Streit Beschreibung
dest Die ID des zu kopierenden Arrays.
dest_index Der Index innerhalb des Arrays, in das kopiert werden soll.
src Die ID des Arrays, von dem kopiert werden soll.
src_index Der Index mit dem Array, von dem aus kopiert werden soll.
length Die Länge (Anzahl der Array-Indizes), die kopiert werden soll.

Kehrt zurück:

N/A


Beispiel:

if !array_equals(inventory_array, item_array)
   {
   var len = array_length_1d(inventory_array);
   array_copy(item_array, 0, inventory_array, 0, len);
   }

Der obige Code prüft zwei Arrays, um festzustellen, ob sie äquivalente Werte enthalten, und wenn sie dies nicht tun, kopiert der Code den gesamten Inhalt des Arrays "inventory_array" in das Array "item_array".
