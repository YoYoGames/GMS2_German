array_create


Beschreibung

Mit dieser Funktion können Sie ein Array einer bestimmten Größe erstellen. Sie geben der Funktion die Länge des zu erstellenden Arrays an und geben das "Handle" für das Array zurück, das Sie dann einer Variablen zuweisen können. Für Arrays, die auf diese Weise erstellt wurden, wird jeder Eintrag auf 0 initialisiert, es sei denn, Sie geben einen (optionalen) Initialisierungswert an. Wenn Sie den zusätzlichen Wert für die Initialisierung des Arrays angeben, werden alle Indizes innerhalb des neuen Arrays auf den Wert 0 gesetzt. Beachten Sie jedoch, dass die Ausführung der Funktion länger dauert.


Syntax:

array_create(size, [value]);


Argumente:

Streit Beschreibung
size Die Größe des zu erstellenden Arrays.
value Der Wert, der zum Initialisieren aller Array-Indizes verwendet werden soll (optional).

Kehrt zurück:

Array (handle)


Beispiel:

instance_array = array_create(100, noone);

Der obige Code erstellt ein neues Array von 100 Einträgen, von denen jeder auf das Schlüsselwort festgelegt ist noone und weisen Sie es dann der Variablen "instance_array" zu.
