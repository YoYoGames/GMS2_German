is_bool


Beschreibung

Diese Funktion gibt zurück, ob eine gegebene Variable ein boolescher Wert ist ( true ior false ) oder nicht. In einigen Fällen möchten Sie prüfen, ob eine Variable in GameMaker Studio 2 einen booleschen Wert enthält und dass Sie diese Funktion verwenden würden.


Syntax:

is_bool(n);


Argumente:

Streit Beschreibung
n Das Argument zu überprüfen.


Kehrt zurück:

Boolean


Beispiel:

if is_bool(val)
   {
   global.Sound = val;
   }
else    {
   global.Sound = true;
   }

Der obige Code überprüft die Variable "val", um zu sehen, ob es sich um eine reelle Zahl handelt, und wenn dies der Fall ist, verwendet sie dann eine globale Variable.
