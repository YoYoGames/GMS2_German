ist nicht definiert


Beschreibung

Diese Funktion gibt zurück, ob eine bestimmte Variable definiert ist oder nicht. In einigen Fällen möchten Sie prüfen und sehen, welchen Datentyp eine Variable in GameMaker Studio 2 und wann Sie diese Funktion verwenden würden. Es kehrt zurück true oder false abhängig davon, ob der Wert definiert ist oder nicht. Weitere Informationen zu Datentypen finden Sie hier.


Syntax:

is_undefined(n);


Argumente:

Streit Beschreibung
n Das Argument zu überprüfen.


Kehrt zurück:

Boolean


Beispiel:

var val = ds_map_find_value(map, 13);
if is_undefined(val)
   {
   show_debug_message("Map entry does not exist!");
   }

Der obige Code überprüft die Variable "val", um zu sehen, ob sie undefiniert ist oder nicht, und zeigt eine Debug-Nachricht an, wenn dies der Fall ist.
