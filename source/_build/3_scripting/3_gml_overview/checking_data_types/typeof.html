Art der


Beschreibung

Diese Funktion gibt den Datentyp einer beliebigen Variablen als String zurück. Die möglichen Rückgabewerte sind in der folgenden Tabelle aufgeführt:

Zeichenfolge Beschreibung
number Die Variable enthält eine reelle Zahl (Gleitkommazahl)
string Die Variable enthält eine Zeichenfolge
array Die Variable verweist auf ein Array
bool Die Variable enthält einen booleschen Wert ( true / false )
int32 Die Variable enthält eine 32-Bit-Ganzzahl
int64 Die Variable enthält eine 64-Bit-Ganzzahl
ptr Die Variable enthält einen Zeiger
undefined Die Variable ist nicht definiert
null Die Variable enthält einen Nullwert (dies sollte normalerweise nicht angezeigt werden)
vec3 Die Variable enthält einen 3-Wert-Vektor
vec4 Die Variable enthält einen 4-Wert-Vektor
unknown Wert ist unbekannt. Dies sollte nie gesehen werden und bedeutet, dass etwas auf der grundlegendsten Ebene schief gelaufen ist, wie bei einem Speicherüberschreiben


Syntax:

typeof(variable);


Argumente:

Streit Beschreibung
variable Die Variable, von der der Datentyp abgerufen werden soll.


Kehrt zurück:

String (see table, above)


Beispiel:

data_str = typeof(global.ExtensionInput);

Der obige Code ruft den Datentyp ab, der von der angegebenen globalen Variablen gehalten wird, und gibt die Zeichenfolge an eine Instanzvariable zurück.
