variable_instanz_get


Beschreibung

Mit dieser Funktion können Sie den Wert von einer gegebenen benannten Variablen erhalten. Sie geben den eindeutigen Instanz-ID-Wert an (der in den Instanzeigenschaften im Raum-Editor zu finden ist oder beim Aufruf der Funktion zurückgegeben wird instance_create_layer ) sowie den Namen der Variablen, um den Wert von as als String zu erhalten (siehe Beispielcode unten). Die Funktion gibt den von der Variablen gehaltenen Wert zurück.

HINWEIS: Diese Funktion wird hauptsächlich für Kompatibilitätsskripts für importierte Projekte und den konvertierten Code für Drag & Drop-Aktionen verwendet.

WICHTIG! Wenn die Variable, die Sie erhalten, nicht existiert, gibt die Funktion das Schlüsselwort zurück undefined und Sie können Fehler bekommen, die das Spiel vom Funktionieren abhalten, wenn Sie Zweifel haben, benutzen Sie die Funktion variable_instance_exists zuerst.


Syntax:

variable_instance_get(instance_id, name);


Argumente:

Streit Beschreibung
instance_id Der eindeutige ID-Wert der zu verwendenden Instanz
name Der Name der Variablen, die abgerufen werden soll (als String)

Kehrt zurück:

Value (any data type) or undefined (if the named variable does not exist)


Beispiel:

if variable_instance_exists(id, "shields")
   {
   var ss = variable_instance_get(id, "shields");
   }
else
   }
   var ss = -1;
   }

Der obige Code prüft, ob eine Variable existiert und wenn dies der Fall ist, wird der Wert, den er enthält, abgerufen und in einer lokalen Variablen gespeichert. Wenn es nicht existiert, wird die lokale Variable auf -1 gesetzt.
