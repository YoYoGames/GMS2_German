variable_global_set


Beschreibung

Mit dieser Funktion können Sie den Wert einer gegebenen globalen Variable festlegen. Sie geben den Namen der globalen Variablen an, um den Wert als Zeichenfolge festzulegen (siehe Beispielcode unten), und dann den einzustellenden Wert (kann ein beliebiger gültiger Datentyp sein ). Wenn die globale Variable nicht bereits im Spiel vorhanden ist, wird sie erstellt und dann der Wert zugewiesen.

HINWEIS: Diese Funktion wird hauptsächlich für Kompatibilitätsskripts für importierte Projekte und den konvertierten Code für Drag & Drop-Aktionen verwendet.


Syntax:

variable_global_set(name, val);


Argumente:

Streit Beschreibung
name Der Name der globalen Variablen, die festgelegt werden soll (als String)
val Der Wert, auf den die globale Variable festgelegt werden soll

Kehrt zurück:

N/A


Beispiel:

if !variable_global_exists("enemy_num")
   {
   variable_global_set("enemy_num", 0);
   }

Der obige Code überprüft, ob eine globale Variable existiert, und wenn dies nicht der Fall ist, wird er erstellt und auf 0 gesetzt.
