async_load


Syntax:

async_load;


Rückgabe: Echt


Beschreibung

Diese Variable hat einen globalen Geltungsbereich und wird verwendet, um eine DS-Map zu speichern, wenn sie in den asynchronen Ereignissen verwendet wird, und -1 zu allen anderen Zeiten. Der tatsächliche Inhalt der DS-Zuordnung hängt vom Typ des asynchronen Ereignisses ab, in dem sie enthalten ist, und von der Funktion, die zum Auslösen des Ereignisses verwendet wurde. Informationen dazu finden Sie auf den einzelnen Seiten für diese Ereignisse.


Beispiel:

if ds_map_find_value(async_load, "id") == back
   {
   if ds_map_find_value(async_load, "status") >= 0
      {
      background_index[0] = back
      }
   }

Der obige Code wird aus dem Image Loaded Asynchrnous Event übernommen und überprüft zuerst die ID des async_load DS-Karte, die ausgefüllt wurde, dann überprüfen Sie den Status des Rückrufs. Wenn der Wert größer oder gleich 0 ist (Signalisierungserfolg), wird das Ergebnis des Rückrufs verwendet, um den Hintergrundindex auf das neu geladene Bild zu setzen.
