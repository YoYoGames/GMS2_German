lebt (veraltet)


Syntax:

lives;


Rückgabe: Echt


Beschreibung

Diese Variable hat einen globalen Geltungsbereich und wird verwendet, um einen numerischen Wert zu speichern, der normalerweise für den Spieler verwendet wird. Denken Sie daran, dass dies nicht immer der Fall ist - nur weil es "Leben" heißt, bedeutet es nicht, dass es für die Leben verwendet werden muss... es kann jeden echten oder ganzzahligen Wert enthalten, den Sie für Ihr Spiel wünschen.

WICHTIG! Diese Variable ist nur zur Unterstützung von älteren Projekten aus früheren Versionen von GameMaker und sollte nicht in neuen Projekten verwendet werden, da sie veraltet ist und in zukünftigen Versionen oder Updates möglicherweise komplett entfernt wird.


Beispiel:

if place_meeting(x, y, obj_Enemy_Bullet)
   {
   health -= 10;
   if health <= 0 lives -=1;
   }

Der obige Code prüft die aktuelle Instanz, um zu sehen, ob es eine Kollision mit der Instanz des in "obj_Enemy_Bullet" indizierten Objekts gibt, und wenn dies der Fall ist, entfernt sie 10 Punkte von der globalen Zustandsvariablen und prüft dann diesen Wert, um zu sehen, ob dies der Fall ist kleiner als oder gleich 0. Wenn dies der Fall ist, löscht die Instanz 1 aus der Variable global life.
