Position entlang Pfad festlegen


Beschreibung

Diese Aktion wird die Position einer Instanz entlang eines Pfades festlegen. Der Wert wird von 0 - 1 normalisiert. Wenn Sie also beispielsweise 0,5 festlegen, wird die Instanz genau in die Mitte des Pfads verschoben. Beachten Sie, dass Sie die Aktion Start Following Path aufgerufen haben müssen, um der Instanz einen Pfad zuzuordnen, damit dieser einen Effekt hat.


Aktionssyntax:


Argumente:

Streit Beschreibung
Position Die Position (von 0 - 1) entlang des Pfades, um die Instanz zu platzieren


Beispiel:

Der obige Aktionsblockcode startet die Instanz entlang eines Pfads und generiert dann einen zufälligen Dezimalwert zwischen 0 und 1. Dieser Wert wird in einer temporären lokalen Variablen gespeichert und zum Festlegen der Position der Instanz entlang des Pfads verwendet.
