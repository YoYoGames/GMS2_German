Raster löschen


Beschreibung

Mit dieser Aktion können Sie eine Rasterdatenstruktur auf einen einzelnen Wert zurücksetzen (jede Zelle im Raster wird auf den angegebenen Wert gesetzt). Sie liefern die Variable, die den Rasterindex enthält (wie von der Aktion Create Grid zurückgegeben ) und geben dann den Wert für das Raster an.


Aktionssyntax:


Argumente:

Streit Beschreibung
Grid Der Index (in einer Variablen gespeichert) des zu löschenden Rasters
Value Der Wert, zu dem das Raster gelöscht werden soll


Beispiel:

Der obige Aktionsblockcode löscht die gegebene Gitterdatenstruktur, um in jeder Zelle 0 zu halten, wenn eine globale Variable kleiner oder gleich 0 ist.
