In Liste einfügen


Beschreibung

Mit dieser Aktion können Sie die Indexposition eines Artikels innerhalb der angegebenen Liste wiederherstellen. Sie liefern die Variable, die den Listenindex speichert (wie von der Aktion Create List zurückgegeben ) und den Wert, nach dem Sie die Indexposition in der Liste suchen möchten, sowie eine Zielvariable zum Speichern der zurückgegebenen Indexposition für den Artikel (die kann als temporäre lokale Variable markiert werden, die bis zum Ende des Skripts oder Ereignisses verwendet wird). Wenn Sie einen Wert in eine Liste einfügen, werden alle Indizes über dem angegebenen Index nach oben verschoben. Wenn Sie also eine Liste mit 4 Elementen (Indexposition 0 bis 3) haben, platzieren Sie ein neues Element an der Indexposition 2, der vorherige Punkt 2 wird zu Punkt 3 und der vorherige Punkt 3 wird zu Punkt 4. Beachten Sie, dass, wenn der gesuchte Wert nicht in der Liste vorhanden ist, es keine Listenindexposition geben wird, die zurückgegeben werden soll, und so die Rückgabe Wert wird -1 sein.


Aktionssyntax:


Argumente:

Streit Beschreibung
List Der Index (in einer Variablen gespeichert) der Liste, in die er eingefügt werden soll
Index Die Indexposition innerhalb der Liste, um einen Wert einzufügen
Value Der Wert, der an der Position gespeichert werden soll


Beispiel:

Der obige Aktionsblockcode verwendet eine Wiederholungsschleife, um die Größe der Liste zu erhalten und dann eine zufällige Position innerhalb der Liste zu erzeugen. Er erzeugt dann einen zufälligen Wert und fügt diesen Wert an der zuvor definierten zufälligen Position in die Liste ein.
