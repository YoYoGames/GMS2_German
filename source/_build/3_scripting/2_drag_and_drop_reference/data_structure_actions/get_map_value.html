Kartenwert erhalten


Beschreibung

Mit dieser Aktion können Sie den Wert abrufen, der einem bestimmten Schlüssel in einer Kartendatenstruktur zugeordnet ist. Sie liefern die Variable, die den Map-Index enthält (wie von der Aktion Create Map zurückgegeben ) und geben dann einen "Schlüssel" (der Bezeichner in der Map für einen Wert ist) und eine Zielvariable für den zurückgegebenen Wert ( welches als temporäre lokale Variable gekennzeichnet werden kann).

HINWEIS: Wenn Sie einen Schlüssel eingeben, der in der Karte nicht vorhanden ist, wird die Aktion zurückgegeben undefined. Sie können dies mit der Aktion Wenn nicht definiert überprüfen.


Aktionssyntax:


Argumente:

Streit Beschreibung
Map Der Index (in einer Variablen gespeichert) der Karte, von dem der Wert abgerufen wird
Key Der Schlüssel zum Erhalten (Real oder String)
Target Die Zielvariable, die den zurückgegebenen Kartenwert enthält


Beispiel:

Der obige Aktionsblockcode prüft auf eine Kollision an der Instanzposition, und wenn einer gefunden wird, wird die eindeutige ID der kollidierenden Instanz in einer temporären lokalen Variablen gespeichert.
