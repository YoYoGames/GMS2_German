Index des Listenelements abrufen


Beschreibung

Mit dieser Aktion können Sie die Indexposition eines Artikels innerhalb der angegebenen Liste wiederherstellen. Sie liefern die Variable, die den Listenindex speichert (wie von der Aktion Create List zurückgegeben ) und den Wert, nach dem Sie die Indexposition in der Liste suchen möchten, sowie eine Zielvariable zum Speichern der zurückgegebenen Indexposition für den Artikel (die kann als temporäre lokale Variable markiert werden, die bis zum Ende des Skripts oder Ereignisses verwendet wird). Beachten Sie, dass, wenn der gesuchte Wert nicht in der Liste vorhanden ist, keine Listenindexposition zurückgegeben wird und der Rückgabewert daher -1 ist.


Aktionssyntax:


Argumente:

Streit Beschreibung
List Der Index (in einer Variablen gespeichert) der Liste, aus der entfernt werden soll
Value Der Wert, für den die Liste überprüft werden soll
Target Die Zielvariable zum Speichern des zurückgegebenen Index


Beispiel:

Der obige Aktionsblockcode prüft auf eine Kollision an der Instanzposition, und wenn einer gefunden wird, wird der eindeutige ID-Wert für die Instanz in einer temporären Variablen gespeichert und dann geprüft, um festzustellen, ob er innerhalb der Listendatenstruktur existiert. Wenn es vorhanden ist, wird das Objekt aus der Liste entfernt und die Instanz zerstört.
