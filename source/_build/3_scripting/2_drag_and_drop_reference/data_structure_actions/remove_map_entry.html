Entfernen Sie den Karteneintrag


Beschreibung

Mit dieser Aktion können Sie einen Schlüssel (und den entsprechenden Wert) aus einer Kartendatenstruktur entfernen. Sie liefern die Variable, die den Kartenindex enthält (wie von der Aktion Create Map zurückgegeben ) und geben dann einen "Schlüssel" (der Bezeichner innerhalb der Karte für einen Wert ist) an, und die Aktion wird diesen Schlüssel aus der Karte entfernen. Beachten Sie, dass diese Aktion nichts zurückliefert, und wenn Sie den Wert für den angegebenen Schlüssel benötigen, sollten Sie die Verwendung Get Map Wert Aktion, bevor es zu entfernen.


Aktionssyntax:


Argumente:

Streit Beschreibung
Map Der Index (in einer Variablen gespeichert) der Karte, aus dem der Wert entfernt werden soll
Key Der zu verwendende Schlüssel (Real oder String)


Beispiel:

Der obige Aktionsblockcode ruft den Wert eines bestimmten Schlüssels aus der Kartendatenstruktur ab und überprüft dann den Wert, um zu sehen, ob er kleiner oder gleich 0 ist. Wenn es der Schlüssel ist, wird er aus der Karte entfernt.
