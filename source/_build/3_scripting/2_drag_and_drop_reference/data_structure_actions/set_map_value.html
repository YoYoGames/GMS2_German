Legen Sie den Kartenwert fest


Beschreibung

Mit dieser Aktion können Sie ein Schlüssel / Wert-Paar in die Kartendatenstruktur einfügen. Sie liefern die Variable, die den Kartenindex enthält (wie von der Aktion Create Map zurückgegeben ) und geben dann einen "Schlüssel" (der den Bezeichner innerhalb der Karte für einen Wert angibt) und dann den "Wert" an, der diesem Schlüssel zugeordnet werden soll. Beide Schlüssel können entweder ganzzahlige Werte (wie 0, 200, 36712, etc...) oder Zeichenfolgen (wie "Name", "Shield" usw.) sein, aber der Wert kann ein beliebiger gültiger Datentyp sein.


Aktionssyntax:


Argumente:

Streit Beschreibung
Map Der Index (in einer Variablen gespeichert) der zu setzenden Map
Key Der zu setzende Schlüssel (Integer oder String)
Value Der Wert, der mit dem angegebenen Schlüssel gespeichert werden soll
