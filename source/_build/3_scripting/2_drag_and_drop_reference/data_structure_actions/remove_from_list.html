Aus Liste entfernen


Beschreibung

Diese Aktion kann verwendet werden, um ein Indexelement aus der angegebenen Liste zu entfernen. Sie liefern die Variable, die den Listenindex speichert (wie von der Aktion Liste erstellt ) und die Indexposition innerhalb der zu entfernenden Liste, wobei die Indexposition zwischen 1 und (Listenlänge -1) liegt. Diese Aktion gibt nichts zurück. Wenn Sie also den Wert an der Indexposition benötigen, sollten Sie Listenelement abrufen vor dem Entfernen des Indexes verwenden.


Aktionssyntax:


Argumente:

Streit Beschreibung
List Der Index (in einer Variablen gespeichert) der Liste, aus der entfernt werden soll
Index Der Index innerhalb der zu entfernenden Liste


Beispiel:

Der obige Aktionsblockcode prüft auf eine Kollision an der Instanzposition, und wenn einer gefunden wird, wird der eindeutige ID-Wert für die Instanz in einer temporären Variablen gespeichert und dann geprüft, um festzustellen, ob er innerhalb der Listendatenstruktur existiert. Wenn es vorhanden ist, wird das Objekt aus der Liste entfernt und die Instanz zerstört.
