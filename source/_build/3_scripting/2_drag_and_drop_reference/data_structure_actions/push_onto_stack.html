Drück auf den Stapel


Beschreibung

Mit dieser Aktion können Sie einen Wert auf eine Stapeldatenstruktur schieben. Sie liefern die Variable, die den Stack-Index enthält (wie von der Aktion Create Stack zurückgegeben ) und geben dann den Wert an, der an den Anfang geschoben werden soll. Bei Stapeldatenstrukturen werden Werte immer auf die "Oberseite" des Stapels und niemals am Ende oder in der Mitte hinzugefügt (wenn Sie diese Funktionalität benötigen, verwenden Sie eine Listendatenstruktur).


Aktionssyntax:


Argumente:

Streit Beschreibung
Stack Der Index (in einer Variablen gespeichert) des zu verwendenden Stapels
Value Der Wert, der auf den Stapel geschoben werden soll


Beispiel:

Der obige Aktionsblockcode erzeugt eine Instanzvariable und eine neue Stapeldatenstruktur. Der Index des Stapels wird in der neuen Variablen gespeichert, und dann wird eine Schleife ausgeführt, die 10 Instanzen erzeugt und ihre eindeutigen ID-Werte auf den Stapel verschiebt.
