Schließen Sie die Ini-Datei


Beschreibung

Diese Aktion schließt die momentan geöffnete Ini-Datei. Diese Aktion sollte in dem Moment aufgerufen werden, wenn Sie mit dem Lesen oder Schreiben in eine geöffnete Ini-Datei fertig sind (Ini-Dateien werden mit der Aktion Open Ini File geöffnet). Wenn Sie diese Aktion nicht verwenden, nachdem Sie die Aktion " In Ini-Datei schreiben" verwendet haben, wird nichts auf die Festplatte geschrieben, da die Dateiinformationen im Speicher verbleiben, bis diese Aktion aufgerufen wird. Anschließend wird sie aus dem Speicher abgerufen und beschrieben die Datei auf der Festplatte. Beachten Sie auch, dass Sie eine Fehlermeldung erhalten, wenn Sie versuchen, eine Ini-Datei zu öffnen, ohne zuvor eine andere (oder dieselbe) geschlossen zu haben.
