Wenn Datei vorhanden ist


Beschreibung

Mit dieser Aktion können Sie prüfen, ob eine Datei mit dem angegebenen Namen existiert. Sie wählen die Datei aus, nach der gesucht werden soll (als Stich und einschließlich der Erweiterung), und die Funktion wird erneut ausgeführt true wenn es existiert, und false wenn es nicht. Beachten Sie, dass dies zurückgegeben wird true wenn die Datei im Spielpaket als Eingeschlossene Datei vorhanden ist oder wenn die Datei während der Ausführung des Spiels gespeichert wurde (weitere Informationen finden Sie im Abschnitt über das Dateisystem ).

Beispiel:

Der obige Aktionsblockcode wird eine Variable zum Halten von Pufferdaten erzeugen und dann prüfen, ob zuvor eine Pufferspeicherdatei erstellt wurde. Wenn die Datei vorhanden ist, wird sie in die von uns erstellte Puffervariable geladen, aber wenn sie nicht existiert, wird sie erstellt, geschrieben und dann zum nächsten Laden geladen.
