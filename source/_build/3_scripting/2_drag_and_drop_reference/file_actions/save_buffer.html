Puffer speichern


Beschreibung

Mit dieser Aktion können Sie eine Datei mit den Daten aus einem Puffer speichern. Sie geben die eindeutige Puffer-ID an (wie beim Erstellen des Puffers mit Create Buffer zurückgegeben ) und geben einen Namen für die Datei an (als String mit der Erweiterung Ihrer Wahl), und der Puffer wird in diese Datei geschrieben erneut geladen mit Load Buffer.


Aktionssyntax:


Argumente:

Streit Beschreibung
Buffer Der eindeutige Puffer-ID-Wert des zu sichernden Puffers
Filename Der Name (als String) der zu speichernden Datei


Beispiel:

Der obige Aktionsblockcode wird eine Variable zum Halten von Pufferdaten erzeugen und dann prüfen, ob zuvor eine Pufferspeicherdatei erstellt wurde. Wenn die Datei vorhanden ist, wird sie in die von uns erstellte Puffervariable geladen, aber wenn sie nicht existiert, wird sie erstellt, geschrieben und dann zum nächsten Laden geladen.
