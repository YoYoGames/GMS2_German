Datei umbenennen


Beschreibung

Mit dieser Aktion können Sie jede Datei umbenennen, die von Ihrem Spiel gespeichert wurde. Sie geben den Namen der zu ändernden Datei (als Zeichenfolge) und dann den neuen Namen (auch als Zeichenfolge und einschließlich der Dateierweiterung) an, und die Datei wird umbenannt. Beachten Sie, dass Sie nicht jede Datei umbenennen, die als EPS - Datei enthält mit Ihrem Spiel gebündelt wurden (siehe den Abschnitt über das Dateisystem für weitere Informationen).


Aktionssyntax:


Argumente:

Streit Beschreibung
Filename Der Name (als Zeichenfolge) der umzubenennenden Datei
New Name Der neue Name (als Zeichenfolge) für die Datei


Beispiel:

Der obige Aktionsblockcode wird prüfen, ob die Datei "checkpoint.sav" existiert, und wenn dies der Fall ist, prüft er, ob die Datei "checkpoint_OLD.sav" existiert. Wenn diese Datei ebenfalls vorhanden ist, wird sie gelöscht und die ursprüngliche Datei "checkpoint.sav" wird in "checkpoint_OLD.sav" umbenannt. Schließlich wird ein Puffer als "checkpoint.sav" gespeichert (im Wesentlichen werden bei jedem Speichern eine gespeicherte Pufferdatei gesichert, so dass immer eine "aktuelle" und eine "alte" Sicherungsdatei vorhanden ist).
