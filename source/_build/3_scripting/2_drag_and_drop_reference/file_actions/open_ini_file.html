Öffnen Sie die Ini-Datei


Beschreibung

Mit dieser Aktion können Sie eine Ini-Datei zum Lesen und / oder Schreiben öffnen. Wenn die Ini-Datei an dem von Ihnen ausgewählten Ort nicht vorhanden ist, erstellt GameMaker Studio 2 möglicherweise eine solche Datei, jedoch nur, wenn Sie Daten in diese Datei schreiben. Wenn Sie nur Informationen aus der Ini-Datei gelesen haben, werden die Standardwerte für die Lesefunktion zurückgegeben, die Ini-Datei wird jedoch nicht erstellt.

Bitte beachten Sie, dass Sie immer nur eine Ini-Datei geöffnet haben können und vergessen Sie nicht, Close Ini File zu verwenden, wenn Sie die Ini-Datei gelesen oder geschrieben haben, da die Informationen bis dahin noch nicht auf der Festplatte gespeichert sind Speicher, bis die Datei geschlossen ist).
