In Ini-Datei schreiben


Beschreibung

Mit dieser Aktion können Sie einen numerischen Wert oder eine Zeichenfolge in eine Ini-Datei schreiben. Sie müssen zuvor die Ini-Datei zum Lesen / Schreiben mit der Aktion Open Ini File geöffnet haben und dann geben Sie mit dieser Aktion den Typ der zu schreibenden Daten an (entweder eine reelle Zahl oder eine Zeichenkette in ""), sowie die Abschnittskopf, der Schlüssel in diesem Abschnitt und der Wert für den Schlüssel. Um die Struktur einer Ini-Datei zu visualisieren, sehen Sie sich dieses Bild an:

Wie Sie sehen können, sind Ini-Dateien in Abschnitte unterteilt - wobei jeder Abschnitt seinen eigenen Kopfzeilennamen hat - und dann wird jeder Abschnitt anschließend in verschiedene Schlüssel / Wert-Paare aufgeteilt (ähnlich einer Kartendatenstruktur ). Beachten Sie, dass Sie Ini-Dateien selbst mit einer beliebigen Textverarbeitungssoftware (z. B. Notepad unter Windows) erstellen und sie dann als Inklusivdatei zum Lesen und Schreiben in Ihr Spiel einbeziehen können.

WICHTIG! Der Versuch, Daten aus einer Ini-Datei zu lesen, die zuvor nicht mit der Aktion Open Ini File geöffnet wurde, führt zu einem schwerwiegenden Fehler und zum Absturz Ihres Spiels.

WICHTIG! Dies schreibt nur die Daten in den Speicher und um schließlich die Daten in die Datei auf der Festplatte zu schreiben, müssen Sie Close Ini File aufrufen, sonst werden die Daten nicht gespeichert.


Aktionssyntax:


Argumente:

Streit Beschreibung
Type Die Art der zu schreibenden Daten, entweder a string (in Anführungszeichen eingeschlossen "") oder a real number
Section Der Abschnitt der Ini-Datei, in den geschrieben werden soll (als String)
Key Der Schlüssel innerhalb des zu schreibenden Abschnitts (als String)
Value Der zu schreibende Wert, entweder eine Zeichenfolge oder eine reelle Zahl basierend auf dem Argument "type" (oben)


Beispiel:

Der obige Aktionsblockcode öffnet ein Ini und schreibt dann Daten in drei verschiedene Schlüssel unter dem gleichen Abschnittskopf (eine Zeichenkette und zwei reelle Zahlen), bevor die Ini-Datei wieder geschlossen wird (wodurch die Daten auf die Platte geschrieben werden).
