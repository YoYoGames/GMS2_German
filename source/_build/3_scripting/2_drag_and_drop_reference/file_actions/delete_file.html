Datei löschen


Beschreibung

Mit dieser Aktion können Sie eine Datei löschen, die zuvor erstellt wurde. Sie geben den Dateinamen (als String) der zu löschenden Datei an. Beachten Sie, dass Sie keine Dateien löschen können, die im Spielpaket enthalten sind, sondern nur solche, die mit den Aktionen " Schließen" oder " Puffer speichern" erstellt wurden (weitere Informationen finden Sie im Abschnitt über das Dateisystem ).

Der obige Aktionsblockcode wird prüfen, ob die Datei "checkpoint.sav" existiert, und wenn dies der Fall ist, prüft er, ob die Datei "checkpoint_OLD.sav" existiert. Wenn diese Datei ebenfalls vorhanden ist, wird sie gelöscht und die ursprüngliche Datei "checkpoint.sav" wird in "checkpoint_OLD.sav" umbenannt. Schließlich wird ein Puffer als "checkpoint.sav" gespeichert (im Wesentlichen werden bei jedem Speichern eine gespeicherte Pufferdatei gesichert, so dass immer eine "aktuelle" und eine "alte" Sicherungsdatei vorhanden ist).
