Lesen Sie die Ini-Datei


Beschreibung

Mit dieser Aktion können Sie Daten aus einer Ini-Datei lesen. Sie müssen zuvor die Ini-Datei zum Lesen / Schreiben mit der Aktion Open Ini File geöffnet haben, und dann geben Sie mit dieser Aktion die Art der zu lesenden Daten an (entweder eine reelle Zahl oder eine in "" eingeschlossene Zeichenkette) Abschnittskopf und der Schlüssel innerhalb dieses Abschnitts, um die Daten abzurufen. Sie können auch einen Standardwert angeben, der entweder eine reelle Zahl eines Strings sein kann, und der Typ sollte mit dem "Typ" übereinstimmen, der in der Aktion festgelegt wurde. Diese Standardaktion wird von der Aktion zurückgegeben, wenn eine der folgenden Situationen eintritt:

  • Die von Ihnen geöffnete Ini-Datei existiert nicht
  • Die referenzierte Abschnittsüberschrift existiert nicht
  • Der Schlüssel, aus dem Sie Daten abrufen möchten, existiert nicht

Unter den obigen Umständen erhalten Sie den Standardwert zurückgegeben lieber als einen Wert aus der Ini-Datei. Welcher Wert auch immer zurückgegeben wird, wird in der von Ihnen angegebenen Zielvariablen gespeichert, die auch zu diesem Zeitpunkt erstellt werden kann, wenn sie als temporäre lokale Variable gekennzeichnet ist.

WICHTIG! Wenn Sie versuchen, Daten aus einer Ini-Datei zu lesen, die zuvor nicht mit der Aktion Open Ini File geöffnet wurde, führt dies zu einem schwerwiegenden Fehler und zum Absturz Ihres Spiels.


Aktionssyntax:


Argumente:

Streit Beschreibung
Type Die Art der zu lesenden Daten, entweder a string (in Anführungszeichen eingeschlossen "") oder a real number
Section Der Abschnitt der Ini-Datei zum Lesen von (als String)
Key Der Schlüssel innerhalb des zu lesenden Abschnitts (als String)
Default Der Standardwert, der zurückgegeben werden soll, sollte entweder die Datei, der Abschnitt oder der Schlüssel nicht vorhanden sein
Target Die Zielvariable für den Rückgabewert der Aktion
