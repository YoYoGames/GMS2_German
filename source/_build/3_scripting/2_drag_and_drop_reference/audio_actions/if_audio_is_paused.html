Wenn Audio pausiert ist


Beschreibung

Mit dieser Aktion können Sie überprüfen, ob eine Instanz einer Sound-Ressource derzeit in Ihrem Spiel pausiert ist. Sie geben die Sound-Ressource an, nach der im Asset-Explorer gesucht werden soll, und die Funktion kehrt zurück true wenn es pausiert ist und falsch, wenn es nicht ist, obwohl wenn Sie das "nicht" -Flag markieren, wird dies umgekehrt und die Funktion wird zurückkehren true wenn der Ton nicht pausiert und false Andernfalls.

Beachten Sie, dass Aktionen zum "if" -Block hinzugefügt werden sollten, wie in der folgenden Abbildung gezeigt:

Diese Aktionen werden jetzt ausgeführt, wenn das "if" ausgewertet wird true, während alle anderen Aktionen nach dem "if" -Block ausgeführt werden.
