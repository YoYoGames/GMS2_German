Audio-Aktionsbibliothek


GameMaker Studio 2 verfügt über eine vollständige Audio-Engine, die auf den Soundformaten *.ogg, *.mp3 und *.wav basiert. Sounds dieser Typen, die der IDE hinzugefügt wurden, können dann mit den unten aufgeführten Audiofunktionen in Ihrem Spiel verwendet werden. Für komplexere Dinge als grundlegende Soundeffekte oder das Abspielen eines einzelnen Musikstücks können Sie sich auf die erweiterten Audiofunktionen beziehen, mit denen Sie die Wiedergabe eines Sounds ändern können. Es gibt auch eine Reihe spezialisierter Funktionen für das Streaming von Audio, das Positionieren von Audio - für 3D-Sound - und für das Gruppieren von Audio.

WICHTIG! Wenn Sie Audio auf dem HTML5-Ziel verwenden, sollten Sie beachten, dass nicht alle Browser Web Audio unterstützen und daher bei der Ausführung möglicherweise kein Sound für Ihr Projekt abgespielt wird. Einen allgemeinen Überblick über die Unterstützung von Web Audio erhalten Sie unter dem folgenden Link: Kann ich Web Audio verwenden?.
