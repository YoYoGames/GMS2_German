Stellen Sie die Geschwindigkeit der Zeitachse ein


Beschreibung

Diese Aktion kann verwendet werden, um die Geschwindigkeit festzulegen, mit der sich eine Zeitachse durch die enthaltenen Momente bewegt. Normalerweise sollte in jedem Spielfeld die Position in der Zeitleiste um 1 erhöht werden. Sie können diesen Wert jedoch ändern, indem Sie diese Aktion auf einen anderen Wert setzen. Der Wert, den Sie auswählen, wird als Multiplikator behandelt. Bei einem Wert von 2 wird die Zeitachse beispielsweise doppelt so schnell ausgeführt, bei einem Wert von 0,5 wird sie mit der halben Geschwindigkeit ausgeführt. Für die Momente bedeutet dies, dass, wenn der Wert größer als eins ist, mehrere Momente innerhalb desselben Spielrahmens auftreten können (keine Momente werden jemals übersprungen), und wenn die Geschwindigkeit kleiner als eins ist, dann nach einem Moment, Es wird Frames geben, in denen nichts passiert, bis der nächste Moment abläuft - wenn Sie beispielsweise die Zeitachse auf 0.1 setzen, erhalten Sie einen Moment auf einem Spielfeld, dann 9 Spielframes ohne Momente und auf dem zehnten Spielfeld das Der nächste Moment wird ausgeführt.

Wenn Sie eine Zeitleiste für die Verwendung der Aktionen " Instanzzeitachse festlegen" oder " Zeitachsenstatus festlegen" festlegen, die Geschwindigkeit jedoch nicht festlegen, wird die Zeitleiste standardmäßig auf 1 Moment pro Spielframe festgelegt.
