Stellen Sie Zeitachsen-Moment ein


Beschreibung

Diese Aktion wird verwendet, um eine Zeitleiste vom aktuellen auf einen neuen Zeitpunkt zu überspringen. Zeitlinien bewegen sich durch verschiedene "Momente" vorwärts und jeder Moment nimmt typischerweise einen Spielrahmen ein (wie das Schritt-Ereignis eines Objekts). Sobald eine Timeline gestartet wurde, wird sie durch die Momente fortschreiten, bis sie das Ende erreicht, an dem sie entweder zum ersten Moment der Timeline zurückkehrt oder ganz aufhört. Dieser lineare Ansatz für Timelines kann jedoch mit dieser Aktion geändert werden, um jederzeit einen anderen Zeitpunkt festzulegen. So können Sie unter bestimmten Umständen eigene Timeline-Loops erstellen oder Abschnitte einer Timeline überspringen usw. Mit dieser Aktion können Sie Stellen Sie einfach den Zeitpunkt ein, zu dem innerhalb der Timeline gesprungen wird, und der nächste Spielframe wird in diesem Moment ausgeführt (beachten Sie, dass Sie der Instanz mit der Aktion Instanzzeitlinie festlegen ein Timeline-Timeline zugewiesen haben müssen).
