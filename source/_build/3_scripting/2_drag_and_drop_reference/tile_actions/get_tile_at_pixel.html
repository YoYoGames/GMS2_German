Get Tile bei Pixel


Beschreibung

Diese Aktion wird verwendet, um einen Kachelindex für eine bestimmte Ebene zu erhalten. Der Layer wird anhand des angegebenen Layernamens angegeben (ein String, wie er im Layer-Editor für den Layer verwendet wird). Anschließend geben Sie eine x- und y-Position ein, um das Kachelfeld zu finden, dessen Index abgerufen werden soll. Die Kachel wird diejenige sein, in der die gegebene Position in dem Raum liegt, und der Index wird auf die Zielvariable zurückgeführt. Die x- und y-Position kann als relativ zur Position der aufrufenden Instanz gekennzeichnet werden, und die Zielvariable kann als temporäre (lokale) Variable gekennzeichnet werden, die bis zum Ende des Ereignisses verwendet werden kann.


Aktionssyntax:


Argumente:

Streit Beschreibung
Layer Die Ebene, auf der die Kachel festgelegt werden soll
x Die zu überprüfende x-Position
y Die zu prüfende y-Position
Target Die Zielvariable zum Speichern des Kachelindex


Beispiel:

Der obige Aktionsblockcode prüft, ob ein Mausklick auf die linke Maustaste gedrückt wird, und wenn einer erkannt wird, erhält er den Index der Kachelzelle, in die die Position der Maus fällt. Dieser Kachelindex wird dann überprüft und wenn er nicht 0 ist, wird er auf 0 gesetzt.
