Fliesen auf Pixel setzen


Beschreibung

Mit dieser Aktion können Sie eine Kachel auf einer bestimmten Ebene auf einen neuen Index setzen. Der Layer wird über den angegebenen Layernamen angegeben (ein String, der dem Layer im Raum-Editor zugewiesen wird). Anschließend geben Sie eine x- und y-Position ein, um die Kachel festzulegen. Die x- und y-Position kann als relativ zur Position der aufrufenden Instanz gekennzeichnet werden, und die einzustellende Kachel ist diejenige, in der die gegebene Position in den Raum fällt, und der verwendete Kachelsatz ist der der Ebene zugewiesene im Raum-Editor. Schließlich geben Sie den Kachelindex an, auf den die Kachel eingestellt werden soll (Kacheln werden von links nach rechts von oben nach unten indiziert, wobei oben links 0 steht). Beachten Sie, dass Sie den Kachelindex immer auf 0 setzen können, um die Kachel zu löschen.


Aktionssyntax:


Argumente:

Streit Beschreibung
Layer Die Ebene, auf der die Kachel festgelegt werden soll
x Die zu überprüfende x-Position
y Die zu prüfende y-Position
Tile Der neue Index der zu verwendenden Kachel


Beispiel:

Der obige Aktionsblockcode prüft, ob ein Mausklick auf die linke Maustaste gedrückt wird, und wenn einer erkannt wird, erhält er den Index der Kachelzelle, in die die Position der Maus fällt. Dieser Kachelindex wird dann überprüft und wenn er nicht 0 ist, wird er auf 0 gesetzt.
