Kacheldatenumwandlung festlegen


Beschreibung

Mit dieser Aktion können Sie eine von drei Kacheltransformationen in den Kacheln ändern. Sie geben die Variable an, die die Kachel für die Kachel enthält, und wählen dann die zu ändernde Transformation aus. Wenn Sie auf das Kontrollkästchen "Anwenden" klicken, wird das Transformieren angewendet und das Häkchen wird entfernt. Das tiledata ist einfach ein Wert, der den Index der Kachel zusammen mit den angewendeten Transformationen sowie den von Ihnen verwendeten benutzerdefinierten Kachelmasken widerspiegelt. Mit dieser Aktion können Sie drei dieser Werte ändern: spiegeln, spiegeln und drehen. Beachten Sie, dass Sie das tiledata normalerweise zuerst entweder mit Get Tile Data In Cell oder Get Tile Data At Pixel abrufen möchten, dann die erforderlichen Transforms anwenden und dann die Kachel in der TileMap mit Set Tile Data In Cell oder Set erneut setzen Kacheldaten bei Pixel, da diese Aktion nur einen Wert festlegt, aber nicht auf die Kachel, von der sie übernommen wurde. Weitere Informationen zu tiledata finden Sie im GML-Abschnitt zu Tilemaps - Tile-Funktionen.
