Get Tile Daten bei Pixel


Beschreibung

Mit dieser Aktion können Sie die Kacheldaten für eine Kachelzelle auf einer Kachelkartenebene von einer Position abrufen. Das tiledata ist einfach ein Wert, der den Index der Kachel zusammen mit den Werten für Spiegeln, Spiegeln und Drehen sowie den benutzerdefinierten Kachelmasken, die Sie verwendet haben, widerspiegelt. Mit dieser Aktion versorgen Sie zuerst die Ebene zum Ziel mit dem Namen der Ebene (eine Zeichenfolge) wie im Raum-Editor definiert, und dann die X- und Y-Achse im Raum, um die Kachel zu erhalten. Der tiledata-Wert, der zurückgegeben wird, stimmt überein mit der Tilemap-Zelle, die sich an diesem Punkt befindet. Wenn Ihre Tileset-Kacheln zum Beispiel 16x16 sind, enthält der Tilemap-Layer Kacheln in Zellen von 16x16 Pixeln. Wenn Sie also diese Aktion zum Targeting der Raumposition (8, 8) verwenden, erhalten Sie die Kacheldaten für die Zelle ( 0, 0) der Kachelkarte.

HINWEIS: Die ausgewählte Ebene muss im Raumeditor als Kachelkarte definiert worden sein, da sonst Fehler auftreten können.


Der zurückgegebene Wert wird dann in der Zielvariablen gespeichert, die zuvor erstellt wurde oder eine neue temporäre Variable sein kann (wenn Sie das Kontrollkästchen "Temp" aktivieren). Weitere Informationen zu tiledata finden Sie im GML-Abschnitt zu Tilemaps - Tile-Funktionen.
