Get Tile In Zelle


Beschreibung

Mit dieser Aktion können Sie einen Kachelindex für eine bestimmte Ebene erhalten. Der Layer wird anhand des angegebenen Layernamens angegeben (ein String, der zum Benennen des Layers im Raum-Editor verwendet wird). Anschließend geben Sie die Zeile und die Spalte der Zelle innerhalb des Layers an, von dem der Kachelindex abgerufen werden soll. Schließlich geben Sie die Zielvariable an, in der der zurückgegebene Kachelindex gespeichert wird. Beachten Sie, dass die Zielvariable als temporäre (lokale) Variable gekennzeichnet werden kann, die bis zum Ende des Ereignisses verwendet werden kann.


Aktionssyntax:


Argumente:

Streit Beschreibung
Layer Die Ebene, von der die Kachel abgerufen werden soll
Column Die Spalte (von links nach rechts) der Zelle
Row Die Zeile (von oben nach unten) der Zelle
Target Die Zielvariable für den Rückgabewert


Beispiel:

Der obige Aktionsblockcode erstellt eine Schleife, um durch die oberste Zeile der gegebenen Raumebene zu gehen, und überprüft dabei, ob die in der Zelle für die Spalte indizierte Kachel auf 0 gesetzt ist. Wenn es nicht der Kachelindex ist für diese Zelle wird auf 0 gesetzt und die temporäre (lokale) Variable wird inkrementiert, bevor die Schleife einen weiteren Satz von Überprüfungen durchführt.
