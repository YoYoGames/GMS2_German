Typen Bibliothek


GameMaker Studio 2 verfügt über verschiedene Datentypen wie Ganzzahlen, Strings, Zeiger usw. und einige davon können in andere konvertiert werden. Dieser Abschnitt des Handbuchs zeigt die verschiedenen Aktionen, die für den Umgang und die Konvertierung verschiedener Datentypen zur Verfügung stehen.

Hinweis: Sie können mehr über die verschiedenen Datentypen finden Sie heraus hier.

Die Typenbibliothek hat die folgenden Aktionen:

Dezimal zu Ganzzahl
Zeichenfolge an Nummer
Nummer zu Zeichenfolge
