Zeichenfolge an Nummer


Beschreibung

Diese Aktion kann eine Zeichenfolge in eine Zahl konvertieren. Sie geben die zu ändernde Zeichenfolge an und geben dann eine Zielvariable zurück, auf die die reelle Zahl zurückgeben soll (diese Variable kann als temporäre lokale Variable gekennzeichnet werden). Beachten Sie, dass die Zeichenfolge nicht konvertiert wird, wenn die Zeichenfolge andere Zeichen als Zahlen und einen Dezimalpunkt hat. Wenn die Zeichenfolge jedoch mit einer Zahl beginnt, werden diese konvertiert und der Rest der Zeichenfolge wird ausgeschlossen. Es ist wichtig zu beachten, dass beim Zurückgeben von Dezimalstellen in diese Aktion die zurückgegebene Zeichenfolge nur auf zwei Dezimalstellen gerundet wird. Im Folgenden finden Sie eine Tabelle der Eingabewerte und was Sie erwarten würden zurückgegeben

Zeichenfolge Zurückgegebener Wert
"123" 123
"123 Hello World" 123
"Hello World" 0 (standardmäßig, wenn die Zeichenfolge nicht konvertiert werden kann, wird 0 zurückgegeben)
"Hello World 123" 0
"123.45678" 123,46


Aktionssyntax:


Argumente:

Streit Beschreibung
Value Die zu konvertierende Zeichenfolge
Target Die Zielvariable, die den zurückgegebenen Wert enthält


Beispiel:

Der obige Aktionsblockcode initialisiert eine globale Variable und öffnet dann eine ini Datei. Eine Zeichenfolge wird aus der Datei gelesen und in einer temporären Variablen gespeichert, bevor sie in eine reelle Zahl konvertiert und der globalen Variablen zugewiesen wird. Das ini ist dann geschlossen.
