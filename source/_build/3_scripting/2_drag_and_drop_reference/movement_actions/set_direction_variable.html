Richtungsvariable einstellen


Beschreibung

Mit dieser Aktion wird eine Richtung für die Instanz festgelegt, die nicht auf 45 ° -Schritten festgelegt ist. Sie liefern den Richtungswert von 0 ° bis 360 ° - wo rechts 0 ° ist, oben ist 90 °, links ist 180 ° und unten ist 270 ° - und Sie können auch das relative Flag setzen, um den Wert, den Sie geben, zu addieren / subtrahieren die aktuelle Richtung.


Aktionssyntax:


Argumente:

Streit Beschreibung
Direction Richtung von 0 ° bis 360 ° (Werte außerhalb des Bereichs werden umgebrochen).


Beispiel:

Der obige Aktionsblockcode prüft auf eine mittlere Maus Drücken Sie auf die Instanz, und wenn eine erkannt wird, wird die Richtung auf einen zufälligen Wert von 0 bis 360 und die Geschwindigkeit auf 1 plus einen zufälligen Wert von 0 bis 1 festgelegt.
