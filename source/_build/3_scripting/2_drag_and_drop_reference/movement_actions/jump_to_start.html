Zum Start springen


Beschreibung

Diese Aktion wird verwendet, um eine Instanz von ihrer aktuellen Position an die Position zu verschieben, an der sie ursprünglich erstellt wurde. Die ursprüngliche Position, an der eine Instanz erstellt wird, wird in zwei Instanzvariablen gespeichert , die aufgerufen werden xstart und ystart. Diese Variablen können in jeder anderen Aktion verwendet werden und können auch so eingestellt werden, dass sie die aufgezeichnete Startposition ändern.
