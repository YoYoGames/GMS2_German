Schnappposition


Beschreibung

Diese Aktion wird verwendet, um eine Instanz an einer bestimmten Gitterkoordinatenposition zu "fangen". Sie geben die Größe der Grid- "Zelle" an, und die Instanz wird an das Grid angehängt. Wenn Sie beispielsweise die Rasterfangwerte für horizontal und vertikal auf 8 setzen und die Instanz auf (4, 6) steht, dann wird sie angezeigt wird an die Position (0, 8) gerastet - im Wesentlichen wird die aktuelle Position auf den nächsten Wert gerundet, der durch die horizontalen und vertikalen Fangeinstellungen, bei denen die Gitterzellen an der oberen linken Ecke des Raums beginnen (0, 0), teilbar ist. Die Standardwerte sind auf einen Snap-Wert von (32, 32) festgelegt, Sie können sie jedoch in einen beliebigen positiven ganzzahligen Wert ändern.

Argumente:

Streit Beschreibung
horizontal Der horizontale Zellenwert, zu dem ein Snapshot erstellt werden soll
vertical Der Wert für die vertikale Zelle, zu dem ein Snapshot erstellt werden soll
