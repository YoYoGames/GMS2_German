Holen Sie die Anzahl der Instanzen


Beschreibung

Mit dieser Aktion können Sie die Anzahl der Instanzen eines beliebigen Objekts finden, das gerade im Raum aktiv ist. Sie können ein beliebiges Objekt aus der Ressourcenliste auswählen oder das Schlüsselwort special verwenden all um die Anzahl aller Instanzen zu erhalten. Sie müssen auch eine Zielvariable angeben, um den Wert zurückzugeben, damit Sie sie in weiteren Aktionen verwenden können.


Aktionssyntax:


Argumente:

Streit Beschreibung
Object Der Name des Objekts, nach dem gesucht werden soll.
Target Die Variable für den Rückgabewert.


Beispiel:

Der obige Aktionsblockcode erstellt eine lokale (temporäre) Variable und weist ihr dann die Anzahl der Instanzen des Objekts "obj_Squirrel" zu. Dieser Wert wird dann überprüft und die Instanz wird zerstört, wenn die Anzahl der Instanzen größer als 100 ist.
