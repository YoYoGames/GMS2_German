Stellen Sie die Animationsgeschwindigkeit ein


Beschreibung

Dieser Aktionsblock setzt die image_speed (Animationsgeschwindigkeit) für das der Instanz zugewiesene Sprite. Sie legen im Sprite-Editor die idealen Frames pro Sekunde fest und können diesen Wert dann zum Ändern verwenden. Die Standardanimationsgeschwindigkeit ist auf 1 gesetzt, was bedeutet, dass das Spiel versucht, die Anzahl der Frames pro Sekunde beizubehalten, die Sie für das Sprite definiert haben. Sie können jedoch Werte größer als 1 verwenden, um schneller zu animieren (Frames können übersprungen werden) Dezimalwerte werden langsamer animiert (Bilder werden in verschiedenen Schritten angezeigt) oder sogar negative Werte, um die Geschwindigkeit der Animation umzukehren.
