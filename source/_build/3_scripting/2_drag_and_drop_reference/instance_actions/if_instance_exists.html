Wenn die Instanz existiert


Beschreibung

Diese Aktion kann verwendet werden, um zu überprüfen, ob eine Instanz eines bestimmten Objekts im Spielraum existiert oder ob eine bestimmte Instanz selbst existiert. Sie geben das Objekt aus dem Asset-Explorer, und wenn eine Instanz in dem Raum vorhanden ist, wird die Aktion zurückgegeben true Sonst wird es zurückkehren false. Beachten Sie, dass Sie hier eine Variable mit der eindeutigen ID für eine bestimmte Instanz (z. B. von der Aktion Objektinstanz erzeugen ) oder sogar den Instanz-ID-Wert verwenden können, der vom Raumeditor beim Hinzufügen der Instanz zurückgegeben wurde.

Wenn Sie das Argument "Nicht" markieren, überprüft die Aktion, ob keine Instanz existiert und wenn keine gefunden wird, wird sie zurückgegeben true und wenn es einen gibt, wird es zurückkehren false.

Beachten Sie, dass Aktionen zum "if" -Block hinzugefügt werden sollten, wie in der folgenden Abbildung gezeigt:

Diese Aktionen werden jetzt ausgeführt, wenn das "if" ausgewertet wird true, während alle anderen Aktionen nach dem "if" -Block ausgeführt werden.

Beispiel:

Der obige Aktionsblockcode prüft auf eine Kollision und gibt den eindeutigen ID-Wert der Instanz zurück, mit der kollidiert wird. Dieser Wert wird in einer temporären Variablen gespeichert und dann überprüft, ob der Wert einer Instanz entspricht, die sich gerade im Raum befindet (wenn keine Kollision gefunden wird, enthält die Variable das Schlüsselwort " noone "), und wenn dies der Fall ist, wird eine Variable in der Instanz gesetzt, mit der kollidiert wird.
