Instanz zerstören


Beschreibung

Sie können diese Aktion immer dann aufrufen, wenn Sie eine Instanz "zerstören" möchten, wodurch sie aus dem Raum entfernt wird. Sie sollten beachten, dass, wenn Sie eine Instanz zerstören, seine Destroy - Ereignis aufgerufen wird unmittelbar nach der Aktion, die die Zerstörung aufruft. Beachten Sie auch, dass die Instanz zwar nicht sofort aus dem Spiel entfernt wird, obwohl das Destroy-Ereignis ausgeführt wird, sobald die Aktion aufgerufen wird, und es wird den im aktuellen Ereignis enthaltenen Code bis zum Ende und nur beim aktuellen Ereignis ausführen ist vorbei wird die Instanz aus dem Spiel entfernt, und zu diesem Zeitpunkt wird das Aufräumereignis aufgerufen.
