Erstellen Sie eine Objektinstanz


Beschreibung

Mit dieser Aktion können Sie eine Instanz eines beliebigen Objekts aus dem Ressourcenbaum erstellen. Sie geben den Namen des Objekts und eine x / y-Position innerhalb des Raums an, in dem es erstellt werden soll. Sie können die "relativen" Flags ankreuzen, um die Instanz relativ zu der Instanz zu positionieren, die die Action-Code-Blöcke ausführt. Wenn Sie eine Instanz im Spiel erstellen, erhalten Sie einen eindeutigen ID-Wert, mit dem Sie die Instanz in weiteren Aktionen identifizieren können. Aus diesem Grund können Sie eine "Ziel" -Variable definieren, die diesen ID-Wert enthält Beispiel unten). Beachten Sie, dass das Ziel als temporäre (lokale) Variable gekennzeichnet werden kann, die für die Aktion erstellt wird und in allen folgenden Aktionen verwendet werden kann.


Aktionssyntax:


Argumente:

Streit Beschreibung
Object Der Name des Objekts, für das eine Instanz erstellt werden soll.
x Die x-Position im Raum zum Erstellen der Instanz (kann relativ zur aufrufenden Instanz sein).
y Die y-Position im Raum zum Erstellen der Instanz (kann relativ zur aufrufenden Instanz sein)
Target Die Variable für den Rückgabewert.
