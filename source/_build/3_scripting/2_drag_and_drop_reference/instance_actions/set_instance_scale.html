Setzen Sie die Instanzskala


Beschreibung

Dieser Aktionsblock setzt die image_xscale und image_yscale Werte für die Instanz. Die Eingabewerte sind hier Modifikatoren, die auf das der Instanz zugewiesene Sprite angewendet werden. Eine Skalierung von 1 (der Standardwert) gibt keine Skalierung (1: 1) an, kleinere Werte werden verkleinert (0,5, z. B. wird die Hälfte) die Breite des Sprites) und größere Werte werden größer. Sie können negative Werte verwenden, um das Sprite zu spiegeln und zu skalieren, es sei denn, der verwendete Wert ist genau -1 (in diesem Fall wird das Sprite einfach um seinen Ursprung gespiegelt oder gespiegelt, ohne Skalierung). Sie können die horizontale oder vertikale Skalierung relativ zu den aktuellen Werten festlegen. In diesem Fall werden Sie den neuen Wert von den aktuellen Skalierungswerten addieren oder subtrahieren.

Beachten Sie, dass das Ändern dieser Werte auch die Erkennung von Kollisionen durch die Instanz ändert, sofern Sie nicht eine separate Komponente angegeben haben mask_index (Kollisionsmaske) im Objekteditor. Bitte beachten Sie, dass für Änderungen in dieser Variablen, die sichtbar sein sollen, die Instanz entweder kein GameMaker Studio 2 haben sollte (und daher wird GameMaker Studio 2 standardmäßig das Sprite zeichnen) oder mit der Funktion " Selbst zeichnen" gezeichnet werden.
