Schreibpuffer


Beschreibung

Diese Aktion kann verwendet werden, um Daten in einen Puffer zu schreiben. Sie geben die eindeutige Puffer-ID zum Schreiben an (wie von der Aktion Create Buffer zurückgegeben ) und dann den zu schreibenden Wert an, der entweder ein reeller Wert oder ein String sein kann. Beachten Sie jedoch, dass alle Werte als Strings gespeichert sind Puffer (mit Lese-Puffer ) müssen Sie möglicherweise eine der Aktionen des Datentyps verwenden, um es von einer Zeichenfolge zurück in den ursprünglichen Wert zu konvertieren. Denken Sie auch daran, dass Puffer sequentiell gelesen und geschrieben werden. Wenn Sie also in einen Puffer schreiben, wird die "Such" -Position um eins verschoben, was bedeutet, dass Sie diese Aktion so oft wie nötig aufrufen.
