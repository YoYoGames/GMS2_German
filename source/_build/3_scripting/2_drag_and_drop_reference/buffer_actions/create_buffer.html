Puffer erstellen


Beschreibung

Diese Aktion kann verwendet werden, um einen Teil des Speichers als Puffer in Ihrem Spiel zuzuordnen, wobei die Funktion den eindeutigen Puffer-ID- Wert zurückgibt, der in einer Variablen gespeichert und für alle weiteren Funktionsaufrufe im Puffer verwendet werden soll. Sie können eine globale Variable oder eine Instanzvariable angeben oder die Aktion eine temporäre lokale Variable zum Speichern der Puffer-ID erstellen lassen. Beachten Sie, dass Sie zusätzliche Puffer erstellen können, indem Sie auf das Pluszeichen klicken neben der Aktion, und wählen Sie eine andere Variable, um die Puffer-ID zu halten.

HINWEIS: Es ist wichtig, dass Sie alle dynamisch erstellten Ressourcen wie diese aus dem Speicher entfernen, wenn Sie sie nicht mehr benötigen, um Speicherlecks zu vermeiden. Wenn Sie mit dem erstellten Puffer fertig sind, sollten Sie ihn wieder freigeben das Andere - Clean Up- Ereignis zum Beispiel). Dies ist besonders wichtig, wenn Sie eine temporäre lokale Variable verwenden, als ob Sie den Puffer nicht vor dem Ende des Ereignis- oder Aktionsskripts entfernen. Dann können Sie nicht mehr darauf zugreifen und können später Probleme haben.
