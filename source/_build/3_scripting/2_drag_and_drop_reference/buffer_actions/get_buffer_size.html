Puffergröße abrufen


Beschreibung

Mit dieser Aktion kann die Größe eines Puffers in Byte abgerufen werden. Sie geben den eindeutigen Puffer-ID-Wert an, der von der Aktion Create Buffer zurückgegeben wird, und geben dann eine Zielvariable an, für die der Größenwert zurückgegeben werden soll. Die Zielvariable kann als temporäre lokale Variable gekennzeichnet werden. In diesem Fall wird sie für Sie erstellt und am Ende des aktuellen Aktionsskripts oder Ereignisses wieder aus dem Speicher entfernt.


Aktionssyntax:


Argumente:

Streit Beschreibung
Buffer Der Pufferindex (in einer Variablen gespeichert)
Target Die Zielvariable, die den zurückgegebenen Wert enthält


Beispiel:

Der obige Aktionsblockcode erhält die Größe des Puffers und verwendet dann diesen Wert, um die Lese- / Schreibposition an das Ende der aktuell gespeicherten Daten zu setzen.
