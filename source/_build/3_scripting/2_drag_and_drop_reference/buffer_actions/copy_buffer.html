Puffer kopieren


Beschreibung

Mit dieser Aktion können alle in einem Puffer gespeicherten Daten in einen anderen kopiert werden. Wenn Sie zwei Puffer verwenden und von einem zum anderen kopieren, müssen beide Puffer zuvor mit der Aktion Puffer erstellen erstellt worden sein, und Sie müssen die eindeutige Puffer-ID angeben, die als Argument von / nach zurückgegeben wird.

Hinweis: Sie können nicht in demselben Puffer kopieren.


Aktionssyntax:


Argumente:

Streit Beschreibung
Buffer Der Puffer - Index (in einer Variable gespeichert ist ) zu kopieren aus
Copy To Der Pufferindex (in einer Variablen gespeichert), in den kopiert werden soll

Der obige Aktionsblockcode erstellt einen neuen Puffer und speichert seine ID in einer temporären Variablen. Es prüft dann, ob der in der gegebenen globalen Variablen indizierte Puffer existiert, und falls dies der Fall ist, wird es in den temporären Puffer kopiert und die Lese- / Schreibposition wird zurück zum Anfang zurückgesetzt, und wenn es nicht existiert, ist es das erstellt.
