Puffer lesen


Beschreibung

Diese Aktion kann verwendet werden, um Daten aus einem zuvor erstellten Puffer zu lesen. Sie geben den eindeutigen Puffer-ID-Wert an (wie von der Aktion Create Buffer zurückgegeben ) und geben eine Variable zum Speichern des zurückgegebenen Werts an. Wenn Sie die Variable als temporäre lokale Variable kennzeichnen, wird sie für Sie erstellt und am Ende des Ereignis- oder Aktionsskripts wieder verworfen. Alle Daten werden als Zeichenfolge zurückgegeben. Wenn Sie also reelle Zahlen gespeichert haben, müssen Sie eine der Datentyp-Aktionen verwenden, um sie von der Zeichenfolge in den Wert zu konvertieren. Denken Sie auch daran, dass Puffer sequentiell gelesen und geschrieben werden und das Lesen aus einem Puffer die Position "suchen" zum nächsten Eintrag verschiebt, was bedeutet, dass Sie zum Lesen verschiedener Daten die Aktion einfach die Anzahl der benötigten Zeiten nennen und speichern jedes zurückgegebene Datenstück an eine Variable.
