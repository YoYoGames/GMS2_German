Wenn Ende des Puffers


Beschreibung

Diese Aktion kann verwendet werden, um zu überprüfen, ob die Lese- / Schreibposition das Ende des Puffers erreicht hat. Sie geben den eindeutigen Puffer-ID-Wert an, der von der Aktion Create Buffer zurückgegeben wird, und dann wird die Aktion zurückgegeben true wenn die Lese- / Schreibposition das Ende aller verfügbaren Daten im Puffer erreicht hat, oder false wenn nicht. Sie können "nicht" markieren, um die Aktion zu überprüfen, wenn die Lese- / Schreibposition nicht am Ende des Puffers ist.
