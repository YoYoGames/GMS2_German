Puffer suchen


Beschreibung

Diese Aktion kann verwendet werden, um eine Position innerhalb eines Puffers zu verschieben (suchen). Sie geben den eindeutigen Puffer-ID-Wert, der von der Aktion Create Buffer zurückgegeben wurde, und anschließend den Offset-Wert in Bytes an, in den verschoben werden soll. Der Offset-Wert wird vom Anfang des Puffers an angelegt, so dass der Aufruf dieser Funktion die Lese- / Schreibposition zum Anfang des Puffers verschiebt, dann die Anzahl der mitgeteilten Bytes verschiebt und die neue Lese- / Schreibposition dort positioniert. Wenn Sie den Offset jedoch als relativ markieren, wird die Lese- / Schreibposition von der aktuellen Position aus vorwärts (oder rückwärts, wenn der Offset ein negativer Wert ist) verschoben.

Argumente:

Streit Beschreibung
