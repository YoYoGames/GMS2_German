Sprite zeichnen


Beschreibung

Diese Aktion wird ein Sprite irgendwo im Raum zeichnen. Sie geben das Sprite, die Position (entweder eine absolute Position innerhalb des Raums oder eine relative Position zu der Instanz, die die Zeichnung ausführt) und den Animationsrahmen des Sprites an, wenn es mehrere Rahmen hat (beachten Sie, dass Bildrahmen von 0 aufwärts nummeriert sind) ). Wenn Sie ein animiertes Sprite haben und möchten, dass es die Rahmen so zeichnet, als wäre es dem Objekt zugewiesen, dann verwenden Sie -1 für den Rahmenindex oder die eingebaute Instanzvariable image_index Beachten Sie jedoch, dass die Animation nicht animiert wird, wenn der Instanz kein Sprite zugewiesen ist. Wenn das zugewiesene Sprite eine andere Anzahl von Frames hat als das gezeichnete Sprite, wird das Sprite mit zusätzlichen Frames oder verpassten Frames gezeichnet.

WICHTIG: Mit dieser Aktion wird das Sprite nicht transformiert, selbst wenn Sie die Bildmischung oder -skalierung geändert haben. Sehen Sie dazu die Aktion Draw Sprite Transformed.
HINWEIS: Diese Aktion dient nur zur Verwendung in den verschiedenen Draw-Ereignissen und zeichnet nichts, wenn sie an anderer Stelle verwendet wird.
