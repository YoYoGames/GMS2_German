Zeichnen Gradient Rechteck


Beschreibung

Mit dieser Aktion wird ein Rechteck an einer bestimmten Position im Raum gezeichnet, wobei ein Satz von Mischfarben verwendet wird, um einen Farbverlauf zu erzeugen. Sie geben die obere linke Position und die untere rechte Position an, und das Rechteck wird zwischen ihnen gezeichnet, und die Position kann entweder eine absolute Position innerhalb des Raums oder eine Position relativ zu der Instanz sein, die die Aktion aufruft. Sie stellen dann die Farben so ein, dass sie von jeder der Ecken überblenden, und können auch festlegen, ob das Rechteck gefüllt oder umrahmt werden kann, indem Sie das Füllfeld unten aktivieren.

HINWEIS: Diese Aktion dient nur zur Verwendung in den verschiedenen Draw-Ereignissen und zeichnet nichts, wenn sie an anderer Stelle verwendet wird.


Aktionssyntax:


Argumente:

Streit Beschreibung
Left Die linke Position, um mit dem Zeichnen zu beginnen
Top Die oberste Position, um mit dem Zeichnen zu beginnen
Right Die richtige Position, um das Zeichnen zu beenden
Bottom Die untere Position, um das Zeichnen zu beenden
Top Left Die Farbe, die von der oberen linken Ecke aus gemischt wird
Top Right Die Farbe wird von oben rechts gemischt
Bottom Left Die Farbe wird von der linken unteren Ecke gemischt
Bottom Right Die Farbe wird von der unteren rechten Ecke gemischt
