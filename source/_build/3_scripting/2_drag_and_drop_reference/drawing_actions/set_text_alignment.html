Textausrichtung einstellen


Beschreibung

Mit dieser Aktion wird die Schriftartausrichtung für alle nachfolgenden Textaktionen zum Zeichnen festgelegt. Sie können die horizontale Ausrichtung auf links, in der Mitte oder rechts und die vertikale Ausrichtung auf oben, in der Mitte oder unten einstellen. Der Text wird relativ zur x / y-Position der Zeichenaktion ausgerichtet, wie in den Bildern unten gezeigt:

Der Text ist horizontal nach links ausgerichtet
Der Text ist horizontal zur Mitte ausgerichtet
Der Text ist horizontal nach rechts ausgerichtet
Der Text ist vertikal nach oben ausgerichtet
Der Text ist vertikal zur Mitte ausgerichtet
Der Text ist vertikal nach unten ausgerichtet

Beachten Sie, dass dadurch die Ausrichtung für alle nachfolgenden Textaufrufe zum Zeichnen festgelegt wird. Sie kann also beispielsweise einmal zu Beginn des Spiels aufgerufen werden - auf jeden Fall nicht nur das Zeichenereignis -, und alle Textaktionen werden verwendet die eingestellte Ausrichtung. Wenn Sie jedoch mehrere Textaktionen zum Ausrichten auf unterschiedliche Weise ausrichten möchten, müssen Sie diese Aktion vor jeder Zeichentextaktion aufrufen, die Sie unterschiedlich ausrichten möchten.


Aktionssyntax:


Argumente:

Streit Beschreibung
H.Align Der horizontale Ausrichtungswert (Links, Mitte, Rechts)
V.Align Der vertikale Ausrichtungswert (Oben, Mitte, Unten)

Der obige Aktionsblockcode legt die Zeichnungsfarbe, die Schriftart und die Ausrichtung für den gezeichneten Text fest.
