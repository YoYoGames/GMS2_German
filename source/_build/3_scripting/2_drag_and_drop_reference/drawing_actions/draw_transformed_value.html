Transformierten Wert zeichnen


Beschreibung

Mit dieser Aktion können Sie eine Zeichenkette und / oder eine Variable oder einen Wert auf den Bildschirm zeichnen. Das Argument caption ist eine Zeichenfolge, die Sie vor dem Wert hinzufügen möchten, und kann bei Bedarf leer gelassen werden. Der Wert kann eine Variable, ein fester Wert oder eine andere Zeichenfolge sein. In diesem Fall sollte er in Anführungszeichen eingeschlossen werden kann bei Bedarf auch leer gelassen werden. Sie geben dann die Position an, an der die Zeichenfolge gezeichnet werden soll, entweder als absolute Position oder als Position relativ zur Instanz, die den Text zeichnet. Zusätzlich können Sie die Skalierung des Textes entweder entlang der horizontalen oder vertikalen Achse sowie der Drehung einstellen (die Winkel für den rotierenden Text sind gegen den Uhrzeigersinn, dh, wenn Sie ihn auf 90 ° einstellen, wird er nach links und -90 gedreht) auf der rechten Seite). Die Skalierung erfolgt nach dem angegebenen Faktor. Eine Skalierung von 0,5 entspricht der halben Breite des Texts. Eine Skalierung von 2 verdoppelt die Breite, wobei 1 der Standardwert für keine Skalierung ist. Beachten Sie, dass die Darstellung des Textes beim Zeichnen von der verwendeten Schriftart, der verwendeten Farbe und der Ausrichtung abhängt. All diese Dinge können vor dem Zeichnen geändert werden, wie im folgenden Beispiel gezeigt.

HINWEIS: Diese Aktion dient nur zur Verwendung in den verschiedenen Draw-Ereignissen und zeichnet nichts, wenn sie an anderer Stelle verwendet wird.


Aktionssyntax:


Argumente:

Streit Beschreibung
Caption Die zu zeichnende Beschriftung (eine Zeichenfolge) kann leer bleiben
Value Der zu zeichnende Wert (Real, Variable oder String)
x Die X-Position zum Zeichnen
y Die y-Position zum Zeichnen
xscale Der Skalierungsfaktor für x (horizontal) ist der Standardwert 1
yscale Der Skalierungsfaktor für y (vertikal) ist der Standardwert 1
Rotation Der Drehwinkel zum Zeichnen des Textes
