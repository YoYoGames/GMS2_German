Linie zeichnen


Beschreibung

Diese Aktion zeichnet eine Linie an einer bestimmten Position im Raum. Sie geben die anfängliche x / y-Position und dann die endgültige x / y-Position an und die Linie wird zwischen ihnen unter Verwendung der aktuellen Zeichenfarbe gezeichnet. Die Positionen können entweder eine absolute Position innerhalb des Raums oder eine Position relativ zu der Instanz sein, die die Aktion aufruft.

HINWEIS: Diese Aktion dient nur zur Verwendung in den verschiedenen Draw-Ereignissen und zeichnet nichts, wenn sie an anderer Stelle verwendet wird.


Aktionssyntax:


Argumente:

Streit Beschreibung
x Die Startposition x
y Die Startposition y
x2 Die finishing x Position
y2 Die endgültige y-Position


Beispiel:

Der obige Aktionsblockcode zeichnet eine rosafarbene Linie über den oberen Rand des Raums.
