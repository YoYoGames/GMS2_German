Zeichnungsfarbe einstellen


Beschreibung

Mit dieser Aktion können Sie die Farbe festlegen, die für beliebige Formen (Rechteck / Ellipse / Linie) oder für jeden nachfolgend gezeichneten Text verwendet werden soll. Beachten Sie, dass dies nicht vor jeder Draw-Aktion aufgerufen werden muss und nur einmal zu Beginn des Spiels gesetzt werden kann. Wenn zum Beispiel Ihr gesamter Text gelb ist, können Sie diese Aktion im Erstell-Ereignis des ersten Raums des Spiels aufrufen, und dann werden alle Texte (und Formen) von nun an gelb gezeichnet. Wenn Sie jedoch mehrere verschiedene Farben verwenden müssen, müssen Sie dies vor jedem Text oder jeder Form, die eine andere Farbe erfordert, aufrufen.

Die Aktion hat ein Kontrollkästchen unten für Alpha verwenden. Dadurch kann die Alpha-Komponente der Farbe verwendet werden, um auch das Zeichnungsalphabet festzulegen (ähnlich wie bei Verwendung von Set Draw Alpha). Wenn Sie die Alpha-Komponente nicht einschließen und nur eine Vollfarbe haben möchten, kann dies deaktiviert werden ( es ist standardmäßig aktiviert).
