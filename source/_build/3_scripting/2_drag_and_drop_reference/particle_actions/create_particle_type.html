Partikeltyp erstellen


Beschreibung

In Ihren Projekten können Sie mehrere unterschiedliche Partikeltypen, die jeweils mit ihren eigenen Eigenschaften wie Richtung und Farbe, etc..., die die unterschiedlichen Partikeltyp Aktionen gesetzt verwenden. Allerdings müssen Sie zuerst mit dieser Aktion einen Partikeltyp erstellen, der im Wesentlichen eine kleine Speichermenge für alle Partikeldetails zuweist, damit Sie die Partikel jederzeit in der Zukunft verwenden können. Sie geben eine Variable an, die den eindeutigen ID-Wert für den Partikeltyp enthält, und dann können Sie in allen zukünftigen Aktionen, die sich auf dieses Partikel beziehen, die Variable aufrufen. Sie können festlegen, dass diese Variable als temporäre lokale Variable gekennzeichnet wird. Dies bedeutet jedoch, dass sie am Ende des Aktionsskripts oder -Ereignisses verworfen wird. Dies bedeutet, dass der Partikeltyp weiterhin vorhanden ist, Sie jedoch ohne Zugriffsmöglichkeit bleiben. Es ist also Vorsicht geboten, wenn man dies verwendet (im Allgemeinen werden Partikeltypen niemals in einer temporären lokalen Variablen gespeichert werden müssen, aber für bestimmte Dinge, wie das Ziehen von Partikeln auf eine Oberfläche, kann es erforderlich sein).

HINWEIS: Ein einmal erstellter Partikeltyp verbleibt für die Dauer des Spiels im Systemspeicher. Wenn Sie ihn also an einem beliebigen Punkt neu erstellen müssen (z. B. wenn Sie die Aktion " Spiel neu starten" aufrufen), müssen Sie ihn zerstören der aktuelle Partikeltyp bekommt man sonst ein Speicherleck. Dies passiert, wenn eine Ressource dem Speicher hinzugefügt und dann "de-referenziert" wird, was bedeutet, dass GameMaker Studio 2 keine Möglichkeit mehr hat, mit dieser Ressource zu interagieren. Dies führt dazu, dass der Speicher durch unzugängliche Ressourcen verstopft wird, was wiederum Probleme mit Ihrem Spiel verursachen kann. Um einen Partikeltyp zu zerstören, wenn er nicht mehr benötigt wird, verwenden Sie die Aktion Partikelart zerstören.


Sie können auch den Mischmodus für die Partikel festlegen. Standardmäßig werden sie mit einem "normalen" Überblendungsmodus gezeichnet, so dass die Farben nicht durch das, was unter dem Partikel gezeichnet wurde, verändert werden. Wenn die Blending Flags markiert sind, werden sie mit additivem Blending gezeichnet, was bedeutet, dass Dinge mit Änderungen gezeichnet werden zu den Farben basierend auf dem, was unter dem Teilchen gezeichnet wurde. Additive Blending bietet sehr helle und "leuchtende" Effekte und ist ideal für Explosionen, Laser, Feuer usw., während normales Mischen besser für Dinge wie Regen, Rauch oder Trümmereffekte ist.
