Partikel-System anhalten


Beschreibung

Diese Aktion wird das gegebene Partikelsystem anhalten. Sie geben den eindeutigen ID-Wert für das zu pausierende System an, wie er beim Anlegen des Systems mit der Aktion Partikelsystem erstellen zurückgegeben wurde. Ein angehaltenes Partikelsystem zeichnet immer noch die Partikel, die erzeugt wurden, aber sie werden nicht mehr ihre Farbe, Position oder Alpha aktualisieren, noch werden sie ihre Lebensdauer vervollständigen und verschwinden, bis Sie das System wieder pausieren. Beachten Sie, dass dies nicht Emittern nicht pausieren, die Teilchen strömen, noch wird es einen Emitter platzt Teilchen stoppen - Emitter noch Partikel erzeugen, wenn genannt, aber wenn das System die Partikel angehalten wird nichts tun, und genau gezogen werden, wo sie bis Sie erstellt werden Setzen Sie den angehaltenen Zustand zurück oder rufen Sie Update Particle System auf.


Aktionssyntax:


Argumente:

Streit Beschreibung
System Der eindeutige ID-Wert für das System, das angehalten werden soll

Der obige Aktionsblockcode prüft eine globale Variable und gibt sie zurück true dann wird das gegebene Partikelsystem pausiert, ansonsten wird es pausiert.
