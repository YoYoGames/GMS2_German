Stellen Sie Partikel Alpha ein


Beschreibung

Mit dieser Aktion wird das Alpha festgelegt, wenn Sie sowohl das Partikel als auch das Alpha während seiner Lebensdauer erstellen. Sie geben zuerst den Partikeltyp-ID-Wert an (der dem Wert entsprechen sollte, der beim Erstellen des Partikels mithilfe der Aktion "Partikeltyp erstellen" zurückgegeben wurde ), und anschließend können Sie die zu verwendenden Alphawerte festlegen.

Sie können drei Alphas auswählen, die verwendet werden, und das Partikel wird sich während der Lebensdauer des Partikels zwischen ihnen vermischen. Wenn zum Beispiel ein Partikel 0,5, 1 und 0 als Alpha-Werte und eine Lebensdauer von 60 Spiel-Frames hat, dann wird es, wenn es erzeugt wird (Frame 0), bei 0,5 Alpha, dann wird es zu einem vollen Alpha von 1 verschwinden (Bild 30), bevor Sie mit einem Alpha von 0 unsichtbar werden (Bild 60).


Aktionssyntax:


Argumente:

Streit Beschreibung
Type ID Der eindeutige ID-Wert für den Partikeltyp, für den die Farbe festgelegt werden soll
Start Das anfängliche zu verwendende Alpha
Middle Das zwischen Alpha zu verwenden
End Das letzte zu verwendende Alpha
