Erstellen Sie Partikelemitter


Beschreibung

Um Ihre Partikel sehen zu können, müssen Sie zunächst ein Partikelsystem erstellt haben, dann einige Partikeltypen definiert haben und schließlich mit dieser Aktion einen Partikelemitter erstellt haben. Um den Emitter zu erstellen, rufen Sie diese Aktion auf und weisen den Emitter einem Partikelsystem zu, das Sie zuvor erstellt haben (siehe Particle System ). Geben Sie dann eine Zielvariable zum Speichern des eindeutigen ID-Werts für weitere Aktionen ein. Sie können festlegen, dass diese Variable als temporäre lokale Variable gekennzeichnet wird. Dies bedeutet jedoch, dass sie am Ende des Aktionsskripts oder Ereignisses verworfen wird. Dies bedeutet, dass der Emitter weiterhin vorhanden ist, Sie jedoch ohne Zugriffsmöglichkeit bleiben Vorsicht ist bei der Verwendung geboten (im Allgemeinen müssen Emitter niemals in einer temporären lokalen Variablen gespeichert werden, aber für bestimmte Dinge, wie zum Beispiel das Zeichnen von Partikeln auf eine Oberfläche, kann es erforderlich sein).

WICHTIG! Wenn Ihr Partikelsystem persistent ist, müssen Sie entweder alle Emitter am Ende jedes Raums mit Destroy Particle Emitter löschen und sie im nächsten Raum nach Bedarf neu erstellen oder ihre ID-Werte in globalen Variablen speichern, damit auf sie zugegriffen werden kann jederzeit in jedem Raum oder lassen Sie sie durch ein persistentes Objekt erstellen, so dass das Objekt bestehen bleibt und über Instanzvariablen auf sie zugreifen kann. Beachten Sie auch, dass Sie das Partikelsystem vorher zerstören sollten, wenn Sie die Aktion " Spiel neu starten" aufrufen, so dass das System und die zugewiesenen Emitter aus dem Speicher entfernt werden, bevor Sie erneut starten. Wenn die Emitter nicht ordnungsgemäß aus dem Systemspeicher wie oben beschrieben entfernt werden, führt dies zu einem Speicherverlust. Dies passiert, wenn eine Ressource dem Speicher hinzugefügt und dann "de-referenziert" wird, was bedeutet, dass GameMaker Studio 2 keine Möglichkeit mehr hat, mit dieser Ressource zu interagieren. Dies führt dazu, dass der Speicher durch unzugängliche Ressourcen verstopft wird, was wiederum Probleme mit Ihrem Spiel verursachen kann.
