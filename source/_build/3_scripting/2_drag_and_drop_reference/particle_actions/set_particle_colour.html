Partikelfarbe einstellen


Beschreibung

Mit dieser Aktion werden die Farben oder Farben festgelegt, die beim Erstellen des Sprites und während seiner Lebensdauer verwendet werden sollen. Sie geben zuerst den Partikeltyp-ID-Wert an (der dem Wert entsprechen muss, der beim Erstellen des Partikels mithilfe der Aktion "Partikeltyp erstellen" zurückgegeben wurde ), und anschließend können Sie die zu verwendenden Farben festlegen.

Sie können drei Farben auswählen, die verwendet werden, und das Sprite wird über die Lebensdauer des Partikels zwischen ihnen gemischt. Zum Beispiel, wenn ein Partikel rote, grüne und gelbe Farben und eine Lebensdauer von 60 Spiel-Frames hat, dann wenn es erstellt wird (Frame 0) wird es rot getönt, dann wird es auf halbem Weg durch sein Leben in Richtung Grün verblassen ( Bild 30), bevor sie am Ende ihrer Lebensdauer gelb werden (Bild 60). Wenn Sie anstelle einer Partikelform ein benutzerdefiniertes Sprite verwenden, wird eine andere Farbe als Weiß mit dem verwendeten Sprite gemischt.

HINWEIS: Die Alpha-Komponente der Farbe wird nicht verwendet, sondern nur die RGB-Komponenten der Farbauswahl. Um das Alpha einzustellen, verwenden Sie die Aktion " Partikel Alpha einstellen".


Aktionssyntax:


Argumente:

Streit Beschreibung
Type ID Der eindeutige ID-Wert für den Partikeltyp, für den die Farbe festgelegt werden soll
Start Die anfängliche Farbe zu verwenden
Middle Die dazwischen Farbe zu verwenden
End Die endgültige Farbe zu verwenden
