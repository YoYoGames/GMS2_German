Aktualisierung des Partikelsystems


Beschreibung

Diese Aktion zwingt das gegebene Partikelsystem, Position, Farbe, Alpha usw. zu aktualisieren. Sie liefern den eindeutigen ID-Wert für das zu aktualisierende System (wie bei der Erstellung des Systems mit der Aktion Particle-System erstellen zurückgegeben ) und jeden einzelnen ID-Wert Wenn Sie die Aktion aufrufen, aktualisiert das angegebene System einen Spielframe, dh Sie können diese Aktion beispielsweise innerhalb einer Wiederholungsschleife aufrufen und das Partikelsystem mehrmals in einem einzelnen Spielframe aktualisieren (nützlich, wenn Sie ein Level haben möchten) um einen Effekt zu haben, der erfordert, dass Partikel gleich zu Beginn vorhanden sind, wie Schnee oder Regen). Sie können diese Aktion auch verwenden, wenn das Partikelsystem mit dem Particle Particle System unterbrochen wurde, um die Partikel im System manuell zu aktualisieren.

Argumente:

Streit Beschreibung
System Der eindeutige ID-Wert für das System, das aktualisiert werden soll
