Emit Partikel


Beschreibung

Diese Aktion wird verwendet, um Partikel von einem Emitter zu erzeugen. Sie liefern die Partikelsystem-ID, zu der der Sender gehört (wie von der Aktion Partikelsystem erzeugt ) und dann die tatsächliche Emitter-ID (wie von der Aktion Partikelemitter erzeugen zurückgegeben ) und schließlich den zu erstellenden Partikeltyp (wie von der Aktion Partikeltyp erstellen ). Sie können dann auswählen, ob Sie die Partikel aus dem Emitter "platzen" oder "streamen" möchten, und geben Sie die Anzahl der Partikel an, die Sie erstellen möchten.

Wenn Sie Partikel streamen möchten, müssen Sie diese Aktion nicht in einem Spielrahmenereignis wie dem Step-Ereignis oder dem Draw-Ereignis platzieren, sondern einfach einmal im Create-Ereignis aufrufen (zum Beispiel) und der Emitter streut Partikel alle Spiel-Frame, bis Sie diese Aktion erneut aufrufen, um sie zu stoppen (indem Sie den Zählwert auf 0 setzen). Wenn Sie die Partikel platzen lassen möchten, können Sie dies in einem beliebigen von Ihnen gewählten Programm aufrufen und die Partikel werden einmal erstellt.


Aktionssyntax:


Argumente:

Streit Beschreibung
System Die ID des Systems, in dem sich der Emitter befindet
Emitter Die ID des zu verwendenden Senders
Type Die ID des zu erstellenden Partikeltyps
Event Der Ereignistyp für die Partikel, entweder "Burst" oder "Stream"
Count Die Anzahl der zu platzenden oder zu streuenden Partikel
