Während


Beschreibung

Die While- Aktion wird zum Ausführen von Schleifen verwendet, die erst beendet werden, wenn eine bestimmte Bedingung zurückgegeben wird true. Sie können beispielsweise eine Variable verwenden und sie auf festlegen false Führen Sie dann die While-Schleife aus, und lassen Sie diese Variable überprüfen. Dann setzen Sie die Variable zu einem bestimmten Zeitpunkt in der Schleife auf true und die Schleife wird beendet. Beachten Sie, dass unabhängig davon, wo Sie die Bedingung innerhalb der while-Schleife festlegen, alle Aktionen innerhalb der Schleife bis zum Ende ausgeführt werden, bevor der Rest des Skripts oder Ereignisses fortgesetzt wird. Die Bedingung ändert sich nicht sofort aus der Schleife.

Wenn Sie Ihrem Code eine While-Aktion hinzufügen, müssen Sie die zu überprüfende Variable angeben, welche Art von Überprüfung durchgeführt werden soll (kleiner als, größer als, gleich, usw.) und den Wert, um die Variable zu überprüfen (Sie kann auch Flagge nicht die While - Schleife Kontrolle zu haben, um zu sehen, wenn die Bedingung nicht wahr ist). Sie fügen dann die Aktionen hinzu, die innerhalb der While-Schleife ausgeführt werden sollen, ähnlich wie Sie eine Aktion zu einem "if" hinzufügen würden, dh: Sie legen sie an die Seite der Aktion anstatt darunter:

Alle zur Seite hinzugefügten Aktionen werden in While enthalten und werden vor dem Ereignis ausgeführt, oder das Skript wird mit den hinzugefügten Aktionen fortgesetzt (wie im folgenden Beispiel gezeigt). Beachten Sie, dass Sie eine Schleife vorzeitig beenden können, wenn Sie den Anruf Pause Aktion.

Nachdem die Schleife beendet wurde, wird an der zufälligen Position eine neue Instanz erstellt, von der wir festgestellt haben, dass sie keine Kollisionen aufweist.
