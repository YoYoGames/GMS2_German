Beschreibung

Die For- Aktion wird zum Ausführen von Schleifen verwendet, die erst enden, wenn ein Wert die angegebene Bedingung erreicht. Sie geben einen Anfangswert als eine Variable (die als temporäre lokale Variable gekennzeichnet werden kann - das ist normalerweise das, was Sie wollen), dann geben Sie die Bedingung an, um die Variable zu überprüfen (zB wenn sie kleiner als ein bestimmter Wert ist), und dann schließlich die Menge, um die Variable bei jeder Iteration der Schleife zu inkrementieren. Also, der allgemeine Ablauf ist wie folgt:

  • Sie initialisieren eine Variable auf einen Wert
  • Sie legen die Bedingung fest, um diese Variable zu überprüfen
  • Sie legen fest, um wie viel die Variable bei jeder Iteration der Schleife erhöht werden soll
  • Sie fügen dann Aktionen in die Schleife ein, und diese werden für jede Iteration ausgeführt, bis die Bedingung zurückkehrt false oder eine Break-Aktion ist aufgetreten.

Wenn Sie Aktionen hinzufügen, die innerhalb der Schleife ausgeführt werden sollen, werden diese so hinzugefügt, als würden Sie eine Aktion zu einem "if" hinzufügen, dh: Sie legen sie auf die Seite der Aktion und nicht darunter:

Alle zur Seite hinzugefügten Aktionen werden in For enthalten sein und werden vor dem Ereignis ausgeführt, oder das Skript wird mit den Aktionen fortgesetzt, die unter hinzugefügt wurden. Beachten Sie, dass Sie eine Schleife vorzeitig beenden können, wenn Sie den Anruf Pause Aktion, und wenn For - Schleifen verwenden Sie müssen nicht erhöhen und kann verringern machen, solange die Bedingung Sie ist angemessen verwenden. Setzen Sie zum Beispiel die Anfangsvariable auf 10, setzen Sie die Bedingung auf weniger als 0 und setzen Sie dann den Inkrementwert auf -1 und die Schleife wird 10-mal durchlaufen, wobei Sie rückwärts von 10 bis 0 zählen.

Argumente:

Streit Beschreibung

Condition Die Bedingung zum Überprüfen der Schleife
Increment Die Menge, um die anfängliche Zählvariable bei jeder Iteration der Schleife zu erhöhen (oder zu dekrementieren)
