Wenn Objekt bei


Beschreibung

Diese Aktion wird verwendet, um zu überprüfen, ob eine Kollision mit einer Instanz eines bestimmten Objekts vorliegt. Sie geben das Objekt an, nach dem gesucht werden soll, die Position - die eine absolute Position oder eine Position relativ zur Instanz sein kann -, und die Aktion kehrt zurück true wenn es zu einer Kollision kommt oder false wenn nicht. Sie können auch überprüfen Sie die „Not“ Flagge zu überprüfen, ob es nicht eine Kollision an der gegebenen Position ist, und die Aktion wird dann wieder true wenn keine Kollisionen gefunden werden und false Andernfalls. Kollisionen werden auf der Grundlage der Kollisionsmaske der aufrufenden Instanz berechnet, die die Kollisionsmaske der Instanz oder des Objekts überlappt, auf die bzw. auf die geprüft wird (als ob sie an der Position "platziert" würde).

WICHTIG! Kollisionen werden nur für diejenigen Instanzen registriert, die eine gültige Kollisionsmaske haben, dh: sie haben ein Sprite, das der Kollisionsmaske zugeordnet ist sprite_index oder ein Sprite, das dem zugewiesen ist mask_index. Wenn eine der Instanzen in der Kollision keine Kollisionsmaske hat, wird die Kollision nicht erkannt, unabhängig davon, was die Instanz zu der Zeit zeichnet.
