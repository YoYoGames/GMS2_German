Wenn Objekt bei


Beschreibung

Diese Aktion wird verwendet, um zu überprüfen, ob es eine Kollision mit einer Instanz eines gegebenen Objekts innerhalb eines gegebenen geformten Bereichs oder entlang einer Linie gibt. Sie geben die Form (Rechteck, Ellipse oder Linie), das Objekt, nach dem gesucht werden soll, und dann die Koordinaten für die obere linke und untere rechte Ecke der verwendeten Form oder im Falle einer Linie die Anfangs- und Endpunkte an Endpositionen. Dies können absolute Positionen innerhalb des Spielraums oder Positionen relativ zu der Instanz sein, die die Aktion ausführt. Sie können sehen, wie dies im folgenden Bild funktioniert:

Wie Sie sehen, hängen die Ergebnisse von der Kollisionsmaske der Instanz ab, nach der gesucht wird. Das bedeutet, dass der Instanz des Objekts, nach dem Sie suchen, ein Sprite zugewiesen sein muss sprite_index oder ein Sprite, das dem zugewiesen ist mask_index. Wenn dies nicht der Fall ist, wird es von diesen Funktionen nicht erkannt, unabhängig davon, ob es ein Sprite zeichnet oder nicht. Es ist erwähnenswert, dass jede Instanz, die vollständig innerhalb der für die Ellipsen- oder Rechteckform festgelegten Grenzen liegt, ebenfalls eine Kollision zurückgibt.

Die Aktion gibt das Keyword zurück noone Wenn keine Kollision zurückgegeben wird, wird der uniq-ID-Wert der Instanz zurückgegeben, bei der eine Kollision festgestellt wurde. Sie können auch den „Not“ Flag überprüfen, um zu überprüfen, ob es nicht eine Kollision an der gegebenen Position ist. Beachten Sie, dass bei einer Kollision mehrerer Instanzen ein einzelner ID-Wert zurückgegeben wird, der einer Instanz (aber nur einer davon) entsprechen kann. Sie können die Zielvariable als temporäre (lokale) Variable kennzeichnen. Dies bedeutet, dass die Aktion diese Variable erstellt, um den Rückgabewert bis zum Ende des Ereignisses zu speichern.

Beachten Sie, dass Aktionen in einen "if" -Block eingefügt werden sollten, wenn sie auf die Seite der Aktion fallen, wie im Bild unten gezeigt:


Aktionssyntax:


Argumente:

Streit Beschreibung
Shape Die zu verwendende Form
Object Das Objekt, das auf Instanzen von
x1 Die x-Position, von der aus gestartet werden soll
y1 Die y-Position, von der aus gestartet werden soll
x2 Die X-Position zum Beenden bei
y2 Die y-Position, um zu enden
Target Die Variable, die als Ziel für den Rückgabewert der Aktion verwendet werden soll

Der Rückgabewert der Aktion wird in einer temporären (lokalen) Variablen gespeichert. Anschließend wird überprüft, ob der zurückgegebene Wert eine Instanz im Raum ist. Ist dies der Fall, wird der Instanz ein Wert aus einer Instanzvariablen entfernt.
