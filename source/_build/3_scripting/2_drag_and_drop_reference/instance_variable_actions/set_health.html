Gesundheit einstellen


Beschreibung

Mit dieser Aktion können Sie eine Instanzvariable für die Integrität erstellen oder festlegen. Sie liefern den Wert entweder gesetzt, die Gesundheit Variable oder für die Gesundheit Variable hinzuzufügen (wenn Sie die relative Checkbox dann wird der Wert auf den aktuellen „Gesundheit“ Wert hinzugefügt werden). Beachten Sie, dass dieser Wert zwischen 0 und 100 liegen sollte, wenn Sie die Aktion " Gesundheitsbalken ziehen" verwenden möchten. Andernfalls kann es sich um einen beliebigen Betrag handeln. Beachten Sie auch, dass im Gegensatz zu normalen Instanzvariablen der Variablenname für "Gesundheit" intern gespeichert wird. Um den Score-Wert abzurufen, müssen Sie die Aktion Get Health verwenden.
