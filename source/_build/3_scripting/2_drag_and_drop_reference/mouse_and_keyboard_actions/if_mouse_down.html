Wenn Mouse Down


Beschreibung

Diese Aktion prüft auf das anfängliche Drücken auf eine Maustaste. Es wird zurückkehren true Bei einem einzelnen Spiel ticken Sie, dass die Maustaste gedrückt ist und false zu allen anderen Zeiten. Wenn Sie das aktivieren not Modifikator, diese Aktion wird umgekehrt, und prüfen, ob die Maustaste nicht gedrückt wurde und zurückkehrt false auf dem Spiel ticken, dass es gedrückt wurde und true zu allen anderen Zeiten.

Beachten Sie, dass Aktionen zum "if" -Block hinzugefügt werden sollten, wie in der folgenden Abbildung gezeigt:
Diese Aktionen werden jetzt ausgeführt, wenn das "if" ausgewertet wird true, während alle anderen Aktionen nach dem "if" -Block ausgeführt werden.


Aktionssyntax:


Argumente:

Streit Beschreibung
Mouse Button Der Name der zu überprüfenden Maustaste (links, Mitte, rechts).
Not Annulliere die Prüfung (wahr wird falsch und umgekehrt)


Beispiel:

Der obige Aktionsblockcode fragt den Status der Maustaste bei jedem Schritt ab, und wenn er gedrückt wird, prüft er, ob die erste Maustaste gedrückt wird. Wenn das Häkchen beim ersten Drücken der Taste gedrückt wird, wird ein Ton wiedergegeben, und während die Taste gedrückt ist, wird die Sprite-Überblendung in Rot geändert. Wenn die Maus nicht gedrückt gehalten wird, erfolgt eine Überprüfung der Maustaste, um die Mischfarbe auf Weiß zurückzusetzen.
