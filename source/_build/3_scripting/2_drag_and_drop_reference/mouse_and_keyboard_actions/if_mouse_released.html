Wenn Maus freigegeben wird


Beschreibung

Diese Aktion überprüft, ob eine Maustaste losgelassen wurde. Es wird zurückkehren true im Einzelspiel ticken Sie, dass die Maustaste losgelassen wird und false zu allen anderen Zeiten. Wenn Sie das aktivieren not Modifikator, diese Aktion wird umgekehrt, und prüfen, ob die Maustaste nicht losgelassen wurde und zurückkehrt false auf dem Spiel ticken, dass es auf und veröffentlicht wurde true zu allen anderen Zeiten.

Beachten Sie, dass Aktionen zum "if" -Block hinzugefügt werden sollten, wie in der folgenden Abbildung gezeigt:

Diese Aktionen werden jetzt ausgeführt, wenn das "if" ausgewertet wird true, während alle anderen Aktionen nach dem "if" -Block ausgeführt werden.


Aktionssyntax:


Argumente:

Streit Beschreibung
Mouse Button Der Name der zu überprüfenden Maustaste (links, Mitte, rechts).
Not Annulliere die Prüfung (wahr wird falsch und umgekehrt)


Beispiel:

Der obige Aktionsblockcode fragt den Status der Maustaste bei jedem Schritt ab, und wenn er gedrückt wird, prüft er, ob die erste Maustaste gedrückt wird. Wenn der Scheck ist true Beim ersten Drücken der Taste wird ein Ton abgespielt. Wenn die Taste gedrückt wird, wird die Sprite-Überblendung in Rot geändert. Wenn die Maus nicht gedrückt gehalten wird, erfolgt eine Überprüfung der Maustaste, um die Mischfarbe auf Weiß zurückzusetzen.
