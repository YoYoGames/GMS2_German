Virtuelle Tastatur ausblenden


Beschreibung

Mit dieser Aktion können Sie die virtuelle Tastatur auf dem Gerät ausblenden, auf dem das Spiel ausgeführt wird. Durch Aufrufen dieser Aktion wird ein asynchrones Systemereignis generiert, in dem das async_load Die DS-Karte wird mit den folgenden Schlüssel / Wert-Paaren aufgefüllt:

  • " event_type " - Dies ist der Status der virtuellen Tastatur, wenn er durch virtuelle Tastaturaktionen ausgelöst wird.

  • " screen_height " - die Höhe der virtuellen Tastatur (in Pixel). Dies ist 0, wenn die Tastatur unsichtbar ist.

  • " keyboard_status " - der aktuelle Status der Tastatur, der als eine der folgenden Zeichenfolgen zurückgegeben wird:

    • "verstecken"
    • "versteckt"
    • "zeigen"
    • "sichtbar"
