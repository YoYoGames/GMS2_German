Schalter


Beschreibung

Diese Aktion erstellt eine neue Switch-Anweisung, der Sie weitere Fälle zur Auswertung hinzufügen können. Sie geben den Wert an, der für den Vergleich mit den Fällen im Switch verwendet werden soll, und fügen dann die verschiedenen Case- Aktionen hinzu, ähnlich wie Sie eine Aktion zu einem "if" hinzufügen würden, dh sie an die Seite der Aktion ablegen darunter:

Beachten Sie, dass Sie keinen anderen Aktionstyp zu einem Switch hinzufügen können, ohne zuvor mindestens einen Case hinzugefügt zu haben.


Aktionssyntax:


Argumente:

Streit Beschreibung
Value Der Wert, der vom Switch für die Auswertung verwendet werden soll


Beispiel:

Der obige Aktionsblockcode ruft den in einer Variablen gespeicherten Wert ab und vergleicht ihn dann mit den verschiedenen möglichen Fällen. Wenn einer der Fälle derselbe wie der Wert ist, wird die Aktion in diesem Fall ausgeführt, andernfalls wird der nächste Fall ausgewertet. Nachdem der Schalter ausgeführt wurde, wird ein Alarm auf 30 gesetzt.
