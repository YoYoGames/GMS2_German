Bibliothek wechseln


Die Aktionsbibliothek " Schalter" enthält die Aktionen, die zum Erstellen von Schalteranweisungen in Ihrem Spielaktionscode erforderlich sind. In einer Reihe von Situationen möchten Sie, dass Ihre Instanzen abhängig von einem bestimmten Wert eine Aktion abschließen. Während Sie dies mit einer Anzahl aufeinander folgender "if" -Anweisungen tun können, kann dies unordentlich werden, wenn die möglichen Auswahlmöglichkeiten über zwei oder drei liegen. In diesen Fällen ist es besser, die Aktion "switch" zu verwenden. Eine Switch-Aktion funktioniert wie folgt:

  • Sie liefern den Wert a (normalerweise von einer Art Ausdruck oder Variable)
  • Der Wert wird mit dem Wert verglichen, der den verschiedenen "case" -Anweisungen zugewiesen wurde
  • Wenn der Wert derselbe wie der Fallwert ist, wird der Inhalt des Falls ausgeführt, andernfalls wird der nächste Fall ausgewertet, usw....
  • Sobald ein Fall ausgeführt wurde oder alle Fälle ausgewertet wurden und fehlgeschlagen sind, wird der Code vom Ende des Switches an fortgesetzt
  • Wenn alle Fälle fehlgeschlagen sind und Sie einen Standardfall haben, wird dies durchgeführt, bevor der Code vom Ende des Switches an fortgesetzt wird
