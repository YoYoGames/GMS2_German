Temporäre Variable deklarieren


Beschreibung

Diese Aktion wird verwendet, um eine temporäre Variable (auch als lokale Variable bezeichnet ) zu deklarieren und einen Anfangswert zu vergeben. Hierbei handelt es sich um eine Variable, die nur für die Dauer des Ereignisses oder Skripts existiert, in dem sie deklariert wurde. Temporäre Variablen sind sehr nützlich zum Speichern von Werten, die nur für eine einzelne Operation benötigt werden, und verhindern, dass Objekte mit Variablen "überladen" werden, die nur einmal verwendet werden dürfen. Sie sind auch sehr nützlich, um komplexe Vorgänge aufzuschlüsseln oder Werte in anderen Fällen festzulegen, wenn Sie den Aktionsbereich ändern. Dies liegt daran, dass als temporär deklarierte Variablen nicht auf eine bestimmte Instanz beschränkt sind, sondern auf das Ereignis oder Skript selbst. Wenn Sie also eine temporäre Variable erstellen und dann den Aktionsbereich mit der Aktion Anwenden auf... ändern, können Sie dies trotzdem tun Überprüfen Sie die temporäre Variable ohne Probleme und ohne den Aktionsbereich zu ändern.

Wenn wir über "Werte" sprechen, meinen wir nicht nur numerische Werte, da eine Variable eine Zeichenfolge, ein Zeiger, eine Ressourcen-ID oder etwas anderes sein kann, das eine Funktion zurückgeben oder verwenden kann, sogar einen Ausdruck. Weitere Informationen finden Sie im Abschnitt über Datentypen.

Beachten Sie, dass viele Aktionen über ein Kontrollkästchen "Temp" verfügen, mit dem Sie eine neue temporäre lokale Variable erstellen können, in der der Rückgabewert der Aktion enthalten ist. Auf diese Variable kann im Aktionsskript oder Ereignis wie auf jede andere Variable zugegriffen werden. Beachten Sie außerdem, dass temporäre lokale Variablen sich auf das Skript oder das Ereignis beziehen, in dem sie erstellt werden. Sie können also von anderen Instanzen im selben Block verwendet werden, wenn Sie die Option Anwenden auf verwenden.

Sie können auch weitere Variablen in derselben Aktion hinzufügen, indem Sie auf das Plussymbol klicken neben der Aktion und geben Sie eine andere temporäre lokale Variable Name und Wert. Ausführlichere Informationen zu Variablen finden Sie im Abschnitt Variablen und Variablenbereich.
