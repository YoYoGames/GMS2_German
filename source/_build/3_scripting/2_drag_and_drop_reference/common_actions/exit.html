Ausfahrt


Beschreibung

Diese Aktion wird je nach dem Kontext, in dem sie aufgerufen wird, zwei Dinge tun. Wenn es innerhalb eines Ereignisses aufgerufen wird, beendet es das Ereignis und geht zum nächsten Ereignis über. Dabei wird der Aktionsblockcode übersprungen, der danach platziert werden kann. Wenn es jedoch in einem Action Block Script verwendet wird, wird nur das Script beendet und zu dem Ereignis, das das Script aufgerufen hat, zurückgekehrt und alle nachfolgenden Aktionen ausgeführt. Beachten Sie, dass diese in der Mitte von einem Skript aufrufen wird keinen Wert aus dem Skript zurückgeben (für, dass die Verwendung Rückkehr Aktion).
