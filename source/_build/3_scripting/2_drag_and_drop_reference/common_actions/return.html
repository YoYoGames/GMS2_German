Rückkehr


Beschreibung

Diese Aktion wird beim Erstellen von Action-Block-Skripten verwendet und wird verwendet, um einen Wert an die Instanz zurückzugeben, die das Skript aufruft. Der Rückgabewert kann alles sein, was eine Variable enthalten kann - wie eine Zeichenfolge, ein numerischer Wert, ein Zeiger oder eine Ressourcen-ID - oder sie kann selbst eine Variable sein. Beachten Sie, dass beim Aufruf der Rückgabeaktion in einem Skript das Skript beendet wird und der angegebene Wert zurückgegeben wird. Nach der Rückgabe werden keine weiteren Aktionen aufgerufen. Wenn Sie jedoch nach dem Aufruf des Skripts Aktionen im Ereignis haben, werden diese ausgeführt immer noch laufen.


Aktionssyntax:


Argumente:

Streit Beschreibung
Return Der Rückgabewert.


Beispiel:

Das obige Aktionsblock-Skript legt eine temporäre Variable für das Schlüsselwort fest noone und überprüft dann eine Instanzvariable, um festzustellen, welchen Wert sie aktuell hat. Wenn die Variable kleiner als 100 ist, wird eine Instanz erstellt und ihre eindeutige Instanz-ID in der temporären Variablen gespeichert, die dann vom Skript zurückgegeben wird.
