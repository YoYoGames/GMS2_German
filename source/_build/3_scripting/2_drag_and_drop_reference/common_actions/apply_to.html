Gelten...


Beschreibung

Diese Aktion kann verwendet werden, um den Aktionsumfang für alle weiteren Aktionen zu ändern, die an diesen angekettet sind. Wenn Sie wie bei einer if- Aktion eine weitere Aktion rechts neben der Aktion " Anwenden auf..." ablegen, wird sie von dem Objekt oder der Instanz ausgeführt, auf die der Bereich angewendet wurde. Der Bereich wird aus dem Popup-Menü ausgewählt, das angezeigt wird, wenn Sie auf den kleinen Pfeil oben rechts in der Aktion klicken. Auf diese Weise können Sie einer anderen Gruppe von Objekten oder sogar einer einzelnen Instanz mit der Instanz-ID zuweisen Führen Sie mehrere Aktionen aus, bevor Sie mit dem Code des aktuellen Objekts fortfahren.

Beachten Sie, dass zum Hinzufügen von Aktionen in den Block "Gilt für" diese an der Seite der Aktion abgelegt werden sollten, wie in der folgenden Abbildung gezeigt:

Diese Aktionen werden nun von dem Objekt oder der Instanz ausgeführt, auf die der Block "Gilt für" abzielt, während alle anderen Aktionen nach dem Block "Gilt für" ausgeführt werden.
