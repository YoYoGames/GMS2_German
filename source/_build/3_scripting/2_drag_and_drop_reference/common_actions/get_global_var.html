Globale Variable erhalten


Beschreibung

Diese Aktion wird verwendet, um den Wert einer globalen Variablen abzurufen und einer anderen Instanz oder temporären (lokalen) Variablen zuzuweisen. Die Variable, der Sie den globalen Wert zuweisen, kann eine zuvor erstellte Variable sein. Alternativ können Sie das Kontrollkästchen "Temp" aktivieren und einen neuen Variablennamen angeben, damit die Aktion eine neue temporäre Variable (lokal) erstellt, die die zurückgegebene Variable enthält Wert.

Beachten Sie, dass Sie in derselben Aktion weitere Variablen hinzufügen können, indem Sie auf das Pluszeichen klicken neben der Aktion und geben Sie einen anderen globalen Variablennamen und temporären lokalen Variablennamen. Weiterführende Informationen zu Variablen finden Sie im Abschnitt Variablen und Variablenumfang.


Aktionssyntax:


Argumente:

Streit Beschreibung
Name Der Name der globalen Variablen, von der der Wert von abgerufen werden soll.
Target Die Zielvariable, die den zurückgegebenen Wert enthält - sie sollte zuvor erstellt worden sein, wenn "Temp" nicht aktiviert ist (siehe unten).
Temp Erstellen Sie eine temporäre (lokale) Variable, die den zurückgegebenen Wert enthält (standardmäßig deaktiviert)
