Setze globale Variable


Beschreibung

Diese Aktion wird verwendet, um einer globalen Variablen entweder einen Wert zuzuweisen (im Prozess zu erstellen) oder um den Wert einer vorhandenen globalen Variablen zu ändern. Globale Variablen sind Variablen, die keiner bestimmten Instanz, sondern dem gesamten Projekt "angehören" und daher von jeder Instanz, jedem Skript oder Code aus aufgerufen und geändert werden können. Weitere Informationen zu globalen Variablen finden Sie im Abschnitt Variablen und Variablenumfang.

Sie geben den Namen der globalen Variablen und dann den Wert an, der bei Bedarf relativ zum ursprünglichen Wert hinzugefügt werden kann. Wenn Sie einen neuen Variablennamen angeben (dh eine Variable, die zuvor im gesamten Spiel noch nie zugewiesen wurde), wird dadurch die neue globale Variable erstellt und ihr der von Ihnen angegebene Wert zugewiesen, während Sie den Namen einer vorhandenen globalen Variablen angeben, es wird einfach den Wert auf den neuen Wert ändern. Beachten Sie, dass durch das Prüfen von Is relativ der Wert der vorhandenen globalen Variablen hinzugefügt wird (verwenden Sie zum Subtrahieren eine negative Zahl), und Sie können auch Ausdrücke und Variablen verwenden, die zuvor als Wert erstellt wurden.

Es ist erwähnenswert, dass wenn wir von "Werten" sprechen, wir nicht nur numerische Werte meinen, da eine Variable eine Zeichenkette, ein Zeiger, eine Ressourcen-ID oder irgendetwas anderes sein kann, das eine Funktion zurückgeben oder verwenden kann. Im Fall einer Zeichenkette wird durch die Überprüfung von Is Relative der Wert mit der zuvor gehaltenen Variablen verkettet. Wenn Ihre Variable also "Hello" war und Sie ihr "Welt" über die relative Prüfung zuweisen, lautet der endgültige Variablenwert "Hello Welt". Es muss jedoch darauf geachtet werden, dass die Variablenwerte vom gleichen Datentyp sind. Wenn Sie beispielsweise eine Ganzzahl und eine Zeichenfolge hinzufügen möchten, führt dies zu einem Compilerfehler.

Sie können auch zusätzliche Variablen in derselben Aktion hinzufügen, indem Sie auf das Plus-Symbol klicken neben der Aktion und geben Sie einen anderen globalen Variablennamen und Wert. Weiterführende Informationen zu Variablen finden Sie im Abschnitt Variablen und Variablenumfang.
