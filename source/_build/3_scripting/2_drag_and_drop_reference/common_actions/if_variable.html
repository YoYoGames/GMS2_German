Wenn Variable


Beschreibung

Diese Aktion wird verwendet, um den Status einer gegebenen Variablen anhand eines bestimmten Ausdrucks anhand eines bestimmten Ausdrucks zu überprüfen. Eine "if" -Anweisung wird immer entweder zurückgeben true oder false und die Ausdrücke zum Überprüfen können eine der folgenden sein:

  • Entspricht - Die Variable und der Wert sind beide genau gleich
  • Kleiner als - Die Variable ist kleiner als der Wert
  • Größer als - Die Variable ist größer als der Wert
  • Kleiner als oder gleich - Die Variable ist kleiner oder gleich dem Wert
  • Größer als oder gleich - Die Variable ist größer oder gleich dem Wert

Wenn Sie das Argument "Not" angeben, werden die obigen Ausdrücke negiert, z. B. wird "gleich" zu " nicht gleich". Beachten Sie auch, dass weder die Variablen- noch die Wertargumente auf numerische Werte beschränkt sind und eine Zeichenfolge, ein Zeiger, eine Ressourcen-ID oder eine andere Funktion sein können, die eine Funktion zurückgeben oder verwenden kann (weitere Informationen finden Sie im Abschnitt Datentypen ) Sie können zwei verschiedene Datentypen nicht vergleichen, ohne einen Fehler zu erhalten. Wenn die Variable beispielsweise eine Zeichenfolge enthält und Sie versuchen, zu überprüfen, ob sie einer Zahl entspricht, erhalten Sie einen Syntaxfehler.

WICHTIGER HINWEIS: Aufgrund von Gleitkommaprozisionsproblemen kann die Überprüfung, ob zwei Werte exakt gleich sind, zurückgegeben werden false, da eine genau 1 sein kann, während die andere 1,00000000000001 sein kann. Dies kann vermieden werden, indem Sie die Aktion " Dezimal zu Ganzzahl" verwenden, bevor Sie die Ausdrücke "größer als" oder "kleiner als" prüfen oder verwenden.


Beachten Sie, dass Aktionen zum "if" -Block hinzugefügt werden sollten, wie in der folgenden Abbildung gezeigt:

Diese Aktionen werden jetzt ausgeführt, wenn das "if" ausgewertet wird true, während alle anderen Aktionen nach dem "if" -Block ausgeführt werden.


Aktionssyntax:


Argumente:

Streit Beschreibung
Variable Der Name der zu überprüfenden Variablen.
Not Legen Sie fest, ob der Ausdruck nicht als wahr ausgewertet wird.
Expression Der Typ des Ausdrucks, der für die Überprüfung verwendet werden soll.
Value Der Wert, um die Variable zu überprüfen.
