Variable zuweisen


Beschreibung

Diese Aktion wird verwendet, um einer Variablen entweder einen Wert zuzuweisen (im Prozess zu erstellen) oder um den Wert einer vorhandenen Variablen zu ändern. Sie geben den Namen der Variablen und dann den Wert an, der bei Bedarf relativ zum ursprünglichen Wert hinzugefügt werden kann. Wenn Sie einen neuen Variablennamen angeben (dh eine Variable, die zuvor noch nie in der Instanz zugewiesen wurde), erstellt dies die neue Variable und weist ihr den von Ihnen angegebenen Wert zu, während Sie den Namen einer vorhandenen Variablen angeben Ändern Sie einfach den Wert in den neuen Wert. Beachten Sie, dass das Überprüfen von Relative den Wert zum vorhandenen Variablenwert hinzufügt (verwenden Sie zum Subtrahieren eine negative Zahl), und Sie können auch Ausdrücke und Variablen verwenden, die zuvor als Wert erstellt wurden.

Es ist erwähnenswert, dass wenn wir von "Werten" sprechen, wir nicht nur numerische Werte meinen, da eine Variable eine Zeichenkette, ein Zeiger, eine Ressourcen-ID oder irgendetwas anderes sein kann, das eine Funktion zurückgeben oder verwenden kann. Im Fall einer Zeichenkette wird durch die Überprüfung von Is Relative der Wert mit der zuvor gehaltenen Variablen verkettet. Wenn Ihre Variable also "Hello" war und Sie ihr "Welt" über die relative Prüfung zuweisen, lautet der endgültige Variablenwert "Hello Welt". Es muss jedoch darauf geachtet werden, dass die Variablenwerte vom gleichen Datentyp sind. Wenn Sie beispielsweise eine Ganzzahl und eine Zeichenfolge hinzufügen möchten, führt dies zu einem Compilerfehler.

Beachten Sie, dass Sie in derselben Aktion weitere Variablen hinzufügen können, indem Sie auf das Pluszeichen klicken neben der Aktion und geben Sie einen anderen Variablennamen und Wert. Weiterführende Informationen zu Variablen finden Sie im Abschnitt Variablen und Variablenumfang.

Argumente:

Streit Beschreibung

Value Der Wert, den die Variable enthalten sollte.
