Skript ausführen


Beschreibung

Diese Aktion kann zum Aufrufen eines benutzerdefinierten Skripts verwendet werden. Sie geben die Skript-Ressource an, die aufgerufen werden soll, und stellen dann alle erforderlichen Argumente bereit, z. B.: alle Werte (Zeichenfolgen, Ganzzahlen, Ressourcenindizes usw.), die Sie in das Skript eingeben möchten, um sie verwenden zu können. Sie können weitere für das Skript erforderliche Argumente hinzufügen, indem Sie auf das Pluszeichen klicken neben der Aktion.

Einige Skripts können auch einen Wert für die spätere Verwendung in der Aktionsblockliste zurückgeben. Sie können also auch die Zielvariable angeben, die diesen Rückgabewert speichern soll, der zuvor erstellt wurde oder ein neuer temporärer Wert sein kann (wenn Sie die Option "Temp" Kontrollkästchen). Sowohl das Ziel- als auch das Argumentfeld können leer gelassen werden, wenn sie nicht benötigt werden.


Aktionssyntax:


Argumente:

Streit Beschreibung
Script Der Name des anzurufenden Skripts.
Argument0... Argument3 Die verschiedenen Argumente (Werte), die an das Skript übergeben werden sollen (nicht verwendete Argumente können leer gelassen werden)
Target Der Name der Variable, die für alle zurückgegebenen Werte verwendet werden soll (kann leer gelassen werden)
