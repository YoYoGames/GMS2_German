ds_map_replace_map

Liste

Ersetzen Sie eine zuvor hinzugefügte ds_map durch eine andere ds_map durch eine andere ds_map.

Syntax:

ds_map_replace_map(id, key, value)


Streit Beschreibung
id Die ID der zu verwendenden ds_map.
key Der zu ersetzende Schlüssel
value Die ID der ds_map, die verwendet wird, um die zuvor im angegebenen Schlüssel gespeicherte ID zu ersetzen.


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie einen ersetzen ds_map das wurde in dem angegebenen "Schlüssel" mit einer anderen Karte gespeichert, die zuvor erstellt wurde. Diese Funktion wurde entwickelt, um JSON-kompatible Karten zu erstellen, die Sie dann verwenden würden json_encode und sollte nur in Verbindung mit dieser Funktionalität verwendet werden.


Beispiel:

var temp_map = ds_map_create();
ds_map_add_list(temp_map, "list", j_list);
ds_map_add(temp_map, "array", j_array);
ds_map_replace_map(j_map, "maps", temp_map);
var j = json_encode(j_map);
ds_map_destroy(temp_map);

Der obige Code erstellt ein ds_map und füllen Sie es mit einem Array und einem ds_list vor dem Ersetzen einer zuvor gespeicherten Karte in der ds_map "j_map".
