ds_map_secure_save

Diese Funktion speichert die angegebene ds_map an einem sicheren Ort auf der Zielplattform.

Syntax:

ds_map_secure_save(map, filename);


Streit Beschreibung
map Die ID der zu verwendenden Datenstruktur
filename Der Name der Datei, in der die Karte gespeichert werden soll


Rückgabe: Boolean


Beschreibung

Diese Funktion speichert den Inhalt der angegebenen ds_map in einer Datei. Die Datei selbst kann fast jede Erweiterung haben (z. B. *.dat, *.json, *.bin usw.) und wird verschlüsselt und an einem sicheren Ort auf der Zielplattform gespeichert. Auf diese Weise können Sie vertrauliche Informationen wie Kaufdetails oder Passwörter lokal auf sichere Weise speichern. Sie können die ds_map dann mit der Funktion erneut laden ds_map_secure_load().


Beispiel:

ds_map_secure_save(purchase_map, "p_data.dat");

Der obige Code speichert die in der Variablen "p_data" indizierte ds_map in die angegebene Datei für den späteren Abruf.
