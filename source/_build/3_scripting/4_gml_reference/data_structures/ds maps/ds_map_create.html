ds_map_create

Erzeugt eine neue ds_map.

Syntax:

ds_map_create();


Rückgabe: Karten-ID


Beschreibung

Diese Funktion wird zum Erstellen einer neuen, leeren ds_map verwendet und gibt ihre ID zurück, die dann für den Zugriff auf die Datenstruktur in allen anderen ds_map-Funktionen verwendet wird.

WICHTIG! Wenn Sie eine Datenstruktur erstellen, handelt es sich bei dem zu identifizierenden Indexwert um einen ganzzahligen Wert, der bei 0 beginnt. Dies bedeutet, dass Datenstrukturen verschiedener Typen denselben Indexwert haben können. Wenn Sie also Zweifel haben, sollten Sie die ds_exists Funktion, bevor Sie auf sie zugreifen. Beachten Sie auch, dass Indizes wiederverwendet werden, sodass ein zerstörter Datenstrukturindexwert anschließend von einem neu erstellten Index verwendet werden kann.


Beispiel:

inventory = ds_map_create();

Der obige Code erstellt eine neue, leere ds_map und speichert ihren ID-Index in der Variablen "inventory".
