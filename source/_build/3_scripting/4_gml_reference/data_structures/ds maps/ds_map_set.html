ds_map_set

Legen Sie einen Kartenschlüssel auf einen Wert fest

Syntax:

ds_map_set(id, key, value)


Streit Beschreibung
id Die ID der zu verwendenden Karte.
key Der Schlüssel zum einstellen.
value Der Wert, auf den der Schlüssel gesetzt werden soll.


Rückgabe: N / A


Beschreibung

Mit dieser Funktion können Sie den Wert eines Schlüssels innerhalb einer bestimmten DS-Karte festlegen. Sie geben den DS-Karten-ID-Wert (wie von der Funktion zurückgegeben) an ds_map_create() Geben Sie dann den Schlüssel, den Sie festlegen möchten, und den Wert ein, auf den Sie ihn einstellen möchten. Schlüssel können Ganzzahlen oder Zeichenfolgen sein. Wenn der angegebene Schlüssel nicht vorhanden ist, wird er für Sie erstellt. Diese Funktion entspricht der Verwendung des DS-Karten-Accessors zum Festlegen / Erstellen eines Kartenschlüssel / Wert-Paares.


Beispiel:

if is_undefined(ds_map_find_value(map, "score"))
    {
    ds_map_set(map, "score", 0);
    }

Der obige Code prüft, ob der angegebene Schlüssel existiert, und wenn nicht, wird er erstellt und festgelegt.
