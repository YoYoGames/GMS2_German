ds_map_write

Diese Funktion schreibt die Daten aus einer ds_map als String.

Syntax:

ds_map_write(id);


Streit Beschreibung
id Die ID der zu verwendenden Datenstruktur


Rückgabe: Zeichenfolge


Beschreibung

Diese Funktion wandelt die ds_map-Daten der angegebenen Map in ein String-Format um, das dann zur einfachen Speicherung in eine *.ini oder eine *.txt-Datei geschrieben werden kann. Diese Zeichenfolge kann später mit Hilfe von in eine neue ds_map zurückgelesen werden ds_map_read().

HINWEIS: Die zurückgegebene Zeichenfolge ist keine lesbare Zeichenfolge, sondern ein Speicherauszug des Inhalts der Datenstruktur


Beispiel:

ini_open("map.ini");
var t_string;
t_string = ds_map_write(inventory);
ini_write_string("Saved", "0", t_string);
ini_close();

Der obige Code öffnet eine Ini-Datei, in die geschrieben werden kann. Es benutzt dann ds_map_write() um einen String zu generieren, der in der temporären Variable "t_string" gespeichert ist. Schließlich schreibt er diese Zeichenfolge in die ini-Datei, bevor er sie schließt.
