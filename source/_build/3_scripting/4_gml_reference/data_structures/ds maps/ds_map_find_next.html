ds_map_find_next

Sucht den nächsten Schlüssel in der ds_map nach dem in der Funktion angegebenen Schlüssel und gibt ihn zurück.

Syntax:

ds_map_find_next(id, key);


Streit Beschreibung
id Die ID der zu verwendenden Karte.
key Der Schlüssel, um den nächsten zu finden.


Rückgabe: Real, String oder undefiniert


Beschreibung

Diese Funktion gibt den nächsten in der ds_map hinter dem in der Funktion angegebenen Schlüssel zurück. Dies kann nützlich sein, wenn Sie durch die ds_map nach etwas suchen müssen, aber sollte vermieden werden, wenn möglich, da es langsam sein kann. Wenn kein solcher Schlüssel existiert, kehrt die Funktion zurück <undefined>. Sie sollten dies immer mit der Taste überprüfen is_undefined() Funktion.



Beispiel:

var size, key, i;
size = ds_map_size(inventory);
key = ds_map_find_first(inventory);
for (i = 0; i < size; i++;)
   {
   if key != "gold" key = ds_map_find_next(inventory, key) else break;
   }

Der obige Code erstellt einige temporäre Variablen und erhält dann die Größe ds_map und findet den ersten Schlüssel, wie er vom Computer in der Karte gespeichert wird. Es verwendet dann a for Durchlaufen Sie die ds_map, um nach dem Schlüsselwert "gold" zu suchen. Wenn es es findet, bricht es die Schleife aus.
