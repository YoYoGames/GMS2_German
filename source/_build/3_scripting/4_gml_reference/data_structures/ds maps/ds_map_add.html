ds_map_add

Fügt den angegebenen Wert und den zugehörigen Schlüssel in die Karte ein.

Syntax:

ds_map_add(id, key, val);


Streit Beschreibung
id Die ID der hinzuzufügenden Karte.
key Der Schlüssel des hinzuzufügenden Werts.
val Der Wert, der der Karte hinzugefügt werden soll.


Rückgabe: Boolean


Beschreibung

Diese Funktion sollte verwendet werden, um Gruppen von Schlüssel / Wert-Paaren in die angegebene ds_map einzufügen. Sie können diese Funktion überprüfen, um festzustellen, ob sie erfolgreich war oder nicht, da sie fehlschlagen kann, wenn in der ds_map bereits derselbe Schlüssel vorhanden ist oder Sie eine nicht vorhandene ds_map als ID der hinzuzufügenden Map angeben. Beide Schlüssel und Werte können entweder aus Ganzzahlen oder Strings bestehen, sodass alle folgenden Punkte akzeptabel sind:

ds_map_add(map, 5, 1);
ds_map_add(map, "level", 100);
ds_map_add(map, 89, "hello world");
ds_map_add(map, "fish", "good");

Sie können auch zu einer Karte hinzufügen, indem Sie den Accessor "?" Verwenden, wie unten gezeigt:

map[? 5] = 1;
map[? "level"] = 100;
map[? 89] = "hello world";
map[? "fish"] = "good";



HINWEIS: Im Gegensatz zu anderen Datenstrukturen in GameMaker Studio 2 wird dieser Schlüssel nicht an den Anfang (oder das Ende) der ds_map gehen, sondern irgendwo in die ds_map.


Beispiel:

inventory = ds_map_create();
ds_map_add(inventory, "hp potion", 1);
ds_map_add(inventory, "gold", 100);

Dadurch wird eine neue Map erstellt, deren ID der Variablen "inventory" zugewiesen wird. Es fügt dann zwei neue Schlüssel zu der Karte hinzu, "hp trance" und "gold", und setzt ihre anfänglichen Werte auf 1 und 100.
