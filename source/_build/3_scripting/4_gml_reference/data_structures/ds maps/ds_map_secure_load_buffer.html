ds_map_secure_load_buffer

Syntax:

ds_map_secure_load_buffer(buffer);


Streit Beschreibung
filename Die Puffer-ID des Puffers, von dem die Kartendaten geladen werden sollen.


Rückgabe: DS-Karte


Beschreibung

Diese Funktion lädt eine sichere gespeicherte DS-Karte aus einem Puffer. Sie müssen zuvor den Puffer in den Speicher geladen haben (mit buffer_load ) und übergibt diese Funktion dann an eine DS-Karte, die mit dem Inhalt des Puffers gefüllt ist. Beachten Sie, dass der Puffer mit der Funktion erstellt worden sein muss ds_map_secure_save_buffer damit dies richtig funktioniert.


Beispiel:

var buff = buffer_load("save.dat");
map = ds_map_secure_load_buffer(buff);
buffer_delete(buff);

Der obige Code lädt eine sicher gespeicherte ds_map aus einem Puffer und speichert seinen Indexwert in einer Variablen für die zukünftige Verwendung.
