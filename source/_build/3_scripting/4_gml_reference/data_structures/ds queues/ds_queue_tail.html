ds_queue_tail

Liest den Wert vom Ende der Warteschlange.

Syntax:

ds_queue_tail(id);


Streit Beschreibung
id Die ID der Datenstruktur, aus der gelesen werden soll.


Rückgabe: Real oder String


Beschreibung

Diese Funktion liest nur den letzten Wert der Warteschlange (das ist "am Ende"). Es wird den Wert nicht herausnehmen, dh er kann auch in Zukunft von dieser Funktion oder der ds_queue_dequeue.


Beispiel:

num = ds_queue_tail(control_queue);

Der obige Code liest den Wert aus der in der Variablen "control_queue" indizierten Warteschlange und speichert den Rückgabewert in der Variablen "num".
