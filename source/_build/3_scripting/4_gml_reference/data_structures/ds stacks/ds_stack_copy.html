ds_stack_copy

Kopiert den Inhalt eines Stapels in einen anderen.

Syntax:

ds_stack_copy(id, source);


Streit Beschreibung
id Die ID des NEUEN Stapels.
source Der ORIGINAL Stapel zum Kopieren nach ID.


Rückkehr: N / A


Beschreibung

Diese Funktion kann verwendet werden, um den Inhalt eines Stapels in einen anderen zu kopieren. Beachten Sie, dass dies nicht den Inhalt aus dem ursprünglichen Stapel entfernen noch den Originalstapel nicht zerstört. Wenn diese Funktion verwendet wird, muss der Stapel, zu dem kopiert wird, zuvor erstellt worden sein und wenn er irgendwelche Elemente vor der Kopie enthielt, werden diese zuerst gelöscht (was bedeutet, dass diese Information verloren geht).


Beispiel:

with (instance_create_layer(x, y, "Enemies", obj_Enemy))
   {
   stack = ds_stack_create();
   ds_stack_copy(stack, other.stack);
   }

Die obige Funktion erstellt eine neue Instanz und erstellt dann in diesem Fall einen neuen ds_stack und kopiert den Inhalt des Stapels in der Instanz, in der der Codeblock ausgeführt wird, in den neu erstellten Instanzstapel.
