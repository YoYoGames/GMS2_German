ds_list_find_value

Syntax:

ds_list_find_value(id, pos);


Streit Beschreibung
id Die ID der zu verwendenden Liste.
pos Die anzuzeigende Position, wobei 0 dem Anfang der Liste entspricht und die Endposition ds_list_size (id) -1 ist.


Rückgabe: Real, String oder Undefined


Beschreibung

Mit dieser Funktion können Sie die gegebene Listenposition überprüfen, und der in der Liste für diese Position enthaltene Wert wird zurückgegeben. Wenn Sie eine Position angeben, die außerhalb der angegebenen Listengröße liegt (z. B. Position 11 in einer Liste mit 10 Werten), kann die Funktion zurückkehren undefined oder 0. Dies liegt daran, dass beim Erstellen der Liste intern die ersten Einträge in der Liste auf 0 gesetzt werden, um Leistungsprobleme zu minimieren, wenn Elemente der Liste anfänglich hinzugefügt werden (obwohl der ds_list_size() Funktion gibt immer noch 0 in einer neu erstellten Liste zurück. Wenn Sie sicherstellen möchten, dass die Liste beim Erstellen "wirklich" leer ist, sollten Sie anrufen ds_list_clear() Nach dem Erstellen der Liste bedeutet dies, dass alle Werte für nicht aufgefüllte Listenplätze zurückgegeben werden undefined, die Sie dann konsequent mit dem überprüfen können is_undefined() Funktion.


Beispiel:

val = ds_list_find_value(list, ds_list_size(list) - 1);

Der obige Code prüft die in der Variablen "Liste" an der letzten Stelle der Liste indizierte Liste und speichert den zurückgegebenen Wert in der Variablen "Val".
