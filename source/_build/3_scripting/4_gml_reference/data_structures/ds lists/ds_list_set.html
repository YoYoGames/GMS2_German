ds_list_set

Syntax:

ds_list_set(id, pos, val);


Streit Beschreibung
id Die ID der Liste, der hinzugefügt werden soll.
pos Die Position innerhalb der Liste, die gesetzt werden soll.
[val2,... max_val] Der in der Liste einzustellende Wert.


Rückgabe: N / A


Beschreibung

Diese Funktion kann verwendet werden, um einen zuvor hinzugefügten Listeneintrag festzulegen. Sie geben die Listen-ID (wie beim Erstellen der Liste zurückgegeben) und die zu setzende Position innerhalb der Liste sowie den Wert für die Liste an. Wenn sich der gesetzte Eintrag außerhalb der Grenzen der Liste befindet (dh Sie legen den Listeneintrag 20 fest, die aktuelle Liste enthält jedoch nur 10 Einträge), wird die Liste bis zur angegebenen Position gefüllt und jeder Eintrag auf 0 gesetzt. Diese Funktion entspricht der Verwendung des DS-Listenzugriffsmittels.


Beispiel:

for (var i = 0; i < ds_list_size(list); i++;)
    {
    ds_list_set(list, i, -1);
    }

Der obige Code fügt den in der Variablen "score" gespeicherten Wert der Liste hinzu, die in der Variablen "sc_list" indiziert ist.
