ds_list_read

Syntax:

ds_list_read(id, str [, legacy]);


Streit Beschreibung
id Die ID der Datenstruktur, in die eingelesen werden soll.
str Die Zeichenfolge, von der gelesen werden soll.
legacy ( optional ) Kann beides sein true oder false oder ganz weggelassen.


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie eine gespeicherte ds_list neu erstellen (eine, die zuvor mit Hilfe einer Zeichenkette als Zeichenkette geschrieben wurde) ds_list_write ). Sie müssen zuerst eine neue ds_list erstellen, in die die Zeichenfolge eingelesen wird. Wenn die ds_list bereits existiert und Informationen darin gespeichert sind, wird diese vor dem Lesen gelöscht. Diese Funktion ist von entscheidender Bedeutung, wenn Sie Speicher- / Speichermechanismen für Ihr Spiel erstellen. Beachten Sie, dass wenn die Datenstruktur mit früheren Versionen von GameMaker, das optionale Argument "Legacy" hinzugefügt werden muss true da sich das String-Format mit dieser Version geändert hat.


Beispiel:

list = ds_list_create();
ini_open("save.ini");
var str = ini_read_string("Lists", "0", "");
if str != ""
   {
   ds_list_read(list, str);
   }
ini_close();

Der obige Code erstellt eine Liste und speichert den Index in der Variablen "list". Es öffnet sich dann ein ini file und liest eine Zeichenkette daraus, um sicherzustellen, dass die Zeichenkette nicht zuerst als leer zurückgegeben wird. Diese Zeichenfolge wird dann in die neu erstellte ds_list eingelesen.
