ds_list_mark_as_map

Syntax:

ds_list_mark_as_map(id, pos);


Streit Beschreibung
id Die ID der zu markierenden Liste.
pos Die Position innerhalb der Liste markieren.


Rückkehr: N / A


Beschreibung

Diese Funktion markiert eine Position in einer zuvor erstellten ds_list mit einer zuvor erstellten ds_map. Diese Funktionalität ist nur für die Verwendung bei der Kodierung von JSON-Strings vorgesehen (die Sie mit erstellen können json_encode ) und die komplementären ds Kartenfunktionen.

HINWEIS: Sobald eine ds_list einen Wert enthält, der als eine andere Liste oder Karte markiert ist, zerstört das Zerstören der Liste auch die markierten Listen und Karten. Dies bedeutet, dass Sie den Listeninhalt nicht manuell durchlaufen müssen und die markierten Datenstrukturen einzeln zerstören müssen, bevor Sie die "Eltern" -Liste zerstören.


Beispiel:

var sub_map = ds_map_create();
ds_map_add(sub_map, "player", player_array);
ds_map_add(sub_map, "enemy", enemy_array);
ds_list_add(j_list, sub_map);
ds_list_mark_as_map(j_list, 0);

Der obige Code erstellt eine ds_map und füllt sie dann mit zwei Schlüsseln, die jeweils ein Array von Werten enthalten. Diese Karte wird dann zu der gegebenen ds_list hinzugefügt und die Position als solche markiert, so dass sie später korrekt codiert werden kann.
