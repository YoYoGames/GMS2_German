ds_grid_get_disk_mean

Ruft den Mittelwert aller Werte innerhalb einer kreisförmigen Region ab.


Syntax:

ds_grid_get_disk_mean(index, xm, ym, r);


Streit Beschreibung
index Der Index des Gitters.
xm Die x-Position der Festplatte im Raster.
ym Die y-Position der Festplatte im Raster.
r Der Radius der Festplatte im Raster.


Rückgabe: Real oder String


Beschreibung

Diese Funktion kann verwendet werden, um den Durchschnittswert für alle Zellen zu finden, die innerhalb des definierten Plattenbereichs eines Gitters gefunden werden (alle Zellenwerte werden addiert und dann durch die Gesamtzahl der Zellen dividiert, die die Platte bilden) Bild unten:


Beispiel:

val = ds_grid_get_disk_mean(grid, 5, 5, 2);

Der obige Code setzt die Variable "val" auf den Mittelwert, der in der gegebenen Platte des in der Variablen "grid" indizierten ds_grid enthalten ist.
