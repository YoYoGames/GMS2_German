ds_grid_value_x

Gibt die x-Position eines Werts innerhalb einer Region eines Gitters zurück.

Syntax:

ds_grid_value_x(index, x1, y1, x2, y2, val);


Streit Beschreibung
index Der Index des Gitters.
x1 Die X-Position der linken Seite der Region im Raster.
y1 Die y-Position des oberen Rands der Region im Raster.
x2 Die x-Position der rechten Seite der Region im Raster.
y2 Die y-Position des unteren Rands der Region im Raster.
val Der Wert zu finden.


Rückgabe: Echt


Beschreibung

Mit dieser Funktion können Sie die x-Koordinate (innerhalb der gegebenen rechteckigen Gitterregion) des gesuchten Wertes erhalten. Beachten Sie, dass, wenn der Wert gesucht wird nicht existiert, wird die Funktion 0 zurück, so ist es eine gute Idee zu überprüfen, ob die Werte existieren zuerst mit ds_grid_value_exists dann führe diese Funktion aus (zusammen mit der ds_grid_value_y Funktion), um die genaue Position innerhalb der zu durchsuchenden Region zu erhalten.


Beispiel:

if ds_grid_value_exists(grid, 0, 1, 5, 6, val)
   {
   xpos = ds_grid_value_x(grid, 0, 1, 5, 6, val);
   ypos = ds_grid_value_y(grid, 0, 1, 5, 6, val);
   }

Der obige Code überprüft ein ds_grid auf einen bestimmten Wert innerhalb einer rechteckigen Region. Wenn es gefunden wird, speichert es dann die x- und y-Position des Wertes in zwei Variablen zur späteren Verwendung.
