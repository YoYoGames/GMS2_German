ds_grid_multiply

Multipliziert den Wert einer gegebenen Rasterzelle mit einem bestimmten Betrag.

Syntax:

ds_grid_multiply(index, x, y, val);


Streit Beschreibung
index Der Index des Gitters.
x Die x-Position der Zelle im Raster.
y Die y-Position der Zelle im Raster.
val Der Wert, der mit der Zelle multipliziert wird.


Rückkehr: N / A


Beschreibung

Diese Funktion multipliziert den Wert einer gegebenen Gitterzelle mit dem angegebenen Betrag.

HINWEIS: Diese Funktion funktioniert nur mit reellen Zahlen, nicht mit Strings.


Beispiel:

ds_grid_multiply(mygrid, 5, 5, 2);

Der obige Code multipliziert den in der angegebenen ds_grid-Zelle gespeicherten Wert mit 2.
