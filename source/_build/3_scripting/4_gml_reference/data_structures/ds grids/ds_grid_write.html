ds_grid_write


Beschreibung

Diese Funktion kann verwendet werden, um das angegebene ds_grid in einen String umzuwandeln, der dann in einer externen Datei gespeichert werden kann (zum Beispiel). Sie können den zurückgegebenen String dieser Funktion mithilfe der Funktion in ein ds_grid zurücklesen ds_grid_read().

HINWEIS: Die zurückgegebene Zeichenfolge ist keine für Menschen lesbare Zeichenfolge, sondern ein Speicherauszug des Inhalts der Datenstruktur.


Syntax:

ds_grid_write(index);

Streit Beschreibung
index Der Index des zu schreibenden Gitters.


Kehrt zurück:

String


Beispiel:

ini_open("Save.ini");
ini_write_string("Save", "0", ds_grid_write(mygrid));
ini_close()

Der obige Code öffnet eine ini-Datei (erstellt sie, falls noch nicht vorhanden) und schreibt dann das angegebene ds_grid als Zeichenfolge in diese Datei.
