ds_grid_write

Verwandelt eine Datenstruktur in eine Zeichenfolge und gibt diese Zeichenfolge zurück.

Syntax:

ds_grid_write(index);


Streit Beschreibung
index Der Index des zu schreibenden Gitters.


Rückgabe: Zeichenfolge


Beschreibung

Mit dieser Funktion kann das angegebene ds_grid in einen String konvertiert werden, der dann zB in einer externen Datei gespeichert werden kann. Sie können die zurückgegebene Zeichenfolge aus dieser Funktion mithilfe der Funktion in ein ds_grid zurücklesen ds_grid_read().

HINWEIS: Die zurückgegebene Zeichenfolge ist keine lesbare Zeichenfolge, sondern ein Speicherauszug des Inhalts der Datenstruktur


Beispiel:

ini_open("Save.ini");
ini_write_string("Save", "0", ds_grid_write(mygrid));
ini_close();

Der obige Code wird eine Ini-Datei öffnen (sie wird erstellt, wenn sie nicht bereits existiert) und dann das gegebene ds_grid als eine Zeichenkette in diese Datei schreiben.
