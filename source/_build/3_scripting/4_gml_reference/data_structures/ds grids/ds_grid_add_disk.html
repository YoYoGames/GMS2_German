ds_grid_add_disk

Fügt allen Zellen in einer kreisförmigen Region in einem Raster einen angegebenen Wert hinzu (kann sowohl eine Zahl als auch eine Zeichenfolge sein).

Syntax:

ds_grid_add_disk(index, xm, ym, r, val);


Streit Beschreibung
index Der Index des Gitters.
xm Die x-Position der Festplatte im Raster.
ym Die y-Position der Festplatte im Raster.
r Der Radius der Festplatte im Raster.
val Der Wert, der den Zellen auf der Festplatte hinzugefügt werden soll.


Rückkehr: N / A


Beschreibung

Diese Funktion kann verwendet werden, um allen Werten der Zellen, die sich innerhalb des definierten Plattenbereichs eines Gitters befinden, einen gegebenen Wert (real oder string) hinzuzufügen. Der hinzuzufügende Wert muss vom gleichen Typ sein, der in den Gitterzellen enthalten ist, dh: Sie können keine Zeichenfolge zu einem Real hinzufügen und umgekehrt, und bei Strings entspricht dies einer Verkettung.


Beispiel:

ds_grid_add_disk(grid, 7, 6, 5, 2);

Dies würde 2 zu allen Werten hinzufügen, die in den Zellen innerhalb des definierten Plattenbereichs des durch die Variable "Gitter" referenzierten ds_grid gehalten werden.
