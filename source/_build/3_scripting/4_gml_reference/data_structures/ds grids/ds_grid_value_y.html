ds_grid_value_y


Beschreibung

Mit dieser Funktion können Sie die y-Koordinate (innerhalb des angegebenen rechteckigen Gitterbereichs) des gesuchten Werts ermitteln. Sie geben den DS-Rasterindex an (wie von zurückgegeben ds_grid_create ) zusammen mit den x / y-Positionen für die oberen linken und unteren rechten Ecken des zu suchenden Gitterbereichs, bevor Sie den Wert angeben, nach dem gesucht werden soll. Wenn der gesuchte Wert nicht vorhanden ist, gibt die Funktion -1 zurück.


Syntax:

ds_grid_value_y(index, x1, y1, x2, y2, val);

Streit Beschreibung
index Der Index des Gitters.
x1 Die x-Position der linken Seite der Region im Raster von 0 bis (Gitterbreite - 1).
y1 Die y-Position des oberen Bereichs der Region im Raster von 0 bis (Rasterhöhe - 1).
x2 Die x-Position der rechten Seite der Region im Raster von 0 bis (Gitterbreite - 1).
y2 Die y-Position des unteren Bereichs der Region im Raster von 0 bis (Rasterhöhe - 1)
val Der Wert zu finden.


Kehrt zurück:

Real


Beispiel:

if ds_grid_value_exists(grid, 0, 1, 5, 6, val)
   {
   xpos = ds_grid_value_x(grid, 0, 1, 5, 6, val);
   ypos = ds_grid_value_y(grid, 0, 1, 5, 6, val);
   }

Der obige Code prüft ein ds_grid auf einen bestimmten Wert innerhalb eines rechteckigen Bereichs. Wenn es gefunden wird, speichert es die x- und y-Position des Werts in zwei Variablen zur späteren Verwendung.
