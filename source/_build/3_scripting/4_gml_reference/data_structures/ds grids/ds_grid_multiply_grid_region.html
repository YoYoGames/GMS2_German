ds_grid_multiply_grid_region

Multipliziert den Inhalt der Zellen in einem Quellraster mit den Zellen in einem Zielraster.

Syntax:

ds_grid_multiply_grid_region(index, source, x1, y1, x2, y2, xpos, ypos);


Streit Beschreibung
index Der Index des Zielrasters.
source Der Index des Quellgitters.
x1 Die linke Position der Zellenregion, die aus dem Quellraster kopiert werden soll.
y1 Die oberste Position der Zellenregion, die aus dem Quellraster kopiert werden soll.
x2 Die richtige Position der Zellenregion, die aus dem Quellraster kopiert werden soll.
y2 Die unterste Position der Zellenregion, die aus dem Quellraster kopiert werden soll.
xpos Die x-Position auf dem Zielgitter multipliziert die Quellregion mit.
ypos Die Y-Position im Zielraster, um die Quellregion mit zu multiplizieren.


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie einen Bereich innerhalb eines gegebenen ds_grid definieren, dann diese Werte nehmen und sie mit denen multiplizieren, die in einem separaten Bereich des gleichen ds_grid oder eines anderen (der zuvor erstellt wurde) gefunden werden. Die ursprüngliche Region bleibt unverändert, während die Region, mit der sie multipliziert wurden, die neuen Werte für jede Zelle speichert.


Beispiel:

ds_grid_multiply_grid_region(mygrid, mygrid, 0, 0, 5, 5, 0, 0);

Dies würde die Region der Zellen von (0,0) bis (5,5) des ds_grid "mygrid" nehmen und sie mit den Zellen von der Position (0,0) desselben ds_grid multiplizieren.
