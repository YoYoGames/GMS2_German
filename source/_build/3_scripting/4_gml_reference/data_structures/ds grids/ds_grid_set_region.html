ds_grid_set_region

Setzt alle Zellen in der Region im Raster mit der angegebenen ID auf den angegebenen Wert (beide können eine Zahl oder eine Zeichenfolge sein).

Syntax:

ds_grid_set_region(index, x1, y1, x2, y2, val);


Streit Beschreibung
index Der Index des Gitters.
x1 Die X-Position der linken Seite der Region im Raster.
y1 Die y-Position des oberen Rands der Region im Raster.
x2 Die x-Position der rechten Seite der Region im Raster.
y2 Die y-Position des unteren Rands der Region im Raster.
val Der Wert, auf den die Regionszellen festgelegt werden sollen.


Rückkehr: N / A


Beschreibung

Diese Funktion kann verwendet werden, um einen rechteckigen Bereich eines gegebenen Rasters auf einen bestimmten Wert zu setzen (der entweder ein reeller Wert oder ein String sein kann), wie in der Abbildung unten gezeigt:


Beispiel:

ds_grid_set_region(grid, 5, 5, 10, 10, 99);

Dies würde alle Zellen innerhalb der Region des Gitters, die in der Variablen "Gitter" von (5,5) bis (10,10) bis 99 indiziert sind, setzen.
