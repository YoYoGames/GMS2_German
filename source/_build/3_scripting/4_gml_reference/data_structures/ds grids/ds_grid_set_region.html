ds_grid_set_region


Beschreibung

Diese Funktion kann verwendet werden, um einen rechteckigen Bereich eines gegebenen Gitters auf einen angegebenen Wert festzulegen (der entweder ein reeller oder ein String sein kann), wie in der folgenden Abbildung dargestellt:


Syntax:

ds_grid_set_region(index, x1, y1, x2, y2, val);

Streit Beschreibung
index Der Index des Gitters.
x1 Die x-Position der linken Seite der Region im Raster.
y1 Die y-Position des oberen Bereichs der Region im Raster.
x2 Die x-Position der rechten Seite der Region im Raster.
y2 Die y-Position des unteren Bereichs der Region im Raster.
val Der Wert, auf den die Regionszellen gesetzt werden sollen.


Kehrt zurück:

N/A


Beispiel:

ds_grid_set_region(grid, 5, 5, 10, 10, 99)

Dies würde alle Zellen in der Region des Gitters setzen, die in der Variablen "grid" von (5,5) bis (10,10) bis 99 indexiert sind.
