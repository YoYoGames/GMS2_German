ds_grid_clear

Löscht das Gitter mit der angegebenen ID auf einen angegebenen Wert.

Syntax:

ds_grid_clear(index, val);


Streit Beschreibung
index Dieser Index des Rasters wird gelöscht.
val Der neue Wert für alle Rasterzellen.


Rückkehr: N / A


Beschreibung

Diese Funktion kann verwendet werden, um ein bestimmtes ds_grid auf einen bestimmten Wert zu löschen. Alle Zellen innerhalb des Gitters enthalten dann diesen Wert, der eine reelle Zahl oder eine Zeichenfolge sein kann. Das Bild unten zeigt, wie das funktioniert:


Beispiel:

ds_grid_resize(global.Grid, room_width / 32, room_height / 32);
ds_grid_clear(global.Grid, -1);

Der obige Code ändert die Größe des in der globalen Variablen "Grid" indizierten ds_grid und löscht ihn dann, so dass jede Zelle den Wert -1 enthält.
