ds_grid_read


Beschreibung

Mit dieser Funktion kann eine Zeichenfolge konvertiert werden, die zuvor von der Funktion erstellt wurde ds_grid_write() zurück in ein ds_grid. Das ds_grid muss zuvor erstellt worden sein (siehe Beispiel unten). Wenn Sie die Datenstruktur mit früheren Versionen von GameMaker, sollten Sie das optionale Argument "Legacy" hinzufügen und auf GameMaker true da sich das Stringformat mit dieser Version geändert hat.


Syntax:

ds_grid_read(index, string [, legacy]);

Streit Beschreibung
index Der Index des zu lesenden Gitters.
string Die Zeichenfolge, die in das ds_grid eingelesen werden soll.
legacy ( optional ) Kann beides sein true oder false oder ganz weggelassen.


Kehrt zurück:

N/A


Beispiel:

grid = ds_grid_create(room_width div 32, room_height div 32);
ini_open("Save.ini");
ds_grid_read(grid, ini_read_string("Save", "0", ""));
ini_close()

Der obige Code erstellt ein ds_grid basierend auf der Größe des Raums (jedes 32 × 32-Quadrat von Pixeln stellt eine Gitterzelle dar) und liest dann einen zuvor gespeicherten Satz von Rasterdaten aus einer INI-Datei in das neue ds-Raster.
