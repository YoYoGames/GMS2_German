ds_grid_resize

Ändert die Größe des Rasters auf eine neue Breite und Höhe.

Syntax:

ds_grid_resize(index, w, h);


Streit Beschreibung
index Dieser Index des Gitters zur Größenänderung.
w Die neue Breite des Gitters.
h Die neue Höhe des Gitters.


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie das angegebene ds_grid auf eine andere Breite und / oder Höhe einstellen. Wenn die Rastergröße größer als das aktuelle Raster ist, haben die neuen Zellen einen Basiswert von 0, und wenn die Größe kleiner ist, gehen die Werte in den Zellen verloren, die nicht mehr innerhalb der neuen Größe sind. Alle anderen Zellen bleiben unberührt.


Beispiel:

ds_grid_resize(global.Grid, room_width / 32, room_height / 32);
ds_grid_clear(global.Grid, -1);

Der obige Code ändert die Größe des in der globalen Variablen "Grid" indizierten ds_grid und löscht ihn dann, so dass jede Zelle den Wert -1 enthält.
