get_integer_async


Beschreibung

Diese Funktion öffnet ein Fenster und zeigt eine Meldung sowie einen Bereich an, in dem der Benutzer einen Wert eingeben kann (der den angegebenen Standardwert enthält, der mit dem Start beginnt). Dies ist eine asynchrone Funktion. Daher blockiert GameMaker Studio 2 nicht das Gerät, auf dem es ausgeführt wird, während es auf Antwort wartet, sondern führt Ereignisse weiterhin normal aus. Nachdem der Benutzer die Zeichenfolge eingegeben und die Schaltfläche "Okay" gedrückt hat, wird ein asynchrones Dialogereignis ausgelöst, bei dem nur für die Dauer dieses Ereignisses eine ds_map in der Variablen gespeichert wird async_load.

Diese Karte enthält die drei Schlüssel. " id "," status ", und " value "." id "ist der Wert, der von der Funktion beim Aufruf zurückgegeben wurde, der" status "wird entweder sein true für das Drücken der Taste "OK" oder false wenn die Nachricht abgebrochen wurde (ggf. nicht alle Zielplattformen erlauben das Abbestellen von Nachrichten). Endlich " value msgstr "" "gibt die Ganzzahl zurück, die der Benutzer als String eingegeben hat (" "wird ein leerer String zurückgegeben, wenn keine Eingabe erfolgt).


Syntax:

get_integer_async(string, default);

Streit Beschreibung
String Die Nachricht, die im Dialog angezeigt werden soll.
default Der Standardwert


Kehrt zurück:

Real


Erweitertes Beispiel:

Das Ereignis mit der linken Maustaste (beispielsweise) des Objekts, das die Nachricht anzeigt, hätte folgenden Code:

msg = get_integer_async("How old are you?", 0);

Oben wird eine Nachricht angezeigt, in der der Benutzer aufgefordert wird, eine Zeichenfolge einzugeben und "OK" zu drücken. Die Funktions-ID wird in der Variablen "msg" gespeichert und im Ereignis asynchronous Dialogs wie folgt verwendet:

var i_d = ds_map_find_value(async_load, "id");
if i_d == msg
   {
   if ds_map_find_value(async_load, "status")
      {
      global.Age = ds_map_find_value(async_load, "value");
      }
   }

Der obige Code überprüft das " id "Schlüssel des zurückgegebenen ds_map gegen den in der Variablen "msg" gespeicherten Wert. Wenn sie gleich sind, wird geprüft, ob "OK" gedrückt wurde (anstatt das Fenster geschlossen / abgebrochen) und ob es zurückkehrt true Es setzt dann eine globale Variable aus der zurückgegebenen Ganzzahl.
