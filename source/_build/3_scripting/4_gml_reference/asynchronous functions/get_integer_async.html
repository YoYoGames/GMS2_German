get_integer_async


Beschreibung

Diese Funktion öffnet ein Fenster und zeigt eine Nachricht sowie einen Platz für den Benutzer an, um einen Wert einzugeben (der den angegebenen Standardwert enthalten wird). Dies ist eine asynchrone Funktion, und so blockiert GameMaker Studio 2 nicht das Gerät, auf dem es ausgeführt wird, während auf Antwort gewartet wird, sondern läuft weiter wie gewohnt. Sobald der Benutzer die Zeichenfolge hat getippt und drückte auf den „Okay“ -Taste, ein asynchrones Dialog Ereignis wird nur ausgelöst, die für die Dauer dieses Ereignisses wird eine ds_map in den Variablen gespeichert hat async_load.

Diese Karte enthält die drei Schlüssel "ID", "Status" und "Wert". "id" ist der Wert, der von der Funktion beim Aufruf zurückgegeben wurde, der "Status" ist entweder true wenn die Taste "Okay" gedrückt wird, oder false wenn die Nachricht abgebrochen wurde (falls nicht alle Zielplattformen die Löschung von Nachrichten zulassen). Schließlich gibt "value" die Ganzzahl zurück, die die Benutzereingabe eingegeben hat (oder 0, wenn keine angegeben wurde).


Syntax:

get_integer_async(string, default);

Streit Beschreibung
String Die Nachricht, die im Dialogfeld angezeigt werden soll.
default Der Standardwert.


Kehrt zurück:

Real


Erweitertes Beispiel:

Das linke Mausereignis (zum Beispiel) des Objekts, das die Nachricht anzeigt, würde den folgenden Code haben:

msg = get_integer_async("How old are you?", 0);

Das obige zeigt eine Nachricht, die anfordert, dass der Benutzer einen String eingibt und "Okay" drückt. Die Funktions-ID ist in der Variablen "msg" gespeichert und wird im asynchronen Dialog " Dialoge " wie folgt verwendet:

var i_d = ds_map_find_value(async_load, "id");
if i_d == msg
   {
   if ds_map_find_value(async_load, "status")
      {
      global.Age = ds_map_find_value(async_load, "value");
      }
   }

Der obige Code überprüft die "ID" -Taste des zurückgegebenen ds_map gegen den in der Variable "msg" gespeicherten Wert. Wenn sie gleich sind, prüft sie, ob "Okay" gedrückt wurde (anstatt dass das Fenster geschlossen / abgebrochen wird) und ob es zurückkehrt true Anschließend wird eine globale Variable aus der zurückgegebenen Ganzzahl gesetzt.
