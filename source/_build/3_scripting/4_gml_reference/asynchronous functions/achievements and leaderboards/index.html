Erfolge und Bestenlisten


Spiele, die für Zielgeräte mit iOS oder Android können auch eine Verbindung zu einem von zwei Leaderboard- und Leistungs-APIs herstellen. Für iOS haben Sie das iTunes Game Center und für Android Sie den Amazon Game Circle oder Google Play verwenden, wobei alle die Funktionen auf dieser Seite verwenden. Diese Funktionen ermöglichen es Ihrem Spiel, Informationen über Punkte und Erfolge an die Bestenlisten zu senden, die Sie im Entwicklerkonto für das von Ihnen erstellte Spiel eingerichtet haben. Dies ist eine hervorragende Möglichkeit, Spieler in die soziale Seite des Spiels einzubeziehen was andere Leute, die dein Spiel gespielt haben, erreicht haben und versuchen, sie zu verbessern.

Wenn Sie diese Funktionen verwenden, sollten Sie beachten, dass sie verfolgen keine Leistungen und Noten automatisch für Sie. Sie zeigen nur die Daten an, die Sie von Ihrem Spiel erhalten, also müssen Sie Ihre eigenen Errungenschaften und das Hi-Score-System programmieren, um den Status der einzelnen Personen im Spiel zu verfolgen, und dann diese Funktionen verwenden, um die Daten dem gewählten Netzwerk hinzuzufügen kann Fortschrittsinformationen für Erfolge anfordern, die das verwenden achievement_load_progress Funktion).

Einige dieser Funktionen lösen ein asynchrones Social-Asynchron-Ereignis aus, das a enthält ds_map von Daten, die sich auf die Funktion beziehen, die sie ausgelöst hat. Die Funktionen, die das Ereignis auslösen, geben auch die Kartenschlüssel / Wert-Paare an, die vom Callback zurückgegeben werden. Ein Callback- Social Event kann jedoch auch ausgelöst werden, wenn Ihr Spiel externe Benachrichtigungen über Herausforderungen usw. erhält ds_map Für diese Ereignisse zurückgegebene Ereignisse werden im Abschnitt " Social Asynchronous Event" erläutert.

HINWEIS: Damit diese Funktionen mit iOS funktionieren, benötigen Sie ein gültiges Game Center-Konto und eine iOS- App-ID für Ihr Spiel.

HINWEIS: Damit diese Funktionen mit Android funktionieren, müssen Sie Ihr Spiel an den Amazon App Store oder an Google Play senden und die erforderlichen Abschnitte einrichten sowie die API in den Android Spieloptionen für Ihr Spiel aktivieren.

Die unten aufgelisteten Funktionen sind die grundlegenden Funktionen, die zur Integration Ihres Spiels in die gewählte Social-Gaming-Zielplattform verwendet werden:

  • Leistung_login_status
  • Leistung_login
  • Leistung_Logout
  • Leistung_verfügbar
  • Leistung_Post
  • achievement_post_score
  • Leistung_Inkrement
  • achievement_show_achievements
  • achievement_show_leaderboards
  • achievement_show_challenge_notifications
  • achievement_get_challenges
  • Leistung_get_pic
  • achievement_load_friends
  • achievement_load_leaderboard
  • achievement_load_progress
  • achievement_send_challenge
  • Leistungsreset
