achievement_show_challenge_notifications


Syntax:

achievement_show_challenge_notifications(receive, local, remote)

Streit Beschreibung
receive Zeige herausgeforderte Pop-ups ( true ) oder nicht ( false ).
local Zeige abgeschlossene lokale Herausforderungen Pop-up ( true ) oder nicht ( false ).
remote Zeige abgeschlossene Remote-Challenges-Pop-ups ( true ) oder nicht ( false ).


Kehrt zurück:

N/A


Beschreibung

Mit dieser Funktion können Sie die verschiedenen Toast-Popup-Benachrichtigungen zu Herausforderungen anzeigen oder unterdrücken. Wenn die Argumente auf festgelegt sind true Diese Nachrichten werden angezeigt und informieren den Spieler über alle erhaltenen lokalen oder entfernten Herausforderungen sowie über die abgeschlossenen Aufgaben und setzen sie auf false unterdrückt diese Benachrichtigungen.


Beispiel:

achievement_show_challenge_notifications(true, false, false);

Der oben stehende Code setzt Challenge Toast-Benachrichtigungen, die nur für empfangene Challenges angezeigt werden.
