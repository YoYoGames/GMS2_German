facebook_send_invite


Syntax:

facebook_send_invite(titlestring, msg, picture_url, redirect_url, ds_map_response);

Streit Beschreibung
titlestring Der Titel des Dialogs (ein String)
msg Die Nachricht, die im Dialogfeld angezeigt wird (eine Zeichenfolge).
picture_url die URL des Bildes, das in den Dialog aufgenommen werden soll.
redirect_url Die "SiteURL" Ihrer App auf der Facebook-Seite der App-Entwicklereinstellungen.
ds_map_response Eine vordefinierte ds_map, die zum Speichern der Antwortdetails verwendet wird.


Kehrt zurück:

N/A


Beschreibung

Mit dieser Funktion können Sie Ihr Facebook-Apprequests-Dialogfeld anzeigen lassen, in dem Ihr Spieler Freunde aus einer Liste auswählen und ihnen eine Einladung senden kann. Die Funktion erfordert, dass Sie ein zuvor erstelltes Objekt angeben ds_map welches dann mit den relevanten Benutzerdetails gefüllt wird. Diese Karte sollte jeden Schritt nach dem Aufruf dieser Funktion abgefragt werden, da die Antwort zu jeder Zeit durchkommen könnte. Wenn dies der Fall ist, sollte die Karte einen Schlüssel " to " haben, der eine ds_list zurückgibt, die alle Facebook-Benutzerkennungen der eingeladenen Personen enthält (siehe weiter unten für weitere Details).

HINWEIS: Die "SiteURL" wird auf den iOS- und Android Modulen ignoriert und ist nur für das HTML5-Modul erforderlich, wenn Sie die Anwendung nicht auf der "SiteURL" hosten (dh: Sie testen lokal).

HINWEIS: Der Benutzer muss nicht angemeldet sein, um diese Funktion zu verwenden, da der Anmeldedialog als Teil der Einladungsanfrage angezeigt wird.


Erweitertes Beispiel:

Sie können eine Einladung für Freunde folgendermaßen senden:

if mouse_check_button_pressed(mb_left)
   {
   map = ds_map_create();
   facebook_send_invite("Catch The Haggis!", "Come and play Catch The Haggis with me...", "http://MacSweeneygames.com/Clown.jpg", "", map);
   }

Der obige Code zeigt einen "Apprequest" -Dialog an, in dem Sie Freunde auswählen können, die eingeladen werden, das Spiel zu spielen, und es erstellt auch einen ds_map um die Facebook IDs der eingeladenen Nutzer zu halten. Dies muss dann im Schritt- Ereignis einer Instanz abgefragt werden, um die Ergebnisse zu erhalten, wie folgt:

if ds_map_exists(map, "to")
   {
   var list = ds_map_find_value(map, "to");
   var size = ds_list_size(list);
   for(var i = 0; i < size; i++;)
      {
      fb_uid[i] = ds_list_find_value(list, i);
      }
   ds_map_clear(map);
   }

Dieser Code überprüft die ds_map auf den Schlüssel "to". Wenn er hinzugefügt wird, greift er auf die ds-Liste zu, die "to" enthält und durchläuft, und setzt das Array "fb_uid" auf die zurückgegebenen Benutzer-IDs.
