facebook_send_invite


Syntax:

facebook_send_invite(titlestring, msg, picture_url, redirect_url, ds_map_response);

Streit Beschreibung
titlestring Der Titel des Dialogs (eine Zeichenfolge)
msg Die Nachricht, die im Dialog erscheint (eine Zeichenfolge).
picture_url Die URL des Bildes, das in den Dialog aufgenommen werden soll.
redirect_url Die "SiteURL" Ihrer App auf der Facebook Developers App-Einstellungsseite.
ds_map_response Eine vorab erstellte ds_map, in der die Antwortdetails gespeichert werden.


Kehrt zurück:

N/A


Beschreibung

Mit dieser Funktion können Sie Ihr Spiel dazu bringen, ein Facebook-Dialogfeld "Anfragen" anzuzeigen, in dem Ihr Spieler Freunde aus einer Liste auswählen und ihnen eine Spieleinladungsnachricht senden kann. Die Funktion erfordert, dass Sie eine zuvor erstellte angeben ds_map die dann mit den relevanten Benutzerdetails gefüllt werden. Diese Map sollte in jedem Schritt nach dem Aufruf dieser Funktion abgefragt werden, da die Antwort jederzeit durchkommen könnte. Wenn dies der Fall ist, sollte die Map einen Schlüssel " to " haben, der eine ds_list mit allen Facebook-Benutzer-IDs der eingeladenen Personen zurückgibt (siehe ausführliches Beispiel unten für weitere Details).

HINWEIS: Die "SiteURL" wird bei den iOS- und Android Modulen ignoriert und ist nur für das HTML5-Modul erforderlich, wenn Sie die Anwendung nicht auf der "SiteURL" hosten (dh Sie testen lokal).

HINWEIS: Der Benutzer muss nicht angemeldet sein, um diese Funktion zu verwenden, da der Anmeldedialog als Teil der Einladungsanfrage angezeigt wird.


Erweitertes Beispiel:

Sie können eine Freunde-Einladung auf folgende Weise senden:

if mouse_check_button_pressed(mb_left)
   {
   map = ds_map_create();
   facebook_send_invite("Catch The Haggis!", "Come and play Catch The Haggis with me...", "http://MacSweeneygames.com/Clown.jpg", "", map);
   }

Der obige Code zeigt einen "Dankeschön" -Dialog, in dem Sie Freunde auswählen können, die zum Spiel eingeladen werden sollen, und es wird auch ein erstellt ds_map um die Facebook-IDs der eingeladenen Benutzer zu speichern. Dies muss dann im Step- Ereignis einer Instanz abgefragt werden, um die Ergebnisse wie folgt zu erhalten:

if ds_map_exists(map, "to")
   {
   var list = ds_map_find_value(map, "to");
   var size = ds_list_size(list);
   for(var i = 0; i < size; i++;)
      {
      fb_uid[i] = ds_list_find_value(list, i);
      }
   ds_map_clear(map);
   }

Dieser Code überprüft ds_map auf den Schlüssel "to". Wenn er hinzugefügt wird, greift er auf die ds-list zu, die "to" enthält, und durchläuft ihn, wobei das Array "fb_uid" so eingestellt wird, dass es die zurückgegebenen Benutzer-IDs enthält.
