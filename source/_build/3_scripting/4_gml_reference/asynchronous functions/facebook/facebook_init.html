facebook_init


Syntax:

facebook_init(appKey)


Kehrt zurück:

N/A


Beschreibung

Bevor Sie eine der Facebook-Funktionen nutzen, müssen Sie diese Funktion aufrufen, um die Facebook-API zu initialisieren und GameMaker Studio 2 mitzuteilen, dass Sie die Facebook-spezifischen Funktionen nutzen werden. Dies sollte nur einmal im Spiel gemacht werden und kann jederzeit durchgeführt werden, aber es wird empfohlen, dass Sie diese Funktion am Anfang vor allem anderen aufrufen. Beachten Sie auch, dass auch wenn Sie verwenden facebook_logout Zu einem bestimmten Zeitpunkt in Ihrem Spiel müssen Sie diese Funktion nicht erneut zum Einloggen verwenden.


Beispiel:

facebook_init();
var permissions = ds_list_create();
ds_list_add(permissions,"public_profile", "user_friends");
facebook_login(permissions);
ds_list_destroy(permissions);

Der obige Code initialisiert Facebook und erstellt dann ein ds_list um die spezifischen Berechtigungen zu halten, die Sie anfordern möchten, und sich schließlich mit diesen Berechtigungen anmeldet.
