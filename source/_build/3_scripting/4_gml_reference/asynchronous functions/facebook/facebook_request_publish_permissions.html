facebook_request_publish_permissions


Syntax:

facebook_request_publish_permissions(permissions)

Streit Beschreibung
permissions Die Berechtigungen, die Sie vom Benutzer erhalten möchten.


Kehrt zurück:

Real


Beschreibung

Mit dieser Funktion können Sie zusätzliche Veröffentlichungsberechtigungen für Facebook anfordern. Diese Berechtigungen werden einem zuvor erstellten als Strings hinzugefügt ds_list Diese wird dann in der Funktion zum Senden der Anfrage verwendet. Der Benutzer muss erfolgreich angemeldet sein (dh er hat angerufen) facebook_login ) und das facebook_status sollte "AUTHORISED" bevor Sie diese Funktion aufrufen. Die Funktion gibt einen eindeutigen ID-Wert für den Anforderungsaufruf zurück oder -1, wenn die Anforderung nicht gesendet wurde (z. B. wenn der Benutzer nicht angemeldet ist).

Diese Funktion löst ein asynchrones soziales Ereignis aus, das die ds_map enthält async_load. Diese ds_map wird mit verschiedenen Schlüssel / Wert-Paaren aufgefüllt, um den Erfolg oder Misserfolg der Anforderung zu melden.


Beispiel:

var request = "publish_actions";
if !facebook_check_permission(request)
   {
   var permissions = ds_list_create();
   ds_list_add(permissions, request);
   request_ID = facebook_request_publish_permissions(permissions);
   ds_list_destroy(permissions);
   }

Der obige Code prüft die aktuellen Facebook-Berechtigungen. Wenn das Spiel die Berechtigung "publish_actions" nicht angefordert hat, wird eine Anfrage gesendet.
