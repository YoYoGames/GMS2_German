facebook_post_message


Syntax:

facebook_post_message(title, caption, message, picture_url, redirect_url, actionname, action_url)

Streit Beschreibung
title Der "Name" des Links.
caption Die Beschriftung des Links (wird unter dem Linknamen angezeigt).
message Die Nachricht, die mit dem Post enthalten sein soll.
picture_url Ein Link zu dem Bild in diesem Beitrag enthalten.
redirect_url Der Link zu diesem Beitrag.
actionname Eine Aktion, die für diesen Post angewendet werden soll.
action_url Der für die obige Aktion zu verwendende Link.


Kehrt zurück:

N/A


Beschreibung

Diese Funktion sendet eine Nachricht direkt an den Benutzer-Feed (dh einen "/ me / feed" -Post an die Facebook-Graph-API). Sie geben den Namen und die Beschriftung sowie eine Nachricht und ein Bild an, die eingeschlossen werden sollen, und dann die URL, auf die verlinkt werden soll. Die letzten beiden Argumente, "actionname" und "actionurl", sind optional und können entweder als leere Strings ("") belassen werden oder sie können auf eine der erlaubten Aktionen von Facebook gesetzt werden, wie zB Kommentieren oder Liken. In diesem Fall sollten Sie auch Geben Sie eine entsprechende URL an (weitere Informationen finden Sie unter Facebook Developers Api Post ).

HINWEIS: Es wird kein Facebook-Dialog für diese Funktion angezeigt, da der Post ohne Benutzerinteraktion "still" gesendet wird. Aus diesem Grund müssen Sie die richtigen Berechtigungen für Login angefordert ( „publish_actions“).

HINWEIS: Der Benutzer muss angemeldet sein, damit die Nachricht funktioniert.


Beispiel:

if mouse_check_button_pressed(mb_left)
   {
   facebook_post_message("Catch The Haggis", "MacSweeneygames.com", "I just got a hi-score playing Catch The Haggis!", "http://MacSweeneygames.com/Clown.jpg", "http://MacSweeneygames.com/CatchTheHaggis.html", "", "");
   }

Mit dem obigen Code wird automatisch eine Nachricht an den Wall-Feed der Nutzer gesendet.
