mp_grid_add_cell

Syntax:

mp_grid_add_cell(id, h, v);


Streit Beschreibung
id Index des mp_grid, das verwendet werden soll
h Horizontale Position der Zelle im Raster
v Vertikale Position der Zelle im Gitter


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können wir einzelne Zellen innerhalb des mp_grid als "verboten" markieren, dh die Pfadfindungsfunktionen werden sie niemals überschreiten. Nun, in der Regel arbeiten Sie in x / y - Koordinaten Raum und nicht in mp_grid Zellkoordinaten. Betrachten wir nun das folgende Bild, das zeigt, wie die beiden korrelieren können:

Wie Sie sehen können, ist mp_grid als eine Reihe von Zellen nummeriert (ab 0,0 in der oberen linken Ecke), und wenn Sie dieses Gitter über den Raum platzieren, gibt es eine direkte Korrelation zwischen den Raumkoordinaten (die ebenfalls bei 0 beginnen, 0 oben links) und die Zellen innerhalb des Gitters. In dem Bild kann man sehen, dass eine Instanz an der Position 260, 130 auf die Gitterzelle 8, 4 fällt. Sie können mathematisch die genaue Zelle berechnen, in der eine Position leicht landet, indem Sie entweder die Zahl durch die Gitterauflösung teilen und dann abrunden oder (wenn Sie die Stärke von 2 Gitterauflösungen wie 8, 16, 32 usw. verwenden) Verwenden Sie die bitweisen Operatoren. Es sollte beachtet werden, dass die bitweise Operation schneller ist. Das folgende Beispiel zeigt beide Methoden.


Beispiel:

with (obj_Wall)
   {
   mp_grid_add_cell(grid, floor(x / 32), floor(y / 32));
   }
with (obj_Door)
   {
   mp_grid_add_cell(grid, x>>5, y>>5);
   }

Der obige Code wird alle "obj_Wall" und alle obj_Door hinzufügen ihre äquivalenten Zellenpositionen in die mp_grid indiziert in der Variable "Grid". Im ersten Teil des Beispiels geschieht dies, indem die x / y-Koordinaten des Objekts genommen und durch die Auflösung des Gitters geteilt werden (unter Verwendung von floor um die Werte als Ganzzahlen zu halten), und im zweiten Teil (da die Gitterauflösung eine Potenz von 2 ist) erfolgt eine Bitverschiebung der x, y-Koordinaten um 5 Stellen nach rechts.
