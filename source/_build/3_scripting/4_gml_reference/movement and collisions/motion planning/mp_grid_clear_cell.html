mp_grid_clear_cell

Syntax:

mp_grid_clear_cell(id, h, v);


Streit Beschreibung
id Index des mp_grid, das verwendet werden soll
h Horizontale Position der zu löschenden Zelle
v vertikale Position der zu löschenden Zelle


Rückgabe: Boolean.


Beschreibung

Diese Funktion kann verwendet werden, um eine bestimmte "Zelle" eines mp_grid zu löschen. Zellen werden nicht als Raumkoordinaten berechnet, sondern als Gitterkoordinaten, wobei (0,0) die obere linke Ecke des Gitters ist. Das bedeutet, dass zum Löschen einer Zelle an einer bestimmten Position im Raum die x- und y-Koordinaten in Zellenkoordinaten geändert werden müssen, die durch die Auflösung von mp_grid geteilt werden. Das folgende Codebeispiel zeigt, wie dies funktioniert.


Beispiel:

with (obj_Box)
   {
   mp_grid_clear_cell(grid, floor(x / 32), floor(y /32));
   instance_destroy();
   }

Der obige Code wird alle "obj_Box" zerstören sich selbst und lassen Sie sie markieren die Zellen, die sie besetzt in der mp_grid in der Variable "Gitter" als frei. In diesem Beispiel finden wir die geeignete Zelle, indem wir die x / y-Koordinate des Objekts nehmen und sie durch die Auflösung des Gitters (mit floor um die Werte als ganze Zahlen zu behalten);
