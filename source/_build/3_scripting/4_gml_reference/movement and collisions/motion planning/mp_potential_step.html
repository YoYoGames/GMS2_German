mp_potential_schritt

Syntax:

mp_potential_step(xgoal, ygoal, stepsize, checkall)


Streit Beschreibung
xgoal Die Zielposition x.
ygoal Die Ziel-Y-Position.
stepsize Die Geschwindigkeit, mit der sich die Instanz in Pixel pro Schritt bewegt.
checkall Ob alle Instanzen geprüft werden sollen (true) oder nur feste (false).


Rückgabe: Boolean.


Beschreibung

Diese Funktion lässt die Instanz einen Schritt in Richtung einer bestimmten Position ausführen, die von xgoal / ygoal definiert wird, und versucht dabei Hindernisse zu vermeiden. Wenn die Instanz auf eine solide Instanz (oder auf eine Instanz, bei der checkall zutrifft) ausgeführt wird, wird die Bewegungsrichtung geändert, um zu versuchen, diese Instanz zu vermeiden und sich um sie herum zu bewegen. Dieser Ansatz funktioniert nicht garantiert, aber in den meisten Fällen wird er die Instanz effektiv zum Ziel hin bewegen. Die Funktion gibt zurück, ob das Ziel erreicht wurde oder nicht.


Beispiel:

if instance_exists(obj_Enemy)
   {
   var inst;
   inst = instance_nearest(x, y, obj_Enemy);
   mp_potential_step(inst.x, inst.y, 5, false);
   }

Der obige Code prüft zunächst, ob sich im Raum irgendwelche Instanzen von "obj_Enemy" befinden. Wenn es dort ist, dann findet das nächste und speichert seine ID in einer Variablen. Diese Variable wird dann verwendet, um zu erzählen mp_potential_step Verschieben Sie die Instanz mit dem Code in Richtung der x / y-Position des Objekts, das mit einer Geschwindigkeit von 5 Pixeln pro Schritt gefunden wurde, und vermeiden Sie nur Instanzen, die als solide gekennzeichnet sind.
