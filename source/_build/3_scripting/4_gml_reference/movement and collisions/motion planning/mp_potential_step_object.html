mp_potential_schritt_objekt

Syntax:

mp_potential_step_object(xgoal, ygoal, stepsize, obj)


Streit Beschreibung
xgoal Die Zielposition x.
ygoal Die Ziel-Y-Position.
stepsize Die Geschwindigkeit, mit der sich die Instanz in Pixel pro Schritt bewegt.
obj Das Objekt, das den Pfad der Instanz blockieren soll, auf der die Funktion ausgeführt wird. Kann ein Objektindex, eine Instanz-ID oder das spezielle Schlüsselwort all sein.


Rückgabe: Boolean.


Beschreibung

Diese Funktion lässt die Instanz einen Schritt in Richtung einer bestimmten Position ausführen, die von xgoal / ygoal definiert wird, und versucht dabei Hindernisse zu vermeiden. Wenn die Instanz auf eine Instanz des Objekts zugreifen würde, das durch das Argument "obj" angegeben wurde, ändert sie die Bewegungsrichtung, um zu versuchen, diese Instanz zu vermeiden und sich um sie herum zu bewegen. Dieser Ansatz funktioniert nicht garantiert, aber in den meisten Fällen wird er die Instanz effektiv zum Ziel hin bewegen. Die Funktion gibt zurück, ob das Ziel erreicht wurde oder nicht.


Beispiel:

if instance_exists(obj_Enemy)
   {
   var inst;
   inst = instance_nearest(x, y, obj_Enemy);
   mp_potential_step_object(inst.x, inst.y, 5, obj_Wall);
   }

Der obige Code prüft zunächst, ob sich im Raum irgendwelche Instanzen von "obj_Enemy" befinden. Wenn es dort ist, dann findet das nächste und speichert seine ID in einer Variablen. Diese Variable wird dann verwendet, um zu erzählen mp_potential_step_object Verschieben der Instanz mit dem Code in Richtung der x / y-Position des Objekts, das mit einer Geschwindigkeit von 5 Pixeln pro Schritt gefunden wurde, wobei nur Instanzen des Objekts "obj_Wall" vermieden werden.
