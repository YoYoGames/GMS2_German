mp_grid_destroy

Syntax:

mp_grid_destroy(id);


Streit Beschreibung
id Index des mp_grid, der zerstört werden soll


Rückkehr: N / A.


Beschreibung

Sie können diese Funktion verwenden, um das angezeigte mp_grid zu zerstören und den zugewiesenen Speicher freizugeben. Es ist wichtig, dass Sie dies immer dann aufrufen, wenn mp_grid beendet ist, oder Sie könnten möglicherweise ein Speicherleck bekommen, was bedeutet, dass Ihr Spiel mit der Zeit langsamer wird und schließlich abstürzt.

HINWEIS: Das Zerstören eines mp_grid, wenn noch Instanzen verwendet werden, führt zu einem Fehler!


Beispiel:

if timer = 0
   {
   mp_grid_destroy(grid);
   room_goto(rm_Menu);
   }

Der obige Code zerstört das mp_grid, das in der Variablen "grid" indiziert ist, wenn die Variable "timer" gleich 0 ist, und dann in ein anderes Zimmer geht.
