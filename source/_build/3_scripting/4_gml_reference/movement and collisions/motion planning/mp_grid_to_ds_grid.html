mp_grid_to_ds_grid

Syntax:

mp_grid_to_ds_grid(source, destination);


Streit Beschreibung
source Index des mp_grid, das verwendet werden soll
destination Index des ds_grid, mit dem die Rasterdaten kopiert werden sollen.


Rückgabe: Boolean.


Beschreibung

Mit dieser Funktion können Sie das gegebene MP-Raster in ein DS-Raster kopieren. Das DS-Grid sollte die gleiche Größe wie das MP-Grid haben, obwohl dies nicht notwendig ist (Daten gehen verloren, wenn sie kleiner sind, und wenn sie größer sind, sind alle zusätzlichen Grid-Zellen 0). Die DS-Gitterzellen enthalten den Wert -1, wenn die MP-Gitterzelle als belegt markiert wurde, oder sie ist 0, wenn dies nicht der Fall ist.


Beispiel:

motion_grid = ds_grid_create(room_width / 32, room_height / 32); mp_grid_to_ds_grid(mp_grid, motion_grid);

Der obige Code erstellt ein neues DS-Grid und kopiert dann die MP-Grid-Daten, die in der Variablen "mp_grid" enthalten sind, in das neue DS-Grid.
