place_snapped

Syntax:

place_snapped(hsnap, vsnap);


Streit Beschreibung
hsnap Das horizontale Einrasten zum Überprüfen.
vsnap Das vertikale Einrasten zum Überprüfen.


Rückgabe: Boolean


Beschreibung

Mit dieser Funktion können Sie überprüfen, ob der Ursprung einer Instanz (ihre X- und Y-Position) mit den von Ihnen angegebenen Werten für hsnap und vsnap auf ein Raster ausgerichtet ist.


Beispiel:

with (obj_Pieces)
   {
   if !place_snapped(32, 32)
      {
      move_snap(32, 32);
      }
   }

Der obige Code prüft alle Instanzen von "obj_Pieces", um zu sehen, ob sie an ein Raster von 32x32 Pixeln gerastert sind, und wenn sie nicht vorhanden sind, werden sie an der nächsten Position in diesem Raster gefangen.
