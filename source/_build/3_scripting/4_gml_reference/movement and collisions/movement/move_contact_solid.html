move_contact_solid

Syntax:

move_contact_solid( dir, maxdist );


Streit Beschreibung
dir Die Richtung zum Einziehen.
maxdist Die maximale Entfernung, die das Objekt zurücklegen kann (-1 oder 0 ist ein Standardwert von 1000 Pixel).


Rückkehr: N / A


Beschreibung

Diese Funktion verschiebt die Instanz, in der der Code ausgeführt wird, um eine festgelegte Anzahl von Pixeln in der angegebenen Richtung, bis sie auf eine Instanz trifft, für die Solid als wahr markiert ist. Sie können -1 oder 0 für maxdist als Standard 1000px verwenden, dh: GameMaker Studio 2 verschiebt die Instanz kontinuierlich um 1000 Pixel, bis sie nicht mehr kollidiert.


Beispiel:

if !place_meeting(x, y + 1, obj_Floor)
   {
   move_contact_solid(270, -1);
   }

Der obige Code wird unterhalb der Instanz nach einer Kollision mit "obj_Floor" suchen, und wenn keine vorhanden ist, wird er nach unten bewegt, bis eine Kollision vorliegt oder 1000 Pixel abgedeckt wurden.
