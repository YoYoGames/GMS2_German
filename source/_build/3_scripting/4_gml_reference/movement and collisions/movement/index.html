Bewegung

In jedem Spiel, Bewegung und Position sind von größter Bedeutung und so hat GameMaker Studio 2 eine vollständige Auswahl an Funktionen, um mit jeder Situation umzugehen. Die zwei Hauptarten zum Verschieben einer Instanz sind entweder das Setzen der tatsächlichen Position oder das Festlegen eines Geschwindigkeits- / Richtungsvektors. Dies kann entweder mithilfe der Variablen für die integrierten Variableninstanzen oder mithilfe bestimmter Bewegungsfunktionen geschehen. Beide Optionen werden in den folgenden Abschnitten erläutert:

Die folgenden Variablen sind alle in jede Instanz "eingebaut" und können gelesen und geändert werden, um unterschiedliche Verhaltensweisen zu ändern, wenn sich die Instanz bewegt (sie wirken sich auf Geschwindigkeits- und Richtungsvektoren aus):

  • Richtung
  • Reibung
  • Schwere
  • Schwerkraft_Richtung
  • hspeed
  • vspeed
  • Geschwindigkeit

Abgesehen von den oben genannten Variablen können Sie die Instanzposition auch direkt wie folgt festlegen:

  • x
  • y
  • xvorhergehend
  • früher
  • xstart
  • ystart

Die folgenden Funktionen können alle verwendet werden, um eine Instanz in irgendeiner Weise zu verschieben, wobei einige die Geschwindigkeits- / Richtungsvektoren beeinflussen und andere die tatsächliche x / y-Position innerhalb des Raums direkt beeinflussen:

  • motion_add
  • Bewegungssatz
  • move_towards_point
  • move_bounce_all
  • move_bounce_solid
  • move_contact_all
  • move_contact_solid
  • move_outside_all
  • move_outside_solid
  • move_random
  • move_snap
  • move_wrap
  • place_snapped
