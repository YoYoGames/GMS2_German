move_towards_point

Syntax:

move_towards_point( x, y, sp );


Streit Beschreibung
x Die x Position des zu bewegenden Punktes.
y Die y-Position des zu bewegenden Punktes.
sp Die Geschwindigkeit, um sich in Pixel pro Sekunde zu bewegen.


Rückkehr: N / A


Beschreibung

Diese Funktion teilt der Instanz, die den Code ausführt, einfach mit, dass sie sich mit einer festgelegten Geschwindigkeit auf einen Sollwert zu bewegt. Denken Sie daran, dass es an Ihnen liegt, ihm zu sagen, was zu tun ist, wenn es einmal da ist, denn es wird nicht von selbst anhalten. Dies bedeutet, dass Sie eine "vibrierende" Instanz erhalten können, wenn sie sich beispielsweise 3 Pixel vorwärts bewegt, die x / y-Zielposition um 2 übersteuert, dann 3 Pixel wieder zurückbewegt, den Zielpunkt um eins überspringt und so weiter.. Es gibt eine Reihe von Möglichkeiten, wie dies vermieden werden kann, z distance_to_point um zu sehen, wie weit die Instanz von der angegebenen Position entfernt ist, bevor sie sich bewegt.

Hinweis: Diese Funktion legt die Geschwindigkeit der Instanz fest. Selbst wenn Sie diesen Code nicht mehr verwenden, bewegt sich die Instanz in der vorherigen Richtung weiter. Daher muss die Instanz festgelegt werden speed auf 0, wenn Sie möchten, dass es aufhört. Siehe das Codebeispiel unten.


Beispiel:

if point_distance(x, y, target.x, target.y) > 5
   {
   move_towards_point(target.x, target.y, 5);
   }
else speed = 0;

Der obige Code wird die Instanz mit einer Geschwindigkeit von 5 Pixeln pro Schritt zur Position der in der Variablen "Ziel" indizierten Instanz bewegen. Sobald diese Position erreicht ist, wird die Geschwindigkeit auf 0 gesetzt.
