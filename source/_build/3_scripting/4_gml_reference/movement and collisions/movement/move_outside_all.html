move_outside_all

Syntax:

move_outside_all(dir, maxdist);


Streit Beschreibung
dir Die Richtung zum Einziehen.
maxdist Die maximale Entfernung, die das Objekt zurücklegen kann (-1 oder 0 ist ein Standardwert von 1000 Pixel).


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie einer Instanz sagen, dass sie aus einer Kollision in jede Richtung und jede Anzahl von Pixeln in jedem Schritt mit einem Wert von -1 oder 0 ausfahren soll, da die maxdist Standard 1000px ist, dh: GameMaker Studio 2 wird die Instanz verschieben kontinuierlich 1000 Pixel hoch, bis es nicht mehr kollidiert.


Beispiel:

if place_meeting(x, y, all)
   {
   move_outside_all(90, 1);
   }

Der obige Code verschiebt die Instanz um 1 Pixel in jedem Schritt, in dem sie mit einer anderen Instanz kollidiert.
