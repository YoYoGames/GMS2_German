move_contact_all

Syntax:

move_contact_all(dir, maxdist);


Streit Beschreibung
dir Die Richtung zum Einziehen.
maxdist Die maximale Entfernung, die das Objekt zurücklegen kann (-1 oder 0 ist ein Standardwert von 1000 Pixel).


Rückkehr: N / A


Beschreibung

Diese Funktion verschiebt die Instanz, die den Code ausführt, um eine festgelegte Anzahl von Pixeln in der angegebenen Richtung, bis sie eine andere Instanz mit einer gültigen Maske erreicht. Sie können -1 oder 0 für maxdist als Standard 1000px verwenden, dh: GameMaker Studio 2 verschiebt die Instanz kontinuierlich um 1000 Pixel, bis sie nicht mehr kollidiert.


Beispiel:

if !place_meeting(x, y + 1, all)
   {
   move_contact_all(270, -1);
   }

Der obige Code wird unter der Instanz nach einer Kollision suchen, und wenn es keine gibt, wird er nach unten verschoben, bis oder bis 1000 Pixel abgedeckt sind.
