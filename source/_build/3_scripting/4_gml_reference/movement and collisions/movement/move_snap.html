move_snap

Syntax:

move_snap( hsnap, vsnap );


Streit Beschreibung
hsnap Das horizontale Fangen (die Größe in Pixel zwischen 'Zellen').
vsnap Das vertikale Fangen (die Größe in Pixel zwischen 'Zellen').


Rückkehr: N / A


Beschreibung

Diese Funktion wird verwendet, um die Instanz an ein Raster mit einer bestimmten Größe zu "schnappen". Es wird an die nächste entsprechende Position im "unsichtbaren" Raster, das die Werte hsnap und vsnap definieren, gerastet.


Beispiel:

with (obj_Pieces)
   {
   if !place_snapped(32, 32)
      {
      move_snap(32, 32);
      }
   }

Der obige Code prüft alle Instanzen von "obj_Pieces", um zu sehen, ob sie an ein Raster von 32x32 Pixeln gerastert sind, und wenn sie nicht vorhanden sind, werden sie an der nächsten Position in diesem Raster gefangen.
