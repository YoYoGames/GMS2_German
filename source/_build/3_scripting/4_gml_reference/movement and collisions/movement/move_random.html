move_random

Syntax:

move_random(hsnap, vsnap);


Streit Beschreibung
hsnap Das horizontale Fangen (die Größe in Pixel zwischen 'Zellen').
vsnap Das vertikale Fangen (die Größe in Pixel zwischen 'Zellen').


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie die Instanz an einer beliebigen Stelle im Raum positionieren, jedoch auf ein "unsichtbares" Raster ausrichten. Ein Wert von 32 für hsnap und vsnap setzt die Instanz auf eine zufällige Position, die auf ein Raster von 32 x 32 Quadraten ausgerichtet ist. Sie können diese Werte auf 1 setzen, um eine Position irgendwo im Raum zu erhalten.

HINWEIS: Diese Funktion wird jedes Mal, wenn das Spiel neu gestartet wird, auf die gleichen Positionen gesetzt, da GameMaker Studio 2 jedes Mal den gleichen anfänglichen zufälligen GameMaker erzeugt, um den Debugging-Code zu vereinfachen. Um dieses Verhalten zu vermeiden, verwenden Sie randomise zu Beginn deines Spiels. Dies trifft nur zu, wenn das Spiel getestet und debuggt wird, da das endgültige ausführbare Paket dieses Verhalten nicht zeigen wird und bei jedem Spiel zufällig sein wird.


Beispiel:

if keyboard_check(vk_space) move_random(1,1);

Dadurch wird die Instanz an eine beliebige Position im Raum verschoben, wenn die Leertaste gedrückt wird.
