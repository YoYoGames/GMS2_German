Bewegungssatz

Syntax:

motion_set(dir, speed);


Streit Beschreibung
dir Die neue Richtung.
speed Die neue Geschwindigkeit


Rückkehr: N / A


Beschreibung

Diese Funktion legt der Instanz, die den Code ausführt, eine neue Bewegungsrichtung und eine neue Geschwindigkeit fest. Beachten Sie, dass dies nicht zur aktuellen Geschwindigkeit und Richtung der Instanzen beiträgt (dafür würden Sie verwenden) motion_add ) sondern zwingt es zu den neuen Einstellungen.


Beispiel:

if irandom(9) = 1 motion_set(random(360), 1 + random(3));

Dieser obige Code bewirkt, dass die Instanz Geschwindigkeit und Richtung in zufälligen Intervallen ändert.
