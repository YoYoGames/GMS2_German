Ort_leer

Syntax:

place_empty(x, y);


Streit Beschreibung
x Die zu überprüfende x-Position.
y Die zu prüfende y-Position.


Rückgabe: Boolean.


Beschreibung

Sie können diese Funktion verwenden, um zu überprüfen, ob die aufrufende Instanz mit einer anderen Instanz in Ihrem Spiel kollidieren würde. Nun sollte beachtet werden, dass die Instanz, auf der der Code ausgeführt wird, eine gültige Kollisionsmaske haben muss (entweder für das Sprite selbst oder über die mask_index ) und es werden nur Kollisionen mit denjenigen Instanzen registriert, die auch eine gültige Maske haben.

Die Funktion selbst arbeitet im Wesentlichen indem die Instanz genommen und auf Kollisionen getestet wird, wenn sie an der durch die x / y-Argumente angegebenen Position platziert wird. Die Kollisionsprüfung kann entweder präzise oder basierend auf der Begrenzungsbox der Instanz erfolgen, abhängig davon, welche Art von Kollisionsmaske ausgewählt wurde.

Beachten Sie, dass die angegebenen x / y-Koordinaten vor der Überprüfung auf die nächste Ganzzahl gesetzt werden.


Beispiel:

if place_empty(mouse_x, mouse_y)
   {
   x = mouse_x;
   y = mouse_y;
   }

Der obige Code prüft auf eine Kollision mit einer anderen Instanz, wenn die aufrufende Instanz an der gleichen Position wie die Maus platziert werden soll. Wenn keine Kollision erkannt wird, werden die x / y-Koordinaten der Instanz auf die der Maus gesetzt.
