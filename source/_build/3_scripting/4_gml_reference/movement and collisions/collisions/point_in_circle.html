point_in_circle

Syntax:

point_in_circle(px, py, x1, y1, rad);


Streit Beschreibung
px Die X-Koordinate des zu überprüfenden Punktes.
py Die y-Koordinate des zu überprüfenden Punktes.
x1 Die x-Koordinate des Kreismittelpunkts.
y1 Die y-Koordinate des Kreismittelpunkts.
rad Der Radius des Kreises.


Rückgabe: Boolean


Beschreibung

Wenn Sie diese Funktion verwenden, definieren Sie einen kreisförmigen Bereich und GameMaker Studio 2 wird herausfinden, ob der angegebene Punkt innerhalb seiner Grenzen liegt oder nicht. Wenn der Punkt innerhalb des definierten Kreises liegt, kehrt die Funktion zurück true Andernfalls wird die Funktion zurückkehren false.


Beispiel:

if point_in_circle(mouse_x, mouse_y, x, y, 16)
   {
   over = true;
   }
else
   {
   over = false;
   }

Der obige Code verwendet die point_in_circle Funktion, um zu prüfen, ob die Mausposition in den definierten Kreisbereich fällt und die Variable auf "over" setzt true wenn es das tut oder false Andernfalls.
