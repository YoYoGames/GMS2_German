rechteckle_in_triangle


Beschreibung

Diese Funktion prüft einen von Ihnen definierten rechteckigen Bereich, um zu sehen, ob er entweder nicht in Kollision ist, sich vollständig innerhalb der Zielgrenzen befindet oder wenn er einen definierten dreieckigen Bereich berührt. Wenn sie sich überhaupt nicht berühren, gibt die Funktion 0 zurück. Wenn sich die Quelle vollständig innerhalb des Ziels befindet, wird 1 zurückgegeben. Wenn sie sich nur überschneiden, wird 2 zurückgegeben. Das folgende Bild veranschaulicht dies:


Syntax:

rectangle_in_triangle(sx1, sy1, sx2, sy2, dx1, dy1, dx2, dy2, dx3, dy3);

Streit Beschreibung
sx1 Die x-Koordinate der linken Seite des Quellrechtecks.
sy1 Die y-Koordinate der Oberseite des Quellrechtecks.
sx2 Die x-Koordinate der rechten Seite des Quellrechtecks.
sy2 Die y-Koordinate der Unterseite des Quellrechtecks.
x1 Die x-Koordinate der ersten Seite des Dreiecks, die geprüft werden soll.
y1 Die y-Koordinate der ersten Seite des Dreiecks, die geprüft werden soll.
x2 Die x-Koordinate der zweiten Seite des Dreiecks, die geprüft werden soll.
y2 Die y-Koordinate der zweiten Seite des Dreiecks, die geprüft werden soll.
x3 Die x-Koordinate der dritten Seite des Dreiecks, die geprüft werden soll.
y3 Die y-Koordinate der dritten Seite des Dreiecks, die geprüft werden soll.


Kehrt zurück:

Real


Beispiel:

var inst = instance_nearest(x, y, obj_Player);
if instance_exists(inst)
   {
   var x1 = x + lengthdir_x(100, image_angle - 45);
   var y1 = y + lengthdir_y(100, image_angle - 45);
   var x2 = x + lengthdir_x(100, image_angle + 45);
   var y2 = y + lengthdir_y(100, image_angle + 45);
   if rectangle_in_triangle(inst.bbox_left, inst.bbox_top, inst.bbox_right, inst.bbox_bottom, x, y, x1, y1, x2, y2) == 1
      {
      can_see = true;
      }
   }

Der obige Code verwendet die rectangle_in_triangle Funktion als "Vision-Kegel", um nach einer Instanz von "obj_player" zu suchen. Wenn sich der volle Begrenzungsrahmen innerhalb des angegebenen dreieckigen Bereichs befindet, wird eine Variable auf festgelegt true.
