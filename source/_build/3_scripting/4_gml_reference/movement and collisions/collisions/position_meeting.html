position_treffen

Syntax:

position_meeting(x, y, obj);


Streit Beschreibung
x Die zu überprüfende x-Position.
y Die zu prüfende y-Position.
obj Das Objekt (oder die Instanz-ID oder die Schlüsselwörter "all" oder "other"), auf die eine Kollision überprüft werden soll.


Rückgabe: Boolean


Beschreibung

Mit dieser Funktion können Sie eine Position auf eine Kollision mit einer anderen Instanz oder auf alle Instanzen eines Objekts überprüfen. Wenn Sie dies verwenden, prüfen Sie einen einzelnen Punkt im Raum auf eine Instanz oder ein Objekt. Diese Überprüfung wird für die Begrenzungsbox der Instanz oder für die Maske der Instanz durchgeführt, wenn für diese Instanz präzise Kollisionen aktiviert sind. Wenn Sie wissen müssen, erhalten Sie die eindeutige Instanz- ID, wenn das Objekt mit Ihnen kollidieren sollte instance_position. Diese Funktion akzeptiert auch die speziellen Schlüsselwörter alle und andere.


Beispiel:

if mouse_check_button(mb_left)
   {
   if !position_meeting(mouse_x, mouse_y, all) instance_create_layer(mouse_x, mouse_y, "Walls", obj_Wall);
   }

Der obige Code prüft auf die linke Maustaste, und wenn er gedrückt wird, prüft er die Maus-x / y-Position auf eine Kollision mit einer beliebigen Instanz. Wenn keine vorhanden ist, wird eine Instanz von "obj_Wall" erstellt.
