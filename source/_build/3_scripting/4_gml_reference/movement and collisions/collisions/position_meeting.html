position_meeting


Beschreibung

Mit dieser Funktion können Sie eine Position auf eine Kollision mit einer anderen Instanz oder auf alle Instanzen eines Objekts prüfen. Wenn Sie dies verwenden, überprüfen Sie einen einzelnen Punkt im Raum auf eine Instanz oder ein Objekt. Diese Prüfung wird gegen die Begrenzungsbox der Instanz oder gegen die Maske der Instanz durchgeführt, wenn für diese Instanz genaue Kollisionen geprüft wurden. Wenn Sie die eindeutige Instanz- ID benötigen, wenn das Objekt, mit dem Sie kollidieren, verwendet werden soll instance_position. Diese Funktion übernimmt auch die speziellen Schlüsselwörter alle und andere.


Syntax:

position_meeting(x, y, obj);

Streit Beschreibung
x Die zu überprüfende x-Position.
y Die zu überprüfende y-Position.
obj Das Objekt (oder die Instanz-ID oder die Schlüsselwörter "all" oder "other"), mit denen auf Kollision geprüft werden soll.


Kehrt zurück:

Boolean


Beispiel:

if mouse_check_button(mb_left)
   {
   if !position_meeting(mouse_x, mouse_y, all) instance_create_layer(mouse_x, mouse_y, "Walls", obj_Wall);
   }

Der obige Code sucht nach der linken Maustaste und wenn er gedrückt wird, prüft er die x / y-Position der Maus auf eine Kollision mit einer Instanz. Wenn keine vorhanden ist, wird eine Instanz von "obj_Wall" erstellt.
