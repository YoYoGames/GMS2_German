place_free


Beschreibung

Mit dieser Funktion können Sie prüfen, ob die aufrufende Instanz mit einer Instanz kollidiert, die in Ihrem Spiel als solide gekennzeichnet ist. Nun sollte beachtet werden, dass die Instanz, die den Code ausführt, über eine gültige Kollisionsmaske verfügen muss (entweder für das Sprite selbst oder durch die mask_index ) und es werden nur Kollisionen mit den markierten Instanzen registriert, die auch über eine gültige Maske verfügen.

Die Funktion selbst funktioniert im Wesentlichen, indem die Instanz genommen und auf Kollisionen geprüft wird, wobei nur Instanzen mit fester Markierung gekennzeichnet sind, wenn sie an der durch die x / y-Argumente angegebenen Position platziert werden. Die Kollisionsprüfung kann entweder genau sein oder auf dem Begrenzungsrahmen der Instanz basieren, abhängig davon, welche Art von Kollisionsmaske ausgewählt wurde.

Beachten Sie, dass die angegebenen x / y-Koordinaten vor der Prüfung auf die nächste Ganzzahl gesetzt werden.


Syntax:

place_free(x, y);

Streit Beschreibung
x Die zu überprüfende x-Position.
y Die zu überprüfende y-Position.


Kehrt zurück:

Boolean


Beispiel:

if place_free(mouse_x, mouse_y)
   {
   x = mouse_x;
   y = mouse_y;
   }

Der obige Code prüft, ob eine Kollision mit einer Instanz mit solider Markierung vorliegt, wenn die aufrufende Instanz an derselben Position wie die Maus platziert werden soll. Wenn keine Kollision erkannt wird, sind die x / y-Koordinaten der Instanz auf die der Maus gesetzt.
