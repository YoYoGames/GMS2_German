collision_ellipse


Beschreibung

Collision_ellipse verwendet die ersten vier Argumente (x1, y1, x2, y2), um die Breite und Höhe einer Ellipse innerhalb des aktuellen Raums zu definieren, und prüft, ob ein Objekt, das durch das Argument "obj" definiert ist, damit kollidiert Bereich. Diese Kollision kann als genau geprüft werden oder nicht. Sie können auch prüfen, ob die Instanz den Code selbst ausführt oder nicht. Betrachten Sie dieses Bild:

Hier in der Mitte wird eine Kollisionsellipse verwendet, um nach Ballobjekten zu suchen. Nun versteht die blauen keine genauen Begrenzungsrahmen hat und wie Sie sehen können, auch wenn das Sprit nicht tatsächlich die Ellipse berührt, kann die Kollision noch passieren (auch wenn Sie die genaue Option in der Funktion auf true gesetzt), wie die Das Begrenzungsfeld dieses Sprites überlappt den durch collision_circle definierten elliptischen Bereich. Auf der anderen Seite werden die grünen Kugeln nur dann als Kollision betrachtet, wenn das eigentliche Sprite die definierte Ellipse überlappt. Um präzise Kollisionen zu berücksichtigen, müssen sowohl das Objekt-Sprite als auch die Kollisionsfunktion genau als aktiviert gekennzeichnet sein. Es sollte auch beachtet werden, dass der Rückgabewert der Funktion die ID einer der Instanzen sein kann, die als in Kollision betrachtet werden.


Syntax:

collision_ellipse(x1, y1, x2, y2, obj, prec, notme);

Streit Beschreibung
x1 Die x-Koordinate der linken Seite der Ellipse, die geprüft werden soll.
y1 Die y-Koordinate der oberen Seite der Ellipse, die geprüft werden soll.
x2 Die x-Koordinate der rechten Seite der Ellipse, die geprüft werden soll.
y2 Die y-Koordinate der unteren Seite der Ellipse, die geprüft werden soll.
obj Das Objekt, das auf Kollisionen geprüft werden soll.
prec Ob die Prüfung auf genauen Kollisionen basiert ( true, der langsamer ist) oder dessen Begrenzungsrahmen im Allgemeinen ( false, schneller).
notme Ob die aufrufende Instanz ggf. ausgeschlossen werden soll ( true ) oder nicht ( false ).


Kehrt zurück:

Instance id or noone


Beispiel:

if collision_ellipse(50, 50, 200, 100, obj_Player, false, true)
   {
   instance_create_layer(obj_Player.x, obj_Player.y, "Effects", obj_Splash);
   }

Dadurch wird eine elliptische Zone innerhalb der Grenzen von 50x, 50y und 200x, 100y für das Objekt "obj_Player" geprüft. Bei einer Kollision mit diesem Objekt wird eine Instanz von "obj_Splash" an den x / y-Koordinaten von obj_Player erstellt.
