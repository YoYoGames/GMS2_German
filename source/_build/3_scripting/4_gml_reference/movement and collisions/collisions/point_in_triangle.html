Punkt_in_Dreieck

Syntax:

point_in_triangle(px, py, x1, y1, x2, y2, x3, y3);


Streit Beschreibung
px Die X-Koordinate des zu überprüfenden Punktes.
py Die y-Koordinate des zu überprüfenden Punktes.
x1 Die X-Koordinate der ersten Ecke des zu überprüfenden Dreiecks.
y1 Die y-Koordinate der ersten Ecke des zu überprüfenden Dreiecks.
x2 Die X-Koordinate der zweiten Ecke des zu überprüfenden Dreiecks.
y2 Die y-Koordinate der zweiten Ecke des zu überprüfenden Dreiecks.
x3 Die x-Koordinate der dritten Ecke des zu überprüfenden Dreiecks.
y3 Die y-Koordinate der dritten Ecke des zu überprüfenden Dreiecks.


Rückgabe: Boolean


Beschreibung

Wenn Sie diese Funktion verwenden, definieren Sie einen dreieckigen Bereich und GameMaker Studio 2, ob der angegebene Punkt innerhalb seiner Grenzen liegt oder nicht. Wenn der Punkt in das definierte Dreieck fällt, kehrt die Funktion zurück true Andernfalls wird die Funktion zurückkehren false.


Beispiel:

var inst = instance_nearest(x, y, obj_Player);
if instance_exists(inst)
   {
   var x1 = x + lengthdir_x(100, image_angle - 45);
   var y1 = y + lengthdir_y(100, image_angle - 45);
   var x2 = x + lengthdir_x(100, image_angle + 45);
   var y2 = y + lengthdir_y(100, image_angle + 45);
   if point_in_triangle(inst.x, inst.y, x, y, x1, y1, x2, y2)
      {
      can_see = true;
      }
   }

Der obige Code verwendet die point_in_triangle Funktion als "Sichtkegel", um nach einer Instanz von "obj_player" zu suchen, indem eine Variable auf gesetzt wird true sollten die Objekte x / y-Position in das definierte Dreieck fallen.
