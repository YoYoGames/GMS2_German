script_get_name


Syntax:

script_get_name(scr);

Streit Beschreibung
scr Der Index des Skripts, für das Sie den Namen erhalten möchten.


Kehrt zurück:

String


Beschreibung

Diese Funktion gibt den Namen als Zeichenfolge des angegebenen Skripts zurück. Dieser Name ist der Name, der für das Skript im Ressourcenbaum des Hauptfensters von GameMaker Studio 2 festgelegt wurde. Weitere Informationen zu Skripts finden Sie im Skript-Editor.


Beispiel:

scr_name = script_get_name(scr_Help_File);

Der obige Beispielcode speichert den Namen des angegebenen Skript-Index in der Variablen "scr_name".
