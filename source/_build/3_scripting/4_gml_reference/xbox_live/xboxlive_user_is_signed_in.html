xboxlive_user_is_signed_in


Beschreibung

Diese Funktion ist nur beim Exportieren mit dem UWP-Ziel für die Xbox One gültig und erfordert, dass Sie die Option " Enable XBox Live " in den UWP-Spieloptionen aktiviert haben. Die Funktion überprüft, ob ein Benutzer angemeldet ist und zurückkehrt true wenn es und gibt false Andernfalls. Wenn kein Benutzer angemeldet ist, können Sie die Funktion aufrufen xboxlive_show_account_picker um die Kontoauswahl zu öffnen und zur Anmeldung aufzufordern.


Syntax:

xboxlive_user_is_signed_in();


Rückgabe: Boolean


Beispiel:

if !xboxlive_user_is_signed_in()
    {
    if !xboxlive_user_is_signing_in()
        {
        xboxlive_show_account_picker();
        }
    }
else
    {
    global.GamerTag = xboxlive_gamertag_for_user();
    }

Der obige Code überprüft, ob ein Benutzer in XBox Live angemeldet ist und wenn er angemeldet ist, erhält der Code den Gamertag für den Benutzer und speichert ihn zur zukünftigen Verwendung in einer globalen Variablen. Wenn sie nicht angemeldet sind, prüft der Code, ob sie gerade angemeldet sind. In diesem Fall wird nichts weiter passieren. Ist dies nicht der Fall, wird die Kontoauswahl geöffnet, und sie werden aufgefordert, zu unterschreiben -im.
