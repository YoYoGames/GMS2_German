Räume

Wenn Sie Räume in GameMaker Studio 2 erstellen, können Sie den dedizierten Raum-Editor verwenden oder dynamische Rauminhalte mithilfe von Code oder vielleicht auch ein wenig von beiden erstellen. Die meisten Aspekte eines Raums können zur Laufzeit geändert, erstellt oder entfernt werden, was bedeutet, dass du prozedurale Dungeons oder dynamische Zerstörung einfach erstellen kannst, wenn du dein Spiel erstellst.

Eine Übersicht über alle verfügbaren Funktionen finden Sie in den folgenden Abschnitten:

  1. Allgemeine Raumfunktionen und Variablen
  2. Allgemeine Layer-Funktionen
  3. Tilemaps
  4. Hintergründe
  5. Sprites

Beim Erstellen von Raumlayern ist es wichtig zu beachten, dass Sie mehrere verschiedene Elementtypen auf einer Ebene haben können. So könntest du zum Beispiel eine Ebene für Feinde haben und auf dieser Ebene die feindlichen Instanzen sowie bestimmte Sprites, die nur mit diesem Gegner zusammenhängen. Oder Sie könnten ein Hintergrundelement mit einem Kachelelement haben, um das Aussehen eines Raums zu erstellen. Dies ist im Raumeditor nicht erlaubt, aber beim dynamischen Erstellen von Dingen können Sie das tun, weshalb Funktionen wie layer_get_all_elements existieren.
