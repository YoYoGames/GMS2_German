layer_sprite_exists


Beschreibung

Sie können diese Funktion verwenden, um zu überprüfen, ob ein Sprite-Element auf einer bestimmten Ebene vorhanden ist. Sie geben die Layer-ID an (die Sie erhalten, wenn Sie die Ebene mit erstellen layer_create() ) oder der Layername (als String - dies wirkt sich auf die Leistung aus) und die Sprite-Element-ID (die Sie erhalten, wenn Sie ein Sprite-Element mit erstellen layer_background_create() oder wenn Sie die Funktion verwenden layer_sprite_get_id() ) und die Funktion wird entweder zurückgeben true wenn das Element existiert, oder false wenn es nicht.

HINWEIS: Diese Funktion arbeitet im Rahmen des aktuellen Zielraums - standardmäßig der Raum, in dem die Funktion aufgerufen wird - der mit der Funktion eingestellt werden kann layer_set_target_room().


Syntax:

layer_sprite_exists(layer_id, sprite_element_id)


Argumente:

Streit Beschreibung
layer_id Der eindeutige ID-Wert des zu targetierenden Layers (oder der Layername als String)
sprite_element_id Der eindeutige ID-Wert des zu überprüfenden Sprite-Elements


Kehrt zurück:

Boolean


Beispiel:

var lay_id = layer_get_id("Asset_Trees");
if layer_sprite_exists(lay_id, global.TreesSprites)
   {
   layer_sprite_destroy(lay_id, global.TreesSprites);
   }

Der obige Code überprüft den Layer "Asset_trees", um zu sehen, ob das gegebene Sprite-Element existiert und falls dies der Fall ist, wird es zerstört (aber nicht der Layer).
