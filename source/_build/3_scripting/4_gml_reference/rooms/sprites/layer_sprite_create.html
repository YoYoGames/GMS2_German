layer_sprite_create


Mit dieser Funktion können Sie einer Ebene, die in Ihrem Projekt verwendet werden soll, eine Sprite-Ressource zuweisen. Sie geben die Layer-ID an (die Sie erhalten, wenn Sie die Ebene mit erstellen layer_create() oder wenn Sie den Ebenennamen zusammen mit verwenden layer_get_id() ), eine Position innerhalb des Raumes und ein Sprite-Index (der Name des Sprites wäre wie im Ressourcenbaum gezeigt), und er wird der Ebene hinzugefügt. Die Funktion gibt den eindeutigen ID-Wert für das Element zurück, der dann in weiteren Layer-Funktionen für Sprites verwendet werden kann.


Syntax:

layer_sprite_create(layer_id, x, y, sprite)


Argumente:

Streit Beschreibung
layer_id Der eindeutige ID-Wert der zu targetenden Ebene
x Die zu verwendende x-Position
y Die zu verwendende y-Position
sprite Der zu verwendende Sprite-Index


Kehrt zurück:

Sprite element ID (Real)


Beispiel:

global.asset_layer = layer_create(10000);
for (var i = 0; i< 10; i++;)
   {
   global.asset_spr_trees[i] = layer_sprite_create(global.asset_layer, random(room_width), room_height - 100, spr_Trees);
   }

Der obige Code erstellt eine neue Ebene und fügt dann 10 neue Sprite-Elemente hinzu und speichert die ID jedes Elements in einem Array.
