layer_sprite_xscale


Beschreibung

Mit dieser Funktion können Sie ändern, ob das angegebene Sprite-Element auf einer Ebene entlang der x-Achse skaliert werden soll oder nicht. Sie geben die Sprite-Element-ID (die Sie erhalten, wenn Sie ein Sprite-Element erstellen mit layer_sprite_create() oder wenn Sie die Funktion verwenden layer_sprite_get_id() ) und stellen Sie dann den Skalierungswert ein. Eine Skala von 1 zeigt keine Skalierung an (1: 1), kleinere Werte werden verkleinert (0,5, zum Beispiel wird die Hälfte der Breite des Sprites verwendet), größere Werte skalieren und negative Werte drehen das Sprite und skalieren es es sei denn, der verwendete Wert ist genau -1 (in diesem Fall wird das verwendete Sprite nur um seine Ursprungsposition ohne Skalierung von rechts nach links gekippt).


Syntax:

layer_sprite_xscale(sprite_element_id, xscale);


Argumente:

Streit Beschreibung
sprite_element_id Der eindeutige ID-Wert des zu ändernden Sprite-Elements
xscale Der xscale-Wert (Standardwert ist 1)


Kehrt zurück:

N/A


Beispiel:

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if layer_sprite_get_xscale(asset_sprite) != 1 || !ayer_sprite_get_yscale(asset_sprite) != 1
   {
   layer_sprite_xscale(asset_sprite, 1);
   layer_sprite_yscale(asset_sprite, 1);
   }

Der obige Code überprüft das Sprite-Element, das dem Layer zugeordnet ist, auf dem die Instanz ausgeführt wird, auf der der Code ausgeführt wird, und wenn es in beide Richtungen skaliert wird, werden sowohl die X-Achsenskalierung als auch die Y-Achsenskalierung auf 1 gesetzt.
