layer_sprite_speed


Beschreibung

Diese Funktion kann verwendet werden, um die Geschwindigkeit eines animierten Bildes festzulegen, das einem Sprite-Element zugewiesen wurde. Sie geben die Sprite-Element-ID (die Sie erhalten, wenn Sie ein Sprite-Element erstellen mit layer_sprite_create() oder wenn Sie die Funktion verwenden layer_sprite_get_id() ) und stellen Sie dann die Animationsgeschwindigkeit ein. Wenn die IDE die Animationsgeschwindigkeit für das Sprite festgelegt hat, das als Frames pro Spiel-Frame berechnet wurde, ist der angegebene Geschwindigkeitswert ein Multiplikator, wobei 1 der Standardwert und 0,5 die halbe Animationsgeschwindigkeit ist, während der Einstellung es auf 1,5 wird es verdoppeln. Wenn die IDE jedoch die Animationsgeschwindigkeit für das Sprite festgelegt hat, das als Frames pro Sekunde berechnet wird, ist der angegebene Geschwindigkeitswert ein Literal-Frames pro Sekunde. Wenn Sie also beispielsweise 5 einstellen, wird die Animation in eine Schleife gesetzt 5 Frames pro Sekunde, in denen das Spiel läuft. Wenn das verwendete Sprite keine Unterbilder hat, hat diese Funktion keine Wirkung.


Syntax:

layer_sprite_speed(sprite_element_id, speed);


Argumente:

Streit Beschreibung
sprite_element_id Der eindeutige ID-Wert des zu ändernden Sprite-Elements
speed Der Geschwindigkeitsmultiplikator / -wert, der für das Element verwendet werden soll


Kehrt zurück:

N/A


Beispiel:

var lay_id = layer_get_id("Assets_water");
var spr_id = layer_sprite_get_id(lay_id, "gfc_water");
layer_sprite_speed(spr_id, 0.5);

Der obige Code ruft die Layer-ID für den Layer "Assets_water" ab und verwendet diese, um die ID des angegebenen Sprite-Elements auf diesem Layer zu erhalten. Diese ID wird dann verwendet, um die Elementbildgeschwindigkeit zu ändern.
