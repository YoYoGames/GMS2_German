layer_sprite_alpha


Beschreibung

Diese Funktion steuert das Alpha (Transparenz) des Sprites auf der Asset-Ebene. Sie geben die Sprite-Element-ID (die Sie erhalten, wenn Sie ein Sprite-Element erstellen mit layer_sprite_create() oder wenn Sie die Funktion verwenden layer_sprite_get_id() ) und legen Sie den zu verwendenden Alpha-Wert fest. Alpha kann zwischen 0 (vollständig transparent) und 1 (vollständig undurchsichtig) mit dem Standard-Alpha-Wert für das Sprite-Element 1 liegen. Beachten Sie, dass wenn die Ebene, der das Sprite-Element zugewiesen wurde, nicht sichtbar ist - oder das Element selbst erstellt wurde unsichtbar - Sie werden keinen Unterschied zu dieser Funktion sehen, bis die Ebene oder das Element wieder sichtbar gemacht wurde.


Syntax:

layer_sprite_alpha(sprite_element_id, alpha);


Argumente:

Streit Beschreibung
sprite_element_id Der eindeutige ID-Wert des zu ändernden Sprite-Elements
alpha Das Alpha für Sprite Sprite, von 0 bis 1 (Standard ist 1)


Kehrt zurück:

N/A


Beispiel:

var lay_id = layer_get_id("Asset_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
layer_sprite_alpha(spr_id, random(1));

Der obige Code erhält den ID-Wert des Sprite-Assets namens "Clouds", das dem Layer "Asset_sky" zugewiesen ist, und setzt sein Alpha auf einen zufälligen Wert zwischen 0 und 1.
