layer_sprite_index


Beschreibung

Diese Funktion kann verwendet werden, um den Bildindex eines Sprite-Objekts festzulegen, das mehrere Teilbilder auf einer Ebene hat. Sie geben die Sprite-Element-ID (die Sie erhalten, wenn Sie ein Sprite-Element erstellen mit layer_sprite_create() oder wenn Sie die Funktion verwenden layer_sprite_get_id() ) und legen Sie dann den zu verwendenden Bildindex fest. Wenn Sie einen Wert außerhalb des Bereichs von Teilbildern festlegen, wird der Bildindex umlaufen.


Syntax:

layer_sprite_index(sprite_element_id, image_index);


Argumente:

Streit Beschreibung
sprite_element_id Der eindeutige ID-Wert des festzulegenden Sprite-Elements
index Der Bildindex, der für das Sprite verwendet werden soll


Kehrt zurück:

N/A


Beispiel:

var lay_id = layer_get_id("sprite_trees");
var spr_id = layer_sprite_get_id(lay_id, "gfc_trees");
layer_sprite_index(spr_id, 1);

Der obige Code ruft die Layer-ID für den Layer "sprite_trees" ab und verwendet diese, um die ID des angegebenen Sprite-Elements auf diesem Layer zu erhalten. Diese ID wird dann verwendet, um den Elementbildindex zu ändern.
