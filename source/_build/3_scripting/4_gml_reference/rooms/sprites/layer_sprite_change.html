layer_sprite_change


Mit dieser Funktion können Sie die einem bestimmten Sprite-Element auf einem Layer zugewiesene Sprite-Ressource ändern. Sie geben die Sprite-Element-ID (die Sie erhalten, wenn Sie ein Sprite-Element erstellen mit layer_sprite_create() oder wenn Sie die Funktion verwenden layer_sprite_get_id() ) und dann ein Sprite-Indexelement-Asset wird geändert. Beachten Sie, dass wenn das Sprite Sub-Images hat, es auch animiert wird (dies kann mit den layer_sprite_index und layer_sprite_speed Funktionen). Beachten Sie, dass Sie als neuen Sprite-Index einen Wert von -1 zuweisen können und kein Sprite angezeigt wird, obwohl das Element weiterhin existiert und später noch geändert werden kann.


Syntax:

layer_sprite_change(sprite_element_id, sprite_index)


Argumente:

Streit Beschreibung
sprite_element_id Der eindeutige ID-Wert des zu ändernden Sprite-Elements
sprite_index Der neue zu verwendende Sprite-Index


Kehrt zurück:

N/A


Beispiel:

var s = layer_sprite_get_id(layer, global.Asset_sprite);
if layer_sprite_get_sprite(s) != spr_Nighttime
   {
   layer_sprite_change(s, spr_nighttime);
   }

Der obige Code überprüft den Sprite-Index des Elements mit der in der globalen Variablen gespeicherten ID global.Asset_sprite und wenn es nicht "spr_Nacht" ist, dann wird das Sprite dem Element zugewiesen.
