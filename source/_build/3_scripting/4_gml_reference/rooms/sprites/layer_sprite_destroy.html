layer_sprite_destroy


Beschreibung

Diese Funktion wird das angegebene Sprite-Element zerstören. Sie liefern die Sprite-ID (die Sie erhalten, wenn Sie das Sprite mit erstellen layer_sprite_create() oder wenn Sie die Layer-ID zusammen mit verwenden layer_get_sprite_id() ) und das wird es entfernen. Beachten Sie, dass die Ebene nicht entfernt wird, sondern nur das Sprite. Wenn es sich um ein Sprite handelt, das im Raumeditor hinzugefügt wurde, wird das Sprite beim nächsten Verlassen des Raums und der Rückkehr erneut erstellt. Wenn der Raum jedoch persistent ist, wird das Sprite entfernt, sofern die Raumpersistenz nicht wieder ausgeschaltet wird.


Syntax:

layer_sprite_destroy(sprite_element_id)


Argumente:

Streit Beschreibung
sprite_element_id Der eindeutige ID-Wert des Sprites, das zerstört werden soll


Kehrt zurück:

N/A


Beispiel:

var lay_id = layer_get_id("Asset_Trees");
if layer_sprite_exists(lay_id, global.Asset_spr_trees)
   {
   layer_sprite_destroy(lay_id);
   }

Der obige Code überprüft den Layer "Asset_Trees", um zu sehen, ob das gegebene Sprite-Element existiert und falls dies der Fall ist, wird es zerstört (aber nicht der Layer).
