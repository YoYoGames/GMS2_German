layer_sprite_get_id


Beschreibung

Diese Funktion kann verwendet werden, um den eindeutigen ID-Wert des Sprite-Elements auf einer Ebene abzurufen. Sie geben die Layer-ID an (die Sie erhalten, wenn Sie die Ebene mit erstellen layer_create() oder wenn Sie den Ebenennamen zusammen mit verwenden layer_get_id() ) und den Namen des Assets, wie er in den Eigenschaften des IDE-Assets definiert ist. Die Funktion gibt den ID-Wert zurück, der diesem Sprite-Element auf dem Layer zugeordnet ist. Beachten Sie, dass diese Funktion speziell für Sprites entwickelt wurde, die in der IDE auf einer Asset-Ebene hinzugefügt wurden, als ob Sie ein Sprite über die Funktion mithilfe von Code zu einer Ebene hinzufügen würden layer_sprite_create, dann wird die eindeutige ID für das hinzugefügte Sprite-Element zurückgegeben (und das neue Element hat keinen Namen).


Syntax:

layer_sprite_get_id(layer_id, asset_name)


Argumente:

Streit Beschreibung
layer_id Der eindeutige ID-Wert der zu targetenden Ebene
asset_name Der eindeutige Name des Assets auf der Ebene, wie im Raumeditor definiert


Kehrt zurück:

Sprite element ID value (Real)


Beispiel:

var lay_id = layer_get_id("Assets_trees");
var back_id = layer_sprite_get_id(lay_id, "gfc_Trees");
layer_sprite_change(back_id, spr_Trees_Winter);

Der obige Code ruft die Layer-ID für den Layer "Assets_trees" ab und verwendet diese, um die ID des Sprite-Elements "gfc_Trees" auf diesem Layer zu erhalten. Diese ID wird dann verwendet, um das Element Sprite zu ändern.
