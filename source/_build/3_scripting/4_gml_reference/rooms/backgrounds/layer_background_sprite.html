layer_background_sprite


Beschreibung

Mit dieser Funktion können Sie den Sprite-Index des Hintergrundelements festlegen. Sie liefern die Hintergrundelement-ID (die Sie erhalten, wenn Sie ein Hintergrundelement mit erstellen layer_background_create() oder wenn Sie die Funktion verwenden layer_background_get_id() ) und geben Sie dann einen zu verwendenden Sprite-Index an. Das Hintergrundelementbild wird durch das neue Sprite ersetzt. Wenn Sie einen Wert von -1 angeben, wird dem Element kein Sprite zugewiesen (es ist jedoch weiterhin vorhanden und kann später wieder einem Sprite zugewiesen werden).

Hinweis: Diese Funktion ersetzt das veraltete layer_background_change().


Syntax:

layer_background_sprite(background_element_id, sprite_index)


Argumente:

Streit Beschreibung
background_element_id Der eindeutige ID-Wert des zu ändernden Hintergrundelements
sprite_index Der Sprite-Index des Sprites, das für das Hintergrundelement verwendet werden soll


Kehrt zurück:

N/A


Beispiel:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_sprite(back_id) != spr_Clouds
   {
   layer_background_sprite(back_id, spr_Clouds);
   }

Der obige Code ruft die Layer-ID für den Layer "Background_sky" ab und verwendet diese, um die ID des Hintergrundelements auf diesem Layer zu erhalten. Diese ID wird dann verwendet, um das dem Element zugewiesene Sprite zu überprüfen und es auf das Sprite "spr_Clouds" zu setzen, falls dies noch nicht geschehen ist.
