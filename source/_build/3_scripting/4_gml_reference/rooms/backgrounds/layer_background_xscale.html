layer_background_xscale


Beschreibung

Mit dieser Funktion kann die Skalierung entlang der X-Achse eines Hintergrundelements festgelegt werden. Sie geben die Hintergrundelement-ID an (die Sie erhalten, wenn Sie ein Hintergrundelement mit erstellen layer_background_create() oder wenn Sie die Funktion verwenden layer_background_get_id() ) und stellen Sie dann die Animationsgeschwindigkeit ein. Der Skalierungswert, den Sie angeben, ist ein Multiplikator, der verwendet wird, um die Art und Weise zu ändern, wie das Hintergrundelement angezeigt wird, wobei ein Wert von 0,5 das Element im Halbmaßstab und ein Wert von 2 im Doppelskala anzeigt. Beachten Sie, dass negative Werte gültig sind und das Element um die Position (0,0) "spiegeln", sodass eine x-Skalierung von -1 das Bild umgekehrt zeigt.


Syntax:

layer_background_xscale(background_element_id, xscale);


Argumente:

Streit Beschreibung
background_element_id Der eindeutige ID-Wert des zu ändernden Hintergrundelements
xscale Der zu verwendende Maßstabswert (1 ist keine Skalierung)


Kehrt zurück:

N/A


Beispiel:

var lay_id = layer_get_id("Background_water");
var back_id = layer_background_get_id(lay_id);
layer_background_xscale(back_id, -1);

Der obige Code ruft die Layer-ID für den Layer "Background_water" ab und verwendet diese, um die ID des Hintergrundelements auf diesem Layer zu erhalten. Diese ID wird dann verwendet, um das Element xscale zu ändern.
