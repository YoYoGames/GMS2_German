layer_background_create


Beschreibung

Mit dieser Funktion können Sie einer Ebene eine Sprite-Ressource zuweisen, die als Hintergrund in Ihrem Projekt verwendet werden soll. Sie geben die Layer-ID an (die Sie erhalten, wenn Sie die Ebene mit erstellen layer_create() ) oder der Name des Layers (als String - dies wirkt sich auf die Performance aus) und ein Sprite-Index (der Name des Sprites wird im Ressourcenbaum angezeigt) und wird dem Layer hinzugefügt. Die Funktion gibt den eindeutigen ID-Wert für das Element zurück, der dann in weiteren Layer-Funktionen für Hintergründe verwendet werden kann.


Syntax:

layer_background_create(layer_id, sprite)


Argumente:

Streit Beschreibung
layer_id Der eindeutige ID-Wert des zu targetierenden Layers (oder der Layername als String)
sprite Der zu verwendende Sprite-Index


Kehrt zurück:

Background element ID (Real)


Beispiel:

global.back_layer = layer_create(10000);
global.back_trees = layer_background_create(global.back_layer, spr_Trees);

Der obige Code erstellt eine neue Ebene und fügt dann ein neues Hintergrundelement hinzu, wobei ein Sprite als Hintergrundbild verwendet wird.
