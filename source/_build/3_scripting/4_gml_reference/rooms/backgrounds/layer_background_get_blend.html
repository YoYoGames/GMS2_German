layer_background_get_blend


Beschreibung

Diese Funktion kann verwendet werden, um die Mischfarbe des Hintergrundelements zu erhalten. Sie geben die Hintergrundelement-ID an (die Sie erhalten, wenn Sie ein Hintergrundelement mit erstellen layer_background_create() oder wenn Sie die Funktion verwenden layer_background_get_id() ), und die Funktion gibt einen reellen Wert zurück, der die zugewiesene Farbe darstellt.


Syntax:

layer_background_get_blend(background_element_id);


Argumente:

Streit Beschreibung
background_element_id Der eindeutige ID-Wert des Hintergrundelements, von dem die Informationen abgerufen werden


Kehrt zurück:

Colour Value (Real)


Beispiel:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_blend(back_id) == c_white
   {
   layer_background_blend(back_id, make_colour_rgb(random(255), random(255), 255));
   }

Der obige Code ruft die Layer-ID für den Layer "Background_sky" ab und verwendet diese, um die ID des Hintergrundelements auf diesem Layer zu erhalten. Diese ID wird dann verwendet, um die Elementmischungsfarbe zu prüfen und wenn sie der Konstante äquivalent ist c_white, dann wird die Ebenenmischung auf eine zufällige Farbe gesetzt.
