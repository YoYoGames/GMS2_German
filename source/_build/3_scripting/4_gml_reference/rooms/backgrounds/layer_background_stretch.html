layer_background_stretch


Beschreibung

Mit dieser Funktion können Sie das Sprite eines Hintergrundelements so umstellen, dass es in den Raum passt oder mit der Auflösung 1: 1 bleibt. Sie liefern die Hintergrundelement-ID (die Sie erhalten, wenn Sie ein Hintergrundelement mit erstellen layer_background_create() oder wenn Sie die Funktion verwenden layer_background_get_id() ) und legen Sie dann das stretch-Argument fest true oder false. Wenn auf eingestellt true Das Element Sprite wird so gestreckt, dass es in den Raum passt (entweder vergrößert oder verkleinert, abhängig von der Größe des Sprites und des Raums), und wenn es auf eingestellt ist false, wird das Element Sprite mit seiner nativen Auflösung angezeigt.


Syntax:

layer_background_stretch(background_element_id, stretch)


Argumente:

Streit Beschreibung
background_element_id Der eindeutige ID-Wert des zu ändernden Hintergrundelements
stretch Der Stretch-Knebel, der sein kann true oder false


Kehrt zurück:

N/A


Beispiel:

if mouse_check_button_pressed(mb_left)
   {
   var back = layer_background_get_id(layer);
   if layer_background_get_stretch(back)
      {
      layer_background_stretch(back, false);
      }
   else
      {
      layer_background_stretch(back, true);
      }
   }

Der obige Code prüft, ob eine Maustaste gedrückt wird, und wenn eine erkannt wird, wird die Streckung des Sprites des Hintergrundelements umgeschaltet, das der Ebene zugewiesen ist, auf der die Instanz ausgeführt wird, auf der der Code ausgeführt wird.
