layer_background_htiled


Beschreibung

Mit dieser Funktion können Sie ändern, ob das angegebene Hintergrundelement auf einer Ebene horizontal gekachelt werden soll oder nicht. Sie geben die Hintergrundelement-ID an (die Sie erhalten, wenn Sie ein Hintergrundelement mit erstellen layer_background_create() oder wenn Sie die Funktion verwenden layer_background_get_id() ), und legen Sie dann den Tiling-Wert fest. Wenn auf eingestellt true dann wird das Element horizontal über den Raum verteilt angezeigt, und wenn festgelegt auf false, das Element wird nicht gekachelt.


Syntax:

layer_background_htiled(background_element_id, tiled);


Argumente:

Streit Beschreibung
background_element_id Der eindeutige ID-Wert des zu ändernden Hintergrundelements
visible Der horizontale Fliesenschalter, der sein kann true oder false


Kehrt zurück:

N/A


Beispiel:

var back = layer_background_get_id(layer);
if !layer_background_get_htiled(back) || !layer_background_get_vtiled(back)
   {
   layer_background_htiled(back, true);
   layer_background_vtiled(back, true);
   }

Der obige Code prüft das Hintergrundelement, das dem Layer zugeordnet ist, auf dem die Instanz läuft, auf der der Code läuft, und wenn es nicht in beide Richtungen gekachelt ist, wird es so eingestellt, dass es vertikal und horizontal gekachelt wird.
