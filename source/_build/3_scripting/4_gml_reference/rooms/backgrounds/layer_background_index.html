layer_background_index


Beschreibung

Mit dieser Funktion kann der Bildindex des Hintergrund-Sprites mit mehreren Teilbildern eingestellt werden. Sie geben die Hintergrundelement-ID an (die Sie erhalten, wenn Sie ein Hintergrundelement mit erstellen layer_background_create() oder wenn Sie die Funktion verwenden layer_background_get_id() ) und legen Sie dann den zu verwendenden Bildindex fest. Wenn Sie einen Wert außerhalb des Bereichs von Teilbildern festlegen, wird der Bildindex umlaufen.


Syntax:

layer_background_index(background_element_id, image_index);


Argumente:

Streit Beschreibung
background_element_id Der eindeutige ID-Wert des zu ändernden Hintergrundelements
index Der Bildindex, der für den Hintergrund verwendet werden soll


Kehrt zurück:

N/A


Beispiel:

var lay_id = layer_get_id("Background_trees");
var back_id = layer_background_get_id(lay_id);
layer_background_index(back_id, 1);

Der obige Code erhält die Layer-ID für den Layer "Background_trees" und verwendet diese, um die ID des Hintergrundelements auf diesem Layer zu erhalten. Diese ID wird dann verwendet, um den Elementbildindex zu ändern.
