layer_background_alpha


Beschreibung

Diese Funktion steuert die Alpha (Transparenz) des Hintergrund-Sprites. Sie geben die Hintergrundelement-ID an (die Sie erhalten, wenn Sie ein Hintergrundelement mit erstellen layer_background_create() oder wenn Sie die Funktion verwenden layer_background_get_id() ) und legen Sie den zu verwendenden Alpha-Wert fest. Alpha kann zwischen 0 (vollständig transparent) und 1 (vollständig undurchsichtig) mit dem Standard-Alpha-Wert für das Hintergrundelement 1 sein. Beachten Sie, dass wenn die Ebene, der das Hintergrundelement zugewiesen wurde, nicht sichtbar ist - oder das Element selbst erstellt wurde unsichtbar - Sie werden keinen Unterschied zu dieser Funktion sehen, bis die Ebene oder das Element wieder sichtbar gemacht wurde.


Syntax:

layer_background_alpha(background_element_id, alpha);


Argumente:

Streit Beschreibung
background_element_id Der eindeutige ID-Wert des zu ändernden Hintergrundelements
alpha Das Alpha für Hintergrundsprite von 0 bis 1 (Standard ist 1)


Kehrt zurück:

N/A


Beispiel:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
layer_background_alpha(back_id, random(1));

Der obige Code erhält den ID-Wert des Hintergrunds, der dem Layer "Background_sky" zugewiesen ist, und setzt dann sein Alpha auf einen zufälligen Wert zwischen 0 und 1.
