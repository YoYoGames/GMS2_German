layer_force_draw_depth


Beschreibung

Diese Funktion erzwingt das Zeichnen aller Ebenen in der angegebenen Z-Tiefe. Dies ändert nicht die Reihenfolge, in der die Layer gerendert werden, und sie werden immer noch in der Reihenfolge der Tiefe gezeichnet. Es ändert sich einfach, welcher z-Wert verwendet wird. Im Allgemeinen müssen Sie sich darüber keine Gedanken machen, aber wenn Sie Ebenen mit einer Tiefe außerhalb des zulässigen Bereichs (-16000 bis 16000) haben, werden sie nicht gerendert, sodass Sie die Z-Tiefe auf einen vernünftigen Wert bringen können - 0 zum Beispiel - und sie werden alle gut gemacht. Beachten Sie, dass dies in der Regel nur für Legacy-Projekte aus früheren Versionen von GameMaker denen Sie höhere oder niedrigere GameMaker als den zulässigen Ebenenbereich haben könnten.


Syntax:

layer_force_draw_depth(force, depth)


Argumente:

Streit Beschreibung
force Ob zu aktivieren (auf true oder deaktivieren (auf false ) Z Tiefenkraft
depth Die neue Z-Tiefe


Kehrt zurück:

N/A


Beispiel:

if !layer_is_draw_depth_forced()
   {
   layer_force_draw_depth(true, 0);
   }

Der obige Code überprüft, ob die Tiefe der Ebene Z erzwungen wird oder nicht, und wenn dies nicht der Fall ist, setzt er die Tiefe Z auf 0 und aktiviert sie.
