layer_get_all_elements


Beschreibung

Sie können diese Funktion verwenden, um die Element-IDs des angegebenen Layers abzurufen. Sie geben die Layer-ID an (die Sie erhalten, wenn Sie die Ebene mit erstellen layer_create() ) oder der Layername (als String - dies wirkt sich auf die Leistung aus) und die Funktion gibt ein Array von IDs zurück, wobei jeder Eintrag im Array eine eindeutige ID für ein Element auf diesem Layer darstellt. Wenn es sich bei dem Layer beispielsweise um einen Asset-Layer handelt, wird das Array mit den ID-Werten für jedes Sprite-Asset gefüllt, das dem Layer zugewiesen ist. Beachten Sie, dass die Verwendung von Code für die Arbeit mit Ebenen bedeutet, dass Sie der gleichen Ebene verschiedene Elementtypen zuweisen können. So können Sie beispielsweise Sprite-Assets zusammen mit Instanzen verwenden. In diesem Fall können Sie dann die Funktion verwenden layer_get_element_type um den Typ des Elements zu erhalten, auf das sich die ID bezieht.


Syntax:

layer_get_all_elements(layer_id)


Argumente:

Streit Beschreibung
layer_id Der eindeutige ID-Wert des Layers, von dem die Elemente abgerufen werden sollen (oder der Layername als String)


Kehrt zurück:

Array (1D, populated with Element IDs)


Beispiel:

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length_1d(a); i++;)
   {
   if layer_get_element_type(a[i]) == layerelementtype_sprite
      {
      layer_sprite_destroy(a[i])
      }
   }

Der obige Code ruft die IDs für alle Instanzelemente ab, die dem Layer der Instanz zugewiesen sind, auf der der Code ausgeführt wird. Der Code überprüft dann, ob einige der zurückgegebenen Elemente Sprite-Assets sind und ob sie dann zerstört werden.
