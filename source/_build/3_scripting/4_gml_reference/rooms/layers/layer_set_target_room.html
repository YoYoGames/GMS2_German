layer_set_target_room


Beschreibung

Wenn Sie diese Funktion aufrufen, weisen Sie GameMaker Studio 2 an, dass alle weiteren Layer-Funktionen auf den gegebenen Raum angewendet werden sollen. Auf diese Weise können Sie Ebenen und Ebeneninhalte in einem Raum, der nicht der aktuelle Raum ist, verfahrensmäßig ändern oder generieren. Wenn Sie mit dem Hinzufügen von Ebenen oder Ebenenelementen zu einem Raum fertig sind, rufen Sie die Funktion auf layer_reset_target_room um das Raumziel zurückzusetzen (oder rufen Sie diese Funktion erneut mit einem Raumargument von -1 auf).

WARNUNG! Bei der Ausrichtung auf einen anderen Raum können Sie alle regulären Ebenenfunktionen verwenden, außer dass Sie keine Instanzen mit erstellen können instance_create_layer oder instance_create_depth, noch wird die Schicht funktionieren layer_add_instance verfügbar sein.


Syntax:

layer_set_target_room(room)


Argumente:

Streit Beschreibung
room Der Raum für alle weiteren Layer-Funktionen


Kehrt zurück:

N/A


Beispiel:

layer_set_target_room(rm_Game);
var l = layer_get_id("SpriteAssets");
repeat(50)
   {
   layer_sprite_create(l, random(room_get_width(rm_Game)), random(room_get_height(rm_Game)), spr_Trees);
   }
layer_reset_target_room();

Der obige Code setzt den Zielraum auf den Raum "rm_Game" und erhält dann die Layer-ID für den Layer "SpriteAssets" in diesem Raum. Diese Layer-ID wird dann verwendet, um 50 zufällige Sprite-Assets zur Ebene hinzuzufügen, bevor das Layer-Ziel auf den aktuellen Raum zurückgesetzt wird.
