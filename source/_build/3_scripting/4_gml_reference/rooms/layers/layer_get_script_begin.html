layer_get_script_begin


Beschreibung

Sie geben die Layer-ID an (die Sie erhalten, wenn Sie die Ebene mit erstellen layer_create() ) oder der Layername (als String - dies wirkt sich auf die Leistung aus) und diese Funktion gibt den Skriptindex des Skripts zurück, das am Anfang des Renderings für diesen Layer ausgeführt wird, oder gibt -1 zurück, wenn kein Skript vorhanden ist zugewiesen.


Syntax:

layer_get_script_begin(layer_id);


Argumente:

Streit Beschreibung
layer_id Der eindeutige ID-Wert des zu targetierenden Layers (oder der Layername als String)


Kehrt zurück:

Script Index (Real) or -1 if no script is assigned


Beispiel:

if layer_get_script_begin(layer) == -1
   {
   layer_script_begin(layer, scr_SetShaderValues);
   }

Der obige Code prüft, ob dem Layer, dem die Instanz, auf der der Code ausgeführt wird, ein Skript zugewiesen ist, und ob ihm ein Skript zugewiesen ist.
