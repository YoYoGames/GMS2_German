layer_element_move


Beschreibung

Sie können diese Funktion verwenden, um ein Element von einem Layer in einen anderen zu verschieben. Sie geben die Element-ID an, die von der Funktion zurückgegeben wurde, mit der das Element oder der Raumeditor oder die Funktion erstellt wurde layer_get_all_elements und geben Sie die Layer-ID ein (die Sie beim Erstellen der Ebene mit layer_create() ) oder der Layername (als String - dies wirkt sich auf die Leistung aus).


Syntax:

layer_element_move(element_id, layer_id)


Argumente:

Streit Beschreibung
element_id Der eindeutige ID-Wert des zu verschiebenden Elements
layer_id Der eindeutige ID-Wert des Layers, in den das Element verschoben werden soll (oder der Layername als String)


Kehrt zurück:

N/A


Beispiel:

var a = layer_get_all_elements(layer);
asset_layer = layer_create(-100);
for (var i = 0; i Alt; array_length_1d(a); i++;)
   {
   if layer_get_element_type(a[i]) == layerelementtype_sprite
      {
      layer_element_move(a[i], asset_layer)
      }
   }

Der obige Code ruft die Elemente auf dem Layer ab, denen die Instanz zugewiesen ist, der der Code zugewiesen ist, und überprüft sie dann, ob es sich um Sprite-Assets handelt. Wenn sie dies tun, werden sie auf den Layer mit der ID in der Variable "asset_layer" verschoben ".
