layer_get_id


Beschreibung

Diese Funktion kann verwendet werden, um den eindeutigen ID-Wert für eine bestimmte Ebene zu erhalten. In der IDE haben alle Layer einen Namen und einen Typ, und um sie bearbeiten oder durch Code ändern zu können, müssen Sie den Layer-ID- Wert angeben. Diese Funktion wird verwendet, um diese ID mithilfe des Namens (einer Zeichenfolge) des Layers (wie in der IDE geschrieben) abzurufen. Wenn Sie mithilfe der Funktion eine neue Ebene mithilfe von Code erstellen layer_create() dann gibt diese Funktion stattdessen den eindeutigen ID-Wert zurück (dynamisch erstellte Layer erhalten keine Namen). Wenn Sie den Namen eines Layers angeben, der im aktuellen Raum nicht vorhanden ist, erhalten Sie einen Fehler, und das Projekt stürzt ab.


Syntax:

layer_get_id(layer_name)


Argumente:

Streit Beschreibung
layer_name Der Name der Ebene (eine Zeichenfolge)


Kehrt zurück:

Layer ID Value (Real), or -1 if the layer specified doesn't exist


Beispiel:

var near = instance_nearest(x, y, obj_Tree);
var layer_id = layer_get_id("Instances Front");
layer_add_instance(layer_id, near);

Der obige Code wird zuerst den Index der nächsten Instanz zu der gegebenen x / y-Position bringen und ihn in einer lokalen Variablen speichern. Er erhält dann die eindeutige Instanz-Layer-ID für den Layer "Instanzen Front" und verschiebt die gefundene Instanz auf diesen Layer.
