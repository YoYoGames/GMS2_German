layer_get_y


Beschreibung

Sie können diese Funktion verwenden, um die y-Position der Ebene innerhalb des aktuellen Bereichs abzurufen. Sie geben die Layer-ID an (die Sie erhalten, wenn Sie die Ebene mit erstellen layer_create() ) oder der Layername (als String - dies wirkt sich auf die Leistung aus) und die Funktion gibt eine reelle Zahl für die x-Position der Ebene relativ zur (0,0) Position des Raums zurück. Standard ist 0.


Syntax:

layer_get_y(layer_id)


Argumente:

Streit Beschreibung
layer_id Der eindeutige ID-Wert des Layers, um die y-Position von zu erhalten


Kehrt zurück:

Real


Beispiel:

var lay_id = layer_get_id("Sprites");
if layer_get_x(lay_id) != 0 || layer_get_x(lay_id) != 0
    {
    layer_x(lay_id, 0);
    layer_y(lay_id, 0);
    }

Der obige Code prüft die gegebene Layer-Position und wenn er nicht auf (0, 0) gesetzt ist, wird er auf diese Position gesetzt.
