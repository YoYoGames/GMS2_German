layer_get_depth


Beschreibung

Sie können diese Funktion verwenden, um den Tiefenwert zu erhalten, der einem bestimmten Layer zugeordnet ist. Sie geben die Layer-ID an (die Sie erhalten, wenn Sie die Ebene mit erstellen layer_create() ) oder der Layername (als String - dies wirkt sich auf die Leistung aus) und die Funktion gibt die Tiefe dieses Layers als reelle Zahl zurück. Beachten Sie, dass die Tiefe als "weiter weg" von der Kamera definiert ist und den "näheren" Abstand zur Kamera verringert. Wenn Sie also drei Ebenen mit den Tiefen -100, 0, 100 haben, zeichnen die Ebenen in der Reihenfolge 100, 0, -100, so dass die "oberste" Ebene (dh die am nächsten zur Kameraansicht und so über alles andere gezeichnet) die Ebene mit der -100 Tiefe ist.

Das folgende Bild zeigt eine schematische Darstellung, wie die Tiefe für Ebenen funktioniert:

Beachten Sie, dass Sie einen Fehler erhalten, wenn Sie einen ungültigen Layer-ID-Wert angeben.


Syntax:

layer_get_depth(layer_id)


Argumente:

Streit Beschreibung
layer_id Der eindeutige ID-Wert des Layers, um die Tiefe von (oder den Layernamen als String) zu erhalten


Kehrt zurück:

Real


Beispiel:

if layer_get_depth(global.temp_layer) != -100
   {
   layer_destroy(global.temp_layer);
   global.temp_layer = layer_create(-100);
   }

Der obige Code prüft die Tiefe einer Layer-ID, die in einer globalen Variablen gespeichert ist, und wenn sie nicht -100 ist, zerstört sie den Layer und erstellt ihn in der Tiefe von -100 neu.
