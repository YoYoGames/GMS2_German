layer_get_depth


Beschreibung

Mit dieser Funktion können Sie den Tiefenwert abrufen, der einer bestimmten Ebene zugeordnet ist. Sie geben die Layer-ID an, die Sie erhalten, wenn Sie den Layer mit erstellen layer_create() ) oder der Ebenenname (als String - dies hat Auswirkungen auf die Leistung) und die Funktion gibt die Ebenentiefe als reelle Zahl zurück. Beachten Sie, dass die Tiefe definiert ist, je höher der Abstand von der Kamera und der Abstand zur Kamera ist. Wenn Sie also drei Ebenen mit den Tiefen -100, 0, 100 haben, werden die Ebenen in der Reihenfolge 100 gezeichnet. 0, -100, so dass die "oberste" Ebene (dh die nächstliegende Ansicht der Kameraansicht und somit alles andere) die Ebene mit der -100 Tiefe ist.

Das folgende Bild zeigt schematisch, wie die Tiefe für Layer funktioniert:

Wenn Sie einen ungültigen Layer-ID-Wert angeben, wird ein Fehler angezeigt.


Syntax:

layer_get_depth(layer_id)


Argumente:

Streit Beschreibung
layer_id Der eindeutige ID-Wert der Ebene, deren Tiefe abgerufen werden soll (oder der Ebenenname als Zeichenfolge).


Kehrt zurück:

Real


Beispiel:

if layer_get_depth(global.temp_layer) != -100
   {
   layer_destroy(global.temp_layer);
   global.temp_layer = layer_create(-100);
   }

Der obige Code prüft die Tiefe einer in einer globalen Variablen gespeicherten Layer-ID. Wenn sie nicht -100 ist, wird der Layer zerstört und in der Tiefe von -100 neu erstellt.
