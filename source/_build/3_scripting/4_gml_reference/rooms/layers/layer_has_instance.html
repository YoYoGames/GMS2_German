layer_has_instance


Beschreibung

Diese Funktion kann verwendet werden, um zu überprüfen, ob eine bestimmte Instanz der gegebenen Ebene aktuell zugewiesen ist. Sie geben die Layer-ID an (die Sie erhalten, wenn Sie die Ebene mit erstellen layer_create() ) oder der Name des Layers (als String - dies wirkt sich auf die Leistung aus) und die Instanz-ID der Instanz, nach der gesucht werden soll. Sie können auch eine geben object_index (dh: der Name des Objekts in der Ressourcenstruktur) und die Funktion überprüft, ob sich Instanzen dieses Objekts auf der gegebenen Ebene befinden. Die Funktion wird zurückkehren true Wenn sich die Instanz in der Ebene befindet und false wenn es das nicht ist.


Syntax:

layer_has_instance(layer_id, instance_id)


Argumente:

Streit Beschreibung
layer_id Der eindeutige ID-Wert des zu targetierenden Layers (oder der Layername als String)
instance_id Die eindeutige Instanz-ID oder der Objektindex der Instanz, nach der gesucht werden soll


Kehrt zurück:

Boolean


Beispiel:

if !layer_has_instance(global.Bullet_Layer, obj_Bullet_Parent)
   {
   layer_destroy(global.Bullet_Layer);
   }

Der obige Code wird prüfen, ob der gegebene Layer Instanzen des Objekts "obj_Bullet_Parent" enthält und wenn nicht, wird der Layer zerstört.
