layer_get_element_type


Beschreibung

Sie können diese Funktion verwenden, um den Elementtyp für das angegebene Element abzurufen. Sie geben den eindeutigen Element-ID-Wert an (z. B. die Funktion, die das Element erstellt hat, oder den Raumeditor), und die Funktion gibt eine der folgenden Konstanten zurück (oder -1, wenn das Element nicht existiert oder der ID-Wert ist) fehlerhaft):

Konstante Beschreibung
layerelementtype_background Das Element ist ein Hintergrund.
layerelementtype_instance Das Element ist eine Instanz.
layerelementtype_sprite Das Element ist ein Sprite-Asset.
layerelementtype_tilemap Das Element ist eine Kachelkarte.
layerelementtype_particlesystem Das Element ist ein Partikelsystem.
layerelementtype_tile Das Element ist eine ältere Hintergrundkachel (dies gilt nur für Projekte, die aus früheren Versionen von GameMaker importiert wurden).

Beachten Sie, dass diese Funktion am nützlichsten ist, wenn Sie der einen Ebene mehrere verschiedene Elementtypen zugewiesen haben und mit dieser Funktion eine Liste aller Elemente auf einer bestimmten Ebene abrufen können layer_get_all_elements.


Syntax:

layer_get_element_type(element_id)


Argumente:

Streit Beschreibung
layer Der eindeutige ID-Wert des Elements, von dem der Typ abgerufen werden soll


Kehrt zurück:

Constant (see table, above) or -1 if element does not exist or is invalid


Beispiel:

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length_1d(a); i++;)
   {
   if layer_get_element_type(a[i]) == layerelementtype_sprite
      {
      layer_sprite_destroy(a[i])
      }
   }

Der obige Code ruft die IDs für alle Instanzelemente ab, die dem Layer der Instanz zugewiesen sind, auf der der Code ausgeführt wird. Der Code überprüft dann, ob einige der zurückgegebenen Elemente Sprite-Assets sind und ob sie dann zerstört werden.
