layer_get_name


Beschreibung

Mit dieser Funktion können Sie den Namen der angegebenen Ebene abrufen. Sie geben den eindeutigen Layer-ID-Wert an. Wenn es sich bei dem Layer um einen der benannten Layer handelt, die im Raumeditor erstellt wurden, gibt die Funktion eine Zeichenfolge mit dem Layernamen zurück. Wenn der Layer nicht zum Raumeditor gehört (dh: er wurde mit erstellt layer_create ) dann wird ein leerer String zurückgegeben.


Syntax:

layer_get_name(layer_id)


Argumente:

Streit Beschreibung
layer_id Der eindeutige ID-Wert des Layers, dessen Name abgerufen werden soll


Kehrt zurück:

String


Beispiel:

var a = layer_get_all();
var layer_list = ds_list_create(); for (var i = 0; i Alt; array_length_1d(a); i++;)
   {
   if layer_get_name(a[i]) != ""
      {
      ds_list_add(layer_list, a[i])
      }
   }

Der obige Code ruft die IDs für alle Ebenen im Raum ab und durchläuft dann diese, um zu überprüfen, ob es sich um benannte Ebenen handelt. Wenn dies der Fall ist, wird ihre ID einer Liste hinzugefügt.
