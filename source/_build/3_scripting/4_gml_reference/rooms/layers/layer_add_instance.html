layer_add_instance


Beschreibung

Diese Funktion kann verwendet werden, um eine bestimmte Instanz von der aktuellen Ebene auf eine andere Ebene zu verschieben. Sie geben die Layer-ID an (die Sie erhalten, wenn Sie die Ebene mit erstellen layer_create() ) oder der Layername (als String - dies wirkt sich auf die Leistung aus) und die Instanz-ID der Instanz, die zwischen den Layern verschoben werden soll. Angenommen, Ihr Player befindet sich auf einer Ebene, die eine geringere Tiefe als eine andere Ebene aufweist, und Sie möchten, dass sie hinter den Ebenen zwischen den beiden Ebenen angezeigt wird. Sie können diese Funktion verwenden, um den Player von der Vordergrundebene zur Hintergrundebene zu verschieben und ihn hinter den anderen Ebenen anzuzeigen, die gezeichnet werden.

HINWEIS: Diese Funktion kann nicht zum Hinzufügen einer neuen Instanz zu einem Layer verwendet werden. Sie müssen die Instanz zuvor erstellt und ihre ID in einer Variablen gespeichert haben.


Syntax:

layer_add_instance


Argumente:

Streit Beschreibung
layer_id Der eindeutige ID-Wert des Instanz-Layers zum Targeting (oder der Layername als String)
instance_id Der eindeutige Instanz-ID-Wert der Instanz, die zur Zielebene verschoben werden soll


Kehrt zurück:

N/A


Beispiel:

var near = instance_nearest(x, y, obj_Tree);
var layer_id = layer_get_id("Instances Front");
layer_add_instance(layer_id, near);

Der obige Code wird zuerst den Index der nächsten Instanz zu der gegebenen x / y-Position bringen und ihn in einer lokalen Variablen speichern. Er erhält dann die eindeutige Instanz-Layer-ID für den Layer "Instanzen Front" und verschiebt die gefundene Instanz auf diesen Layer.
