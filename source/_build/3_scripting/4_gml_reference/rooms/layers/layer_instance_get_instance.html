layer_instance_get_instance


Beschreibung

Mit dieser Funktion kann die eindeutige Instanz-ID des angegebenen Instanzelements abgerufen werden. Sie geben die ID - Instanz Element (siehe Codebeispiel unten, wie diese zu erhalten), und die Funktion einen echten Wert zurück, der die einzigartige darstellt Instanz - ID für das Element. Wenn das Element keine Instanz ist, gibt die Funktion -1 zurück.


Syntax:

layer_instance_get_instance(element_id)


Argumente:

Streit Beschreibung
element_id Der eindeutige ID-Wert des Instanzelements, dessen ID abgerufen werden soll


Kehrt zurück:

Real (Unique ID value for an instance)


Beispiel:

elements = layer_get_all_elements("Instances");
for (var i = 0; i < array_length_1d(elements); i++)
    {
     if (layer_get_element_type(elements[i]) == layerelementtype_instance)
        {
        var layerelement = elements[i];
        var inst = layer_instance_get_instance(layerelement);
        inst.x = inst.x + 10;
        }
    }

Der obige Code prüft, ob alle Instanzelemente auf einer Ebene abgerufen werden. Dann wird der eindeutige ID-Wert abgerufen und mit diesem Wert um 10 Pixel nach rechts verschoben.
