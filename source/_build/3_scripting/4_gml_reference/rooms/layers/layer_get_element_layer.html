layer_get_element_layer


Beschreibung

Sie können diese Funktion verwenden, um die Layer-ID abzurufen, auf der sich das angegebene Element befindet. Sie geben den eindeutigen Element-ID-Wert an (z. B. wie die Funktion zurückgegeben wurde, mit der das Element erstellt wurde, oder den Raumeditor), und die Funktion gibt die eindeutige ID des Layers zurück, auf dem das Element gefunden wurde. Wenn die angegebene Element-ID nicht gültig ist, gibt die Funktion -1 zurück.


Syntax:

layer_get_element_layer(element_id)


Argumente:

Streit Beschreibung
element_id Der eindeutige ID-Wert des Elements, von dem die Ebene abgerufen werden soll


Kehrt zurück:

Layer ID or -1 if the element is invalid (ie: does not exist)


Beispiel:

element = layer_get_element_layer(asset_1);

Der obige Code ruft die Layer-ID für das Element mit der in der Variablen "asset_1" gespeicherten ID ab und speichert sie in einer Variablen.
