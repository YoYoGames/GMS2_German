layer_set_visible


Beschreibung

Mit dieser Funktion können Sie die Sichtbarkeit einer Ebene umschalten. Wenn eine Ebene unsichtbar ist, wird nichts, das der Ebene zugewiesen ist, gezeichnet, und wenn Instanzen der Ebene zugewiesen werden, wird ihr Draw-Ereignis nicht einmal ausgeführt (ähnlich wie beim Festlegen der Instanzvariablen) visible zu false ). Beachten Sie, dass alle Instanzen, die bereits als unsichtbar gekennzeichnet sind, nicht als sichtbar markiert werden, wenn die Ebene, auf der sie sich befinden, auf sichtbar gesetzt wird, da dies zwei unabhängige Einstellungen sind, obwohl ihr Verhalten das gleiche ist.


Syntax:

layer_set_visible(layer_id, visible)


Argumente:

Streit Beschreibung
layer_id Der eindeutige ID-Wert des zu targetierenden Layers (oder der Layername als String)
visible Ob die Schicht sichtbar sein soll ( true ) oder nicht ( false )


Kehrt zurück:

N/A


Beispiel:

var lay_id = layer_get_id("Instances");
if layer_get_visible(lay_id)
   {
   layer_set_visible(lay_id, false);
   }
else
   {
   layer_set_visible(lay_id, true);
   }

Der obige Code ruft den ID-Wert für den Layer "Instances" im Raumeditor ab und prüft anhand der ID, ob der Layer sichtbar ist oder nicht, und schaltet die Sichtbarkeit des Layers abhängig vom zurückgegebenen Wert um.
