layer_shader


Beschreibung

Mit dieser Funktion können Sie jeder Ebene eine Shader-Ressource zuweisen. Die Ebene wird dann mit diesem Shader gerendert. Sie geben entweder den eindeutigen ID-Wert des Layers oder den Namen des Layers (als String - dies wirkt sich auf die Leistung aus) zusammen mit der ID des zu verwendenden Shaders aus. Der Shader muss zuvor im Ressourcenbaum erstellt worden sein, und der Shader-Index (der Name der Shader-Ressource) wird dann an diese Funktion übergeben. Wenn der zugewiesenen Ebene Instanzen hinzugefügt wurden, wird der Shader auf alle Zeichnungsereignisse angewendet, die von der Instanz verwendet werden. Wenn die Instanz beispielsweise ein Draw GUI-Begin-Ereignis enthält, wird der Shader automatisch darauf angewendet. Der Shader wirkt sich auch auf alle anderen auf dieser Ebene gezeichneten Grafikelemente aus, wie etwa Sprite-Elemente oder Tilemaps. Beachten Sie, dass die Funktion nicht für Zeichnungs- oder Schrittereignisse, sondern nur zu Beginn des Raums im Raumerstellungscode oder im Ereignisereignis / Raumstartereignis einer Instanz aufgerufen werden soll.


Syntax:

layer_shader(layer_id, shader)


Argumente:

Streit Beschreibung
layer_id Der eindeutige ID-Wert der Ebene für das Ziel (oder der Ebenenname als Zeichenfolge)
shader Der Shader-Index, der der Ebene zugewiesen werden soll


Kehrt zurück:

N/A


Beispiel:

var lay_id = layer_get_id("Instances");
layer_shader(lay_id, shd_Sepia);

Der obige Code weist die Shader-Ressource "shd_Sepia" der angegebenen Ebene für alle Zeichnungen zu.
