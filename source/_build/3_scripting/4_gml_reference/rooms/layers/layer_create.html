layer_create


Beschreibung

Diese Funktion wird verwendet, um eine neue Ebene innerhalb des aktuellen Raums zu erstellen. Sie geben die Tiefe an, die die neue Ebene einnehmen soll, und die Funktion gibt den eindeutigen Layer-ID-Wert zurück, der dann für den Zugriff auf die Ebene in weiteren Funktionen verwendet werden kann. Sie können auch ein optionales "name" -Argument angeben, mit dem Sie der Ebene einen bestimmten Namen (als Zeichenfolge) zuweisen können. Anstatt die Ebenen-ID für den Zugriff auf die Ebene zu verwenden, können Sie stattdessen auch diese Namenszeichenfolge verwenden Beachten Sie, dass die Verwendung von Namenszeichenfolgen einen größeren Einfluss auf die Leistung hat, als wenn Sie nur den ID-Wert verwenden.

Wenn Sie einen Tiefenwert angeben, ist dieser Wert umso höher, je weiter Sie von der Kamera entfernt sind und je weiter Sie sich von der Kamera entfernt befinden. Wenn Sie also drei Ebenen mit den Tiefen -100, 0, 100, haben Ebenen werden in der Reihenfolge 100, 0, -100 gezeichnet, so dass die "oberste" Ebene (dh die Kameraansicht am nächsten und somit über alles andere gezeichnet) die Ebene mit der Tiefe -100 ist.

Das folgende Bild zeigt eine schematische Darstellung, wie die Tiefe für Ebenen funktioniert:

Beachten Sie, dass Sie zwar mehr als eine Ebene einer einzelnen Tiefe zuweisen können, aber Sie können nicht garantieren, dass Ebenen mit derselben Tiefe in einer bestimmten Reihenfolge gezeichnet werden. Daher ist es am besten, nur eine Ebene zuzuweisen zu irgendeiner gegebenen Tiefe. Beachten Sie auch, dass die minimale und maximale Schichttiefe zwischen -16000 und 16000 liegt und alles außerhalb dieser Tiefen nicht gerendert wird. Wenn Sie eine Tiefe außerhalb dieses Bereichs benötigen, müssen Sie die Funktion verwenden layer_force_draw_depth.


Syntax:

layer_create(depth, [name])


Argumente:

Streit Beschreibung
depth Die Tiefe, in der die Ebene erstellt werden soll.
name Der Name, der dem Layer zugewiesen wird (ein String - optional)


Kehrt zurück:

Layer ID (Real)


Beispiel:

global.Bullet_Layer = layer_create(-100);

Der obige Code erstellt einen neuen Layer mit einem Tiefenwert von -100 und weist seinen ID-Wert einer globalen Variablen zu.
