layer_tilemap_destroy

Beschreibung

Diese Funktion zerstört das angegebene Templemaplement. Sie liefern die Kachelkarten-ID (die Sie beim Erstellen der Kachelkarte mit erhalten layer_tilemap_create() oder wenn Sie die Layer-ID zusammen mit verwenden layer_get_tilemap_id() ) und das wird es entfernen. Beachten Sie, dass damit nicht die Ebene, sondern nur die Kachelkarte daraus entfernt wird. Wenn die Kachelkarte im Raumeditor hinzugefügt wurde, wird die Kachelkarte beim nächsten Verlassen des Raums erneut erstellt. Wenn der Raum jedoch persistent ist, wird die Kachelkarte entfernt, sofern die Raumpersistenz nicht wieder ausgeschaltet wird.


Syntax:

layer_tilemap_destroy(tilemap_element_id)


Argumente:

Streit Beschreibung
tilemap_element_id Der eindeutige ID-Wert der zu vernichtenden Kachelkarte


Kehrt zurück:

N/A


Beispiel:

var lay_id = layer_get_id("Tiles_trees");
var tile_id = layer_tilemap_get_id(lay_id);
if layer_tilemap_exists(lay_id, tile_id)
   {
   layer_tilemap_destroy(tile_id);
   }

Der obige Code überprüft den Layer "Tiles_trees", um zu sehen, ob das angegebene Tilemap-Element existiert und wenn dies der Fall ist, wird es zerstört (aber nicht der Layer).
