tilemap_get_at_pixel

Beschreibung

Mit dieser Funktion können Sie die Kacheldaten von einer Position (innerhalb des Raumes) des Tilemap-Elements abrufen. Sie geben die Element-ID des Tilemap-Elements an (die Sie beim Erstellen eines Tilemap-Elements mit erhalten layer_tilemap_create() oder wenn Sie die Funktion verwenden layer_tilemap_get_id() ) sowie die x- und y-Position im Raum, um die Kacheldaten zu erhalten, und die Funktion gibt die Kacheldaten "blob" zurück. Diese Daten sind im Wesentlichen eine Bitmaske, die den Kachelindex, die Flip / Rotate / Mirror-Booleans und alle Maskendaten enthält, die angewendet wurden (siehe tilemap_set_mask für Details), und der resultierende Datenwert kann dann in den Kachelfunktionen verwendet werden, um die Kacheleigenschaften zu ändern. Wenn Sie die Kacheldaten von einer bestimmten Kachelzelle abrufen müssen, sollten Sie die Funktion verwenden tilemap_get stattdessen.

WICHTIG! Wenn die Kacheln in der Kachelkarte unverändert geblieben sind (dh sie werden nicht gedreht oder gedreht usw.), dann ist der Rückgabewert der Kachel-Daten genau gleich dem Index der Kachel auf der Kachel. Sie können also "Kollisionsabbildungen" von Kacheln erstellen, indem Sie zum Beispiel im Tileset eine Kachel bei Index 1 verwenden. Dann können Sie mit dieser Funktion nach 1 oder 0 (einer leeren Kachel) suchen, um Kollisionen zu berechnen.


Syntax:

tilemap_get_at_pixel(tilemap_element_id, x, y);


Argumente:

Streit Beschreibung
tilemap_element_id Der eindeutige ID-Wert des Tilemap-Elements, von dem das tiledata-Element abgerufen werden soll
x Die Position entlang der X-Achse, um die Kacheldaten zu erhalten (in Raumpixeln)
y Die Position entlang der Y-Achse, um die Kacheldaten von (in Raumpixeln) zu erhalten


Kehrt zurück:

Real (-1 if there is an error)


Beispiel:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get_at_pixel(map_id, 64, 128);
data = tile_set_flip(data, true);
tilemap_set_at_pixel(map_id, data, 64, 128);

Der obige Code ruft die ID für die Kachel "Clouds" auf der Ebene "Tiles_Sky" ab und verwendet diese, um die Daten von der Kachel an Position (64, 128) zu erhalten. Diese Kacheldaten werden dann umgedreht, bevor sie erneut zum Setzen der Kachel auf der Kachelkarte verwendet werden.
