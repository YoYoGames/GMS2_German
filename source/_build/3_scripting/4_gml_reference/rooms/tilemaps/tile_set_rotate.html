tile_set_rotate

Beschreibung

Diese Funktion kann verwendet werden, um einen gegebenen Satz von Fliesen zu setzen, um die Kachel um 90 Grad zu drehen oder nicht. Sie geben die tiledata, die mit der Funktion abgerufen werden kann tilemap_get und legen Sie dann das rotate-Argument auf entweder fest true Wenn Sie möchten, dass die Kachel gedreht wird, oder false Wenn Sie möchten, dass die Kachel in ihrem standardmäßigen, nicht gedrehten Zustand ist. Die Funktion gibt einen modifizierten tiledata-Satz zurück, der dann mit dem tilemap_set Funktion.


Syntax:

tile_set_rotate(tiledata, rotate)


Argumente:

Streit Beschreibung
tiledata die tiledata zu setzen
rotate Einstellen true drehen und false um es so wie es ist zu verlassen


Kehrt zurück:

Real


Beispiel:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var bool = !tile_get_rotate(data);
data = tile_set_rotate(data, bool);
tilemap_set(map_id, data, mx, my);

Der obige Code ruft die Kachelkarten-ID von der gegebenen Ebene ab und erhält dann die X- und Y-Zellenposition für die Kachel unter der Maus. Diese Position wird dann verwendet, um die tiledata zu erhalten, die gedreht wird und dann verwendet wird, um die Fliese erneut zu setzen.
