layer_tilemap_create

Beschreibung

Mit dieser Funktion können Sie einer Ebene eine Tileset-Ressource zuweisen, die als Tilemap in Ihrem Projekt verwendet werden soll. Sie geben die Layer-ID an, die Sie erhalten, wenn Sie den Layer mit erstellen layer_create() ) oder der Ebenenname (als String - dies hat Auswirkungen auf die Leistung) und dann eine Anfangsposition (x, y), um die Tilemap zum Raum hinzuzufügen, die zu verwendende Kachel-Ressource und dann die Breite und Höhe der Tilemap in Zellen (z. B. eine Breite von 20 und eine Höhe von 10 erstellt eine Tilemap mit 200 Zellen, die 20 Zellen breit und 10 Zellen hoch sind), wobei die Größe der Zellen selbst durch den ausgewählten Kachelsatz definiert wird. Es ist erwähnenswert, dass Sie keine Kacheln an negativen Positionen innerhalb der Tilemap platzieren können. Daher müssen alle Kacheln innerhalb der Zellbereiche von 0 bis Breite und 0 bis Höhe liegen.


Syntax:

layer_tilemap_create(layer_id, x, y, tile set, width, height)


Argumente:

Streit Beschreibung
layer_id Der eindeutige ID-Wert der Ebene für das Ziel (oder der Ebenenname als Zeichenfolge)
x Die x-Position der Tilemap im Raum
y Die y-Position der Tilemap im Raum
tile set Der zu verwendende Kachelsatzindex
width Die Breite Tilemap (in Zellen)
height Die Höhe der Tilemap (in Zellen)


Kehrt zurück:

tilemap element ID (Real)


Beispiel:

global.back_layer = layer_create(10000);
global.back_tiles = layer_tilemap_create(global.back_layer, 0, 0, tmap_Trees, 16, 32);

Der obige Code erstellt eine neue Ebene und fügt dann ein neues Tilemap-Element hinzu, wobei die Position der Tilemap auf (0,0) sowie der zu verwendende Kachelsatz sowie die Breite und Höhe der Tilemap festgelegt wird.
