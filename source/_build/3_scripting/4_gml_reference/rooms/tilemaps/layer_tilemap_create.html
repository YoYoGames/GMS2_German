layer_tilemap_create

Beschreibung

Mit dieser Funktion können Sie einer Ebene eine Sprite-Ressource zuweisen, die in Ihrem Projekt als Kachelkarte verwendet werden soll. Sie geben die Layer-ID an (die Sie erhalten, wenn Sie die Ebene mit erstellen layer_create() ) oder der Layername (als String - dies wirkt sich auf die Leistung aus) und dann eine anfängliche (x, y) Position, um die Tilemap dem Raum hinzuzufügen, die zu verwendende Tile Set Ressource und dann die Breite und Höhe des Kachelkarte in Zellen (dh eine Breite von 20 und eine Höhe von 10 werden eine Kachelkarte mit 200 Zellen erzeugen, die 20 Zellen breit und 10 Zellen groß ist), wobei die Größe der Zellen selbst durch den gewählten Kachelsatz definiert wird. Beachten Sie, dass Sie keine Kacheln an negativen Positionen innerhalb der Kachel platzieren können. Daher müssen alle Kacheln innerhalb der Zellenräume 0 bis Breite 0 bis Höhe platziert werden.


Syntax:

layer_tilemap_create(layer_id, x, y, tile set, width, height)


Argumente:

Streit Beschreibung
layer_id Der eindeutige ID-Wert des zu targetierenden Layers (oder der Layername als String)
x Die x-Position der Tilemap im Raum
y Die y-Position der Tilemap im Raum
tile set Der zu verwendende Kachelset-Index
width Die Breite Tilemap (in Zellen)
height Die Höhe der Kachelkarte (in Zellen)


Kehrt zurück:

tilemap element ID (Real)


Beispiel:

global.back_layer = layer_create(10000);
global.back_tiles = layer_tilemap_create(global.back_layer, 0, 0, tmap_Trees, 16, 32);

Der obige Code erstellt eine neue Ebene und fügt ihr ein neues Element hinzu, wobei die Position der Karte auf (0,0) gesetzt wird, sowie die zu verwendende Ebene und die Breite und Höhe der Karte.
