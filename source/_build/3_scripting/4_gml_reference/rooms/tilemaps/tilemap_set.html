tilemap_set

Beschreibung

Diese Funktion kann verwendet werden, um eine beliebige Zelle (Rasterquadrat) innerhalb des Tilemap-Elements auf der Ebene auf eine neue Kachel zu setzen. Sie geben die Element-ID des Tilemap-Elements an (die Sie beim Erstellen eines Tilemap-Elements mit erhalten layer_tilemap_create() oder wenn Sie die Funktion verwenden layer_tilemap_get_id() ) und dann die Kacheldaten sowie die Position innerhalb der Kachelkarte. Sie können Kacheldaten mit der Funktion abrufen tilemap_get und dann benutze die tile_ Funktionen, um es vor dem Einstellen der Zelle mit dieser Funktion zu ändern. Die Funktion wird zurückkehren true wenn die Kachel erfolgreich eingestellt wurde und false wenn ein Problem aufgetreten ist und es nicht festgelegt wurde.


Syntax:

tilemap_set(tilemap_element_id, tiledata, xcell, ycell)


Argumente:

Streit Beschreibung
tilemap_element_id Der eindeutige ID-Wert des zu ändernden Elements teilemap
tiledata Die zu setzenden Kacheldaten
xcell Die Position der Zelle (Gitter) entlang der X-Achse
ycell Die Position der Zelle (Gitter) entlang der Y-Achse


Kehrt zurück:

Boolean


Beispiel:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
data = tile_set_flip(data, true);
tilemap_set(map_id, data, 0, 0);

Der obige Code ruft die ID für die Kachelkarte auf der Ebene "Tiles_Sky" ab und verwendet diese, um die Daten von der Kachel an der Zelle (0, 0) abzurufen. Diese Kacheldaten werden dann umgedreht, bevor sie erneut zum Setzen der Kachel auf der Kachelkarte verwendet werden.
