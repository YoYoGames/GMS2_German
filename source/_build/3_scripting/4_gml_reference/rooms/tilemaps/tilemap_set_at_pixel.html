tilemap_set_at_pixel

Beschreibung

Diese Funktion kann verwendet werden, um eine Zelle innerhalb des Tilemap-Elements auf der Ebene auf eine neue Kachel zu setzen, wobei die tatsächliche Position der Kachel innerhalb des Raums geändert wird. Sie geben die Element-ID des Tilemap-Elements an (die Sie beim Erstellen eines Tilemap-Elements mit erhalten layer_tilemap_create() oder wenn Sie die Funktion verwenden layer_tilemap_get_id() ), und dann die Kachel Daten zu setzen sowie die Position innerhalb des Raumes. Sie können Kacheldaten mit der Funktion abrufen tilemap_get_at_pixel und dann benutze die tile_ Funktionen, um es zu ändern, bevor Sie es mit dieser Funktion erneut einstellen. Die Funktion wird zurückkehren true wenn die Kachel erfolgreich eingestellt wurde und false wenn ein Problem aufgetreten ist und es nicht festgelegt wurde.


Syntax:

tilemap_set_at_pixel(tilemap_element_id, tiledata, x, y)


Argumente:

Streit Beschreibung
tilemap_element_id Der eindeutige ID-Wert des zu ändernden Elements teilemap
tiledata Die Kachel stellt Daten ein, die festgelegt werden sollen
xcell Die x Position (im Raum)
ycell Die y-Position (im Raum)


Kehrt zurück:

Boolean


Beispiel:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get_at_pixel(map_id, 4, 4);
data = tile_set_flip(data, true);
tilemap_set_at_pixel(map_id, data, 4, 4);

Der obige Code ruft die ID für die Kachelkarte auf der Ebene "Tiles_Sky" ab und verwendet diese, um die Daten von der Kachel an Position (4, 4) zu erhalten. Diese Kacheldaten werden dann umgedreht, bevor sie erneut zum Setzen der Kachel auf der Kachelkarte verwendet werden.
