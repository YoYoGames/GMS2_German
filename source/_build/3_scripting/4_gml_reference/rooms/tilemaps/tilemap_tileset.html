tilemap_tileset

Beschreibung

Mit dieser Funktion können Sie die Fliesensatz-Ressource ändern, die einem bestimmten Tilemap-Element auf einer Ebene zugewiesen ist. Sie geben die Element-ID des Tilemap-Elements an (die Sie beim Erstellen eines Tilemap-Elements mit erhalten layer_tilemap_create() oder wenn Sie die Funktion verwenden layer_tilemap_get_id() ) und liefern dann einen Kachelset-Index und die Kachelkarte erhält das neue Sprite.


Syntax:

tilemap_tileset(tilemap_element_id, tileset_index)


Argumente:

Streit Beschreibung
tilemap_element_id Der eindeutige ID-Wert des zu ändernden Elements teilemap
tileset_index Der neue zu verwendende Kachelsatzindex


Kehrt zurück:

N/A


Beispiel:

var lay_id = layer_get_id("Tiles_trees");
var tile_id = layer_tilemap_get_id(lay_id);
if tilemap_get_tileset(tile_id) != ts_Nighttime
   {
   tilemap_tileset(tile_id, ts_Nighttime);
   }

Der obige Code prüft den aktuellen Kachelsatz, der der Timemap auf dem Layer "Tiles_trees" zugewiesen wurde, und wenn er nicht "ts_Nighttime" ist, wird dieser Kachelsatz dem Kachelplan zugewiesen.
