layer_tilemap_get_id

Beschreibung

Diese Funktion kann verwendet werden, um den eindeutigen ID-Wert des Tilemap-Elements auf einer Ebene abzurufen. Sie geben die Layer-ID an (die Sie erhalten, wenn Sie die Ebene mit erstellen layer_create() ) oder der Layername (als String - dies wirkt sich auf die Leistung aus) und die Funktion gibt den ID-Wert zurück, der dem Element kachelmap auf dem Layer zugeordnet ist. Beachten Sie, dass diese Funktion speziell für die Verwendung mit Tilemaps entwickelt wurde, die in der IDE hinzugefügt wurden, so als ob Sie eine Tilemap zu einem Layer hinzufügen, indem Sie mithilfe der Funktion Code verwenden layer_tilemap_create, dann wird die eindeutige ID für das hinzugefügte Tilemap-Element zurückgegeben.


Syntax:

layer_tilemap_get_id(layer_id)


Argumente:

Streit Beschreibung
layer_id Der eindeutige ID-Wert des zu targetierenden Layers (oder der Layername als String)


Kehrt zurück:

tilemap element ID value (Real)


Beispiel:

var lay_id = layer_get_id("tilemap_trees");
var tile_id = layer_tilemap_get_id(lay_id);
layer_tilemap_destroy(tile_id);

Der obige Code ruft die Layer-ID für den Layer "tellemap_trees" ab und verwendet diese, um die ID des Tilemap-Elements auf diesem Layer zu erhalten. Diese ID wird dann verwendet, um die Kachelkarte von der Ebene zu entfernen.
