tile_get_rotate

Beschreibung

Diese Funktion kann verwendet werden, um zu überprüfen, ob die Kachel in einem gegebenen Satz von Kacheln um 90 Grad gedreht wurde oder nicht. Sie geben die tiledata, die mit der Funktion abgerufen werden kann tilemap_get und die Funktion wird zurückkehren true wenn die Kachel gedreht wird, oder false wenn nicht.


Syntax:

tile_get_rotate(tiledata)


Argumente:

Streit Beschreibung
tiledata die tiledata zu überprüfen


Kehrt zurück:

Boolean


Beispiel:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var bool = !tile_get_rotate(data);
data = tile_set_rotate(data, bool);
tilemap_set(map_id, data, mx, my);

Der obige Code ruft die Kachelkarten-ID von der gegebenen Ebene ab und erhält dann die X- und Y-Zellenposition für die Kachel unter der Maus. Diese Position wird dann verwendet, um die tiledata zu erhalten, die gedreht wird und dann verwendet wird, um die Fliese erneut zu setzen.
