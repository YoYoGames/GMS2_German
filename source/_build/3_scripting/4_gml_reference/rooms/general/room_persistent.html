room_persistent


Beschreibung

Diese Variable kann verwendet werden, um das persistente Flag für den aktuellen Raum zu erhalten und zu setzen. Wenn auf eingestellt true Der Raum wird als hartnäckig betrachtet. In diesem Fall wird jedes Mal, wenn Sie den Raum verlassen und wieder zurückkommen, der Zustand der Instanzen in diesem Raum beibehalten. Wenn es jedoch als markiert ist false Jedes Mal, wenn Sie in den Raum zurückkehren, wird es in den Ausgangszustand zurückgesetzt. Sie sollten beachten, dass ein persistenter Raum erheblich mehr Speicher benötigt als ein normaler Raum und es wird nicht empfohlen, zu viele davon in Ihrem Spiel zu haben.


Syntax:

room_persistent;


Kehrt zurück:

Boolean


Beispiel:

if lives < 1
   {
   room_persistent = false;
   room_goto(rm_start);
   }

Der obige Code prüft die Variable "lebt" und wenn sie kleiner als 1 ist, wird sie die Raumpersistenz auf setzen false und dann die Zimmer wechseln.
