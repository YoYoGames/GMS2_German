room_instance_clear


Beschreibung

Diese Funktion löscht einen Raum aller Instanzen (für die Instanzen, die entfernt werden, dürfen keine Destroy-Ereignisse aufgerufen werden).


Syntax:

room_instance_clear(index);


Argumente:

Streit Beschreibung
index Der Index des Raums, in dem alle Instanzen gelöscht werden sollen.


Kehrt zurück:

N/A


Beispiel:

global.rm = room_add();
room_assign(rm_Base, global.rm);
room_instance_clear(global.rm);

Der obige Code fügt dem Spiel einen neuen Raum hinzu und kopiert dann den Inhalt des Raums, der als "rm_Base" indiziert ist. Es löscht dann alle Instanzen von diesem neuen Raum.
