room_set_persistent


Beschreibung

Mit dieser Funktion können Sie die Dauer eines beliebigen Raums in Ihrem Spiel (mit Ausnahme des aktuellen ) ändern (oder festlegen). Ein Zimmer mit Ausdauer gekennzeichnet als true Es behält den Status aller Instanzen in diesem Raum bei, wenn der Spieler verlässt und dann zurückkehrt, während die Persistenz als markiert ist false Es wird jedes Mal auf den Anfangszustand zurückgesetzt. Sie sollten beachten, dass ein persistenter Raum erheblich mehr Speicher benötigt als ein normaler Raum und es wird nicht empfohlen, zu viele davon in Ihrem Spiel zu haben.

HINWEIS: Diese Funktion schaltet die Persistenz NICHT ab, wenn der Raum bereits besucht wurde! Ein persistenter Raum, der einmal besucht wurde, wird in Erinnerung gehalten und um Persistenz auszuschalten, sollten Sie in diesen Raum gehen und den room_persistent Variable zu false und dann den Raum wieder verlassen.



Syntax:

room_set_persistent(index, val);


Argumente:

Streit Beschreibung
index Der Index des Raumes, um die Persistenz von zu setzen.
val Ob der Raum dauerhaft sein soll ( true ) oder nicht ( false ).


Kehrt zurück:

N/A


Beispiel:

global.myroom = room_add();
room_set_width(global.myroom, 640);
room_set_height(global.myroom, 480);
room_set_persistent(global.myroom, true);

Dies erstellt einen neuen Raum und speichert seinen Index in der Variable "global.myroom". Es wird dann seine Breite auf 640 Pixel einstellen, seine Höhe auf 480 Pixel, seine Beschriftung auf "Game Room" und seine Persistenz auf "True".
