room_set_width


Beschreibung

Mit dieser Funktion können Sie die Breite eines beliebigen Raums in Ihrem Spiel (mit Ausnahme des aktuellen ) ändern (oder festlegen).


Syntax:

room_set_width(index, w);


Argumente:

Streit Beschreibung
index Der Index des Raumes, um die Breite einzustellen.
w Die neue Breite des Raums in Pixel.


Kehrt zurück:

N/A


Beispiel:

global.myroom = room_add();
room_set_width(global.myroom, 640);
room_set_height(global.myroom, 480);
room_set_persistent(global.myroom, false);

Dies erstellt einen neuen Raum und speichert seinen Index in der Variable "global.myroom". Es wird dann seine Breite auf 640 Pixel einstellen, seine Höhe auf 480 Pixel, seine Beschriftung auf "Game Room" und seine Beständigkeit auf "false".
