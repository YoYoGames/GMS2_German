zufällig


Beschreibung

Diese Funktion eignet sich für Wahrscheinlichkeiten, bei denen keine Ganzzahl (ganze Zahl) zurückgegeben werden muss. Zum Beispiel, random(100) gibt einen Wert von 0 bis 99 zurück, dieser Wert kann jedoch 22.56473 sein! In dieser Funktion können Sie auch reelle Zahlen und keine ganzen Zahlen verwenden. random(0.5), der einen Wert zwischen 0 und 0,4999999 zurückgibt.

ANMERKUNG: Diese Funktion gibt bei jeder erneuten Ausführung des Spiels denselben Wert zurück, da GameMaker Studio 2 jedes Mal den gleichen zufälligen Start-Seed generiert, um das Debuggen von Code zu erleichtern. Um dieses Verhalten zu vermeiden, verwenden Sie randomise zu Beginn deines Spiels.


Syntax:

random(n);


Streit Beschreibung
n Der obere Bereich, aus dem die Zufallszahl ausgewählt wird.


Kehrt zurück:

Real


Beispiel:

if random(10) >= 9
   {
   score += 100;
   }

Dies führt zu einer ungefähr eins zu zehn Chance, die Punktzahl um 100 zu erhöhen.
