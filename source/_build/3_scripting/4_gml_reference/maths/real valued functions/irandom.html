irandom

Gibt eine zufällige Ganzzahl zwischen 0 und n zurück. Dies schließt n ein, wenn n selbst ein ganzzahliger Wert ist.

Syntax:

irandom(n);


Streit Beschreibung
n Der obere Bereich, aus dem die Zufallszahl ausgewählt wird.


Rückgabe: Ganzzahl


Beschreibung

Diese sehr nützliche Funktion gibt nur Ganzzahlen (ganze Zahlen) zurück. So können Sie beispielsweise eine Zufallszahl von 0 bis 9 erhalten irandom(9) und es wird eine Zahl von 0 bis einschließlich 9 zurückgeben... reelle Zahlen können auch verwendet werden, aber der obere Wert wird ausgeschlossen, so irandom(9.7) gibt nur einen Wert von 0 bis 9 zurück. Die Funktion hat eine Obergrenze von $7fffffffffffffff LL, daher ist Vorsicht geboten, wenn sehr große Zahlen verwendet werden.

HINWEIS: Diese Funktion gibt bei jedem Spielstart den gleichen Wert zurück, da GameMaker Studio 2 jedes Mal den gleichen anfänglichen Zufallswert generiert, um den Debugging-Code zu vereinfachen. Um dieses Verhalten zu vermeiden, verwenden Sie randomise zu Beginn deines Spiels.


Beispiel:

if irandom(9) = 1
   {
   score += 100;
   }

Dies wird eine Wahrscheinlichkeit von einer von zehn ergeben (da 0 enthalten ist), 100 Punkte zu addieren.
