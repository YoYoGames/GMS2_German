irandom


Beschreibung

Diese sehr nützliche Funktion gibt nur Ganzzahlen (ganze Zahlen) zurück. So können Sie beispielsweise eine Zufallszahl von 0 bis 9 erhalten irandom(9) und es wird eine Zahl von 0 bis einschließlich 9 zurückgegeben. Beachten Sie, dass auch Floats verwendet werden können, der obere Wert nach dem Punkt jedoch ausgeschlossen wird irandom(9.7) gibt nur einen Wert von 0 bis 9 zurück. Die Funktion hat eine Obergrenze von $7fffffffffffffff LL, daher ist bei der Verwendung sehr großer Zahlen Vorsicht geboten.

HINWEIS: Diese Funktion gibt bei jeder erneuten Ausführung des Spiels denselben Wert zurück, da GameMaker Studio 2 jedes Mal den gleichen zufälligen Start-Seed generiert, um das Debuggen von Code zu erleichtern. Um dieses Verhalten zu vermeiden, verwenden Sie randomise zu Beginn deines Spiels.


Syntax:

irandom(n);


Streit Beschreibung
n Der obere Bereich, aus dem die Zufallszahl ausgewählt wird.


Kehrt zurück:

Integer


Beispiel:

if irandom(9) = 1
   {
   score += 100;
   }

Dies führt zu einer Eins-zu-Zehn-Chance (da 0 enthalten ist), der Punktzahl 100 hinzuzufügen.
