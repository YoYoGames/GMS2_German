random_get_seed

Gibt den aktuellen Zufallssamen zurück.

Syntax:

random_get_seed();


Rückgabe: Echt


Beschreibung

Eine Zufallszahl zu erzeugen GameMaker Studio 2 beginnt mit einem Zufallsstartnummer. Mit dieser Funktion können Sie diesen Startwert abrufen, der dann gespeichert werden kann, um eine bestimmte Reihe zufälliger Ereignisse zu reproduzieren. Angenommen, Sie haben ein prozedural erstelltes Spiel, möchten aber, dass der Spieler in zuvor entdeckte Bereiche zurückkehren kann. Nun, am Anfang jedes Bereichs können Sie den Zufallssaat mit dieser Funktion speichern und so können Sie jedes Mal, wenn Sie dorthin zurückkehren müssen, verwenden random_set_seed um den Anfangswert des Samens GameMaker Studio 2 und GameMaker Studio 2 so zu zwingen, die gleichen "zufälligen" Funktionen wie zuvor zu reproduzieren.


Beispiel:

ini_open("Save.ini")
ini_write_real("Levels", string(level), random_get_seed());
ini_close();

Der obige Code wird den zufälligen Startwert erhalten und in einer Ini-Datei speichern.
