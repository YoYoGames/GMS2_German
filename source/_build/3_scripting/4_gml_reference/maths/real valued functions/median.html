Median

Gibt den Median (mittlere Zahl) einer Menge von Realwertargumenten zurück.

Syntax:

median(val1, val2, ... max_val);


Streit Beschreibung
val0... max_val Die zu vergleichenden Werte.


Rückgabe: Echt


Beschreibung

Diese Funktion gibt den Median der Eingabewerte zurück, also den Mittelwert. Wenn die Anzahl der Argumente gerade ist, wird der kleinere der beiden mittleren Werte zurückgegeben und die Funktion kann so viele Argumente wie erforderlich haben (beachten Sie, dass mehr Argumente dazu führen, dass die Funktion langsamer analysiert wird), die alle echte Werte sein müssen. Dies bedeutet zum Beispiel, median(43, 12, 25, 19, 6) würde 19 zurückgeben, da es der mittlere Wert zwischen dem ganzen Rest ist.


Beispiel:

x = median( 0, x, room_width );
y = median( 0, y, room_height );

Dies wird den Spieler davon abhalten, irgendeine Seite des Raums zu verlassen, indem er den Median als Klammer verwendet. Wenn sich der Spieler zum Beispiel links von der Raumgrenze bewegt, wird sein x kleiner als 0 sein. Dies bedeutet, dass die mittlere Zahl des ersten der Mediane 0 ist, so dass der Spieler zu (0, y).
