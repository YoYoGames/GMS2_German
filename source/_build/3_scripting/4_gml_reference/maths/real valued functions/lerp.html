Lerp

Gibt die lineare Interpolation zweier Eingabewerte in der angegebenen Menge zurück.

Syntax:

lerp(a, b, amt)


Streit Beschreibung
a Der erste Wert.
b Der zweite Wert.
amt Der zu interpolierende Betrag


Rückgabe: Echt.


Beschreibung

Mit dieser Funktion können Sie den Wert finden, der der Position zwischen zwei anderen Werten für einen bestimmten Prozentsatz entspricht. Also wenn Sie zum Beispiel:

lerp(0, 10, 0.5)

Sie erhalten den Rückgabewert von 5, also 50% der Eingabewerte. Sie können auch mit dieser Funktion extrapolieren, indem Sie einen positiven oder negativen Wert für den Interpolationsbetrag angeben.

lerp(0, 10, 2)

gibt einen Wert von 20 zurück.


Beispiel:

xx = lerp(x, x + hspeed, room_speed);
yy = lerp(y, y + vspeed, room_speed);

Der obige Code verwendet die lineare Interpolationsfunktion, um vorherzusagen, wo sich eine Instanz nach einer Sekunde Spielzeit bewegt hätte.
