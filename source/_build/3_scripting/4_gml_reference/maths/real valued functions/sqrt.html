sqrt

Gibt die Quadratwurzel von val zurück. Val kann kein negativer Wert sein.

Syntax:

sqrt(val);


Streit Beschreibung
val Die Nummer, um die Quadratwurzel zu erhalten.


Rückgabe: Echt


Beschreibung

Wenn Sie eine Zahl mit sich selbst multiplizieren, erhalten Sie das Quadrat dieser Zahl, aber manchmal möchten Sie das Gegenteil tun und die Quadratwurzel einer Zahl erhalten. Um herauszufinden, welche Zahl mit sich selbst multipliziert wurde, um einen bestimmten positiven Wert zu erhalten, verwenden wir diese Funktion. Beispielsweise: sqrt(9) gibt 3 seit 3 ​​* 3 = 9 zurück.


Beispiel:

num = sqrt(exp);

Der obige Code setzt die Variable "num" so, dass sie die Quadratwurzel des in "exp" enthaltenen Wertes enthält.
