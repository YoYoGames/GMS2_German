random_range


Beschreibung

Diese Funktion gibt eine Zufallszahl zwischen dem angegebenen Bereich zurück, und dieser Rückgabewert muss keine Ganzzahl sein. Zum Beispiel, random_range(20,50) gibt eine Zufallszahl von 20 bis 50 zurück, der Wert kann jedoch eine reelle Zahl wie 38,65265 sein. Reelle Zahlen können auch als Eingabeargumente verwendet werden.

ANMERKUNG: Diese Funktion gibt bei jeder erneuten Ausführung des Spiels denselben Wert zurück, da GameMaker Studio 2 jedes Mal den gleichen zufälligen Start-Seed generiert, um das Debuggen von Code zu erleichtern. Um dieses Verhalten zu vermeiden, verwenden Sie randomise zu Beginn deines Spiels.


Syntax:

random_range(n1, n2);


Streit Beschreibung
n1 Das untere Ende des Bereichs, aus dem die Zufallszahl ausgewählt wird.
n2 Das obere Ende des Bereichs, aus dem die Zufallszahl ausgewählt wird.


Kehrt zurück:

Real


Beispiel:

score += random_range(500, 600);

Dies erhöht die Gesamtpunktzahl um 500 bis 600.
