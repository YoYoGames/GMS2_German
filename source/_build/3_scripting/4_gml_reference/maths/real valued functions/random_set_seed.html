random_set_seed


Beschreibung

Um eine Zufallszahl zu generieren, beginnt GameMaker Studio 2 mit einer zufälligen Startnummer. Mit dieser Funktion können Sie diesen Startwert auf einen bekannten Wert setzen und so das Ergebnis aller zufälligen Ereignisse danach zwingen, bei jeder Ausführung des Programms gleich zu sein. Diese Funktion kann beispielsweise in Verbindung mit verwendet werden random_get_seed Erstellen von prozedural generierten Inhalten und Speichern der Ergebnisse ohne große Speicherstände (Sie speichern nur den Startwert, es ist keine weitere Notwendigkeit erforderlich). Sollten Sie wirklich zufällige Ergebnisse für alles benötigen, sollten Sie das verwenden randomise() Funktion.

ANMERKUNG: Obwohl dieser Startwert für jede Zielplattform konsistente Ergebnisse liefert, können die Ergebnisse aufgrund der unterschiedlichen Arbeitsweise der einzelnen Ziele von Plattform zu Plattform variieren.


Syntax:

random_set_seed(val);


Streit Beschreibung
val Der Samen zum setzen.


Kehrt zurück:

N/A


Beispiel:

if debug random_set_seed(1);

Der obige Code setzt den Zufallsstartwert nur dann auf 1, wenn die Variable "debug" wahr ist.
