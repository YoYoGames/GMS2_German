random_set_seed

Legt den Ausgangswert fest, aus dem Zufallszahlen und Ergebnisse gezogen werden.

Syntax:

random_set_seed(val);


Streit Beschreibung
val Der zu setzende Samen.


Rückkehr: N / A


Beschreibung

Um eine Zufallszahl zu erzeugen, startet GameMaker Studio 2 mit einer zufälligen Startnummer. Mit dieser Funktion können Sie diesen Ausgangswert auf einen bekannten Wert setzen und so das Ergebnis aller zufälligen Ereignisse nach jeder Ausführung des Programms als "erzwingen". Zum Beispiel kann diese Funktion in Verbindung mit verwendet werden random_get_seed um prozedural generierten Inhalt zu erstellen und die Ergebnisse zu speichern, ohne riesige Savegames zu haben (Sie speichern nur den Seed, brauchen nichts anderes). Sollten Sie wirklich zufällige Ergebnisse für alles benötigen, sollten Sie das verwenden randomise() Funktion.

HINWEIS: Während dieser Seed auf jeder Zielplattform konsistente Ergebnisse liefert, können die Ergebnisse aufgrund der unterschiedlichen Funktionsweise jedes Ziels zwischen den Plattformen variieren.


Beispiel:

if debug random_set_seed(1);

Der obige Code setzt den Random-Seed nur dann auf 1, wenn die Variable "Debug" wahr ist.
