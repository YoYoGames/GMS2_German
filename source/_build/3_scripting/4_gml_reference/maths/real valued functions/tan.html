bräunen


Beschreibung

In einem rechtwinkligen Dreieck ist tan (val) = Gegenteil / Angrenzend, wobei val einer der drei Winkel ist.

HINWEIS: Der zurückgegebene Wert wird in Bogenmaß angegeben, nicht in Grad.
ANMERKUNG: Pi / 2, (3 pi / 2), -pi / 2 und eine große Anzahl anderer Werte werden aufgrund ihrer zurückkehrenden Unendlichkeit mit dieser Funktion fehlerhaft. Eine Graphdarstellung davon würde bei diesen Werten Asymptoten erzeugen.


Syntax:

tan(val);


Streit Beschreibung
val Der Winkel (im Bogenmaß), um den Tangens von zurückzugeben.


Kehrt zurück:

Real


Beispiel:

val = tan( pi );

Damit wird val auf 0 gesetzt.
