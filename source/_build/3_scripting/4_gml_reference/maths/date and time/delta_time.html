Delta_Zeit

Die Zeit verging in Mikrosekunden zwischen einem Schritt und einem anderen.

Syntax:

delta_time


Rückgabe: Echt


Beschreibung

Der Zweck des Delta-Timings besteht darin, die Auswirkungen der Verzögerung oder Langsamkeit von Computern zu eliminieren, die versuchen, komplexe Grafiken oder viel Code zu verarbeiten. Es ist ein Wert, der der Geschwindigkeit von Objekten hinzugefügt werden kann, damit sie sich unabhängig von der Verzögerung mit der gleichen Geschwindigkeit bewegen. Dies wird in GameMaker Studio 2 mithilfe der schreibgeschützten Variablen erreicht delta_time Jeder Schritt misst die Zeit, die in Mikrosekunden zwischen einem Schritt und dem nächsten verstrichen ist (1 Mikrosekunde ist 1.000.000stel Sekunde). Daher die Variable delta_time kann verwendet werden, um zu berechnen, um wie viel schneller (zum Beispiel) ein Spielcharakter sich bewegen muss, um eine Verzögerungsspitze im Spiel auszugleichen.


Beispiel:

speed = spd * (ot - delta_time);

Der obige Code wird die Geschwindigkeit der Instanz unter Verwendung der Delta-Zeit einstellen, um die Verzögerung zu korrigieren. Die Variable "ot" würde verwendet werden, um den zuvor berechneten delta_time-Wert zu halten, so dass das Verhältnis verwendet werden kann, um den Basisgeschwindigkeitswert zu multiplizieren (gehalten in der Variablen "spd") und so eine konsistente Geschwindigkeit für die Instanz zu erhalten.
