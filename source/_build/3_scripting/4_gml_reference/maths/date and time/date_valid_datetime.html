date_valid_datetime

Gibt zurück, ob eine bestimmte Datetime gültig ist.

Syntax:

date_valid_datetime(year, month, day, hour, minute, second);


Streit Beschreibung
year Das Jahr zu überprüfen.
month Der zu überprüfende Monat.
day Der Tag zu überprüfen.
hour Die Stunde zu überprüfen.
minute Die Minute zu überprüfen.
second Die zweite Überprüfung.


Rückgabe: Boolean


Beschreibung

Mit dieser Funktion können Sie einen Datetime-Wert überprüfen, um zu sehen, ob er gültig ist (returns) true ) oder nicht (zurück false ). Beachten Sie, dass diese Funktion nur ein gültiges Datetime-Datum als nach dem 1.1.1970 betrachtet und alles davor zurückgibt false, so frühest du prüfen können:

date_valid_datetime(1970, 01, 01, 0, 0, 0);




Beispiel:

if date_valid_datetime(2011, 9, 15, 10, 3, 30))
   {
   mydatetime = date_create_datetime( 2011, 9, 15, 10, 3, 30 );
   }

Dies setzt "mydatetime" auf den 15. September 2011, 10: 03.30, wenn es ein gültiger Wert ist (was es ist).
