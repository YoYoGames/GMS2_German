date_compare_datetime

Gibt an, welche von zwei Zeiten an bestimmten Daten die letzte ist.

Syntax:

date_compare_datetime(date1, date2);


Streit Beschreibung
date1 Das erste Datum.
date2 Die Datetime zum Vergleich.


Rückgabe: Echt


Beschreibung

Mit dieser Funktion können Sie zwei Daten überprüfen, um zu sehen, welche früher oder später als die andere ist. Die Funktion gibt -1 zurück, wenn date1 früher ist, 1 wenn date1 später ist und 0, wenn sie gleich sind, obwohl da dies auch Zeit in Anspruch nimmt, ist eine 0 für das Matching sehr selten (es muss buchstäblich die selbe Sekunde sein) am selben Tag desselben Monats desselben Jahres, um eine 0 zurückzugeben.


Beispiel:

d = date_compare_datetime(date_create_datetime( 2011, 9, 15, 11, 4, 0 ), date_current_datetime());

Dies würde d auf den entsprechenden Wert setzen, abhängig davon, welches der Datumswerte am frühesten war, wahrscheinlich -1, da das aktuelle Datum dem 15. September 2011 voraus sein würde.
