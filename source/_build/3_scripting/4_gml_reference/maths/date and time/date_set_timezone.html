date_set_timezone

Legt die Basiszeitzone für alle Datums- und Uhrzeitfunktionen fest.

Syntax:

date_set_timezone(timezone);


Streit Beschreibung
timezone Die Zeitzone für die Basiszeit.


Rückkehr: N / A


Beschreibung

Legt die Basiszeitzone fest, die für alle übrigen Datums- und Uhrzeitfunktionen verwendet wird. Diese Zeitzone kann entweder lokal (wie vom System festgelegt) oder UTC sein und Sie verwenden eine der folgenden Konstanten, um festzulegen, welche verwendet wird (standardmäßig ist dies die lokale Zeit):

Konstante Beschreibung
timezone_local Verwenden Sie die vom System festgelegte lokale Zeitzone
timezone_utc Verwenden Sie koordinierte Weltzeit


Beispiel:

if date_get_timezone() != timezone_utc
   {
   date_set_timezone(timezone_utc);
   }

Dieser Code überprüft die Basiszeitzoneneinstellung für das Spiel und wenn es nicht UTC ist, ändert es es.
