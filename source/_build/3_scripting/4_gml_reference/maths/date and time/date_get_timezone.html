date_get_timezone

Legt die Basiszeitzone für alle Datums- und Uhrzeitfunktionen fest.

Syntax:

date_get_timezone();


Rückgabe: Konstante


Beschreibung

Ruft die Basiszeitzone ab, die für den gesamten Rest der Datums- und Uhrzeitfunktionen verwendet wird. Diese Zeitzone kann entweder lokal (wie vom System festgelegt) oder UTC sein und die Funktion gibt eine der folgenden Konstanten zurück:

Konstante Beschreibung
timezone_local Verwenden Sie die vom System festgelegte lokale Zeitzone
timezone_utc Verwenden Sie koordinierte Weltzeit


Beispiel:

if date_get_timezone() != timezone_utc
   br>    date_set_timezone(timezone_utc);
   }

Dieser Code überprüft die Basiszeitzoneneinstellung für das Spiel und wenn es nicht UTC ist, ändert es es.
