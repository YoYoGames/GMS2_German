dot_product_normalised

Syntax:

dot_product_normalised(x1, y1, x2, y2)


Streit Beschreibung
x1 Die x-Koordinate des ersten Vektors.
y1 Die y-Koordinate des ersten Vektors.
x2 Die x-Koordinate des zweiten Vektors.
y2 Die y-Koordinate des zweiten Vektors.


Rückgabe: Echt


Beschreibung

Das Skalarprodukt ist ein Wert, der die Winkelbeziehung zwischen zwei Vektoren ausdrückt, und wird gefunden, indem zwei Vektoren genommen werden, diese miteinander multipliziert werden und dann die Ergebnisse addiert werden. Der Name "Punktprodukt" wird von dem zentrierten Punkt "·" abgeleitet, der häufig verwendet wird, um diese Operation zu bezeichnen (der alternative Name "Skalarprodukt" betont die Skalar- und nicht die Vektornatur des Ergebnisses).

Also, in 2D ist das Skalarprodukt der Vektoren a [x1, y1] und b [x2,2] x1x2 + y1y2, was bedeutet, dass das dot_product in GameMaker Studio 2 folgt berechnet wird:

a · b = (x1*x2)+(y1*y2);

Was ist mit dem normalisierten Skalarprodukt? Das normierte Skalarprodukt wurde so korrigiert, dass der Rückgabewert in den Bereich von -1 und 1 gebracht wird (siehe Normalisierte Vektoren für detailliertere Informationen), was in vielen Fällen außerordentlich nützlich ist, insbesondere wenn es um Beleuchtung und andere geht 3D Funktionen.


Beispiel:

var x1, y1, x2, y2;
x1 = lengthdir_x(1, image_angle);
y1 = lengthdir_y(1, image_angle);
x2 = o_Player.x - x;
y2 = o_Player.y - y;
if dot_product_normalised(x1, y1, x2, y2) > 0 seen=true else seen=false;

Der obige Code erzeugt einen Vektor unter Verwendung des Bildwinkels der Instanzen und erhält dann den Vektor des Spielerobjekts "o_Player" zu sich selbst. Schließlich berechnet es das Skalarprodukt dieser zwei Vektoren und wenn es größer als 0 ist, setzt es die Variable "gesehen" auf wahr, und wenn es gleich oder kleiner als 0 ist, setzt es es auf "falsch".
