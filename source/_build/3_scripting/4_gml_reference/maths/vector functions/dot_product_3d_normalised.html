dot_product_3d_normalised

Syntax:

dot_product_3d_normalised(x1, y1, z1, x2, y2, z2)


Streit Beschreibung
x1 Die x-Koordinate des ersten Vektors.
y1 Die y-Koordinate des ersten Vektors.
z1 Die z-Koordinate des ersten Vektors.
x2 Die x-Koordinate des zweiten Vektors.
y2 Die y-Koordinate des zweiten Vektors.
z2 Die z-Koordinate des zweiten Vektors.


Rückgabe: Echt


Beschreibung

Das Skalarprodukt ist ein Wert, der die Winkelbeziehung zwischen zwei Vektoren ausdrückt, und wird gefunden, indem zwei Vektoren genommen werden, diese miteinander multipliziert werden und dann die Ergebnisse addiert werden. Der Name "Punktprodukt" wird von dem zentrierten Punkt "·" abgeleitet, der häufig verwendet wird, um diese Operation zu bezeichnen (der alternative Name "Skalarprodukt" betont die Skalar- und nicht die Vektornatur des Ergebnisses).

Die eigentliche mathematische Formel kann folgendermaßen geschrieben werden:


In 2D ist das Skalarprodukt der Vektoren a [x1, y1] und b [x2, y2] x1x2 + y1y2, also in 3D das Skalarprodukt der Vektoren a [x1, y1, z1] und b [x1, y1, z1] ist x1x2 + y1y2 + z1z2. Also wird das dot_product_3d in GameMaker Studio 2 folgt berechnet:

a · b = (x1*x2) + (y1*y2) + (z1*z2);

Was ist mit dem normalisierten Skalarprodukt? Das normierte Skalarprodukt wurde so korrigiert, dass der Rückgabewert in den Bereich von -1 und 1 gebracht wird (siehe Normalisierte Vektoren für detailliertere Informationen), was in vielen Fällen außerordentlich nützlich ist, insbesondere wenn es um Beleuchtung und andere geht 3D Funktionen.


Beispiel:

var x1, y1, x2, y2;
x1 = 0;
y1 = 1;
z1 = 0;
x2 = o_Player.x - x;
y2 = o_Player.y - y;
z2 = o_Player.z - z;
if dot_product_3d_normalised(x1, y1, z1, x2, y2, z2) > 0 above=true else above=false;

Der obige Code erzeugt eine Vektornormale entlang der Achse y der Instanzen (aufwärts) und erhält dann den Vektor des Spielerobjekts "o_Player" zu sich selbst. Schließlich berechnet es das Skalarprodukt dieser beiden Vektoren und wenn es größer tan 0 ist, setzt es die Variable "über" auf wahr, und wenn es kleiner oder gleich 0 ist, setzt es es auf "falsch".
