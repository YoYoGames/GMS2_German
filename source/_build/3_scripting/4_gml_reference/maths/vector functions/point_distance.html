Punktentfernung

Syntax:

point_distance(x1, y1, x2, y2);


Streit Beschreibung
x1 Die x-Koordinate der ersten Komponente des Vektors
y1 Die y-Koordinate der ersten Komponente des Vektors
x2 Die X-Koordinate der zweiten Komponente des Vektors
y2 Die y-Koordinate der zweiten Komponente des Vektors


Rückgabe: Echt


Beschreibung

Diese Funktion gibt die Länge eines Vektors zurück, der durch die angegebenen Komponenten [x1, y1] und [x2, y2] gebildet wird. Wenn wir zum Beispiel im Bild unten den Abstand zwischen der Spielerschiffsposition und der gegnerischen Position ermitteln wollen, damit wir berechnen können, ob der Feind nahe genug ist, um darauf zu schießen, würden wir diese Funktion verwenden (der genaue Code ist in der Beispiel unter dem Bild):


Beispiel:

var ex, ey;
ex = instance_nearest(x, y, enemy).x;
ey = instance_nearest(x, y, enemy).y;
if point_distance(x, y, ex, ey) < 200
   {
   instance_create_layer(x, y, "Bullets", obj_Missile)
   }

Der obige Code wird die X- und Y-Koordinaten des nächsten Feindes erhalten und sie dann verwenden, um die Entfernung (Länge) des von ihnen gebildeten Vektors und die Spielerkoordinaten zu überprüfen. Wenn der Wert kleiner als 200 ist, erstellt das Player-Objekt eine Instanz von "obj_Missile".
