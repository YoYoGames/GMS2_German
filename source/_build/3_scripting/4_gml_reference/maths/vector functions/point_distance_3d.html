Punkt_Abstand_3d

Syntax:

point_distance_3d(x1, y1, z1, x2, y2, z2);


Streit Beschreibung
x1 Die x-Koordinate der ersten Komponente des Vektors
y1 Die y-Koordinate der ersten Komponente des Vektors
z1 Die z-Koordinate der ersten Komponente des Vektors
x2 Die X-Koordinate der zweiten Komponente des Vektors
y2 Die y-Koordinate der zweiten Komponente des Vektors
z2 Die z-Koordinate der zweiten Komponente des Vektors


Rückgabe: Echt


Beschreibung

Diese Funktion nimmt die gelieferten Komponenten des Vektors und gibt die Länge (Entfernung) des Vektors zurück. Es funktioniert genau so wie point_distance() aber mit der Zugabe von Faktorisierung in den z-Wert (Tiefe) für den Einsatz im 3D-Raum.


Beispiel:

var inst, ex, ey, ez;
inst = instance_nearest(x, y, enemy);
if inst
   {
   ex = inst.x;
   ey = inst.y;
   ez = inst.z;
   if point_distance_3d(x, y, z, ex, ey, ez) < 200
      {
      instance_create_layer(x, y, "Bullets", obj_Missile)
      }
   }

Der obige Code wird die x- und y- und z-Koordinaten des nächsten Feindes erhalten und sie dann verwenden, um die Entfernung (Länge) des von ihnen gebildeten Vektors und die Spielerkoordinaten zu überprüfen. Wenn der Wert kleiner als 200 ist, erstellt das Player-Objekt eine Instanz von "obj_Missile".
