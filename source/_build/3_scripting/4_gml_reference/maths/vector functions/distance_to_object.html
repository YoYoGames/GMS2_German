distance_to_object

Syntax:

distance_to_object( obj );


Streit Beschreibung
obj Das Objekt, nach dem gesucht werden soll.


Rückgabe: Echt.


Beschreibung

Diese Funktion berechnet die Entfernung von der Kante der Begrenzungsbox der aufrufenden Instanz zur nächsten Kante der nächsten Instanz des angegebenen Objekts. Das Objekt kann ein Objektindex oder eine spezifische Instanz-ID sowie das Schlüsselwort Sonstige sein, und der Abstand wird in Pixeln zurückgegeben. Beachten Sie, dass die Ergebnisse falsch sind, wenn eines der Objekte kein Sprite oder keine Maske definiert hat.


Beispiel:

if distance_to_object(obj_Player) < range
   {
   canshoot = true;
   }

Der obige Code prüft auf die Entfernung zum Spielerobjekt und wenn er kleiner als der in der Variablen "range" gespeicherte Wert ist, wird die Variable "canshoot" auf "true" gesetzt.
