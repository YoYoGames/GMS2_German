Punkt_Richtung

Syntax:

point_direction(x1, y1, x2, y2)


Streit Beschreibung
x1 Die x-Koordinate der ersten Komponente des Vektors
y1 Die y-Koordinate der ersten Komponente des Vektors
x2 Die X-Koordinate der zweiten Komponente des Vektors
y2 Die y-Koordinate der zweiten Komponente des Vektors


Rückgabe: Echt


Beschreibung

Diese Funktion gibt die Richtung eines Vektors zurück, der durch die angegebenen Komponenten [x1, y1] und [x2, y2] in Bezug auf die festen x / y-Koordinaten des Raums gebildet wird. Wenn wir zum Beispiel im Bild unten die Richtung von der Position des Spielerschiffes zur feindlichen Position sehen wollen, damit wir eine Rakete auf den Feind abfeuern können, würden wir diese Funktion verwenden (der genaue Code ist im Beispiel unter dem Bild) ):


Beispiel:

var ex, ey;
ex = instance_nearest(x, y, enemy).x;
ey = instance_nearest(x, y, enemy).y;
with (instance_create_layer(x, y, "Bullets", obj_Missile))
   {
   direction = point_direction(x, y, ex, ey);
   }

Der obige Code wird die x- und y-Koordinaten des nächsten Feindes erhalten und diese dann an ein Aufzählungsobjekt übergeben, das in der Funktion point_direction verwendet wird, um seine Bewegungsrichtung korrekt einzustellen.
