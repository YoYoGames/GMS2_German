shader_set_uniform_f


Syntax:

shader_set_uniform_f(handle, value1 [, value2, value3, value4]);


Streit Beschreibung
handle Der Handle der Shader-Konstante wird gesetzt.
value1... value4 Der Gleitkommawert (oder die Werte), auf den der Shader festgelegt werden soll.


Kehrt zurück:

N/A


Beschreibung

Mit dieser Funktion können Sie den Wert (oder die Werte) eines Shaders konstant setzen. Sie müssen zuvor den "Griff" der Konstante mit der Funktion erhalten haben shader_get_uniform, und Sie müssen wissen, um welche Art von Konstante es sich handelt, um die richtige Anzahl von Fließkommawerten zu übergeben, dh: wenn Sie eine haben vec2 Sie müssen zwei Werte an die Funktion übergeben.

Eine Ausnahme von der oben genannten Regel besteht beim Umgang mit 32-Bit-Farbwerten. Sie können bis zu acht 32-Bit-Farbwerte in den Shader übernehmen und sie werden automatisch in vec4-Werte von 0 - 1 umgewandelt. Dazu müssen Sie jedoch den Namen der Shader-Konstante verwenden in_Colour, für eine einzelne Farbe, oder in_Colour0, in_Colour1 usw... bis zu in_Colour7 für mehrere Eingabewerte. Die folgenden Beispiele zeigen, wie dies im Shader selbst aussehen würde:

   //Single colour
   attribute vec2 in_Position;
   attribute vec4 in_Colour;
   attribute vec2 in_TextureCoord;

   //Multiple colours
   attribute vec2 in_Position;
   attribute vec4 in_Colour0;
   attribute vec4 in_Colour1;
   attribute vec2 in_TextureCoord;

Beachten Sie auch, dass Sie, wenn Sie komprimierte Normalen usw. verwenden möchten, ein 4-Byte-Farbformat verwenden, dann müssen Sie auch das gleiche "in_Colour" -Format verwenden, wie oben gezeigt.

HINWEIS: Alle Uniformen müssen nach dem Aufruf der Funktion gesetzt werden shader_set() und bevor Sie anrufen shader_reset.
WICHTIG! Diese Funktion ist mit der Testlizenz des Produkts nicht verfügbar.


Beispiel:

shader_set(shader_Glass);
shader_params = shader_get_uniform(shader_glass, "u_vParams");
shader_set_uniform_f(shader_params, 0.1, 0.8, 0.25);
draw_self();
shader_reset();

Der obige Code erhält das Handle der Shader-Konstante "u_vParams" (a vec3 ), dann setze diese Konstante auf die angegebenen Werte.
