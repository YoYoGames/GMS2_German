instance_furest


Beschreibung

Diese Funktion überprüft alle Instanzen des gegebenen Objekts, um zu sehen, welches am weitesten von dem gegebenen Ursprungspunkt entfernt ist. Wenn keine Instanzen des Objekts vorhanden sind, gibt die Funktion das Schlüsselwort noone zurück. Wenn jedoch Instanzen vorhanden sind, wird die Funktion zurückgegeben id der gefundenen Instanz. Beachten Sie, dass die Instanz, auf der der Code ausgeführt wird, als Instanz des zu prüfenden Objekts erstellt wurde.


Syntax:

instance_furthest(x, y, obj);


Argumente:

Streit Beschreibung
x Die x-Position, um nach weit entfernten Instanzen zu suchen.
y Die y-Position, um nach weit entfernten Instanzen zu suchen.
obj Das Objekt, das auf Instanzen von.


Kehrt zurück:

Real (either an instance ID value or the keyword noone)


Beispiel:

var inst;
inst = instance_furthest(x, y, object_index);
if inst != id
   {
   draw_line(x, y, inst.x, inst.y);
   }

Der obige Code findet die am weitesten entfernte Instanz, die denselben Objekt-Index teilt wie die Instanz, auf der der Code ausgeführt wird, und speichert die ID in einer Variablen. Diese Variable wird dann überprüft, um festzustellen, ob sie mit der ID der aufrufenden Instanz identisch ist. Ist dies nicht der Fall, wird eine Linie zwischen den beiden Instanzen gezogen.
