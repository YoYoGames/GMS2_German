instance_create_depth


Beschreibung

Mit dieser Funktion können Sie eine neue Instanz des angegebenen Objekts an jedem beliebigen Punkt im Raum und in der angegebenen Tiefe erstellen. Die Tiefe kann ein beliebiger Wert sein, wobei je niedriger die Tiefe ist, je näher die Kamera an die Dinge gezogen wird und je höher die Tiefe, desto weiter entfernt wird eine Instanz in der Tiefe -200 über eine Instanz in der Tiefe von +300 gezogen ( beispielsweise). Beachten Sie, dass diese Funktion tatsächlich eine Raumschicht für die Instanz erstellen, da alle Instanzen auf einer Ebene im Raum sein muss, aber da dies eine verwaltete Schicht (dh: nicht eine, die Sie durch den Code oder in dem Raum erstellt haben, aber eine, die GameMaker Studio 2 automatisch erstellt hat) Sie können nicht darauf zugreifen, und die layer Instanzvariable gibt -1 zurück.

WICHTIG! Es gibt eine minimale und maximale Ebenentiefe von -16000 bis 16000. Alles, was sich auf einer Ebene außerhalb dieses Bereichs befindet, wird nicht gezeichnet, obwohl alle Ereignisse weiterhin normal ausgeführt werden.

Diese Funktion gibt die id der neuen Instanz, die dann in einer Variablen gespeichert oder für den Zugriff auf diese Instanz verwendet werden kann. Beachten Sie, dass diese Funktion auch das Create-Ereignis der Instanz aufruft, die gerade erstellt wird, bevor Sie mit dem Code oder den Aktionen für das Ereignis fortfahren, das die Funktion aufgerufen hat.


Syntax:

instance_create_depth(x, y, depth, obj);


Argumente:

Streit Beschreibung
x Die x-Position, an der das Objekt erstellt wird
y Die y-Position, an der das Objekt erstellt wird
depth Die Tiefe, der die erstellte Instanz zugewiesen werden soll
obj Der Objektindex des Objekts, für das eine Instanz erstellt werden soll


Kehrt zurück:

Real (instance ID value)


Beispiel:

var inst;
inst = instance_create_depth(x, y, -10000, obj_Bullet);
with (inst)
   {
   speed = other.shoot_speed;
   direction = other.image_angle;
   }

Der obige Code erstellt eine neue Instanz des Objekts "obj_Bullet" und speichert die Instanz-ID in einer Variablen. Diese Variable wird dann zur Zuweisung verwendet speed und direction zu der neuen Instanz.
