instance_place_list


Beschreibung

Mit dieser Funktion können Sie eine Position mit allen Instanzen eines Objekts auf Kollision prüfen, indem Sie die Kollisionsmaske der Instanz verwenden, die den Code für die Überprüfung ausführt. Wenn Sie dies verwenden, fordern Sie GameMaker Studio 2 effektiv auf, die Instanz an die neue Position zu verschieben, auf eine Kollision zu prüfen, zurück zu gehen und Ihnen mitzuteilen, ob eine Kollision gefunden wurde oder nicht. Dies funktioniert für präzise Kollisionen, jedoch nur, wenn sowohl die Instanz als auch die Instanzen des Objekts, für das geprüft wird, präzise Kollisionsmasken ausgewählt haben, andernfalls werden nur Kollisionen mit Begrenzungsrahmen angewendet. Die Funktion füllt eine (zuvor erstellte) DS-Liste mit der eindeutigen Liste id Werte aller Instanzen des Objekts, die als Kollision mit der aufrufenden Instanz betrachtet werden, und Sie haben die Möglichkeit, die Liste anhand der Entfernung vom Ursprung der Instanz, die die Prüfung durchführt, bis zum Ursprung der gefundenen Instanzen zu sortieren in einer Kollision. Beachten Sie, dass die Funktion auch das spezielle Schlüsselwort akzeptiert all In diesem Fall werden alle Fälle aufgelistet, bei denen eine Kollision festgestellt wurde. Die Funktion gibt die Anzahl der gefundenen Instanzen zurück oder 0, wenn keine gefunden wurde.

Beachten Sie, dass die angegebenen x / y-Koordinaten vor der Überprüfung auf die nächste ganze Zahl gerundet werden. Wenn dies nicht der Fall ist , was Sie benötigen (oder Sie ein älteres GameMaker Produkt verwendet haben), sollten Sie dies tun floor die Koordinaten im Scheck: instance_place_list(floor(x), floor(y), obj_Enemy, list, true).


Syntax:

instance_place_list(x, y, obj, list, ordered);

Streit Beschreibung
x Die x-Position, die auf Instanzen geprüft werden soll.
y Die y-Position, um nach Instanzen zu suchen.
obj Das Objekt, das auf Instanzen von überprüft werden soll.
list Die DS-Liste, die zum Speichern der IDs kollidierender Instanzen verwendet werden soll.
ordered Ob die Instanzen nach Entfernung vom Ursprung geordnet sein sollen oder nicht.


Kehrt zurück:

Int (The number of instances found to be in collision)


Beispiel:

var _list = ds_list_create();
var _num = instance_place_list(x, y, obj_Enemy, _list, false);
if _num > 0
    {
    for (var i = 0; i < _num; ++i;)
        {
        instance_destroy(_list[| i]);
        }
    }
ds_list_destroy(_list);

Der obige Code prüft auf Kollision mit allen Instanzen, die an der Position der aufrufenden Instanzen gefunden werden. Diese werden zu einer DS-Liste hinzugefügt, die dann durchgeschleift wird, um jede der Kollisionen zu zerstören.
