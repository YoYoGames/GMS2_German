instance_place


Beschreibung

Mit dieser Funktion können Sie eine Position für eine Kollision mit einer anderen Instanz oder alle Instanzen eines Objekts mithilfe der Kollisionsmaske der Instanz prüfen, die den Code für die Prüfung ausführt. Wenn Sie dies verwenden, fordern Sie GameMaker Studio 2 effektiv auf, die Instanz an die neue Position zu verschieben, auf eine Kollision zu prüfen, zurück zu gehen und Ihnen mitzuteilen, ob eine Kollision gefunden wurde oder nicht. Dies funktioniert für präzise Kollisionen, jedoch nur, wenn sowohl die Instanz als auch das Objekt, auf das geprüft wird, präzise Kollisionsmasken ausgewählt haben, andernfalls werden nur Kollisionen mit Begrenzungsrahmen angewendet. Diese Funktion gibt die eindeutige Instanz zurück id Wenn das Objekt nicht mehr benötigt wird, ist es etwas schneller, die Funktion zu verwenden place_meeting. Diese Funktion akzeptiert auch die Sonderschlüsselwörter alle und andere und gibt das Schlüsselwort noone zurück, wenn keine Kollision auftritt, oder den eindeutigen Instanz-ID-Wert der Instanz, wenn eine Kollision auftritt.

Beachten Sie, dass die angegebenen x / y-Koordinaten vor der Überprüfung auf die nächste ganze Zahl gerundet werden. Wenn dies nicht der Fall ist , was Sie benötigen (oder Sie ein älteres GameMaker Produkt verwendet haben), sollten Sie dies tun floor die Koordinaten im Scheck: instance_place(floor(x), floor(y), obj_Enemy).


Syntax:

instance_place(x, y, obj);


Argumente:

Streit Beschreibung
x Die x-Position, die auf Instanzen geprüft werden soll.
y Die y-Position, um nach Instanzen zu suchen.
obj Das Objekt, das auf Instanzen von überprüft werden soll.


Kehrt zurück:

Real (either an instance ID value or the keyword noone)


Beispiel:

var _inst = instance_place(x, y, obj_Enemy);
if _inst != noone
   {
   hp -= _inst.dmg;
   instance_destroy(_inst);
   }

Der obige Code prüft auf eine Kollision mit Instanzen von "obj_Enemy", und wenn es einen gibt, wird die Variable "hp" um den in der Variablen "dmg" der kollidierenden Instanz gespeicherten Betrag reduziert und anschließend die kollidierende Instanz zerstört.
