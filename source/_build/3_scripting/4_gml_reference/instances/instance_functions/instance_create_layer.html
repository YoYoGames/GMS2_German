instance_create_layer


Beschreibung

Mit dieser Funktion können Sie eine neue Instanz des angegebenen Objekts an jedem beliebigen Punkt im Raum und auf der angegebenen Ebene erstellen. Der Layer kann anhand des Layer-ID-Werts identifiziert werden (wie von der Funktion zurückgegeben) layer_create() ) oder durch den Namen der Ebene (als String, zum Beispiel "instance_layer") wie im Raumeditor definiert. Diese Funktion gibt die id der neuen Instanz, die dann in einer Variablen gespeichert oder für den Zugriff auf diese Instanz verwendet werden kann. Beachten Sie, dass diese Funktion auch das Create-Ereignis der Instanz aufruft, die gerade erstellt wird, bevor Sie mit dem Code oder den Aktionen für das Ereignis fortfahren, das die Funktion aufgerufen hat.

WICHTIG! Es gibt eine minimale und maximale Ebenentiefe von -16000 bis 16000. Alles, was sich auf einer Ebene außerhalb dieses Bereichs befindet, wird nicht gezeichnet, obwohl alle Ereignisse weiterhin normal ausgeführt werden.


Syntax:

instance_create_layer(x, y, layer_id, obj);


Argumente:

Streit Beschreibung
x Die x-Position, an der das Objekt erstellt wird
y Die y-Position, an der das Objekt erstellt wird
layer_id Die Layer-ID (oder der Name), der die erstellte Instanz zugewiesen werden soll
obj Der Objektindex des Objekts, für das eine Instanz erstellt werden soll


Kehrt zurück:

Real (instance ID value)


Beispiel:

var inst;
inst = instance_create_layer(x, y, "instance_layer", obj_Bullet);
with (inst)
   {
   speed = other.shoot_speed;
   direction = other.image_angle;
   }

Der obige Code erstellt eine neue Instanz des Objekts "obj_Bullet" und speichert die Instanz-ID in einer Variablen. Diese Variable wird dann zur Zuweisung verwendet speed und direction zu der neuen Instanz.
