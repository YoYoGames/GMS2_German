Instanzposition


Beschreibung

Mit dieser Funktion können Sie eine Position auf eine Kollision mit einer anderen Instanz oder auf alle Instanzen eines Objekts überprüfen. Wenn Sie dies verwenden, prüfen Sie einen einzelnen Punkt im Raum auf eine Instanz oder ein Objekt. Diese Überprüfung wird für die Begrenzungsbox der Instanz oder für die Maske der Instanz durchgeführt, wenn für diese Instanz präzise Kollisionen aktiviert sind und die eindeutige Instanz zurückgegeben wird id. Wenn Sie die ID der kollidierenden Instanz nicht benötigen, sollten Sie sie verwenden position_meeting stattdessen. Diese Funktion akzeptiert auch die speziellen Schlüsselwörter all und andere und gibt das Schlüsselwort noone zurück, wenn keine Kollision auftritt oder der eindeutige ID-Wert der Instanz, die bei einer Kollision gefunden wird.


Syntax:

instance_position( x, y, obj );


Argumente:

Streit Beschreibung
x Die x-Position, um nach Instanzen zu suchen.
y Die y-Position, um nach Instanzen zu suchen.
obj Das Objekt, das auf Instanzen von.


Kehrt zurück:

Real (either an instance ID value or the keyword noone)


Beispiel:

var inst;
inst = instance_position(mouse_x, mouse_y, obj_Pause_Button);
if inst != noone
   {
   with (inst) image_index=1;
   instance_create_layer(room_width / 2, 0, "Controllers", obj_Menu);
   }

Der obige Code wird auf eine Kollision mit einer Instanz von "obj_Pause_Button" an der Mausposition prüfen, und wenn es einen gibt, wird er dann die zurückgegebene ID verwenden, um seinen image_index auf einen neuen Wert zu setzen, bevor eine neue Instanz des Objekts "obj_Menu" erstellt wird ".
