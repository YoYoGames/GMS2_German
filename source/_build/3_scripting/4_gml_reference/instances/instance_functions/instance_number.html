Instanznummer


Beschreibung

Mit dieser Funktion können Sie herausfinden, wie viele aktive Instanzen des angegebenen Objekts im Raum existieren. Bitte beachten Sie, dass diejenigen Instanzen, die mit der Option deaktiviert wurden, nicht aktiviert sind instance_deactivate Funktionen werden bei dieser Prüfung nicht berücksichtigt.


Syntax:

instance_number(obj);


Argumente:

Streit Beschreibung
obj Das Objekt, um die Anzahl der Instanzen von.


Kehrt zurück:

Real


Beispiel:

if instance_number(object_index) < 50    {
   instance_create_layer(random(room_width), random(room_height), "Instances", object_index);
   }

Der obige Code prüft die Anzahl der Instanzen, die vom selben Objekt wie die aktuelle Instanz erstellt werden, und erstellt dann, wenn weniger als 50 vorhanden sind, einen anderen an einer zufälligen Position innerhalb des Raums.
