instance_deactivate_object


Syntax:

instance_deactivate_object(obj);


Streit Beschreibung
obj Das zu deaktivierende Objekt oder die Instanz (das Schlüsselwort all kann ebenfalls verwendet werden).


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie eine einzelne Instanz oder alle Instanzen eines bestimmten Objekts von allen zuvor aktivierten deaktivieren. Beachten Sie, dass Sie, wenn Sie eine Instanz oder ein Objekt, das als Persistent gekennzeichnet wurde, deaktiviert haben, es erneut mit der Funktion reaktivieren müssen instance_activate_object vor dem Umkleiden, sonst wird es nicht übernommen und stattdessen verworfen.

HINWEIS: Wenn Sie eine Instanz beim Raumstart deaktivieren (dh aus dem Raumerstellungscode oder aus einem Instanzenereignis einer Instanz im Raum), führen alle Instanzen, die vom Raumeditor aus im Raum platziert sind, ihr create-Ereignis noch aus bevor sie deaktiviert werden.

WARNUNG: Das Deaktivieren von Instanzen, bei denen die Physik aktiviert ist, verhindert NICHT, dass ihre Fixtures in der Physiksimulation interagieren. Dafür sollten Sie ihre einstellen phy_active Variable zu true oder false wenn Sie die Instanzen aktivieren / deaktivieren.


Beispiel:

instance_deactivate_object(obj_Enemy);
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_activate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false);

Der obige Code deaktiviert alle Instanzen des Objekts "obj_Enemy" und aktiviert dann eine Region innerhalb des Raums.
