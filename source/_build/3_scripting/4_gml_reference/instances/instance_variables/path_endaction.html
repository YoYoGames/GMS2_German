Pfad_endaktion


Beschreibung

Diese Variable kann verwendet werden, um die Reaktion einer Instanz zu erhalten oder zu ändern, wenn sie das Ende des aktuellen Pfads erreicht. Normalerweise würden Sie dies einstellen, wenn Sie den Pfad mit starten path_start Sie können dieses Verhalten jedoch abhängig von einer beliebigen Anzahl von Ereignissen in Ihrem Spiel ändern. Die verfügbaren Werte werden mit den folgenden Konstanten ausgedrückt:
Konstante Beschreibung
path_action_stop Stoppen Sie den Pfad.
path_action_restart Fortsetzen von der Startposition, Springen zum Start, wenn der Pfad nicht geschlossen ist.
path_action_continue Starten Sie den Pfad erneut von der aktuellen Position.
path_action_reverse Kehren Sie die Geschwindigkeit des Pfades um (führen Sie den Pfad umgekehrt aus).


Syntax:

path_endaction;


Kehrt zurück:

Constant (see table above)


Beispiel:

if path_endaction == path_action_stop
   {
   path_endaction = choose(path_action_restart, path_action_continue, path_action_reverse);
   }

Der obige Code prüft die Pfad-Ende-Aktion und wenn er auf Stopp gesetzt ist, wird eine neue Pfad-Ende-Aktion zufällig ausgewählt.
