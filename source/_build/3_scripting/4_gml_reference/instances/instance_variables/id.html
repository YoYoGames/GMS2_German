Ich würde


Beschreibung

Diese schreibgeschützte Variable enthält die eindeutige Identifikationsnummer für die Instanz. Jede Instanz, die Sie erstellen, erhält eine Nummer, die intern verwendet wird, um diese Instanz zu identifizieren, und mit dieser Variablen können Sie sie referenzieren. Die ID wird auch zurückgegeben (und kann in einer Variablen gespeichert werden), wenn eine Instanz mit erstellt wird instance_create_layer.
HINWEIS: Der Wert dieser Variablen ist nicht identisch mit dem Bezeichner, den die Instanz im Raumeditor erhält, und unterscheidet sich auch von der ID instance_id Array, das alle IDs aller momentan aktiven Instanzen enthält.


Syntax:

id;


Kehrt zurück:

Real (instance ID value)


Beispiel:

for(var i = 0; i < instance_count; i++;)
   {
   if instance_id[i] != id
      {
      instance_id[i].scr += 5;
      }
   }

Der obige Code addiert 5 zur "scr" -Variable für jede aktive Instanz in dem Raum mit Ausnahme derjenigen, die den Code ausführt. Dazu werden ALLE aktiven Instanzen durchgeschleift (indem das Array "instance_id" verwendet wird, um jede aktive Instanz-ID zurückzugeben) und mit der integrierten ID-Variablen verglichen, die die ID der ursprünglichen Instanz darstellt, in der der Code ausgeführt wird.
