lebt


Syntax:

lives;


Kehrt zurück:

Real


Beschreibung

Diese eingebaute Instanzvariable wird verwendet, um einen numerischen Wert für die Objektlebensdauer zu speichern (beachten Sie, dass dies nicht notwendigerweise immer der Fall ist). Nur weil sie "Leben" genannt wird, heißt das nicht, dass sie für die Leben verwendet werden muss kann einen reellen oder ganzzahligen Wert enthalten, den Sie für Ihr Spiel wünschen).


Beispiel:

if place_meeting(x, y, obj_Enemy_Bullet)
   {
   health -= 10;
   if health <= 0 lives -=1;
   }

Der obige Code prüft die aktuelle Instanz, um festzustellen, ob eine Kollision mit der Instanz des in "obj_Enemy_Bullet" indizierten Objekts vorliegt. Wenn dies der Fall ist, werden 10 Punkte aus der Integritätsvariablen entfernt, und dieser Wert wird überprüft, um zu sehen, ob er niedriger ist als oder gleich 0. Wenn dies der Fall ist, entfernt die Instanz 1 aus der Lives-Variable.
