Beschreibung

Diese integrierte Variable wird für jede Instanz in einem Raum erstellt und enthält den Layer-ID-Wert des Layers, dem die Instanz zugewiesen ist. Dieser Wert kann dann in anderen Funktionen wie verwendet werden layer_get_depth oder es kann geändert werden, um die Instanz auf eine andere Ebene zu verschieben. Wenn jedoch die zugewiesene Ebene im aktuellen Raum nicht vorhanden ist, wird ein Fehler angezeigt, der das Schließen des Spiels erzwingt. Beim Zuweisen eines Layers müssen Sie die eindeutige Layer-ID angeben, die von der Funktion zurückgegeben wird layer_get_id() (wenn benannte Raumebenen verwendet werden) oder wie von der Funktion zurückgegeben layer_create() (wenn Sie zur Laufzeit eigene Layer erstellen).

WICHTIG! wenn Sie die Instanz mit dem erstellt haben instance_create_depth() Funktion oder haben die depth Variabel wird die der Instanz zugewiesene Ebene zu einer "verwalteten" Ebene, die GameMaker Studio 2 automatisch steuert und verwaltet. In diesen Fällen layer Variable gibt -1 zurück.


Syntax:

layer;


Kehrt zurück:

Layer ID value (Real)


Beispiel:

layer = layer_create(-1000);

Der obige Code erstellt eine neue Ebene mit einer Tiefe von -1000 und legt dann die Instanz fest layer Variable zur zurückgegebenen Layer-ID und verschiebt die Instanz von der Ebene, auf der sie sich gerade befindet, zur neuen Ebene, die gerade erstellt wird.
