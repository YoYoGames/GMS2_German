Instanzvariablen


Wenn Sie ein neues Objekt erstellen, werden bestimmte Variablen bereits mit den Standardwerten initialisiert. Diese Variablen nennen wir Instanzvariablen, da sie für jede Instanz des Objekts in Ihren Spielräumen erstellt werden. Sobald eine Instanz erstellt wurde, sind die Werte dieser Variablen für diese Instanz und nur für diese Instanz eindeutig. Einige dieser Variablen werden Sie häufig verwenden, wie x und y, während andere weniger benötigt werden, je nachdem, was Sie tun möchten, aber im Allgemeinen sind sie sehr praktisch und wenn möglich, sollten Sie sie verwenden, anstatt eigene benutzerdefinierte Variablen zu erstellen.

Die meisten Instanzvariablen können sowohl gesetzt als auch gelesen werden. Sie können die Eigenschaften und das Verhalten einer Instanz ändern, indem Sie einfach den Wert einer bestimmten Variablen anpassen. Sie können beispielsweise verhindern, dass eine Instanz gezeichnet wird, indem Sie einfach die Option visible eingebaute Variable zu false. Im Folgenden finden Sie die verschiedenen Variablen, die für alle Instanzen aller Objekte in Ihrem Spiel initialisiert werden.

Diese Variablen befassen sich mit allgemeinen Instanzeigenschaften:
  • Ich würde
  • sichtbar
  • solide
  • beständig
  • Tiefe
  • Schicht
  • Alarm
  • Objekt_index

Diese Variablen beziehen sich alle auf das der Instanz zugewiesene Sprite und können verwendet werden, um zu ändern, was gezeichnet wird und wie:
  • sprite_index
  • sprite_width
  • sprite_height
  • sprite_xoffset
  • sprite_yoffset
  • Bild_alpha
  • Bild_Winkel
  • Bild_Mischung
  • Bildindex
  • Bildnummer
  • Bildgeschwindigkeit
  • image_xscale
  • Bild_Skalierung

Diese Variablen befassen sich mit der Kollisionsmaske:
  • Maskenindex
  • bbox_bottom
  • bbox_left
  • bbox_recht
  • bbox_top

Diese Variablen befassen sich mit der Instanzposition und -bewegung:
  • Richtung
  • Reibung
  • Schwere
  • Schwerkraft_Richtung
  • hspeed
  • vspeed
  • Geschwindigkeit
  • xstart
  • ystart
  • x
  • y
  • xvorhergehend
  • früher

Diese Variablen beziehen sich auf Pfade und wie die Instanz mit einem interagiert, falls zugewiesen:
  • Pfad_Index
  • path_position
  • path_positionvorhergehend
  • Pfad_Geschwindigkeit
  • path_scale
  • Pfad_Orientierung
  • Pfad_endaktion

Diese Variablen dienen zum Festlegen einer Instanz für die Verwendung einer Zeitachse:
  • timeline_index
  • timeline_running
  • timeline_speed
  • timeline_position
  • timeline_loop

  • Physik
