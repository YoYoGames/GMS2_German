sprite_xoffset


Beschreibung

Diese schreibgeschützte Variable gibt das lokale xoffset (die x-Komponente des im Sprite-Editor definierten Ursprungs) des Sprites zurück, das der Instanz zugewiesen wurde. Dieser xoffset wird in Pixel zurückgegeben und ist abhängig von image_xscale, so dass (zum Beispiel) wenn ein quadratisches 32x32-Sprite mit dem x-Ursprung bei 16 definiert ist, wenn dieses Sprite um 2 skaliert wird, diese Variable 32 zurückgibt. Wenn Sie den unskalierten xoffset benötigen, sollten Sie verwenden sprite_get_xoffset.

Informationen zu den Sprites finden Sie hier.


Syntax:

sprite_xoffset;


Kehrt zurück:

Real


Beispiel:

if x - sprite_xoffset < 0
   {
   x = sprite_xoffset;
   }

Der obige Code überprüft, ob sich die linke Kante des Sprites der Instanz außerhalb des Raums befindet. Wenn dies der Fall ist, wird die Instanz so verschoben, dass das gesamte Sprite sichtbar ist.
