beständig


Beschreibung

Diese Variable kann gelesen werden, um herauszufinden, ob die Instanz als markiert ist persistent oder nicht, oder es kann verwendet werden, um Persistenz zu setzen true (anhaltend) oder false (nicht persistent) für die Instanz. Eine persistente Instanz ist eine Instanz, die von Raum zu Raum "übertragen" wird, was bedeutet, dass sie zum Beispiel nur einmal zu Beginn des Spiels erstellt werden muss und in allen weiteren Räumen vorhanden sein wird. Vorsicht ist geboten bei Beharrlichkeit, da es leicht ist, den Überblick über persistente Instanzen zu verlieren, was später zu Problemen in der Entwicklung des Spiels führen kann.

Ein ständiges Objekt wird noch sein Spiel Start, Spielende, Zimmer Start- und Zimmer End Ereignisse ausgelöst, aber wenn Sie das Spiel neu starten (mit zum Beispiel des game_restart() Funktion) werden alle persistenten Objekte entfernt und existieren nur, wenn sie vom Spiel neu erstellt werden. Beachten Sie außerdem, dass wenn Sie ein persistentes Objekt deaktivieren, es nicht mehr von einem Raum in einen anderen übergeht, es sei denn, es wird wieder aktiviert, bevor das Ereignis Raumende ausgelöst wird.


Syntax:

persistent;


Kehrt zurück:

Boolean


Beispiel:

if !persistent
   {
   persistent = true;
   }

Der obige Code überprüft die Instanz, um festzustellen, ob die Persistenz als markiert ist false und wenn die Instanz NICHT persistent ist, wird das Persistenz-Flag auf gesetzt true.
