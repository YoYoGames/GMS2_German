Bild_Winkel


Beschreibung

Dieser Wert legt den Winkel (Drehung) des Sprites fest und wird in Grad gemessen, wobei der rechte Wert 0º beträgt, wobei der Wert 90º beträgt, wobei der linke Wert 180º und der fallende Wert 270º beträgt. Setzen Sie diese Variable auf 0, um das zu zeichnende Sprite wie im Sprite-Editor definiert zurückzusetzen. Bitte beachten Sie, dass für Änderungen in dieser Variablen, die sichtbar sein sollen, die Instanz entweder kein Zeichenereignis haben sollte (und deshalb GameMaker Studio 2 standardmäßig das Sprite zeichnet) oder mit einer der erweiterten Zeichenfunktionen wie gezeichnet wird draw_self() oder draw_sprite_ext().


Syntax:

image_angle;


Kehrt zurück:

Real


Beispiel:

image_angle = point_direction(x, y, mouse_x, mouse_y);

Der obige Code wird das Sprite der Instanz so drehen, dass es immer auf die Mausposition zeigt.
