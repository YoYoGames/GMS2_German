path_scale


Beschreibung

Dieser Wert kann verwendet werden, um den Maßstab des aktuell zugewiesenen Pfads für die Instanz mit einem Standardwert von 1 zu ermitteln oder festzulegen. Dies ist ein Skalarwert. Daher ist 1 ein Maßstab von 1: 1, während der Wert 2 ist. Zum Beispiel wird das Doppelte der Skala und die Einstellung auf 0,5 wäre die Skalierung zu halbieren.

Funktionen zu Pfaden finden Sie hier.


Syntax:

path_scale;


Kehrt zurück:

Real


Beispiel:

path_scale = 1 + random(2);

Der obige Code wird die Skalierung des aktuellen Pfads auf eine zufällige Größe zwischen 1 (Originalgröße) und 3 (3-fache Originalgröße) einstellen.
