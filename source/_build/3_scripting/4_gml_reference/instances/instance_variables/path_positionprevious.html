path_positionvorhergehend


Beschreibung

Diese Variable kann verwendet werden, um die Position einer Instanz entlang ihres aktuellen Pfads im vorherigen Schritt zu ermitteln oder festzulegen, und ist ein normalisierter Wert zwischen 0 und 1, dh: 0 ist die Startposition des Pfads und 1 wäre die Endposition. Es ist ähnlich wie die xprevious und yprevious funktioniert in, wie es funktioniert, nur ist es für Pfade spezifisch. Dies kann nützlich sein, wenn ein Pfadfolger vorübergehend gestoppt wird, wenn etwas im Weg ist (siehe Beispielcode unten).

Funktionen zu Pfaden finden Sie hier.


Syntax:

path_positionprevious;


Kehrt zurück:

Real


Beispiel:

var xx = x + lengthdir_x(speed, direction);
var yy = y + lengthdir_y(speed, direction);
if collision_circle(xx, yy, 16, obj_Player, false,true) path_position = path_positionprevious;

Der obige Code prüft einen Bereich vor der Instanz auf eine Kollision mit dem Objekt "obj_Player", und falls es einen gibt, setzt er die Instanz zurück auf die vorherige Position, die sie auf dem aktuellen Pfad belegte.
