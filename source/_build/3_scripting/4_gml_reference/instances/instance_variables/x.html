x


Beschreibung

Der x-Wert einer Instanz ist die horizontale Position im aktuellen Raum, gemessen in Pixeln. Dieser Wert kann entweder 0, positiv oder negativ sein, wobei 0 die linke Seite des Raums ist und die Bewegung nach rechts x erhöht wird, die Bewegung nach links wird um x verringert (ein negativer Wert für x bedeutet, dass die Instanz die linke Seite des Raums verlassen hat) ). Sie können xa sogar einen echten Wert wie 12.345 geben, aber da Sie keine.345 Pixel erhalten, scheint sich die Instanz nicht verschoben zu haben, obwohl diese Werte von GameMaker Studio 2 und intern verwendet werden und absolut gültig sind.

Indem Sie die x (und y) Werte einstellen, können Sie das Objekt um den Raum zu der Position Ihrer Wahl springen lassen, oder Sie können sie um kleinere Beträge addieren und subtrahieren, um die Illusion von Bewegung zu erzeugen, ohne das eingebaute zu verwenden speed und direction Funktionen. Dies sollte berücksichtigt werden, wenn Sie Ihre Spiele so oft machen, dass eine Art von Bewegung sich mehr für einen bestimmten Spielstil eignet als für einen anderen.


Syntax:

x;


Kehrt zurück:

Real


Beispiel:

if keyboard_check(vk_left) x += 5;
if keyboard_check(vk_right) x -= 5;

Der obige Code wird die Pfeiltasten verwenden, um die X-Position hinzuzufügen oder zu subtrahieren und so die Illusion der Bewegung zu erzeugen.
