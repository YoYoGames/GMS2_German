sprite_index


Beschreibung

Diese Variable gibt den Index des aktuellen Sprites für die Instanz zurück oder -1, wenn der Instanz kein Sprite zugeordnet ist. Sie können es ändern, um der Instanz ein anderes Sprite zu geben, indem Sie ihm den Namen eines Sprites aus dem Ressourcenbaum geben oder indem Sie eine Variable verwenden, in die ein extern geladenes Sprite indiziert ist. Das Ändern des Sprites ändert nicht den Index des aktuell sichtbaren Rahmens. Wenn Sie also das Sprite auf Bild 3 ändern, wird das neue Sprite mit diesem Rahmen gezeichnet (vorausgesetzt, es hat die gleiche Anzahl von Rahmen).

Informationen zu den Sprites finden Sie hier.


Syntax:

sprite_index;


Kehrt zurück:

Real


Beispiel:

with (obj_Check)
   {
   if !collision_line(x, y, other.x, other.y, obj_Wall, false, true)
      {
      sprite_index = spr_spotted;
      }
   else
      {
      sprite_index = spr_clear;
      }
   }

Der obige Code durchläuft alle Instanzen von "obj_Check", wobei er nach einer Kollisionslinie zwischen ihnen und der Instanz sucht, die den Code ausführt. Das Sprite dieser Instanzen wird abhängig vom Rückgabewert (true oder false) für die Kollisionslinie geändert.
