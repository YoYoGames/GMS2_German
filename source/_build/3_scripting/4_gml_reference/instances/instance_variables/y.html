y


Beschreibung

Der y-Wert einer Instanz ist die vertikale Position im aktuellen Raum, gemessen in Pixeln. Dieser Wert kann entweder 0, positiv oder negativ sein, wobei 0 die Oberkante des Raums ist und die Abwärtsbewegung erhöht sich um y, die Aufwärtsbewegung verringert sich um y (ein negativer Wert für y bedeutet, dass die Instanz außerhalb des oberen Bereichs des Raums ist). Sie können sogar einen echten Wert wie 12.345 angeben, aber da Sie keine.345 Pixel erhalten, scheint sich die Instanz nicht verschoben zu haben, obwohl diese Werte von GameMaker Studio 2 verwaltet und intern verwendet werden und absolut gültig sind.

Indem Sie die y (und x) Werte einstellen, können Sie das Objekt um den Raum zu der Position Ihrer Wahl springen lassen, oder Sie können sie um kleinere Beträge addieren und subtrahieren, um die Illusion von Bewegung zu erzeugen, ohne das eingebaute zu verwenden speed und direction Funktionen. Dies sollte berücksichtigt werden, wenn Sie Ihre Spiele so oft machen, dass eine Art von Bewegung sich mehr für einen bestimmten Spielstil eignet als für einen anderen.


Syntax:

y;


Kehrt zurück:

Real


Beispiel:

if keyboard_check(vk_down) y += 5;
if keyboard_check(vk_up) y -= 5;

Der obige Code wird die Pfeiltasten verwenden, um die Y-Position zu addieren oder zu subtrahieren und so die Illusion der Bewegung zu erzeugen.
