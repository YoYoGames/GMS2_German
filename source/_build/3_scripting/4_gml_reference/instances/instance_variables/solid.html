solide

Beschreibung

Eine Instanz kann durch die Objekteigenschaften im Objekt-Editor als solide markiert werden oder durch Ändern des Werts dieser integrierten Variable. Ob solid ist eingestellt auf true Wenn eine Kollision erkannt wird, wird die Kollisionsinstanz automatisch an die Position zurückversetzt, die sie im Schritt vor der Kollision hatte (und dann werden alle Codes oder Aktionen im Kollisionsereignis ausgeführt). Wenn es auf eingestellt ist false, alle Positionierung muss durch das Kollisionsereignis behandelt werden.


Syntax:

solid;


Kehrt zurück:

Boolean


Beispiel:

if other.visible = true
   {
   x = xprevious;
   y = yprevious;
   }

Der obige Code prüft das sichtbare Flag der "anderen" Instanz in einer Kollision und wenn es auf gesetzt ist true Verschiebe diese Instanz zurück an ihre vorherige Position.
