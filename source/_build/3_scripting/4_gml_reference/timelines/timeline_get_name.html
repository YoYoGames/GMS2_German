timeline_get_name

Ruft den Namen einer bestimmten Zeitlinie ab.

Syntax:

timeline_get_name( ind );


Streit Beschreibung
ind Der Index der Zeitlinie zum Überprüfen des Namens von.


Rückgabe: Zeichenfolge


Beschreibung

Diese Funktion kann verwendet werden, um den Namen einer Zeitlinie als String zu erhalten. wenn die Zeitleiste dynamisch mit dem erstellt wurde timeline_add Funktion hat der zurückgegebene Name das Format "__newtimeline N " wobei " N " die Nummer der Zeitlinie ist (beginnend mit 0). Bitte beachten Sie, dass dies nur eine Zeichenkette ist und nicht verwendet werden kann, um direkt auf die Zeitleiste zu verweisen - dafür benötigen Sie den Zeitleistenindex. Sie können diese Zeichenfolge jedoch verwenden, um den Zeitachsenindex mithilfe der zurückgegebenen Zeichenfolge zusammen mit der Funktion abzurufen asset_get_index().


Beispiel:

str = timeline_get_name(global.tl);

Der obige Code speichert den Namen der Zeitlinie, die in der Variablen "global.tl" indiziert ist.
