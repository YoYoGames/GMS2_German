Zeitleisten

Zeitleisten sind ein mächtiger Mechanismus zur Steuerung von Ereignissen in deinem Spiel und basieren auf "Momenten", bei denen ein "Moment" nur ein einziger Spielschritt ist. Diese Zeitleisten werden im Allgemeinen aus dem Ressourcenbaum erstellt, indem Sie den Zeitachseneditor ähnlich wie ein Objekt verwenden, dh Sie erstellen eine Zeitleiste und dann in jedem Moment, in dem Sie eine Aktion ausführen müssen, fügen Sie einen Code oder ein DnD™ -Symbol hinzu es durchzuführen. Nach dem Erstellen können Sie Zeitachsen über den Code stoppen, starten und bearbeiten. Sie können sogar einzelne Momente modifizieren, indem Sie ein Skript definieren und es dynamisch von einem Controller-Objekt in die Zeitleiste einfügen (zum Beispiel), was es zu einem sehr mächtigen und vielseitigen Werkzeug beim Erstellen Ihrer Spiele macht.

Die folgenden Funktionen dienen zum Hinzufügen und Bearbeiten von Timelines in Ihrem Spiel:

  • timeline_exists
  • timeline_get_name
  • timeline_add
  • timeline_delete
  • timeline_moment_add_script
  • timeline_moment_clear
  • timeline_clear
  • timeline_größe
  • timeline_max_moment

Die folgenden Variablen sind für alle Instanzen "eingebaut", um das Arbeiten mit Zeitleisten zu erleichtern und Ihnen die Kontrolle über Folgendes zu geben:

  • timeline_index
  • timeline_running
  • timeline_speed
  • timeline_position
  • timeline_loop

Beachten Sie, dass Sie hier weitere Informationen zu Instanzvariablen finden: Instanzvariablen.
