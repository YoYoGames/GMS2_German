timeline_exists

Überprüft, ob eine bestimmte Zeitlinie existiert.

Syntax:

timeline_exists(ind);


Streit Beschreibung
ind Der Index der Zeitlinie, nach der gesucht werden soll.


Rückgabe: Boolean


Beschreibung

Mit dieser Funktion können Sie prüfen und sehen, ob eine Zeitleiste existiert (returns) true ) oder nicht (zurück false ). Dies ist besonders nützlich, wenn Sie Timelines dynamisch mithilfe von timeline_add Funktion, aber Sie sollten beachten, dass ein Fehler ausgelöst wird, wenn Sie nach einer nicht initialisierten Variablen suchen (die andernfalls dem Index einer Zeitlinie zugewiesen wäre).


Beispiel:

if timeline_exists(global.tl) timeline_delete(global.tl);

Der obige Code prüft, ob eine Zeitlinie existiert und in der globalen Variablen "t1" indiziert ist, und wenn dies der Fall ist, löscht er diese Zeitlinie.
