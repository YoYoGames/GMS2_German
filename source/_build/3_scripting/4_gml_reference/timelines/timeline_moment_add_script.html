timeline_moment_add_script

Fügt der Zeitleiste im angegebenen Schritt ein Skript hinzu.

Syntax:

timeline_moment_add_script(ind, step, script);


Streit Beschreibung
ind Der Index der Zeitlinie, um einen Moment hinzuzufügen.
step Der Moment (Schritt) zum Hinzufügen.
script Der Index des einzufügenden Skripts.


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie Skripten zu Timelines (die Skripte müssen bereits in den Spielressourcen vorhanden sein) dynamisch zu jedem beliebigen "Moment" innerhalb dieser Zeitlinie hinzufügen, wobei ein "Moment" einem Spielschritt (oder Schritt) entspricht. Auf diese Weise können Sie eine neue Zeitleiste mit dem erstellen timeline_add Funktionieren Sie und fügen Sie an einem beliebigen Punkt ein anderes Verhalten hinzu, oder ändern Sie einfach eine zuvor erstellte Zeitlinienressource mit neuen Verhaltensweisen. Beachten Sie, dass das Skript bei Verwendung dieser Funktion keine zusätzlichen Argumente erfordern kann.


Beispiel:

global.tl = timeline_add();
var i = room_speed * 60;
repeat(3)
   {
   timeline_moment_add_script(global.tl, i, choose(scr_AlienAttack_1, scr_AlienAttack_2, scr_AlienAttack_3);
   i += room_speed * 60;
   }

Der obige Code erstellt eine neue Zeitlinie und speichert seinen Index in der Variablen "global.tl". Es wird dann drei Skripte zu der zufällig ausgewählten Zeitlinie in Intervallen von einer Minute hinzufügen.
