GameMaker Studio 2 Referenzhandbuch


Dieser Abschnitt des Handbuchs ist ein Nachschlagewerk für GameMaker Studio 2 Language (GML). Hier finden Sie alle verfügbaren Funktionen sowie die erforderlichen Argumente und Codebeispiele, um zu zeigen, wie sie verwendet werden können. Informationen zur Verwendung von GML finden Sie im Abschnitt zur GML-Übersicht des Handbuchs.

In den folgenden Abschnitten werden die Funktionen erläutert, die Ihnen zur Bearbeitung der verschiedenen Ressourcen in Ihrem Projekt zur Verfügung stehen. Dazu gehören das Erstellen neuer dynamischer Ressourcen oder das Bearbeiten und Ändern der Ressourcen, die über den Ressourcenbaum zu Ihrem Spiel hinzugefügt werden.

GameMaker Studio 2 bietet zahlreiche Möglichkeiten, um Instanzen innerhalb Ihres Spielzimmers zu verschieben, von der Einstellung der Geschwindigkeits- und Richtungsvektoren bis hin zum einfachen "Platzieren" einer Instanz an einer bestimmten Position sowie der Verwendung ausgefeilterer Methoden wie Pfad- finden. Es gibt auch eine Reihe verschiedener Methoden zum Erkennen von Kollisionen zwischen zwei Instanzen. Welche Sie wählen, hängt stark von der Art des Spiels ab, die Sie erstellen, und von der genauen Situation, in der Sie die Kollisionsfunktionen verwenden müssen.
