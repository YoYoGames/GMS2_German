GameMaker Studio 2 Sprachreferenz


Dieser Abschnitt des Handbuchs ist ein Referenzhandbuch für die Sprache von GameMaker Studio 2 (GML). Sie können alle verfügbaren Funktionen, die hier dokumentiert sind, zusammen mit den erforderlichen Argumenten und Codebeispielen finden, um zu zeigen, wie sie verwendet werden können. Informationen zur Verwendung von GML finden Sie im Abschnitt GML-Übersicht des Handbuchs.

In den folgenden Abschnitten werden die Funktionen erläutert, mit denen Sie die verschiedenen Ressourcen in Ihrem Projekt bearbeiten können. Dazu gehören das Erstellen neuer dynamischer Ressourcen oder das Bearbeiten und Ändern von Funktionen, die über den Ressourcenbaum zu Ihrem Spiel hinzugefügt werden.

GameMaker Studio 2 gibt es eine Vielzahl von Möglichkeiten, mit denen Sie Instanzen innerhalb Ihres Spielraums verschieben können, angefangen von Geschwindigkeits- und Richtungsvektoren zum einfachen "Platzieren" einer Instanz an einer bestimmten Position, als auch von komplexeren Methoden wie finden. Es gibt auch eine gute Anzahl verschiedener Methoden, um Kollisionen zwischen zwei Instanzen zu erkennen. Welche Sie wählen, hängt stark von der Art des Spiels und der genauen Situation ab, in der Sie die Kollisionsfunktionen verwenden müssen.
