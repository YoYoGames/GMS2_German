keyboard_check_direct


Beschreibung

Diese Funktion kehrt zurück true wenn die Taste mit dem jeweiligen Schlüsselcode gedrückt wird, oder false Ist dies nicht der Fall, prüfen Sie die Hardware direkt. Es erlaubt einige weitere Überprüfungen, insbesondere können Sie die Tastencodes vk_lshift, vk_lcontrol, vk_lalt, vk_rshift, vk_rcontrol und vk_ralt verwenden, um zu prüfen, ob die linke oder rechte Shift-, Steuer- oder Alt-Taste gedrückt wird. Die Funktion nimmt einen von der Funktion zurückgegebenen Keycode-Wert an ord() (nur Großbuchstaben von AZ oder Zahlen von 0-9) oder eine der vk_* Konstanten auf der Hauptseite der Tastatureingabe.

HINWEIS: Diese Funktion ist nur für das Standard-Windows-Ziel verfügbar und das Ergebnis ist unabhängig davon, welche Anwendung den Fokus hat.


Syntax:

keyboard_check_direct(key);

Streit Beschreibung
key Der Schlüssel zum Überprüfen des Down-Status von.


Kehrt zurück:

Boolean


Beispiel:

if keyboard_check_direct(vk_ralt) || keyboard_check_direct(vk_lalt)
   {
   crouch = true;
   }

Der obige Code prüft, ob entweder die linke oder die rechte Alt-Taste gedrückt wurde, und wenn dies der Fall ist, setzt die Variable "crouch" auf "true".
