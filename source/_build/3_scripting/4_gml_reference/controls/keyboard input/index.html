Tastatureingabe


Beim Umgang mit der Tastatur in GameMaker Studio 2 Ihnen eine Reihe von Funktionen zur Verfügung, mit denen Sie verschiedene Tastaturzustände wie gedrücktes oder freigegebenes erkennen können. Es gibt auch einige, die alle Tastendrücke als Zeichenfolge speichern oder Ihnen sagen können, welche Taste zuletzt gedrückt wurde, und andere, mit denen Sie den Tastaturzustand vollständig löschen können.

HINWEIS: Diese Funktionen sind nur für Windows / Mac / Ubuntu-Desktop-Plattformen vorgesehen. Möglicherweise werden einige der eingebauten Variablen und Konstanten auf anderen Plattformen nicht gültig sein, und viele Funktionen funktionieren auf Mobiltelefonen nicht.


Jedes Eingabezeichen einer Taste (oder mehrerer Tasten) wird durch seinen UTF8- Code definiert, bei dem es sich um einen numerischen Wert handelt. Dieser Wert kann für jedes Zeichen mit der Taste abgerufen werden ord() Funktion, aber GameMaker Studio 2 hat auch eine Reihe von constants für die am häufigsten verwendeten Tastaturtasten und Sonderfunktionen. Normalerweise verwenden Sie eine Kombination aus ord() mit dem keyboard_check*() Funktionen, etwa so:

if keyboard_check(ord("A"))
    {
    hspeed = -5;
    }


Das obige überprüft also die "A" -Taste und wenn es gedrückt wird, wird die horizontale Geschwindigkeit des Objekts auf -5 gesetzt. Beachten Sie, dass mit ord() funktioniert auf diese Weise nur korrekt, wenn die Eingabezeichenfolge nur ein Zeichen lang ist und eine Zahl zwischen 0 und 9 oder ein Großbuchstabe aus römischen Buchstaben von A bis Z ist. Die Funktion ord() gibt einen vollen UTF8-Wert zurück, aber der keyboard_check*() Funktionen erkennen nur A - Z und 0 - 9. Was aber, wenn Sie die Pfeiltasten verwenden möchten? Oder wenn Sie eine Aktion mit der Umschalttaste ändern möchten? Nun, dafür hat GameMaker Studio 2 eine Reihe von vk_ Konstanten ( vk_ steht für virtuelle Tastatur ), die Sie anstelle von verwenden können ord():

Konstante Beschreibung

Im Folgenden finden Sie ein kleines Beispiel für die Verwendung von vk_ Konstanten:

if keyboard_check_pressed(vk_tab)
    {
    instance_create_layer(x, y, "Controllers", obj_Menu);
    }


Der obige Code erkennt, ob die "Tab" -Taste gedrückt wird, und erstellt eine Instanz des Objekts "obj_Menu", falls dies der Fall ist.

Wenn Sie nach einem Schlüsselzeichen suchen müssen, das nicht 0 - 9, A - Z oder eine der VK - Konstanten ist, sollten Sie eine der keyboard_* Variablen wie keyboard_lastchar zum Beispiel:
