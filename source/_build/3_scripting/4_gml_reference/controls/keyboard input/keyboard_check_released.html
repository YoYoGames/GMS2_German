keyboard_check_released

Gibt zurück, ob die angegebene Taste auf der Tastatur gerade losgelassen wurde.

Syntax:

keyboard_check_released(key);


Streit Beschreibung
key Der Schlüssel zum Überprüfen des freigegebenen Status von.


Rückgabe: Boolean


Beschreibung

Mit dieser Funktion können Sie prüfen, ob ein Schlüssel freigegeben wurde oder nicht. im Gegensatz zu den keyboard_check Funktion wird diese Funktion nur einmal ausgeführt, wenn die Taste gedrückt wird. Um sie erneut auszulösen, muss die Taste zuerst gedrückt und dann wieder losgelassen werden.


Beispiel:

if keyboard_check_released(ord("P"))
   {
   instance_create_layer(0, 0, "Controllers", obj_Pause);
   }

Der obige Code prüft, ob die Taste "P" losgelassen wurde, und wenn dies der Fall ist, erstellen Sie eine Instanz von "obj_Pause".
