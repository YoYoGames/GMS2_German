keyboard_string


Beschreibung

Diese Variable enthält eine Zeichenfolge, die die letzten (höchstens) 1024 auf der Tastatur eingegebenen Zeichen enthält. Diese Zeichenfolge enthält nur druckbare Zeichen eingegeben haben, aber es wird durch Löschen des letzten Zeichens auf Drücken der Zurück - Taste richtig reagieren. Diese Variable ist nicht schreibgeschützt und Sie können sie ändern, um sie beispielsweise auf "" (eine leere Zeichenfolge) zu setzen, wenn Sie sie bereits behandelt haben, und Sie können sie mit den String-Funktionen bearbeiten. Beachten Sie, dass bei Verwendung der virtuellen Tastatur auf dem Bildschirm nur diese Variable mit der Tastatureingabe aktualisiert wird.


Syntax:

keyboard_string;


Kehrt zurück:

String


Beispiel:

if string_length(keyboard_string) > 15
   {
   keyboard_string = string_copy(keyboard_string, 1, 15);
   }

Der obige Code begrenzt die Länge der Tastaturzeichenfolge auf 15 Zeichen. Entfernen Sie die Zeichen, die diese Grenze überschreiten, indem Sie die ersten 15 Zeichen in die Variable zurückkopieren.
