keyboard_string

Hält eine Zeichenfolge der letzten Tasten gedrückt.

Syntax:

keyboard_string;


Rückgabe: Zeichenfolge


Beschreibung

Diese Variable enthält eine Zeichenfolge mit den letzten (maximal) 1024 Zeichen, die auf der Tastatur eingegeben wurden. Diese Zeichenfolge enthält nur druckbare Zeichen eingegeben haben, aber es wird durch Löschen des letzten Zeichens auf Drücken der Zurück - Taste richtig reagieren. Diese Variable ist nicht schreibgeschützt und Sie können sie ändern, z. B. um sie auf "" (eine leere Zeichenfolge) zu setzen, wenn Sie sie bereits behandelt haben, oder die String-Funktionen verwenden, um sie zu manipulieren.


Beispiel:

if string_length(keyboard_string) > 15
   {
   keyboard_string = string_copy(keyboard_string, 1, 15);
   }

Der obige Code begrenzt die Länge der Tastaturzeichenfolge auf 15 Zeichen und entfernt diejenigen, die diese Grenze überschreiten, indem die ersten fünfzehn Zeichen zurück in die Variable kopiert werden.
