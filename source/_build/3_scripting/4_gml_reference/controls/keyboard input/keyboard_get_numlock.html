keyboard_get_numlock

Ermitteln Sie den Status der Numberlock-Taste auf der Tastatur.

Syntax:

keyboard_get_numlock();


Rückgabe: Boolean


Beschreibung

Sie können diese Funktion verwenden, um den Status der Keypad Number Lock zu finden, wobei " Wahr " für "Ein" zurückgegeben wird und " Falsch" für "Aus" zurückgegeben wird.

HINWEIS: Diese Funktionalität ist nur in den Windows-Exe-Builds verfügbar und funktioniert auf keinem anderen Gerät.


Beispiel:

if keyboard_get_numlock()
   {
   keyboard_set_numlock(false);
   }
else
   {
   keyboard_set_numlock(true);
   }

Der obige Beispielcode wird den Status des Numberlock-Schlüssels erhalten und wenn er eingeschaltet ist (true), wird er auf off gesetzt (false) und umgekehrt.
