device_is_keypad_open

Überprüft das Gerät, ob eine Tastatur geöffnet ist oder nicht.

Syntax:

device_is_keypad_open


Rückgabe: Boolean


Beschreibung

Dies überprüft das Gerät auf eine Tastatur und wenn eine verfügbar ist, kehrt es zurück true Sonst kehrt es zurück false. Bitte beachten Sie, dass diese Funktion hauptsächlich für Android Geräte Android ist.

Für Benutzer mit einem Sony Xperia Play gibt es in GameMaker Studio 2 eine Set-Taste / Tastenbelegung, so dass Sie die Tastaturkonstanten vk_up, vk_down, vk_left, vk_right für die Joypad-Tasten und Triangle verwenden können ord("T"), Quadrat ist ord("S"), Kreis ist vk_alt + vk_backspace, während der Zurück- Knopf einfach ist vk_backspace Cross ist vk_space, Select ist vk_return und Start ist vk_shift, der L Trigger ist ord("L") und R Trigger ist ord("R").


Beispiel:

if device_is_keypad_open()
   {
   global.Setting = 2;
   }
else
   {
   global.Setting = 1;
   }

Der obige Code prüft auf eine Tastatur und ändert dann abhängig vom zurückgegebenen Wert eine globale Variable.
