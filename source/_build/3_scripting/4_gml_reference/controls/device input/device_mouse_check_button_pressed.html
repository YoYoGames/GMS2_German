device_mouse_check_button_pressed

Gibt zurück, ob eine bestimmte Gerätetaste gedrückt wurde.

Syntax:

device_mouse_check_button_pressed(device, button);


Streit Beschreibung
device Das Gerät (von 0 - n ), das überprüft wird.
button Die Schaltfläche des Geräts, das überprüft wird.


Rückgabe: Boolean


Beschreibung

Diese Funktion gibt zurück true oder false abhängig davon, ob das von Ihnen angegebene Gerät "berührt" (angeklickt) wurde oder nicht. Das Geräteargument bezieht sich auf die Berührungsnummer, die von 0 bis n reichen kann (die maximale Anzahl von Berührungen, die erkannt werden können, hängt sehr stark von dem verwendeten Gerät und dem Betriebssystem ab, das es ausführt). Diese Funktion wird nur einmal durch den eigentlichen Pressvorgang und die Standardkonstanten von mb_left und mb_right kann verwendet werden, um nach beiden Schaltflächen zu suchen.

Beispiel:

if device_mouse_check_button_pressed(0, mb_left)
{
press=true;
}

Der obige Code überprüft, ob das Gerät gedrückt wurde, und setzt eine Variable auf wahr, wenn dies der Fall ist.
