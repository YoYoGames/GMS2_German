device_mouse_dbclick_enable

Legt fest, ob ein doppelter Linksklick als mb_right erkannt werden soll oder nicht.

Syntax:

device_mouse_dbclick_enable(bool);


Streit Beschreibung
bool Aktivieren Sie die Doppelklickerkennung ( true ) oder aus ( false ).


Rückgabe: Boolean


Beschreibung

Mit dieser Funktion kann das Gerät so eingestellt werden, dass es einen Doppelklick auf das Symbol erkennt mb_left (linke Maustaste) als mb_right (rechte Maustaste) tippen oder nicht. Standardmäßig ist dies auf eingestellt true Dies bedeutet, dass jedes Mal, wenn der Benutzer den Gerätebildschirm zweimal schnell und nacheinander antippt, der Rückgabewert derselbe ist, als wenn die rechte Maustaste geklickt worden wäre. Wenn diese Option aktiviert ist, wird die erste Berührung als erkannt mb_left und der zweite als mb_right Vergewissern Sie sich, dass der von Ihnen verwendete Code dies berücksichtigt.


Beispiel:

if os_type == os_windows || os_type == os_mac
   {
   device_mouse_dbclick_enable(false);
   }

Der obige Code überprüft, ob das Gerät, auf dem das Spiel ausgeführt wird, ein Windows-PC oder ein Mac ist, und wenn es einer von beiden ist, wird die Doppeltippfunktion deaktiviert.
