Geräteeingabe


Wenn Ihr Spiel auf Plattformen ausgerichtet ist, die Multitouch erlauben, sollten Sie diese Funktionen verwenden. Sie funktionieren genauso wie ein PC, wenn mehrere Mäuse angeschlossen wären und die Konstanten, nach denen gesucht wird, dieselben sind wie die für die Standardmausfunktionen:

Konstante Beschreibung
mb_left Die linke Maustaste
mb_middle Die mittlere Maustaste (dies ist möglicherweise nicht für alle Zielplattformen gültig)
mb_right Die rechte Maustaste
mb_none Keine Maustaste
mb_any Irgendeine der Maustasten


Die Funktionen können alle einen "Berührungs-" Wert annehmen, der einer Berührung auf der Geräteanzeige entspricht, jedoch hängt die maximale Anzahl von Berührungen, die erkannt werden können, sehr von dem Gerät und dem Betriebssystem ab, das ausgeführt wird. Zum Beispiel können einige Low-End- Android Geräte möglicherweise nur 2 oder 3 Berührungen gleichzeitig erkennen, während ein UWP-Gerät möglicherweise bis zu 10 erkennen kann. Beachten Sie auch, dass auf Touchscreen-Geräten ein Doppeltippen als zurückgegeben wird das Äquivalent eines Ereignisses mit der rechten Maustaste. Wenn Sie z. B. mehrere aufeinander folgende Taps in schneller Folge erkennen müssen, müssen Sie nach beiden suchen mb_left und mb_right. Diese Funktionalität kann mit aktiviert oder deaktiviert werden device_mouse_dbclick_enable.

Im Folgenden finden Sie eine Liste aller gerätespezifischen Steuerungsfunktionen:

  1. device_mouse_check_button
  2. device_mouse_check_button_pressed
  3. device_mouse_check_button_released
  4. device_mouse_dbclick_enable
  5. device_mouse_x
  6. device_mouse_y
  7. device_mouse_raw_x
  8. device_mouse_raw_y
  9. device_mouse_x_to_gui
  10. device_mouse_y_to_gui
  11. device_is_keypad_open
  12. device_get_tilt_x
  13. device_get_tilt_y
  14. device_get_tilt_z

Bitte beachten Sie, dass auf mobilen Geräten ( Android, iOS und Windows Phone), können Sie die Zurück - Taste jederzeit verwenden, da es auf der Tastatur Backspace abgebildet wird, die das ist GameMaker Studio 2 Tastatur konstant vk_backspace Sie können also Prüfungen für diesen Schlüssel durchführen und auf diesen Geräten Aktionen ausführen, als ob die Zurück-Schaltfläche verwendet wurde.
