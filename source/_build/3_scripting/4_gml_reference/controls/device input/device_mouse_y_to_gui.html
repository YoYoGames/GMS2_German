device_mouse_y_to_gui

Gibt die x-Position der Maus relativ zur GUI-Ebene zurück, wenn sie gedrückt wird.

Syntax:

device_mouse_y_to_gui(device);


Streit Beschreibung
device Das Gerät (von 0 - n ), das überprüft wird.


Rückgabe: Echt


Beschreibung

Diese Funktion gibt die y-Position der Berührung auf dem Gerät in Bezug auf die GUI-Ebene zurück. Da die GUI-Ebene unabhängig von der Bildschirmauflösung und der Raumgröße auf eine andere Größe eingestellt werden kann, kann es manchmal schwierig sein, den genauen Punkt auf der GUI-Ebene zu ermitteln, an dem ein Benutzer den Bildschirm berührt hat. Mit dieser Funktion können Sie diese Position basierend auf der tatsächlichen GUI-Größe erhalten, wodurch GUI-Schaltflächen und HUD-Elemente einfacher zu erstellen und zu interagieren sind. Wenn Sie dies auf einem der HTML5- oder PC- und Mac-Module ausführen, wird dieser Wert ständig aktualisiert, solange das Gerät (normalerweise eine Maus) angeschlossen ist, jedoch für mobile Geräte nur aktualisiert wird, während der Bildschirm läuft berührte, und beachten Sie, dass die maximale Anzahl der Berührungen, die erkannt werden können, sehr stark von dem Gerät abhängig ist, das verwendet wird, und dem Betriebssystem, das es ausführt.

Beispiel:

if device_mouse_check_button(0, mb_left)
   {
   if device_mouse_y_to_gui(0) > 0 && device_mouse_y_to_gui(0) < 32
      {
      pressed = true;
      }
   else
      {
      pressed = false;
      }
   }

Der obige Code überprüft, ob die Geräte-Maus gedrückt wird, und wenn dies der Fall ist, fragt er die Position des Geräts x auf der GUI-Ebene ab, um zu sehen, ob sie innerhalb der Parameter liegt. Wenn dies der Fall ist, wird die Variable auf "gedrückt" gesetzt true sonst setzt es es um false.
