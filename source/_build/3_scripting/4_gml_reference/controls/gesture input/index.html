Gesteneingabe


Die Geräteeingabefunktionen eignen sich hervorragend für einfache Berührungen auf dem Bildschirm oder zum Ermitteln der Geräteausrichtung. Manchmal möchten Sie jedoch, dass Ihre Projekte komplexere Dinge erkennen, z. B. können Sie eine Instanz durch einen Raum ziehen oder "streichen". Diese Dinge können mit den Gerätefunktionen erledigt werden, aber um die Dinge einfacher zu machen, verfügt GameMaker Studio 2 über eine vollständige Palette von Gestenereignissen, um Doppeltippen, Ziehen, Flicks, Kneifen und Rotationen sowohl auf Instanz- als auch auf globaler Ebene zu erkennen Informationen über die Veranstaltung zurückgeben. Mit den hier aufgeführten Funktionen können Sie besser steuern, wie und wann diese Ereignisse ausgelöst werden.

WICHTIG! Diese Funktionen funktionieren nicht auf der HTML5-Zielplattform. Sie geben keinen Fehler, wenn sie verwendet werden, sondern werden einfach ignoriert.


Die folgenden Abschnitte listen alle verschiedenen Gestenfunktionen auf:

Die folgenden Funktionen können verwendet werden, um die verschiedenen Werte zu ermitteln, die zum Berechnen von Ziehen, Ziehen, Kneifen usw. verwendet werden:

  1. gesten_get_drag_time
  2. gesten_get_drag_distance
  3. gesten_get_flick_speed
  4. gesten_get_double_tap_time
  5. Geste_get_double_tap_distance
  6. gesten_get_pinch_distance
  7. gesten_get_pinch_angle_towards
  8. gesten_get_pinch_angle_away
  9. gesten_get_rotate_time
  10. gesten_get_rotate_angle

Die folgenden Funktionen können verwendet werden, um die verschiedenen Werte festzulegen, die zum Berechnen von Ziehen, Ziehen, Kneifen usw. verwendet werden:

  2. Gesten_Zug_Distanz
  3. Gesten_flick_speed
  4. gesten_double_tap_time
  5. Geste_double_tap_distance
  6. gesten_pinch_distanz
