Gesteneingabe


Die Geräteeingabefunktionen sind allesamt gut und eignen sich für einfache Berührungen auf dem Bildschirm oder für die Ausrichtung des Geräts. Manchmal möchten Sie jedoch, dass Ihre Projekte komplexere Dinge erkennen, z. B. eine Instanz durch einen Raum ziehen oder "ziehen" können. Diese Funktionen können mit den Gerätefunktionen ausgeführt werden. Zur Vereinfachung bietet GameMaker Studio 2 eine umfassende Palette an Gestenereignissen zur Erkennung von doppeltem Antippen, Ziehen, Wischen, Drücken und Drehen auf einer Instanz- und einer globalen Ebene sowie auf Informationen über die Veranstaltung zurückgeben. Die hier aufgelisteten Funktionen ermöglichen Ihnen eine bessere Kontrolle darüber, wie und wann diese Ereignisse ausgelöst werden.

WICHTIG! Diese Funktionen funktionieren auf der HTML5-Zielplattform nicht. Sie geben keinen Fehler aus, wenn sie verwendet werden, sondern werden einfach ignoriert.


In den folgenden Abschnitten werden alle verschiedenen Gestenfunktionen aufgeführt:

Die folgenden Funktionen können verwendet werden, um die verschiedenen Werte zu ermitteln, die zum Berechnen des Flickens, Ziehens, Kneifens usw. verwendet werden:

  1. gesture_get_drag_time
  2. gesture_get_drag_distance
  3. gesture_get_flick_speed
  4. gesture_get_double_tap_time
  5. gesture_get_double_tap_distance
  6. gesture_get_tap_count
  7. gesture_get_pinch_distance
  8. gesture_get_pinch_angle_towards
  9. gesture_get_pinch_angle_away
  10. gesture_get_rotate_time
  11. gesture_get_rotate_angle

Die folgenden Funktionen können verwendet werden, um die verschiedenen Werte festzulegen, die zum Berechnen des Flimmerns, Ziehens, Kneifens usw. verwendet werden:

  1. gesture_drag_time
  2. gesture_drag_distance
  3. gesture_flick_speed
  4. gesture_double_tap_time
  5. gesture_double_tap_distance
  6. gesture_tap_count
  7. gesture_pinch_distance
  8. gesture_pinch_angle_towards
  9. gesture_pinch_angle_away
  10. gesture_rotate_time
  11. gesture_rotate_angle
