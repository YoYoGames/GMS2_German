gesture_drag_distance


Beschreibung

Diese Funktion wird verwendet, um die Entfernung festzulegen, die ein Ziehen-Gesten- Ereignis durch die Bewegung einer Berührung oder eines Klicks auslösen soll. Die Entfernung wird in Zoll gemessen und hat einen Standardwert von 0,1.


Syntax:

gesture_drag_distance(distance);


Streit Beschreibung
distance Die Entfernung (in Zoll), die für die Erkennung von Ziehgestenereignissen festgelegt werden soll.


Kehrt zurück:

N/A


Beispiel:

if gesture_get_drag_distance() != 0.1
    {
    gesture_drag_distance(0.1);
    }

Der obige Code prüft, ob der Ziehabstand für Gesten auf 0,1 Zoll eingestellt ist. Ist dies nicht der Fall, wird er auf diesen Wert gesetzt.
