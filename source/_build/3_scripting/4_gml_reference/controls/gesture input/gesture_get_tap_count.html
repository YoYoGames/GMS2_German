gesture_get_tap_count


Beschreibung

Mit dieser Funktion wird geprüft, ob die Stufenzählung aktiviert oder deaktiviert ist. Die Funktion kehrt zurück true wenn es aktiviert ist, und false Andernfalls können Sie die Stufenzählung mithilfe der Funktion aktivieren oder deaktivieren gesture_tap_count. Wenn diese Option aktiviert ist, verfügt jedes Tap-Ereignis über einen zusätzlichen "tapcount" -DS-Karteneintrag, der die Anzahl der Taps registriert hat. Wenn diese Option aktiviert ist, werden alle Tapping-Events ausgelöst, dh: zwei Taps lösen sowohl das Single-Tapping-Ereignis als auch das aus das Double Tap-Ereignis, wobei der Tap-Count-Wert des Single-Tap-Ereignisses 1 ist und der Tap-Count-Wert des Double-Tap-Ereignisses 2. Der Wert für den Tap-Count-Wert wird auf 0 zurückgesetzt, nachdem die für eine Double-Tap-Erkennung festgelegte Zeit abgelaufen ist (siehe Funktion) gesture_double_tap_time ). Wenn die Stufenzählung deaktiviert ist, wird der erste Stufenanschlag erst registriert, wenn die doppelte Stufenzeit verstrichen ist und keine zweite Stufe erkannt wurde. Beachten Sie, dass diese standardmäßig aktiviert ist.


Syntax:

gesture_get_tap_count();


Kehrt zurück:

Bool


Beispiel:

if !gesture_get_tap_count()
    {
    getsure_tap_count(true);
    }

Der obige Code prüft, ob die Stufenzählung aktiviert ist, und wenn nicht, wird sie eingeschaltet.
