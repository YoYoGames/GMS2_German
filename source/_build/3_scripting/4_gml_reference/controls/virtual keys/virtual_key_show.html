virtual_key_show

Zeigt einen bestimmten erstellten virtuellen Schlüssel auf dem Bildschirm des Geräts an.

Syntax:

virtual_key_show(index);


Streit Beschreibung
index Der Index des anzuzeigenden virtuellen Schlüssels.


Rückkehr: N / A


Beschreibung

Nachdem Sie Ihren virtuellen Schlüssel für Ihre Geräte erstellt haben, müssen Sie möglicherweise debuggen und testen, ob er richtig positioniert ist und ordnungsgemäß funktioniert. Dazu würden Sie diese Funktion aufrufen, indem Sie den Index des virtuellen Schlüssels verwenden, den Sie sehen möchten (zuvor erstellt und gespeichert mit virtual_key_add ), die ein farbiges Rechteck zeichnen, um den Schlüssel auf dem Bildschirm darzustellen. Sobald Sie mit den Dingen zufrieden sind, wird empfohlen, dass Sie Ihre eigene Grafik in ihrem Bereich erstellen und platzieren.

HINWEIS: Das Rechteck gezeichnet wird durch die Mischung, Farb- und Alpha - Optionen, die von den entsprechenden Satz beeinflusst wird Zeichenfunktionen.


Beispiel:

if global.Debug
   {
   virtual_key_show(global.Left);
   }
else
   {
   virtual_key_hide(global.Left);
   }

Der obige Code überprüft die globale Variable "global.Debug" und wenn er "true" testet, wird der in der Variable "global.Left" indizierte virtuelle Schlüssel auf dem Bildschirm gezeichnet, und wenn er falsch ist, wird der Schlüssel versteckt.
