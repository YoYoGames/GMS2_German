virtual_key_show


Beschreibung

Nachdem Sie Ihren virtuellen Schlüssel für Ihre Geräte erstellt haben, müssen Sie möglicherweise prüfen, ob er richtig positioniert ist und ordnungsgemäß funktioniert. Dazu rufen Sie diese Funktion mit dem Index des virtuellen Schlüssels auf, den Sie anzeigen möchten (zuvor erstellt und gespeichert mit) virtual_key_add ), wodurch ein farbiges Rechteck gezeichnet wird, das den Schlüssel auf dem Bildschirm darstellt. Sobald Sie mit den Dingen zufrieden sind, sollten Sie Ihre eigene Grafik erstellen und in ihrem Bereich platzieren.

HINWEIS: Das Rechteck gezeichnet wird durch die Mischung, Farb- und Alpha - Optionen, die von den entsprechenden Satz beeinflusst wird Zeichenfunktionen.


Syntax

virtual_key_show(index);

Streit Beschreibung
index Der Index des anzuzeigenden virtuellen Schlüssels.


Kehrt zurück:

N/A


Beispiel:

if global.Debug
   {
   virtual_key_show(global.Left);
   }
else
   {
   virtual_key_hide(global.Left);
   }

Der obige Code prüft die globale Variable "global.Debug". Wenn sie den Wert "true" überprüft, wird der in der Variablen "global.Left" indizierte virtuelle Schlüssel auf dem Bildschirm angezeigt. Wenn er falsch ist, wird der Schlüssel ausgeblendet.
