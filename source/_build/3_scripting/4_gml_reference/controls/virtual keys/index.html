Virtuelle Schlüssel


Damit Ihre Spiele auf Mobilgeräten mit einem Touchscreen ordnungsgemäß funktionieren, müssen Sie möglicherweise Bereiche des Bildschirms zuweisen können, um auf bestimmte Dinge zu reagieren, wenn sie vom Benutzer berührt werden. Dazu wird einem intern zugeordneten Standardtastaturschlüssel ein "virtueller Schlüssel" zugewiesen und anschließend die von Ihnen generierten Tastaturereignisse zur Steuerung Ihrer Anwendung verwendet. Es ist erwähnenswert, dass diese "virtuellen Tasten" auf die Bildschirmposition und nicht auf die Raumposition abgebildet werden, so dass die x / y-Werte basierend auf der Bildschirmgröße absolut sind. Dies bedeutet, dass Sie sich keine Gedanken über die Verwendung von Ansichten oder die relativen Raumkoordinaten machen müssen und einfach Ihre Schlüssel-Sprites zeichnen und Ihre virtuellen Schlüssel im Draw-GUI-Ereignis eines Objekts definieren können.

HINWEIS: Diese Funktionen funktionieren nicht auf den Zielplattformen Windows, Ubuntu (Linux) oder MacOS!

Die verfügbaren virtuellen Schlüsselfunktionen sind:

  1. virtuelle_schlüssel_add
  2. virtual_key_show
  3. virtual_key_hide
  4. virtual_key_delete
