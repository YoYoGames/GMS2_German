Gamepad_get_device_count

Gibt die Anzahl der verbundenen Game-Pads oder die Anzahl der für Game-Pads verfügbaren Slots zurück.

Syntax:

gamepad_get_device_count();


Rückgabe: Echt


Beschreibung

Diese Funktion wird Ihnen eines von zwei Dingen sagen. Entweder die Anzahl der angeschlossenen Spielepads oder die Anzahl der verfügbaren "Slots" für Spielepads, mit denen verbunden werden soll. Der tatsächliche Rückgabewert hängt von der Plattform und der internen Konfiguration dieser Plattform ab und daher sollte diese Funktion in Verbindung mit der Funktion verwendet werden gamepad_is_connected um die genaue Anzahl der angeschlossenen Pads jederzeit zu überprüfen.


Beispiel:

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
   {
   if gamepad_is_connected(i) global.gp[i] = true else global.gp[i] = false;
   }

Der obige Code durchläuft die verfügbaren Game-Pads (oder Gamepad-Slots) und überprüft dann jedes für ein verbundenes Gamepad. Der zurückgegebene Wert wird dann verwendet, um ein globales Array auf zu setzen true oder false zur Verwendung in zukünftigen Überprüfungen.
