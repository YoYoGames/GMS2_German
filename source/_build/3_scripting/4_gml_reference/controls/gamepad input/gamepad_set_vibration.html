gamepad_set_vibration


Syntax:

gamepad_set_vibration(device, left_motor, right_motor);

Streit Beschreibung
device Welches Gamepad-Gerät "Slot" überprüft werden soll.
left_motor Die Vibrationsstärke des linken Motors von 0 bis 1.
right_motor Die Vibrationsstärke des rechten Motors von 0 bis 1.


Kehrt zurück:

N/A


Beschreibung

Mit dieser Funktion können Sie die Vibration der Gamepad-Motoren einstellen, wobei jeder Motor einen Wert von 0 (keine Vibration) bis 1 (volle Vibration) verwendet. Beachten Sie, dass es für diese Funktion keine Zeitbegrenzung gibt. Daher müssen Sie eine Variable oder einen Alarm verwenden, um die Vibration auszuschalten (Motoren auf 0 setzen), nachdem eine bestimmte Zeit verstrichen ist. Andernfalls vibriert das Gamepad unbegrenzt.

HINWEIS: Diese Funktion ist derzeit nur für die Standard- Zielmodule von Windows, PS4 und Xbox One verfügbar.


Beispiel:

if lives = 0
   {
   gamepad_set_vibration(0, 1, 1);
   alarm[0] = room_speed / 2;
   }

Der obige Code würde (zum Beispiel) in einem Kollisionsereignis verwendet, um das Gamepad für eine halbe Sekunde in "Slot" 0 zu vibrieren, wobei der eingestellte Alarm verwendet wird, um es nach Ablauf dieser Zeit wieder auszuschalten.
