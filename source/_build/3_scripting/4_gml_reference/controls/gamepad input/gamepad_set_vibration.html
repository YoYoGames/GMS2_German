Gamepad_set_vibration

Steuert die Gerätevibration

Syntax:

gamepad_set_vibration(device, left_motor, right_motor);


Streit Beschreibung
device Welches Gamepad-Gerät "Slot" überprüft.
left_motor Die Menge der Vibrationen vom linken Motor von 0 bis 1.
right_motor Die Menge an Vibrationen vom rechten Motor von 0 bis 1.


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie die Vibration der Gamepad-Motoren einstellen, wobei jeder Motor einen Wert von 0 (keine Vibration) bis 1 (volle Vibration) verwendet. Beachten Sie, dass es für diese Funktion keine zeitliche Begrenzung gibt. Sie müssen also nach Ablauf einer bestimmten Zeit eine Variable oder einen Alarm verwenden, um die Vibration abzuschalten (Motoren auf 0 setzen), sonst vibriert das Gamepad unbegrenzt.

HINWEIS: Diese Funktion ist derzeit nur für die Standardzielmodule Windows, PS4 und Xbox One verfügbar.


Beispiel:

if lives = 0
   {
   gamepad_set_vibration(0, 1, 1);
   alarm[0] = room_speed / 2;
   }

Der obige Code würde (zum Beispiel) in einem Kollisionsereignis verwendet werden, um das Gamepad, das in "Slot" 0 eingesteckt wird, für eine halbe Sekunde vibrieren zu lassen, wobei der eingestellte Alarm verwendet wird, um es nach Ablauf dieser Zeit wieder auszuschalten.
