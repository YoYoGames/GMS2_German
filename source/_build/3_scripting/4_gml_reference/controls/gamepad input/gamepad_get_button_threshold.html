gamepad_get_button_threshold

Gibt die aktuelle Schwellenwerteinstellung der analogen Tasten zurück.

Syntax:

gamepad_get_button_threshold(device);


Streit Beschreibung
device Welches Gamepad-Gerät "Slot" überprüft.


Rückgabe: Echt


Beschreibung

Diese Funktion kann verwendet werden, um die aktuelle Schwellenwerteinstellung der analogen Tasten für ein bestimmtes Gerät zu erkennen. Der Standardschwellwert für alle analogen Tasten ist 0,5, wobei der Bereich von 0 bis 1 reicht. Der Schwellwert legt fest, an welcher Stelle die Taste als "gedrückt" für Spiele angesehen wird, bei denen sie als digitale Taste fungieren müssen.


Beispiel:

if gamepad_get_button_threshold(0) != 0.5
   {
   gamepad_set_button_threshold(0, 0.5);
   }

Der obige Code überprüft die analoge Tastenschwelle des Gamepads, der mit dem Gerät "Slot" 0 verbunden ist, und wenn es nicht der Standardwert von 0,5 ist, wird es auf diesen Wert gesetzt.
