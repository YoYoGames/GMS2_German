gamepad_get_mapping


Beschreibung

Diese Funktion kann verwendet werden, um den Mapping-String für das Gamepad abzurufen. Diese Zeichenfolge lautet entweder:

  • Die aktuelle Kartenzeichenfolge für das Gamepad, zum Beispiel:

    "050000005e040000fd020000ffff3f00, Xbox Wireless Controller, a: b0, b: b1, Start: b4, dpdown: h0.4, dpleft: h0.8, dpright: h0.2, dpup: h0.1, Anleitung: b6, leftshoulder: b9 linker Stick: b7, linker Auslöser: a4, leftx: a0, lefty: a1, rechter Schulter: b10, rechter Stick: b8, rechter Auslöser: a5, rightx: a2, righty: a3, x: b2, y: b3, Plattform: android "


  • Eine Zeichenfolge mit "no mapping", wenn kein Mapping-Set vorhanden ist (dies ist nicht dasselbe wie wenn kein Mapping verfügbar ist)

  • Eine Zeichenfolge mit "Geräteindex außerhalb des zulässigen Bereichs", wenn der Slotindex kein gültiger Gamepad-Index ist

  • Eine leere Zeichenfolge "", wenn für das angeschlossene Gamepad kein Mapping verfügbar ist

Weitere Informationen zur Formatierung der zurückgegebenen Map-Zeichenfolge finden Sie in der Funktion gamepad_test_mapping().


Syntax:

gamepad_get_mapping(index);

Streit Beschreibung
index Über welchen Gamepad-Index "Slot" das Mapping abgerufen wird.


Kehrt zurück:

String


Beispiel:

var _gpMap = gamepad_get_mapping(global.PadIndex);
show_debug_message("Gamepad Mapping = " + _gpMap);

Der obige Code ruft die Map-Zeichenfolge für den angegebenen Gamepad-Steckplatz ab und gibt sie zum Debuggen an die Konsole aus.
