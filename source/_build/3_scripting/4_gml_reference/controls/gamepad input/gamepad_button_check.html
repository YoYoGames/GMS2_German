gamepad_button_check

Gibt zurück, ob eine bestimmte Gamepad-Schaltfläche momentan inaktiv ist.

Syntax:

gamepad_button_check(device, button);


Streit Beschreibung
device Welches Gamepad-Gerät "Slot" überprüft.
button Welche Gamepad - Taste konstant zu überprüfen.


Rückgabe: Boolean


Beschreibung

Diese Funktion wird zurückkehren true oder false abhängig davon, ob die vorgegebene Gamepad-Taste als gedrückt erkannt wird oder nicht.


Beispiel:

if gamepad_button_check(0, gp_face1)
   {
   if canshoot = true
      {
      audio_play_sound(snd_Shoot, 0, false);
      instance_create_layer(x, y, "Bullets", obj_Bullet)
      canshoot = false;
      alarm[0] = room_speed / 2;
      }
   }

Der obige Code erkennt, ob der untere Knopf der oberen Fläche (das "X" auf einem PS3-Controller) gedrückt wird, und wenn dies der Fall ist, wird er eine "Kugel" -Instanz schießen und einen Alarm setzen.
