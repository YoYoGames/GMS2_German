gamepad_button_count

Gibt die Gesamtzahl der unterstützten Schaltflächen für das Gerät zurück.

Syntax:

gamepad_button_count(device);


Streit Beschreibung
device Welches Gamepad-Gerät "Slot" überprüft.


Rückgabe: Echt


Beschreibung

Diese Funktion gibt die Gesamtzahl der Tasten zurück, die für das Gamepad verfügbar sind, das mit dem angegebenen Gerät "Slot" verbunden ist.


Beispiel:

b_num = gamepad_button_count(0);

Der obige Code gibt die Anzahl der verfügbaren Tasten auf dem Gamepad zurück, die an das Gerät "slot" 0 angeschlossen sind, und speichert den Wert in der Variablen "b_num".
