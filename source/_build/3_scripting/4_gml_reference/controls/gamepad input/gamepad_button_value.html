gamepad_button_value


Beschreibung

Mit dieser Funktion können Sie den aktuellen Wert einer analogen Taste von 0 bis 1 erhalten, wobei 0 kein Druck und 1 der volle Druck ist. Sie geben den Gamepad-Slot-Index zur Überprüfung an, entweder mit einer Tastenkonstante (wie hier aufgeführt) oder einem ganzzahligen Wert zwischen 0 und gamepad_button_count() -1.


Syntax:

gamepad_button_value(device, button);

Streit Beschreibung
device Welches Gamepad-Gerät "Slot" überprüft werden soll.
button Welche Gamepad - Taste konstant zu überprüfen.


Kehrt zurück:

Real


Beispiel:

speed = gamepad_button_value(0, gp_shoulderrb) * 4;

Der obige Code verwendet den analogen Auslösewert von dem in Gerät "slot" 0 eingesteckten Gamepad, um die Geschwindigkeit der Instanz einzustellen.
