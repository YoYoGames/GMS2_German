gamepad_axis_value

Gibt einen Wert basierend auf der aktuellen Achsenposition zurück.

Syntax:

gamepad_axis_value(device, axisIndex);


Streit Beschreibung
device Welches Gamepad-Gerät "Slot" überprüft.
axisIndex Der zu überprüfende Achsindex (siehe Liste der Konstanten ).


Rückgabe: Echt


Beschreibung

Sie können diese Funktion verwenden, um den Wert der verschiedenen Achsen von einem bestimmten Gamepad zu erhalten. Der Rückgabewert wird für jede der verfügbaren horizontalen und vertikalen Achsen zwischen -1 und 1 liegen (die von dieser Funktion verwendeten Konstanten finden Sie hier ).


Beispiel:

var haxis = gamepad_axis_value(0, gp_axislh);
var vaxis = gamepad_axis_value(0, gp_axislv);
direction = point_direction(0, 0, haxis, vaxis);
speed = point_distance(0 ,0, haxis, vaxis) * 5;

Der obige Code legt die Geschwindigkeit und die Richtung der Instanz basierend auf den zurückgegebenen Werten der linken Analogsteuerung des Gamepads fest, die in den Geräte- "Schlitz" 0 eingesteckt sind.
