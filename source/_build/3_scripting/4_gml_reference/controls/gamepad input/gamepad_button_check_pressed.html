gamepad_button_check_pressed


Beschreibung

Diese Funktion kehrt zurück true oder false abhängig davon, ob die angegebene Gamepad-Taste als gedrückt erkannt wurde oder nicht. Beachten Sie, dass diese Funktion beim ersten Drücken der Taste nur einmal ausgelöst wird. Zum erneuten Auslösen muss die Taste zuerst losgelassen und dann erneut gedrückt werden. Wenn Sie einen kontinuierlichen Tastendruck prüfen müssen, sollten Sie die Funktion verwenden gamepad_button_check.


Syntax:

gamepad_button_check_pressed(device, button);

Streit Beschreibung
device Welches Gamepad-Gerät "Slot" überprüft werden soll.
button Welche Gamepad - Taste konstant zu überprüfen.


Kehrt zurück:

Boolean


Beispiel:

if gamepad_button_check_pressed(0, gp_start)
   {
   audio_play_sound(snd_Start, 0, false);
   room_goto(rm_Level_1);
   }

Der obige Code erkennt, ob die Taste "Start" des an Gerät "slot" 0 angeschlossenen Gamepads gedrückt wurde oder nicht, und ändert den Raum, falls dies der Fall ist.
