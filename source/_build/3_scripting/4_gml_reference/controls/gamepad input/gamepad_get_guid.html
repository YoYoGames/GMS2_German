gamepad_get_guid


Beschreibung

Mit dieser Funktion können Sie die GUID für das mit dem angegebenen Slot-Index verbundene Gamepad abrufen. Sie geben den Gamepad-Slot zur Überprüfung an (von 0 bis 11), und die Funktion gibt eine der folgenden Zeichenfolgen zurück:

  • " none ", wenn keine GUID verfügbar ist oder kein Gamepad angeschlossen ist
  • " Geräteindex außerhalb des Bereichs ", wenn der Gamepad-Slotindex außerhalb des Bereichs liegt
  • Die GUID-Zeichenfolge, etwa "050000005e040000fd020000ffff3f00", wenn dem Steckplatz ein Gamepad zugewiesen ist und die GUID dafür abgerufen werden kann

Diese Funktion wird normalerweise in Verbindung mit verwendet gamepad_get_description() So generieren Sie einen Teil der SDL-Zeichenfolge, die für die Neuzuordnung von Controllern unter Windows, Android und macOS erforderlich ist.


Syntax:

gamepad_get_guid(index);

Streit Beschreibung
index Welches Gamepad "Slot" -Index zu überprüfen ist


Kehrt zurück:

String


Beispiel:

var _guid = gamepad_get_guid(global.PadIndex);
var _desc = gamepad_get_description(global.PadIndex);
global.GamepadID = _guid + "," + _desc;

Der obige Code ruft die GUID- und Namensbeschreibungszeichenfolgen ab, verkettet sie und speichert die endgültige Zeichenfolge in einer globalen Variablen zur Nahtreferenz.
