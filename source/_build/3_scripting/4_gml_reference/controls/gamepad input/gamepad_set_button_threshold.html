gamepad_set_button_threshold


Beschreibung

Diese Funktion kann verwendet werden, um die aktuellen Schwellenwerteinstellungen der analogen Tasten für ein bestimmtes Gerät festzulegen. Der Standardschwellenwert für alle analogen Tasten beträgt 0,5, wobei der Bereich zwischen 0 und 1 liegt. Der Schwellenwert definiert, an welchem ​​Punkt die Taste für Spiele als "gedrückt" angesehen wird, bei denen sie als digitale Taste fungieren müssen.


Syntax:

gamepad_set_button_threshold(device, threshold);

Streit Beschreibung
device Welches Gamepad-Gerät "Slot" überprüft werden soll.
threshold Der neue Schwellenwert (von 0 - 1, Standardeinstellung 0,5).


Kehrt zurück:

N/A


Beispiel:

if gamepad_get_button_threshold(0) != 0.5
   {
   gamepad_set_button_threshold(0, 0.5);
   }

Der obige Code prüft den analogen Tastenschwellenwert des Gamepads, das an Gerät "slot" 0 angeschlossen ist. Wenn der Standardwert nicht 0,5 ist, wird er auf diesen Wert gesetzt.
