gamepad_button_check_released


Beschreibung

Diese Funktion kehrt zurück true oder false abhängig davon, ob die angegebene Gamepad-Taste als freigegeben erkannt wird oder nicht. Beachten Sie, dass diese Funktion für die Taste nur einmal ausgelöst wird, wenn sie losgelassen wird. Zum erneuten Auslösen muss der Knopf zuerst gedrückt und dann wieder losgelassen werden.


Syntax:

gamepad_button_check_released(device, button);

Streit Beschreibung
device Welches Gamepad-Gerät "Slot" überprüft werden soll.
button Welche Gamepad - Taste konstant zu überprüfen.


Kehrt zurück:

Boolean


Beispiel:

if gamepad_button_check_released(0, gp_select)
   {
   audio_play_sound(snd_Button, 0, false);
   global.Pause = !global.Pause;
   }

Der obige Code erkennt, ob die Schaltfläche "Auswählen" des an Gerät "slot" 0 angeschlossenen Gamepads gedrückt wurde oder nicht, und schaltet die globale "Pause" -Variable um.
