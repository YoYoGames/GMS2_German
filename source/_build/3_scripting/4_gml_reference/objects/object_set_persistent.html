object_set_persistent

Legt die Persistenz des angegebenen Objekts fest.

Syntax:

object_set_persistent(index, pers);


Streit Beschreibung
index Der Index des Objekts, das geändert werden soll.
pers Ob das Objekt als persistent gekennzeichnet werden soll ( true ) oder nicht ( false ).


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie die Persistenz eines bestimmten Objekts festlegen. Ein persistentes Objekt wird von Raum zu Raum "übertragen" und behält seine Eigenschaften zu jeder Zeit bei (weitere Informationen zur Persistenz finden Sie unter Definieren von Objekten ). Das Festlegen der Persistenz des Objekts bedeutet, dass alle Instanzen dieses Objekts, die nach dem Ändern der Persistenz erstellt wurden, persistent erstellt werden, während Instanzen, die sich bereits im Raum befinden, nicht betroffen sein können.

Bitte beachten Sie, dass dies keine Instanzfunktion ist! Sie können das Persistent-Flag einzelner Instanzen mit der Option persistent Variable und so haben zehn Instanzen alle als persistent markiert, obwohl das Objekt in der IDE das Flag auf false gesetzt hat, und selbst wenn Sie das persistente Flag des Objekts mit dieser Funktion auf true ändern, alle Instanzen, die sich derzeit im Raum befinden bleibt so, wie sie waren, und nur Instanzen, die nach dem Aufruf der Funktion erstellt wurden, beginnen mit persistent als wahr markiert.


Beispiel:

if room == rm_final object_set_persistent(obj_Player, 0); instance_create_layer(32, 32, "Instances", obj_Player);

Der obige Code überprüft, in welchem ​​Raum sich die Instanz gerade befindet, und wenn er sich in der angegebenen "rm_final" befindet, setzt er die Persistenz des Objekts "obj_Player" auf "false" und erstellt dann die Instanz.
