object_set_mask

Legt den Maskenindex des angegebenen Objekts fest.

Syntax:

object_set_mask(index, spr);


Streit Beschreibung
index Der Index des Objekts, das geändert werden soll.
spr Das neue Sprite, das als Objektmaske verwendet werden soll.


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie den Maskenindex eines bestimmten Objekts einstellen (weitere Informationen zu Masken finden Sie im Objekt-Editor ). Dies bedeutet, dass alle Instanzen dieses Objekts, die nach dem Ändern der Maske erstellt wurden, mit dieser neuen Maske erstellt werden, während Instanzen, die sich bereits im Raum befinden, nicht betroffen sein können. Sie können dies auf -1 setzen, um ein Masken-Sprite zu entfernen, und so standardmäßig auf die für das Sprite des Objekts definierte Maske (oder auf keine Masken, wenn kein Sprite gewählt wurde).

Bitte beachten Sie, dass dies keine Instanzfunktion ist! Sie können den Maskenindex für einzelne Instanzen mit Hilfe von mask_index Variable und so haben zehn Instanzen alle mit einer anderen Maske als das Objekt, aus dem sie erstellt wurden, und selbst wenn Sie den Maskenindex des Objekts mit dieser Funktion ändern, bleiben alle Instanzen, die sich gerade im Raum befinden, unverändert Instanzen, die nach dem Aufruf der Funktion erstellt wurden, beginnen mit der neuen Maske.


Beispiel:

if global.level=10 object_set_mask(obj_Platform, spr_mask_10);

Der obige Code prüft den Wert der globalen Variablen und ändert dann den Maskenindex des Objekts "obj_Platform", wenn es gleich zehn ist.
