object_set_visible

Legt die Sichtbarkeit des angegebenen Objekts fest.

Syntax:

object_set_visible( index, vis );


Streit Beschreibung
index Der Index des Objekts, das geändert werden soll.
vis Die neue Sichtbarkeit des Objekts (wahr = sichtbar, falsch = unsichtbar).


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie ein bestimmtes Objekt als sichtbar kennzeichnen oder nicht (weitere Informationen zum Flag "sichtbar" finden Sie unter Objekte definieren ). Das bedeutet, dass alle Instanzen dieses Objekts, die nach dem Ändern von visible erstellt wurden, mit diesem neuen Status erstellt werden, während Instanzen, die sich bereits im Raum befinden, nicht davon betroffen sind.

Bitte beachten Sie, dass dies keine Instanzfunktion ist! Sie können das Sichtbarkeits-Flag für einzelne Instanzen mit der Option visible Variable und so haben zehn Instanzen alle als sichtbar markiert, obwohl das Objekt in der IDE das Flag auf false gesetzt hat, und auch wenn Sie das sichtbare Flag des Objekts mit dieser Funktion auf true ändern, alle Instanzen, die sich derzeit im Raum befinden wird so bleiben, wie sie waren, und nur Instanzen, die nach dem Aufruf der Funktion erstellt wurden, beginnen mit sichtbar als wahr gekennzeichnet.


Beispiel:

if !object_get_visible(obj_Block) object_set_visible(obj_Block, true);

Der obige Code prüft das sichtbare Flag des Objekts "obj_Block" und wenn es falsch ist, wird es geändert, um als wahr gekennzeichnet zu werden.
