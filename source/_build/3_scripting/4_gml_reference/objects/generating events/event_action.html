Ereignis_Aktion

Der Index der Aktion, die gerade ausgeführt wird.

Syntax:

event_action;


Rückgabe: Echt


Beschreibung

Diese schreibgeschützte Variable gab den Index der aktuell ausgeführten Aktion zurück, beginnend mit 0 bei früheren Versionen von GameMaker. Dies ist jedoch eine veraltete Funktion in GameMaker Studio 2. Es wurde nur für die Legacy-Unterstützung zur Verfügung gestellt und gibt immer 0 zurück, da alle Aktionen verkettet sind, um die Ausführungsgeschwindigkeit zu verbessern.


Beispiel:

num = event_action;

Der obige Code speichert die aktuelle Aktionsnummer in der Variablen "num".
