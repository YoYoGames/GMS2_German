Ereignis_perform

Führt ein bestimmtes Ereignis der aufrufenden Instanz aus.

Syntax:

event_perform(type, numb);


Streit Beschreibung
type Der Typ des auszuführenden Ereignisses. Siehe unten.
numb Die spezifische Ereigniskonstante oder der Wert, falls erforderlich (ansonsten 0).


Rückkehr: N / A


Beschreibung

Diese Funktion führt das angegebene Ereignis mit dem angegebenen Argument für die Instanz aus, auf der der Code ausgeführt wird. Es gibt viele Optionen, die eine vollständige Simulation aller möglichen Ereignisse erlauben, aber beachten Sie, dass dies buchstäblich nur den gesamten Code des Ereignisses ausführt und das Spiel nichts verändert, damit es manuell ausgelöst wird, zum Beispiel wenn Sie eine Tastatur ausführen Drücken Sie Ereignis, wird das Ereignis ausgelöst, aber die relevante Taste wird nicht als gedrückt erkannt. Oder wenn Sie ein Alarmereignis ausführen, wird der Alarmzähler nicht auf -1 gesetzt, sondern läuft weiter abwärts.

Erweitertes Beispiel

Um zu verstehen, wie diese Funktion verwendet werden kann, sehen Sie sich die folgenden Beispiele an:

event_perform(ev_keypress, ord("W"));

Dadurch wird das Ereignis ausgeführt, das mit der Tastaturprüfung "W" verknüpft ist (ohne dass tatsächlich eine Tastaturbetätigung generiert wird).

event_perform(ev_step, ev_step_begin);

Dies würde das Ereignis " Begin Step" ausführen (wenn es von einem der Schritt-Events aufgerufen wird, würde dies dazu führen, dass der Start-Step-Event-Code zweimal ausgeführt wird).

event_perform(ev_create, 0);

Dies würde das Create- Ereignis der Instanz ausführen.
