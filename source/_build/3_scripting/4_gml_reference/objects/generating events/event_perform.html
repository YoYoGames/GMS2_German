event_perform

Führt ein bestimmtes Ereignis der aufrufenden Instanz aus.

Syntax:

event_perform(type, numb);


Streit Beschreibung
type Die Art des auszuführenden Ereignisses. Siehe unten.
numb Die spezifische Ereigniskonstante oder der Wert, falls erforderlich, verwenden Sie andernfalls einfach 0.


Rückgabe: N / A


Beschreibung

Diese Funktion führt das angegebene Ereignis mit dem angegebenen Argument für die Instanz aus, die den Code ausführt. Es gibt viele Optionen, die eine vollständige Simulation aller möglichen Ereignisse zulassen. Beachten Sie jedoch, dass dies buchstäblich den gesamten Code des Ereignisses durchführt und das Spiel nichts ändert, um es manuell auszulösen, z. B. wenn Sie sich für eine Tastatur entscheiden Ereignis drücken, das Ereignis wird ausgelöst, aber die entsprechende Taste wird nicht als gedrückt erkannt. Wenn Sie ein Alarmereignis durchführen, wird der Alarmzähler nicht auf -1 gesetzt, sondern der Countdown wird fortgesetzt.

Erweitertes Beispiel

Um zu verstehen, wie diese Funktion verwendet werden kann, sehen Sie sich die folgenden Beispiele an:

event_perform(ev_keypress, ord("W"));

Dies würde das Ereignis ausführen, das der Taste Keyboard Check Pressed "W" zugeordnet ist (ohne tatsächlich einen Tastendruck zu erzeugen).

event_perform(ev_step, ev_step_begin);

Dies würde das Ereignis " Begin Step" ausführen (wenn es von einem der Step- Events aufgerufen wird, würde der Start-Event-Code zweimal ausgeführt werden).

event_perform(ev_create, 0);

Dies würde das Create- Ereignis der Instanz ausführen.
