object_set_sprite

Legt das Sprite des angegebenen Objekts fest.

Syntax:

object_set_sprite( index, spr );


Streit Beschreibung
index Der Index des Objekts, das geändert werden soll.
spr Das Sprite, das dem Objekt zugewiesen werden soll.


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie den Sprite-Index eines bestimmten Objekts festlegen. Dies bedeutet, dass alle Instanzen dieses Objekts, die nach der Änderung von sprite_index erstellt wurden, mit diesem neuen sprite_index erstellt werden, während Instanzen, die sich bereits im Raum befinden, nicht davon betroffen sind.

Bitte beachten Sie, dass dies keine Instanzfunktion ist! Sie können den Sprite-Index für einzelne Instanzen mit der Taste sprite_index Variable und so haben zehn Instanzen alle mit einem anderen Sprite zu dem Objekt, aus dem sie erstellt werden, und selbst wenn Sie den Sprite-Index des Objekts mit dieser Funktion ändern, bleiben alle Instanzen, die derzeit im Raum sind, so wie sie waren und nur Instanzen, die nach dem Aufruf der Funktion erstellt wurden, beginnen mit dem neuen Sprite.


Beispiel:

switch (room)
   {
   case rm_start: object_set_sprite(obj_Player, spr_uniform); break;
   case rm_middle: object_set_sprite(obj_Player, spr_swimsuit); break;
   case rm_end: object_set_sprite(obj_Player, spr_casual); break;
   }
instance_create_layer(32, 32, "Instances", obj_Player);

Der obige Code setzt das Objekt "obj_Player" Sprite-Index auf verschiedene Werte abhängig von dem Raum, in dem die Instanz, in der der Code gerade ausgeführt wird, ist. Er erstellt dann eine Instanz von "obj_Player".
