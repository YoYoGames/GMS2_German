audio_emitter_set_listener_mask


Beschreibung

Diese Funktion kann verwendet werden, um die Bitmaske für einen Sender so einzustellen, dass alle über den Sender abgespielten Töne nur von den angegebenen Zuhörern abgespielt werden. Sie können eine Bitmaske erstellen, indem Sie die Option verwenden audio_get_listener_info() und dann die bitweise verwenden or ("|"), um eine benutzerdefinierte Maske mit diesen Zuhörern zu erstellen, von denen Sie den gewünschten Sound benötigen, und diese benutzerdefinierte Maske dann auf den Emitter anzuwenden. Diese Maske überschreibt die Standardmaske oder diejenige, die Sie mit der Funktion festgelegt haben audio_set_listener_mask().


Syntax:

audio_emitter_set_listener_mask(emitterID, mask);


Streit Beschreibung
emitterID Die eindeutige ID des Emitters zum Einstellen der Maske von
mask Die Bitmaskendaten, die für den Sound festgelegt werden sollen


Kehrt zurück:

N/A


Beispiel:

s_emit = audio_emitter_create();
audio_emitter_set_listener_mask(snd, global.PlayerMask);

Der obige Code erstellt einen Audio-Emitter und setzt dann den / die zu empfangenden Listener unter Verwendung der in einer globalen Variablen gespeicherten Maskendaten.
