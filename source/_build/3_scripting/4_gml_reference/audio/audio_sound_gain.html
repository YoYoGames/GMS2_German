audio_sound_gain


Beschreibung

Mit dieser Funktion können Sie einen Sound über einen bestimmten Zeitraum ein- oder ausblenden, oder er kann verwendet werden, um die Tonverstärkung sofort einzustellen. Die Zeit wird in Millisekunden gemessen und die Funktion erfordert, dass Sie einen Endpegel der Verstärkung eingeben, damit der Klang bis zum Ende dieser Zeit erreicht werden kann. Diese Verstärkung kann zwischen 0 (lautlos) und einem beliebigen Wert größer als 0 liegen. Normalerweise würden Sie die maximale Lautstärke als 1 betrachten. Alles über 1 kann verwendet werden. Abhängig vom verwendeten Sound und der zu kompilierenden Plattform können Sie dies jedoch tun Verzerrungen oder Beschneidungen erhalten, wenn der Ton wiedergegeben wird. Beachten Sie, dass die Verstärkungsskala linear ist. Um die Verstärkung sofort zu ändern, setzen Sie einfach das Zeitargument auf 0.

Diese Funktion wirkt sich auf alle Instanzen des Sounds aus, die aktuell im Raum abgespielt werden, wenn der Index eine Soundressource ist. Die endgültige Lautstärke ist die Lautstärke, bei der alle weiteren Instanzen des Sounds abgespielt werden. Wenn Sie jedoch den Index verwendet haben, der von einer Funktion wie zurückgegeben wurde audio_play_sound es beeinflusst nur diese eine Instanz des Sounds.


Syntax:

audio_sound_gain(index, volume, time);


Streit Beschreibung
index Der Index des Sounds, für den die Verstärkung eingestellt werden soll.
volume Wert für die Musiklautstärke.
time Die Länge für die Änderung der Verstärkung in Millisekunden.


Kehrt zurück:

N/A


Beispiel:

if val
   {
   var snd = audio_play_sound(snd_fountain);
   audio_sound_gain(snd, 0, 0);
   audio_sound_gain(snd, 1, 5000);
   }

Der obige Code überprüft eine Variable und ob sie zurückgegeben wird true Der Index eines zu spielenden Sounds wird dann der lokalen Variablen "snd" zugewiesen. Diese Variable wird dann verwendet, um die Lautstärke dieses bestimmten Sounds auf 0 zu reduzieren und die volle Lautstärke innerhalb von 5 Sekunden einzublenden.
