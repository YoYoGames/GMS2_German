audio_sound_gain


Beschreibung

Mit dieser Funktion können Sie einen Sound über einen bestimmten Zeitraum ein- oder ausblenden, oder Sie können ihn verwenden, um die Soundverstärkung sofort einzustellen. Die Zeit wird in Millisekunden gemessen, und die Funktion erfordert, dass Sie eine Endverstärkung eingeben, damit der Klang bis zum Ende dieser Zeit erreicht ist. Diese Verstärkung kann zwischen 0 (leise) und 1 (volle Lautstärke) liegen und der Maßstab ist linear, so dass ein Wert von 0,5 das halbe Volumen wäre. Um die Verstärkung sofort zu ändern, setzen Sie einfach das Argument time auf 0.

Diese Funktion betrifft alle Instanzen des Sounds, die gerade im Raum gespielt werden, wenn der Index eine Sound-Ressource ist, und die endgültige Lautstärke ist die Lautstärke, mit der alle weiteren Instanzen des Sounds wiedergegeben werden. Wenn Sie jedoch den von einer Funktion wie zurückgegebenen Index verwendet haben audio_play_sound Es betrifft nur diese eine Instanz des Sounds.


Syntax:

audio_sound_gain(index, volume, time);


Streit Beschreibung
index Der Index des Sounds, für den die Verstärkung eingestellt werden soll.
volume Wert für die Musiklautstärke.
time Die Länge für die Änderung der Verstärkung in Millisekunden.


Kehrt zurück:

N/A


Beispiel:

if val
   {
   var snd = audio_play_sound(snd_fountain);
   audio_sound_gain(snd, 0, 0);
   audio_sound_gain(snd, 1, 5000);
   }

Der obige Code prüft eine Variable und ob sie zurückkehrt true Es wird dann den Index eines zu spielenden Sounds der lokalen Variablen "snd" zuweisen. Diese Variable wird dann verwendet, um die Lautstärke dieses spezifischen Sounds auf 0 zu reduzieren und über 5 Sekunden auf die volle Lautstärke zu schalten.
