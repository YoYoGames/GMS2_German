audio_listener_get_data


Beschreibung

Diese Funktion erstellt ein ds_map und füllen Sie sie mit den Werten für Position, Geschwindigkeit und Ausrichtung für den jeweiligen Listener. Der Standardlistenerindex ist 0, Sie können die Funktion jedoch verwenden audio_get_listener_info um die verschiedenen Indizes für die Zielplattform verfügbar zu machen. Wenn Sie einen falschen Listener-Index angeben, gibt die Funktion -1 zurück.

HINWEIS: Sie sind verantwortlich für die Zerstörung der zurückgegebenen ds_map Verwenden Sie die entsprechende Funktion.

Das ds_map wird folgende Schlüssel enthalten:

  1. "x" - Die x-Position des Zuhörers
  2. "y" - Die y-Position des Zuhörers
  3. "z" - Die z-Position des Hörers
  4. "vx" - Die Geschwindigkeit der X-Achse des Zuhörers
  5. "vy" - Die Geschwindigkeit der y-Achse des Zuhörers
  6. "vz" - Die Geschwindigkeit der Z-Achse des Hörers
  7. "lookat_x" - Die x-Komponente des Blickvektors des Zuhörers
  8. "lookat_y" - Die y-Komponente des Blickvektors des Zuhörers
  9. "lookat_z" - Die z-Komponente des Look-at-vectors des Listeners
  10. "up_x" - Die x-Komponente des Aufwärtsvektors des Hörers
  11. "up_y" - Die Y-Komponente des Aufwärtsvektors des Hörers
  12. "up_z" - Die z-Komponente des Aufwärts-Vektors des Hörers


Syntax:

audio_listener_get_data(index);


Streit Beschreibung
index Der Listener, für den die Daten abgerufen werden sollen (Standard 0).


Kehrt zurück:

ds_map


Beispiel:

var num = audio_get_listener_count();
for(var i = 0; i < num; ++i;)
   {
   var info = audio_get_listener_info(i);
   var data = audio_listener_get_data(info[? "index"]);
   if data[? "x"] != 0
      {
      audio_listener_set_position(info[? "index"], 0, 0, 0);
      }
   ds_map_destroy(info);
   ds_map_destroy(data);
   }

Der obige Code prüft die Anzahl der verfügbaren Listener, durchläuft sie dann, und wenn ihre x-Position nicht 0 ist, setzt sie ihre Position auf 0, 0, 0.
