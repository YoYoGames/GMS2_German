audio_stop_recording


Beschreibung

Diese Funktion stoppt die Aufnahme auf dem angegebenen Rekorderkanal (der Kanalindex wird zurückgegeben, wenn Sie die Funktion aufrufen audio_start_recording ). Wenn Sie die Aufnahme stoppen, werden keine weiteren asynchronen Audioaufnahmeereignisse für den angegebenen Rekorderkanal ausgelöst. Sie würden diese Funktion normalerweise im tatsächlichen asynchronen Ereignis verwenden, um sicherzustellen, dass Sie alle Daten erfasst haben.

ANMERKUNG: Die meisten Plattformen mit Ausnahme von HTML5 unterstützen die Aufnahme von Audio in irgendeiner Form. Dies bedeutet jedoch nicht, dass alle Geräte dies zulassen, auch wenn die Plattform dies tut audio_get_recorder_count() function gibt einen Wert größer als 0 zurück, um zu überprüfen, ob Aufnahmegeräte verfügbar sind, bevor die restlichen Aufnahmefunktionen verwendet werden.


Syntax:

audio_stop_recording(channel_index);


Streit Beschreibung
channel_index Der Index des Rekorderkanals, der angehalten werden soll.


Kehrt zurück:

N/A


Beispiel:

audio_stop_recording(audio_channel);

Der obige Code weist GameMaker Studio 2 an, die Aufnahme auf dem angegebenen Audiokanalindex zu beenden.
