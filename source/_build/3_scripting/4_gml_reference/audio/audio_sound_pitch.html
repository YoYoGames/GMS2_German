audio_sound_pitch


Beschreibung

Diese Funktion kann verwendet werden, um die Tonhöhe eines bestimmten Sounds zu ändern. Der Sound kann entweder einer sein, der von einem Index für einen einzelnen wiedergegebenen Sound referenziert wird, der in einer Variablen gespeichert wurde, wenn der audio_play_sound oder audio_play_sound_at Funktionen oder ein tatsächliches Sound-Asset aus dem Ressourcenbaum. Wenn es sich um einen Index eines spielenden Sounds handelt, wird nur diese Instanz geändert. Wenn Sie jedoch ein Sound-Asset aus dem Ressourcenbaum verwenden, werden alle weiteren Instanzen dieses Sounds geändert.

Das Tonhöhenargument ist ein Tonhöhenmultiplikator, bei dem der Eingabewert die aktuelle Tonhöhe um diesen Betrag multipliziert, so dass der Standardwert von 1 keine Tonhöhenänderung ist, während ein Wert von weniger als 1 die Tonhöhe und mehr als 1 die Tonhöhe anhebt Tonhöhe. Es empfiehlt sich, kleine Inkremente für diese Funktion zu verwenden, da ein Wert unter 0 oder über 5 möglicherweise nicht hörbar ist. Es ist erwähnenswert, dass die erlaubte Gesamttonhöhenänderung auf (1/256) - 256 Oktaven beschränkt ist, so dass jeder darüber oder darunter liegende Wert nicht registriert wird.

HINWEIS: Der oben angegebene geklammerte Wert ist der Versuch von GameMaker Studio 2, den Bereich zu GameMaker Studio 2, aber dieser Wert ist nicht auf allen Zielplattformen garantiert. iOS beispielsweise klemmt auf (1/256) - 8, sodass Sie möglicherweise auf jeder Zielplattform experimentieren müssen und verschiedene Versionen einer Klangressource haben müssen, die jeweils vorverschoben sind, falls Sie höhere oder niedrigere Oktavwerte benötigen.


Syntax:

audio_sound_pitch(index, pitch);


Streit Beschreibung
index Der Index des Sounds zu ändern.
pitch Der Tonhöhenmultiplikator (Standard 1).


Kehrt zurück:

N/A


Beispiel:

var s_engine = audio_play_sound(snd_CarEngine, 10, false);
switch (gear)
   {
   case 1: audio_sound_pitch(s_engine, 0.8); break;
   case 2: audio_sound_pitch(s_engine, 0.9); break;
   case 3: audio_sound_pitch(s_engine, 0.95); break;
   case 4: audio_sound_pitch(s_engine, 1); break;
   case 5: audio_sound_pitch(s_engine, 1.2); break;
   }

Der obige Code ändert die Tonhöhe des Audiomaterials, das von dem in der Variablen "s_engine" indizierten Klang gespielt wird, basierend auf dem Wert der Variablen "gear".
