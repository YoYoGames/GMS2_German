audio_master_gain


Beschreibung

Mit dieser Funktion können Sie den absoluten Wert für die globale Lautstärke aller Sounds und Musik festlegen. Es basiert auf einer linearen Skala von 0 (leise) bis 1 (volle Lautstärke) und wirkt sich auf die relative Lautstärke aller Sounds und Musik aus Ihrem Spiel aus. Beachten Sie, dass Sie auf manchen Plattformen eine Verstärkung von mehr als 1 haben können, obwohl ein Wert von 1 als "volle Lautstärke" betrachtet wird und alles, was größer ist, Audio-Clipping verursachen kann.


Syntax:

audio_master_gain(gain);


Streit Beschreibung
gain Wert für das globale Volumen (0 bis 1).


Kehrt zurück:

N/A


Beispiel:

if keyboard_check(vk_up)
   {
   if vol < 1 vol += 0.05;
   audio_master_gain(vol);
   }
if keyboard_check(vk_down)
   {
   if vol > 0 vol -= 0.05;
   audio_master_gain(vol);
   }

Der obige Code prüft auf Tastendrücke der Auf- und Ab-Pfeiltasten, die dann die Variable "vol" erhöhen oder erniedrigen. Diese Variable wird dann verwendet, um den globalen Gewinn aller Sounds und Musik im Spiel einzustellen.
