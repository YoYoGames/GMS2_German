audio_get_name


Beschreibung

Diese Funktion gibt den Namen einer bestimmten Audio-Ressource als Zeichenfolge zurück. Der "Index" -Wert kann der Wert der Ressource selbst sein (wie im Ressourcenbaum zu sehen) oder der eindeutige ID-Wert, der bei der Wiedergabe des Sounds gegeben wird. audio_play_sound(). Beachten Sie, dass die zurückgegebene Zeichenfolge nicht der Ressourcen-ID entspricht und nicht für den Zugriff auf die Ressource selbst verwendet werden kann. Sie sollte daher nur für die Anzeige oder die Fehlerprüfung verwendet werden.


Syntax:

audio_get_name(index);


Streit Beschreibung
index Der Index des zu überprüfenden Sounds.


Kehrt zurück:

String


Beispiel:

var snd = audio_play_sound(choose(snd_One, snd_Two, snd_Three), 0, false);
var name = audio_get_name(snd);
show_debug_message("Sound = " + name);

Der obige Code spielt einen zufälligen Sound ab, der aus drei verschiedenen Sound-Ressourcen ausgewählt wurde, und zeigt dann eine Debug-Nachricht mit seinem Namen an.
