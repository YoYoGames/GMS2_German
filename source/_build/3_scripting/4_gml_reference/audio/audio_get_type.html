audio_get_type


Beschreibung

Beim Hinzufügen von Audio zu GameMaker Studio 2 kann es entweder gestreamt oder im Speicher abgelegt werden. Wenn Sie wissen möchten, ob ein bestimmter Soundindex für gestreamtes Audio ist oder nicht, können Sie diese Funktion verwenden, die 1 für Streaming, 0 für Sound im Speicher und -1 zurückgibt, wenn ein Fehler vorliegt oder der Index nicht auf a zeigt gültige Tonquelle


Syntax:

audio_get_type(index);


Streit Beschreibung
index Der Index des zu überprüfenden Sounds.


Kehrt zurück:

Real


Beispiel:

type = audio_get_type(snd_Music_1);

Der obige Code überprüft den Audio-Typ, der in der Variablen "snd_Music_1" indexiert wurde, und speichert den zurückgegebenen Wert in der Variablen "type".
