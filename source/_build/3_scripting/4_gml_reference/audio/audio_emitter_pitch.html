audio_emitter_pitch


Beschreibung

Diese Funktion kann verwendet werden, um die Tonhöhe aller vom gegebenen Sender emittierten Töne zu ändern. Es ist ein Tonhöhenmultiplikator, bei dem der Eingangswert die aktuelle Tonhöhe um diesen Betrag multipliziert, so dass der Standardwert von 1 keine Tonhöhenänderung ist, während ein Wert von weniger als 1 die Tonhöhe und mehr als 1 die Tonhöhe erhöht. Es empfiehlt sich, kleine Inkremente für diese Funktion zu verwenden, da ein Wert unter 0 oder über 5 möglicherweise nicht hörbar ist.

HINWEIS: Wenn ein Ton durch den Emitter geschleift wird, wird die Änderung der Tonhöhe nicht erkannt, es sei denn, der Ton wird gestoppt und erneut geloopt!


Syntax:

audio_emitter_pitch(emitter, pitch);


Streit Beschreibung
emitter Der Index des Senders ändert sich.
pitch Der Tonhöhenmultiplikator (Standard 1).


Kehrt zurück:

N/A


Beispiel:

switch (gear)
   {
   case 1: audio_emitter_pitch(s_emit, 0.8); break;
   case 2: audio_emitter_pitch(s_emit, 0.9); break;
   case 3: audio_emitter_pitch(s_emit, 0.95); break;
   case 4: audio_emitter_pitch(s_emit, 1); break;
   case 5: audio_emitter_pitch(s_emit, 1.2); break;
   }

Der obige Code ändert die Tonhöhe des Audiomaterials, das von dem in der Variablen "s_emit" indexierten Emitter gespielt wird, basierend auf dem Wert der Variablen "gear".
