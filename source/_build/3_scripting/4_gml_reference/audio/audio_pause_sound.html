audio_pause_sound


Beschreibung

Mit dieser Funktion können Sie jeden aktuell laufenden Ton anhalten. Der Sound kann eine einzelne Instanz eines Sounds sein (der Index für einzelne Sounds, die abgespielt werden, kann in einer Variablen gespeichert werden, wenn Sie den audio_play_sound oder audio_play_sound_at Funktionen) oder ein Sound-Asset, in welchem ​​Fall alle Instanzen des gegebenen Sounds pausiert werden.


Syntax:

audio_pause_sound(index);


Streit Beschreibung
index Der Index des zu pausierenden Sounds.


Kehrt zurück:

N/A


Beispiel:

if keyboard_check_pressed(ord("P"))
   {
   global.Pause = !global.Pause;
   if global.Pause
      {
      audio_pause_sound(snd_Waterfall);
      }
   else
      {
      audio_resume_sound(snd_Waterfall);
      }
   }

Der obige Code prüft auf ein Drücken der Tastaturtaste "P", und wenn er einen Code erkennt, setzt er die globale Variable "Pause" auf true oder false und unterbricht dann entweder den in der Variablen "snd_Waterfall" indexierten Sound oder nimmt den Sound aus dem angehaltenen Zustand wieder auf.
