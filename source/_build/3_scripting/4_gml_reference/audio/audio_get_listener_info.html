audio_get_listener_info


Beschreibung

Diese Funktion erstellt ein ds_map und füllen Sie sie mit Informationen für den angegebenen Listener.

HINWEIS: Sie sind verantwortlich für die Zerstörung der zurückgegebenen ds_map Verwenden Sie die entsprechende Funktion.

Das ds_map wird folgende Schlüssel enthalten:

  1. "Name" - Der Name des Listeners als String, wobei "Standard" der Standard-Listener-Name auf den meisten Zielplattformen ist
  2. "Maske" - Die Bitmaske für den Hörer
  3. "index" - Der eindeutige Indexwert des Listeners

Der Maskenwert kann verwendet werden, um einen Sound oder Emitter so einzustellen, dass er von mehreren Zuhörern gleichzeitig abgespielt werden kann or "|" um eine Maske für den Sound zu erzeugen (siehe Beispielcode unten), während der Index verwendet wird, um die Eigenschaften wie Position oder Geschwindigkeit für einen bestimmten Listener mit Funktionen wie audio_listener_set_position().


Syntax:

audio_get_listener_info(num);


Streit Beschreibung
num Die Listener-Nummer, für die die Daten abgerufen werden sollen.


Kehrt zurück:

ds_map


Beispiel:

var num = audio_get_listener_count();
var mask = 0; for(var i = 0; i < num; ++i;)
   {
   var info = audio_get_listener_info(i);
   var m = audio_listener_get_data(info[? "mask"]);
   mask = mask | m;
   ds_map_destroy(info);
   }
audio_set_listener_mask(mask);

Der obige Code prüft die Anzahl der verfügbaren Listener, dann durchlaufen die Schleifen ihre Maskenbits, die dann kombiniert werden, um eine einzelne Bitmaske zu erzeugen, die auf den globalen Listener angewendet wird.
