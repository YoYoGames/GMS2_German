audio_sound_set_listener_maske


Beschreibung

Diese Funktion kann verwendet werden, um die Bitmaske für einen Sound so einzustellen, dass er nur von den angegebenen Listenern abgespielt wird. Sie können eine Bitmaske erstellen, indem Sie die Option verwenden audio_get_listener_info() und dann die bitweise verwenden or ("|"), um eine benutzerdefinierte Maske mit den Zuhörern zu erstellen, von denen Sie den gewünschten Sound benötigen, und diese benutzerdefinierte Maske dann auf den Sound anzuwenden, nachdem Sie ihn abgespielt haben. Diese Maske überschreibt die Standardmaske oder diejenige, die Sie mit der Funktion festgelegt haben audio_set_listener_mask().

HINWEIS: Diese Funktion kann nur für einen Sound verwendet werden, nachdem er auf Wiedergabe eingestellt wurde und erfordert die soundID, wie sie von der Funktion zur Wiedergabe des Sounds zurückgegeben wird.


Syntax:

audio_sound_set_listener_mask(soundID, mask);


Streit Beschreibung
soundID Die eindeutige ID des Sounds zum Einstellen der Maske von
mask Die Bitmaskendaten, die für den Sound festgelegt werden sollen


Kehrt zurück:

N/A


Beispiel:

var snd = audio_play_sound(snd_PlayerDead, 10, false);
audio_sound_set_listener_mask(snd, global.PlayerMask);

Der obige Code gibt einen Ton wieder und setzt dann den oder die Listener, die gespielt werden sollen, indem er die in einer globalen Variablen gespeicherten Maskendaten verwendet.
