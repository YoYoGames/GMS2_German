audio_group_set_gain


Beschreibung

Mit dieser Funktion können Sie eine Gruppe von Sounds über einen bestimmten Zeitraum ein- oder ausblenden oder sie können verwendet werden, um die Gruppenverstärkung sofort einzustellen. Die Zeit wird in Millisekunden gemessen, und die Funktion erfordert, dass Sie eine endgültige Verstärkungsebene für die Gruppe eingeben, die am Ende dieser Zeit erreicht wurde. Diese Verstärkung kann zwischen 0 (leise) und 1 (volle Lautstärke) liegen und der Maßstab ist linear, so dass ein Wert von 0,5 das halbe Volumen wäre. Um die Verstärkung sofort zu ändern, setzen Sie einfach das Zeitargument auf 0. Beachten Sie, dass Sie auf manchen Plattformen eine Verstärkung von mehr als 1 haben können, obwohl ein Wert von 1 als "volle Lautstärke" und etwas mehr als Audio-Clipping betrachtet werden kann.


Syntax:

audio_group_set_gain(groupID, volume, time);


Streit Beschreibung
groupID Der Index der anzuhaltenden Audiogruppe (wie im Audiogruppen-Fenster definiert )
volume Der endgültige Wert für das Gruppenvolumen.
time Die Länge der Änderung der Verstärkung in Millisekunden.


Kehrt zurück:

N/A


Beispiel:

if keyboard_check_pressed(vk_space)
   {
   audio_group_set_gain(audiogroup1, 0, 5000);
   }

Der obige Code sucht nach der "Space" -Taste und blendet dann den Audio-Pegel für "audiogruppe1" über 5 Sekunden auf 0 herunter.
