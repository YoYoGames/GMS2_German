audio_get_recorder_count


Beschreibung

Diese Funktion gibt die Anzahl der Audioaufnahmequellen (wie Mikrofone usw.) zurück, die derzeit für Ihr Spiel verfügbar sind. Wenn Sie beispielsweise einen Rückgabewert von vier haben, erhalten Sie Audioeingang an den Quellen 0, 1, 2 und 3, wobei einer dieser Werte derjenige ist, den Sie zum Starten der Aufnahme verwenden, um die Funktion zu verwenden audio_start_recording(). Dieser Wert kann sich jederzeit ändern, wenn Personen Mikrofone oder andere Eingangsquellen an das Gerät anschließen oder entfernen. Beachten Sie, dass Sie die Funktion verwenden können audio_get_recorder_info Informationen zu jedem verbundenen Gerät erhalten.

WICHTIG! Diese Funktion gibt immer 0 zurück, wenn sie für das HTML5-Ziel verwendet wird.


Syntax:

audio_get_recorder_count();


Kehrt zurück:

Integer


Beispiel:

if audio_get_recorder_count() > 0
   {
   channel_index = audio_start_recording(0);
   }

Der obige Code prüft die verfügbaren Audiorecorder. Wenn einer oder mehrere vorhanden sind, beginnt die Aufnahme mit dem ersten indizierten (Quelle 0).
