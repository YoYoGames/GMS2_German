audio_create_sync_group


Beschreibung

Erstellt eine Synchronisierungsgruppe und gibt einen eindeutigen ID-Wert zurück, der dann in allen weiteren Audiofunktionsaufrufen für diese Gruppe verwendet werden soll. Wenn Sie möchten, dass die Gruppe eine Schleife bildet, dann übergeben Sie sie true, ansonsten hineingehen false Beachten Sie jedoch, dass alle Tracks, die später hinzugefügt werden, die gleiche Länge haben müssen, wenn Sie sie loopen möchten. Beachten Sie, dass Sie beim Erstellen einer Synchronisierungsgruppe den Speicher und die damit verbundenen Sounds freigeben müssen, wenn Sie die Option nicht verwenden audio_destroy_sync_group() Funktion - zum Beispiel in den Ereignissen Room End oder Destroy.


Syntax:

audio_create_sync_group(loop);


Streit Beschreibung
loop Ob die Spuren in der Gruppe eine Schleife bilden sollen ( true ) oder nicht ( false ).


Kehrt zurück:

index


Beispiel:

sg = audio_create_sync_group(true);
audio_play_in_sync_group(sg, sound1);
audio_play_in_sync_group(sg, sound2);
audio_sound_gain(sound2, 0, 0);
audio_play_in_sync_group(sg, sound3);
audio_sound_gain(sound3, 0, 0);
audio_play_in_sync_group(sg, sound4);
audio_sound_gain(sound4, 0, 0);
audio_start_sync_group(sg);

Das obige erzeugt eine neue Synchronisationsgruppe und weist den Index der Gruppe der Variablen "sg" zu. Vier Sounds werden dann zur Gruppe hinzugefügt, wobei die Verstärkung für drei von ihnen auf 0 gesetzt wird. Schließlich wird die Sync-Gruppe abgespielt.
