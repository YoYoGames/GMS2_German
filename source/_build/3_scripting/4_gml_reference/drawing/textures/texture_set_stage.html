texture_set_stage

Gibt an, in welcher Textur "slot" eine Textur hinzugefügt wird, wenn mit Shadern gearbeitet wird.

Syntax:

texture_set_stage(stage, tex);


Streit Beschreibung
stage Die Textur "slot" zu verwenden.
tex Die zu verwendende Textur.


Rückkehr: N / A


Beschreibung

Diese Funktion setzt den "Slot" einer bestimmten Stufe als eine zu verwendende Textur. Die Anzahl der verfügbaren "Slots" hängt von der Plattform ab, auf der Sie kompilieren, wobei maximal 8 für Windows, Mac und Linux verfügbar sind, aber auf Android Geräten mit niedrigeren Android (zum Beispiel) kann diese Zahl so niedrig wie 2 sein Es ist auch erwähnenswert, dass die erste Stufe "Slot" immer automatisch von GameMaker Studio 2.

HINWEIS: Diese Funktion wird nichts außerhalb des Kontextes eines laufenden Shaders tun! Weitere Informationen finden Sie unter Shaders.


Beispiel:

glassshader_bgsampler = shader_get_sampler_index(GlassShader, "s_BackgroundSampler")
spr = sprite_get_texture(sprite_index, 0);
shader_set(GlassShader);
texture_set_stage(glassshader_bgsampler, spr);
shader_reset();

Der obige Code erhält das Handle für den Sampler innerhalb des Shaders, der als "GlassShader" indiziert ist, und setzt dann diese Shader-Konstante auf die gegebene Sprite-Textur.
