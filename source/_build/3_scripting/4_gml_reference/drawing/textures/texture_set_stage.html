texture_set_stage


Beschreibung

Mit dieser Funktion wird für die gegebene Stufe "Slot" eine zu verwendende Textur festgelegt. Die Anzahl der verfügbaren "Slots" hängt von der Plattform ab, auf der Sie kompilieren. Maximal 8 stehen für Windows, Mac und Linux zur Verfügung. Bei Android Geräten der unteren Android (z. B.) kann diese Anzahl jedoch nur 2 betragen Es ist auch erwähnenswert, dass der "erste Slot" der ersten Stufe immer automatisch von GameMaker Studio 2.

HINWEIS: Diese Funktion bewirkt nichts außerhalb eines laufenden Shaders! Weitere Informationen finden Sie unter Shader.


Syntax:

texture_set_stage(stage, tex);

Streit Beschreibung
stage Die Textur "Slot" zu verwenden.
tex Die zu verwendende Textur.


Kehrt zurück:

N/A


Beispiel:

glassshader_bgsampler = shader_get_sampler_index(GlassShader, "s_BackgroundSampler")
spr = sprite_get_texture(sprite_index, 0);
shader_set(GlassShader);
texture_set_stage(glassshader_bgsampler, spr);
shader_reset();

Der obige Code erhält das Handle für den Sampler im Shader, der als "GlassShader" indiziert wird, und setzt diese Shader-Konstante auf die angegebene Sprite-Textur.
