texture_is_ready


Beschreibung

Diese Funktion kann verwendet werden, um zu überprüfen, ob eine bestimmte Textur-Seite entpackt wurde und betriebsbereit ist oder ob eine Gruppe von Textur-Seiten entpackt wurde und verwendet werden kann. Sie geben die eindeutige Textur-Seiten-ID an (wie vom zurückgegeben texturegroup_* Funktionen) oder die Texturgruppen-ID-Zeichenfolge (wie im Texture Group Editor definiert), und die Funktion wird zurückgegeben true wenn sie ausgepackt wurden oder false Andernfalls.


Syntax:

texture_is_ready(tex_id);

Streit Beschreibung
tex_id Die Textur-Seiten-ID (eine Ganzzahl) oder eine Texturgruppen-ID (eine Zeichenfolge)


Kehrt zurück:

Boolean


Beispiel:

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length_1d(_tex_array); ++i;)
    {
    if texture_is_ready(_tex_array[i])
        {
        texture_prefetch(_tex_array[i]);
        }
    }

Der obige Code ruft die Textur-Seiten-IDs für die Texturgruppe "MainMenu" ab und prüft, ob sie entpackt sind, und wenn sie diese sind, werden sie in VRAM platziert.
