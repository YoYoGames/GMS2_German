texture_get_uvs


Beschreibung

Diese Funktion gibt ein 1D-Array mit 4 Elementen zurück, die die UV-Koordinaten für das Bild auf der Textur-Seite darstellen, und füllt das Array mit den folgenden Werten:

  • [0] = links
  • [1] = oben
  • [2] = richtig
  • [3] = unten

Dieser Wert kann dann in anderen Zeichenfunktionen verwendet werden, insbesondere in allgemeinen 3D- und einigen 2D-Grundfunktionen sowie den Shader-Funktionen. Wenn Sie den UVS für ein Sprite benötigen, können Sie das verwenden sprite_get_uvs und für eine Schrift, font_get_uvs.


Syntax:

texture_get_uvs(texid)


Argumente:

Streit Beschreibung
texid Das Bild-Asset, um die Textur zu erhalten


Kehrt zurück:

1D Array (4 elements)


Beispiel:

var tex = texture_get_uvs(surf_back);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_left = tex[3];

Der obige Code speichert die UV-Koordinaten für die gegebene Oberflächentextur in einem lokalen Array und weist dann die Werte den Instanzvariablen zu.
