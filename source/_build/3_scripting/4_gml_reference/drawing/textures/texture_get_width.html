texture_get_width

Ruft die Breite eines bestimmten Texturelements ab.

Syntax:

texture_get_width(tex);


Streit Beschreibung
tex Der zu verwendende Texturseiten-Asset-Zeiger


Rückgabe: Echt


Beschreibung

Gibt die Breite der Textur mit der angegebenen ID zurück. Dies ist immer ein Wert zwischen 0 und 1. Dies kann dann verwendet werden, wenn Texturen auf Modelle oder Grundelemente abgebildet werden.


Beispiel:

tex_w = texture_get_width(sprite_get_texture(spr_Model_tex, 0));

Der obige Code erhält die Breite der Textur, die einem Sprite-Asset entnommen wurde.
