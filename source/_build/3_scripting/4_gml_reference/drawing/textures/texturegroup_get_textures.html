texturegroup_get_textures


Beschreibung

Diese Funktion kann verwendet werden, um die Textur-Seiten-IDs der einzelnen Seiten abzurufen, aus denen eine Texturgruppe besteht. Sie geben die Zeichenfolge für die Texturgruppen-ID (wie im Texture Group Editor definiert) ein, und die Funktion gibt ein 1D-Array zurück, wobei jeder Eintrag im Array eine einzelne Textur-Seiten-ID ist. Wenn die Funktion fehlschlägt, dh eine ungültige Gruppe ist angegeben oder der Gruppe ist keine Textur zugewiesen, ist das Array leer (Länge 0).


Syntax:

texturegroup_get_textures(tex_id);

Streit Beschreibung
tex_id Der Name der zu überprüfenden Texturgruppe (eine Zeichenfolge)


Kehrt zurück:

1D array


Beispiel:

var _tex_array = texturegroup_get_textures( "MainMenu");
for (var i = 0; i < array_length_1d(_tex_array); ++i;)
    {
    if texture_is_ready(_tex_array[i])
        {
        texture_prefetch(_tex_array[i]);
        }
    }

Der obige Code ruft die Textur-Seiten-IDs für die Texturgruppe "MainMenu" ab und prüft, ob sie entpackt sind, und wenn sie diese sind, werden sie in VRAM platziert.
