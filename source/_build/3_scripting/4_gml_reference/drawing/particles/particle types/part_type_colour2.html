Teil_Typ_farbe2

Stellt den Partikeltyp so ein, dass er zwischen zwei Farben übergeht

Syntax:

part_type_colour2(ind, colour1, colour2);


Streit Beschreibung
ind Der Index des zu ändernden Partikeltyps.
colour1 Die Farbe, bei der das Partikel beginnt.
colour2 Die Farbe, mit der das Partikel endet.


Rückkehr: N / A


Beschreibung

Mit dieser Funktion kann für jedes Partikel, das mit dem angegebenen Typ erstellt wurde, ein zweifarbiger Farbverlauf festgelegt werden. Die erste Farbe ist diejenige, mit der alle Teilchen beginnen, und die zweite Farbe ist diejenige, auf der das Teilchen enden wird, und über die Lebensdauer der Teilchen von der einen Farbe zur anderen wird eine gleichmäßige Gradientenänderung der Farbe auftreten.


Beispiel:

part_type_colour2( part_Health, c_red, c_white);

Der obige Code bewirkt, dass alle Partikel, die mit dem in der Variable "part_Health" indizierten Typ erstellt wurden, während ihrer Lebensdauer eine Farbmischung von rot nach weiß aufweisen.
