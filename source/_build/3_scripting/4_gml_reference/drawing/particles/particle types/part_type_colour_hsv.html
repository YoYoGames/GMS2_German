part_type_colour_hsv

Stellen Sie die Partikelfarbe aus einem Farbton-, Sättigungs- und Wertebereich ein.

Syntax:

part_type_colour_hsv(ind, hmin, hmax, smin, smax, vmin, vmax);


Streit Beschreibung
ind Der Index des zu ändernden Partikeltyps.
hmin Das Minimum der Farbtonkomponente der endgültigen Farbe kann sein.
hmax Das Maximum der Farbtonkomponente der endgültigen Farbe kann sein.
smin Das Minimum der Sättigungskomponente der endgültigen Farbe kann sein.
smax Das Maximum der Sättigungskomponente der endgültigen Farbe kann sein.
vmin Das Minimum der Wertkomponente der finalen Farbe kann sein.
vmax Das Maximum der Wertkomponente der finalen Farbe kann sein.


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie für alle Partikel des angegebenen Typs einen Farbton-, Sättigungs- und Wertebereich festlegen. Sie geben einen minimalen Wert und einen maximalen Wert für jede der drei Komponenten an und die erzeugten Partikel haben eine zufällige Farbe basierend auf dem gegebenen Parameterbereich. Auf diese Weise können Sie Partikel mit demselben Farbton, aber unterschiedlichen Sättigungen oder verschiedenen Farbtönen, jedoch mit dem gleichen Wert (Helligkeit) usw. erzeugen. Alle Werte müssen zwischen 0 und 255 liegen.


Beispiel:

part_type_hsv(global.Stars, 0, 255, 0, 255, 255, 255 );

Der obige Code setzt jedes emittierte Partikel des in der globalen Variable "Stars" indizierten Partikeltyps so, dass es unterschiedliche Farben und Sättigungen, aber denselben Wert (Leuchtkraft) aufweist.
