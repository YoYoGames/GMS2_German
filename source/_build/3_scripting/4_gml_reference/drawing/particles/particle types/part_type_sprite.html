part_type_sprite

Legt die Form der erstellten Partikel auf ein Sprite fest.

Syntax:

part_type_sprite(ind, sprite, animate, stretch, random);


Streit Beschreibung
ind Der Index des zu ändernden Partikeltyps.
sprite Der Index des Sprites, auf den der Partikeltyp eingestellt werden soll.
animate Ob der Animation des Sprites normal (true) oder nicht (false) folgen soll.
stretch Gibt an, ob die Animation des Sprites an die Partikellebensdauer angepasst werden soll (true) oder nicht (false).
random Gibt an, ob ein zufälliges Teilbild ausgewählt werden soll (true) oder nicht (false).


Rückkehr: N / A


Beschreibung

Diese Funktion kann verwendet werden, um einen Partikeltyp so einzustellen, dass er ein benutzerdefiniertes Sprite aus den Spielelementen verwendet. Sie können das Sprite auswählen und dann GameMaker Studio 2, das Sprite zu verwenden und es zu animieren. Wenn Sie sich für animiert entscheiden, können Sie die Teilbilder auch in einem Intervall ändern, in dem die Animation beginnt und endet am Anfang und am Ende der Lebensdauer jedes erzeugten Partikels (so dass, wenn Sie eine Lebensdauer von mindestens 15 Stufen und höchstens 30 Stufen haben, wird die Animation unabhängig vom Wert der Lebensdauer angepasst und schneller ausgeführt für die kürzere Lebensdauer und langsamer für die längere). Sie können auch ein zufälliges Unterbild auswählen, so dass, wenn das Sprite nicht animiert ist, ein anderes Bild zufällig ausgewählt wird, um das Sprite zu erstellen, oder wenn das Partikel animiert ist, wird die Animation an einem zufälligen Punkt gestartet.


Beispiel:

global.p1 = part_type_create();
part_type_sprite(global.p1 , spr_Coins, 1, 0, 0);
part_type_size(global.p1, 1, 3, 0, 0);
part_type_scale(global.p1, 1, 1);
part_type_colour1(global.p1, c_white);
part_type_alpha2(global.p1, 1, 0);
part_type_speed(global.p1, 2, 4, 0, 0);
part_type_direction(global.p1, 0, 180, 0, 0);
part_type_gravity(global.p1, 0.20, 270);
part_type_orientation(global.p1, 0, 0, 0, 0, 1);
part_type_blend(global.p1, 1);
part_type_life(global.p1, 15, 60);

Der obige Code setzt den in der globalen Variablen "p1" indizierten Partikeltyp auf die Form des in "spr_Coins" indizierten Sprites und setzt dann verschiedene andere Partikeleigenschaften.
