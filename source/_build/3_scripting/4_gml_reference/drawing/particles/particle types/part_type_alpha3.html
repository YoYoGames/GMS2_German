part_type_alpha3

Legt einen Partikeltyp fest, um während seiner gesamten Lebensdauer zwischen drei Alphawerten zu wechseln.

Syntax:

part_type_alpha3(ind, alpha1, alpha2, alpha3);


Streit Beschreibung
ind Der Index des zu ändernden Partikeltyps.
alpha1 Das Ausgangs-Alpha des Partikels.
alpha2 Der halbe Punkt Alpha des Teilchens.
alpha3 Das Endalpha des Partikels.


Rückkehr: N / A


Beschreibung

Diese Funktion kann verwendet werden, um einen Drei-Alpha (Transparenz) -Wertgradienten für jedes Partikel festzulegen, das vom gegebenen Typ erzeugt wurde. Das erste Alpha ist das, mit dem alle Teilchen beginnen, und das zweite Alpha ist dasjenige, das zur Hälfte seiner Lebenszeit vermischt wird, und das dritte Alpha ist dasjenige, mit dem das Teilchen enden wird. Eine sanfte Gradientenänderung wird durch die Alphas über die Lebensdauer der Partikel von einem zum anderen auftreten.


Beispiel:

part_type_alpha3( part_Health, 0.5, 1, 0);

Der obige Code bewirkt, dass alle Partikel, die mit dem in der Variablen "part_Health" indizierten Typ erstellt wurden, über ihre Lebensdauer eine Alpha-Mischung von 0,5 bis 1 bis 0 aufweisen.
