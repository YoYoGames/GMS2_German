part_type_blend

Legt einen Mischmodus für Partikeltypen fest.

Syntax:

part_type_blend(ind, additive);


Streit Beschreibung
ind Der Index des zu ändernden Partikeltyps.
additive Ob die Partikel additiv (richtig) oder normal (falsch) gemischt werden sollten.


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie Ihre Partikel mit einem additiven Mischmodus zeichnen ( true ) oder nicht ( false ). Additive Blending ist ein spezieller Blending-Modus, der die Luminanzwerte jedes Partikels addiert, so dass helle Farben allmählich heller werden (bis sie weiß erscheinen), während dunkle Farben mehr und mehr transparent werden, während Schwarz fast unsichtbar ist. Diese Funktion kann einige schöne Partikeleffekte ergeben, ist aber möglicherweise nicht immer notwendig. Zum Beispiel würde eine Rauchspur eine additive Abmischung aufweisen, um den Effekt grau zu halten, aber ein Flammeneffekt würde wahrscheinlich dazu führen, dass die Partikel durchsichtiger und heller werden.


Beispiel:

part_type_blend(part_Fire, true);

Der obige Code setzt alle Partikel, die aus dem in der Variablen "part_Fire" indizierten Partikeltyp erzeugt werden, auf einen additiven Mischmodus.
