Teil_Typ_Richtung

Legt die Richtungseigenschaften für den angegebenen Partikeltyp fest.

Syntax:

part_type_direction(ind, dir_min, dir_max, dir_incr, dir_wiggle);


Streit Beschreibung
ind Der Index des zu ändernden Partikeltyps.
dir_min Die minimale Richtung, in der das Partikel beginnen kann.
dir_max Die maximale Richtung, in der das Partikel beginnen kann.
dir_incr Wie stark sollte die Partikelrichtung pro Schritt ansteigen oder abfallen?
dir_wiggle Wie viel sollte zufällig von der Richtung des Partikels pro Schritt addiert oder subtrahiert werden.


Rückkehr: N / A


Beschreibung

Diese Funktion wird verwendet, um die Richtung des Teilchens zu bestimmen, wenn es erzeugt wird, und es kann auch verwendet werden, um das Teilchen während seiner Lebensdauer zu vergrößern oder zu verkleinern. Die minimalen und maximalen Richtungswerte sind standardmäßig auf 0 gesetzt, was bedeutet, dass sich das Partikel ohne Abweichungen nach rechts bewegt, aber diese Werte können geändert werden, um die Bewegungsrichtung zu begrenzen, wenn die Standard-Richtungen von GameMaker Studio 2 richtig sind, also 90 oben, 180 links und 270 unten sein. Wenn Sie sie auf denselben Wert einstellen, bewegen sich die Partikel nur in einer Richtung.

Sie können auch einen Inkrementwert festlegen, der eine Gradzahl zur Richtung über seine Lebensdauer hinweg addiert (wenn eine positive Zahl vorhanden ist) oder subtrahieren (wenn eine negative Zahl vorhanden ist). Dieser Wert kann mindestens (+/-) 0,01 betragen.

Schließlich können Sie den Faktor "wackeln" einstellen. Dies ist ein Wert, der zufällig zu jedem Schritt der Lebensdauer der Partikel addiert oder subtrahiert wird. Offensichtlich sind größere Werte ausgeprägter als kleinere, und dieser Wert kann sogar negativ sein, wobei der maximale Bereich zwischen -20 und 20 liegt.


Beispiel:

part_type_shape(particle1, pt_shape_pixel);
part_type_size(particle1, 0.19, 0.79, 0.01, 0);
part_type_scale(particle1, 5, 5);
part_type_colour3(particle1, 32768, 8454016, 8454016);
part_type_alpha3(particle1, 0, 0.80, 0);
part_type_speed(particle1, 8, 10, -0.50, 1);
part_type_direction(particle1, 0, 359, 1, 20);
part_type_gravity(particle1, 0, 270);
part_type_orientation(particle1, 0, 359, 0, 0, 1);
part_type_blend(particle1, 1);
part_type_life(particle1, 10, 50);

Der obige Code wird verschiedene Partikelwerte einschließlich der Richtung einstellen, die ein zufälliger Wert zwischen 0 und 359 Grad sein wird. Es wird auch um 1 Grad für jeden Schritt erhöht und es wird eine zufällige Menge von 0 bis 20 hinzugefügt.
