part_type_alpha1

Legt fest, dass ein Partikeltyp während seiner gesamten Lebensdauer einen einzelnen Alphawert hat.

Syntax:

part_type_alpha1(ind, alpha1);


Streit Beschreibung
ind Der Index des zu ändernden Partikeltyps.
alpha1 Das Alpha des Teilchens.


Rückkehr: N / A


Beschreibung

Diese Funktion wird verwendet, um einen Partikeltyp so einzustellen, dass er einen einzelnen Alpha-Wert (Transparenz) für die Gesamtdauer der Lebensdauer jedes einzelnen Partikels hat, und dies kann von 0 (transparent) bis 1 (undurchsichtig) sein.


Beispiel:

part_type_alpha1(global.Snow_Part, 0.5);

Der obige Code setzt alle Partikel, die aus dem in der globalen Variablen "Snow_Part" indizierten Partikeltyp erstellt wurden, auf einen Alpha-Wert von 0,5 (halbtransparent).
