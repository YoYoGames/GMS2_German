Teil_Typ_Geschwindigkeit

Legt die minimale und maximale Geschwindigkeit für ein bestimmtes Partikel fest.

Syntax:

part_type_speed(ind, speed_min, speed_max, speed_incr, speed_wiggle);


Streit Beschreibung
ind Der Index des zu ändernden Partikeltyps.
speed_min Die Mindestgeschwindigkeit, bei der das Partikel beginnen kann.
speed_max Die maximale Geschwindigkeit, mit der das Partikel beginnen kann.
speed_incr Wie stark sollte die Partikelgeschwindigkeit pro Schritt erhöht oder verringert werden?
speed_wiggle Wie viel sollte zufällig hinzugefügt oder von der Geschwindigkeit des Partikels pro Schritt subtrahiert werden.


Rückkehr: N / A


Beschreibung

Diese Funktion wird verwendet, um die Geschwindigkeit des Teilchens zu bestimmen, wenn es erzeugt wird, und es kann auch verwendet werden, um das Teilchen während seiner Lebensdauer zu beschleunigen oder zu verringern. Die minimalen und maximalen Geschwindigkeitswerte sind standardmäßig auf 1 eingestellt, was bedeutet, dass sich das Partikel bei jedem Schritt um 1 Pixel bewegt. Wenn sie auf denselben Wert eingestellt sind, ist die Anfangsgeschwindigkeit für alle Partikel gleich, wenn sie jedoch auf andere Werte eingestellt ist, Jedes Partikel beginnt mit einer zufälligen Geschwindigkeit innerhalb des angegebenen Bereichs. Wenn Sie beispielsweise mindestens 3 und höchstens 6 Pixel haben, werden alle Partikel mit einer Geschwindigkeit zwischen drei und sechs Pixel pro Schritt erstellt.

Sie können auch einen Inkrementwert festlegen, der bewirkt, dass sich das Partikel während seiner Lebensdauer schneller (bei einer positiven Zahl) oder langsamer (bei einer negativen Zahl) bewegt, wobei die angegebene Menge der Partikelgeschwindigkeit bei jedem einzelnen Schritt hinzugefügt wird. Dieser Wert kann mindestens (+/-) 0,01 betragen.

Schließlich können Sie den Faktor "wackeln" einstellen. Dies ist ein Wert, der zufällig zur Geschwindigkeit jedes Schritts der Partikellebensdauer addiert oder subtrahiert wird. Offensichtlich sind größere Werte ausgeprägter als kleinere, und dieser Wert kann sogar negativ sein, wobei der maximale Bereich zwischen -20 und 20 liegt.


Beispiel:

part_type_shape(particle2, pt_shape_flare);
part_type_size(particle2, 0.01, 0.50, 0.01, 20);
part_type_scale(particle2, 0.10, 0.10);
part_type_colour3(particle2, 4210816, 4227327, 8240895);
part_type_alpha3(particle2, 0.20, 0.80, 0);
part_type_speed(particle2, 5, 10, -0.20, 20);
part_type_direction(particle2, 0, 359, -1, 20);
part_type_gravity(particle2, 0, 270);
part_type_orientation(particle2, 0, 359, 0, 20, 1);
part_type_blend(particle2, 1);
part_type_life(particle2, 1, 20);

Der obige Code wird verschiedene Partikelwerte einschließlich der Geschwindigkeit einstellen, die eine minimale Geschwindigkeit von fünf und eine maximale Geschwindigkeit von zehn einstellt, wobei die Geschwindigkeit um einen Wert von -0,2 pro Schritt verlangsamt wird. Die Geschwindigkeit wackelt auch mit einem zufälligen Wert zwischen 0 und 20 für jeden Schritt.
