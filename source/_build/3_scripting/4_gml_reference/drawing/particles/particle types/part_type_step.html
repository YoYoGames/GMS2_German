part_type_step

Legt fest, dass ein Partikel während seiner gesamten Lebensdauer einen anderen Schritt erstellt.

Syntax:

part_type_step(ind, step_number, step_type);


Streit Beschreibung
ind Der Index des zu ändernden Partikeltyps.
step_number Die Menge des neuen Partikels, die pro Schritt erstellt werden soll.
step_type Der Index des zu erstellenden Partikeltyps.


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie Ihren Partikeltyp jeden Schritt bis zum Ende seiner Lebensdauer in einen anderen Typ umwandeln. Sie können einen negativen Wert für die Schrittzahl festlegen, und basierend darauf wird ein Partikel in einem zufälligen Intervall ausgegeben, sodass beispielsweise ein Wert von -3 eine Wahrscheinlichkeit von 1: 3 hat, bei jedem Schritt ein Partikel zu emittieren.

HINWEIS: Verwenden Sie NIEMALS den gleichen Partikeltyp, da dies zu einer Endlosschleife führt und das Spiel in Sekundenschnelle zum Absturz bringen kann! Seien Sie auch sehr vorsichtig, wenn Sie dies verwenden, da dies die Anzahl der Partikel auf dem Bildschirm stark erhöhen und Ihre Spiele merklich verlangsamen kann.


Beispiel:

particle2 = part_type_create();
part_type_shape(particle2, pt_shape_smoke);
part_type_size(particle2, 0.01, 0.01, 0.01, 0);
part_type_scale(particle2, 1, 1);
part_type_colour3(particle2, 4227327, 8454143, 4227327);
part_type_alpha3(particle2,1, 0.50, 0.80);
part_type_speed(particle2, 0, 0, 0, 0);
part_type_direction(particle2, 0, 359, 0, 0);
part_type_gravity(particle2, 0, 270);
part_type_orientation(particle2, 0, 0, -2, 0, 1);
part_type_blend(particle2, 1);
part_type_life(particle2, 160, 160);
particle3 = part_type_create();
part_type_shape(particle3, pt_shape_flare);
part_type_size(particle3, 0.20, 0.20, 0.02, 0);
part_type_scale(particle3, 1, 1);
part_type_colour3(particle3, 16777215, 8454143, 16777215);
part_type_alpha3(particle3, 0, 0.50, 0);
part_type_speed(particle3, 3, 3, -0.03, 0);
part_type_direction(particle3, 0, 359, 0, 0);
part_type_gravity(particle3, 0, 270);
part_type_orientation(particle3, 0, 0, 0, 0, 1);
part_type_blend(particle3, 1);
part_type_life(particle3, 40, 80);
part_type_step(particle2, 2, particle3);

Der obige Code erstellt und definiert zwei Partikel und setzt den Partikel, der in der Variablen "particle2" indiziert ist, um in jedem Schritt seiner Lebensdauer 2 Partikel von "particle3" zu erzeugen.
