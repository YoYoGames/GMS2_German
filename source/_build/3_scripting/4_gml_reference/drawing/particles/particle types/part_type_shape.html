Teil_Typ_Form

Legt die Form der erstellten Partikel auf eine der vierzehn eingebauten Formen fest.

Syntax:

part_type_shape(ind, shape);


Streit Beschreibung
ind Der Index des festzulegenden Partikeltyps.
shape Die Form zur Auswahl.


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie die Sprite-Form für den Partikeltyp festlegen, wobei die folgenden Konstanten akzeptabel sind:

Konstante Beschreibung
ind Der Index des festzulegenden Partikeltyps.
pt_shape_pixel Ein 1x1 Pixel. (Dies ist die Standardeinstellung.)
pt_shape_disk Ein gefüllter Kreis.
pt_shape_square Ein gefülltes Quadrat.
pt_shape_line Eine 8px breite horizontale Linie.
pt_shape_star Ein mit fünf Punkten gefüllter Stern.
pt_shape_circle Ein 3px umrissener Kreis.
pt_shape_ring Ein Kreis mit einem inneren Leuchten (sieht wie eine Blase aus).
pt_shape_sphere Ein Kreis mit einem äußeren Leuchten, in der Mitte fest und nach außen leuchtend.
pt_shape_flare Ein harsch leuchtender Punkt (sieht aus wie ein echter Stern in der Nacht).
pt_shape_spark Ein Funkeffekt wie ein Stern mit mehreren verblassenden Punkten.
pt_shape_explosion Eine quadratische Rauchwolke erfordert mehrere Farben, um einer Explosion zu ähneln.
pt_shape_cloud Eine dünne Wolke erfordert eine Skalierung und mehrere Partikel, die einer Wolke ähneln.
pt_shape_smoke Eine glatte Version des Explosionseffekts. Verwenden Sie mehrere, um eine Rauchwolke zu erstellen.
pt_shape_snow Eine generische Schneeflockeform.




Beispiel:

global.p1 = part_type_create();
part_type_shape(global.p1, pt_shape_square);
part_type_size(global.p1, 1, 3, 0, 0);
part_type_scale(global.p1, 1, 1);
part_type_colour1(global.p1, c_white);
part_type_alpha2(global.p1, 1, 0);
part_type_speed(global.p1, 2, 4, 0, 0);
part_type_direction(global.p1, 0, 180, 0, 0);
part_type_gravity(global.p1, 0.20, 270);
part_type_orientation(global.p1, 0, 0, 0, 0, 1);
part_type_blend(global.p1, 1);
part_type_life(global.p1, 15, 60);

Der obige Code setzt den in der globalen Variablen "p1" indizierten Partikeltyp auf die Form eines Quadrats und setzt dann verschiedene andere Partikeleigenschaften.
