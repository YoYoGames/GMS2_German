part_type_colour_rgb

Stellen Sie die Partikelfarbe aus einem roten, grünen und blauen Bereich ein.

Syntax:

part_type_colour_rgb(ind, rmin, rmax, gmin, gmax, bmin, bmax p>


Streit Beschreibung
ind Der Index des zu ändernden Partikeltyps.
rmin Das Minimum der roten Komponente der endgültigen Farbe kann sein.
rmax Das Maximum der roten Komponente der endgültigen Farbe kann sein.
gmin Das Minimum der grünen Komponente der endgültigen Farbe kann sein.
gmax Das Maximum der grünen Komponente der endgültigen Farbe kann sein.
bmin Das Minimum der blauen Komponente der endgültigen Farbe kann sein.
bmax Das Maximum der blauen Komponente der endgültigen Farbe kann sein.


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie die Mischung von roten, grünen und blauen Farben für alle Partikel des angegebenen Typs einstellen. Sie geben einen minimalen Wert und einen maximalen Wert für jede der drei Komponenten an und die erzeugten Partikel haben eine zufällige Farbe basierend auf dem gegebenen Parameterbereich. Alle Werte müssen zwischen 0 und 255 liegen.


Beispiel:

part_type_rgb(global.Blood_Part, 0, 255, 0, 0, 0, 0);

Der obige Code setzt jedes emittierte Partikel des in der globalen Variablen "Blood_Part" indizierten Partikeltyps auf nur unterschiedliche Rottöne.
