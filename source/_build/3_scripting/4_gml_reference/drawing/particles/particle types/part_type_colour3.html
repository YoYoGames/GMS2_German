part_type_colour3

Legt einen Partikeltyp fest, der während seiner gesamten Lebensdauer zwischen drei Farben wechselt.

Syntax:

part_type_colour3( ind, colour1, colour2, colour3 );


Streit Beschreibung
ind Der Index des zu ändernden Partikeltyps.
colour1 Die Farbe, bei der das Partikel beginnt.
colour2 Die Farbe des Teilchens wird die Hälfte seiner Lebensdauer betragen.
colour3 Die Farbe, mit der das Partikel endet.


Rückkehr: N / A


Beschreibung

Mit dieser Funktion kann für jedes Partikel, das mit dem angegebenen Typ erstellt wurde, ein dreifarbiger Farbverlauf festgelegt werden. Die erste Farbe ist diejenige, mit der alle Teilchen beginnen werden, und die zweite Farbe ist diejenige, die zur Hälfte ihrer Lebensdauer gemischt wird, und die dritte Farbe ist diejenige, mit der das Teilchen enden wird. Eine gleichmäßige Gradientenänderung wird durch die Farben über die Lebensdauer der Partikel von einer Farbe zur anderen auftreten.


Beispiel:

part_type_colour3( part_Health, c_red, c_white, c_maroon);

Der obige Code bewirkt, dass alle Partikel, die mit dem in der Variable "part_Health" indizierten Typ erstellt wurden, eine Farbmischung von rot nach weiß haben, um über ihre Lebenszeit rot zu werden.
