part_type_alpha2

Legt fest, dass ein Partikeltyp bei einem Alpha-Wert beginnt und während seiner gesamten Lebensdauer zu einem anderen Alpha-Wert übergeht.

Syntax:

part_type_alpha2(ind, alpha1, alpha2);


Streit Beschreibung
ind Der Index des zu ändernden Partikeltyps.
alpha1 Das Anfangs-Alpha des Partikels.
alpha2 Das Ende Alpha des Partikels.


Rückkehr: N / A


Beschreibung

Diese Funktion kann verwendet werden, um einen Alpha-Wert (Transparenz) -Gradient für jedes Partikel festzulegen, das vom angegebenen Typ erstellt wurde. Das erste Alpha ist dasjenige, mit dem alle Teilchen beginnen, und das zweite Alpha ist dasjenige, mit dem das Teilchen enden wird, und eine allmähliche Gradientenänderung wird gegenüber dem Alpha über die Lebensdauer der Teilchen von einem zum anderen auftreten. Dies kann von 0 (transparent) bis 1 (undurchsichtig) sein.


Beispiel:

part_type_alpha2(global.Snow_Part, 0, 1);

Der obige Code setzt alle Partikel, die aus dem in der globalen Variablen "Snow_Part" indizierten Partikeltyp erstellt wurden, auf einen Alpha-Wert von 0 (transparent) und dann am Ende ihrer Lebensdauer in ein Alpha von 1 (undurchsichtig).
