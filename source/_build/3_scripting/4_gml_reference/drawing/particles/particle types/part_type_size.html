Teil_Typ_Größe

Legt die minimale und maximale Startgröße des Partikeltyps fest.

Syntax:

part_type_size(ind, size_min, size_max, size_incr, size_wiggle);


Streit Beschreibung
ind Der Index des zu ändernden Partikeltyps.
size_min Die Mindestgröße, ab der das Partikel beginnen kann.
size_max Die maximale Größe, ab der das Partikel beginnen kann.
size_incr Wie viel sollte der Partikel pro Schritt erhöhen oder verringern?
size_wiggle Wie viel sollte zufällig von der Partikelgröße pro Schritt addiert oder subtrahiert werden.


Rückkehr: N / A


Beschreibung

Diese Funktion wird verwendet, um die Größe des Partikels zu bestimmen, wenn es erzeugt wird, und es kann auch verwendet werden, um die Partikel während ihrer Lebensdauer zu vergrößern oder zu verkleinern. Die Werte für die minimale und maximale Größe sind standardmäßig auf 1 eingestellt (das heißt, dass die Größe das gleiche Verhältnis wie das Basispartikel nach dem part_type_scale Skalierung wurde festgelegt) und wenn sie auf denselben Wert eingestellt sind, ist die anfängliche Größe für alle Partikel gleich, aber wenn sie auf unterschiedliche Werte eingestellt ist, beginnt jedes Partikel mit einer zufälligen Größe innerhalb des Bereichs, zum Beispiel wenn Sie haben ein Minimum von 0,5 und ein Maximum von 2, werden alle Partikel mit einer Größe zwischen der Hälfte und dem Doppelten der Basisskala erzeugt.

Sie können auch einen Inkrementwert festlegen, der das Partikelsprite während der Lebensdauer des Partikels wachsen lässt (wenn eine positive Zahl vorhanden ist) oder verkleinern (wenn eine negative Zahl vorhanden ist), wobei die angegebene Menge in jedem Schritt zur Partikelgröße addiert wird. Dieser Wert kann mindestens (+/-) 0,01 betragen.

Schließlich können Sie den Faktor "wackeln" einstellen. Dies ist ein Wert, der zufällig zur Größe jedes Schritts der Lebensdauer der Partikel addiert oder subtrahiert wird. Offensichtlich sind größere Werte ausgeprägter als kleinere, und dieser Wert kann sogar negativ sein, wobei der maximale Bereich zwischen -20 und 20 liegt.


Beispiel:

part_type_shape(particle1, pt_shape_explosion);
part_type_size(particle1, 2, 2, 0.10, 0.01);
part_type_scale(particle1, 1, 1);
part_type_colour2(particle1, 16777215, 12632256);
part_type_alpha3(particle1, 0, 0.20, 0);
part_type_speed(particle1, 0.20, 0.20, 0, 0);
part_type_direction(particle1, 0, 359, 0, 20);
part_type_orientation(particle1, 0, 359, 0, 0, 1);
part_type_blend(particle1, 0);
part_type_life(particle1, 40, 60);

Der obige Code legt verschiedene Partikelwerte fest, einschließlich der Größe, die auf die doppelte Größe eingestellt wird, wenn jedes Partikel erzeugt wird, und erhöht dann die Größe um 0,01 pro Schritt mit einem Wiggle von 0,1 ebenfalls.
