effect_create_below

Erstellt unter allen Instanzen einen einfachen Partikeleffekt.

Syntax:

effect_create_below(kind, x, y, size, colour);


Streit Beschreibung
kind Die Art des Effekts (verwenden Sie eine der hier aufgeführten Konstanten).
x Die x Positionierung des Effekts, falls relevant.
y Die y-Positionierung des Effekts, falls relevant.
size Die Größe des Effekts.
colour Die Farbe des Effekts.


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie einen einfachen Effekt unter allen Instanzen Ihres Raums erstellen (er wird tatsächlich in einer Tiefe von 100000 erstellt). Wenn der Effekt etwas anderes ist ef_rain oder ef_snow Dann können Sie eine x / y-Position definieren, um den Effekt zu erstellen, und die Größe kann ein Wert von 0, 1 oder 2 sein, wobei 0 klein, 1 mittel und 2 groß ist.

Es ist erwähnenswert, dass bei diesen Effekten die Zeichnung ein- und ausgeschaltet werden kann und die Zeichnung angehalten werden kann, indem die Funktionen verwendet werden part_system_automatic_draw und part_system_automatic_update mit dem entsprechenden Wert für den Partikelsystemindex (wobei 0 für die Effekte unterhalb und 1 für die obigen Effekte steht).


Beispiel:

if speed > 0
   {
   effect_create_below(ef_smoke, x, y, 0, c_gray);
   }

Der obige Code erzeugt bei jedem Schritt, bei dem die Instanzgeschwindigkeit größer als 0 ist, bei den x, y-Koordinaten der Instanz eine kleine graue Rauchwolke.
