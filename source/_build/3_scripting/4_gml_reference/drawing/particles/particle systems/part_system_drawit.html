part_system_drawit

Zeichnet die Partikel in einem bestimmten System.

Syntax:

part_system_drawit(ind);


Streit Beschreibung
ind Der Index des Partikelsystems, um die Partikel zu zeichnen.


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie ein Partikelsystem zwingen, alle aktuellen Partikel auf dem Bildschirm zu zeichnen. Ob part_system_automatic_draw Ist diese Option ausgeschaltet, zeigt diese Funktion die Partikel an, wenn sie im Zeichenereignis einer Instanz verwendet wird, oder sie kann verwendet werden, wenn das Zeichenziel auf eine Fläche eingestellt ist (mit surface_set_target ) um die Partikel im System auf diese Oberfläche zu ziehen.


Beispiel:

if surface_exists(surf)
   {
   surface_set_target(surf);
   part_system_drawit();
   surface_reset_target();
   }

Der obige Code prüft, ob die Oberfläche in der Variable "surf" indiziert ist, und wenn er es tut, dann setzt er das Zeichnungsziel auf die Oberfläche, zeichnet die Partikel und setzt dann das Zeichnungsziel zurück, so dass alle normalen Zeichnungen einmal auf dem Bildschirm erscheinen Mehr.
