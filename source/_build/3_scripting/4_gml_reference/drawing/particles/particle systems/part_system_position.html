part_system_position

Setzt die Position des Partikelsystems.

Syntax:

part_system_position(ind, x, y);


Streit Beschreibung
ind Der Index des Partikelsystems soll sich ändern.
x Die neue x-Koordinate des Partikelsystems.
y Die neue y-Koordinate des Partikelsystems.


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie die Basisposition für das Partikelsystem relativ zur (0,0) Position des Raums festlegen, dh alle weiteren Partikelfunktionen, die zu diesem System gehören, werden nun relativ zur neuen Position gezeichnet. Standardmäßig ist diese Position immer (0,0), aber unter ganz besonderen Umständen möchten Sie vielleicht etwas anderes ändern.

HINWEIS: Diese Funktion ändert alles innerhalb des Partikelsystems. Wenn Sie also einen Emitter an der Position (100,100) haben und dann die Position des Partikelsystems auf (0,100) einstellen, zeichnet der Emitter nun bei (100,200). Das gleiche gilt, wenn Sie das System verschieben und dann den Emitter erstellen, obwohl er bei (100,100) erstellt wird (100,200).


Beispiel:

if mouse_check_button_pressed(mb_left)
   {
   part_system_position(global.Sname, mouse_x, mouse_y);
   }

Der obige Code prüft auf das Drücken der Maustaste und wenn er einen entdeckt, wird das Partikelsystem, das in der globalen Variablen "Sname" indiziert ist, auf die Maus x / y-Position verschoben
