part_particles_create_colour

Erzeuge Partikel aus einem System an einer bestimmten Position und mit einer bestimmten Farbe.

Syntax:

part_particles_create_colour(ind, x, y, parttype, colour, number);


Streit Beschreibung
ind Der Index des Partikelsystems.
x Die x-Koordinate, an der die Partikel erstellt werden sollen.
y Die y-Koordinate, an der die Partikel erstellt werden sollen.
parttype Der Index (Typ) der zu erstellenden Partikel.
colour Die Farbmischung für die Partikel.
number Die Anzahl der zu erstellenden Partikel.


Rückkehr: N / A


Beschreibung

Diese Funktion eignet sich ideal für Effekte, die keine der von Partikelemittern gebotenen Funktionen erfordern (z. B. um Rauch von einer Rakete oder einen einfachen Explosionseffekt zu erzeugen), da Sie Partikel schnell und einfach an jeder Position in Partikel erzeugen können das Spielzimmer. Sie können die Partikel auch "on the fly" färben, während sie mit dieser Funktion erzeugt werden. Diese Farbe wird die vordefinierte Farbe des Partikels übersteuern, aber diese Farbe wird nicht über die Lebensdauer der Partikel gemischt. Beachten Sie, dass Sie zuvor das Partikelsystem und den Partikeltyp für diese Funktion erstellt haben müssen.


Beispiel:

if speed > 1
   {
   part_particles_create_colour(sname, x, y, p_MagicSmoke, choose(c_aqua, c_lime, c_fuschia, c_orange), 1);
   }

Der obige Code erzeugt bei jedem Schritt ein einzelnes Partikel, bei dem die Instanz mit dem Code eine höhere Geschwindigkeit als 1 aufweist. Diese Partikel haben eine zufällige Farbe.
