part_system_clear

Löscht ein Partikelsystem auf seine Standardeinstellungen.

Syntax:

part_system_clear(ind);


Streit Beschreibung
ind Der Index des zu reinigenden Partikelsystems.


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie das indizierte System auf seinen Standardzustand zurücksetzen, indem Sie alle Emitter entfernen und die Tiefe und Position zurücksetzen (falls sie geändert wurden). Seien Sie vorsichtig bei der Verwendung dieser Funktion, so als ob Sie eine Instanzeinstellung oder ein Bursting oder eine andere Aktion unter Verwendung eines mit diesem System verbundenen Senders haben, Sie erhalten eine Fehlermeldung, es sei denn, Sie verwenden das part_emitter_exists zu überprüfende Funktion. Es besteht auch keine Notwendigkeit, die part_emitter_destroy Funktion, da dies automatisch erledigt wird.

HINWEIS: Mit dieser Funktion werden die sichtbaren Partikel im Raum gelöscht, die Partikeleigenschaften werden jedoch nicht gelöscht, noch werden sie aus dem Speicher entfernt. Dafür sollten Sie die Funktionen verwenden part_type_clear und part_type_destroy.


Beispiel:

part_system_clear(global.Sname);

Der obige Code löscht das in der globalen Variablen "Sname" indizierte Partikelsystem in seinen Standardzustand.
