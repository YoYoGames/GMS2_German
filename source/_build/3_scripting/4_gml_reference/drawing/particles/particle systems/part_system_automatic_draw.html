part_system_automatic_draw

Legt fest, ob die Partikel im Partikelsystem automatisch gezeichnet werden oder nicht.

Syntax:

part_system_automatic_draw(ind, automatic);


Streit Beschreibung
ind Der Index des Partikelsystems soll sich ändern.
automatic Ob automatische Zeichnung aktiviert ist (wahr) oder nicht (falsch).


Rückkehr: N / A


Beschreibung

Mit dieser Funktion kann die Zeichnung eines Partikelsystems abgeschaltet werden, so dass keine Aktualisierungen (automatisch oder nicht) am System angezeigt werden. Dies ist eine rein visuelle Option und wenn festgelegt auf false Sie werden nicht in der Lage sein, die Partikel zu sehen, da sie nicht gezeichnet sind, aber sie existieren immer noch und ändern ihre Position, Farbe usw. wie sie es normalerweise tun würden. Wenn das automatische Zeichnen GameMaker Studio 2 ist, können Sie GameMaker Studio 2 explizit GameMaker Studio 2, den aktuellen Status des Partikelsystems mithilfe der Funktion zu zeichnen part_system_drawit und wenn Sie diese Funktion auf true Sie können das automatische Zeichnen wieder einschalten.

Eine Sache zu beachten ist, dass wenn Sie die einfachen Effekte verwenden, die von den Funktionen erstellt werden effect_create_above oder effect_create_below dann können Sie die Werte von 0 (für die unteren Effekte) oder 1 (für die oben genannten Effekte) als Partikelsystemindex verwenden und so die automatische Zeichnung für diese ebenfalls umschalten (dies wird auch funktionieren, um die Zeichnung für die DnD™ -Partikeleffekte umzuschalten).


Beispiel:

part_system_automatic_draw(global.Sname, false);

Der obige Code schaltet die automatische Zeichnung für das in der globalen Variablen "Sname" indizierte Partikelsystem aus.
