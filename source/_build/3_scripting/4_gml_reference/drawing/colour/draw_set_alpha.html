draw_set_alpha

Legt die Basis-Alpha-Mischung für die Zeichenfunktionen fest.

Syntax:

draw_set_alpha(alpha);


Streit Beschreibung
alpha Das zu setzende Alpha (zwischen 0 und 1)


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie den Basis-Alpha für das Spiel festlegen. Dieser Wert kann von 0 bis 1 eingestellt werden, wobei 0 vollständig transparent und 1 vollständig undurchsichtig ist (der Standardwert) und sich auf alle weiteren Zeichnungen, einschließlich Hintergründe, Sprites, Schriftarten, Grundelemente und 3D, auswirkt.

Beachten Sie, dass wenn Sie ein Asset zeichnen und ein Alpha-Argument als Teil der Funktion angeben (z. B. bei Verwendung von draw_sprite_ext() ), dann wird der hier eingestellte Alpha-Wert überschrieben, also ein Sprite mit zeichnen draw_sprite_ext Mit einem Alpha-Argument von 1 wird das Sprite vollständig deckend gezeichnet, unabhängig davon, mit welchem ​​Alpha es gesetzt wurde draw_set_alpha (Dies funktioniert auch bei der Verwendung draw_self() und Einstellung der image_alpha der Instanz). Beachten Sie auch, dass wenn eine Instanz eine Standardzeichnung ist (dh ein Sprite, aber kein Draw-Ereignis), der image_alpha Variable wird mit dem eingestellten Alpha geklemmt draw_set_alpha(), so dass ein image_alpha von 1 und ein Draw Alpha von 0,5 wird das Sprite auf Alpha 0,5 zeichnen, während ein image_alpha von 0,25 und ein Zeichen Alpha von 0,5 wird das Sprite bei 0,25 Alpha ziehen.


Beispiel:

draw_set_alpha(0.5);
draw_set_colour(c_black);
draw_text(x+5, y+5, "LEVEL 1");
draw_set_alpha(1);
draw_set_colour(c_white);
draw_text(x, y, "LEVEL 1");

Der obige Code zeichnet einen Text an der angegebenen Position mit einem Schatteneffekt, der durch das modifizierte Zeichen Alpha und Farbe erzeugt wird.
