draw_set_alpha


Beschreibung

Mit dieser Funktion können Sie das Basis-Draw-Alpha für das Spiel festlegen. Dieser Wert kann von 0 bis 1 festgelegt werden, wobei 0 vollständig transparent und 1 vollständig undurchsichtig ist (Standardwert). Er wirkt sich auf alle weiteren Zeichnungen aus, einschließlich Hintergründe, Sprites, Schriftarten, Grundelemente und 3D.

Beachten Sie Folgendes, wenn Sie eines der Elemente zeichnen und als Teil der Funktion ein Alpha-Argument angeben (z. B. bei der Verwendung von) draw_sprite_ext() ), dann wird der hier eingestellte Alpha-Wert überschrieben, also ein Sprite mit zeichnen draw_sprite_ext Mit einem Alpha-Argument von 1 wird das Sprite unabhängig von der Alpha-Einstellung vollständig undurchsichtig gezeichnet draw_set_alpha (Dies funktioniert auch bei Verwendung von draw_self() und das einstellen image_alpha der Instanz). Beachten Sie auch, dass, wenn eine Instanz eine Standardzeichnung ist (dh: ein Sprite, aber kein Draw-Ereignis), die image_alpha Die Variable wird mit der Alpha-Gruppe geklemmt draw_set_alpha(), so dass ein image_alpha von 1 und einem Draw-Alpha von 0,5 wird das Sprite mit Alpha 0,5 gezeichnet, während ein image_alpha von 0,25 und ein Draw-Alpha von 0,5 zieht das Sprite mit 0,25 Alpha.


Syntax:

draw_set_alpha(alpha);


Streit Beschreibung
alpha Das einzustellende Alpha (zwischen 0 und 1)


Kehrt zurück:

N/A


Beispiel:

draw_set_alpha(0.5);
draw_set_colour(c_black);
draw_text(x+5, y+5, "LEVEL 1");
draw_set_alpha(1);
draw_set_colour(c_white);
draw_text(x, y, "LEVEL 1");

Der obige Code zeichnet an der angegebenen Position Text mit einem Schatteneffekt, der durch das geänderte Zeichnungsalpha und die Farbe erstellt wird.
