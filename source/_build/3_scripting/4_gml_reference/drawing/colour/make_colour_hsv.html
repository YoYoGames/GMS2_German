make_colour_hsv


Beschreibung

GameMaker Studio 2 bietet diese und andere Funktionen, mit denen der Benutzer seine eigenen Farben erstellen kann. Diese spezielle Funktion benötigt drei Komponenten, den Farbton, die Sättigung und den Wert (auch als "Helligkeit" bekannt), um die gewünschte Farbe zu erzeugen. Diese Werte liegen zwischen 0 und 255, so dass Sie damit 16.777.216 (256 * 256 * 256) Farben erstellen können! Unten sehen Sie ein Bild davon, wie diese Komponenten aussehen, wenn sie voneinander getrennt sind:

Das Bild links ist eine Aufschlüsselung der einzelnen Komponenten der Funktion, und rechts ist eine Illustration, wie sich diese Komponenten auf die Endfarbe auswirken. Wie Sie sehen, können Sie auch ohne Änderung des Farbtons (er bleibt auf 0, rot) die Farbe mit den beiden anderen Einstellungen drastisch ändern. Die Sättigung bewirkt, dass der Farbton von grau (0) auf reinen Ton (255) wechselt. und mit dem Wert, der den Wert von dunkler (0) bis heller (255) erhöht.


Syntax:

make_colour_hsv(hue, sat, val);

Streit Beschreibung
hue Der Farbton der Farbe
sat Wie gesättigt ist die Farbe
val Wie dunkel ist die Farbe?


Kehrt zurück:

Int


Beispiel:

col = make_colour_hsv(100, 145, 255);

Der obige Code verwendet die Funktion, um eine Farbe zu erstellen und ihren Wert für die spätere Verwendung in der Variablen "col" zu speichern.
