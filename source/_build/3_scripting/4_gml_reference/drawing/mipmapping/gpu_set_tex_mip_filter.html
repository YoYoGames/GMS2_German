gpu_set_tex_mip_filter


Beschreibung

Mit dieser Funktion können Sie den aktuellen MIP-Filtermodus auf einen der drei unterstützten Typen einstellen. Sie geben die Konstante, die den mip Filtermodus bezeichnet, die Sie benötigen, wo Sie zwischen Punkt Filterung (Default - Einstellung, dh keine Filterung), lineare Filterung (dies wird auch als trilineare Filterung bekannt) oder aniostropic Filterung (siehe Konstanten Tabelle wählen unten).

Konstante Beschreibung
tf_point Dadurch wird die Filterung zwischen Mipmap-Ebenen deaktiviert, was zu sichtbaren Texturübergängen führen kann, aber die beste Leistung erzielt.
tf_linear Dadurch wird eine lineare Filterung zwischen Mipmap-Ebenen (dies wird auch als trilineare Filterung bezeichnet ) aktiviert, wodurch die Texturübergänge geglättet werden, was jedoch die Leistung beeinträchtigt.
tf_anisotropic Dies ermöglicht eine anisotrope Filterung zwischen Mipmap-Ebenen, was die Texturübergangsqualität erheblich verbessert und die Unschärfe, die bei anderen Filtermodi sichtbar ist, reduzieren kann. Die Performance wird jedoch am meisten beeinträchtigt.


Syntax:

gpu_get_tex_mip_filter(filter);

Streit Beschreibung
filter Der zu verwendende MIP-Filtermodus (eine Konstante, Standardeinstellung: tf_point ).


Kehrt zurück:

N/A


Beispiel:

if keyboard_check(ck_enter)
    {
    switch(gpu_get_tex_mip_filter())
        {
        case tf_point: gpu_set_tex_mip_filter(tf_linear); break;
        case tf_linear: gpu_set_tex_mip_filter(tf_anisotropic); break;
        case tf_anisotropic: gpu_set_tex_mip_filter(tf_point); break;
        }
    }

Der obige Code prüft die Tastatur und wenn die angegebene Taste gedrückt wird, wird der aktuelle Mip-Filter abgerufen und der Wert zum nächsten umgeschaltet, wobei die verschiedenen Modi durchlaufen werden.
