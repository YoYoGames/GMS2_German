gpu_set_tex_mip_enable_ext


Beschreibung

Mit dieser Funktion können Sie einstellen, ob das Mipmapping für alle Texturgruppen, die im Texturgruppen- Manager auf einem Shader-Sampler ausgewählt wurden, ausgeschaltet, für alles eingeschaltet oder eingeschaltet ist. Sie geben den Indexwert für den Shader-Sampler an (wie von der Funktion zurückgegeben) shader_get_sampler_index() und dann eine der unten aufgeführten Konstanten.

Konstante Beschreibung
mip_off Das Mipmapping ist deaktiviert.
mip_on Das Mipmapping für alle Texturen ist aktiviert.
mip_markedonly Das Mipmapping ist für Texturen aktiviert, für die es in den Optionen der Texturgruppe aktiviert ist (Standard).


Syntax:

gpu_set_tex_mip_enable_ext(sampler_index, setting);


Streit Beschreibung
sampler_index Der Index des Shader Samplers wird abgerufen
setting Die Mipmap-Einstellung (eine Konstante, Standard: mip_markedonly )


Kehrt zurück:

N/A


Beispiel:

Beispiel:

var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background");
var _spriteTex = sprite_get_texture(sprite_index, 0);
shader_set(shd_Glass);
if gpu_get_tex_mip_enable_ext(_sampleIndex) != mip_on
    {
    gpu_set_tex_mip_enable_ext(_sampleIndex, mip_on);
    }
texture_set_stage(_sampleIndex , _spriteTex);
shader_reset();

Der obige Code aktiviert das Mipmapping für den gegebenen Shader-Textur-Sampler, wenn er nicht bereits aktiviert wurde.
