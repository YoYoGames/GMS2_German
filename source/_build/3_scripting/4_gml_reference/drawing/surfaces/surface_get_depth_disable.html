surface_get_depth_disable


Syntax:

surface_get_depth_disable();


Kehrt zurück:

Boolean


Beschreibung

Diese Funktion prüft, ob die automatische Tiefenpuffergenerierung für alle Flächen aktiviert ist. Normalerweise verfügen alle Oberflächen über Tiefenpuffer. Wenn Sie 3D-Objekte zeichnen, werden die Objekte dann nach Tiefe sortiert. Durch das Zuweisen von Tiefenpuffern wird jedoch die Größe der Flächen im Wesentlichen verdoppelt. Dies kann ein übermäßiger und unnötiger Aufwand sein, insbesondere wenn Ihr Spiel sehr viel Arbeitsspeicher bietet intensiv. In diesem Fall können Sie dies mit dieser Funktion überprüfen und ggf. den Tiefenpuffer für Flächen mit der Funktion deaktivieren surface_depth_disable().

HINWEIS: Diese Funktion ist derzeit nur für die Sony Playstation 4 Zielplattform gültig.


Beispiel:

if surface_get_depth_disable() == false
    {
    surface_depth_disable(true);
    }

Der obige Code prüft den aktuellen Status des Puffers für die Oberflächentiefe. Wenn er aktiviert ist, wird er stattdessen deaktiviert.
