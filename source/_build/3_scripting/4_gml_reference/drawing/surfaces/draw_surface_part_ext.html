draw_surface_part_ext


Syntax:

draw_surface_part_ext(id, left, top, w, h, x, y, xscale, yscale, colour, alpha);

Streit Beschreibung
id Der eindeutige ID-Wert der zu zeichnenden Oberfläche.
left Die linke Position in der Oberfläche des zu zeichnenden Teils.
top Die oberste Position in der Oberfläche des zu zeichnenden Teils.
w Die Breite des zu zeichnenden Teils von links.
h Die Höhe des zu zeichnenden Teils von oben.
x Die x-Position, an der die Oberfläche gezeichnet werden soll.
y Die y-Position, in der die Oberfläche gezeichnet werden soll.
xscale Die horizontale Skalierung des Teils sollte mit gezeichnet werden.
yscale Die vertikale Skalierung des Teils sollte mit gezeichnet werden.
colour Die Farbe, mit der das Teil verschnitten wird, sollte gezeichnet werden.
alpha Die Alpha Transparenz, mit der das Teil gezeichnet werden soll.


Kehrt zurück:

N/A


Beschreibung

Diese Funktion zeichnet einen Teil der ausgewählten Oberfläche an der angegebenen Position nach den gleichen Regeln wie nach draw_surface_part Jetzt können Sie das Teil skalieren, eine Farbe mit ihm mischen oder das Alpha ändern, wenn Sie es auf den Bildschirm zeichnen (wie beim Zeichnen einer Fläche mit draw_surface_ext ).

HINWEIS: Beim Arbeiten mit Oberflächen besteht die Möglichkeit, dass sie aufgrund ihrer Speicherung im Texturspeicher jederzeit nicht mehr existieren können. Sie sollten IMMER überprüfen, ob eine Oberfläche vorhanden ist surface_exists bevor Sie direkt auf sie verweisen. Weitere Informationen finden Sie unter Oberflächen.


Beispiel:

draw_surface_part_ext(surf, 8, 8, 32, 32, x, y, 2, 0.5, c_black, 1);

Dadurch wird ein 32x32-Pixelbereich von 8x8 Pixel in die in der Variablen "surf" indizierte Oberfläche gezeichnet. Es wird gedehnt, um seine übliche Breite zu verdoppeln, aber die Hälfte seiner üblichen Höhe. Es wird undurchsichtig sein, und es wird mit Schwarz gemischt (um es in eine Silhouette zu verwandeln).
