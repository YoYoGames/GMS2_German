Oberflächenkopie_part


Syntax:

surface_copy_part(destination, x, y, source, xs, ys, ws, hs);

Streit Beschreibung
destination Die ID der Oberfläche, auf die die andere Oberfläche kopiert werden soll.
x Die x-Position, in die kopiert werden soll.
y Die y-Position, in die kopiert werden soll.
source Die ID der zu kopierenden Oberfläche.
xs Die x-Position in der Quelloberfläche, aus der kopiert werden soll.
ys Die y-Position in der Quelloberfläche, von der kopiert werden soll.
ws Die Breite des Bereichs in der Quelloberfläche, von der kopiert werden soll.
hs Die Höhe des Bereichs in der Quelloberfläche, von der kopiert werden soll.


Kehrt zurück:

N/A


Beschreibung

Diese Funktion nimmt das Bild einfach von einer Oberfläche und kopiert es auf eine andere an der angegebenen lokalen Position innerhalb dieser Oberfläche (wobei die Position (0,0) die obere linke Ecke der Zieloberfläche ist). Sie können eine lokale x- und y-Position angeben, von der kopiert werden soll, sowie die Breite und Höhe des Abschnitts. Bitte beachten Sie, dass dies Koordinaten sind, die auf der Oberflächengröße basieren und nicht auf der Position, an der die Oberfläche im Raum gezeichnet wird (weitere Informationen finden Sie auf der Hauptseite zu Oberflächen ). Wenn die Zieloberfläche bereits Informationen hat, wird diese durch die Kopie überschrieben werden, und die Funktion der Quelloberfläche nicht in irgendeiner Weise ändern.

HINWEIS: Beim Arbeiten mit Oberflächen besteht die Möglichkeit, dass sie aufgrund ihrer Speicherung im Texturspeicher jederzeit nicht mehr existieren können. Sie sollten IMMER überprüfen, ob eine Oberfläche vorhanden ist surface_exists bevor Sie direkt auf sie verweisen. Weitere Informationen finden Sie unter Oberflächen.


Beispiel:

if view_current == 0
   {
   surface_copy_part(surf, 0, 0, temp_surf, 0, 0, view_xview[1] - mouse_x, view_yview[1] - mouse_y);
   }
else
   {
   draw_surface(surf, 0, 0);
   }

Der obige Code überprüft die aktuelle Ansicht und wenn es view [0] ist, kopiert er die in der Variablen "temp_surf" indizierte Oberfläche auf die Oberfläche, die in der Variable "surf" indiziert ist. Der kopierte Bereich entspricht einem Rechteck, das durch die relative Position der Maus innerhalb der Oberfläche gebildet wird, wie sie in Ansicht [1] gezeichnet wäre. Wenn die aktuelle Ansicht etwas anderes als Ansicht [0] ist, wird die Oberfläche "Surfen" auf den Bildschirm gezeichnet.
