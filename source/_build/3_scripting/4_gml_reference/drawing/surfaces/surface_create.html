Oberflächen_Erstellen


Syntax:

surface_create(w, h);

Streit Beschreibung
w Die Breite der zu erstellenden Oberfläche.
h Die Höhe der zu erstellenden Oberfläche.


Kehrt zurück:

Real


Beschreibung

Diese Funktion wird zum Erstellen einer Oberfläche verwendet und gibt den Index der Oberfläche zurück, der in einer Variablen für zukünftige Funktionsaufrufe gespeichert werden soll. Wenn die Oberfläche zum ersten Mal erstellt wird, enthält sie möglicherweise "Rauschen", da es sich im Grunde nur um einen Speicherbereich handelt, der für diesen Zweck zur Seite gelegt wird (und der Speicher möglicherweise noch Informationen enthält). Daher empfiehlt es sich, die Oberfläche vor der Verwendung zu löschen eine Funktion wie draw_clear_alpha.

Es wird dringend empfohlen, alle Oberflächen mit einer Stärke von 2 zu erstellen, z. B. 16, 128, 512 oder 1024 Pixel. Dies ist nicht unbedingt notwendig auf bestimmten Plattformen (wie Windows und MacOS), aber es wird sicherlich die Kompatibilität auf diesen Zielen erhöhen, während für HTML5 und Geräte es wichtig ist und es sehr wichtig ist, dass Sie sich daran erinnern oder später Probleme auftreten.


Beispiel:

if !surface_exists(surf)
   {
   surf = surface_create(1024, 1024);
   surface_set_target(surf);
   draw_clear_alpha(c_black, 0);
   surface_reset_target();
   view_surface_id[0] = surf;
   }

Der obige Code überprüft, ob eine Oberfläche existiert, und wenn dies nicht der Fall ist, wird eine Oberfläche erzeugt, die 1024 Pixel breit und 1024 Pixel hoch ist, wobei der Index der Variablen "Surf" zugewiesen wird. Das Zeichenziel wird dann auf die neue Oberfläche gesetzt, die gelöscht und transparent gemacht wird, bevor das Zeichenziel auf die Anzeige zurückgesetzt wird. Abschließend wird die Oberfläche einer Ansicht zugewiesen.
