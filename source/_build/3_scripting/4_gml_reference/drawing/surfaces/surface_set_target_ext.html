Oberflächensatz_ziel_ext


Syntax:

surface_set_target_ext(index, surface_id);

Streit Beschreibung
index Der zu verwendende Renderzielindex (von 0 bis 3).
surface_id Die ID der zu verwendenden Oberfläche.


Kehrt zurück:

Boolean


Beschreibung

Diese Funktion wird für die Shader-Funktionen verwendet und legt den MRT (0 - 3) für native Level-Shader fest (DX9, DX11, OpenGL).

HINWEIS: Beim Arbeiten mit Oberflächen besteht die Möglichkeit, dass sie aufgrund ihrer Speicherung im Texturspeicher jederzeit nicht mehr existieren können. Sie sollten IMMER überprüfen, ob eine Oberfläche vorhanden ist surface_exists bevor Sie direkt auf sie verweisen. Weitere Informationen finden Sie unter Oberflächen.


Beispiel:

surface_set_target_ext(0, surf);

Der obige Code setzt das Renderziel 0 auf die in der Variablen "surf" indizierte Oberflächen-ID.
