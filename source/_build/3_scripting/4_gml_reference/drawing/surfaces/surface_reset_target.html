Oberflächenziel_ziel


Syntax:

surface_reset_target();


Kehrt zurück:

N/A


Beschreibung

Mit dieser Funktion setzen Sie alle weiteren Zeichnungen von der Zielfläche zurück auf den Bildschirm. Bitte beachten Sie, dass Sie die Funktion zuerst aufgerufen haben müssen, um mit dem Zeichnen auf eine Oberfläche zu beginnen surface_set_target und dann, wenn Sie fertig sind, für jedes von Ihnen festgelegte Oberflächenziel, sonst wird nichts auf dem Bildschirm gezeichnet, da alle weiteren Zeichnungen (auch in anderen Fällen) auf der Oberfläche ausgeführt werden. Sie sollten auch erkennen, dass nichts zu sehen sein wird, wenn die Oberfläche selbst im Draw-Ereignis einer Instanz nicht auf dem Bildschirm gezeichnet wird.

HINWEIS: Wenn Sie zuvor kein Renderziel mit der Funktion festgelegt haben surface_set_target Mit dieser Funktion wird automatisch (dh ohne Fehlermeldung) die weitere Codeausführung für das Ereignis beendet.


Beispiel:

if view_current = 0
   {
   surface_set_target(surf);
   with (obj_Effect)
      {
      draw_self();
      }
   surface_reset_target();
   }
else
   {
   draw_surface(surf, 0, 0);
   }

Der obige Code überprüft, welche Ansicht gerade gezeichnet wird, und wenn es view [0] ist, setzt er das Zeichenziel auf eine Fläche und zeichnet alle Instanzen des Objekts "obj_Effect", bevor das Zeichenziel erneut zurückgesetzt wird. Wenn die Ansicht nicht Ansicht [0] ist, wird die Oberfläche auf den Bildschirm gezeichnet.
