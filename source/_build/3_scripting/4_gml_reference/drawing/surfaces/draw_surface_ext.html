draw_surface_ext


Syntax:

draw_surface_ext(id, x, y, xscale, yscale, rot, col, alpha);

Streit Beschreibung
id Der eindeutige ID-Wert der zu zeichnenden Oberfläche.
x Die x-Position, an der die Oberfläche gezeichnet werden soll.
y Die y-Position, in der die Oberfläche gezeichnet werden soll.
xscale Die horizontale Skala.
yscale Die vertikale Skala.
rot Die Drehung oder der Winkel, um die Oberfläche zu zeichnen.
col Die Farbe, mit der die Oberfläche gemischt werden soll.
alpha Die Alpha-Transparenz zum Zeichnen der Oberfläche.


Kehrt zurück:

N/A


Beschreibung

Diese Funktion zeichnet die angegebene Oberfläche wie in der Funktion draw_surface aber mit zusätzlichen Optionen zum Ändern der Skalierung, Überblendung, Rotation und Alpha der gezeichneten Oberfläche. Ändern dieser Werte nicht die Ressource in irgendeiner Weise ändern (nur, wie es gezeichnet wird).

HINWEIS: Beim Arbeiten mit Oberflächen besteht die Möglichkeit, dass sie aufgrund ihrer Speicherung im Texturspeicher jederzeit nicht mehr existieren können. Sie sollten IMMER überprüfen, ob eine Oberfläche vorhanden ist surface_exists bevor Sie direkt auf sie verweisen. Weitere Informationen finden Sie unter Oberflächen.


Beispiel:

draw_surface_ext(surf, 0, 0, 2, 2, 0, c_red, 0.5);

Der obige Code zeichnet eine Oberfläche, die in der Variablen "surf" an der Position (0,0) im Raum indiziert ist, mit der doppelten Originalskala, gemischt rot und halbtransparent.
