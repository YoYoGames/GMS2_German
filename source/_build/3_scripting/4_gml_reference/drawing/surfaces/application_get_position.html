application_get_position


Syntax:

application_get_position();


Kehrt zurück:

Array


Beschreibung

Wenn Sie in den Spieloptionen für eine Zielplattform "Seitenverhältnis beibehalten" aktiviert haben, setzt GameMaker Studio 2 automatisch die Zeichenposition für die Anwendungsoberfläche so, dass sie korrekt zentriert und auf der angegebenen Anzeige skaliert wird. Wenn Sie diese Oberfläche jedoch bearbeiten und selbst zeichnen möchten, können Sie mit dieser Funktion genau herausfinden, wo in der Anzeige oder im Fenster die Oberfläche gezeichnet wurde, damit Sie sie dort selbst zeichnen oder die Benutzeroberfläche ausrichten können Bilder oder Bilder nachzeichnen.

Die Funktion gibt ein Array mit vier Tasten zurück, wobei die Tasten 0 und 1 die x- und y-Position der oberen linken Ecke der Oberfläche und die Tasten 2 und 3 das x und y der unteren rechten Ecke der Oberfläche sind relativ zur Größe der Anzeige oder des Fensters.


Beispiel:

var a = application_get_position();
xx = a[0];
yy = a[1];
ww = a[2] - a[0];
hh = a[3] - a[1];

Der obige Code wird die Position der Anwendungsoberfläche sowie die absolute Breite und Höhe in Bezug auf das Anzeigefenster erhalten und sie für zukünftige Verwendung in vier Variablen speichern.
