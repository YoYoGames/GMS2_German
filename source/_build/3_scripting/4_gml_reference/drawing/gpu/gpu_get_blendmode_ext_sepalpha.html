gpu_get_blendmode_ext_sepalpha


Beschreibung

Diese Funktion kann verwendet werden, um den aktuellen erweiterten Mischmodus, der zum Zeichnen verwendet wird, abzurufen und die Mischmodi in zwei Teile zu trennen - die RGB-Komponente und die Alpha-Komponente. Die Funktion gibt ein 4-Elemente-1D-Array mit folgenden Elementen zurück:

  • [0] = Quellmischmodus minus Alphakomponente (Standard ist bm_src_alpha (Wie, als, wie))
  • [1] = Ziel-Mischmodus minus Alphakomponente (Standard ist bm_inv_src_alpha (1-As, 1-As, 1-As))
  • [2] = Quell-Alpha-Komponente (Standard ist bm_src_alpha (Wie))
  • [3] = Ziel-Alpha-Komponente (Standard ist bm_inv_src_alpha (1-As))

Die Werte des Arrays werden eine der folgenden Konstanten sein ("s" bezeichnet einen Wert aus der Quelle, während "d" einen Wert aus dem Ziel angibt):

Konstante Mischfaktor (Rot, Grün, Blau, Alpha)
bm_zero (0, 0, 0, 0)
bm_one (1, 1, 1, 1)
bm_src_colour (Rs, Gs, Bs, As)
bm_inv_src_colour (1-Rs, 1-Gs, 1-Bs, 1-As)
bm_src_alpha (Wie, als, wie, als)
bm_inv_src_alpha (1-As, 1-As, 1-As, 1-As)
bm_dest_alpha (Anzeige, Anzeige, Anzeige, Anzeige)
bm_inv_dest_alpha (1-Ad, 1-Ad, 1-Ad, 1-Ad)
bm_dest_colour (Rd, Gd, Bd, Ad)
bm_inv_dest_colour (1-Rd, 1-Gd, 1-Bd, 1-Ad)
bm_src_alpha_sat (f, f, f, 1) mit f = min (As, 1-Ad)

Wenn Sie auf diese Weise die erweiterten Mischmodi erstellen, werden die RGB-Komponenten aus einem Mischmodus und die Alphakomponente aus einem anderen ausgewählt, so dass Sie beispielsweise eine andere Mischform verwenden bm_zero (0, 0, 0, 0) als RGB-Mischmodus und bm_one Als Alpha-Mischmodus erhalten Sie eine Mischung aus (0,0,0,1).

Beachten Sie, dass Sie diese Werte direkt innerhalb des Arrays ändern und das vollständige Array als einzelnes Argument an die Funktion übergeben können (wie im folgenden Beispiel gezeigt).

WARNUNG! Diese Funktion ist derzeit nur auf der Windows- Zielplattform verfügbar.


Syntax:

gpu_get_blendmode_ext_sepalpha();


Kehrt zurück:

4 element 1D Array (see above for constants)


Beispiel:

var bm = gpu_get_blendmode_ext_sepalpha();
bm[0] = bm_src_alpha;
bm[1] = bm_inv_src_alpha;
gpu_set_blendmode_ext_sepalpha(bm);

Der obige Code erstellt ein lokales Array und ruft den aktuellen erweiterten Mischmodus mit separaten Alpha-Werten ab. Dieses Array wird dann manipuliert und zurückgegeben, um den Mischmodus einzustellen.
