gpu_set_texfilter


Beschreibung

Diese Funktion kann verwendet werden, um die Texturfilterung (lineare Interpolation) aller auf dem Spielbildschirm gezeichneten Bilder einzustellen. Wenn dies aktiviert ist ( true ) Alle Texturen werden beim Zeichnen geglättet (dies schließt auch Sprites ein, da sie auch als Texturen gelten), was bedeutet, dass beim Skalieren oder Verschieben, wenn kein 1: 1 Pixelverhältnis vorhanden ist, ein "Verschmieren" über verschiedene Pixel hinweg erfolgt Bilder erscheinen verschwommen, abhängig vom verwendeten Kunststil. Wenn dies deaktiviert ist ( false ) dann werden Bilder basierend auf dem nächsten Pixel gezeichnet, wenn skaliert oder bewegt wird, was zu "blockartigen" Bildern führen kann. Der Standardwert ist false und dies kann auch in den Global Game Options für einzelne Zielplattformen geändert werden.

HINWEIS: Diese Einstellung überschreibt alle Texturstufeninterpolationseinstellungen für Shader, die die Funktion verwenden gpu_set_texfilter_ext.


Syntax:

gpu_set_texfilter(enable);


Argumente:

Streit Beschreibung
enable Aktivieren oder Deaktivieren der Texturfilterung ( true / false )


Kehrt zurück:

N/A


Beispiel:

if gpu_get_texfilter()
   {
   gpu_set_texfilter(false);
   }
else
   {
   gpu_set_texfilter(true);
   }

Der obige Code überprüft, ob die Texturfilterung aktiviert oder deaktiviert ist und schaltet sie entsprechend um.
