gpu_set_ztestenable


Beschreibung

Mit dieser Funktion kann der Z-Buffer-Test ein- oder ausgeschaltet werden, was sich auf die Art und Weise des Zeichnens auswirkt (im Allgemeinen nur bei der Arbeit mit 3D-Projekten). Im Grunde genommen werden beide Objekte gerendert, wenn der Z-Test deaktiviert ist und Sie zwei Objekte auf den gleichen Platz zeichnen, unabhängig davon, ob der eine das andere überzeichnet, was zu unnötigen Zeichenaufrufen führt. Wenn Sie dies einschalten, wird der Z-Buffer getestet, um zu sehen, ob ein Objekt "sichtbar" ist und nicht gezeichnet wird, wenn dies nicht der Fall ist. Beachten Sie, dass dies das Standardverhalten ist. Sie können dies jedoch ändern, indem Sie den für den Z-Buffer-Test verwendeten Vergleichstyp ändern (siehe Funktion) gpu_set_zfunc.

Standardmäßig ist der Z-Buffer-Test deaktiviert ( false ).


Syntax:

gpu_set_ztestenable(enable);


Argumente:

Streit Beschreibung
enable Aktivieren oder Deaktivieren von Z-Buffer-Tests ( true oder false ).


Kehrt zurück:

N/A


Beispiel:

gpu_set_ztestenable(true);
draw_sprite(spr_Background, 0, 0, 0);
gpu_set_ztestenable(false);

Der obige Code schaltet den Z-Buffer-Test ein, um ein Hintergrund-Sprite zu zeichnen, und schaltet ihn dann wieder aus, um mit dem Zeichnen fortzufahren.
