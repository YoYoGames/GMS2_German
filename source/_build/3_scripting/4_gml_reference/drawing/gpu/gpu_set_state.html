gpu_set_state


Beschreibung

Diese Funktion wird den aktuellen GPU-Status mit dem übergebenen festlegen ds_map. Die mitgelieferte Karte kann mit der Funktion erstellt werden gpu_get_state


Syntax:

gpu_set_state(ds_map);


Argumente:

Streit Beschreibung
ds_map Der GPU-Status, der als a festgelegt wird ds_map.


Kehrt zurück:

N/A


Beispiel:

gpu_set_state(gpu_map);

Der obige Code setzt den GPU-Status unter Verwendung der in der Variablen "gpu_map" bereitgestellten Abbildung.
