GPU-Steuerung

Wenn Sie Objekte in einem Raum zeichnen, benötigen Sie oft eine genauere Kontrolle darüber, wie genau gezeichnet wird. Möglicherweise möchten Sie, dass dort keine Alpha-Überblendung vorhanden ist, oder Sie müssen möglicherweise aussortiert werden, oder Sie müssen möglicherweise die Texturinterpolation ein- oder ausschalten. All das ist möglich - und noch viel mehr! - Verwenden Sie die unten aufgeführten GPU-Controller-Funktionen. Beachten Sie, dass die meisten dieser Funktionen einen globalen Geltungsbereich haben und dass, sobald sie festgelegt oder geändert wurden, sich darauf auswirken wird, wie alles von da an gezeichnet wird.

Die folgenden Funktionen können verwendet werden, um die Werte verschiedener GPU-Eigenschaften abzurufen:

  • gpu_get_blendenable
  • gpu_get_ztestenable
  • gpu_get_zfunc
  • gpu_get_zwriteenable
  • gpu_get_fog
  • gpu_get_cullmode
  • gpu_get_blendmode
  • gpu_get_blendmode_ext
  • gpu_get_blendmode_ext_sepalpha
  • gpu_get_blendmode_src
  • gpu_get_blendmode_dest
  • gpu_get_blendmode_srcalpha
  • gpu_get_blendmode_telepha
  • gpu_get_colorwriteenable
  • gpu_get_alphatestenable
  • gpu_get_alphetesref
  • gpu_get_texfilter
  • gpu_get_texfilter_ext
  • gpu_get_texrepeat
  • gpu_get_texrepeat_ext

Die folgenden Funktionen können verwendet werden, um die Werte verschiedener GPU-Eigenschaften festzulegen:

  • gpu_set_blendenable
  • gpu_set_ztestenable
  • gpu_set_zfunc
  • gpu_set_zwriteenable
  • gpu_set_fog
  • gpu_set_cullmode
  • gpu_set_blendmode
  • gpu_set_blendmode_ext
  • gpu_set_blendmode_ext_sepalpha
  • gpu_set_colorwriteenable
  • gpu_set_alphatestenable
  • gpu_set_alphadestref
  • gpu_set_texfilter
  • gpu_set_texfilter_ext
  • gpu_set_texrepeat
  • gpu_set_texrepeat_ext

Mit diesen Funktionen kann der GPU-Stack manipuliert werden:

  • gpu_push_state
  • gpu_pop_state
  • gpu_get_state
  • gpu_set_state
