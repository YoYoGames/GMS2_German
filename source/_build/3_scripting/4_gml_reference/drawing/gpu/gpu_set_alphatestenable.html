gpu_set_alphatestenable


Beschreibung

Diese Funktion aktiviert oder deaktiviert den Alphatest für dein Spiel (standardmäßig ist dies deaktiviert). Wenn Sie den Alphatest einschalten, können Sie die Begleitfunktion verwenden gpu_set_alphatestref um den "Cut-Off" -Wert einzustellen, bei dem alle Alpha-Werte auf 0 gesetzt werden. Das Bild unten zeigt den Unterschied, der beim Ein- oder Ausschalten des Alphatests zu sehen ist:

HINWEIS: Diese Funktion kann die Leistung von iOS-, Android und Windows Phone Geräten beeinträchtigen.


Syntax:

gpu_set_alphatestenable(enable);


Argumente:

Streit Beschreibung
enable Aktiviere oder deaktiviere Alphatests ( true / false )


Kehrt zurück:

N/A


Beispiel:

if !gpu_get_alphatestenable()
   {
   gpu_set_alphatestenable(true);
   gpu_set_alphatestref(128);
   }

Der obige Code wird prüfen, ob der Alphatest aktiviert ist, und wenn nicht, schaltet er den Alphatest ein und setzt den Testschwellenwert auf 128 (nur Pixel mit einem Alpha über 0,5 werden gezeichnet).
