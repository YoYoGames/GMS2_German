gpu_get_blendmode_ext


Beschreibung

Diese Funktion kann verwendet werden, um den aktuellen erweiterten Mischmodus abzurufen, der zum Zeichnen verwendet wird. Die Funktion gibt ein 2-Element-1D-Array mit den folgenden Elementen zurück:

  • [0] = Quell-Mischmodus (Standardeinstellung ist bm_src_alpha )
  • [1] = Ziel-Mischmodus (Standardeinstellung ist bm_inv_src_alpha )

Die Werte des Arrays sind eine der folgenden Konstanten ("s" bezeichnet einen aus der Quelle entnommenen Wert, während ein "d" einen Wert aus dem Ziel angibt):

Konstante Mischungsfaktor (Rot, Grün, Blau, Alpha)
bm_zero (0, 0, 0, 0)
bm_one (1, 1, 1, 1)
bm_src_colour (Rs, Gs, Bs, As)
bm_inv_src_colour (1-Rs, 1-Gs, 1-Bs, 1-As)
bm_src_alpha (As, As, As, As)
bm_inv_src_alpha (1-As, 1-As, 1-As, 1-As)
bm_dest_alpha (Anzeige, Anzeige, Anzeige, Anzeige)
bm_inv_dest_alpha (1-Ad, 1-Ad, 1-Ad, 1-Ad)
bm_dest_colour (Rd, Gd, Bd, Ad)
bm_inv_dest_colour (1-Rd, 1-Gd, 1-Bd, 1-Ad)
bm_src_alpha_sat (f, f, f, 1) wobei f = min (As, 1-Ad)

Beachten Sie, dass Sie diese Werte ändern und das gesamte Array an das übergeben können gpu_set_blendmode_ext Funktion als einzelnes Argument.


Syntax:

gpu_get_blendmode_ext();


Kehrt zurück:

2 element 1D Array (see above for constants)


Beispiel:

var bm = gpu_get_blendmode_ext();
bm[0] = bm_src_alpha;
gpu_set_blendmode_ext(bm);

Der obige Code ruft den aktuellen erweiterten Mischmodus ab, ändert die Quelle und stellt dann den erweiterten Mischmodus erneut ein.
