gpu_set_fog


Beschreibung

Diese Funktion kann zum Aktivieren oder Deaktivieren des Nebelzeichnens verwendet werden. Nebel kann in 3D-Spielen verwendet werden, um Instanzen in der Ferne unscharf erscheinen zu lassen oder sogar verschwinden zu lassen. Dies hilft beim Erzeugen von Atmosphäre und maskiert die Tatsache, dass Sie keine Instanzen zeichnen, die weit entfernt sind. Sie legen fest, ob es aktiviert ist ( true ) oder deaktiviert ( false ), die Farbe, die der Nebel zum Überblenden verwenden soll, sowie die Start- und Endstrecken. Beachten Sie, dass der Nebelanfang den Radius relativ zu der Ansichtskamera anzeigt, an der der Nebel beginnt, und das Nebelende anzeigt, bei welchem ​​Radius (auch relative) Instanzen vollständig unsichtbar sind.

Die Funktion kann vier einzelne Argumente (wie oben angegeben) oder ein Array mit der folgenden Struktur annehmen (der folgende Beispielcode zeigt diese Methode):

  • [0] = aktiviert toggle (ein Boolescher, entweder true oder false ), Standard false
  • [1] = Farbe (echt), Standard c_black
  • [2] = Startdistanz (real), Standard 0
  • [3] = Endabstand (real), Standard 1


Syntax:

gpu_set_fog(enable, colour, start, end);


Argumente:

Streit Beschreibung
enable Aktivieren oder deaktivieren Sie den Nebel
colour Die Nebelfarbe
start Die Entfernung zum Beginn der Anwendung von Nebel aus
end Die Entfernung, um den Nebel zu beenden


Kehrt zurück:

N/A


Beispiel:

var fog_a = gpu_get_fog();
fog_a[1] = c_red;
gpu_set_fog(fog_a);

Der obige Code ruft die aktuellen Nebeleinstellungen ab und setzt dann das Farbelement des Arrays auf c_red bevor Sie den Nebel mit dem geänderten Array erneut einstellen.
