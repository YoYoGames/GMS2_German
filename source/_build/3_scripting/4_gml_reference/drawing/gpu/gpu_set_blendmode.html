gpu_set_blendmode


Beschreibung

Wenn GameMaker Studio 2 ein Pixel zeichnet, gibt es eine Quellfarbe (die Farbe des Pixels, die wir zeichnen werden) und eine Zielfarbe (die Farbe, die bereits in dem Pixel ist, zu dem wir zeichnen) Neue Farbe des Pixels, die Quell- und Zielfarben werden entsprechend dem ausgewählten Mischmodus berechnet. Jede Komponente der Farben wird als Gleitkommawert zwischen 0 und 1 gespeichert, und die neue Farbe wird berechnet, indem jede Komponente der Quellfarbe um einen Faktor multipliziert wird und indem jede Komponente der Zielfarbe mit einem anderen Faktor multipliziert und dann addiert wird ergibt zusammen Komponente für Komponente.

Mit dieser Funktion können Sie den Mischmodus auf vier grundlegende Arten unter Verwendung der folgenden Konstanten einstellen:

Konstante Beschreibung Erweiterter Mischmodus
bm_normal Normales Mischen (der Standard-Mischmodus). ( bm_src_alpha, bm_inv_src_alpha )
bm_add Additive Mischung. Helligkeitswerte von hellen Bereichen werden hinzugefügt. ( bm_src_alpha, bm_one )
bm_subtract Subtraktive Mischung. Helligkeitswerte von hellen Bereichen werden subtrahiert. ( bm_zero, bm_inv_src_colour )
bm_max Maximale Mischung. Ähnlich wie beim additiven Mischen. ( bm_src_alpha, bm_inv_src_colour )

Wie Sie der obigen Tabelle entnehmen können, handelt es sich bei diesen vier Mischmodi wirklich um Composites von erweiterten Mischmodi, die auf der Seite beschrieben werden gpu_set_blendmode_ext.


Syntax:

gpu_set_blendmode(mode);


Argumente:

Streit Beschreibung
mode Der zu verwendende Mischmodus (siehe obige Tabelle)


Kehrt zurück:

N/A


Beispiel:

gpu_set_blendmode(bm_add);
draw_circle_colour(100, 100, 50, c_white, c_black, 0);
gpu_set_blendmode(bm_normal);

Dies wird das Schwarz in Transparenz verwandeln und einen "Leucht" -Effekt erzeugen, da das Weiß außen stark ist und vom Kreismittelpunkt aus immer schwächer wird. Der Mischmodus wird nach dem Zeichnen des Kreises zurückgesetzt, um sicherzustellen, dass die additive Mischung nicht auch auf alle anderen Elemente angewendet wird.
