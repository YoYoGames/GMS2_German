gpu_set_texrepeat


Beschreibung

Diese Funktion kann verwendet werden, um GameMaker Studio 2 anzuzeigen, ob sich Texturen wiederholen sollen ( true ) oder nicht ( false ) wenn sie für Modelle und Primitive verwendet werden. Normalerweise liegen Texturbreiten und -höhen im Bereich von 0-1, aber wenn Sie standardmäßig einen Wert größer als 1 angeben, wird der Rest nicht gezeichnet (die gesamte Textur wird mit einem "leeren" Abschnitt gezeichnet, der dem Prozentsatz über 1 entspricht Also würde ein Wert von 2 für eine Texturbreite oder -höhe die Textur in der Hälfte des Raums zeichnen und die andere Hälfte leer lassen. Wenn Sie diese Funktion jedoch verwenden, um "repeat" auf "true" zu setzen, wird die Textur automatisch für alle Modelle und Grundelemente wiederholt.

HINWEIS: Sprites und Hintergründe werden immer gezeichnet, ohne sich zu wiederholen. Wenn Sie also ein Sprite oder einen Hintergrund zeichnen, wird dieser Wert auf zurückgesetzt false. Dieses Standardverhalten wirkt sich nicht auf die Einstellungen der Texturstufe aus (dh: die Funktion wird aufgerufen, aber die interne Einstellung wird nicht auf false gesetzt).
HINWEIS: Diese Einstellung überschreibt alle Texturstufenwiederholungen, die für Shader bei Verwendung der Funktion festgelegt wurden gpu_set_texrepeat_ext.


Syntax:

gpu_set_texrepeat(enable);


Argumente:

Streit Beschreibung
enable Aktivieren oder Deaktivieren der Texturfilterung ( true / false )


Kehrt zurück:

N/A


Beispiel:

if gpu_get_texrepeat()
   {
   gpu_set_texrepeat(false);
   }
else
   {
   gpu_set_texrepeat(true);
   }

Der obige Code überprüft, ob Texturwiederholung aktiviert oder deaktiviert ist und schaltet sie entsprechend um.
