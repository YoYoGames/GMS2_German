draw_light_get


Beschreibung

Diese Funktion erhält die angegebenen Lichtparameter als Array mit den folgenden 6 Elementen -

  • [0] = aktiviert / deaktiviert ( true / false )
  • [1] = Lichtart (siehe untenstehende Konstanten)
  • [2] = x Position
  • [3] = y-Position
  • [4] = z-Position
  • [5] = Lichtradius (nur für Punktlichter, wird 0 für Richtung)
  • [6] = helle Farbe (ein echter)

Der Leuchtentyp kann eine der folgenden zwei Konstanten sein:

Konstante Beschreibung
lighttype_dir Das Licht ist ein gerichtetes Licht
lighttype_point Das Licht ist ein Punktlicht


Syntax:

draw_light_get(ind)


Argumente:

Streit Beschreibung
ind Die Indexnummer des Lichts (von 0 bis 7)


Kehrt zurück:

Real


Beispiel:

light_a = draw_light_get(1);
if light_a[5] < 200
   {
   light_a[5] += 5;
   draw_light_define_point(1, 200, 123, 50, light_a[5], c_white);
   }

Der obige Code wird die Werte erhalten, die verwendet werden, um das Licht zu definieren, das als "1" indiziert wird, dann prüfe den Radius und wenn er kleiner als 200 ist, wird er erhöht und der Lichtradius auf den neuen Wert gesetzt.
