draw_sprite

Zeichnet ein Sprite an einer bestimmten Position.

Syntax:

draw_sprite(sprite, subimg, x, y);


Streit Beschreibung
sprite Der Index des zu zeichnenden Sprites.
subimg Das Unterbild (Frame) des zu zeichnenden Sprites (image_index oder -1 entspricht dem aktuellen Animationsrahmen des Objekts).
x Die x-Koordinate, an der das Sprite gezeichnet werden soll.
y Die y-Koordinate, an der das Sprite gezeichnet werden soll.


Rückkehr: N / A


Beschreibung

Diese Funktion zeichnet das gegebene Sprite und das Teilbild an einer Position innerhalb des Spielraums. Für das Sprite können Sie die Instanzvariable verwenden sprite_index um das aktuelle Sprite zu erhalten, das der Instanz zugewiesen ist, auf der der Code ausgeführt wird, oder Sie können ein beliebiges anderes Sprite-Asset verwenden. Das gleiche gilt für das Teilbild, da dieses auch auf die Instanzvariable eingestellt werden kann image_index Dadurch wird das Sub-Image auf das für das aktuelle Instanz-Sprite ausgewählte Sub-Image gesetzt (beachten Sie, dass Sie ein anderes Sprite zeichnen und immer noch den Sub-Image-Wert für die aktuelle Instanz verwenden können) oder Sie können einen anderen Wert zum Zeichnen verwenden ein bestimmtes Teilbild des ausgewählten Sprites. Wenn der Wert größer als die Anzahl der Unterbilder ist, wird GameMaker Studio 2 die Nummer automatisch wiederholen, um das entsprechende Bild auszuwählen (z. B. wenn das gezeichnete Sprite 5 Unterbilder mit den Nummern 0 bis 4 hat und wir den Index setzen Wert auf 7, dann zeichnet die Funktion Teilbild 3, nummeriert 0). Schließlich ist die x- und y-Position die Position innerhalb des Raums, in der das Sprite gezeichnet wird, und es ist auf den Sprite x-Offset und y-Offset zentriert.


Beispiel:

draw_sprite(sprite_index, image_index, x, y);
draw_sprite(spr_Halo, 0, x, y-32);

Dadurch wird das zugewiesene Sprite (sprite_index) mit dem aktuellen Teilbild an der x- und y-Position der Instanz innerhalb des Raums gezeichnet. Es wird dann das erste Bild des Sprites, das mit "spr_Halo" indiziert ist, an der gleichen x- und y-Position, aber 32 Pixel über 'zeichnen.
