Zeichnen von Sprites und Fliesen

Obwohl Sie Hintergründe im Raumeditor einrichten und Sprites Instanzen zuordnen können, ist dies oftmals nicht flexibel genug für das von Ihnen erstellte Spiel. Wenn Sie beispielsweise über eine Instanz verfügen, die ein Menü zeichnen muss, möchten Sie möglicherweise mehrere Button-Sprites gleichzeitig anzeigen, oder wenn Sie ein Spiel mit scrollenden Parallaxen-Hintergründen haben, müssen Sie sie in der Lage sein, sie durch Code zu setzen Controller-Instanz eines Typs. In diesen Situationen sollten Sie die folgenden Funktionen verwenden, mit denen Sie Sprite- und Hintergrundelemente an beliebiger Stelle im Raum zeichnen können. Einige der Funktionen ermöglichen es auch, die Form, Größe, Farbe und das Alpha des Sprites zu ändern, um Spezialeffekte zu erzielen.

Sie können einer Instanz ein Sprite zuweisen, das standardmäßig von GameMaker Studio 2 mit den Eigenschaften der verschiedenen Sprite-Variablen gezeichnet wird. Wenn Sie jedoch etwas in das Draw-Ereignis einfügen, übernehmen Sie die Kontrolle und müssen GameMaker Studio 2 genau mitteilen was zu zeichnen Das bedeutet, wenn Sie die folgenden Funktionen verwenden müssen, um genau zu definieren, was die Instanz zeichnen soll und wo im Raum sie gezeichnet werden soll:

  • draw_self
  • draw_sprite
  • draw_sprite_ext
  • draw_sprite_general
  • draw_sprite_part
  • draw_sprite_part_ext
  • draw_sprite_stretched
  • draw_sprite_stretched_ext
  • draw_sprite_pos
  • draw_sprite_tiled
  • draw_sprite_tiled_ext

Wenn Sie Sprites im SWF-Format verwenden, gibt es zusätzliche Optionen zur Steuerung des beim Zeichnen verwendeten Anti-Aliasing (weitere Informationen zu diesen Sprites finden Sie unter Importieren von Sprites, die keine Bitmap- Sprites sind). Beachten Sie, dass diese Funktionen auf globaler Ebene funktionieren und die AA für alle Vektor-SWF-Sprites festlegen, die nach dem Aufruf gezeichnet werden.

  • draw_enable_swf_aa
  • draw_set_swf_aa_level
  • draw_get_swf_aa_level

Neben den oben aufgeführten Funktionen gibt es auch zwei spezielle Funktionen für die Verwendung mit Skelettanimationen (weitere Informationen zu diesen Sprites finden Sie unter Importieren von Sprites, die keine Bitmap sind, und den Abschnitt Skelettanimationen für spezifische Funktionen, die sich auf diese Sprites beziehen.):

  • draw_skeleton
  • draw_skeleton_instance
  • draw_skeleton_collision
  • draw_skeleton_time

Kacheln werden automatisch basierend auf den darin enthaltenen Daten, dem verwendeten Kachelsatz und den Tileemap-Eigenschaften gezeichnet. Sie können jedoch mit den folgenden Funktionen einzelne Kacheln oder Tilemaps zeichnen:

  • draw_tile
  • tilemap_draw
