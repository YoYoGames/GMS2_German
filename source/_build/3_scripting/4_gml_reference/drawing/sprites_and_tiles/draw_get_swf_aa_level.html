draw_get_swf_aa_level

Ermitteln des aktuellen Anti-Aliasing-Levels für Vektor-Sprites im SWF-Format.

Syntax:

draw_get_swf_aa_level();


Rückgabe: Echt


Beschreibung

Diese Funktion kann verwendet werden, um die Anti-Aliasing-Ebene (AA) für SWF-Vektor-Sprites zu erhalten. Der Rückgabewert liegt zwischen 0 und 1 und zeigt an, wie "glatt" die Kanten dieser Sprites gezeichnet werden. Sie können die AA-Stufe mit der Funktion einstellen draw_set_swf_aa_level().

WICHTIG! Diese Funktion ist mit der Testlizenz des Produkts nicht verfügbar.



Beispiel:

if draw_get_swf_aa_level() == 0
   {
   draw_enable_swf_aa(true);
   draw_set_swf_aa_level(0.5);
   }

Der obige Code überprüft den AA-Wert für SWF-Format-Sprites und wenn er 0 ist, aktiviert er AA und setzt den Wert auf 0,5.
