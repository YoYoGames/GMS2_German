draw_sprite_stretched

Zeichnet ein Sprite an einer bestimmten Position, gestreckt.

Syntax:

draw_sprite_stretched(sprite, subimg, x, y, w, h);


Streit Beschreibung
sprite Der Index des zu zeichnenden Sprites.
subimg Der Subimg (Rahmen) des zu zeichnenden Sprites (image_index oder -1 entspricht dem aktuellen Animationsrahmen des Objekts).
x Die x-Koordinate, an der das Sprite gezeichnet werden soll.
y Die y-Koordinate, an der das Sprite gezeichnet werden soll.
w Die Breite des Bereichs, in dem sich das gestreckte Sprite befindet.
h Die Höhe des Bereichs, in dem sich das gestreckte Sprite befindet.


Rückkehr: N / A


Beschreibung

Diese Funktion nimmt einfach eine Sprite-Ressource und streckt sie über die angegebene Breite und Höhe, so dass sie das Gebiet einnimmt. Wie mit draw_sprite Sie können ein Sprite und ein Teilbild für das Zeichnen festlegen, dann die x / y-Position im Raum, in der das Sprite gezeichnet werden soll, und schließlich eine Breite und eine Höhe (die Pixelwerte sein müssen). Das folgende Bild zeigt, wie diese Funktion funktioniert:

HINWEIS: Beim Zeichnen mit dieser Funktion werden der Sprite x Offset und y Offset ignoriert und das Sprite wird mit der oberen linken Ecke an der angegebenen x / y Position im Raum gezeichnet.


Beispiel:

draw_sprite_stretched(sprite_index, image_index, x, y, sprite_width / 2, sprite_height / 2);

Dadurch werden die den Instanzen zugewiesenen Sprites und ihr Teilbild mit der linken Ecke an der Instanz x / y-Position gezeichnet. Seine Breite ist auf das gleiche wie das Sprite eingestellt, und die Höhe ist die Sprite-Höhe geteilt durch zwei.
