draw_til

Beschreibung

Mit dieser Funktion können Sie jede beliebige Kachel aus einem Kachel-Set zeichnen - konkurrieren Sie mit Rotationen, Flips und Spiegeln - und setzen Sie den Frame (falls animiert) und die Position im Raum. Der Kachelsatz-ID-Wert ist der Kachelset-Index, der in der IDE festgelegt wurde, und die Kacheldaten sind das Daten- "Blob", das alle Informationen über den Kachelindex und die angewendeten Transformationen enthält. Sie können die Funktion verwenden tilemap_get die Kachel - Daten zu erhalten, die dann die manipuliert werden kann, unter Verwendung von Tile - Funktionen, oder Sie können es selbst konstruieren, um den folgenden Fliesen Konstanten zusammen mit dem Indexwert der Kachel mit zu erstellen:

Konstante Beschreibung
tile_rotate Wird verwendet, um das Rotationsbit eines Kachel-Datenblobs zu setzen / zu erhalten.
tile_mirror Wird verwendet, um das Spiegelungsbit eines Kachel-Datenblobs zu setzen / zu erhalten.
tile_flip Wird verwendet, um das Flip-Bit eines Kacheldatenblobs zu setzen / zu erhalten.
tile_index_mask Eine spezielle Konstante für "und" mit dem Kachel-Daten-Blob, um den Kachel-Index zu extrahieren.


Du würdest bitweise "oder" diese Konstanten mit dem Kachel-Indexwert verwenden, um das letzte tiledata zu erhalten:

var tiledata = 8 | tile_rotate | tile_flip;


Der obige Code erstellt einfach eine lokale Variable zum Speichern der tiledata für die Kachel mit dem Index 8 auf der Kachel, und sie wird gedreht und gewendet. Dies kann dann verwendet werden, um die Kachel bei diesem Index von dem von Ihnen angegebenen Kachelsatz zu zeichnen. Dies ist nicht mit einem Kachelkarten-Layer verknüpft und die Kachel wird in der Ebenentiefe der Instanz, die die Zeichnung ausführt, an der x / y-Position gezeichnet, die Sie im Raum angeben. Beachten Sie, dass Sie den Rahmen auch zum Zeichnen festlegen können. Wenn die indizierte Kachel Teil eines Animationssatzes ist, können Sie hier die Bildnummer einstellen (von 0 bis zur Anzahl der Bilder - 1) und die indizierte Kachel ist einfach der Indikator dafür, welches animierte Kachelset Sie gerade verwenden Sei der gegebene Rahmen, der gezeichnet wird. Wenn die Kachel nicht Teil einer animierten Gruppe ist, sollte frame auf 0 gesetzt werden.


Syntax:

draw_tile(tile set, tiledata, frame, x, y)


Argumente:

Streit Beschreibung
tile set Die ID der zu verwendenden Kachel
tiledata Die zu verwendenden Kacheldaten
frame Die Bildnummer für die animierte Kachel (Standard 0 für nicht animierte Kacheln)
x Die x-Position im Raum zum Zeichnen
y Die y-Position im Raum zum Zeichnen


Kehrt zurück:

Real


Beispiel:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
tile_set_flip(data, true);
draw_tile(set_Wall, data, 0, x, y);

Der obige Code wird die Daten auf einer Kachel von der gegebenen Kachelkarte erhalten, dann das "Flip" -Flag setzen und diese Daten verwenden, um die an der Instanzposition gekippte Kachel zu zeichnen.
