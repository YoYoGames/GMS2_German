draw_tile

Beschreibung

Mit dieser Funktion können Sie eine beliebige Kachel aus einem Kachelsatz zeichnen - mit Drehungen, Flips und Spiegeln konkurrieren - den Rahmen (wenn animiert) und die Position innerhalb des Raums festlegen. Der Tile-Set-ID-Wert ist der Tile-Set-Index, wie er in der IDE festgelegt ist, und die Tile-Daten sind die Daten "Blob", die alle Informationen zum Tile-Index und den angewendeten Transformationen enthalten. Sie können die Funktion verwenden tilemap_get die Kachel - Daten zu erhalten, die dann die manipuliert werden kann, unter Verwendung von Tile - Funktionen, oder Sie können es selbst konstruieren, um den folgenden Fliesen Konstanten zusammen mit dem Indexwert der Kachel mit zu erstellen:

Konstante Beschreibung
tile_rotate Wird verwendet, um das Rotationsbit eines Kachel-Datenblobs festzulegen / abzurufen.
tile_mirror Wird verwendet, um das Spiegelbit eines Kachel-Datenblobs zu setzen / abzurufen.
tile_flip Wird verwendet, um das Flip-Bit eines Kachel-Datenblobs zu setzen / zu erhalten.
tile_index_mask Eine spezielle Konstante, die für das "und" -Zeichen mit dem Kachel-Datenblob zum Extrahieren des Kachelindex bestimmt ist


Sie würden diese Konstanten mit dem Tile-Indexwert bitweise "oder" eingeben, um die endgültigen Tiledata zu erhalten:

var tiledata = 8 | tile_rotate | tile_flip;


Der obige Code erstellt einfach eine lokale Variable, um die Tiledata für die Kachel mit dem Index 8 im Kachelsatz zu speichern, und diese wird gedreht und gespiegelt. Dies kann dann verwendet werden, um die Kachel an diesem Index aus dem von Ihnen angegebenen Kachelsatz zu zeichnen. Dies ist nicht mit einem Tilemap-Layer verknüpft, und die Kachel wird an der Layer-Tiefe der Instanz gezeichnet, die die Zeichnung ausführt, an der X / Y-Position, die Sie im Raum angeben. Beachten Sie, dass Sie den Rahmen auch zum Zeichnen einstellen können. Wenn die indizierte Kachel Teil eines Animationssatzes ist, können Sie hier die Bildnummer festlegen (von 0 bis zur Anzahl der Bilder - 1), und die indizierte Kachel ist einfach der Indikator für den animierten Kachelsatz, den Sie verwenden sei der gegebene Rahmen, der gezeichnet wird. Wenn die Kachel nicht Teil eines animierten Sets ist, sollte der Frame auf 0 gesetzt werden.


Syntax:

draw_tile(tile set, tiledata, frame, x, y)


Argumente:

Streit Beschreibung
tile set Die ID des zu verwendenden Kachelsatzes
tiledata Die zu verwendenden Kacheldaten
frame Die Bildnummer für die zu verwendende animierte Kachel (Standardwert 0 für nicht animierte Kacheln)
x Die x-Position innerhalb des zu zeichnenden Raums
y Die y-Position im Raum zum Zeichnen


Kehrt zurück:

Real


Beispiel:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
tile_set_flip(data, true);
draw_tile(set_Wall, data, 0, x, y);

Der obige Code ruft die Daten einer Kachel aus der angegebenen Kachelkarte ab, setzt dann das "Flip" -Flag und verwendet diese Daten, um die Kachel an der Instanzposition zu zeichnen.
