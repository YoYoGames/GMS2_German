draw_set_swf_aa_level

Legen Sie die Anti-Aliasing-Stufe für Vektor-Sprites im SWF-Format fest.

Syntax:

draw_set_swf_aa_level(AA);


Streit Beschreibung
AA Der zu verwendende Anti-Aliasing-Wert von 0 bis 1.


Rückkehr: N / A


Beschreibung

Diese Funktion kann verwendet werden, um die Anti-Aliasing-Ebene (AA) für SWF-Vektor-Sprites festzulegen. Dies kann ein reeller Wert von 0 bis 1 sein und die Kanten dieser Sprites "glätten". Beachten Sie, dass Sie, um diesen Effekt zu sehen, zuerst AA mit der Funktion aktiviert haben müssen draw_enable_swf_aa().

WICHTIG! Diese Funktion ist mit der Testlizenz des Produkts nicht verfügbar.



Beispiel:

if draw_get_swf_aa_level() == 0
   {
   draw_enable_swf_aa(true);
   draw_set_swf_aa_level(0.5);
   }

Der obige Code überprüft den AA-Wert für SWF-Format-Sprites und wenn er 0 ist, aktiviert er AA und setzt den Wert auf 0,5.
