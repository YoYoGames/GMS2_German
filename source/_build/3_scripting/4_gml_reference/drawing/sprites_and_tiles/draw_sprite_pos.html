draw_sprite_pos

Zeichnet ein Sprite, das zwischen vier vorgegebenen Punkten und mit einer Alpha-Mischung gestreckt ist.

Syntax:

draw_sprite_pos(sprite, subimg, x1, y1, x2, y2, x3, y3, x4, y4, alpha);


Streit Beschreibung
sprite Der Index des zu zeichnenden Sprites.
subimg Der Subimg (Rahmen) des zu zeichnenden Sprites (image_index oder -1 entspricht dem aktuellen Animationsrahmen des Objekts).
x1 Die erste x-Koordinate.
y1 Die erste y-Koordinate.
x2 Die zweite x-Koordinate.
y2 Die zweite y-Koordinate.
x3 Die dritte x-Koordinate.
y3 Die dritte y-Koordinate.
x4 Die vierte x-Koordinate.
y4 Die vierte y-Koordinate.
alpha Das Alpha des Sprites (von 0 bis 1, wobei 0 transparent und 1 undurchsichtig ist).


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie ein verzerrtes Sprite über den durch die vier Eckkoordinaten definierten Bereich zeichnen. Die ersten beiden Argumente sind das zu zeichnende Sprite und das Teilbild des Sprites (das gleiche wie in der Funktion) draw_sprite ) aber die nächsten sind diejenigen, die die Position jeder der vier Ecken der Begrenzungsbox des gegebenen Sprites definieren. Diese sollten im Uhrzeigersinn angegeben werden, also die erste Koordinate ist oben links, dann oben rechts, dann unten rechts und schließlich unten links. Sie können auch einen Wert für das Alpha des Sprites festlegen, um es mit Transparenz zu zeichnen. Das folgende Bild zeigt, wie diese Funktion funktioniert:

ACHTUNG: Das obige Bild dient nur zur Veranschaulichung, und wenn Sie diese Funktion für ein Sprite verwenden, erhalten Sie unterschiedliche Ergebnisse und können aufgrund der Art und Weise, wie ein Sprite aus einem Vierer von Primitiven konstruiert wird, ein "Scheren" der Textur erfahren.
HINWEIS: Diese Funktion ist nur für Bitmap- Sprites und nicht für SWF- oder JSON (Spine) -Sprites geeignet.


Beispiel:

draw_sprite_pos(sprite_index, image_index, x - 100, y - 50, x - 50, y +150, x + 100, y + 200, x + 100, y, 1);

Der obige Code zeichnet das Sprite, das der Instanz zugeordnet ist, die den Code um die x / y-Position der Instanz herum verzerrt, und mit einem vollständig undurchsichtigen Alpha.
