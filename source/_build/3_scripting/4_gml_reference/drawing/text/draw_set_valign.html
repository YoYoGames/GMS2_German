draw_set_valign


Beschreibung

Mit dieser Funktion wird Text entlang der vertikalen Achse ausgerichtet. Wenn Sie die vertikale Ausrichtung ändern, ändern Sie die Position und Richtung, in der der gesamte weitere Text gezeichnet wird. Der Standardwert ist dabei fa_top. Folgende Konstanten werden akzeptiert:

Konstante Ausrichtung
fa_top
fa_middle
fa_bottom


Syntax:

draw_set_valign(valign);

Streit Beschreibung
valign Vertikale Ausrichtung.


Kehrt zurück:

N/A


Beispiel:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
draw_text(100, 32, "Score: " + string(score));

Der obige Code zeichnet die Punktzahl um die Mitte des Textes.
