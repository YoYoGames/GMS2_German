draw_text_ext


Beschreibung

Diese Funktion zeichnet den Text ähnlich wie draw_text Erst jetzt können Sie den Abstand zwischen den einzelnen Textzeilen festlegen - sollte der Text mehr als eine Zeile einnehmen - und die Breite (in Pixel) der Zeichenfolge pro Zeile begrenzen, sodass GameMaker Studio 2 automatisch aufgeteilt wird, wenn eine Zeile diesen Wert überschreitet der Text bis zur nächsten Zeile an der nächstgelegenen verfügbaren Leerstelle (wenn der Text keine Leerräume enthält, wird dieser Wert für die maximale Breite überschritten). Beachten Sie, dass alle Leerzeichen am Anfang der Zeichenfolge entfernt werden, bevor sie zum Zeichnen analysiert werden. Beachten Sie außerdem, dass der Wert -1 für das Zeilentrennungsargument standardmäßig auf der Höhe des "M" -Zeichen in der ausgewählten Schriftart basiert.


Syntax:

draw_text_ext(x, y, string, sep, w);

Streit Beschreibung
x Die x-Koordinate der gezeichneten Zeichenfolge.
y Die y-Koordinate der gezeichneten Zeichenfolge.
string Die Zeichenfolge zum Zeichnen.
sep Der Abstand in Pixel zwischen Textzeilen.
w Die maximale Breite der Zeichenfolge in Pixel vor einem Zeilenumbruch.


Kehrt zurück:

N/A


Beispiel:

draw_text_ext(100, 50, keyboard_string, 3, 300);

Der obige Code zeichnet den vom Benutzer eingegebenen Text auf der Tastatur und teilt ihn in neue Zeilen auf, wenn die Zeichenfolgenlänge für diese Zeile 300 Pixel überschreitet. Der Code behält auch einen Abstand von 3 Pixeln zwischen den Zeilen bei, falls dies auftritt.
