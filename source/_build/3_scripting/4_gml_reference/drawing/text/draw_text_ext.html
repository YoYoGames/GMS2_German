zeichen_text_ext

Zeichnet eine Zeichenfolge an einer bestimmten Position mit einem bestimmten Abstand und innerhalb eines begrenzten Bereichs.

Syntax:

draw_text_ext(x, y, string, sep, w);


Streit Beschreibung
x Die X-Koordinate der gezeichneten Zeichenfolge.
y Die Y-Koordinate der gezeichneten Zeichenfolge.
string Die zu zeichnende Zeichenfolge.
sep Der Abstand zwischen den Textzeilen in Pixeln.
w Die maximale Breite in Pixel der Zeichenfolge vor einem Zeilenumbruch.


Rückkehr: N / A


Beschreibung

Diese Funktion zeichnet Text auf ähnliche Weise draw_text Nur jetzt können Sie den Abstand zwischen jeder Textzeile einstellen - sollte der Text mehr als eine Zeile umfassen - und die Breite (in Pixel) der Zeichenkette pro Zeile begrenzen. GameMaker Studio eine Zeile diesen Wert überschreiten, wird GameMaker Studio 2 automatisch geteilt der Text in die nächste Zeile. Ein Wert von -1 für das Zeilentrennungsargument wird standardmäßig auf eine Trennung basierend auf der Höhe des "M" -Zeichens in der ausgewählten Schriftart gesetzt.


Beispiel:

draw_text_ext(100, 50, keyboard_string, 3, 300);

Der obige Code zeichnet jeden Text, den der Benutzer in die Tastatur eingibt, auf jedes Mal auf, wenn die Zeichenfolgenlänge für diese Zeile 300 Pixel überschreitet. Der Code wird auch eine Trennung von 3 Pixeln zwischen Zeilen beibehalten, falls dies auftritt.
