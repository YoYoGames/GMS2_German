Zeichnungs_text_farbe

Zeichnen Sie eine Zeichenfolge an einer bestimmten Position mit einem Farbverlauf.

Syntax:

draw_text_colour(x, y, string, c1, c2, c3, c4, alpha);


Streit Beschreibung
x Die X-Koordinate der gezeichneten Zeichenfolge.
y Die Y-Koordinate der gezeichneten Zeichenfolge.
string Die zu zeichnende Zeichenfolge.
c1 Die Farbe für den oberen linken Teil des gezeichneten Texts.
c2 Die Farbe für die obere rechte Ecke des gezeichneten Textes.
c3 Die Farbe für die untere rechte Ecke des gezeichneten Textes.
c4 Die Farbe für die linke untere Ecke des gezeichneten Textes.
alpha Das Alpha für den Text.


Rückkehr: N / A


Beschreibung

Diese Funktion zeichnet Text auf ähnliche Weise draw_text Nur jetzt können Sie die Farben auswählen, die Sie zum Einfärben des Textes verwenden möchten, sowie den Alpha-Wert, und diese neuen Werte werden anstelle der Basiszeichnungsfarbe und Alpha verwendet.

HINWEIS: Die Farbverlaufsmischung ist für das HTML5-Ziel nicht verfügbar, es sei denn, WebGL ist aktiviert, Sie können jedoch weiterhin die Mischfarben festlegen und die Schriftart mit der ersten angegebenen Farbe mischen. Jedes Mischen auf diese Weise erzeugt jedoch eine doppelte Schrift, die dann im Cache gespeichert und bei Bedarf verwendet wird, was alles andere als optimal ist. Wenn Sie mehrere Farbänderungen verwenden, verlangsamt dies die Leistung Ihres Spiels. Sie können die Schriftart-Cache-Größe festlegen, um zu versuchen, dies zu begrenzen, sollte es notwendig sein, die Funktion zu verwenden font_set_cache_size.


Beispiel:

draw_set_colour(c_white);
draw_text(100, 100, "Health");
draw_text_colour(100, 200, string(health), c_lime, c_lime, c_green, c_green, 1);

Der obige Code zeichnet zwei Textabschnitte in derselben Zeile, wobei der erste Text weiß gezeichnet wird (da dies die Basiszeichnungsfarbe ist) und der zweite Text mit einem lindgrünen bis normalen grünen Farbverlauf gezeichnet wird.
