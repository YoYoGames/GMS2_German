draw_text_ext_transformed_colour


Beschreibung

Diese Funktion ist eine Kombination der Basis draw_text Funktion mit der draw_text_ext. draw_text_transformed und draw_text_colour Funktionen. Dies ist die allgemeinste aller Funktionen zum Zeichnen von Text und bietet eine große Flexibilität beim Zeichnen. Beachten Sie, dass das Argument "width" auf einer Skala von 1 basiert. Wenn sich die Skala unterscheidet, sollte dieser Wert proportional geändert werden. Wenn die Basisbreite für einen Zeilenumbruch beispielsweise 300 ist und Sie die Skalierung auf 2 festlegen, wird der Text falsch angezeigt und überschreitet die angegebene Breite. Stattdessen sollten Sie die Breite auf 150 setzen, um die Skalierung zu kompensieren.

HINWEIS: Die Farbverlaufsüberblendung ist für das HTML5-Ziel nur verfügbar, wenn WebGL aktiviert ist. Sie können jedoch weiterhin die Überblendungsfarben festlegen und die Schrift wird mit der ersten angegebenen Farbe gemischt. Durch das Mischen auf diese Weise wird jedoch eine doppelte Schriftart erstellt, die dann im Cache gespeichert und bei Bedarf verwendet wird. Dies ist alles andere als optimal und wenn Sie mehrere Farbwechsel verwenden, wird die Leistung Ihres Spiels beeinträchtigt. Sie können die Größe des Schrift-Cache festlegen, um dies zu versuchen und einzuschränken, falls dies mit der Funktion erforderlich ist font_set_cache_size.


Syntax:

draw_text_ext_transformed_colour(x, y, string, sep, w, xscale, yscale, angle, c1, c2, c3, c4, alpha);

Streit Beschreibung
x Die x-Koordinate der gezeichneten Zeichenfolge.
y Die y-Koordinate der gezeichneten Zeichenfolge.
string Die Zeichenfolge zum Zeichnen.
sep Der Abstand in Pixel zwischen Textzeilen.
w Die maximale Breite der Zeichenfolge in Pixel vor einem Zeilenumbruch.
xscale Die horizontale Skala.
yscale Die vertikale Skala.
angle Der Winkel des Textes
c1 Die Farbe für den oberen linken Bereich des gezeichneten Textes.
c2 Die Farbe für den oberen rechten Bereich des gezeichneten Textes.
c3 Die Farbe für den unteren rechten Bereich des gezeichneten Textes.
c4 Die Farbe für den unteren linken Bereich des gezeichneten Textes.
alpha Das Alpha für den Text.


Kehrt zurück:

N/A


Beispiel:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
draw_text_ext_transformed_colour(200, 200, "My name is" + global.name +".", 20, 200, 0.5, 0.5, 180, c_blue, c_blue, c_navy, c_navy, 1);

Dadurch wird der angegebene Text mit einem blauen bis dunkelblauen Farbverlauf an der Position (200,200) im Raum gedreht, so gedreht, dass er auf dem Kopf steht und die Hälfte der normalen Punktgröße der Schrift beträgt. Es werden 20 Pixel zwischen jeder Zeile angezeigt, falls mehr als eine Zeile gezeichnet wird und jede Zeile eine maximale Breite von 200 Pixeln hat.
