draw_text_ext_transformed_colour

Zeichnet eine Zeichenfolge mit den von Ihnen definierten Attributen für Farbe, Skalierung, Winkel oder Abstand.

Syntax:

draw_text_ext_transformed_colour(x, y, string, sep, w, xscale, yscale, angle, c1, c2, c3, c4, alpha);


Streit Beschreibung
x Die X-Koordinate der gezeichneten Zeichenfolge.
y Die Y-Koordinate der gezeichneten Zeichenfolge.
string Die zu zeichnende Zeichenfolge.
sep Der Abstand zwischen den Textzeilen in Pixeln.
w Die maximale Breite in Pixel der Zeichenfolge vor einem Zeilenumbruch.
xscale Die horizontale Skala.
yscale Die vertikale Skala.
angle Der Winkel des Textes.
c1 Die Farbe für den oberen linken Teil des gezeichneten Texts.
c2 Die Farbe für die obere rechte Ecke des gezeichneten Textes.
c3 Die Farbe für die untere rechte Ecke des gezeichneten Textes.
c4 Die Farbe für die linke untere Ecke des gezeichneten Textes.
alpha Das Alpha für den Text.


Rückkehr: N / A


Beschreibung

Diese Funktion ist eine Kombination der Basis draw_text Funktion mit dem draw_text_ext, draw_text_transformed und draw_text_colour Funktionen. Somit ist dies die allgemeinste aller Funktionen zum Zeichnen von Text und bietet eine große Flexibilität beim Zeichnen. Beachten Sie, dass das Argument "width" auf einer Skala von 1 basiert. Wenn die Skala also anders ist, sollte dieser Wert proportional geändert werden. Wenn die Basisbreite für einen Zeilenumbruch beispielsweise 300 ist und Sie die Skalierung auf 2 festlegen, wird der Text falsch angezeigt und die angegebene Breite wird überschritten. Stattdessen sollten Sie die Breite auf 150 einstellen, um die Skalierung zu kompensieren.

HINWEIS: Die Farbverlaufsmischung ist für das HTML5-Ziel nicht verfügbar, es sei denn, WebGL ist aktiviert, Sie können jedoch weiterhin die Mischfarben festlegen und die Schriftart mit der ersten angegebenen Farbe mischen. Jedes Mischen auf diese Weise erzeugt jedoch eine doppelte Schrift, die dann im Cache gespeichert und bei Bedarf verwendet wird, was alles andere als optimal ist. Wenn Sie mehrere Farbänderungen verwenden, verlangsamt dies die Leistung Ihres Spiels. Sie können die Schriftart-Cache-Größe festlegen, um zu versuchen, dies zu begrenzen, sollte es notwendig sein, die Funktion zu verwenden font_set_cache_size.


Beispiel:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
draw_text_ext_transformed_colour(200, 200, "My name is" + global.name +".", 20, 200, 0.5, 0.5, 180, c_blue, c_blue, c_navy, c_navy, 1);

Dies wird den gegebenen Text mit einer blauen bis dunkelblauen Abwärtsneigung an der Position (200, 200) im Raum zeichnen, gedreht, um auf den Kopf gestellt zu sein und um die Hälfte der normalen Punktgröße der Schriftart. Zwischen jeder Zeile befinden sich 20 Pixel, wenn mehr als eine Zeile gezeichnet wird und jede Zeile eine maximale Breite von 200 Pixel hat.
