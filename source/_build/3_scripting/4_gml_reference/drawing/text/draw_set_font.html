draw_set_font

Legt die Schriftart zum Zeichnen von Text fest.

Syntax:

draw_set_font(font);


Streit Beschreibung
font Der Name der zu verwendenden Schriftart.


Rückkehr: N / A


Beschreibung

Diese Funktion stellt die Schriftart ein, die für alle weiteren Textzeichnungen verwendet werden soll. Diese Schriftart muss zu den Schriftarten-Assets des Spiels hinzugefügt worden sein oder entweder mit der Schriftart erstellt worden sein font_add oder font_add_sprite. Sie können diese Einstellung auch auf -1 setzen, um Zeichentext mit einer 12-Punkt-Arial-Schriftart zu zeichnen.


Beispiel:

draw_set_colour(c_blue);
draw_set_font(fnt_Game);
draw_text(200, 200, "Hello World" );

Der obige Code wird den gegebenen Text unter Verwendung der Schriftart zeichnen, die in der Variablen "fnt_Game" indiziert und blau gefärbt ist.
