draw_vertex_texture_colour


Syntax:

draw_vertex_texture_colour(x, y, xtex, ytex, col, alpha)


Streit Beschreibung
x Die x-Koordinate des Scheitelpunkts.
y Die y-Koordinate des Scheitelpunkts.
xtex Die x-Koordinate innerhalb der Textur.
ytex Die y-Koordinate innerhalb der Textur.
col Die Farbe, die mit der Textur an diesem Scheitelpunkt vermischt werden soll (-1 oder c_white für kein Mischen).
alpha Das Alpha, um diesen Scheitelpunkt mit (0-1) zu zeichnen.


Kehrt zurück:

N/A


Beschreibung

Diese Funktion definiert die Position eines texturierten Scheitelpunkts für ein Grundelement. Das endgültige Aussehen des Grundelements hängt von dem für das Zeichnen ausgewählten Grundtyp ab (siehe draw_primitive_begin Weitere Informationen), die Reihenfolge, in der Sie die Scheitelpunkte hinzufügen, die Position der Start- und Endpunkte, die Sie für das Texturmuster angeben, sowie die von Ihnen festgelegten Werte für Farbe und Alpha. Um das Aussehen der Textur beizubehalten und nur das Alpha zu ändern, den Wert -1 (oder c_white ) kann für das Farbargument verwendet werden. Um das Primitive zu beenden und zu zeichnen, musst du anrufen draw_primitive_end.

HINWEIS: Damit eine Textur wiederholt werden kann, muss sie eine Stärke von zwei haben, dh: 32x32, 128x128 usw.


Beispiel:

draw_set_colour(c_white);
var tex = sprite_get_texture(spr_Background);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture_colour(0, 480, 0, 1, c_green, 1);
draw_vertex_texture_colour(0, 0, 0, 0, c_fuchsia, 1);
draw_vertex_texture_colour(640, 480, 1, 1, c_red, 1);
draw_vertex_texture_colour(640, 0, 1, 0, c_blue, 1);
draw_primitive_end();

Der obige Code zeichnet einen 4-Punkt-Dreieckstreifen (der ein Rechteck bildet), wobei die Textur in der Variablen "tex" texturiert ist. Die gesamte Textur wird verwendet, um das fertige Grundelement zu überdecken, und es wird mit vier verschiedenen Farben gemischt.
