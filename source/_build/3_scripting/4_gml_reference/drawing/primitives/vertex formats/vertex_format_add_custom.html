vertex_format_add_custom

Fügen Sie dem Vertexformat benutzerdefinierte Werte hinzu.

Syntax:

vertex_format_add_custom(type, usage);


Streit Beschreibung
type Die Daten geben, dass dieser Brauch Vertexdaten halten (siehe unten die Typkonstanten sehen).
usage Die Verwendung, die die Daten erhalten (siehe die unten aufgeführten Nutzungskonstanten ).


Rückkehr: N / A


Beschreibung

GameMaker Studio 2 Sie GameMaker Studio 2 an, einen benutzerdefinierten Wert (oder Werte) als Teil des neu erstellten Vertex-Formats zu übernehmen. Die verfügbaren Werte, die verwendet werden sollen, werden durch die ausgewählte Datentypkonstante definiert, die unten aufgeführt ist:

Konstante Beschreibung
vertex_type_float1 Ein einzelner Gleitkommawert
vertex_type_float2 Zwei Gleitkommawerte
vertex_type_float3 Drei Gleitkommawerte
vertex_type_float4 Vier Gleitkommawerte
vertex_type_colour Vier Komponentenwerte (r, g, b, a)
vertex_type_ubyte4 Vier vorzeichenlose Bytewerte (von 0 bis 255)

Die Verwendung dieser Konstanten muss ebenfalls definiert werden, damit die Werte innerhalb des erstellten Shaders korrekt "gebunden" werden können. Dies ist notwendig, da DX und OpenGL unterschiedliche Anforderungen haben. Wenn Sie sie nicht richtig binden, werden sie nicht direkt im Shader ankommen. Die verfügbaren Nutzungskonstanten, die Sie auswählen können, sind unten aufgeführt und die von Ihnen verwendeten sind abhängig von den Besonderheiten des erstellten Shaders:

Konstante Beschreibung
vertex_usage_position Positionswerte (x, y, z)
vertex_usage_colour Farbwerte (r, g, b, a)
vertex_usage_normal Eckpunktnormale Werte (nx, ny, nz)
vertex_usage_textcoord UV-Koordinaten (u, v)
vertex_usage_blendweight das Mischgewicht der Eingabematrix (zB für Skelettanimation)
vertex_usage_blendindices die Indizes der zu verwendenden Matrizen (z. B. für Skelettanimation)
vertex_usage_depth Vertex-Tiefenpufferwert
vertex_usage_tangent Tangenswerte
vertex_usage_binormal binormale Werte
vertex_usage_fog Nebelwerte
vertex_usage_sample Sampler-Index


Beispiel:

vertex_format_begin();
vertex_format_add_textcoord();
vertex_format_add_custom(vertex_type_float3, vertex_usage_position);
my_format = vertex_format_end();

Der obige Code erstellt ein neues Vertexformat mit nur Textur und 3 benutzerdefinierten Fließkommawerten für die Position. Es speichert dann die Format-ID in der Variablen "my_format".
