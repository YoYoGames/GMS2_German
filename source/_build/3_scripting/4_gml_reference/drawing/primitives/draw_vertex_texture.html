draw_vertex_texture


Syntax:

draw_vertex_texture(x, y, xtex, ytex)


Streit Beschreibung
x Die x-Koordinate des Scheitelpunkts.
y Die y-Koordinate des Scheitelpunkts.
xtex Die Breite der zu verwendenden Texturprobe.
ytex Die Höhe der zu verwendenden Texturprobe.


Kehrt zurück:

N/A


Beschreibung

Diese Funktion definiert die Position eines texturierten Scheitelpunkts für ein Grundelement. Das endgültige Aussehen des Grundelements hängt von dem für das Zeichnen ausgewählten Grundtyp ab (siehe draw_primitive_begin Weitere Informationen), die Reihenfolge, in der Sie die Scheitelpunkte hinzufügen, und die Position des Startpunkts, den Sie für die Textur festlegen. Um das Primitive zu beenden und zu zeichnen, musst du anrufen draw_primitive_end.

Sie können auch nur einen Teil der Textur über die Oberfläche des Primitivs zeichnen, indem Sie die entsprechende xtex / ytex-Position für die Abtastung festlegen (diese Koordinaten werden auch als UV- Punkte bezeichnet). Dieser Punkt wird durch einen Wert zwischen 0 und 1 definiert, wobei die obere linke Ecke des zu verwendenden Bildes der (0,0) -Punkt ist und die untere rechte Ecke der (1,1) -Punkt ist. Auf diese Weise können Sie ein Grundelement nur mit einem Teil des Quellbildassets strukturieren.

Beim Zeichnen eines texturierten Primitivs wird die Textur mit der mit eingestellten Farbe gemischt draw_set_colour. Dies bedeutet, dass zum Zeichnen der Textur ohne Mischen dies auf festgelegt werden sollte c_white wie im nachstehenden Beispiel gezeigt.

HINWEIS: Damit eine Textur wiederholt werden kann, muss sie eine Stärke von zwei haben, dh: 32x32, 128x128 usw.


Beispiel:

draw_set_colour(c_white);
var tex = sprite_get_texture(spr_Background);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 480, 0, 1);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(640, 480, 1, 1);
draw_vertex_texture(640, 0, 1, 0);
draw_primitive_end();

Der obige Code zeichnet einen 4-Punkt-Dreieckstreifen (der ein Rechteck bildet), wobei die Textur in der Variablen "tex" gespeichert ist. Die gesamte Textur wird verwendet, um das fertige Grundelement abzudecken.
