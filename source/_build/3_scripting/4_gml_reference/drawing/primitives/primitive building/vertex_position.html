Vertexposition

Legen Sie die 2D-Position für den Scheitelpunkt fest.

Syntax:

vertex_position(buffer, x, y);


Streit Beschreibung
buffer Der Puffer, in den die Informationen geschrieben werden.
x Die x-Position.
y Die y-Position.


Rückkehr: N / A


Beschreibung

Diese Funktion fügt dem aktuell definierten Eckpunkt für das benutzerdefinierte Grundelement 2D-Positionsdaten hinzu. Sie liefern den Puffer zum Schreiben der Daten sowie die x- und y-Position zum Zeichnen.


Beispiel:

vertex_position(buff, x - 100, y - 125);

Der obige Code legt die Position des definierten Eckpunkts fest.
