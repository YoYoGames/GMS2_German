vertex_create_buffer

Erstellen Sie einen neuen Vertex-Puffer.

Syntax:

vertex_create_buffer();


Rückgabe: Zeiger


Beschreibung

Mit dieser Funktion können Sie einen neuen Vertex Buffer erstellen. Dies ist ein spezieller Grow- Puffer, der von GameMaker Studio 2 erstellt wurde und GameMaker Studio 2 ist, um Primitive zu erstellen (zum Beispiel für Shader). Die Funktion gibt einen "Zeiger" (Index) für den Puffer zurück, der dann in allen weiteren Aufrufen verwendet werden sollte.

Wenn Sie einen Vertex-Puffer verwenden, der mit dieser Funktion erstellt wurde, rufen Sie einfach auf vertex_begin Beginnen Sie mit der Zuweisung von Vertex-Daten, um mit der Definition Ihres benutzerdefinierten Grundelements zu beginnen, das dann im Puffer für die Übergabe an den Shader bereitgehalten wird. Der Puffer kann bei Bedarf wiederverwendet werden (außer Sie haben den vertex_freeze Funktion), bei jedem Aufruf von vertex_begin Löschen der vorherigen Pufferdaten, die bereit sind, die neuen Daten zu akzeptieren.


Beispiel:

v_buff = vertex_create_buffer();

Der obige Code erstellt einen neuen neuen Vertex-Puffer und speichert sein Handle in der Variablen "v_buff".
