vertex_ubyte4

Fügen Sie einem Knoten vier vorzeichenlose Byte-Werte hinzu.

Syntax:

vertex_ubyte4(buffer, byte, byte, byte, byte);


Streit Beschreibung
buffer Der Puffer, in den die Informationen geschrieben werden sollen.
byte Der erste Eingabewert.
byte Der zweite Eingabewert.
byte Der dritte Eingabewert.
byte Der vierte Eingabewert.


Rückgabe: N / A


Beschreibung

Diese Funktion fügt den Scheitelpunktdaten vier vorzeichenlose Bytewerte (0 - 255) hinzu. Der Scheitelpunkt muss korrekt formatiert sein, um dies mit dem zu akzeptieren vertex_format_add_custom Funktion.


Beispiel:

vertex_ubyte4(buff, irandom(255), irandom(255), irandom(255), 127);

Der obige Code fügt den zu definierenden Vertexdaten vier Werte hinzu.
