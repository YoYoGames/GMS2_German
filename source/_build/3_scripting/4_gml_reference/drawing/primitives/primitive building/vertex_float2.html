Vertex_float2

Fügen Sie einem Knoten Gleitkommawerte hinzu.

Syntax:

vertex_float2(buffer, float, float);


Streit Beschreibung
buffer Der Puffer, in den die Informationen geschrieben werden.
float Der erste Eingabewert.
float Der zweite Eingabewert.


Rückkehr: N / A


Beschreibung

Diese Funktion fügt den Scheitelpunktdaten zwei Gleitkommawerte hinzu. Der Eckpunkt muss korrekt formatiert worden sein, um dies mit dem Befehl zu akzeptieren vertex_format_add_custom Funktion.


Beispiel:

vertex_float2(buff, 0.05, 0.01);

Der obige Code fügt den zu definierenden Vertex-Daten zwei Gleitpunktwerte hinzu.
