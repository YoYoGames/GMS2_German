vertex_create_buffer_from_buffer_ext

Erstellen Sie einen Vertex-Puffer einer bestimmten Größe.

Syntax:

vertex_create_buffer_from_buffer_ext(buffer, format, src_offset, vert_num);


Streit Beschreibung
buffer Der Puffer, aus dem der Vertex-Puffer erstellt werden soll.
format Das zu verwendende primitive Vertexformat.
src_offset Der Offset innerhalb des Quellpuffers, aus dem kopiert werden soll.
vert_num Die Anzahl der Scheitelpunkte, die der Puffer haben sollte.


Rückgabe: Zeiger


Beschreibung

Wie bei der Funktion vertex_create_buffer, diese Funktion wird einen neuen Vertex-Puffer erstellen, nur die Vertex-Daten, die es speichert, werden jetzt aus dem regulären Puffer kopiert, der als die Quelle angegeben wird. Der Puffer ist mit dem Vertex-Format für die Erstellung von Primitiven zur Verwendung mit (zum Beispiel) Shadern vorformatiert, und Sie können auch einen Offset innerhalb des Quellpuffers angeben, aus dem kopiert werden soll, und die Anzahl der Scheitelpunkte, die der endgültige Puffer haben sollte. Beachten Sie, dass die Scheitelpunktdaten möglicherweise beschädigt sind, wenn die Anzahl der Scheitelpunkte nicht mit den kopierten Scheitelpunkten übereinstimmt.


Beispiel:

vertex_format_begin();
vertex_format_add_position_3d();
vertex_format_add_colour();
vertex_format_add_textcoord();
var my_format = vertex_format_end();
v_buff = vertex_create_buffer_from_buffer(global.modelBuff, myFormat, 0, 512);

Der obige Code erstellt ein neues Vertex-Format und erstellt dann einen neuen Vertex-Puffer aus einem zuvor erstellten regulären Puffer, wobei das benutzerdefinierte Vertex-Format mit 0 Versatz darauf angewendet wird. Die Funktion teilt dem neuen Vertex-Puffer mit, dass er 512 Vertices erwarten sollte.
