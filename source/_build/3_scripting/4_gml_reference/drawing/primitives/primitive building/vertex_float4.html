vertex_float4

Fügen Sie einem Knoten Gleitkommawerte hinzu.

Syntax:

vertex_float4(buffer, float, float, float, float);


Streit Beschreibung
buffer Der Puffer, in den die Informationen geschrieben werden.
float Der erste Eingabewert.
float Der zweite Eingabewert.
float Der dritte Eingabewert.
float Der vierte Eingabewert.


Rückkehr: N / A


Beschreibung

Diese Funktion fügt den Scheitelpunktdaten vier Gleitkommawerte hinzu. Der Eckpunkt muss korrekt formatiert worden sein, um dies mit dem Befehl zu akzeptieren vertex_format_add_custom Funktion.


Beispiel:

vertex_float4(buff, 0.05, 0.01, room_width / x, room_height / y);

Der obige Code fügt den zu definierenden Eckendaten vier Gleitkommawerte hinzu.
