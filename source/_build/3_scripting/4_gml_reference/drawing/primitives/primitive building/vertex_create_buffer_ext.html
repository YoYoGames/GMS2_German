vertex_create_buffer_ext

Erstellen Sie einen Vertex-Puffer einer bestimmten Größe.

Syntax:

vertex_create_buffer_ext(size);


Streit Beschreibung
size Die anfängliche Größe des Puffers (in Byte).


Rückgabe: Zeiger


Beschreibung

Wie bei der Funktion vertex_create_buffer, diese Funktion erstellt einen neuen Vertex-Puffer. Dies ist ein spezieller Grow- Puffer, der von GameMaker Studio 2 erstellt wurde und GameMaker Studio 2 ist, um Primitive für die Verwendung mit Shadern zu erstellen. Sie können eine anfängliche Anfangsgröße für den Puffer (in Bytes) angeben und es wird ein "handle" (Index) für den Puffer zurückgegeben, der dann in allen weiteren Aufrufen des Puffers verwendet werden sollte.


Beispiel:

v_buff = vertex_create_buffer_ext(1024 * 1024);

Der obige Code erstellt einen neuen Vertex-Puffer, zunächst 1 MB groß, und speichert sein Handle in der Variablen "v_buff".
