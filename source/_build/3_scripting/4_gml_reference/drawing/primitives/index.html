Primitive zeichnen und erstellen


Ein Grundelement ist eine Zeichnungsart, bei der Punkte verwendet werden, um zu definieren, was auf dem Bildschirm angezeigt wird. Sie können aus einzelnen Punkten, Linien oder Dreiecken bestehen und können farbig sein, Alpha-Blending haben und sogar eine Textur erhalten, was bedeutet, dass sie verwendet werden können, um einige sehr aufregende Effekte in einem 2D-Spiel zu erzeugen! Zum Beispiel können Sie ein Rechteck mit mehreren Dreiecks-Primitiven erstellen, es dann mit einem Flag-Sprite texturieren und im Draw-Ereignis einer Instanz dieses "wave" im Wind haben und das verwendete Sprite verzerren, indem Sie einfach die Punkte ändern, die sich bilden das Primitive.

WARNUNG! Diese Funktionen funktionieren nicht mit dem HTML5-Modul, es sei denn, Sie haben OpenGL in den Spieloptionen aktiviert.

GameMaker Studio 2 können Sie eigene Vertex-Formate definieren, aus denen Sie Ihre eigenen benutzerdefinierten Grundelemente erstellen können. Dies kann Shader-Operationen erheblich beschleunigen oder dazu verwendet werden, ihre Fähigkeiten zu erweitern und überraschende Effekte zu erzeugen.
