Zeichenpfad

Zeichnet eine Pfadressource.

Syntax:

draw_path(path, x, y, absolute);


Streit Beschreibung
path Der Pfad zum Zeichnen
x Die x-Koordinate, an der der Pfad gezeichnet wird
y Die y-Koordinate, in der der Pfad gezeichnet wird
absolute Ob der Pfad an der absoluten Position gezeichnet ist ( true ) oder die relative Position ( false )


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie in GameMaker Studio 2 einen Pfad zum Bildschirm zeichnen. Der Pfad wird als einfache Linie gezeichnet und kann entweder relativ zur aufrufenden Instanz oder an der absoluten Position sein, an der er im Pfadeditor oder über Code erstellt wurde. Diese Funktion ist sehr nützlich beim Debuggen von dynamischen Pfaden (zB solche, die für Instanzen mit dem mp_grid_path Funktion).


Beispiel:

if mp_grid_path(grid, path, x, y, obj_Player.x, obj_Player.y, 1)
   {
   draw_path(path, x, y, false);
   }

Der obige Code wird die verwenden mp_grid_path Funktion um einen Pfad zu generieren und in der Variable "Pfad" zu speichern. Wenn der Pfad erfolgreich erstellt wurde, wird er auf dem Bildschirm an einer Position relativ zu der Instanz gezeichnet, auf der der Code ausgeführt wird.
