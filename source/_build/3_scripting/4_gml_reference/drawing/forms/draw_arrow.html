Zeichnungs_Zeichnung

Zeichnet einen Pfeil mit einem ein Pixel breiten Stamm und dem vom Benutzer definierten Punkt.

Syntax:

draw_arrow(x1, y1, x2, y2, size);


Streit Beschreibung
x1 Die x-Koordinate des Zeilenanfangs.
y1 Die y-Koordinate des Zeilenanfangs.
x2 Die X-Koordinate des Linienendes (wo die Pfeilspitze endet).
y2 Die y-Koordinate des Linienendes (wo die Pfeilspitze endet).
size Die Länge des Pfeils in Pixel.


Rückkehr: N / A


Beschreibung

Diese Funktion zeichnet einen Pfeil von Punkt (x1, y1) zu Punkt (x2, y2). Der Stamm des Pfeils wird entlang dieser Punkte gezeichnet, wobei der eigentliche Pfeilkopf am Ende gezeichnet wird, wobei die Größe der Pfeilspitze durch das Argument "Größe" definiert ist und als Teil des Stamms berechnet wird, so dass der Endpunkt ist immer mit der durch x2, y2 definierten Position ausgerichtet. Die Breite der Pfeilspitze wird automatisch proportional zur Länge berechnet.


Beispiel:

draw_arrow(x, y, mouse_x, mouse_y, 10);

Der obige Code zeichnet einen Pfeil von der Position der Instanz zur Position der Maus, wobei der Pfeilkopf 10 Pixel lang ist.
