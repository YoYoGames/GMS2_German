draw_button

Zeichnet eine einfache Schaltfläche in beliebiger Größe, entweder gedrückt oder nicht gedrückt.

Syntax:

draw_button(x1, y1, x2, y2, up);


Streit Beschreibung
x1 Die x-Koordinate der Schaltfläche links
y1 Die y-Koordinate des oberen Bereichs der Schaltfläche
x2 Die x-Koordinate der rechten Seite der Schaltfläche
y2 Die y-Koordinate der Unterseite der Schaltfläche
up Ob der Button aktiv ist ( true ) oder runter ( false )


Rückgabe: N / A


Beschreibung

Diese Funktion zeichnet eine sehr einfache, rechteckige Schaltfläche mit der aktuell ausgewählten Zeichenfarbe und dem Alpha-Zeichen. Das Argument aufwärts definiert, wie der abgeschrägte Kanteneffekt aussieht (siehe Abbildung unten):

HINWEIS: Wenn Sie eine Form mit einem Shader zeichnen möchten, sollten Sie wissen, dass die meisten Shader die folgenden Eingaben erwarten: Scheitelpunkt, Textur, Farbe. Bei Verwendung dieser Funktion werden jedoch nur Vertex- und Farbdaten übergeben, sodass der Shader möglicherweise nichts zeichnet (oder etwas zeichnet, aber nicht korrekt). Wenn Sie auf diese Weise Formen zeichnen müssen, sollte der Shader entsprechend angepasst werden.



Beispiel:

draw_button(100, 100, 200, 150, !mouse_check_button(mb_left));

Dadurch wird eine Schaltfläche gezeichnet, die bei gedrückter linker Maustaste angezeigt wird.
