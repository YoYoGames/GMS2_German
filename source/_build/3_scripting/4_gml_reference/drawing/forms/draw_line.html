draw_line

Zeichnet eine Linie ein Pixel breit.

Syntax:

draw_line(x1, y1, x2, y2);


Streit Beschreibung
x1 Die x-Koordinate des Zeilenanfangs.
y1 Die y-Koordinate des Zeilenanfangs.
x2 Die x-Koordinate des Endes der Linie.
y2 Die y-Koordinate des Endes der Linie.


Rückgabe: N / A


Beschreibung

Mit dieser Funktion können Sie zwischen zwei beliebigen Punkten im Spielzimmer eine 1 Pixel breite Linie zeichnen.

HINWEIS: Wenn Sie eine Form mit einem Shader zeichnen möchten, sollten Sie wissen, dass die meisten Shader die folgenden Eingaben erwarten: Scheitelpunkt, Textur, Farbe. Bei Verwendung dieser Funktion werden jedoch nur Vertex- und Farbdaten übergeben, sodass der Shader möglicherweise nichts zeichnet (oder etwas zeichnet, aber nicht korrekt). Wenn Sie auf diese Weise Formen zeichnen müssen, sollte der Shader entsprechend angepasst werden.



Beispiel:

draw_set_colour(c_lime);
draw_line(50,50,150,50);

Dadurch wird eine hellgrüne horizontale Linie vom Punkt (50,50) zum Punkt (150,50) gezeichnet.
