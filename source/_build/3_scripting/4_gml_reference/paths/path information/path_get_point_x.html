path_get_point_x

Gibt die X-Koordinate des angegebenen Definitionspunkts eines bestimmten Pfads zurück.

Syntax:

path_get_point_x(index, n);


Streit Beschreibung
index Der Index des zu überprüfenden Pfads.
n Die zu prüfende Punktidentität.


Rückgabe: Echt.


Beschreibung

Diese Funktion gibt die x-Position (in Raumkoordinaten) des Punkts zurück, den Sie für den von Ihnen indizierten Pfad eingegeben haben. Wenn der Punkt außerhalb des Bereichs des Pfads liegt (dh ein Pfad hat 8 Punkte und Sie fragen nach der x-Position von Punkt 10), wird der Wert 0 zurückgegeben.


Beispiel:

if path_position = 1
   {
   var i;
   i = floor(random(path_get_number(mypath)));
   x = path_get_point_x(mypath, i);
   y = path_get_point_y(mypath, i);
   path_position = (1 / path_get_number(mypath)) * i;
   }

Der obige Code überprüft, ob sich eine Instanz am Ende eines Pfades befindet. Wenn dies der Fall ist, wird ein zufälliger Punkt auf dem Pfad ausgewählt und die Instanz an diesen Punkt verschoben.
