Pfad_Get_Name

Gibt den Namen eines bestimmten Pfades zurück.

Syntax:

path_get_name(index);


Streit Beschreibung
index Der Index des zu überprüfenden Pfads.


Rückgabe: Zeichenfolge.


Beschreibung

Diese Funktion gibt den Namen des Pfads zurück, auf den als Zeichenfolge verwiesen wird. Der Name ist, was auch immer verwendet wurde, um den Pfad im Editor zu definieren oder (wenn der Pfad durch eine Code-Funktion erstellt wurde), wird eine Zeichenkette mit dem Format "_newpathXX" zurückgegeben, wobei "XX" die Nummer des erzeugten Pfades ist. beginnend bei 0 und jedes Mal um eins erhöht, wenn ein neuer Pfad erstellt wird. Bitte beachten Sie, dass dies nur eine Zeichenkette ist und nicht verwendet werden kann, um den Pfad direkt zu referenzieren - dazu benötigen Sie den Pfadindex. Sie können diese Zeichenfolge jedoch verwenden, um den Pfadindex mithilfe der zurückgegebenen Zeichenfolge zusammen mit der Funktion abzurufen asset_get_index().


Beispiel:

name = path_get_name(mypath);

Dies setzt "name" auf den Namen des Pfades, der in der Variablen "mypath" indiziert ist.
