Pfade

GameMaker Studio 2 verfügt über eine umfangreiche Sammlung von Funktionen und integrierten Variablen zur Verwendung von Pfaden in Ihrem Spiel. Dieser Abschnitt behandelt die grundlegenden Aspekte, die beim Umgang mit Pfaden verwendet werden, die zuvor im Pfad-Editor erstellt wurden. Es gibt jedoch weitere Funktionen, mit denen Sie eigene Pfade mithilfe von Code erstellen und auf verschiedene Weise bearbeiten können.

Folgende Grundfunktionen existieren, die sich mit Pfaden befassen:

  • Pfad_Start
  • Pfad_Ende

Die folgenden Variablen sind bei der Erstellung in alle Instanzen integriert und können zum Abrufen von Informationen zu einem Pfad oder zum Ändern bestimmter Eigenschaften verwendet werden:

  • Pfad_Index
  • path_position
  • path_positionvorhergehend
  • Pfad_Geschwindigkeit
  • path_scale
  • Pfad_Orientierung
  • Pfad_endaktion

Beachten Sie, dass Sie hier weitere Informationen zu Instanzvariablen finden: Instanzvariablen.

Weitere Funktionen für Pfade finden Sie in den folgenden zwei Abschnitten:

  1. Pfadinformationen
  2. Pfade ändern
