Pfad_Löschen

Löscht einen bestimmten Pfad.

Syntax:

path_delete(index);


Streit Beschreibung
index Der Index des zu löschenden Pfads.


Rückkehr: N / A


Beschreibung

Sie können diesen Code verwenden, um einen Pfad aus dem Speicher zu entfernen. Wenn dieser Pfad dynamisch mit erstellt wurde path_add Die Variable, die den Pfadindex enthält, ist nicht länger gültig für den Zugriff auf den Pfad, da dieser Pfad nicht mehr existiert. Wenn der Pfad mit dem Pfad-Editor erstellt wurde, kann dieser Pfad nicht mehr im gesamten Spiel aufgerufen werden.

WICHTIG! Diese Funktion ist mit der Testlizenz des Produkts nicht verfügbar.



Beispiel:

var t_path;
t_path = path_add();
if mp_grid_path(grid, t_path, x, y, obj_Player.x, obj_Player.y, 1)
   {
   path_assign(mypath, t_path);
   }
path_delete(t_path);

Der obige Code erstellt einen Pfad und speichert seinen Index in einer lokalen Variablen. Dieser Pfad wird dann zum Speichern eines mp_grid_path generierter Pfad, der, wenn er seinen Weg zum Ziel gefunden hat, dem Pfad zugewiesen wird, der in "mypath" indiziert ist. Schließlich wird der "t_path" gelöscht.
