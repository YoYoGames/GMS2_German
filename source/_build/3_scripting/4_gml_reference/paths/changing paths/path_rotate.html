path_rotate

Dreht den angegebenen Pfad in Bezug auf seinen Mittelpunkt.

Syntax:

path_rotate(index, angle);


Streit Beschreibung
index Der Index des zu spiegelnden Pfads.
angle Der Winkel der Drehung Grad.


Rückkehr: N / A


Beschreibung

Sie können diese Funktion verwenden, um einen bestimmten Pfad um seinen Mittelpunkt zu drehen. Denken Sie daran, dass in GameMaker Studio 2 (sofern Sie keine Physik verwenden) die Winkel gegen den Uhrzeigersinn berechnet werden. Wenn Sie den Pfad um 90 Grad drehen, wird der Pfad nach links gedreht. Diese Funktion ändert die tatsächliche Pfadressource und beeinflusst somit permanent, wie der Pfad von allen Instanzen im Spiel vom Moment der Verwendung der Funktion bis zum Ende des Spiels verwendet wird.


Beispiel:

path_rotate(path0, 90);

Dies würde Pfad0 um 90 Grad gegen den Uhrzeigersinn drehen.
