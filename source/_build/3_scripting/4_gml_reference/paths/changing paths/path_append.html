path_append

Verbindet zwei Wege zusammen.

Syntax:

path_append(index, path);


Streit Beschreibung
index Der neue Index des Pfades.
path Der Index des Pfades, der an 'index' angehängt wird.


Rückkehr: N / A


Beschreibung

Mit diesem Code können Sie einen Pfad an einen anderen anhängen und diese effektiv verbinden. Damit dies funktioniert, müssen beide Pfade vorher existieren (entweder im Pfadeditor erstellt oder mit Hilfe von path_add ).

Hinweis: das wird den Pfad nicht entfernen, der von den Ressourcen angefügt wird. Es ist immer noch da, nur hat es jetzt keine Punkte mehr und wenn du es nicht mehr benutzen willst, solltest du es mit der Funktion entfernen path_delete.


Beispiel:

path_append(mypath, mppath); path_delete(mppath);

Der obige Code hängt die Pfaddaten, die in der Variablen "mppath" indiziert sind, an den Pfad an, der in der Variablen "mypath" indiziert ist, und löscht dann den "mppath" aus dem Speicher.
