path_change_point

Ändert einen vorhandenen Pfad zu Punktattributen.

Syntax:

path_change_point(index, n, x, y, speed);


Streit Beschreibung
index Der Index des Pfades zum Ändern des Punktes von.
n Der definierende Punkt zum Ändern der Attribute von.
x Die neue x-Koordinate (relativ zum Pfad) des Punkts.
y Die neue y-Koordinate (relativ zum Pfad) des Punkts.
speed Der neue Geschwindigkeitsfaktor des Punktes.


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie die Position und / oder den Geschwindigkeitsfaktor eines zuvor für einen Pfad definierten Punkts ändern (der Pfad kann im Pfadeditor oder über Code mit Hilfe von erstellt worden sein path_add ).


Beispiel:

for (var i = 0; i < path_get_number(path0); i ++)
   {
   var px = path_get_point_x(path0, i) + 32 - random(64);
   var py = path_get_point_y(path0, i) + 32 - random(64);
   path_change_point(path0, i, px, py, 100);
   }

Der obige Code durchläuft alle Punkte im Pfad, der als "path0" indiziert ist, und positioniert alle Punkte in einer zufälligen Position innerhalb eines Bereichs von 64x64 Pixeln neu.
