path_duplicate

Dupliziert einen vorhandenen Pfad und gibt seinen Index zurück.

Syntax:

path_duplicate(index);


Streit Beschreibung
index Der Index des vorhandenen Pfads, der dupliziert werden soll.


Rückgabe: Echt


Beschreibung

Diese Funktion nimmt einen Pfad und kopiert ihn in einen neuen Pfad. Der neue Pfad wird dabei erstellt, und der Index wird zurückgegeben, damit er in allen weiteren Aufrufen verwendet wird, um diesen neuen Pfad zu verwenden.

WICHTIG! Diese Funktion ist mit der Testlizenz des Produkts nicht verfügbar.



Beispiel:

mypath = path_duplicate(choose(path_1, path_2, path_3, path_4));

Der obige Code wählt eine von vier Pfadressourcen aus und dupliziert sie, wobei der Index des neuen Pfads in der Variablen "mypath" gespeichert wird.
