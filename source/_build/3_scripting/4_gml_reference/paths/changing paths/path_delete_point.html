path_delete_point

Löscht einen Punkt aus einem vorhandenen Pfad.

Syntax:

path_delete_point(ind, n);


Streit Beschreibung
index Der Index des Pfads, aus dem ein Punkt gelöscht werden soll.
n Der zu löschende Definitionspunkt.


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie einen Punkt aus dem angegebenen Pfad entfernen. Path-Punkte werden von 0 nummeriert, so dass der erste Punkt immer der 0te Punkt und der letzte Pfadpunkt sein wird path_get_number() - 1.


Beispiel:

path_delete_point(mypath, path_get_number(mypath) - 1);

Dies würde den letzten Punkt auf dem Pfad löschen, der in der Variablen "mypath" indiziert ist.
