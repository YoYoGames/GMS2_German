Pfad_Zuordnung

Dupliziert einen vorhandenen Pfad.

Syntax:

path_assign(index, path);


Streit Beschreibung
index Der Index des Pfades, der überschrieben werden soll.
path Der Index des Pfades, der 'index' überschreibt.


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie die Pfaddaten von einem Pfad zu einem anderen kopieren. Der Pfad, in den kopiert wird , wird zuerst gelöscht (falls ein Pfadpfad vorhanden ist) und vollständig durch den Pfad überschrieben, von dem kopiert wird. Kein Pfad wird dabei gelöscht und das Ergebnis sind zwei Pfade mit zwei verschiedenen Indizes, aber die exakt gleiche Form und die gleichen Punkte.


Beispiel:

mypath = path_add();
path_assign(mypath, choose(path_1, path_2, path_3));

Der obige Code erstellt einen neuen, leeren Pfad, der in der Variablen "Mypath" indiziert ist, und kopiert dann die Pfaddaten von einer der drei verfügbaren Pfadressourcen.
