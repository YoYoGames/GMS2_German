Pfad_Rescale

Skaliert den gegebenen Pfad in Bezug auf seinen Mittelpunkt.

Syntax:

path_rescale(index, xscale, yscale);


Streit Beschreibung
index Der Index des Skalierungswegs.
xscale Womit die aktuelle horizontale Skalierung multipliziert werden soll. Standardskala ist 1.
yscale Womit die aktuelle vertikale Skalierung multipliziert werden soll. Standardskala ist 1.


Rückkehr: N / A


Beschreibung

Diese Funktion kann verwendet werden, um den gegebenen Pfad entlang beider (oder entweder) der vertikalen und horizontalen Achse neu zu skalieren, wobei grundsätzlich jeder der Pfadpunkte an eine neue Position bewegt wird, die dieser Skalierung um die Mitte des Pfads entspricht. Diese Funktion ändert die tatsächliche Pfadressource und beeinflusst somit permanent, wie der Pfad von allen Instanzen im Spiel vom Moment der Verwendung der Funktion bis zum Ende des Spiels verwendet wird.


Beispiel:

path_rescale(path0, 3, 3);

Dies würde path0 auf das Dreifache seiner ursprünglichen Größe skalieren.
