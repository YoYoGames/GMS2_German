path_reverse

Kehrt den angegebenen Pfad um.

Syntax:

path_reverse(index);


Streit Beschreibung
index Der Index des zu ändernden Pfads.


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie die Reihenfolge umkehren, in der die einzelnen Wegpunkte nummeriert sind. Wenn also zB der Weg 5 Punkte hat, wird Punkt 0 zu Punkt 4, Punkt 1 wäre Punkt 3 und Punkt 2 würde nicht geändert. Die tatsächliche Position der Punkte bleibt gleich, nur die Reihenfolge, in der sie bearbeitet werden, wird geändert. Diese Funktion ändert die tatsächliche Pfadressource und beeinflusst somit permanent, wie der Pfad von allen Instanzen im Spiel vom Moment der Verwendung der Funktion bis zum Ende des Spiels verwendet wird.


Beispiel:

path_reverse(path0);

Dies würde die Reihenfolge umkehren, in der alle Punkte auf dem in "path0" indizierten Pfad verarbeitet werden.
