phy_speed_y

Die y-Komponente des Geschwindigkeitsvektors.

Syntax:

phy_speed_y;


Rückgabe: Echt


Beschreibung

Diese Variable kann verwendet werden, um die y-Komponente des linearen Geschwindigkeitsvektors der Instanz zu erhalten oder zu ändern und wird in Pixel pro Schritt definiert (für Pixel pro Sekunde, siehe phy_linear_velocity_y ). Ändern Sie dies für eine statische Instanz (dh eine Instanz mit 0 Dichte) wird daraus eine kinematische Instanz.


Beispiel:

if phy_speed_y != 0
   {
   phy_speed_y = 0;
   }

Der obige Code prüft die y-Komponente des linearen Geschwindigkeitsvektors und wenn es etwas anderes als 0 ist, wird es auf 0 gesetzt.
