phy_kinematisch

Gibt zurück, ob die Instanz ein kinematischer Körper ist oder nicht.

Syntax:

phy_kinematic;


Rückgabe: Boolean


Beschreibung

Diese schreibgeschützte Variable wird zurückgegeben true wenn die Instanz als kinematisches Objekt klassifiziert wird, oder false wenn es das nicht ist. Eine kinematische Instanz hat eine unendliche Masse (eine Dichte von 0), kann sich aber bewegen. Um also eine Instanz kinematisch zu machen, würden Sie zuerst eine statische Instanz erstellen und dann eine oder mehrere der Instanzvariablen für die Bewegung festlegen (dh: phy_speed_x ).


Beispiel:

if !phy_kinematic
   {
   phy_speed_x = 5;
   }

Der obige Code überprüft, ob die Instanz kinematisch ist. Ist dies nicht der Fall, wird die horizontale Geschwindigkeit auf 5 gesetzt.
