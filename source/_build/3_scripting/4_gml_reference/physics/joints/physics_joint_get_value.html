physics_joint_get_value

Diese Funktion testet eine Verbindung und gibt einen Wert zurück, der von der Art der getesteten Verbindung und der verwendeten Konstante abhängt.

Syntax:

physics_joint_get_value(joint, value)


Streit Beschreibung
joint Der Index des Gelenks, das Sie testen möchten
value Die Konstante für die gemeinsame Eigenschaft, die Sie testen möchten


Rückgabe: Echt


Beschreibung

Indem Sie eine Reihe vordefinierter Konstanten verwenden, können Sie GameMaker Studio 2 bitten, Ihnen eine Reihe von Informationen über den Zustand eines bestimmten Gelenks zu geben. Dies ist sehr nützlich, da es Ihnen die Möglichkeit gibt, Verbindungen zu löschen oder das Verhalten einer Instanz zu ändern, je nachdem, was Sie gerade benötigen. Es gibt eine Reihe von Konstanten, die in dieser Funktion verwendet werden können und sie finden Sie hier: Physics Joint Constants, aber seien Sie sich bewusst, dass komplexe Berechnungen durchgeführt werden, wenn Sie diese aufrufen, also sollten sie mit Vorsicht verwendet werden und nur wenn nötig und notieren dass viele einzigartig für eine bestimmte Art von Gelenk sind.

Wenn die Eigenschaft nicht existiert (wenn Sie zum Beispiel ein Rollengelenk für die prismatische Translation überprüfen), ist 0 der Rückgabewert.


Beispiel:

var reactionForceX, reactionForceY, reactionForce;
if shipJoint
   {
   reactionForceX = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);
   reactionForceY = physics_joint_get_value(shipJoint, phy_joint_reaction_force_y);
   reactionForce = point_distance(0, 0, reactionForceX, reactionForceY);
   if reactionForce > 2
      {
      physics_joint_delete(shipJoint);
      shipJoint = -1;
      }
   }

Der obige Code überprüft, ob die Variable "shipJoint" einen gemeinsamen Index enthält, und wenn dies der Fall ist, berechnet er dann die Kraft, die auf diesen Knoten angewendet wird, unter Verwendung der zwei Konstanten. Wenn die Gesamtkraft größer als 2 ist, wird das Gelenk gelöscht.
