physics_joint_gear_create

Erstellen Sie eine Zahnradverbindung mit zwei Drehgelenken oder einem Drehgelenk und einem Prismengelenk.

Syntax:

physics_joint_gear_create(inst1, inst2, joint_1, joint_2, ratio)


Streit Beschreibung
inst1 Die erste Instanz zum Verbinden mit dem Gelenk
inst2 Die zweite Instanz, um sich mit dem Gelenk zu verbinden
joint_1 Ein zuvor definiertes Drehgelenk
joint_2 Ein zuvor definiertes Dreh- oder Prismengelenk
ratio Stellen Sie das Geschwindigkeitsverhältnis zwischen den beiden Gelenken ein


Rückgabe: Index des Gelenks


Beschreibung

Wenn Sie eine anspruchsvolle mechanische Vorrichtung erstellen möchten, können Sie Zahnräder verwenden. Im Prinzip können Sie in GameMaker Studio 2 Verzahnungen erstellen, indem Sie Compound-Instanzen zum Modellieren von Verzahnungen verwenden. Dies ist jedoch nicht sehr effizient und könnte für den Autor mühsam sein! Zum Glück gibt es eine einfachere Methode, und das ist eine Zahnradverbindung zu verwenden. Um eins zu machen, müssen Sie Ihre Vorrichtungen vorher definiert haben und die zwei grundlegenden Verbindungen geschaffen haben, die Ihr Zahnrad umfassen werden - diese müssen aus einem Drehgelenk und entweder einem prismatischen Gelenk oder einem anderen Drehgelenk bestehen. Das Bild unten zeigt, wie eine Ausrüstung normalerweise in einem Spiel erstellt wird: Was passiert also? Nun, sobald die zwei Gelenke in den Gang hinzugefügt werden, wird sich die Interaktion mit der einen auf die andere auswirken. Wenn Sie also in obigem Beispiel inst2 drehen, bewegt sich inst3 auf und ab, oder wenn Sie inst3 auf und ab bewegen dann wird sich inst2 drehen. Sie können auch die Getriebeübersetzung ändern, was bedeutet, dass Sie eine Instanz mehr (oder weniger) bewegen müssen, um den gewünschten Effekt zu erzielen. Der Code im Beispiel unten zeigt, wie etwas wie das obige Bild erstellt werden kann.

Hinweis: Wenn Sie eine der beiden Instanzen, die an der Verzahnung beteiligt sind, löschen müssen (oder nur ihre Gelenke löschen), müssen Sie die Verzahnung zuerst mit löschen physics_joint_delete() sonst wirst du einen Fehler bekommen!


Beispiel:

var t_fix, g_fix, inst1, inst2, inst3, r_joint, p_joint;
g_fix = physics_fixture_create();
physics_fixture_set_box_shape(g_fix, 40, 10);
t_fix = physics_fixture_create();
physics_fixture_set_circle_shape(t_fix, 10);
physics_fixture_set_density(t_fix, 0.5);
inst1 = instance_create_layer(60, room_height - 30, "Background", obj_Ground);
inst2 = instance_create_layer(40, room_height - 300, "Instances", obj_Cog);
inst3 = instance_create_layer(150, room_height - 300, "Instances", obj_Barrel);
physics_fixture_bind(g_fix, inst1);
physics_fixture_bind(t_fix, inst2);
physics_fixture_bind(t_fix, inst3);
r_joint = physics_create_revolute_joint(inst1, inst2, 40, room_height - 300, -80, 80, 1, 10, 0.5, 1, 0);
p_joint = physics_create_prismatic_joint(inst1, inst3, 150, room_height - 300, 0, 1, -10, 10, true, 0, 0, 0, 0);
physics_create_gear_joint(inst2, inst3, r_joint, p_joint, 0.5);

Der obige Code erstellt und definiert zwei Fixtures und erstellt dann drei Instanzen, eine "obj_Ground" und zwei weitere, "obj_Cog" und "obj_Barrel". Die Fixtures werden dann an diese Instanzen gebunden und es werden zwei Joints erstellt. Eine Drehverbindung zwischen dem Boden und dem Zahnrad und eine prismatische Verbindung zwischen dem Boden und dem Lauf. Schließlich wird eine Verzahnungsverbindung zwischen den Zahn- und Laufinstanzen unter Verwendung der zuvor definierten Dreh- und Prismengelenke erzeugt.
