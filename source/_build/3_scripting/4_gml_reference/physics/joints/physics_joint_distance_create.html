physics_joint_distance_create

Erstellen Sie eine Verbindung für eine gerade Linie, die zwei Instanzen verbindet, für die jeweils Vorrichtungen definiert und ihnen zugewiesen wurden.

Syntax:

physics_joint_distance_create(inst1, inst2, w_anchor1_x, w_anchor1_y, w_anchor2_x, w_anchor2_y, col)


Streit Beschreibung
inst1 Die erste Instanz zum Verbinden mit dem Gelenk
inst2 Die zweite Instanz, um sich mit dem Gelenk zu verbinden
w_anchor1_x Die erste x-Koordinate für das Gelenk innerhalb der Spielwelt
w_anchor1_y Die erste y-Koordinate für das Gelenk innerhalb der Spielwelt
w_anchor2_x Die zweite x Koordinate für das Gelenk innerhalb der Spielwelt
w_anchor2_y die zweite y-Koordinate für das Gelenk innerhalb der Spielwelt
col Ob die beiden Instanzen kollidieren können (wahr) oder nicht (falsch)


Rückgabe: Index des Gelenks


Beschreibung

Eines der einfachsten Gelenke ist ein Distanzgelenk, das besagt, dass der Abstand zwischen zwei Punkten in zwei Fällen konstant sein muss. Wenn Sie eine Distanzverbindung angeben, sollten die beiden Instanzen bereits erstellt und eine Fixture zugewiesen sein. Anschließend definieren Sie die zwei Ankerpunkte in Raumkoordinaten. Der erste Ankerpunkt ist mit der Instanz 1 verbunden, der zweite Ankerpunkt ist mit der Instanz 2 verbunden, und der Abstand zwischen diesen Punkten impliziert die Länge der Abstandsbeschränkung. Das Bild unten zeigt, wie das funktioniert:

Wie Sie sehen können, werden die Ankerpunkte als Raumkoordinaten angegeben, daher muss bei der Definition besonders darauf geachtet werden, dass die Instanzen gleichzeitig mit den Gelenken erstellt werden und nicht durch den Raumeditor in den Raum platziert werden. Sie sollten auch erkennen, dass die Gelenke unabhängig von der Größe des Sprites der Instanzen oder der Befestigungen, die sie angebracht haben, erstellt werden. Wenn Sie also eine Distanzverbindung an einer anderen Stelle als dem Ursprung der Instanz erstellen, ist sie immer noch gültig und beschränkt die beiden Instanzen auf die Position, an der sie erstellt wurde. Wenn Sie den Wert "col" auf festlegen true dann können die beiden Instanzen interagieren und miteinander kollidieren, aber nur wenn sie Kollisionsereignisse haben, wenn sie jedoch auf gesetzt sind false sie werden nicht kollidieren, egal was passiert.


Beispiel:

var mainFixture, o_id;
mainFixture = physics_fixture_create();
physics_fixture_set_circle_shape(mainFixture, sprite_get_width(sprite_index) / 2);
o_id=instance_create_layer(x+300, y, "Instances", obj_Rudder);
physics_fixture_bind(mainFixture, id);
physics_fixture_bind(mainFixture, o_id);
physics_joint_distance_create(id, o_id, x + 50, y, o_id.x - 50, o_id.y, 0);
physics_fixture_delete(mainFixture);

Der obige Code erstellt und definiert ein neues Fixture und erstellt dann eine Instanz von "obj_Rudder". Das Fixture wird dann der Instanz zugewiesen, auf der der Code ausgeführt wird, sowie der neu erstellten, und es wird ein Joint zwischen ihnen erstellt. Schließlich wird das Fixture gelöscht, da es nicht mehr benötigt wird.
