physics_joint_set_value

Diese Funktion setzt eine bestimmte Verbindungseigenschaft auf den angegebenen Wert.

Syntax:

physics_joint_set_value(joint, field, value)


Streit Beschreibung
joint Der Index des Gelenks, das Sie ändern möchten
field Die Konstante für die gemeinsame Eigenschaft, die Sie ändern möchten
value Der neue Wert für die gemeinsame Eigenschaft


Rückkehr: N / A


Beschreibung

Bestimmte Gelenkeigenschaften können auch nach der Herstellung des Gelenks verändert und eingestellt werden, jedoch nur an den Gelenken, die einen Motor haben, dh - Drehgelenke und Prismengelenke. Es gibt eine Reihe von Konstanten, die in dieser Funktion verwendet werden können und sie können hier gefunden werden: Physics Joint Constants.


Beispiel:

if physics_joint_get_value(revJoint, phy_joint_max_motor_torque) < 2
   {
   physics_joint_set_value(revJoint, phy_joint_max_motor_torque, 2);
   {

Der obige Code überprüft, ob das maximale Motordrehmoment der Gelenke auf weniger als 2 eingestellt ist. Wenn dies der Fall ist, wird der Wert auf 2 gesetzt.
