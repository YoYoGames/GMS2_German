Physik Gemeinsame Konstanten

Es gibt eine große Anzahl von Konstanten innerhalb des GameMaker Studio 2 GML, die spezifisch für Gelenke zwischen Fixtures in der Physikwelt sind. Diese können in Verbindung mit vielen der verschiedenen Gelenkfunktionen verwendet werden, um während der Simulation der Physik Informationen in Echtzeit zu setzen oder abzurufen. Sie sollten sich jedoch bewusst sein, dass komplexe Berechnungen durchgeführt werden, wenn Sie diese aufrufen, so dass sie mit Vorsicht und nur bei Bedarf verwendet werden sollten und beachten Sie, dass viele für einen bestimmten Gelenkstyp einzigartig sind. Beachten Sie auch, dass Sie zwar alle diese Werte mit der entsprechenden Funktion abrufen können, jedoch nur diejenigen, die als nicht schreibgeschützt gekennzeichnet sind.

Die folgenden Konstanten können auf jeden der verfügbaren Verbindungstypen angewendet werden:

Konstante Beschreibung Schreibgeschützt
phy_joint_anchor_1_x Die x-Koordinate des ersten Ankerpunkts des Gelenks im Raum Yes
phy_joint_anchor_1_y Die y-Koordinate des ersten Ankerpunkts des Gelenks im Raum Yes
phy_joint_anchor_2_x Die x-Koordinate des zweiten Ankerpunkts des Gelenks im Raum Yes
phy_joint_anchor_2_y Die y-Koordinate des zweiten Ankerpunkts des Gelenks im Raum Yes
phy_joint_reaction_force_x Dies ist die Reaktionskraft, die auf die zweite Instanz in einer Verbindung an der x-Ankerposition angewendet wird Yes
phy_joint_reaction_force_y Dies ist die Reaktionskraft, die auf die zweite Instanz in einer Verbindung an der y-Ankerposition angewendet wird Yes
phy_joint_reaction_torque Dies ist das Drehmoment, das in der zweiten Position in einer Verbindung an der Ankerposition angewendet wird Yes

Diese Konstanten sind für jene Gelenke, an denen ein Motor angebracht ist (drehend, prismatisch, Rad):

Konstante Beschreibung Schreibgeschützt
phy_joint_max_motor_force Der Wert, der beim Erstellen der Verbindung für die maximale Motorkraft angegeben wurde No
phy_joint_max_motor_torque Der Wert, der beim Erstellen des Gelenks für das maximale Motordrehmoment angegeben wurde No
phy_joint_motor_force Die aktuelle Motorkraft Yes
phy_joint_motor_speed Die aktuelle Motordrehzahl No
phy_joint_motor_torque Das aktuelle Motordrehmoment Yes

Für ein Drehgelenk können Sie die folgende Konstante verwenden (sowie die Motorkonstanten, falls eine hinzugefügt wurde):

Konstante Beschreibung Schreibgeschützt
phy_joint_angle Der Winkel, den eine Linie zwischen den beiden Ankerpunkten des Gelenks bildet. Dies wird unter Verwendung der physikalischen Weltkoordinaten berechnet
( nicht die Raumkoordinaten von GameMaker Studio 2 ) im Bogenmaß. 		Yes
phy_joint_angle_limits Aktivieren oder deaktivieren Sie die Winkelbegrenzung für das Gelenk. Setzen Sie den Wert auf true aktivieren oder false Etwas deaktivieren. No
phy_joint_upper_angle_limit Die obere Winkelgrenze für die Verbindung in Grad. No
phy_joint_lower_angle_limit Die untere Winkelgrenze für die Verbindung in Grad. No

Für ein prismatisches Gelenk können Sie folgende Konstante verwenden:

Konstante Beschreibung Schreibgeschützt
phy_joint_translation Ruft den Abstand zwischen den x / y-Ankerkoordinaten und den lokalen x / y-Koordinaten ab. Yes
phy_joint_speed Die aktuelle Bewegungsgeschwindigkeit. Yes

Für eine Riemenscheibenverbindung können Sie die folgenden Konstanten verwenden (ebenso wie für Distanzverbindungen):

Konstante Beschreibung Schreibgeschützt
phy_joint_length_1 Dies gibt die Länge des Gelenks von den ersten lokalen x / y-Koordinaten zu den ersten x / y-Koordinaten des Ankers zurück No
phy_joint_length_2 Dies wird die Länge der Verbindung von den zweiten lokalen x / y-Koordinaten zu den zweiten x / y-Koordinaten des Ankers zurückgeben Yes

Für eine Distanz-, Schweiß- und Radverbindung können Sie die folgenden Konstanten (wie auch für die Riemenscheibenverbindungen) verwenden:

Konstante Beschreibung Schreibgeschützt
phy_joint_damping_ratio Das Dämpfungsverhältnis ist nichtdimensional und definiert die "Federung" des Gelenks. Der Wert für diese Konstante liegt typischerweise zwischen 0 und 1, kann aber größer sein, und bei 1 ist die Dämpfung kritisch, was bedeutet, dass alle Oszillationen verschwinden sollten. No
phy_joint_frequency Dies wird die Oszillationsfrequenz für das Gelenk in Hertz zurückgeben (oder einstellen), und typischerweise sollte die Frequenz weniger als die Hälfte der Frequenz des Zeitschritts sein, wie durch die Funktion festgelegt physics_world_update_speed. No

Für eine Reibverbindung können Sie die folgenden Konstanten verwenden:

Konstante Beschreibung Schreibgeschützt
phy_joint_max_torque Der maximale Drehmomentwert für das Gelenk. No
phy_joint_max_force Der maximale Kraftwert für das Gelenk. No

Für eine Seilverbindung können Sie folgende Konstante verwenden:

Konstante Beschreibung Schreibgeschützt
phy_joint_max_length Die maximale Erweiterung für die Verbindung zwischen den beiden Ankerpunkten. No
