physics_joint_delete

Lösche ein Gelenk aus der Physikwelt.

Syntax:

physics_joint_delete(joint)


Streit Beschreibung
joint Der Index des Gelenks, das Sie löschen möchten


Rückkehr: N / A


Beschreibung

Sobald zwei Instanzen mit physikalischen Darstellungen durch ein Gelenk gebunden wurden, kann dies jederzeit wieder gelöscht werden. Normalerweise passiert dies automatisch, wenn eine der beiden Instanzen zerstört wird oder wenn der Raum endet, aber manchmal möchten Sie dies manuell tun. In diesen Fällen würden Sie diese Funktion verwenden.

Es sollte auch beachtet werden, dass, wann immer eine Instanz, die Teil einer Zahnradverbindung ist, zerstört wird, die Zahnradverbindung unter Verwendung dieser Funktion gelöscht werden sollte, bevor irgendeine der an der Bildung des Zahnrades beteiligten Instanzen (aber die verbleibenden Verbindungen werden automatisch gelöscht werden), z Beispiel im destroy-Ereignis der Instanz.


Beispiel:

var reactionForceX, reactionForceY, reactionForce;
if shipJoint
   {
   reactionForceX = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);
   reactionForceY = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);
   reactionForce = sqrt((reactionForceX + reactionForceX) + (reactionForceY + reactionForceY));
   if reactionForce > 2
      {
      physics_joint_delete(shipJoint);
      shipJoint = -1;
      }
   }

Der obige Code überprüft, ob die Variable "shipJoint" einen gemeinsamen Index enthält, und wenn dies der Fall ist, berechnet er dann die Kraft, die auf diesen Knoten angewendet wird, unter Verwendung der zwei Konstanten. Wenn die Gesamtkraft größer als 2 ist, wird das Gelenk gelöscht.
