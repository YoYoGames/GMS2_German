physik_welt_update_speed

Legt die Geschwindigkeit fest, mit der das Physiksystem aktualisiert wird.

Syntax:

physics_world_update_speed(speed)


Streit Beschreibung
speed die Anzahl der Male pro Sekunde, die das Physiksystem aktualisiert


Rückkehr: N / A


Beschreibung

GameMaker Studio 2 Dinge auf der Grundlage der GameMaker Studio 2 aktualisiert, sodass GameMaker Studio 2 im Verlauf einer Sekunde 30 Schritte GameMaker Studio 2, wenn Sie es auf 30 setzen. Jedoch für die Physik-Funktionen, die nicht genug sein können und Sie möchten, dass Dinge mit einer etwas höheren Geschwindigkeit aktualisiert werden, um Stabilität oder Präzision zu erhöhen. Zu diesem Zweck verwenden wir die Funktion physics_world_update_speed welches die Aktualisierungsgeschwindigkeit für das Physiksystem unabhängig von der Raumgeschwindigkeit einstellt. Dies bedeutet, dass Sie eine Raumgeschwindigkeit von 30 haben könnten, aber die Physik auf 60 setzen, was die Geschwindigkeit verdoppelt, mit der das Physiksystem seine Berechnungen aktualisiert und durchführt, verglichen mit der Geschwindigkeit, mit der der Schritt aktualisiert wird.

Hinweis: Sie können dies derzeit nicht auf weniger als die Zimmergeschwindigkeit einstellen, zukünftige Aktualisierungen können dies jedoch ändern.


Beispiel:

physics_world_update_speed(room_speed * 2);

Der obige Code setzt die Aktualisierungsgeschwindigkeit des Physiksystems auf das Doppelte der Raumgeschwindigkeit.
