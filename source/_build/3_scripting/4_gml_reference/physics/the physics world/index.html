Die Physikwelt

Bevor irgendwelche physikalischen Reaktionen in GameMaker Studio 2 auftreten können, müssen Sie zuerst die physikalische Welt definieren, in der sie passieren werden. Wenn Sie dies nicht tun, wird GameMaker Studio 2 standardmäßig die grundlegenden Kollisionsfunktionen verwenden und alle anderen physikalischen Codes oder Einstellungen werden ignoriert. Sie können die Physikwelt über den Raum-Editor einrichten, Sie können jedoch dieselben Eigenschaften auch über Code mit den folgenden Funktionen festlegen:

  • physik_welt_erschaffen
  • Physik_Weltgravität
  • physics_world_update_iterationen
  • physik_welt_update_speed
  • physics_world_draw_debug
  • physics_draw_debug
  • physik_pause_enable
