physik_pause_enable

Pausiert oder unterbricht die Simulation der Physik.

Syntax:

physics_pause_enable(flag)


Streit Beschreibung
flag Dies kann auf eingestellt werden true um die Simulation anzuhalten, oder false um es wieder zu beginnen.


Rückkehr: N / A


Beschreibung

Damit eine Physiksimulation funktioniert, muss sie normalerweise kontinuierlich sein und kann nicht gestoppt und gestartet werden, oder Instanzen können plötzlich von einem Ort zum anderen im Raum bewegt werden. Allerdings gibt es Momente, in denen Sie die Simulation, wie (zum Beispiel) pausieren müssen das Gerät o pausiert, und so würden Sie diese Funktion verwenden. Es wird die Simulation angehalten, wenn das Flag auf gesetzt ist true und es werden keine weiteren Physik-Berechnungen durchgeführt, bis das Flag gesetzt ist false nochmal.

HINWEIS: Dies ist von besonderem Nutzen, wenn Sie alle Instanzen in einem Raum deaktivieren möchten, da ein physischer Körper auch dann, wenn er deaktiviert wird, weiterhin in der Physikwelt berechnet und simuliert wird.


Beispiel:

if os_is_paused()
   {
   physics_pause_enable(true);
   instance_deactivate_all(true);
   instance_create_layer(x, y, "Controllers", obj_PauseMenu);
   }

Der obige Code prüft, ob das Betriebssystem pausiert wurde und wenn es dann ist, unterbricht es die Physikwelt, deaktiviert alles außer sich selbst und erstellt dann eine Pause-Menü-Instanz.
