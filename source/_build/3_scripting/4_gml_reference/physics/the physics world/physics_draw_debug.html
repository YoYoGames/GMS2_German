physics_draw_debug

Diese Funktion zeichnet ein grundlegendes Schema der Physikeigenschaften für die Instanz.

Syntax:

physics_draw_debug()

Rückkehr: N / A


Beschreibung

Wenn Sie ein Physiksystem in GameMaker Studio 2 erstellen, ist es oft wichtig, eine visuelle Darstellung dessen zu haben, was tatsächlich mit einer Instanz geschieht. Dies kann durch Verwendung erreicht werden physics_draw_debug, die einen schematischen Überblick über die physikalischen Eigenschaften der Instanz gibt, auf der der Code ausgeführt wird. Hier ist ein Bild davon, wie es aussieht:

Es sollte beachtet werden, dass dies im Draw-Ereignis der Instanz geschehen muss, damit es funktioniert, und es wird mit der aktuell definierten Zeichenfarbe gezeichnet.


Beispiel:

draw_set_colour(c_red);
physics_draw_debug();

Der obige Code zeichnet eine Darstellung der physikalischen Eigenschaften, die mit dieser Instanz verbunden sind, unter Verwendung der Farbe Rot.
