physics_fixture_set_kinematic

Diese Funktion zeigt an, dass das Gerät sich kinematisch verhält.

Syntax:

physics_fixture_set_kinematic(fixture)


Streit Beschreibung
fixture der Index der Vorrichtung


Rückkehr: N / A


Beschreibung

Es wird bestimmte Fälle in einem Spiel auf Physikbasis geben, die du bewegen willst, aber nicht durch Kräfte wie Schwerkraft oder Kräfte, die durch Kollisionen mit dynamischen Objekten entstehen (z. B. bewegliche Plattformen in einem Plattformspiel) ). Für solche Objekte bedeutet das einfache Setzen der Dichte einer Fixture auf 0, dass die Physik davon ausgeht, dass das Objekt statisch sein soll. Sollten Sie jedoch die phy_speed_x, phy_speed_y oder phy_angular_velocity Variablen für eine Instanz, an die eine solche Fixture gebunden ist, wird sie in ein kinematisches Objekt umgewandelt und beginnt mit der Bewegung. Ändern Sie die Instanz von statisch in kinematisch. Die mittlere Simulation kann in Bezug auf die Verarbeitung teuer sein (abhängig vom Zustand der Instanzen in der Physikwelt). Daher wird empfohlen, die Vorrichtung vor dem Binden mit dieser Funktion kinematisch zu setzen ein Beispiel, so dass es von Kollisionen und Kräften (wie der Schwerkraft) unbeeinflusst bleibt, sich aber mit konstanter linearer und / oder Winkelgeschwindigkeit bewegen kann.


Beispiel:

physics_fixture_set_kinematic(fix_Cloud);

Der obige Code zeigt an, dass das in "fix_Cloud" indizierte Fixture als kinematisch markiert sein sollte.
