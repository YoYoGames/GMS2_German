physics_fixture_set_box_shape

Mit dieser Funktion wird die Form der Fixture an die einer Box (Rechteck) mit den angegebenen Abmessungen angepasst.

Syntax:

physics_fixture_set_box_shape(fixture, halfWidth, halfHeight)


Streit Beschreibung
fixture der Index der Vorrichtung
halfWidth die halbe Breite der Box
halfHeight die halbe Höhe der Box


Rückkehr: N / A


Beschreibung

Diese Funktion definiert eine Boxform für Ihr Gerät. Es benötigt die halbe Breite und Höhe, da die Physikwelt diesen Wert weit mehr als ganze Breite / Höhe verwendet, um Dinge wie Kollisionen zu bestimmen.


Beispiel:

physics_fixture_set_box_shape(fix_Border, room_width/2, 10);

Der obige Code wendet eine Boxform auf die Fixture an, die in der Variable "fix_Border" mit einer Breite des Raums und einer Höhe von 20 Pixeln indiziert ist.
