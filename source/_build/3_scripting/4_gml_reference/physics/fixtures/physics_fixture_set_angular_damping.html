physics_fixture_set_angular_damping

Diese Funktion legt den Wert für die Winkeldämpfung eines Scheinwerfers fest.

Syntax:

physics_fixture_set_angular_damping(fixture, damping)


Streit Beschreibung
fixture der Index der Vorrichtung
damping die Winkeldämpfung der Leuchte, normalerweise zwischen 0 und 1


Rückkehr: N / A


Beschreibung

Wenn Sie an ein sich drehendes Objekt in der "realen Welt" denken, es sei denn, es hat einen Motor oder ist es im Raum, wird es im Laufe der Zeit aufgrund des Einflusses äußerer Kräfte (wie Reibung mit der umgebenden Luft) langsamer. Wir können die Funktion verwenden physics_fixture_set_angular_damping() in GameMaker Studio 2, um diesen Effekt zu simulieren und die Rotationsgeschwindigkeit von Instanzen in der GameMaker zu reduzieren, ohne dass sich eine rotierende Instanz unendlich weiter drehen würde. Dämpfungsparameter sollten zwischen 0 und unendlich liegen, wobei 0 keine Dämpfung und unendlich die volle Dämpfung bedeutet. Normalerweise verwenden Sie einen Dämpfungswert zwischen 0 und 1, aber Sie können einen beliebigen nicht negativen Wert verwenden, falls erforderlich.


Beispiel:

physics_fixture_set_angular_damping(fix_Ball, 0.1);

Der obige Code setzt die Winkeldämpfung der in "fix_ball" indizierten Vorrichtung auf 0,1.
