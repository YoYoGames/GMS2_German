physics_fixture_set_chain_shape

Diese Funktion stellt die Form der Vorrichtung auf die einer Kette ein.

Syntax:

physics_fixture_set_chain_shape(fixture, loop)


Streit Beschreibung
fixture Der Index der Vorrichtung
loop Ob die Kette zu schließen ( true ) oder nicht ( false ).


Rückkehr: N / A


Beschreibung

Diese Funktion definiert eine "Ketten" -Fixorform. Eine Kettenhalterung besteht aus einer Anzahl von Punkten, die dann unter Verwendung von Kantenformen miteinander verbunden werden, um sie zu verbinden. Die Funktion selbst nimmt den Index (ID) des Fixtures und kann dem Fixture mitteilen, ob er loopt oder nicht. Einstellen auf true Verbindet den letzten Punkt mit dem ersten Punkt mit einer Kante und setzt ihn auf false wird nicht. Beachten Sie, dass diese Funktion das Fixture darauf vorbereitet, die Punkte zu akzeptieren, die zur Herstellung der Kette erforderlich sind. Diese sollten nach dem Aufruf dieser Funktion hinzugefügt werden physics_fixture_add_point, ähnlich wie beim Bau eines Polygon Fixtures.

Wenn Sie eine Kettenhalterung erstellen, müssen Sie ihr mindestens zwei Punkte geben, aber Sie sind nicht auf die Anzahl der nachfolgenden Punkte beschränkt, die Sie zum fertigen Gerät hinzufügen können.


Beispiel:

var xx = -100;
var yy = room_height / 2;
var fix = physics_fixture_create(); physics_fixture_set_chain_shape(fix, false);
for (var i = 0; i < 10; i++;)
   {
   physics_fixture_add_point(fix, xx, yy);
   xx += 50 + random(150);
   yy = (room_height / 2) - 250 + irandom(500);
   }
physics_fixture_bind(fix, id);

Der obige Code erzeugt eine Reihe von "Kanten" -Vorrichtungen mit verschiedenen Höhen über die Länge des Raums.
