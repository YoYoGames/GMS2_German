physics_fixture_set_riction

Diese Funktion legt die Reibung einer Vorrichtung fest.

Syntax:

physics_fixture_set_friction(fixture, friction)


Streit Beschreibung
fixture der Index der Vorrichtung
friction die Reibung der Vorrichtung


Rückkehr: N / A


Beschreibung

Reibung ist die Kraft, die der relativen Bewegung von gegeneinander gleitenden Materialelementen widersteht, was sich in der GameMaker Studio 2 als Impulsverlust auswirkt, der durch die Kollision von zwei Instanzen mit an sie gebundenen Scheinwerfern verursacht wird. Wenn also zwei Instanzen kollidieren, wird ihre Bewegung von diesem Wert beeinflusst, wobei eine hohe Reibung einen größeren Impulsverlust verursacht als ein niedrigerer Wert. Beachten Sie, dass die Reibung normalerweise auf einen Wert zwischen 0 und 1 eingestellt ist. Sie können jedoch jeden nicht negativen Wert verwenden, falls erforderlich.


Beispiel:

physics_fixture_set_friction(fix_Brick, 0.1);

Der obige Code setzt die Reibung der in "fix_brick" indizierten Vorrichtung auf 0,1.
