physics_fixture_set_density

Diese Funktion legt die Dichte eines Scheinwerfers fest.

Syntax:

physics_fixture_set_density(fixture, density)


Streit Beschreibung
fixture der Index der Vorrichtung
density die Dichte der Befestigung berechnet als kg / m²


Rückkehr: N / A


Beschreibung

Die Dichte von etwas ist definiert als seine Masse pro Einheitsvolumen, was im Grunde bedeutet, wie viel Masse in den Raum eingepfercht ist, den es in der Welt aufnimmt. So hätte ein Ballon eine geringe Dichte, da er eine geringe Masse hat, die einen großen Raum einnimmt, während ein Führungsstab eine hohe Dichte haben würde, da er eine große Masse hat und dennoch wenig Raum einnimmt.

In der 2D-Physik von GameMaker Studio 2 die GameMaker Studio 2 automatisch aus den Werten berechnet, die Sie für die Dichte und die Oberfläche der Form eingeben, die Sie für das Fixture definieren. Dies hat einen direkten Einfluss darauf, wie viel Trägheit eine Leuchte hat und wie sie auf Kollisionen und Kräfte reagiert. Wenn man also eine kleine Form mit hoher Dichte macht, hat sie eine sehr große Masse (wie ein Bleistab), Wenn Sie jedoch eine große Form mit einer geringen Dichte definieren, hat sie eine viel kleinere Masse (wie ein Ballon).

Hinweis: Wenn Sie möchten, dass die Instanz, die das Fixture erhält, statisch ist (dh in der Spielwelt unbeweglich), sollte die Dichte auf 0 gesetzt werden, was im Wesentlichen eine unendliche Dichte ergibt.


Beispiel:

physics_fixture_set_density(fix_Balloon, 0.1);

Der obige Code setzt die Dichte des in "fix_Ballon" indizierten Fixtures auf 0.1.
