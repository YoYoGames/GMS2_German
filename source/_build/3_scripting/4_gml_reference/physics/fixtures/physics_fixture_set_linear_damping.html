physics_fixture_set_linear_damping

Diese Funktion stellt den linearen Dämpfungswert eines Scheinwerfers ein.

Syntax:

physics_fixture_set_linear_damping(fixture, damping)


Streit Beschreibung
fixture der Index der Vorrichtung
damping die Dämpfung der Vorrichtung, normalerweise zwischen 0 und 1


Rückkehr: N / A


Beschreibung

Dämpfung wird verwendet, um die physikalische Weltgeschwindigkeit von Instanzen zu reduzieren, und unterscheidet sich von Reibung darin, dass Reibung nur auftritt, wenn zwei Instanzen mit Vorrichtungen in Kontakt sind. Dämpfung ist auch viel billiger zu simulieren als Reibung, aber bitte beachten Sie, dass die Dämpfung kein Ersatz für Reibung ist; Die zwei Effekte können und sollten zusammen verwendet werden. Hier ist ein Bild zur Veranschaulichung der Dämpfung:

Dämpfungsparameter sollten zwischen 0 und unendlich liegen, wobei 0 keine Dämpfung und unendlich die volle Dämpfung bedeutet. Normalerweise verwenden Sie einen Dämpfungswert zwischen 0 und 1, aber Sie können einen beliebigen nicht negativen Wert verwenden, falls erforderlich.


Beispiel:

physics_fixture_set_linear_damping(fix_Ball, 0.1);

Der obige Code wird die lineare Dämpfung der in "fix_ball" indizierten Vorrichtung auf 0,1 einstellen.
