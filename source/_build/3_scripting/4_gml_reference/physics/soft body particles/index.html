Weiche Körperpartikel

GameMaker Studio 2 verwendet die Box2D-Bibliothek für Festkörperphysik und erlaubt daher keine Physik des weichen Körpers. Die Liquid Fun- Erweiterung fügt dem Physiksystem jedoch Partikel hinzu, die die Erzeugung von simulierten weichen Körpern unter Verwendung der Starrkörperphysik ermöglichen. Grundsätzlich können Sie eine Form von Partikeln erzeugen, die den Zusammenhalt erhalten und so einen ganzen Körper simulieren, der durch die Wechselwirkung der Partikel den Eindruck eines weichen Körpers vermittelt. Sie können die Physikpartikel auch verwenden, um Flüssigkeiten und andere Teilchenbasierte Physiksysteme zu simulieren.

Die folgenden Abschnitte beziehen sich alle auf die Verwendung von Physikpartikeln, um Gruups (weiche Körper) und andere Effekte zu erzeugen:

Bevor Sie beginnen, Ihre Physik-Teilchen zu erstellen, müssen Sie die grundlegenden physikalischen Eigenschaften für sie festlegen, so wie Sie es in der normalen Physik tun würden. Beachten Sie jedoch, dass sich das Festlegen dieser Eigenschaften auf alle zuvor erstellten Partikel sowie auf alle neu erstellten Partikel auswirkt, sodass Sie diese Funktionen normalerweise nur zu Beginn des Spiels oder Levels verwenden würden. Es stehen jedoch bestimmte Funktionen zur Verfügung, um die Partikel- Flags für einzelne Partikel (und Partikelgruppen) zu setzen, die ihre Eigenschaften wie "Bounciness" oder "Viscosity" ändern, so dass Sie verschiedene Partikelarten innerhalb der Bereiche dieser globalen Einstellungen erstellen können.

Folgende Funktionen stehen zum Einstellen und Abrufen globaler Partikelsystemdaten zur Verfügung:

  • physics_particle_count
  • physics_particle_group_count
  • physics_particle_set_max_count
  • physics_particle_set_radius
  • physics_particle_set_density
  • physics_particle_set_damping
  • physics_particle_set_gravity_scale
  • physics_particle_get_max_count
  • physics_particle_get_radius
  • physics_particle_get_density
  • physics_particle_get_damping
  • physics_particle_get_gravity_scale

Sie haben auch die folgenden Funktionen, um Flags für einzelne Partikel (oder Gruppen von Partikeln) zu setzen sowie Informationen wie Position oder Farbe von einem einzelnen Partikel oder einer einzelnen Gruppe zu erhalten:

  • physics_particle_set_flags
  • physics_particle_set_category_flags
  • physics_particle_set_group_flags
  • physics_particle_get_group_flags
  • physics_particle_group_get_inertia
  • physics_particle_group_get_centre_x
  • physics_particle_group_get_centre_y
  • physics_particle_group_get_vel_x
  • physics_particle_group_get_vel_y
  • physics_particle_group_get_ang_vel
  • physics_particle_group_get_x
  • physics_particle_group_get_y
  • physics_particle_group_get_angle
  • physics_particle_get_data
  • physics_particle_get_data_particle
  • physics_particle_group_get_data

Sie können Partikel auf zwei Arten erstellen - einzeln oder in Gruppen. Einzelne Partikel verhalten sich genau so, wie sie in den globalen Einstellungen und den Flags definiert sind, die bei ihrer Erstellung festgelegt wurden. Gruppen von Partikeln können jedoch abhängig von ihren Gruppenflags unterschiedliche Verhaltensweisen aufweisen. Diese Gruppenflags können auf unterschiedliche Werte gesetzt werden, um der Gruppe verschiedene Eigenschaften wie Durchdringung und Steifigkeit zu geben.

Die folgenden Funktionen sind zum Erstellen von Partikeln verfügbar:

  • physics_particle_create
  • physics_particle_group_begin
  • physics_particle_group_circle
  • physics_particle_group_box
  • physics_particle_group_polygon
  • physics_particle_group_add_point
  • physics_particle_group_end
  • physics_particle_group_join
  • physics_particle_group_delete

Sie können Partikel nicht nur erstellen, sondern auch einzeln oder in einer Region löschen. Folgende Funktionen stehen zum Löschen von Partikeln aus dem System zur Verfügung:

  • physics_particle_delete
  • physics_particle_delete_region_circle
  • physics_particle_delete_region_box
  • physics_particle_delete_region_poly

Es sei denn, Sie haben die Debug-Zeichnung mit aktiviert physics_draw_debug(), sehen Sie keine Partikel auf dem Bildschirm, wenn Sie sie erstellen. Um die Partikel tatsächlich zu sehen, müssen Sie sie im Draw-Event einer Instanz mit den folgenden Funktionen auf den Bildschirm zeichnen (beachten Sie, dass Sie sie auch mit dem regulären zeichnen können) draw_sprite Funktionen, wenn Sie die Daten für jedes Partikel mithilfe der Funktion erhalten physics_particle_get_data ):

  • physics_particle_draw
  • physics_particle_draw_ext
