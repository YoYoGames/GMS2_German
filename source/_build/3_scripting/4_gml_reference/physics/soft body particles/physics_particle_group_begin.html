physics_particle_group_begin

Beginnen Sie die Erstellung einer weichen Körperpartikelgruppe mit den gegebenen Parametern.

Syntax:

physics_particle_group_begin(flags, groupflags, x, y, ang, xv, yv, ang_velocity, col, alpha, strength, category)


Streit Beschreibung
flags Die Flags, die auf das Partikel gesetzt werden sollen.
groupflags Die Gruppen-Flags, die auf das Partikel gesetzt werden sollen (siehe Beschreibung unten).
x Die x-Position, um die Partikelgruppe zu erstellen.
y Die Y-Position zum Erstellen der Partikelgruppe bei.
ang Der Winkel (in Grad) zum Drehen der Gruppe wird erstellt.
xv Die anfängliche horizontale Geschwindigkeit.
yv Die anfängliche vertikale Geschwindigkeit.
ang_velocity Die anfängliche Winkelgeschwindigkeit.
col Die Grundfarbe für die Partikelgruppe.
alpha Das Basis-Alpha für die Partikelgruppe.
strength Stellen Sie die Kohäsionsstärke zwischen den Partikeln in der Gruppe ein.
category Die benutzerdefinierte Kategorie, zu der die Gruppe gehört.


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie eine Gruppe von Partikeln in einem Raum erstellen. Eine Gruppe von Partikeln ist eine, in der die Partikel, die für die Erzeugung einer bestimmten Form erforderlich sind, gleichzeitig als Gruppe erstellt werden, so dass Sie simulierte weiche Körper mit verschiedenen Eigenschaften erstellen können, die durch die verwendeten Flags festgelegt werden. Die "Flags" sind der Rückgabewert einer Kombination der Partikelkonstanten (siehe Funktion) physics_particle_create für Details zu diesen), während die "groupFlags" eine Kombination der folgenden Gruppenkonstanten sind (oder 0 für keine Flags):

Konstante Beschreibung
phy_particle_group_flag_solid Eine feste Partikelgruppe verhindert, dass sich andere Vorrichtungen darin aufhalten. Dringt etwas in sie ein, so drückt die feste Partikelgruppe die angreifende Vorrichtung zurück auf ihre Oberfläche, wodurch eine feste Partikelgruppe eine besonders starke Abstoßungskraft besitzt.
phy_particle_group_flag_rigid Starre Partikelgruppen sind solche, deren Form sich nicht ändert, selbst wenn sie mit anderen Vorrichtungen kollidieren.



Diese Gruppenflags verwenden die Bit-Maskierung, um einen endgültigen Ausgabewert zu erstellen, der dann geprüft wird, um die verschiedenen grundlegenden Eigenschaften der Gruppe außer denen, die durch die Partikelflags festgelegt sind, festzulegen. Sie können keine, eine oder beide dieser Konstanten verwenden, indem Sie das bitweise oder "|" um die entsprechenden Bits zu maskieren (wie im folgenden Beispiel gezeigt).

Außer den Flags können Sie auch die Position im Raum festlegen, um die Partikelgruppe bei (die angegebene Position ist der Massenschwerpunkt der fertigen Gruppe), die anfängliche horizontale und vertikale Geschwindigkeit sowie die Winkelgeschwindigkeit von die Gruppe, mit der Sie bewegliche, rotierende Gruppen erstellen können.

Sie können auch die Anfangsfarbe und das Alpha aller Partikel in der Gruppe festlegen, wobei die Werte beim Zeichnen der Partikel mithilfe der Funktion verwendet werden physics_particle_draw(). Danach setzt man die Kohäsionskraft der Teilchen in die Gruppe, was eine andere Art zu sagen ist, wie stark sie aneinander haften. Die Stärke kann ein Wert zwischen 0 und 1 sein, wobei 0 keine Kohäsion ist und 1 maximale Kohäsion ist.

Schließlich müssen Sie die Kategorie für Ihre Partikelgruppe angeben. Dieser Wert ist ein beliebiger ganzzahliger Wert, den Sie jeder Partikelgruppe, die ähnliche Eigenschaften haben soll, geben, und sie können verwendet werden, um später Flags für diese bestimmte Kategorie sowie eine Reihe anderer Dinge zu setzen.

HINWEIS: Sie können einen beliebigen ganzzahligen Wert für die Kategorie mit Ausnahme von 0 verwenden, der von der Simulation reserviert ist. Mit dieser Option können Sie alle Kategorien in anderen Funktionen auswählen (z. B. zum Zeichnen).

Es ist wichtig zu beachten, dass das Aufrufen dieser Funktion nicht die Gruppe im Physikraum erzeugt. Diese Funktion startet einfach den Prozess und legt die Basispartikeleigenschaften für die Gruppe fest. Um die Partikelgruppe tatsächlich zu erstellen, müssen Sie ihr ein Formular geben (siehe Beispiel unten) und dann die Funktion aufrufen physics_particle_group_end() An diesem Punkt wird es erstellt.

HINWEIS: Die von Ihnen festgelegten Partikelarten-Flags beeinflussen auch die Kohäsion der Partikelgruppe. Wenn Sie die Partikel zum Beispiel als Wasserart markieren, bricht die Gruppe beim Zusammenstoß mit einer anderen Fixture auseinander Wenn Sie das Partikel als Federtyp kennzeichnen, behält es seine Kohäsion bei einer Kollision bei.


Beispiel:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(100);
mLastGroup = physics_particle_group_end();

Der obige Code speichert die Flags für den Partikeltyp und die Partikelgruppeneigenschaften in Variablen und erstellt daraus eine kreisförmige Partikelgruppe mit einem Radius von 100 px an der Mausposition.
