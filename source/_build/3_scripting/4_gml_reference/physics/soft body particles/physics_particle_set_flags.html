physics_particle_set_flags

Legen Sie die Partikelart-Flags für ein bestimmtes Partikel fest.

Syntax:

physics_particle_set_flags(index, flags)


Streit Beschreibung
index Der Index des Partikels.
flags Die Flags, die auf das Partikel gesetzt werden sollen.


Rückgabe: Echt


Beschreibung

Mit dieser Funktion können Sie die Partikelflags für ein einzelnes Partikel ändern. Das index Wert ist die Partikel-ID, die von der Funktion zurückgegeben wird physics_particle_create, während die Flags der Rückgabewert einer Kombination der folgenden Konstanten sind:

Konstante Beschreibung
phy_particle_flag_water Die Standardeigenschaften für ein weiches Körperpartikel.
phy_particle_flag_zombie Ein Zombie-Partikel wird in einem einzigen Schritt zerstört, während alle anderen auf diese Weise markiert werden.
phy_particle_flag_wall Dies definiert das Teilchen als statisch und erzeugt es im Wesentlichen als ein unbewegliches Objekt in der physikalischen Simulation, da es in einer festen Position bleibt, egal was mit ihnen kollidiert. Sie sollten dieses Flag verwenden, anstatt die Dichte auf 0 zu setzen.
phy_particle_flag_spring Federpartikel erzeugen den Effekt, dass sie aneinander haften, wie durch eine Feder. Partikel, die mit diesem Flag erstellt wurden, sind paarweise "verbunden", wobei jedes Partikel mit demjenigen verbunden ist, das ihm zum Zeitpunkt der Erstellung am nächsten war. Sobald sie gepaart sind, ändern die Teilchen die "Partner" nicht mehr, und je weiter eine äußere Kraft sie voneinander wegzieht, desto größer ist die Kraft, mit der sie kollidieren werden, wenn diese äußere Kraft entfernt wird. Beachten Sie, dass unabhängig davon, wie weit die gepaarten Partikel voneinander entfernt sind, die Verbindung zwischen ihnen nicht unterbrochen wird.
phy_particle_flag_elastic Elastische Partikel verformen sich und können auch prallen, wenn sie in der Physiksimulation mit anderen starren Körpern kollidieren.
phy_particle_flag_viscous Ein viskoses Teilchen ist eines, das "Anhaftung" oder "Klebrigkeit" wie Öl aufweist. Viskose Partikel verklumpen und haften mehr zusammen.
phy_particle_flag_powder Pulverpartikel erzeugen einen Streueffekt, wie er bei Sand oder Staub auftreten kann.
phy_particle_flag_tensile Zugpartikel werden verwendet, um den Effekt der Oberflächenspannung oder der straffen Krümmung auf der Oberfläche eines Flüssigkeitskörpers zu erzeugen. Sie könnten zum Beispiel verwendet werden, um die Oberflächenspannung zu erzeugen, die Sie auf einem Wassertropfen sehen würden. Sobald die Spannung gebrochen ist, prallen die Partikel auf, als wären sie elastisch, ziehen sich aber auch weiterhin an. Infolgedessen neigen Partikel dazu, Cluster zu bilden, wenn sie aufprallen.
phy_particle_flag_colourmixing Farbmischende Partikel nehmen einen Teil der Farbe anderer Partikel an, mit denen sie kollidieren. Beachten Sie, dass, wenn nur eines der beiden kollidierenden Partikel farbmischend ist, das andere Partikel seine Farbe vor der Kollision behält.



Diese Flags verwenden Bitmaskierung, um einen endgültigen Ausgabewert zu erstellen, der dann überprüft wird, um die verschiedenen grundlegenden Eigenschaften des Partikels festzulegen (wobei die Basiseigenschaft immer die von phy_particle_flag_water ). Wenn Sie z. B. eine viskose Flüssigkeit mit Oberflächenspannung simulieren möchten, verwenden Sie die bitweise oder "|" um die entsprechenden Bits wie im folgenden Beispiel zu maskieren. Auf diese Weise können Sie für jedes einzelne erstellte Partikel unterschiedliche Eigenschaften (außer den globalen Eigenschaften) festlegen.


Beispiel:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
physics_particle_set_flags(p, flags);

Der obige Code erstellt eine Variable zum Speichern des Flags-Werts und verwendet diese dann, um die Flags für ein zuvor erstelltes Partikel mit dem in der Variablen "p" gespeicherten Index festzulegen.
