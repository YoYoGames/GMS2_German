physics_particle_group_get_data

Erhalte Daten zu einer Gruppe von Partikeln in deiner Physiksimulation.

Syntax:

physics_particle_group_get_data(group, buffer, flags)


Streit Beschreibung
group Der Gruppenindex (ID) der Partikelgruppe, von der die Daten abgerufen werden sollen.
buffer Der (zuvor erstellte) Puffer, der zum Speichern der Daten verwendet werden soll.
flags Die Flags, die zum Extrahieren von Daten zu bestimmten Partikeltypen verwendet werden.


Rückgabe: Pufferdaten


Beschreibung

Diese Funktion gibt verschiedene Informationen über eine Partikelgruppe in der Physiksimulation unter Verwendung der angegebenen Flags zurück. Der Gruppenindex (seine ID) ist derjenige, der von der Funktion zurückgegeben wurde physics_particle_group_end() und der verwendete Puffer muss zuvor mit der Funktion erstellt worden sein buffer_create(). Es sollte vom Typ "grow" sein, wobei die Größe ungefähr der der erwarteten Rückgabedaten entspricht. Die Flags selbst werden mit den unten angegebenen Konstanten gesetzt, und Sie würden das bitweise oder "|" um einen einzelnen Flag-Wert zu erzeugen, um die gewünschte Information zu erhalten.

Konstante Beschreibung Datentyp
phy_particle_data_flag_typeflags Der Flag-Wert für das Partikel. buffer_u32
phy_particle_data_flag_position Die x- und y-Position des Partikels. 2 x buffer_f32
phy_particle_data_flag_velocity Die horizontale und vertikale Geschwindigkeit. 2 x buffer_f32
phy_particle_data_flag_colour Die Farbe und der Alpha-Wert (hexadezimal). buffer_f32
phy_particle_data_flag_category Die Partikelkategorie (wie bei der Erstellung der Gruppe definiert, zu der sie gehört). buffer_f32




Beispiel:

var count = physics_particle_group_count(gp);
var flags = phy_particle_data_flag_position | phy_particle_data_flag_colour;
if (count > 0)
   {
   var buffer = buffer_create(count * 12, buffer_grow, 4);
   physics_particle_group_get_data(gp, buffer, flags);
   for (var n = 0; n < count; n++;)
      {
      var xx = buffer_read(buffer, buffer_f32);
      var yy = buffer_read(buffer, buffer_f32);
      var argb = buffer_read(buffer, buffer_u32);
      var alpha = (argb >> 24) & 255;
      draw_sprite_ext(sprBlob, 0, xx, yy, 1, 1, 0, c_green, alpha);
      }
   buffer_delete(buffer);
   }

Der obige Code erhält die Position und Geschwindigkeit jedes Teilchens in der Gruppe, die durch die Variable "gp" indiziert wird, speichert die Pufferdaten in einer Anzahl von Variablen und verwendet diese, um ein Sprite an der Position jedes Teilchens in der Gruppe zu zeichnen.
