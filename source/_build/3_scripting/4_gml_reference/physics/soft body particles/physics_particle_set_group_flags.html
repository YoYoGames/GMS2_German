physics_particle_set_group_flags

Legen Sie die Gruppentyp-Flags für eine Gruppe von Partikeln fest.

Syntax:

physics_particle_set_group_flags(group, flags)


Streit Beschreibung
group Die einzustellende Partikelgruppe.
flags Die Flags, die auf das Partikel gesetzt werden sollen.


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie die Gruppenflags für eine Gruppe von Partikeln ändern. Das group Wert ist der Wert, der beim Erstellen der Partikelgruppe mit der Funktion zurückgegeben wurde physics_particle_group_end, während die Flags der Rückgabewert einer Kombination der folgenden Konstanten sind:

Konstante Beschreibung
phy_particle_group_flag_solid Eine feste Partikelgruppe verhindert, dass sich andere Vorrichtungen darin aufhalten. Dringt etwas in sie ein, so drückt die feste Partikelgruppe die angreifende Vorrichtung zurück auf ihre Oberfläche, wodurch eine feste Partikelgruppe eine besonders starke Abstoßungskraft besitzt.
phy_particle_group_flag_rigid Starre Partikelgruppen sind solche, deren Form sich nicht ändert, selbst wenn sie mit anderen Vorrichtungen kollidieren.



Diese Gruppenflags verwenden die Bit-Maskierung, um einen endgültigen Ausgabewert zu erstellen, der dann geprüft wird, um die verschiedenen grundlegenden Eigenschaften der Gruppe außer denen, die durch die Partikelflags festgelegt sind, festzulegen. Sie können keine, eine oder beide dieser Konstanten verwenden, indem Sie das bitweise oder "|" um die entsprechenden Bits zu maskieren (wie im folgenden Beispiel gezeigt).


Beispiel:

var flags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid;
physics_particle_set_group_flags(group, flags);

Der obige Code erstellt eine Variable, um den Flags-Wert zu speichern und verwendet sie dann, um die Flags aller Partikel festzulegen, die in der Gruppe erstellt wurden, die durch die Variable "group" indiziert wird.
