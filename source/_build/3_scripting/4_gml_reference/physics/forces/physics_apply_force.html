physics_apply_force

Diese Funktion wendet eine Kraft auf die aktuelle Instanz an einer Position im Raum mit einer durch einen Vektor definierten Stärke an.

Syntax:

physics_apply_force(xpos, ypos, xforce, yforce)


Streit Beschreibung
xpos Die x-Koordinate in dem Raum, in dem die Kraft angewendet wird
ypos Die y-Koordinate in dem Raum, in dem die Kraft angewendet wird
xforce die x-Komponente des Kraftvektors
yforce die y-Komponente des Kraftvektors


Rückkehr: N / A


Beschreibung

Wenn man mit Physik arbeitet, muss man nicht nur die Schwerkraft auf Objekte anwenden, sondern auch Kräfte. Eine Kraft ist jeder Einfluss, der bewirkt, dass ein Objekt eine Änderung der Geschwindigkeit, eine Richtungsänderung oder eine Formänderung erfährt und in der Physik von GameMaker Studio 2 genauso definiert wird, wie wir die Schwerkraft definieren, indem wir den Vektor aus der zwei xforce / yforce-Komponenten. Es ist anzumerken, dass die Kräfte nicht auf den Massenschwerpunkt des Objekts, sondern auf einen Punkt im Raum wirken und dass sie die Richtung und Geschwindigkeit sowie das Drehmoment (Spin) in Abhängigkeit von anderen wirkenden Kräften ändern auf dem Objekt (wie die Schwerkraft). Hier ist ein Beispielbild zur Veranschaulichung:

Im obigen Beispiel hat das Federobjekt am Boden eine Kraft auf das Kugelobjekt ausgeübt, die es in eine andere Richtung bewegen lässt. Die genaue Kraft wird durch den Vektor definiert, den wir aus den Komponenten xforce / yforce in Bezug auf die xpos / ypos-Koordinaten erhalten - was bedeutet, dass die Kraft als die Entfernung von xpos / ypos zu xforce / yforce in Newton berechnet wird (ein Newton ist) definiert als die Menge an Nettokraft, die benötigt wird, um eine Masse von einem Kilogramm mit einer Geschwindigkeit von einem Meter pro Quadratsekunde zu beschleunigen) und die Richtung ist der Winkel, den wir von xpos / ypos zu xforce / yforce erhalten. Der Code für eine Kollision wie oben würde in das Kollisionsereignis des Federobjekts gehen und würde ungefähr so ​​aussehen wie im folgenden Beispiel.

HINWEIS: Die Physikwelt verwendet für alle Berechnungen Zähler, und das Verhältnis der Pixel pro Meter wird im Raumeditor oder unter Verwendung definiert physics_world_create.


Beispiel:

with (other)
   {
   physics_apply_force(x, y, 0, -30);
   }

Der obige Code wendet Code auf das andere Objekt in der Kollision mit einer Kraft von 30 Newton und einer Richtung an.
