steam _get_user_persona_name

Syntax:

steam_get_user_persona_name(steamID);


Streit Beschreibung
steam ID Die eindeutige Steam ID für einen Benutzer.


Rückgabe: Echt


Beschreibung

Diese Funktion kann verwendet werden, um den Benutzernamen (Bildschirmname) für eine bestimmte Benutzer-ID abzurufen. Wenn Sie die Benutzer-ID angeben, gibt die Funktion einen eindeutigen Async-ID-Wert zurück, mit dem die Details im Steam Async-Ereignis korrekt überprüft werden können. Diese Veranstaltung wird die async_load DS-Karte, die mit den folgenden Schlüssel / Wert-Paaren gefüllt wird:

  1. " id " - Die asynchrone ID, die von der Funktion zurückgegeben wurde, die das Ereignis ausgelöst hat

  3. " event_type " - Die Zeichenfolge " user_persona_name "

  5. " steam id" "- Der Steam ID-Wert, mit dem der Benutzername abgerufen wird.

  7. " persona_name " - Der Benutzername für die angegebene Steam ID


Beispiel:

request = steam_get_user_persona_name(global.UGC_UserID);

Der obige Code fordert den Benutzernamen der Benutzer-ID an, die in der globalen Variablen "UGC_UserID" gespeichert ist, und speichert den zurückgegebenen Wert in einer Variablen für das Parsing im Async-Ereignis.
