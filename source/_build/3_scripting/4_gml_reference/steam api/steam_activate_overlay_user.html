steam _activate_overlay_user

Syntax:

steam_activate_overlay_user(dialog_name, steamid);


Streit Beschreibung
dialog_name Der Dialog zum Öffnen des Overlays (siehe unten).
steam id Die zu verwendende Steam Benutzer-ID oder Gruppen-ID.


Rückkehr: N / A


Beschreibung

Diese Funktion öffnet das Steam Overlay für einen der ausgewählten Dialoge, die sich auf die angegebene Benutzer-ID beziehen. Die möglichen verfügbaren Dialogstrings sind:

  • " steam id " - öffnet den Steam Community Webbrowser auf der Seite des Benutzers oder der Gruppe
  • " Chat " - öffnet ein Chat-Fenster für den angegebenen Benutzer


Beispiel:

var key = keyboard_lastkey;
switch (key)
   {
   case vk_f1: steam_activate_overlay_user("steamid", global.GameGroupID); break;
   case vk_f2: steam_activate_overlay_user("chat", global.FriendID); break;
   }

Der obige Code fragt die zuletzt gedrückte Taste ab und wenn es die Funktionstaste 1 oder die Funktionstaste 2 ist, wird das Steam Overlay geöffnet, um entweder die in der globalen Variable "GamegroupID" gespeicherte Steam Gruppe zu sehen oder es öffnet sich das Chat-Fenster chat mit dem Benutzer, der in der globalen Variablen "FriendID" gespeichert ist.
