steam _get_quota_total

Syntax:

steam_get_quota_total();


Rückgabe: Echt


Beschreibung

Bei der Verwendung von Steam - Cloud - Dateien zu speichern und zu synchronisieren, müssen Sie die Quote des Raumes, der Ihr Spiel brauchen einrichten. Dieses Kontingent wird für jedes Cloud-fähige Spiel pro Nutzer und pro Spiel erzwungen. Wenn beispielsweise das Kontingent für Spiel X 1 Megabyte beträgt, kann jeder Steam Account, der Game X besitzt, höchstens speichern, 1 Megabyte Daten zu diesem Spiel in der Cloud. Alle anderen Cloud-fähigen Spiele, die der Benutzer besitzt (z. B. Spiel Y), sind von den von Game X gespeicherten Daten nicht betroffen. Das Standardkontingent für neue Steam Works-Spiele beträgt ein Gigabyte. Sie können dies jedoch über das Steam Werksmenü ändern Panel für Ihr Spiel.

HINWEIS: Sobald das Kontingent erschöpft ist, schlagen Dateischreibvorgänge fehl. Wenn Sie denken, dass das Kontingent möglicherweise für den Benutzer Ihres Spiels erschöpft ist, sollten Sie Code erstellen, um damit umzugehen, da Sie nichts tun, indem Sie Benutzer in einer Situation lassen, in der sie die Dinge nicht beheben können eine schlechte Erfahrung deines Spiels.


Beispiel:

if steam_is_cloud_enabled_for_app()
   {
   quota = steam_get_quota_total();
   }

Der obige Code überprüft, ob die steam Cloud für das Spiel aktiviert ist. Wenn dies der Fall ist, wird die Größe des Speicherkontingents ermittelt und in einer Variablen gespeichert.
