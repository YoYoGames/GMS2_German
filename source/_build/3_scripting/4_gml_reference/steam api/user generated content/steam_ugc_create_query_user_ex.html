steam _ugc_create_query_user_ex

Syntax:

steam_ugc_create_query_user_ex(list_type, match_type, sort_order, page, account_id, creator_app_id, consumer_app_id);


Streit Beschreibung
list_type Der Typ der zu erstellenden Datenliste (siehe die unten aufgeführten Konstanten).
match_type Der Typ der abzufragenden UGC-Elemente (siehe die unten aufgeführten Konstanten).
sort_order Die Art und Weise, wie die Daten geordnet werden sollen (siehe die unten aufgeführten Konstanten).
page Die Seitenzahl, die abgefragt werden soll.
account_id Die Steam Konto-ID.
creator_app_id Die ID der Artikelersteller-App
consumer_app_id Die Nutzer-App-ID.


Rückgabe: Abfrage Handle


Beschreibung

Diese Funktion kann verwendet werden, um die UGC-Datenbank abzufragen. Die Funktion benötigt den ID-Wert für den Benutzer und die ID des Spiels, das den Gegenstand und / oder die ID des Spiels, das den Gegenstand geschaffen hat, verbrauchen wird. Außerdem müssen Sie die folgenden Konstanten für den abzufragenden Datentyp, den abzufragenden Elementtyp und die Reihenfolge, in der die zurückgegebenen Elemente sortiert werden, sowie eine Seitenzahl verwenden. Beachten Sie, dass eine Abfrage eine maximale Anzahl zurückgibt von 50 Artikeln.

Die Funktion gibt einen eindeutigen Abfrage-Handle-Wert zurück, der in einer Variablen zur Verwendung in den anderen Abfragefunktionen gespeichert werden sollte. Beachten Sie, dass diese Funktion nur die Abfrage vorbereitet, aber nicht sendet - dafür müssen Sie die Funktion aufrufen steam_ugc_send_query - und Sie können weiter verwenden steam_ugc_query_*() Funktionen, um die Suchanfrage zu verfeinern, bevor sie tatsächlich gesendet wird.


Beispiel:

query_handle = steam_ugc_create_query_user_ex(ugc_list_Published, ugc_match_Items, ugc_sortorder_TitleAsc, page, global.AccountID, 0, global.GameID);

Der obige Code erstellt eine Abfrageanforderung und speichert sie in einer Variablen für die zukünftige Verwendung.
