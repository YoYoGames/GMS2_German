steam _ugc_download

Syntax:

steam_ugc_download(ugc_handle, dest_filename);


Streit Beschreibung
ugc_handle Das eindeutige Handle für die Vorschau, die heruntergeladen werden soll.
dest_filename Der Dateiname, mit dem die Vorschau gespeichert werden soll.


Rückgabe : Async-ID


Beschreibung

Mit dieser Funktion können Sie ein Vorschaubild für ein bestimmtes UGC-Objekt herunterladen. Das ugc_handle ist der eindeutige Identifizierungswert für das Bild (den Sie mit der Funktion erhalten können) steam_ugc_send_query() ), und der Zieldateiname ist der Name (und der lokale Pfad in der Steam Sandbox), den Sie der Bilddatei geben möchten, wenn der Download abgeschlossen ist.

Wenn Sie diese Funktion verwenden, wird ein Steam Asynchronous- Ereignis ausgelöst, um die Details der angeforderten Image-Datei zu melden, die die folgenden Schlüssel / Wert-Paare enthält async_load ds_map:

  1. " id " - Die von der aufrufenden Funktion zurückgegebene asynchrone ID

  3. " Ergebnis " - Das Ergebnis der Operation (ein realer Wert). Dies ist entweder die GML-Konstante ugc_result_success oder eine andere reelle Zahl. Sie sollten also nach dieser Konstante suchen, um sicherzustellen, dass der Aufruf erfolgreich war, und wenn sonst etwas nicht korrekt funktioniert hat. Der Rest der möglichen Werte wird als Ergebnis des Steam "EResult" -Werts angezeigt und Sie sollten sehen steamclientpublic.h in den SDK-Headern für alle 89 möglichen Werte.

  5. " event_type " - Dieser Schlüssel enthält den Wert " ugc_download "

  7. " original_filename " - Dieser Schlüssel enthält den ursprünglichen Namen der Bilddatei auf dem Server (eine Zeichenkette)

  9. " dest_filename " - Dieser Schlüssel enthält den Bilddateinamen, den Sie übergeben haben (eine Zeichenkette)

  11. " ugc_handle " - Diese Taste hält die ugc_handle Wert, den Sie an die aufrufende Funktion übergeben haben


Erweitertes Beispiel:

In diesem Beispiel rufen wir zuerst die Funktion auf und speichern den asynchronen ID-Wert in einer Variablen:

steam_get = steam_ugc_download(steam_handle, "\UGC\Preview_file.png");

Dies würde dann eine Dateianforderung an die Steam API senden, die ein asynchrones Ereignis erzeugt, mit dem wir uns wie folgt befassen würden:

var event_id = async_load[? "id"];
if event_id == steam_get
   {
   var type = async_load[? "event_type"];
   if type == "ugc_download"
      {
      sprite_delete(preview_sprite);       preview_sprite = sprite_add(async_load[? "dest_filename"], 0, false, false, 0, 0);
      }
   }

Der obige Code überprüft den Ereignistyp und erstellt dann ein Sprite aus dem heruntergeladenen Bild.
