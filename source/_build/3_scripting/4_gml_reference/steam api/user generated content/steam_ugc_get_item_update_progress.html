steam _ugc_get_item_update_progress

Syntax:

steam_ugc_get_item_update_progress(ugc_update_handle, info_map);


Streit Beschreibung
ugc_update_handle Der eindeutige Handle für den UGC, der aktualisiert werden soll.
info_map Ein (zuvor erstellter) DS-Kartenindex.


Rückgabe: Boolean


Beschreibung

Diese Funktion kann verwendet werden, um den Aktualisierungsstatus eines Artikels zu verfolgen. Sie geben den Item-Handle (wie von der Funktion zurückgegeben) steam_ugc_start_item_update ) und eine leere DS-Karte, die dann mit den folgenden Schlüssel / Wert-Paaren gefüllt wird:

  1. " status_code " - Der Steam Statuscode

  3. " status_string " - Eine Zeichenfolge für den aktuellen Status

  5. " bytes_processed " - Die bisher verarbeiteten Bytes

  7. " Bytes_total " - Die Gesamtzahl der Bytes im Update

Wenn ein Fehler auftritt, kehrt die Funktion zurück false und die Karte ist leer, sonst kehrt die Funktion zurück true.


Beispiel:

var uploadMap = ds_map_create();
steam_ugc_get_item_update_progress(global.itemHandle, uploadMap);
var statusCode = uploadMap[? "status_code"];
var status = uploadMap[? "status_string"];
var processed = uploadMap[? "bytes_processed"];
var total = uploadMap[? "bytes_total"];
draw_text(32, 0, "Upload info for item:" + string(global.itemHandle));
draw_text(32, 15, "status code:" + string(statusCode));
draw_text(32, 30, "status:" + string(status));
draw_text(32, 45, "bytes processed:" +string(processed));
draw_text(32, 60, "bytes total:" + string( total));
ds_map_destroy(uploadMap);

Der obige Code fragt den Upload-Status des in der globalen Variable "itemHandle" indizierten Objekts ab ds_map um die Informationen zu speichern. Dies wird dann analysiert und die resultierenden Werte auf den Bildschirm gezeichnet.
