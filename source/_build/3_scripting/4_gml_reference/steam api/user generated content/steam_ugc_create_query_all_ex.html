steam _ugc_create_query_all_ex

Syntax:

steam_ugc_create_query_all_ex(query_type, match_type, page, creator_app_id, consumer_app_id);


Streit Beschreibung
query_type Der Typ der zu erstellenden Datenabfrage (siehe die unten aufgeführten Konstanten).
match_type Der Typ der abzufragenden UGC-Elemente (siehe die unten aufgeführten Konstanten).
page Die Seitenzahl, die abgefragt werden soll.
creator_app_id Die ID der Artikelersteller-App
consumer_app_id Die Nutzer-App-ID.


Rückgabe: Abfrage Handle


Beschreibung

Diese Funktion kann verwendet werden, um die UGC-Datenbank abzufragen. Die Funktion benötigt die ID des Spiels, das das Element und / oder die ID des Spiels, das das Element erstellt hat, konsumiert, und Sie müssen die folgenden Konstanten für den zu erstellenden Abfragetyp verwenden, den Typ des abzufragenden Elements und die Seitenzahl, die abgefragt werden soll. Beachten Sie, dass eine Abfrage eine maximale Anzahl von 50 Elementen zurückgibt.

Abfragetyp Konstante Beschreibung
ugc_query_RankedByVote Siehe die Steam Dokumentation.
ugc_query_RankedByPublicationDate -
ugc_query_AcceptedForGameRankedByAcceptanceDate -
ugc_query_RankedByTrend -
ugc_query_FavoritedByFriendsRankedByPublicationDate -
ugc_query_CreatedByFriendsRankedByPublicationDate -
ugc_query_RankedByNumTimesReported -
ugc_query_CreatedByFollowedUsersRankedByPublicationDate -
ugc_query_NotYetRated -
ugc_query_RankedByTotalVotesAsc -
ugc_query_RankedByVotesUp -
ugc_query_RankedByTextSearch -
Übereinstimmungstyp konstant Beschreibung
ugc_match_Items Siehe die Steam Dokumentation.
ugc_match_Items_Mtx -
ugc_match_Items_ReadyToUse -
ugc_match_Collections -
ugc_match_Artwork -
ugc_match_Videos -
ugc_match_Screenshots -
ugc_match_AllGuides -

ugc_match_IntegratedGuides -
ugc_match_UsableInGame -
ugc_match_ControllerBindings -



Die Funktion gibt einen eindeutigen Abfrage-Handle-Wert zurück, der in einer Variablen zur Verwendung in den anderen Abfragefunktionen gespeichert werden sollte. Beachten Sie, dass diese Funktion nur die Abfrage vorbereitet, aber nicht sendet - dafür müssen Sie die Funktion aufrufen steam_ugc_send_query - und Sie können weiter verwenden steam_ugc_query_*() Funktionen, um die Suchanfrage zu verfeinern, bevor sie tatsächlich gesendet wird.


Beispiel:

query_handle = steam_ugc_create_query_all_ex(ugc_query_RankedByVote, page, global.AccountID, 0, global.GameID);

Der obige Code erstellt eine Abfrageanforderung und speichert sie in einer Variablen für die zukünftige Verwendung.
