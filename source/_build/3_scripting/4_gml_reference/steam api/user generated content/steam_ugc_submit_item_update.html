steam _ugc_submit_item_update

Syntax:

steam_ugc_submit_item_update(ugc_update_handle, change_note);


Streit Beschreibung
ugc_update_handle Der eindeutige Handle für den UGC, der aktualisiert werden soll.
change_note Die Änderungshinweise (eine Zeichenfolge), die für den Artikel verwendet werden sollen.


Rückgabe : Async-ID


Beschreibung

Diese Funktion wird den UGC-Eintrag, der von dem angegebenen Handle nicht angegeben wurde, an die Steam Workshop-Server senden und die Änderungshinweise hinzufügen, die für den angegebenen Eintrag verwendet werden sollen. Die Änderungshinweise sollten eine Zeichenfolge sein und das Update-Handle ist der Wert, der beim Aufruf der Funktion zurückgegeben wurde steam_ugc_start_item_update. Bei Verwendung dieser Funktion wird ein Async-ID-Wert zurückgegeben, der dann analysiert werden kann, wenn das Steam Asynchronous- Ereignis ausgelöst wird, um die Erstellung des Elements zu melden. Das Ereignis enthält die folgenden Schlüssel / Map-Werte in der async_load ds_map:

  1. " id " - Die von der aufrufenden Funktion zurückgegebene asynchrone ID

  3. " Ergebnis " - Das Ergebnis der Operation (ein realer Wert). Dies ist entweder die GML-Konstante ugc_result_success oder eine andere reelle Zahl. Sie sollten also nach dieser Konstante suchen, um sicherzustellen, dass der Aufruf erfolgreich war, und wenn sonst etwas nicht korrekt funktioniert hat. Der Rest der möglichen Werte wird als Ergebnis des Steam "EResult" -Werts angezeigt und Sie sollten sehen steamclientpublic.h in den SDK-Headern für alle 89 möglichen Werte.

  5. " event_type " - Dieser Schlüssel enthält den Wert " ugc_update_item "

  7. " legal_agreement_required " - Wird sein true oder false (Weitere Informationen finden Sie in den Steam Dokumenten )


Beispiel:

var app_id = steam_get_app_id();
var updateHandle = steam_ugc_start_item_update(app_id, global.Publish_ID);
steam_ugc_set_item_title(updateHandle, "My workshop item(3)!");
steam_ugc_set_item_description( updateHandle, "testing workshop...");
steam_ugc_set_item_visibility(updateHandle, ugc_visibility_public);
var tagArray;
tagArray[0] = "Test";
tagArray[1] = "New";
steam_ugc_set_item_tags(updateHandle, tagArray);
steam_ugc_set_item_preview(updateHandle, "promo.jpg");
steam_ugc_set_item_content(updateHandle, "WorkshopContent1");
requestId = steam_ugc_submit_item_update(updateHandle, "Version 1.2");

Der obige Code ruft die Spiel-ID ab und verwendet diese zusammen mit einer zuvor gespeicherten veröffentlichten Datei-ID, um ein Update-Handle für das Element zu generieren. Dieser Punkt wird dann verwendet, um verschiedene Informationen zu aktualisieren, bevor das Update an die Workshop-Server gesendet wird.
