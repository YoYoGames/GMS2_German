steam _file_write

Syntax:

steam_file_write(filename, data, size);


Streit Beschreibung
filename Der Name der Datei, in die geschrieben werden soll.
data Die zu schreibenden Daten (eine Zeichenfolge).
size die Größe der zu schreibenden Daten.


Rückgabe: Echt


Beschreibung

Sie können diese Funktion verwenden, um Daten in eine Datei zu schreiben, die dann mit der Steam Cloud synchronisiert wird, wenn der Benutzer das Spiel verlässt. Wenn die Datei nicht existiert, erstellt diese Funktion sie für Sie, und wenn sie bereits existiert, überschreibt sie alle Daten, die bereits in der Datei gespeichert sind, mit der neuen Datenfolge. Die Funktion gibt einen Wert von 0 zurück, wenn es aus irgendeinem Grund fehlschlägt, und einen Wert größer als 0, wenn es erfolgreich ist.


Beispiel:

var fname="SaveData.txt";
var data = string(global.Level) + "|" + string(global.Points) + "|" + string(global.HP);
var len = string_length(data);
steam_file_write_file(fname, data, len);

Der obige Code bereitet eine Reihe von lokalen Variablen vor und verwendet sie dann, um eine Datei zu schreiben (oder zu erstellen), die dann mit der Steam Cloud synchronisiert wird.
