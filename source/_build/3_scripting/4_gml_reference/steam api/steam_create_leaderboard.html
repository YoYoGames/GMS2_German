steam _create_leaderboard

Syntax:

steam_create_leaderboard(lb_name, sort_method, display_type);


Streit Beschreibung
lb_name Der Name der Bestenliste, die Sie erstellen (eine Zeichenfolge).
sort_method Die Methode zum Sortieren der Leaderboard-Einträge (siehe die unten aufgeführten Konstanten).
display_type Die Art und Weise, wie dem Benutzer die Bestenliste angezeigt wird (siehe die unten aufgeführten Konstanten).


Rückgabe: Echt


Beschreibung

Mit dieser Funktion können Sie eine neue Rangliste für Ihr Spiel erstellen. Das erste Argument ist eine Zeichenfolge, die den Namen Ihrer Bestenliste definiert. Dieser Name sollte bei allen weiteren Funktionsaufrufen für die zu erstellende Bestenliste verwendet werden. Sie können dann die Sortiermethode (siehe die unten aufgeführten Konstanten) sowie die Art und Weise, in der die Information angezeigt wird, definieren (wiederum eine der unten aufgeführten Konstanten). Die Funktion gibt einen Wert von -1 zurück, wenn die Rangliste nicht erstellt werden kann, oder einen Wert größer als -1, wenn sie erfolgreich war.

HINWEIS: Wenn Sie zuvor eine Bestenliste mit demselben Namen erstellt haben (entweder über Code oder über Ihre Steam Seite für das Spiel), wird diese Funktion ignoriert!

Konstante sortieren Beschreibung
lb_sort_none Keine Sortierung. Die Information wird "so wie sie ist" angezeigt.
lb_sort_ascending Sortieren Sie die Bestenliste in aufsteigender Reihenfolge.
lb_sort_deszendieren Sortieren Sie die Rangliste in absteigender Reihenfolge.
Konstante anzeigen Beschreibung
lb_disp_none Zeigen Sie das Leaderboard "wie es ist".
lb_disp_numerisch Zeigen Sie die Bestenliste als numerische Anzeige an.
lb_disp_time_sec Zeigen Sie die Leaderboard-Werte als Zeiten an, wobei der Basiswert Sekunden ist.
lb_disp_time_ms Zeigen Sie die Leaderboard-Werte als Zeiten an, wobei der Basiswert Millisekunden ist.


Beispiel:

steam_create_leaderboard("Game Times", lb_sort_ascending, lb_disp_time_sec);

Der obige Code erstellt eine Rangliste namens "Spielzeiten" und legt sie so fest, dass die Ergebnisse in aufsteigender Reihenfolge und mit einer Anzeige in Sekunden angezeigt werden.
