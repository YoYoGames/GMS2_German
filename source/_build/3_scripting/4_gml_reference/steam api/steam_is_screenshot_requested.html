steam _is_screenshot_requested

Syntax:

steam_is_screenshot_requested();


Rückgabe: Boolean


Beschreibung

Diese Funktion fragt die Steam API ab, um zu sehen, ob der Schlüssel zum Erstellen eines Screenshots des Spiels gedrückt wurde. Die Funktion wird nur zurückkehren true für einen Schritt (Spiel Tick), wenn die Taste gedrückt wurde, und wird zurückkehren false zu allen anderen Zeiten.

HINWEIS: Dies nimmt keinen Screenshot für Sie. Dies signalisiert nur, dass die Taste gedrückt wurde und Sie die Funktionen von GameMaker Studio 2 müssen screen_save oder screen_save_part um eine lokale Kopie der hochzuladenden Datei zu speichern.


Beispiel:

if steam_is_screenshot_requested()
   {
   var file = "Catch_The_Haggis_" + string(global.scrn_num) + ".png");
   screen_save(file)
   steam_send_screenshot(file, window_get_width(), window_get_height());
   global.scrn_num += 1;
   }

Der obige Code fragt die Steam API nach einer Screenshot-Anfrage ab. Wenn dies der Fall ist, wird ein eindeutiger Name für die Bilddatei generiert, ein Screenshot wird erstellt und die Datei wird an die Steam Community-Seite für den Benutzer gesendet.
