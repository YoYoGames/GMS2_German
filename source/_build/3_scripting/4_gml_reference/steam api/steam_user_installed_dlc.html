steam _user_installed_dlc

Syntax:

steam_user_installed_dlc(dlc_id);


Streit Beschreibung
dlc_id Der eindeutige Bezeichner für den zu überprüfenden DLC (eine ganze Zahl).


Rückgabe: Boolean


Beschreibung

Wenn für Ihr Spiel DLC erstellt wurde, können Sie diese Funktion verwenden, um zu überprüfen, ob der Benutzer sie installiert hat, bevor auf zugehörige Dateien zugegriffen wird. Die Funktion kehrt zurück true Wenn der Player den Inhalt installiert hat und false wenn der Benutzer dies nicht tut, aber beachten Sie, dass der Benutzer auch die DLC besitzen muss, also sollten Sie die zusätzliche Funktion verwenden steam_user_owns_dlc um zu überprüfen, dass es auch vor der Verwendung gehört.


Beispiel:

global.Level_Max = 100;
if steam_user_owns_dlc(10354)
   {
   if steam_user_installed_dlc(10354)
      {
      global.Level_max = 200;
      }
   }

Der obige Code prüft, ob der Benutzer den DLC mit der ID 10354 gekauft und installiert hat, und wenn dies der Fall ist, setzt er eine globale Variable auf einen anderen Wert.
