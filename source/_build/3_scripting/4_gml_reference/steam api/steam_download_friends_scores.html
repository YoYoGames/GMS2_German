steam _download_friends_scores

Syntax:

steam_download_friends_scores(lb_name);


Streit Beschreibung
lb_name Der Name der Bestenliste, von der Sie die Noten herunterladen (eine Zeichenfolge).


Rückgabe: Echt


Beschreibung

Mit dieser Funktion können Sie nur die Punkte auf der Bestenliste abrufen, die zu den Personen gehören, die im Steam Client als "Freunde" markiert sind. Wenn also Ihre Bestenliste 200 Einträge hat und 50 davon Ihre Freunde sind, ruft diese Funktion nur diese 50 Ergebnisse ab. Der Name der Bestenliste ist eine Zeichenfolge, die beim Erstellen der Bestenliste mithilfe der Funktion definiert wurde steam_create_leaderboard, und die Funktion wird einen Wert zurückgeben, der dann verwendet werden kann, um den Rückruf im Steam Async-Ereignis zu identifizieren, oder es gibt -1 zurück, wenn es fehlgeschlagen ist.


Beispiel:

score_get = steam_download_friends_scores("Game Scores");

Dies sendet eine Anfrage an den Steam Server für die Werte der Benutzerfreunde von der gegebenen Bestenliste und speichert die asynchrone ID der Anfrage in der Variablen "score_get". Dies wird dann im Steam Async Event behandelt, wie im Extended Example für steam_download_scores.
