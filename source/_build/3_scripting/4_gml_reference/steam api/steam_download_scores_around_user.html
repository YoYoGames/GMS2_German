steam _download_scores_around_user

Syntax:

steam_download_scores_around_user(lb_name, range_start, range_end);


Streit Beschreibung
lb_name Der Name der Bestenliste, von der Sie die Noten herunterladen (eine Zeichenfolge).
range_start Die Startposition innerhalb der Bestenliste.
range_end Die Endposition innerhalb der Bestenliste.


Rückgabe: Echt


Beschreibung

Diese Funktion wird verwendet, um Ranglisteneinträge für den aktuellen Benutzereintrag abzurufen. Das range_start Parameter ist die Anzahl der Einträge, die vor dem aktuellen Benutzereintrag abgerufen werden sollen range_end Parameter ist die Anzahl der Einträge nach dem aktuellen Benutzereintrag, und der aktuelle Benutzereintrag ist immer in den Ergebnissen enthalten. Wenn der aktuelle Benutzer z. B. auf einer gegebenen Bestenliste die Nummer 5 hat, wird der Startbereich auf -2 und der Endbereich auf 2 zurückgesetzt. Es werden 5 Einträge zurückgegeben: 3 bis 7. Wenn nicht genügend Einträge in der Bestenliste vorhanden sind wird der Steam den Bereich anpassen, um zu versuchen, die Anzahl der angeforderten Einträge zurückzugeben. Wenn der Benutzer beispielsweise # 1 auf der Rangliste ist, Start auf -2 gesetzt ist und Ende auf 2 gesetzt ist, gibt Steam die ersten 5 Einträge in der Rangliste zurück.


Beispiel:

score_get = steam_download_scores_around_user("Game Scores", -4, 5);

Dies sendet eine Anfrage an den Steam Server für einen Bereich von 10 Punkten von der Bestenliste "Spielstände", zentriert auf den Spieler und speichert die asynchrone ID der Anfrage in der Variablen "score_get". Dies wird dann im Steam Async Event behandelt, wie im Extended Example für steam_download_scores.
