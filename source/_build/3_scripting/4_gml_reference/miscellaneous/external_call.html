External_call


Syntax:

external_call(id, args[0...15]);


Streit Beschreibung
id Der Name der Funktion, die Sie anrufen möchten
args[0...10] Die verschiedenen Argumente, die Sie an die externe Funktion übergeben möchten


Kehrt zurück:

Value (the type of value returned will depend on the defined function)


Beschreibung

Wenn Sie einen externen Funktionsaufruf an a angelegt haben dll oder dylib verwenden external_define, können Sie diese Funktion verwenden, um sie dann aufzurufen. Sie geben den Namen der zuvor definierten Funktion sowie jedes der benötigten Argumente an (jedes Argument muss vom richtigen Typ sein, entweder real oder string), und die Funktion gibt das Ergebnis des externen Aufrufs zurück.

WICHTIG! Diese Funktion ist mit der Testlizenz des Produkts nicht verfügbar.


Beispiel:

my_function = external_define("MyDLL.dll", "MyMin", dll_cdecl, ty_real, 2, ty_real, ty_real);
var _a = external_call(my_function, x, y);

Der obige Beispielcode ruft eine zuvor definierte externe Funktion auf und speichert den zurückgegebenen Wert in einer lokalen Variablen.
