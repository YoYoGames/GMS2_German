os_get_language

Diese Funktion gibt je nach Browser oder Gerätesprache einen anderen Wert zurück.

Syntax:

os_get_language()


Rückgabe: Zeichenfolge


Beschreibung

Diese Funktion gibt eine Zeichenfolge mit dem aus zwei Buchstaben bestehenden Sprachcode für das Betriebssystem zurück, auf dem das Spiel ausgeführt wird, wie im ISO639- Standard festgelegt. Wenn die Information nicht verfügbar ist, wird sie einfach eine leere Zeichenfolge "" oder "en" für "englische" Sprache enthalten. Beachten Sie, dass einige Sprachen auch einen relevanten Regionalcode haben, um die verschiedenen Regionen desselben Landes zu unterscheiden os_get_region().

Die folgende Tabelle zeigt einige der wichtigsten zweistelligen Sprachencodes gemäß ISO 639:

Sprache Code
Arabisch ar
Chinesisch z
dänisch da
Englisch de
Französisch fr
Deutsche de
griechisch el
Italienisch es
japanisch ja
norwegisch Nein
Polieren pl
Portugiesisch pt
Russisch ru
Spanisch es
Schwedisch SV



HINWEIS: Dies ist nicht der Code für den Standort, der zurückgegeben wird, sondern der Sprachencode des Betriebssystems!


Beispiel:

switch (os_get_language())
   {
   case "es": ini_open("spanish.ini"); break;
   case "fr": ini_open("french.ini"); break;
   case "it": ini_open("italian.ini"); break;
   default: ini_open("english.ini"); break;
   }

Der obige Code überprüft die Sprache des Betriebssystems und öffnet eine andere Sprache *.ini Datei abhängig vom zurückgegebenen Wert.
