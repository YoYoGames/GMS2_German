os_is_pausiert

Überprüft, ob das Betriebssystem des Geräts pausiert ist oder nicht.

Syntax:

os_is_paused()


Rückgabe: Boolean


Beschreibung

Bei der Entwicklung für mobile Geräte müssen Sie überprüfen können, ob das Gerät noch auf Ihr Spiel ausgerichtet ist oder nicht, insbesondere wenn Ihr Spiel auf einem iOS-, Android oder Windows Phone Gerät läuft und dann durch Telefonanrufe und unterbrochen werden kann Andere Apps. Mit dieser Funktion können Sie prüfen, ob das Gerät pausiert ist oder nicht.

Es ist erwähnenswert, dass diese Funktion jeden Code ausführt, der in dem Code-Block auf Geräten enthalten ist, aber dann wird GameMaker: Studio sofort angehalten, nachdem das Gerät das Spiel als Hintergrundprozess zwischengespeichert hat. Wenn Sie also beispielsweise den unten angegebenen Beispielcode haben, wird die zu erstellende Instanz erst nach dem Neustart des Spiels aus dem zwischengespeicherten Zustand tatsächlich angezeigt oder ausgeführt. Dies bedeutet, dass Sie möglicherweise lieber die haben os_is_paused Funktion ruft einen Alarm auf und dann im Alarmereignis (das eingestellt wird, wenn das Gerät pausiert ist, aber nicht läuft, bis es nicht pausiert wird), erzeuge ein Pausenmenü, starte Musik neu usw.

Wenn Sie diese Funktion auf Windows-, Ubuntu- (Linux), MacOS- oder HTML5-Plattformen verwenden, wird die Funktion nur für einen Schritt umgeschaltet true Wenn das Spiel den Fokus verliert, kehrt es zurück false bis das Spiel wieder fokussiert wird (und es wieder verliert). Dies liegt daran, dass diese Plattformen keine Apps und Spiele zwischenspeichern und diese daher weiterhin im Hintergrund laufen.

HINWEIS: Auf Mobilgeräten werden Sounds und Musik angehalten, wenn das Betriebssystem pausiert ist. Sie werden nicht neu gestartet, wenn Sie sie nicht mit der entsprechenden Funktion starten.


Beispiel:

if os_is_paused()
   {
   if !instance_exists(obj_PauseMenu) instance_create_layer(0, 0, "Controllers", obj_PauseMenu);
   }

Der obige Code überprüft, ob das Spiel pausiert wurde, und wenn er dies getan hat, erstellt er eine Instanz des Objekts "obj_PauseMenu", falls noch keine vorhanden ist.
