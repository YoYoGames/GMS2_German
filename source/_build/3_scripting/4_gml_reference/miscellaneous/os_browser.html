os_browser

Diese Variable hat eine andere Konstante, abhängig vom Browser, in dem das Spiel gerade ausgeführt wird.

Syntax:

os_browser


Rückgabe: Konstante


Beschreibung

Diese Variable enthält eine der verschiedenen Konstanten, die GameMaker Studio 2 Ihnen mitteilen muss, in welchem ​​Browser Sie das Spiel gerade ausführen (sofern vorhanden). Folgende Konstanten können zurückgegeben werden:

Konstante Beschreibung
browser_not_a_browser Das Spiel wird nicht in einem Browser gespielt
browser_unknown Unbekannter Browser
browser_ie Internet Explorer
browser_ie_mobile Internet Explorer auf einem mobilen Gerät
browser_firefox Mozilla Firefox
browser_chrome Google Chrome
browser_safari Safari
browser_safari_mobile Safari auf einem mobilen Gerät
browser_opera Oper
browser_tizen Tizen-Browser für mobile Geräte
browser_windows_store Windows App

Beispiel:

if os_browser == browser_not_a_browser
   {
   global.Config = 0;
   }
else
   {
   global.Config = 1;
   }

Der obige Code prüft, ob das Spiel in einem Browser ausgeführt wird oder nicht, und setzt eine globale Variable auf einen Wert, der vom Ergebnis der Überprüfung abhängig ist.
