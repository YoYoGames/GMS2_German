script_execute

Syntax:

script_execute(scr, arg0, arg1, arg2, ...);


Streit Beschreibung
scr Die ID des Skripts, das Sie aufrufen möchten
arg0... arg14 Die verschiedenen Argumente, die Sie an das Skript übergeben möchten


Rückgabe: N / A


Beschreibung

Es kann vorkommen, dass Sie ein Skript auf dynamische Weise an eine Instanz übergeben möchten, und für diese Fälle stellen wir das bereit script_execute Funktion. Es führt eine vom Benutzer erstellte Skriptressource aus und übergibt dem Skript bis zu 15 Argumente. Weitere Informationen zu Skripts finden Sie unter Skripts.


Beispiel:

script_execute(choose(scr_Up, scr_Down, scr_Left, scr_Right), irandom(5));

Der obige Beispielcode wird verwendet script_execute Wählen Sie eine der vier vom Benutzer erstellten Skriptressourcen (in diesem Fall in Bezug auf die Bewegung) und übergeben Sie eine zufällige Ganzzahl als Argument0.
