os_device

Diese Variable hat eine andere Konstante, abhängig vom Gerät, in dem das Spiel gerade ausgeführt wird.

Syntax:

os_device


Rückgabe: Konstante


Beschreibung

Diese Variable enthält eine der verschiedenen Konstanten, die GameMaker Studio 2 Ihnen mitteilen muss, auf welchem ​​Gerät Sie das Spiel gerade ausführen. Folgende Konstanten können zurückgegeben werden:

Konstante Beschreibung
device_ios_ipad iPad
device_ios_ipad_retina Neueres iPad mit Retina-Displaygröße von 2048 x 1536
device_ios_iphone6 iPhone6 ​​mit Displaygröße 1334 x 750
device_ios_iphone6plus Größeres iPhone 6 mit 1920 x 1080 Display
device_ios_iphone5 iPhone5 mit Displaygröße 640 x 1136)
device_ios_iphone Älteres iPhone / iPod Touch (480 x 320 Bildschirm) oder Android Telefon
device_ios_iphone_retina Neueres iPhone / iPod Touch mit Retina-Display von 960 x 640
device_emulator Das Gerät ist eigentlich ein Emulator ( Windows Phone oder Android )
device_tablet Android Tablet
device_ios_unknown Unbekanntes oder nicht iOS

Beispiel:

if os_browser = browser_not_a_browser
   {
   switch (os_device)
      {
      case device_ios_ipad: global.Config = 2; break;
      case device_ios_iphone: global.Config = 3; break;
      case device_ios_iphone_retina: global.Config = 4; break;
      case device_ios_unknown: global.Config = 5; break;
      }
   }
else
   {
   global.Config = 1;
   }

Der obige Code prüft, ob das Spiel in einem Browser ausgeführt wird oder nicht, und setzt eine globale Variable auf einen Wert, der vom Ergebnis der Überprüfung abhängig ist.
