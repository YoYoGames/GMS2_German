external_define


Syntax:

external_define(dll, name, calltype, restype, argnumb, argtype[0], argtype[1], ...argtype[10]) ;


Streit Beschreibung
dll Der Name der DLL-Datei (Zeichenfolge)
name Der Name der Funktion (String)
calltype Die Aufrufkonvention wird verwendet
restype Der Typ des zu erwartenden Ergebnisses
argnumb Die Anzahl der Argumente (0 - 10)
argtype[0... 10] Die verschiedenen Arten von Argumenten, die verwendet werden


Kehrt zurück:

Real (Unique function ID value)


Beschreibung

Diese Funktion kann verwendet werden, um einen externen Funktionsaufruf für eine bestimmte DLL (für Windows) oder dylib (für Mac) zu definieren. Diese Datei kann entweder eine enthaltene Datei oder Teil einer Erweiterung sein. Sie geben den Namen (und den Pfad) der Datei und dann den Namen der Funktion an, die Sie definieren möchten. Als nächstes müssen Sie die zu verwendende Aufrufkonvention definieren (siehe untenstehende Liste der Konstanten) sowie die Art des zu erwartenden Ergebnisses (ebenfalls eine Konstante, wie unten aufgeführt). Schließlich müssen Sie die Anzahl der Argumente angeben, die die Funktion annehmen kann (von 0 bis 15), und für jedes Argument müssen Sie auch seinen Typ angeben. Bitte beachten Sie, dass für Funktionen mit 4 oder mehr Argumente, alle von ihnen muss Typ ty_real.

HINWEIS: Dies gilt nur für dll oder dylib, die als "Enthaltene Dateien" zur GameMaker Studio 2 IDE hinzugefügt wurden. Es funktioniert nicht mit den Dateien, die als Erweiterungen hinzugefügt wurden, da diese erfordern, dass Sie die Funktionen im Erweiterungspaket selbst definieren.
WICHTIG! Diese Funktion ist mit der Testlizenz des Produkts nicht verfügbar.
Konstante Beschreibung
dll_cdecl Dies ist der standardmäßige C-, C ++ - Aufruf
dll_stdcall Dies ist der Standard-WinAPI-Aufruf (Windows dll nur)
ty_real Ein reelles Zahlenargument
ty_string ein nullterminiertes String-Argument


Beispiel:

my_funcion = external_define("MyDLL.dll", "MyMin", dll_cdecl, ty_real, 2, ty_real, ty_real);

Der obige Beispielcode definiert eine externe Funktion namens "MyMin" mit zwei Argumenten.
