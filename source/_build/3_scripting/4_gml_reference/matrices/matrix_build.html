matrix_build


Beschreibung

Diese Funktion kann verwendet werden, um Ihre eigene benutzerdefinierte Matrix zu erstellen, und gibt einen Indexwert für diese neue Matrix zurück, der in einer Variablen gespeichert werden sollte, um sie später zu referenzieren und zu verwenden. Die Matrix selbst wird als 16-Wert-1D-Array erstellt, wobei die ersten 4 Elemente Zeile 1, die zweiten 4 Elemente Zeile 2 usw. einer 4x4-Matrix sind.

HINWEIS: Wenn Sie auf diese Weise eine neue Matrix erstellen, lautet die Reihenfolge der Operation YXZ.


Syntax:

matrix_build(x, y, z, xrotation, yrotation, zrotation, xscale, yscale, zscale);


Argumente:

Streit Beschreibung
x Die x-Komponente des Translationsvektors.
y Die y-Komponente des Übersetzungsvektors.
z Die z-Komponente des Translationsvektors.
xrotation Der Winkel zum Drehen um die x-Achse.
yrotation Der Winkel zum Drehen um die y-Achse.
xzotation Der Winkel zum Drehen um die z-Achse.
xscale Die x-Skalierungsmenge.
yscale Die y-Skalierungsmenge.
zscale Die z-Skalierungsmenge.


Kehrt zurück:

4x4 Matrix


Beispiel:

t_matrix = matrix_build(x, y, 0, 0, 90, 0, 1, 2, 1);

Der obige Code erstellt eine neue Matrixtransformation und speichert den resultierenden Matrixindex in der Variablen "t_matrix".
