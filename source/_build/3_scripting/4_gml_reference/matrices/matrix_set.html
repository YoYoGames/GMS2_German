matrix_set


Beschreibung

Diese Funktion nimmt ein 1D Array von 16 Werten, entsprechend einem gegebenen 4x4 Matrix Typ, wobei Elemente [0 - 3] Zeile 1, Elemente [4 - 7] Zeile 2, etc... sein würden (siehe Bild auf die Hauptseite ). Sie können eine solche Matrix mit dem erstellen matrix_build oder matrix_get Funktionen oder einfach das Array selbst aufbauen und in die Funktion übernehmen. Die verfügbaren Matrixtypen sind Ansicht, Projektion und Welt, für die Sie eine der folgenden Konstanten verwenden würden:

Konstante Beschreibung
matrix_view Die aktuelle Ansichtsmatrix
matrix_projection Die aktuelle Projektionsmatrix
matrix_world Die aktuelle Weltmatrix


Syntax:

matrix_set(type, matrix);


Argumente:

Streit Beschreibung
type Der Typ der Matrix, um die Werte von zu erhalten (siehe die oben aufgeführten Konstanten )
matrix Die Matrixdaten als Array


Kehrt zurück:

NA


Beispiel:

matrix_set(matrix_world, m_array);

Der obige Code setzt die Werte der aktuellen Weltmatrix auf diejenigen, die in der Matrix "m_array" gespeichert sind.
