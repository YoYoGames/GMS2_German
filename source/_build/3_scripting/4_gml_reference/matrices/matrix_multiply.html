matrix_multiply


Beschreibung

Mit dieser Funktion können Sie zwei Matrix-Arrays miteinander multiplizieren, um eine neue Transformationsmatrix zu erstellen. Die Funktion gibt den neuen Matrix-Index zurück, der zur zukünftigen Verwendung in einer Variablen gespeichert werden sollte.

HINWEIS: Sie können eine Matrixkonstante nicht als Argument für diese Funktion verwenden. Wenn Sie also die (Beispiel-) Ansichtsmatrix mit einer benutzerdefinierten Matrix multiplizieren möchten, die Sie erstellt haben, müssen Sie zuerst aufrufen matrix_get(type) Ordnen Sie die Ansichtsmatrixwerte einer Array-Variablen zu, multiplizieren Sie sie mit Ihrer benutzerdefinierten Matrix und legen Sie dann die ausgewählte Matrix (Ansicht, Projektion oder Welt) fest.


Syntax:

matrix_multiply(matrix1, matrix2);


Argumente:

Streit Beschreibung
matrix1 Der erste zu verwendende Matrixindex.
matrix2 Der zweite zu verwendende Matrixindex.


Kehrt zurück:

4x4 Matrix


Beispiel:

var v_matrix = matrix_get(matrix_view);
var new_matrix = matrix_multiply(v_matrix, my_matrix);
matrix_set(matrix_view, new_matrix);

Der obige Code ruft die aktuelle Ansichtsmatrix ab, multipliziert diese mit einer benutzerdefinierten Matrix und verwendet dann den resultierenden Matrixindex, um die Ansichtsmatrix erneut festzulegen.
