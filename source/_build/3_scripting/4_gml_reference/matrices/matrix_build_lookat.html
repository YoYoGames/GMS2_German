matrix_build_lookat


Beschreibung

Diese Funktion erstellt eine "Look-at" (View) -Matrix basierend auf den unten angegebenen Parametern. Da diese Funktion die Ansichtsmatrix und nicht die Projektionsmatrix ändert, sollten Sie zunächst die Projektionsmatrix mit der anderen Matrixfunktion initialisieren matrix_build_projection_perspective Verwenden Sie diese Funktion, um die Ansichtskamera innerhalb der Projektion zu verschieben.

So stellen Sie die Ansicht, die Sie zuerst die Position Sie definieren müssen schauen, die durch die Parameter angezeigt (xFrom, yfrom, zfrom). Als nächstes müssen Sie die Richtung, die Sie betrachten angeben und dies, indem Sie einen zweiten Punkt erfolgt mit den Argumenten suchen Richtung (xto, yto, ZTO). Schließlich können Sie die Kamera immer noch um die Linie vom Betrachtungspunkt zum Betrachtungspunkt drehen. Um dies anzugeben, müssen Sie einen "Aufwärts" -Vektor angeben - die Richtung, die in der Kamera nach oben gerichtet ist. Dies wird durch die letzten drei Argumente (xup, yup, zup) gegeben.


Syntax:

matrix_build_lookat(xfrom, yfrom, zfrom, xto, yto, zto, xup, yup, zup);


Argumente:

Streit Beschreibung
xfrom Die x-Koordinate der Position, von der aus gesucht werden soll.
yfrom Die y-Koordinate der Position, von der aus betrachtet werden soll.
zfrom Die z-Koordinate der Position, von der aus gesucht werden soll.
xto Die x-Koordinate der zu betrachtenden Position.
yto Die y-Koordinate der zu betrachtenden Position.
zto Die z-Koordinate der zu betrachtenden Position.
xup Die x-Koordinate des "up" -Vektors.
yup Die y-Koordinate des "up" -Vektors.
zup Die z-Koordinate des "up" -Vektors.


Kehrt zurück:

Matrix index


Beispiel:

viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0);
projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0);
camera_set_view_mat(view_camera[0], viewmat);
camera_set_proj_mat(view_camera[0], projmat);

Der obige Code erstellt eine neue Anschauungsmatrix und eine orthografische Matrix, speichert ihre IDs in lokalen Variablen und verwendet sie zum Festlegen der Ansichts- und Projektionsmatrizen für die dem Ansichtsanschluss zugewiesene Kamera [0].
