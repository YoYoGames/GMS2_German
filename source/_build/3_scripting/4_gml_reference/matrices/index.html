Matrixfunktionen

Eine Matrix ist eine Sammlung von Zahlen, die in einer festen Anzahl von Zeilen und Spalten angeordnet sind. Normalerweise sind die Zahlen reelle Zahlen, aber das muss nicht immer der Fall sein. Das folgende Bild zeigt allgemein, wie eine Matrix mit vier Zeilen und vier Spalten aufgebaut wird:

Die oberste Zeile ist Zeile 1, die Spalte ganz links Spalte 1, und diese Matrix ist eine 4x4-Matrix, da sie vier Zeilen und vier Spalten hat (Matrizen anderer Größe können konstruiert werden). Bei der Beschreibung von Matrizen ist das Format immer Zeilen / Spalten und jede Zahl, die eine Matrix bildet, wird als Element der Matrix bezeichnet. Die Elemente in einer Matrix haben spezifische Positionen, die durch ihre Zeilen- und Spaltenposition beschrieben sind, wobei die obere linke Ecke der Matrix Zeile 1 Spalte 1 ist. In dem obigen Matrixbeispiel hat das Element in Zeile 1 Spalte 1 den Wert "1". Das Element in Zeile 2 Spalte 3 ist der Wert "4.8".

Matrizen sind für viele Dinge sehr wichtig, werden aber am meisten im Umgang mit 3D-Raum verwendet, zum Beispiel können sie zum Einstellen der Kameraansicht oder zum Übersetzen oder Transformieren eines Modells verwendet werden.
