matrix_build_projection_perspective_fov


Beschreibung

Diese Funktion erstellt eine perspektivische Projektionsmatrix basierend auf dem Sichtfeld unter Verwendung der unten aufgeführten Parameter.


Syntax:

matrix_build_projection_perspective_fov(fov_y, aspect, znear, zfar);


Argumente:

Streit Beschreibung
fov Der Winkel des Sichtfelds.
aspect Das Seitenverhältnis des Sichtfelds.
znear Die nahe Clipping-Ebene.
zfar Die ferne Clipping-Ebene.


Kehrt zurück:

Matrix index


Beispiel:

projmat = matrix_build_projection_perspective_fov(60, 320/240, 1.0, 32000.0);
camera_set_proj_mat(view_camera[0], projmat);

Der obige Code erzeugt eine Projektionsfeldmatrix, die dann in einer Variablen gespeichert wird. Diese Matrix wird dann verwendet, um die Kamera einzurichten, die dem Port [0] zugeordnet ist.
