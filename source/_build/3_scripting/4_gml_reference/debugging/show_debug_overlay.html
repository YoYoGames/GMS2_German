show_debug_overlay


Beschreibung

Diese Funktion kann verwendet werden, um das Standard-Debug-Overlay beim Testen des Spiels ein- und auszuschalten, und ist standardmäßig deaktiviert. Das Debug-Overlay zeigt eine grafische CPU / GPU-Verwendungsleiste im eigentlichen Spielfenster selbst zusammen mit dem aktuellen realen fps-Wert, der Anzahl der Textur-Swaps und der Anzahl der Vertex-Chargen (beachten Sie, dass Textur-Swaps und Vertex-Chargen niemals Null sind und normalerweise Zeige Werte von 2 oder 3, da auch bei einem leeren Raum noch keine Objekte in GameMaker Studio 2 gezeichnet werden müssen).

Diese Leiste ist in Abschnitte unterteilt, wobei jeder Abschnitt 1/60 Sekunden lang ist. Wie Sie aus dem obigen Bild sehen können, besteht die Leiste aus verschiedenen Farben:

  • Grün - Eingabe / Ausgabe Verarbeitung (zB: Tastatur, Maus, Gamepad, Netzwerk usw.)
  • Rot - Die Aktualisierungsgeschwindigkeit des Schrittes
  • Gelb - Die Zeit, die für das Zeichenereignis benötigt wird
  • Orange -Debug Aktualisierungszeit, die normalerweise nur sichtbar ist, wenn Sie das Debug-Modul verwenden
  • Weiß - Die verbleibende GPU ist die Zeit, die vergeht, bis die GPU das Rendern des Frames beendet hat, bevor der nächste beginnen kann
  • Cyan - Die Zeit zum Rendern des Texts
  • Grau - Die Zeit, die benötigt wird, um jeden Zeichenschritt zu löschen
  • Dunkelrot - Der GPU-Flush, wie lange die GPU benötigt, um Bilder aus dem Speicher zu löschen

Mit dieser Funktion können Sie das Overlay hinzufügen, ob im Debug-Modus oder nicht, und auf diese Weise können Sie sehen, wie effizient Ihr Spiel läuft und einen visuellen Hinweis bekommen, wie es die verfügbaren Ressourcen nutzt, ohne den Overhead zu haben der Debugger läuft neben.


Syntax:

show_debug_overlay(enable);

Streit Beschreibung
enable einschalten ( true ) oder aus ( false ) das Debug-Overlay.


Kehrt zurück:

N/A


Beispiel:

if global.debug
   {
   show_debug_overlay(true);
   }
else
   {
   show_debug_overlay(false);
   }

Der obige Code schaltet das Debugging abhängig vom Wert einer globalen Variable ein oder aus.
