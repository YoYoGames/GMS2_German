fps


Beschreibung

In GameMaker Studio 2 gibt es zwei Möglichkeiten, um die Geschwindigkeit zu ermitteln, mit der das Spiel ausgeführt wird. Das game speed (wie in den Spieloptionen angegeben) und der fps (Bilder pro Sekunde). Diese Werte sind oft verwechselt, aber im Grunde ist eine die Anzahl der Spielschritte, die GameMaker Studio 2 in einer Sekunde GameMaker Studio 2 soll, während die andere (die fps) die Anzahl der CPU - Schritte ist, die GameMaker Studio 2 tatsächlich in einem GameMaker Studio 2 zweitens bis zu einem Höchstwert der Raumgeschwindigkeit. Um die wahren fps zu erhalten, d. die tatsächliche Anzahl der CPU-Schritte pro Spielschritt, verwenden Sie die fps_real Variable.

Diese schreibgeschützte Variable gibt die aktuellen fps als ganzzahligen Wert zurück. Bitte beachten Sie, dass die Funktion nur einmal in jedem Schritt Ihres Spiels aktualisiert wird und so von einem Wert zum anderen zu springen scheint, aber das ist ganz normal.


Syntax:

fps


Kehrt zurück:

Real


Beispiel:

if debug_mode
   {
   draw_text(32, 32, "FPS = " + string(fps));
   }

Der obige Code überprüft, ob sich das Spiel im Debug-Modus befindet und wenn dies der Fall ist, werden die aktuellen fps auf dem Bildschirm angezeigt.
