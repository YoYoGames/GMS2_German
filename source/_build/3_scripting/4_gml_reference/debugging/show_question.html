show_question


Beschreibung

Diese Funktion erstellt ein Popup-Meldungsfeld mit zwei Schaltflächen für "Ja" und "Nein". Es kehrt zurück true oder false abhängig davon, welche der beiden Tasten der Benutzer drückt.

HINWEIS: Diese Funktion ist für Debug - Anwendung ist nur auf Ziele Desktop, sondern auf alle anderen Ziele ist veraltet.


Syntax:

show_question(str);

Streit Beschreibung
str Die Zeichenfolge, die in der Popup-Frage angezeigt werden soll.


Kehrt zurück:

Boolean


Beispiel:

if score > 500
   {
   if show_question("Continue to next room?")
      {
      room_goto(rm_Level2);
      }
   else game_end();
   }

Der obige Code prüft die Punktzahl, und wenn der Wert über 500 liegt, werden Sie gefragt, ob Sie fortfahren möchten oder nicht. Wenn Sie auf die Schaltfläche "Ja" klicken, wird ein anderer Raum aufgerufen ausgewählt wird das Spiel beendet.
