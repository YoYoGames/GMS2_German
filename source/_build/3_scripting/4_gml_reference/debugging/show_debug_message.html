show_debug_message


Beschreibung

Diese Funktion erstellt eine benutzerdefinierte Debugmeldung, die zur Laufzeit im Compilerfenster angezeigt wird. Beachten Sie, dass die Nachricht entweder eine Zeichenfolge oder eine reelle Zahl sein kann, aber wenn Sie beide benötigen, müssen Sie zuerst die Zahl in eine Zeichenfolge konvertieren string Funktion (siehe Beispiel unten) und wenn die Zahl mehr als zwei Dezimalstellen hat, sollten Sie verwenden string_format Um sie standardmäßig anzuzeigen, werden die Dezimalstellen auf die nächsten zwei Dezimalstellen gerundet (dh "1.2468" würde im Ausgabefenster als "1.25" angezeigt).

Debug-Nachrichten, die mit dieser Funktion angezeigt werden, werden im Compiler-Ausgabefenster unten in der IDE sowie in der Graphansicht des Debuggers angezeigt, wenn das Spiel im Debug-Modus ausgeführt wird. Wenn Sie nur Nachrichten im Debug-Modus sehen möchten, sollten Sie wahrscheinlich verwenden debug_event stattdessen.


Syntax:

show_debug_message(string);

Streit Beschreibung
string Die benutzerdefinierte Debug-Nachricht zum Anzeigen.


Kehrt zurück:

N/A


Beispiel:

if !instance_exists(obj_Exit)
   {
   show_debug_message("Exit not created!");
   show_debug_message("Error Num: " + string(global.error));
   game_end();
   }

Der obige Code überprüft, ob eine Instanz existiert, und wenn dies nicht der Fall ist, wird eine Debug-Nachricht im Kompilierungsfenster angezeigt und das Spiel wird beendet.
