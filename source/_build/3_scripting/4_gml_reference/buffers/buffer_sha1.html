puffer_sha1

Syntax:

buffer_sha1(buffer, offset, size);


Streit Beschreibung
buffer Der Index des zu verwendenden Puffers.
offset Der Datenversatzwert.
size Die Größe des Puffers.


Rückgabe: Zeichenfolge


Beschreibung

In der Kryptographie ist SHA-1 eine kryptografische Hash-Funktion, die von der National Security Agency der Vereinigten Staaten entwickelt wurde und in mehreren weit verbreiteten Anwendungen und Protokollen wie dem bekannten Git verwendet wird, um dort nach Dateiänderungen zu suchen. Diese Funktion nimmt Eingabedaten aus einem Puffer und gibt einen 160-Bit-Nachrichtenauszug im ASCII-Format zurück. Auf diese Weise können Sie einen sicheren Schlüssel generieren, der gespeichert und verwendet werden kann, um die Integrität der Informationen zu prüfen, die zum Beispiel an einen externen Server gesendet (oder von diesem empfangen) werden. Wenn Sie dies auf Puffer anwenden, die diese Funktion verwenden, müssen Sie die Puffer-ID des zu verwendenden Puffers angeben, dann einen Offset-Wert (in Bytes) für den Beginn und dann eine Größe (erneut in Byte) für die zu hashende Region.


Beispiel:

check_string = buffer_sha1(buff, 0, buffer_get_size(buff));

Der obige Code erzeugt einen sha1-Hash für die gesamten Daten, die in dem Puffer gespeichert sind, der durch die Variable "buff" indiziert wird, und speichert den zurückgegebenen Hash in der Variablen "check_string".
