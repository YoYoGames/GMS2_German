Pufferkopie

Syntax:

buffer_copy(src_buffer, src_offset, size, dest_buffer, dest_offset);


Streit Beschreibung
src_buffer Der Index des Puffers kopieren aus.
src_offset Der Datenoffset, von dem aus kopiert werden soll (in Byte).
size Die Größe der zu kopierenden Daten (in Byte).
dest_buffer Der Index des Puffers zu kopieren.
dest_offset Die Offsetposition zum Kopieren der Daten in (in Byte).


Rückkehr: N / A


Beschreibung

Diese Funktion kann verwendet werden, um ein Segment (oder alle) der in einem Puffer gespeicherten Daten auf ein anderes zu kopieren. Wenn Sie zwei Puffer verwenden und von einem zum anderen kopieren, müssen beide Puffer zuvor mit dem Puffer erstellt worden sein buffer_create Funktion (zum Beispiel), und Sie können einen Datenoffset (in Bytes) für den Startpunkt der Daten angeben, die aus dem Quellpuffer relativ zum Anfang des Puffers kopiert werden sollen, sowie einen weiteren Datenoffset, um die Position zu definieren Kopieren Sie die Daten in den Zielpuffer.

Hinweis: Sie können nicht in demselben Puffer kopieren.


Beispiel:

buffer_copy(buff1, 0, 2048, buff2, 2048);

Der obige Code kopiert die Daten, die in dem Puffer gespeichert sind, der in der Variablen "buff1" indiziert ist, und fügt ihn dann in den Puffer ein, der in der Variablen "buff2" indiziert ist, jedoch um 2048 Bytes versetzt.
