buffer_get_surface

Syntax:

buffer_get_surface(buffer, surface, mode, offset, modulo);


Streit Beschreibung
buffer Der Index des zu verwendenden Puffers.
surface Der Index der zu verwendenden Oberfläche.
mode Die Daten, die von der Oberfläche abgerufen werden sollen (siehe Liste der Konstanten unten).
offset Der Datenversatzwert.
modulo Der Versatz vom Ende der Zeile zum Anfang der nächsten


Rückgabe: N / A


Beschreibung

Mit dieser Funktion können Sie die Informationen von einer Oberfläche in einen bestimmten Puffer schreiben. Der Puffer muss zuvor erstellt worden sein und sollte ein 1-Byte-ausgerichteter Wachstumspuffer mit einer ungefähren Größe (in Byte) der Oberfläche sein, die Sie schreiben werden. Sie können den Kopiermodus für die zu speichernden Daten auswählen, der durch die unten aufgeführten Konstanten sowie die Versatz- und Modulo-Werte festgelegt wird. Sowohl Offset als auch Modulo sind Byte-Werte, wobei der Offset die Startposition innerhalb des Puffers definiert und Modulo die Anzahl der am Ende jeder Zeile verbleibenden Bytes (zum Speichern zusätzlicher Daten). Beachten Sie, dass die Funktion jedes Pixel der Oberfläche mit einer BGRA- Formatierung auf dem Windows-Ziel in den Puffer schreibt. Bei anderen Zielen kann es jedoch je nach Betriebssystem oder Gerät unterschiedlich sein.

Konstante Beschreibung
buffer_surface_copy Kopiert alle Daten von der Oberfläche.


Beispiel:

buffer_get_surface(buff, application_surface, buffer_surface_copy, 0, 0);

Der obige Code kopiert die in der Anwendungsoberfläche gespeicherten Daten in den Puffer, der in der Variablen "buff" ohne Offset- oder Modulo-Werte indiziert ist.
