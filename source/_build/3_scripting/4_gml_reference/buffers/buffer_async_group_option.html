buffer_async_group_option

Syntax:

buffer_async_group_option(option, value);


Streit Beschreibung
option Die Option zum Einstellen.
value Der einzustellende Wert (kann je nach Option string oder real sein).


Rückgabe: N / A


Beschreibung

Mit dieser Funktion können Sie einige plattformspezifische Optionen für die zu speichernde Puffergruppe einstellen. Die verfügbaren Optionen sind wie folgt:

  • " subtitle " oder " slottitle " - Der Wert für diese Option wäre eine Zeichenfolge, die dem Benutzer angezeigt wird, wenn er seine Sicherungsdaten im Betriebssystem verwaltet. Dies ist nur beim Speichern von Daten wichtig, nicht beim erneuten Laden.

  • " showdialog " - Der Wert für diese Option ist ein Boolean true oder false. Wenn eingestellt auf true Wenn Sie GameMaker Studio 2 speichern oder laden, wird die Systembenutzeroberfläche angezeigt. Andernfalls wird eine Schnellspeicherung / Schnellladung ohne angezeigte Benutzeroberfläche durchgeführt. Normalerweise benötigen Sie dies nur, wenn Sie mehrere Speicherslots unterstützen, damit der Benutzer einen Slot auswählen kann. Wenn Sie jedoch nur einen Slot pro Benutzer unterstützen, setzen Sie diesen Wert auf false.

  • " savepadindex " - Bei dieser Option geben Sie den Pad-Index des Players an, der speichert oder lädt, und das System schreibt Daten in den Speicherordner dieses Players und liest Daten aus diesem Ordner.

  • " saveslotsize " - Bei dieser Option müssen Sie die tatsächliche Größe in Byte angeben, die Sie speichern möchten (damit Sie eine buffer_seek und buffer_tell um das zum Beispiel zu bekommen). Beachten Sie, dass die Angabe dieses Wertes nicht obligatorisch ist, da allen Speicherungen ein Mindestspeicherplatz zugewiesen wird, der normalerweise etwa 10 MB beträgt.

WICHTIG - Diese Funktion ist derzeit nur für die Zielmodule PSVita, PS4 und XBox One gültig. Bei allen anderen Zielen tut es nichts.


Beispiel:

buffer_async_group_begin("save_folder_name");
buffer_async_group_option("showdialog", false);
buffer_async_group_option("slottitle", "Catch The Haggis Save");
buffer_async_group_option("subtitle", "All your haggis are saved here!");
save = buffer_save_async(buff, "Player_Save.sav", 0, 16384);
buffer_async_group_end();

Der obige Code startet eine Puffergruppe und legt dann die Gruppenoptionen fest, bevor 4 Dateien asynchron gespeichert werden. Die Gruppendefinition wird dann beendet (ab diesem Zeitpunkt beginnt das Speichern).
