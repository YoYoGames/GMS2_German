buffer_base64_decode

Syntax:

buffer_base64_decode(string);


Streit Beschreibung
string Die Base64-codierte Zeichenfolge zum Decodieren


Rückgabe: Echt


Beschreibung

Mit dieser Funktion können Sie eine base64 kodierte Zeichenkette dekodieren (erstellt mit der buffer_base64_encode Funktion) in einen Puffer. Diese Funktion erstellt den Puffer (als ein 1-Byte-ausgerichteter "Grow" -Puffer) und gibt den eindeutigen Index für den Puffer zurück, der in allen weiteren Funktionsaufrufen verwendet werden sollte.

HINWEIS: Es ist wichtig, dass Sie alle dynamisch erstellten Ressourcen wie diese aus dem Speicher entfernen, wenn Sie sie nicht mehr benötigen, um Speicherlecks zu vermeiden. Wenn Sie den erstellten Puffer fertiggestellt haben, sollten Sie ihn wieder freigeben buffer_delete.


Beispiel:

ini_open("Save.ini");
buff = buffer_base64_decode(ini_read_string("Save", "Slot1", ""));
ini_close();

Der obige Code wird eine Ini-Datei öffnen und dann eine Zeichenfolge daraus in die Dekodierungsfunktion lesen. Diese Funktion gibt einen Pufferindex zurück, der in der Variablen "buff" gespeichert ist und die zuvor codierten und gespeicherten Daten enthält. Die Ini-Datei wird dann geschlossen.
