Puffer_peek

Syntax:

buffer_peek(buffer, offset, type);


Streit Beschreibung
buffer Der Index des zu verwendenden Puffers.
offset Die Offset-Position (in Bytes) innerhalb des Puffers zum Lesen der angegebenen Daten.
type Die Art der Daten, die aus dem Puffer gelesen werden sollen (siehe die Liste der Konstanten hier ).


Liefert: Real / String


Beschreibung

Mit dem buffer_read Englisch: www.weisang.info/index.php?id=143&t...h=ddcdcdcdc2 Mit dieser Funktion können Sie Daten aus dem gegebenen Puffer an der aktuellen "Such" -Position lesen, wobei jedes Datenelement gelesen wird, wobei diese Position durch die gelesenen oder geschriebenen Bytes fortgeschrieben wird Es kann jedoch erforderlich sein, dass Sie ein bestimmtes Datenelement lesen, ohne die aktuelle Suchposition ändern zu wollen. In diesem Fall würden Sie diese Funktion verwenden. Sie geben einfach die Funktion mit einer Puffer-ID und dann eine Offset-Position (vom Pufferstart) innerhalb dieses Puffers aus, von der gelesen werden soll, sowie den Datentyp, den Sie lesen möchten.

HINWEIS: Die Verwendung des falschen Datentyps für die gelesenen Daten führt zu fehlerhaften Werten!


Beispiel:

var red = buffer_peek(buff, 1, buffer_u8);
var green = buffer_peek(buff, 2, buffer_u8);
var blue = buffer_peek(buff, 3, buffer_u8);
image_blend = make_colour_rgb(red, green, blue);

Der obige Code erhält drei Werte aus drei verschiedenen Positionen innerhalb des Puffers, der in der Variablen "Buff" indiziert ist, und verwendet dann diese Werte, um die Bildmischung der Instanz festzulegen.
