buffer_poke

Syntax:

buffer_poke(buffer, offset, type, value);


Streit Beschreibung
buffer Der Index des zu verwendenden Puffers.
offset Die Offset-Position (in Byte) innerhalb des Puffers zum Schreiben der angegebenen Daten.
type Die Art der Daten, die in den Puffer geschrieben werden sollen (siehe Liste der Konstanten hier ).
value Die Daten, die dem Puffer entsprechend dem angegebenen Typ hinzugefügt werden.


Rückkehr: N / A


Beschreibung

Mit dem buffer_write Funktion können Sie Daten in den gegebenen Puffer an der aktuellen "Such" -Position schreiben, wobei jedes Datenstück diese Position vorrückt, indem die Bytes geschrieben oder gelesen werden. Es kann jedoch erforderlich sein, dass Sie ein bestimmtes Datenelement ändern, ohne die aktuelle Suchposition ändern zu wollen. In diesem Fall würden Sie diese Funktion verwenden. Sie liefern einfach die Funktion mit einem Pufferindex und dann die Offsetposition vom Pufferstart (in Bytes) innerhalb des Puffers, in den geschrieben werden soll, sowie den Datentyp und den Wert, der geschrieben werden soll.


Beispiel:

buffer_poke(buff, 3, buffer_u8, colour_get_blue(image_blend));

Der obige Code fügt den blauen Komponentenwert der für die Bildmischung verwendeten Farbe in den Puffer ein, der in der Variablen "buff" indiziert ist, an der dritten Position im Puffer und als vorzeichenloser 8-Bit-Wert.
