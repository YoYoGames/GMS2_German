buffer_set_surface

Syntax:

buffer_set_surface(buffer, surface, mode, offset, modulo);


Streit Beschreibung
buffer Der Index des zu verwendenden Puffers.
surface Der Index der zu verwendenden Oberfläche.
mode Die Daten, die aus dem Puffer abgerufen werden sollen (siehe Liste der Konstanten unten).
offset Der Datenversatzwert.
modulo Der Offset vom Ende der Zeile bis zum Anfang des nächsten


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie die Informationen aus einem Puffer auf eine gegebene Oberfläche schreiben. Sowohl der Puffer als auch die Oberfläche müssen zuvor erstellt worden sein. Sie können den Kopier- "Modus" für die gespeicherten Daten wählen, der durch die unten aufgeführten Konstanten sowie die Offset- und Modulo-Werte eingestellt wird. Sowohl offset als auch modulo sind Byte-Werte, wobei der Offset die Startposition innerhalb des Puffers definiert und modulo die Anzahl der Bytes ist, die am Ende jeder Zeile verblieben (zum Speichern zusätzlicher Daten), als der Puffer geschrieben wurde.

Konstante Beschreibung
0 Kopiert alle Daten aus dem Puffer.


Beispiel:

buffer_set_surface(buff, application_surface, 0, 0, 0);

Der obige Code kopiert alle Daten, die in dem Puffer gespeichert sind, der in der Variablen "buff" indiziert ist, auf die Anwendungsoberfläche ohne Offset- oder Modulo-Werte.
