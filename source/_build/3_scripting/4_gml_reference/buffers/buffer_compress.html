buffer_compress


Syntax:

buffer_compress(buffer, offset, size);


Streit Beschreibung
buffer Der Index des zu komprimierenden Puffers.
offset Der zu komprimierende Offset innerhalb des Puffers (in Byte).
size Die Größe des zu komprimierenden Pufferbereichs (in Byte).


Rückgabe: Pufferindex


Beschreibung

Mit dieser Funktion können Sie einen Teil (oder alle) eines Puffers mithilfe der Zlib-Komprimierung komprimieren. Sie geben die ID des zu komprimierenden Puffers an (wie von zurückgegeben buffer_create() ), der Offset in dem Puffer, der in Byte verwendet werden soll, und die Größe der zu komprimierenden Pufferdaten (auch in Byte). Die Funktion gibt einen neuen Puffer-ID-Wert für den komprimierten Puffer zurück oder einen Wert unter 0, falls er aus irgendeinem Grund fehlgeschlagen ist. Diese Funktion ändert den ursprünglichen Puffer nicht.


Beispiel:

var srcBuff = buffer_create(1024, buffer_grow, 1);
buffer_write(srcBuff, global.DataString);
var cmpBuff = buffer_compress(srcBuff, 0, buffer_tell(srcBuff));
buffer_save(cmpBuff, "Player_Save.sav");
buffer_delete(srcBuff);
buffer_delete(cmpBuff);

Der obige Code erstellt einen Puffer und füllt ihn dann mit den Daten eines Strings. Dieser Puffer wird dann komprimiert und gespeichert, und sowohl der Quellpuffer als auch der komprimierte Puffer werden gelöscht
