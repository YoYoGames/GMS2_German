Puffer_Tell

Syntax:

buffer_tell(buffer);


Streit Beschreibung
buffer Der Index des zu verwendenden Puffers.


Rückgabe: Echt


Beschreibung

Wenn Sie Daten in einen Puffer lesen oder schreiben, verwenden Sie die buffer_read oder buffer_write Die aktuelle "Such" -Position wird um die geschriebenen oder gelesenen Bytes vorgerückt, und mit dieser Funktion können Sie die aktuelle "Such" -Position für die Verwendung in anderen Pufferfunktionen erhalten. Wenn Ihre Pufferausrichtung beispielsweise auf 4 Byte festgelegt ist und Sie ein einzelnes Datenelement schreiben, das 1 Byte groß ist, führen Sie a buffer_tell, erhalten Sie einen Rückgabewert von 1. Wenn Sie jedoch ein weiteres Datenelement schreiben, das ebenfalls 1 Byte groß ist, machen Sie a buffer_tell, erhalten Sie einen Rückgabewert von 5, da die Ausrichtung die Daten an dieser Position "aufgefüllt" hat.


Beispiel:

var pos = buffer_tell(buff); buffer_seek(buff, buffer_seek_start, 0);
val[0] = buffer_read(buff, buffer_S16);
val[1] = buffer_read(buff, buffer_S16);
val[2] = buffer_read(buff, buffer_S16);
buffer_seek(buff, buffer_seek_start, pos);

Der obige Code speichert die aktuelle Suchposition innerhalb des Puffers, der in der Variablen "buff" indiziert ist, auf die lokale Variable "pos". Die Puffersuchposition wird dann auf den Anfang des Puffers gesetzt, und drei Datenstücke werden in ein Array eingelesen, bevor schließlich die Puffersuchposition wieder auf die vorherige Position zurückgesetzt wird.
