Puffer_md5

Syntax:

buffer_md5(buffer, offset, size);


Streit Beschreibung
buffer Der Index des zu verwendenden Puffers.
offset Der Datenversatzwert.
size Die Größe des Puffers.


Rückgabe: Zeichenfolge


Beschreibung

In der Kryptographie ist MD5 (Message-Digest-Algorithmus 5) eine weit verbreitete kryptographische Hash-Funktion mit einem 128-Bit-Hash-Wert und wurde in einer Vielzahl von Sicherheitsanwendungen eingesetzt. Es wird auch häufig verwendet, um die Integrität von Daten zu überprüfen. Diese Funktion nimmt die Eingabedaten aus einem bestimmten Puffer (oder einem Teil des Puffers) und gibt den 32-stelligen hexadezimalen MD5-Hash zurück, der für diese Daten eindeutig ist. Auf diese Weise können Sie einen sicheren Schlüssel generieren, der gespeichert und verwendet werden kann, um die Integrität der Informationen zu prüfen, die zum Beispiel an einen externen Server gesendet (oder von diesem empfangen) werden. Wenn Sie dies auf Puffer anwenden, die diese Funktion verwenden, müssen Sie die Puffer-ID des zu verwendenden Puffers angeben, dann einen Offset-Wert (in Bytes) für den Beginn und dann eine Größe (erneut in Byte) für die zu hashende Region.


Beispiel:

check_string = buffer_md5(buff, 0, buffer_get_size(buff));

Der obige Code erstellt einen md5-Hash für die vollständigen Daten, die im Puffer gespeichert sind, der durch die Variable "buff" indiziert wird, und speichert den zurückgegebenen Hash in der Variablen "check_string".
