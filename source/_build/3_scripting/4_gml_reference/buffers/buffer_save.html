Pufferspeicherung

Syntax:

buffer_save(buffer, filename);


Streit Beschreibung
buffer Der Index des zu sichernden Puffers.
filename Der Name der zu speichernden Datei.


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie den Inhalt eines Puffers in einer Datei speichern, die mit Hilfe der Taste in den Speicher zurückgelesen werden kann buffer_load Funktion.


Beispiel:

buffer_save(buff, "Player_Save.sav");

Speichert den aktuellen Inhalt des Puffers mit der in der Variablen "buff" gespeicherten ID in einer Datei.
