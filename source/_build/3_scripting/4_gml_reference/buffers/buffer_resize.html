Puffergröße

Syntax:

buffer_resize(buffer, newsize);


Streit Beschreibung
buffer Der Index des Puffers, um die Größe von zu ändern.
newsize Die neue Größe des Puffers (in Byte).


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie die Größe eines gegebenen Puffers auf die von Ihnen angegebene Größe (in Byte) ändern.


Beispiel:

if buffer_get_size(buff) < 16384
   {
   buffer_resize(buff, 16384);
   }

Der obige Code überprüft die Größe des Puffers, der in der Variablen "buff" indiziert ist, und wenn er kleiner als der angegebene Wert ist, wird die Größe des Puffers geändert.
