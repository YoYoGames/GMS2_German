Puffer_Größe von

Syntax:

buffer_sizeof(type);


Streit Beschreibung
type Die Art der Daten, die überprüft werden sollen (siehe Liste der Konstanten hier ).


Rückkehr: N / A


Beschreibung

Diese Funktion gibt die Größe (in Bytes) einer der angegebenen Datenkonstanten ( hier aufgelistet) zurück.


Beispiel:

var b = 12 * buffer_sizeof(buffer_u8);
buff = buffer_create(b, buffer_fixed, 1);

Der obige Code berechnet zuerst die Größe des zu erstellenden Puffers, indem er den vorzeichenlosen 8-Bit-Datentyp mit 12 multipliziert (da wir den Puffer für 12 Datenstücke verwenden), und verwendet dann diesen Wert, um einen festen Puffer auf den richtigen Wert zu setzen Größe.
