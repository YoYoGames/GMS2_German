Puffer_lesen

Syntax:

buffer_read(buffer, type)


Streit Beschreibung
buffer Der Index des Puffers, von dem gelesen werden soll.
type Die Art der Daten, die aus dem Puffer gelesen werden sollen (siehe Liste der Konstanten unten).


Liefert: Real / String


Beschreibung

Diese Funktion kann verwendet werden, um Daten aus einem zuvor erstellten Puffer zu lesen. Der Rückgabewert hängt von der Art der Daten ab, die Sie lesen, was an sich durch die folgenden Konstanten definiert ist:

Konstante Beschreibung
buffer_u8 Eine vorzeichenlose 8-Bit-Ganzzahl. Dies ist ein positiver Wert von 0 bis 255.
buffer_s8 Eine vorzeichenbehaftete 8-Bit-Ganzzahl. Dies kann ein positiver oder negativer Wert von -128 bis 127 sein (0 wird als positiv eingestuft).
buffer_u16 Eine vorzeichenlose 16-Bit-Ganzzahl. Dies ist ein positiver Wert von 0 - 65.535.
buffer_s16 Eine vorzeichenbehaftete 16-Bit-Ganzzahl. Dies kann ein positiver oder negativer Wert von -32.768 bis 32.767 sein (0 wird als positiv eingestuft).
buffer_u32 Eine vorzeichenlose 32-Bit-Ganzzahl. Dies ist ein positiver Wert von 0 bis 4.294.967.295.
buffer_s32 Eine vorzeichenbehaftete 32-Bit-Ganzzahl. Dies kann ein positiver oder negativer Wert von -2.147.483.648 bis 2.147.483.647 sein (0 wird als positiv eingestuft).
buffer_u64 Eine vorzeichenlose 64-Bit-Ganzzahl. Dies kann ein positiver oder negativer Wert von - (2 63 ) bis 2 63 - 1 sein.
buffer_f16 Ein 16-Bit-Float. Dies kann ein positiver oder negativer Wert im Bereich von +/- 65504 sein. ( Wird derzeit nicht unterstützt!)
buffer_f32 Ein 32-Bit-Float. Dies kann ein positiver oder negativer Wert im Bereich von +/- 16777216 sein.
buffer_f64 Ein 64-Bit-Float. Dies kann ein positiver oder negativer Wert von - (2 52 ) bis 2 52 - 1 sein.
buffer_bool Ein boolescher Wert. Kann nur 1 oder 0 sein ( true oder false )
buffer_string Eine Zeichenfolge beliebiger Größe.
buffer_text Eine Zeichenfolge beliebiger Größe ohne das abschließende abschließende Nullzeichen.

HINWEIS: Die Verwendung des falschen Datentyps für die gelesenen Daten führt zu fehlerhaften Werten!


Beispiel:

var cmd = buffer_read(buff, buffer_S16 );

Der obige Code liest aus dem Puffer mit der in der Variablen "buff" gespeicherten ID einen 16bit-Wert mit Vorzeichen in die lokale Variable "cmd".
