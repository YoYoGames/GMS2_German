Puffer_Write

Syntax:

buffer_write(buffer, type, value)


Streit Beschreibung
buffer Der Index des Puffers, in den geschrieben werden soll.
type Der Typ der Daten, die in den Puffer geschrieben werden sollen (siehe Liste der Konstanten unten).
value Die Daten zum Schreiben.


Rückkehr: N / A


Beschreibung

Diese Funktion kann verwendet werden, um Daten in einen zuvor erstellten Puffer zu schreiben. Die Daten, die Sie schreiben, müssen mit dem Argument "type" dieser Funktion übereinstimmen, was bedeutet, dass Sie beispielsweise nicht versuchen können, eine Zeichenkette als vorzeichenlose 16-Bit-Ganzzahl zu schreiben. Die folgenden Konstanten können verwendet werden, um den Datentyp zu definieren:

Konstante Beschreibung
buffer_u8 Eine vorzeichenlose 8-Bit-Ganzzahl. Dies ist ein positiver Wert von 0 bis 255.
buffer_s8 Eine vorzeichenbehaftete 8-Bit-Ganzzahl. Dies kann ein positiver oder negativer Wert von -128 bis 127 sein (0 wird als positiv eingestuft).
buffer_u16 Eine vorzeichenlose 16-Bit-Ganzzahl. Dies ist ein positiver Wert von 0 - 65.535.
buffer_s16 Eine vorzeichenbehaftete 16-Bit-Ganzzahl. Dies kann ein positiver oder negativer Wert von -32.768 bis 32.767 sein (0 wird als positiv eingestuft).
buffer_u32 Eine vorzeichenlose 32-Bit-Ganzzahl. Dies ist ein positiver Wert von 0 bis 4.294.967.295.
buffer_s32 Eine vorzeichenbehaftete 32-Bit-Ganzzahl. Dies kann ein positiver oder negativer Wert von -2.147.483.648 bis 2.147.483.647 sein (0 wird als positiv eingestuft).
buffer_u64 Eine vorzeichenlose 64-Bit-Ganzzahl. Dies kann ein positiver oder negativer Wert von - (2 63 ) bis 2 63 - 1 sein.
buffer_f16 Ein 16-Bit-Float. Dies kann ein positiver oder negativer Wert im selben Bereich wie eine vorzeichenbehaftete 16-Bit-Ganzzahl sein. (Derzeit nicht unterstützt!)
buffer_f32 Ein 32-Bit-Float. Dies kann ein positiver oder negativer Wert im selben Bereich wie eine vorzeichenbehaftete 32-Bit-Ganzzahl sein.
buffer_f64 Ein 64-Bit-Float. Dies kann ein positiver oder negativer Wert von - (2 63 ) bis 2 63 - 1 sein.
buffer_bool Ein boolescher Wert. Kann nur 1 oder 0 sein ( true oder false )
buffer_string Eine Zeichenfolge beliebiger Größe, die mit einem abschließenden Nullzeichen abgeschlossen wird.
buffer_text Eine Zeichenfolge beliebiger Größe ohne das abschließende abschließende Nullzeichen.


Beispiel:

buffer_seek(buff, buffer_seek_start, 0);
buffer_write(buff, buffer_s16, 0);
buffer_write(buff, buffer_s16, x);
buffer_write(buff, buffer_s16, y);

Der obige Code findet den Anfang des Puffers mit der ID, die in der Variablen "buff" gespeichert ist, und schreibt eine Reihe vorzeichenbehafteter 16-Bit-Integer-Werte an ihn.
