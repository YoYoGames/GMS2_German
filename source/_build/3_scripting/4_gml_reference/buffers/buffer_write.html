buffer_write

Syntax:

buffer_write(buffer, type, value)


Streit Beschreibung
buffer Der Index des Puffers, in den geschrieben werden soll.
type Der Datentyp, der in den Puffer geschrieben werden soll (siehe Liste der Konstanten unten).
value Die zu schreibenden Daten.


Rückgabe: N / A


Beschreibung

Mit dieser Funktion können Daten in einen zuvor erstellten Puffer geschrieben werden. Die Daten, die Sie schreiben, müssen mit dem Argument "type" dieser Funktion übereinstimmen. Dies bedeutet, dass Sie nicht versuchen können, eine Zeichenfolge als 16-Bit-Ganzzahl ohne Vorzeichen zu schreiben. Die folgenden Konstanten können zur Definition des Datentyps verwendet werden:

Konstante Beschreibung
buffer_u8 Eine vorzeichenlose 8-Bit-Ganzzahl. Dies ist ein positiver Wert von 0 bis 255.
buffer_s8 Eine vorzeichenbehaftete 8-Bit-Ganzzahl. Dies kann ein positiver oder negativer Wert von -128 bis 127 sein (0 wird als positiv eingestuft).
buffer_u16 Eine vorzeichenlose 16-Bit-Ganzzahl. Dies ist ein positiver Wert von 0 - 65.535.
buffer_s16 Eine vorzeichenbehaftete 16-Bit-Ganzzahl. Dies kann ein positiver oder negativer Wert von -32.768 bis 32.767 sein (0 wird als positiv eingestuft).
buffer_u32 Eine vorzeichenlose 32-Bit-Ganzzahl. Dies ist ein positiver Wert von 0 bis 4.294.967.295.
buffer_s32 Eine vorzeichenbehaftete 32-Bit-Ganzzahl. Dies kann ein positiver oder negativer Wert von -2.147.483.648 bis 2.147.483.647 sein (0 wird als positiv eingestuft).
buffer_u64 Eine 64-Bit-Ganzzahl ohne Vorzeichen.
buffer_f16 Ein 16-Bit-Float. Dies kann ein positiver oder negativer Wert innerhalb desselben Bereichs wie eine vorzeichenbehaftete 16-Bit-Ganzzahl sein. (Derzeit nicht unterstützt!)
buffer_f32 Ein 32-Bit-Float. Dies kann ein positiver oder negativer Wert innerhalb desselben Bereichs wie eine 32-Bit-Ganzzahl mit Vorzeichen sein.
buffer_f64 Ein 64-Bit-Float.
buffer_bool Ein boolescher Wert. Kann nur 1 oder 0 sein ( true oder false )
buffer_string Eine Zeichenfolge beliebiger Größe, die mit einem Null-Abschlusszeichen abgeschlossen wird.
buffer_text Eine Zeichenfolge beliebiger Größe ohne das abschließende Nullendezeichen.


Beispiel:

buffer_seek(buff, buffer_seek_start, 0);
buffer_write(buff, buffer_s16, 0);
buffer_write(buff, buffer_s16, x);
buffer_write(buff, buffer_s16, y);

Der obige Code findet den Start des Puffers, wobei die ID in der Variablen "buff" gespeichert ist. Sie schreibt eine Reihe von 16-Bit-Ganzzahlen mit Vorzeichen in den Puffer.
