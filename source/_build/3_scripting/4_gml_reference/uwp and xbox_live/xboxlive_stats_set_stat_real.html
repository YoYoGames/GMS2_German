xboxlive_stats_set_stat_real


Beschreibung

Mit dieser Funktion kann der Wert einer Statistik für die angegebene Benutzer-ID festgelegt werden. Sie geben die Benutzer-ID an, wie sie beispielsweise von der Funktion zurückgegeben wird xboxlive_get_user(), dann die Stat-Zeichenfolge, die gesetzt werden soll ( wenn die Stat-Zeichenfolge noch nicht vorhanden ist, wird eine neue Statistik erstellt und auf den angegebenen Wert gesetzt ) und einen Wert (eine reelle Zahl), um die Statistik auf festzulegen. Beachten Sie, dass der Statenname nur alphanumerisch sein darf und keine Symbole oder Leerzeichen enthalten darf.

Beim Festlegen des Statistikwerts wird jeder vorherige Wert überschrieben. Daher müssen Sie festlegen, ob der Statistikwert aktualisiert werden soll (dh prüfen, ob der Höchstwert tatsächlich der höchste Wert ist), indem Sie ihn mit dem aktuellen Statistikwert vergleichen mit dem neuen, bevor Sie es einstellen.

Die Funktion gibt -1 zurück, wenn ein Fehler aufgetreten ist oder die Benutzer-ID ungültig ist, oder einen anderen Wert, wenn die Funktion erfolgreich aufgerufen wurde.

WICHTIG! Diese Funktion ist nur beim Exportieren mit dem UWP-Ziel für die Xbox One gültig, und Sie müssen die Option XBox Live aktivieren in den UWP-Spieloptionen aktiviert haben.


Syntax:

xboxlive_stats_set_stat_real(user_id, stat, value);

Streit Beschreibung
user_id Der Benutzer-ID-Zeiger, für den die Statistik festgelegt werden soll
stat Die zu setzende Statistik (eine Zeichenfolge)
value Der Wert, auf den die Statistik gesetzt werden soll (ein Real)


Kehrt zurück:

Real


Beispiel:

if game_over == true
    {
    if xboxlive_stats_get_stat(p_user_id, "PercentDone") < 100
        {
        var _val = (global.LevelsFinished / global.LevelsTotal) * 100;
        xboxlive_stats_set_stat_real(p_user_id, "PercentDone", _val);
        }
    }

Der obige Code prüft eine Variable und ob es ist true Dann wird der Wert des Stat "PercentDone" überprüft. Ist dieser Wert kleiner als 100, wird ein Wert generiert und die Statistik auf diesen Wert gesetzt.
