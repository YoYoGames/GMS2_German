xboxlive_achievements_set_progress


Beschreibung

Mit dieser Funktion können Sie den Fortschritt einer Leistung aktualisieren. Sie geben die Benutzer-ID an, wie sie beispielsweise von der Funktion zurückgegeben wird xboxlive_get_user und dann die Erfolgszeichenfolge (dies ist die Leistungs-ID, die bei der Erstellung im XDP / Windows Dev Center zugewiesen wurde). Schließlich setzen Sie den Fortschrittswert auf einen Wert zwischen 0 und 100. Beachten Sie, dass das Leistungssystem Aktualisierungen ablehnt, die niedriger sind als der aktuelle Fortschrittswert.


Syntax:

xboxlive_achievements_set_progress(user_id, achievement, progress);

Streit Beschreibung
user_id Die Benutzer-ID des Benutzers, für den die Leistung festgelegt werden soll
achievement Die zu setzende Leistung (eine Zeichenfolge)
progress Der einzustellende Fortschrittswert (von 0 bis 100)


Kehrt zurück:

N/A


Erweitertes Beispiel:

Das folgende Beispiel zeigt, wie diese Funktion verwendet werden kann. Um mit zu beginnen, rufen Sie es bei einer Veranstaltung wie Room Start oder Create an:

xboxlive_stats_get_social_leaderboard(user_id, "GlobalTime", 20, 1, false, false);

Der obige Code würde aufgerufen, um eine Liste aller globalen Leaderboard-Positionen für das Spiel zu erhalten, und es wird ein Rückruf für soziales asynchrones Ereignis generiert, den wir auf folgende Weise behandeln würden:

var _progress = (global.Level / global.LevelMax) * 100;
xboxlive_achievements_set_progress(user_id, "Game_Completed", _progress);

Der obige Code erstellt einen Prozentwert basierend auf einigen globalen Variablen und verwendet diesen dann zum Festlegen des Leistungsfortschritts für die Leistung "Game_Completed".
