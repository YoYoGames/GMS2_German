xboxlive_get_activating_user


Beschreibung

Mit dieser Funktion können Sie den Benutzer-ID-Zeiger für den Benutzer abrufen, der das Spiel von der Konsole aus gestartet hat.

HINWEIS: Wie Sie diesen Zeiger verwenden, hängt davon ab, was Sie im Spiel tun möchten. Sie sollten sich jedoch nicht darauf verlassen, dass er die aktuell abgespielte Benutzer-ID ist, da sich der aktivierende Benutzer abmelden kann, während das Spiel läuft.
WICHTIG! Diese Funktion ist nur beim Exportieren mit dem UWP-Ziel für die Xbox One gültig, und Sie müssen die Option XBox Live aktivieren in den UWP-Spieloptionen aktiviert haben.


Syntax:

xboxlive_get_activating_user();


Kehrt zurück:

Pointer


Beispiel:

global.activUser = xboxlive_get_activating_user();

Das obige speichert den aktivierenden Benutzer-ID-Zeiger in einer globalen Variablen
