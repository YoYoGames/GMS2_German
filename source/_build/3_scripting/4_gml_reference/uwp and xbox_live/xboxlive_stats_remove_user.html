xboxlive_stats_remove_user


Beschreibung

Mit dieser Funktion kann ein bestimmter Benutzer aus dem Statistik-Manager entfernt (dessen Registrierung aufgehoben werden), indem die Statistikdaten auf den Live-Server geleert werden. Gemäß der XBox-Dokumentation muss das Spiel den Benutzer nicht aus dem Statistikmanager entfernen, da das XBox-Betriebssystem die Statistiken sowieso regelmäßig spült. Sie geben die Benutzer-ID an, wie sie beispielsweise von der Funktion zurückgegeben wird xboxlive_get_user(), und die Funktion gibt -1 zurück, wenn ein Fehler aufgetreten ist oder die Benutzer-ID ungültig ist, oder einen anderen Wert, wenn die Funktion erfolgreich aufgerufen wurde.

HINWEIS: Wenn Sie den Benutzer entfernen, kann ein Fehler zurückgegeben werden, wenn die für den Benutzer festgelegten Statistiken ungültig sind (z. B. die Statennamen, die nicht alphanumerische Zeichen enthalten).


Die Funktion generiert einen Rückruf, der ein asynchrones Systemereignis auslöst. Dieses Ereignis wird die spezielle DS-Karte haben async_load die dann für die folgenden Schlüssel analysiert werden:

  • " id " - Hält die Konstante achievement_stat_event

  • " eventname " - enthält die Zeichenfolge " localUserRemoved "

  • " userid " - Die der Anfrage zugeordnete Benutzer-ID

  • " error " - 0 wenn erfolgreich, ein anderer Wert, wenn ein Fehler aufgetreten ist

  • " errormessage " - Eine Zeichenfolge mit einer Fehlermeldung, falls vorhanden

Wenn Sie die Statistikdaten jederzeit auf den Live-Server leeren möchten, ohne den Benutzer zu entfernen, können Sie die Funktion verwenden xboxlive_stats_flush_user().

WICHTIG! Diese Funktion ist nur beim Exportieren mit dem UWP-Ziel für die Xbox One gültig, und Sie müssen die Option XBox Live aktivieren in den UWP-Spieloptionen aktiviert haben.


Syntax:

xboxlive_stats_remove_user(user_id);

Streit Beschreibung
user_id Die zu entfernende Benutzer-ID (ein Zeiger)


Kehrt zurück:

Real


Beispiel:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
    {
    user_id[i] = xboxlive_get_user(i);
    xboxlive_stats_remove_user(user_id[i]);
    }

Der obige Code durchläuft die gespeicherten Benutzerkonto-IDs und entfernt jeden Benutzer aus dem Statistik-Manager.
