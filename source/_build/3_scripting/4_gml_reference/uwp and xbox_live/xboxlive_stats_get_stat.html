xboxlive_stats_get_stat


Beschreibung

Mit dieser Funktion kann ein einzelner stat-Wert für einen bestimmten Benutzer vom stat-Manager abgerufen werden. Sie geben die Benutzer-ID an, wie sie beispielsweise von der Funktion zurückgegeben wird xboxlive_get_user() und dann die stat-Zeichenfolge, wie sie beim Erstellen mit der Zeichenfolge definiert wurde xboxlive_stats_set_stat_* Funktionen. Der Rückgabewert kann entweder ein String oder ein Real (abhängig von der zu überprüfenden Statistik) oder der GML-Konstante sein undefined wenn die stat nicht existiert oder ein fehler aufgetreten ist.

WICHTIG! Diese Funktion ist nur beim Exportieren mit dem UWP-Ziel für die Xbox One gültig, und Sie müssen die Option XBox Live aktivieren in den UWP-Spieloptionen aktiviert haben.


Syntax:

xboxlive_stats_get_stat(user_id, stat);

Streit Beschreibung
user_id Der Benutzer-ID-Zeiger, von dem die Statnamen abgerufen werden
stat Die zu setzende Statistik (eine Zeichenfolge)


Kehrt zurück:

Real, string or undefined


Beispiel:

if game_over == true
    {
    if xboxlive_stats_get_stat(p_user_id, "PercentDone") < 100
        {
        var _val = (global.LevelsFinished / global.LevelsTotal) * 100;
        xboxlive_stats_set_stat_real(p_user_id, "PercentDone", _val);
        }
    }

Der obige Code prüft eine Variable und ob es ist true Dann wird der Wert des Stat "PercentDone" überprüft. Ist dieser Wert kleiner als 100, wird ein Wert generiert und die Statistik auf diesen Wert gesetzt.
