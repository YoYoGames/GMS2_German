xboxlive_pad_count_for_user


Beschreibung

Mit dieser Funktion können Sie herausfinden, wie viele Pad- "Slots" für den aktuellen Benutzer aktiv sind (siehe xboxlive_pad_for_user() für weitere Details).

WICHTIG! Diese Funktion ist nur beim Exportieren mit dem UWP-Ziel für die Xbox One gültig, und Sie müssen die Option XBox Live aktivieren in den UWP-Spieloptionen aktiviert haben.


Syntax:

xboxlive_pad_count_for_user(user_id, slot);

Streit Beschreibung
user_id Die zu überprüfende Benutzer-ID (ein Zeiger)


Kehrt zurück:

Integer


Beispiel:

global.slots[0] = xboxlive_pad_count_for_user(user_id[0])

Der obige Code speichert die Anzahl der Pads, die dem angegebenen Benutzer-ID-Zeiger zugeordnet sind.
