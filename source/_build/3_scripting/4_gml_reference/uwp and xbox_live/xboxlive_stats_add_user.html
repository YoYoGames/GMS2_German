xboxlive_stats_add_user


Beschreibung

Mit dieser Funktion kann ein bestimmter Benutzer-ID-Zeiger zum Statistikmanager hinzugefügt werden. Dies muss geschehen, bevor Sie eine der anderen Statistikfunktionen verwenden, um das Spiel automatisch mit dem Xbox-Live-Server zu synchronisieren und die neuesten Werte abzurufen. Sie geben die Benutzer-ID an, wie sie beispielsweise von der Funktion zurückgegeben wird xboxlive_get_user(), und die Funktion gibt -1 zurück, wenn ein Fehler aufgetreten ist oder die Benutzer-ID ungültig ist, oder einen anderen Wert, wenn die Funktion erfolgreich aufgerufen wurde.

Die Funktion generiert einen Rückruf, der ein asynchrones Systemereignis auslöst. Dieses Ereignis wird die spezielle DS-Karte haben async_load die dann für die folgenden Schlüssel analysiert werden:

  • " id " - Hält die Konstante achievement_stat_event

  • " eventname " - enthält die Zeichenfolge " localUserAdded "

  • " userid " - Die der Anfrage zugeordnete Benutzer-ID

  • " error " - 0 wenn erfolgreich, ein anderer Wert, wenn ein Fehler aufgetreten ist

  • " errormessage " - Eine Zeichenfolge mit einer Fehlermeldung, falls vorhanden
WICHTIG! Diese Funktion ist nur beim Exportieren mit dem UWP-Ziel für die Xbox One gültig, und Sie müssen die Option XBox Live aktivieren in den UWP-Spieloptionen aktiviert haben.


Syntax:

xboxlive_stats_add_user(user_id);

Streit Beschreibung
user_id Die hinzuzufügende Benutzer-ID (ein Zeiger)


Kehrt zurück:

Real


Beispiel:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
    {
    user_id[i] = xboxlive_get_user(i);
    xboxlive_stats_add_user(user_id[i]);
    }

Der obige Code erhält die Anzahl der verfügbaren Benutzerkonten, durchläuft diese und führt die Konto-ID einem Array zu. Außerdem wird der Benutzer beim Statistik-Manager registriert.
