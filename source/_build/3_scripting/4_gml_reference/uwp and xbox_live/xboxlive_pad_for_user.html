xboxlive_pad_for_user


Beschreibung

Xbox-Benutzer können ihrer Benutzer-ID verschiedene Pads zuweisen, von denen jeder einen "Slot" füllt. Diese Slots sind von 0 aus nummeriert und können einen beliebigen Pad-Index enthalten, sodass Slot 0 dem Pad 3 und Slot 1 dem Pad 2 zugewiesen werden kann. Diese Funktion kann verwendet werden, um das Pad zu erhalten, das einem bestimmten Slot zugewiesen ist. Dies ist besonders nützlich, wenn ein Benutzer ein Gamepad für eine bestimmte Zeit ohne Eingabe lassen muss, da sich das Gerät ausschaltet. Sie können also jede Benutzer-ID mit dieser Funktion überprüfen, um sicherzustellen, dass sie über ein aktives Gamepad verfügen reagieren Sie nicht entsprechend, indem Sie beispielsweise Ihr Spiel pausieren. Wenn das Pad aktiv ist, wird sein Indexwert zurückgegeben. Ist dies nicht der Fall, wird die Funktion -1 zurückgeben.

Wenn ein Pad in einem Slot inaktiv wird, "rutschen" spätere Slots in die freie Position. Wenn Sie also drei aktive Pads den Slots 0, 1 und 2 zugewiesen haben und das Pad in Slot 0 inaktiv ist, sind die Slots 1 und to wird jetzt zu den Slots 0 und 1, daher sollte Slot 0 immer ein aktives Pad enthalten, wenn nicht alle Pads ausgeschaltet sind (in diesem Fall wird -1 zurückgegeben).

WICHTIG! Diese Funktion ist nur beim Exportieren mit dem UWP-Ziel für die Xbox One gültig, und Sie müssen die Option XBox Live aktivieren in den UWP-Spieloptionen aktiviert haben.


Syntax:

xboxlive_pad_for_user(user_id, slot);

Streit Beschreibung
user_id Die zu überprüfende Benutzer-ID (ein Zeiger)
user_id Der zu überprüfende Index (eine ganze Zahl) des Gamepad-Slots


Kehrt zurück:

Pointer


Beispiel:

if xboxlive_pad_for_user(user_id, 0) == -1
    {
    for(var i = 0; i < gamepad_get_device_count(); i++;)
        {
        var u_id = xboxlive_user_for_pad(i);
        if u_id == -1
            {
            async_ID = xboxlive_pad_for_user(i, u_id);
            }
        }
    }

Der obige Code prüft den Steckplatz 0 auf den angegebenen Benutzer-ID-Zeiger, um festzustellen, ob Gamepads zugewiesen wurden, und falls nicht, wird einer zugewiesen.
