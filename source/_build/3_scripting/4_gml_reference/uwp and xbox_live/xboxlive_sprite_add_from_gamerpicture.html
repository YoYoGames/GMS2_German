xboxlive_sprite_add_from_gamerpicture


Beschreibung

Mit dieser Funktion können Sie das Spielerbild für einen bestimmten Benutzer-ID-Zeiger erhalten. Die Funktion arbeitet asynchron und löst ein asynchrones Ereignis " Image Loaded" aus, um Sie darüber zu informieren, dass die Funktion abgeschlossen ist (wie bei HTML5 / URL-Aufrufen von) sprite_add() ). Beim Ereignis Asynchronous Image Loaded wird eine DS-Map in der eingebauten Variablen gespeichert async_load. Die Karte enthält folgende Informationen:

  • " Dateiname ": Der vollständige Pfad zur angeforderten Datei.
  • " id ": Die ID der Ressource, die Sie geladen haben. Dies entspricht der Variablen, der Sie die Ressource zugewiesen haben.
  • " status ": Gibt für einen Fehler einen Wert unter 0 zurück.
HINWEIS: Wenn Sie ein Sprite zur Laufzeit dynamisch in Ihr Spiel laden, müssen Sie es unbedingt mit entfernen sprite_delete() Wenn es nicht mehr benötigt wird, besteht ansonsten die Gefahr eines Speicherverlusts, der das Spiel verlangsamt und letztendlich zum Absturz führt.
WICHTIG! Diese Funktion ist nur beim Exportieren mit dem UWP-Ziel für die Xbox One gültig, und Sie müssen die Option XBox Live aktivieren in den UWP-Spieloptionen aktiviert haben.


Syntax:

xboxlive_sprite_add_from_gamerpicture(user_id, imagesize, xorig, yorig);

Streit Beschreibung
user_id Die Benutzer-ID (ein Zeiger), für die das Spielerbild abgerufen werden soll
imagesize Größe in Pixeln des zurückzugebenden Sprites
xorig Geben Sie die x-Position des Ursprungs im Sprite an
yorig Geben Sie die y-Position des Ursprungs im Sprite an


Kehrt zurück:

Real


Beispiel:

var _a = 0;
var _num = xboxlive_get_user_count();
for (var i = 0; i < _num; ++i;)
    {
    var _uid = xboxlive_get_user(i);
    if _uid != pointer_null
        {
        global.UserName[_a] = xboxlive_gamedisplayname_for_user(_uid);
        global.UserScore[_a] = xboxlive_gamerscore_for_user(_uid);
        global.UserRep[_a] = xboxlive_reputation_for_user(_uid);
        global.UserAvatar[_a] = xboxlive_sprite_add_from_gamerpicture(_uid, 256, 0, 0);
        ++a;
        }
    }

Der obige Code durchläuft die angemeldeten Benutzer und speichert ihre Spielerdaten in verschiedenen globalen Arrays.
