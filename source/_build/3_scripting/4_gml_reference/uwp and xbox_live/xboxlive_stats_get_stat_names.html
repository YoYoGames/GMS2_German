xboxlive_stats_get_stat_names


Beschreibung

Diese Funktion kann verwendet werden, um alle definierten Statistiken des Statistik-Managers für einen bestimmten Benutzer abzurufen. Sie geben die Benutzer-ID an, wie sie beispielsweise von der Funktion zurückgegeben wird xboxlive_get_user(), und die Funktion gibt ein Array von Strings mit den Statistiken für den Benutzer zurück. Wenn ein Fehler auftritt oder der Benutzer keine Statistiken hat, wird das Array weiterhin zurückgegeben, hat jedoch eine Elementanzahl von 0.

WICHTIG! Diese Funktion ist nur beim Exportieren mit dem UWP-Ziel für die Xbox One gültig, und Sie müssen die Option XBox Live aktivieren in den UWP-Spieloptionen aktiviert haben.


Syntax:

xboxlive_stats_get_stat_names(user_id);

Streit Beschreibung
user_id Der Benutzer-ID-Zeiger, von dem die Statnamen abgerufen werden


Kehrt zurück:

Array


Beispiel:

var _stat_str = xboxlive_stats_get_stat_names(user_id);
for (var i = 0; i < array_length_1d(_stat_str); ++i;)
    {
    xboxlive_stats_delete_stat(user_id, _stat_str[i]);
    }

Der obige Code ruft alle definierten Statistiken für den in der angegebenen Variablen gespeicherten Benutzer-ID-Zeiger ab, durchläuft dann das Array und löscht die angegebene Statistik aus dem Statistik-Manager.
