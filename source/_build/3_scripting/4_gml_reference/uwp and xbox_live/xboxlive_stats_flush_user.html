xboxlive_stats_flush_user


Beschreibung

Diese Funktion kann verwendet werden, um die Statistikdaten für einen bestimmten Benutzer vom Statistikmanager auf den Live-Server zu leeren, um sicherzustellen, dass der Server mit den aktuellen Werten auf dem neuesten Stand ist. Laut der XBox-Dokumentation sollten Entwickler darauf achten, dass dies nicht zu oft aufgerufen wird, da die XBox die Anforderungen auf ein Ratenlimit beschränkt und das XBox-Betriebssystem die Statistiken ohnehin automatisch etwa alle 5 Minuten löscht.

HINWEIS: Wenn Sie den Benutzer entfernen, kann ein Fehler zurückgegeben werden, wenn die für den Benutzer festgelegten Statistiken ungültig sind (z. B. die Statennamen, die nicht alphanumerische Zeichen enthalten).


Um die Funktion zu verwenden, geben Sie die Benutzer-ID ein, wie sie beispielsweise von der Funktion zurückgegeben wird xboxlive_get_user(), und die Funktion gibt -1 zurück, wenn ein Fehler aufgetreten ist oder die Benutzer-ID ungültig ist, oder einen anderen Wert, wenn die Funktion erfolgreich aufgerufen wurde. Die Funktion generiert auch einen Rückruf, der ein asynchrones Systemereignis auslöst. Dieses Ereignis enthält die spezielle DS-Map async_load, die für die folgenden Schlüssel analysiert werden sollte:

  • " id " - Hält die Konstante achievement_stat_event

  • " eventname " - enthält die Zeichenfolge " StatisticUpdateComplete "

  • " userid " - Die der Anfrage zugeordnete Benutzer-ID

  • " error " - 0 wenn erfolgreich, ein anderer Wert, wenn ein Fehler aufgetreten ist

  • " errormessage " - Eine Zeichenfolge mit einer Fehlermeldung, falls vorhanden

Beachten Sie, dass Statistiken automatisch gelöscht werden, wenn Sie einen Benutzer aus dem Statistik-Manager entfernen. Sehen xboxlive_stats_remove_user() für mehr Informationen.

WICHTIG! Diese Funktion ist nur beim Exportieren mit dem UWP-Ziel für die Xbox One gültig, und Sie müssen die Option XBox Live aktivieren in den UWP-Spieloptionen aktiviert haben.


Syntax:

xboxlive_stats_flush_user(user_id);

Streit Beschreibung
user_id Die zu verwendende Benutzer-ID (ein Zeiger)


Kehrt zurück:

Real


Beispiel:

for(var i = 0; i < xarray_length_1d(user_id); ++i;)
    {
    xboxlive_stats_flush_user(user_id[i]);
    }

Der obige Code durchläuft die gespeicherten Benutzerkonto-IDs und leert dann die Statistikdaten für jeden Benutzer.
