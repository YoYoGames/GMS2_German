xboxlive_stats_delete_stat


Beschreibung

Mit dieser Funktion kann eine Statistik für die angegebene Benutzer-ID aus dem Statistik-Manager gelöscht werden. Sie geben die Benutzer-ID an, wie sie beispielsweise von der Funktion zurückgegeben wird xboxlive_get_user() und dann die zu löschende stat-Zeichenfolge. Dadurch wird der stat-Wert gelöscht und aus dem stat-Manager entfernt, dh er wird von den Funktionen nicht mehr zurückgegeben xboxlive_stats_get_stat_names oder xboxlive_stats_get_stat.

Die Funktion gibt -1 zurück, wenn ein Fehler aufgetreten ist oder die Benutzer-ID ungültig ist, oder einen anderen Wert, wenn die Funktion erfolgreich aufgerufen wurde

WICHTIG! Diese Funktion ist nur beim Exportieren mit dem UWP-Ziel für die Xbox One gültig, und Sie müssen die Option XBox Live aktivieren in den UWP-Spieloptionen aktiviert haben.


Syntax:

xboxlive_stats_delete_stat(user_id, stat);

Streit Beschreibung
user_id Der Benutzer-ID-Zeiger, von dem die Statistik gelöscht werden soll
stat Die zu löschende Statistik (eine Zeichenfolge)


Kehrt zurück:

Real


Beispiel:

for(var i = 0; i < xboxlive_get_user_count(); i++;)
    {
    user_id[i] = xboxlive_get_user(i);
    xboxlive_stats_delete_stat(user_id[i], "HighScore");
    }

Der obige Code durchläuft die verbundenen Benutzer und löscht die angegebenen Stat für jeden der Stat Manager.
