uwp_is_suspending


Beschreibung

Diese Funktion kehrt zurück true für ein einzelnes Bild (und false ansonsten) auf ähnliche Weise wie os_is_paused(). Das Spiel hat dann eine Sekunde Zeit, um den Spielzustand zu speichern, bevor es möglicherweise vom System beendet wird. Sobald das Spiel mit dem Speichern der Daten fertig ist, sollte es anrufen uwp_suspend() sauber suspendieren. Das Spiel sollte wahrscheinlich auch einen Pausenzustand annehmen, sodass der Spieler, wenn er vom System wieder aufgenommen wird (wenn er nicht vollständig beendet wurde), nicht direkt in eine Spielsituation zurückgeworfen wird.


Syntax:

uwp_is_suspending();


Kehrt zurück:

Boolean


Beispiel:

if xboxliveis_suspending()
    {
    scr_Save_Game_Data();
    xboxlivesuspend();
    }

Der obige Code prüft, ob die App gesperrt ist und ob sie ein Skript zum Speichern der Spieldaten aufruft, bevor das Spiel ausgesetzt wird.
