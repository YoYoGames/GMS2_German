game_get_speed


Beschreibung

Diese Funktion kann verwendet werden, um die Spielgeschwindigkeit entweder als Anzahl von Spiel-Frames pro Sekunde oder als Anzahl von Mikro-Sekunden pro Spiel-Frame zu erhalten. Beachten Sie, dass dies nicht der tatsächliche FPS-Wert für die Fahrgeschwindigkeit ist fps_real Variable), sondern die Anzahl der Spiel-Frames (FPS), die das Spiel versuchen wird, jede Sekunde beizubehalten, oder die Länge jedes Spiel-Frames in Mikrosekunden, die das Spiel zu erhalten versuchen wird (MPF). Wenn Sie diese Funktion verwenden, müssen Sie eine der folgenden Konstanten angeben, die den Typ des Rückgabewerts bestimmen:

Konstante Beschreibung
gamespeed_fps Ruft die Spielgeschwindigkeit mit Bildern pro Sekunde ab.
gamespeed_microseconds Ruft die Spielgeschwindigkeit in Mikrosekunden pro Frame ab.


Wenn also beispielsweise die Spielgeschwindigkeit in den Spieloptionen auf 30 eingestellt ist und Sie den FPS-Typ verwenden, gibt die Funktion 30 zurück. Wenn Sie jedoch den MPF ​​verwenden, gibt die Funktion 33333 zurück.


Syntax:

game_get_speed(type);

Streit Beschreibung
type Die Art der Methode, die verwendet wird, um die Geschwindigkeit des Spiels zu erhalten (siehe obige Konstanten).


Kehrt zurück:

Real


Beispiel:

if game_get_speed(gamespeed_fps) != 60
    {
    game_set_speed(60, gamespeed_fps);
    }

Der obige Code überprüft, ob das Spiel mit einer Spielgeschwindigkeit von 60 FPS läuft und wenn nicht, wird es auf 60 FPS gesetzt.
