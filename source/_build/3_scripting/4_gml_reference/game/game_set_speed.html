game_set_speed


Beschreibung

Diese Funktion kann verwendet werden, um die Spielgeschwindigkeit einzustellen. Sie können dies auf zwei Arten festlegen - entweder als Game-Frames pro Sekunde (FPS) oder als Mikrosekunden pro Game-Frame (MPF) - mit einer der beiden folgenden Konstanten:

Konstante Beschreibung
gamespeed_fps Legt die Spielgeschwindigkeit mit Frames pro Sekunde fest.
gamespeed_microseconds Legt die Spielgeschwindigkeit in Mikrosekunden pro Bild fest.


Eine Spielgeschwindigkeit von 30 Bildern pro Sekunde wäre beispielsweise 33333 Mikrosekunden pro Spielrahmen, was dann durch diese Funktion ausgedrückt wird als entweder:

game_set_speed(30, gamespeed_fps);

oder:

game_set_speed(33333, gamespeed_microseconds);


Syntax:

game_set_speed(speed, type);

Streit Beschreibung
speed Die neue Spielgeschwindigkeit (entweder als FPS oder MPF).
type Die Art der Methode, mit der die Spielgeschwindigkeit eingestellt wird (siehe Konstanten oben).


Kehrt zurück:

N/A


Beispiel:

if os_browser == browser_not_a_browser
    {
    game_set_speed(60, gamespeed_fps);
    }
else
    {
    game_set_speed(30, gamespeed_fps);
    }

Der obige Code prüft, ob das Spiel in einem Browser ausgeführt wird, und stellt die Spielgeschwindigkeit entsprechend als FPS-Wert ein.
