game_load_buffer

Syntax:

game_load_buffer(buffer);


Streit Beschreibung
buffer Die Puffer-ID zum Laden von.


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie einen zuvor gespeicherten Spielstand laden. Das Spiel wird von einem zuvor erstellten "Grow" -Puffer geladen (siehe Puffer ) und der Puffer muss einen Spielstatus gespeichert haben, der verwendet wurde game_save_buffer Funktion.


Beispiel:

if keyboard_check_pressed(ord("L"))
   {
   if global.Checkpoint game_load_buffer(save_buff);
   }

Der obige Code lädt einen zuvor gespeicherten Spielstatus nur dann aus dem Puffer, der in der Variablen "save_buff" indiziert ist, wenn die globale Variable ist true, wenn der Spieler die Taste "L" drückt.
