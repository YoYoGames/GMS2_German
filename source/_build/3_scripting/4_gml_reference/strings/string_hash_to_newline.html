string_hash_to_newline


Beschreibung

Diese Funktion gibt eine Zeichenfolge zurück, bei der das Symbol "#" in eine neue Zeile konvertiert wurde. Also eine Zeichenfolge formatiert als:

"Hello#World"

Würde auf den Bildschirm gedruckt werden als:

Hello
World

WICHTIG! Diese Funktion ist für die Importkompatibilität zwischen GameMaker Studio 2 und früheren Versionen der Software vorgesehen. Daher sollte diese Funktion nicht verwendet werden, es sei denn, um veraltete Funktionen zu replizieren. Stattdessen sollten Sie die Escape-Zeichen verwenden, um eine neue Zeile zu markieren. Weitere Informationen zur Importkompatibilität finden Sie hier.


Syntax:

string_hash_to_newline(string);

Streit Beschreibung
string Die Zeichenfolge, die über mehrere Zeilen konvertiert werden soll


Kehrt zurück:

String


Beispiel:

var str = string_hash_to_newline("Hello#World");
draw_text(32, 32, str);

Der obige Code konvertiert die Zeichenfolge mit dem Hash-Symbol in eine Zeichenfolge, die auf zwei Zeilen aufgeteilt ist, und druckt sie dann auf dem Bildschirm aus.
