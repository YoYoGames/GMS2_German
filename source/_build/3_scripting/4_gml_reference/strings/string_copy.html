Zeichenkette


Beschreibung

Mit dieser Funktion können Sie einfach eine Anzahl von Zeichen innerhalb einer Zeichenfolge auswählen, die in eine andere Zeichenfolge kopiert werden soll. Das erste Zeichen in einer Zeichenfolge wird immer als "1" nummeriert, um z. B. die ersten fünf Zeichen einer Zeichenfolge zu kopieren string_copy(str, 1, 5). Ein weiteres Beispiel wird nachfolgend bereitgestellt.


Syntax:

string_copy(str, index, count);

Streit Beschreibung
str Die Zeichenfolge, von der kopiert werden soll.
index Die Position des ersten Zeichens in der zu kopierenden Zeichenfolge (nummeriert von 1)
count Die Anzahl der Zeichen, beginnend mit der Position des ersten, zum Kopieren


Kehrt zurück:

String


Beispiel:

name = keyboard_string;
if string_length(name) > 15 )
      {
      keyboard_string = string_copy(name, 1, 15 );
      }

Der obige Code ermöglicht es dem Spieler, eine Zeichenfolge über die Tastatur einzugeben, die dann in der Variablen gespeichert wird name. Diese Variable wird dann überprüft, um zu sehen, ob sie die maximale Länge von 15 Zeichen überschreitet, und wenn dies der Fall ist, wird der keyboard_string durch eine 15-stellige Kopie des name Zeichenfolge. Dies begrenzt effektiv die Spielereingabe auf 15 Zeichen.
