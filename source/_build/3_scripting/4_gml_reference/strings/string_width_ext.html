string_width_ext


Beschreibung

Diese Funktion gibt die maximale Breite (in Pixel) der Eingabezeichenfolge zurück, wobei die Zeilentrennung und die Zeilenumbruchbreite berücksichtigt werden (definiert als die Anzahl der Pixel, die die Zeichenfolge belegen kann, bevor ein Zeilenumbruch eingefügt wird). Es ist sehr praktisch für die Berechnung von Abständen zwischen Textelementen basierend auf der maximalen Breite eines Strings, der über mehrere Zeilen verteilt ist, so wie er gezeichnet würde draw_text_ext Verwenden Sie die aktuell definierte Schriftart. Trennung und Breite können auf -1 gesetzt werden, um den Standardabstand zu erhalten.


Syntax:

string_width_ext(string, sep, w);

Streit Beschreibung
string Die Zeichenfolge zum Messen der Breite von.
sep Der Abstand zwischen den Textzeilen in Pixeln, als ob die Zeichenfolge gezeichnet wurde.
w Die maximale Breite (in Pixel) der Zeichenfolge vor einem Zeilenumbruch, als ob die Zeichenfolge gezeichnet wurde.


Kehrt zurück:

Real


Beispiel:

var ww;
ww = string_width_ext(str_Story_Text[1], -1, 100);
draw_text_ext(32, 32, str_Story_Text[1], -1, 100);
draw_text_ext(32 + ww, 32, str_Story_Text[2], -1, 100);

Der obige Code wird die Breite der gegebenen Zeichenfolge unter Berücksichtigung der Zeilentrennung und der Zeilenumbrüche erhalten und dann zwei Textzeilen zeichnen, wobei die zurückgegebene gesamte Zeichenfolgenbreite als Trennzeichen verwendet wird.
