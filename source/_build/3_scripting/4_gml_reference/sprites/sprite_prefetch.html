sprite_prefetch


Beschreibung

Diese Funktion kann verwendet werden, um eine Texturseite mit dem angegebenen Sprite vorabzuholen (in den Texturspeicher zu platzieren). Sie geben den Sprite-Index so an, wie er beim Erstellen des Sprite-Assets definiert wurde, und die Textur-Seite, auf der er sich befindet, wird in den Speicher geladen. Beachten Sie, dass die Funktion zurückkehrt -1 Wenn für die ausgewählte Ressource kein Prefetch unterstützt wird oder die Zielplattform HTML5 ist, wird sie zurückgegeben 0 wenn alles richtig funktioniert hat.

ANMERKUNG: Es gibt einen Leistungsverlust, da die Textur auf den meisten Geräten in den Texturspeicher hochgeladen wird. Daher wird empfohlen, dass Sie anrufen sprite_prefetch auf einer beliebigen Grafik zu Beginn eines Levels, um Staus zu vermeiden.


Syntax:

sprite_prefetch(ind)

Streit Beschreibung
ind Der abzurufende Sprite-Index


Kehrt zurück:

Int (-1 or 0)


Beispiel:

sprite_prefetch(spr_Player_Aura);

Mit dem obigen Code wird das referenzierte Sprite im Texturspeicher einsatzbereit gespeichert.
