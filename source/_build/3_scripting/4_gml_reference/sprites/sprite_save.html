sprite_save


Syntax:

sprite_save(ind, subimg, fname);

Streit Beschreibung
ind Der Index des zu speichernden Sprites.
subimg Das Teilbild des zu speichernden Sprites.
fname Der Dateiname für das gespeicherte Sprite.


Kehrt zurück

N/A


Beschreibung

Diese Funktion kann verwendet werden, um ein beliebiges Teilbild eines beliebigen Sprites auf der Disc zu speichern und den angegebenen Dateinamen zu erhalten. Dieses Bild muss ein sein *.png Datei.

HINWEIS: Abhängig von der ausgewählten Zielplattform sind Sie eingeschränkt, wo Sie Dateien speichern und laden können. Weitere Informationen finden Sie unter Dateiverarbeitung.


Beispiel:

var surf, spr_custom;
surf = surface_create(32, 32);
surface_set_target(surf);
draw_clear_alpha(c_black, 0);
draw_sprite(spr_Body, 0, 0, 0);
draw_sprite(spr_Clothes, 0, 0, 0);
draw_sprite(spr_Hair, 0, 0, 0);
spr_custom = sprite_create_from_surface(surf, 0, 0, 32, 32, true, true, 16, 16);
surface_reset_target();
surface_free(surf);
sprite_save(spr_custom, 0, "Player_Custom_Sprite.png");
sprite_delete(spr_Custom);

Der obige Code erstellt eine Oberfläche und speichert ihren Index in der lokalen Variablen "surf". Er zielt dann auf diese Oberfläche, löscht sie und zieht mehrere Sprites übereinander. Er erstellt ein neues Sprite aus dem auf der Oberfläche gezeichneten zusammengesetzten Bild und weist seinen Index der lokalen Variablen "spr_Custom" zu, bevor er den von der Oberfläche verwendeten Speicher freigibt. Schließlich wird das resultierende Sprite in einer Datei gespeichert und dann ebenfalls aus dem Speicher entfernt.
