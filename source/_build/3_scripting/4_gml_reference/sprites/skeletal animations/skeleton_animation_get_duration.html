skeleton_animation_get_duration


Syntax:

skeleton_animation_get_duration(animname);

Streit Beschreibung
animname Der Name (eine Zeichenfolge) des zu verwendenden Animationssatzes.


Kehrt zurück:

Real


Beschreibung

Diese Funktion gibt die Zeit zurück, die benötigt wird, damit der angegebene Animationssatz ausgeführt wird, bevor zum Anfang zurückgespult wird. Der Rückgabewert ist in Sekunden.

WICHTIG! Diese Funktion ist mit der Testlizenz des Produkts nicht verfügbar.


Beispiel 1:

time += delta_time / 1000000;
var duration = skeleton_animation_get_duration(skeleton_animation_get());
var frame = floor((image_number * (mTime / duration)) + 0.5) % image_number;
image_index = frame;
draw_self();

Der obige Code wird die image_index auf den richtigen Wert für das aktuell zugewiesene Skeleton-Animations-Sprite.

Beispiel2:

time += delta_time / 1000000;
var d = skeleton_animation_get_duration("walk");
if time > d time -= d;
draw_skeleton_time(sprite_index, "walk", "skin1", time, x, y, image_xscale, image_yscale, image_angle, c_white);

Der obige Code wird den gegebenen Skeleton-Animations-Sprite unter Verwendung der Delta-Zeit zeichnen, um den zu zeichnenden Rahmen festzulegen.
