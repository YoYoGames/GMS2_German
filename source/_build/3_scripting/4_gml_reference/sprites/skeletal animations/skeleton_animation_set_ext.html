skeleton_animation_set_ext


Syntax:

skeleton_animation_set_ext(animname, track);

Streit Beschreibung
animname Der Name (eine Zeichenfolge) des zu verwendenden Animationssatzes.
track Die Spur für das verwendete Animationssatz.


Kehrt zurück:

N/A


Beschreibung

Wenn Sie einen Animationssatz für Ihr Skeleton-Animations-Sprite festlegen, wird intern Spur 0 zugewiesen. Sie können jedoch weitere Sets für weitere Tracks (1 und höher) zuweisen und auf diese Weise Ihr Sprite zwei (oder mehr) Animationen gleichzeitig abspielen lassen. Beachten Sie, dass das Sprite in diesem Fall in Ihrem Animationsprogramm entworfen sein sollte, sodass die "Basis" -Animation in Spur 0 nur Keyframes für bestimmte Bones hat, während die nächsten Tracks Keyframes für die anderen enthalten. Wenn Sie zum Beispiel eine "walk" -Animation haben und ihre Keyframes nur die Beine des Sprites betreffen, können Sie eine Reihe von Kopfanimationen und Armanimationen haben, die Sie jeweils in den Spuren 1 und 2 einstellen können Grundbewegung und passen Sie sich während des Spiels in Echtzeit an.

WICHTIG! Diese Funktion ist mit der Testlizenz des Produkts nicht verfügbar.


Beispiel:

if mouse_check_button(mb_left)
   {
   skeleton_animation_set_ext("bodyfight", 1);
   }
else
   {
   skeleton_animation_set_ext("bodywalk", 1);
   }

Der obige Code ändert das Animationsset, das von Track 1 verwendet wird, in den "Bodyfight" -Satz oder den "Bodywalk" -Satz, wenn die linke Maustaste gedrückt wird.
