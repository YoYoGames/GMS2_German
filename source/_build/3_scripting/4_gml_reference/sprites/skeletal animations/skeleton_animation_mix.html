skeleton_animation_mix


Syntax:

skeleton_animation_mix(animfrom, animto, duration);

Streit Beschreibung
animfrom Der Name (eine Zeichenfolge) der ersten Animation, von der interpoliert wird.
animto Der Name (eine Zeichenfolge) der zweiten Animation, auf die interpoliert wird.
duration Die Dauer der Interpolation (von 0 bis 1)


Kehrt zurück:

N/A


Beschreibung

Sie können die Animationssätze einfach mit der Taste wechseln skeleton_animation_set Funktion, aber dies kann zu einem Überspringen oder Stottern führen, wenn eine Animation gegen eine andere ausgetauscht wird. Um dies zu verhindern, können Sie den Mischungswert zwischen zwei Animationssätzen einstellen und das Sprite interpoliert zwischen diesen. normalerweise würden Sie dies im Create Event der Instanz mit der Skelettanimation tun, da es nur einmal gesetzt werden muss, und GameMaker Studio 2 interpoliert alle weiteren Änderungen an dem Sprite unter Verwendung der Animationssätze in dieser Instanz. Beachten Sie, dass der Wert für die Dauer von 0 bis 1 ist, wobei ein Wert von 0,5 eine "halbe und eine halbe" Interpolation von einem Satz zum anderen hätte.

WICHTIG! Diese Funktion ist mit der Testlizenz des Produkts nicht verfügbar.


Beispiel:

skeleton_animation_set("walk");
skeleton_animation_mix("walk", "jump", 0.2);
skeleton_animation_mix("jump", "walk", 0.4);

Der obige Code würde in das Create Event einer Instanz mit einem Skeleton-Animations-Sprite gehen und die Animationsmix-Dauer für die Interpolation zwischen den beiden Animations-Sets "walk" und "jump" einstellen.
