skeleton_animation_list


Syntax:

skeleton_animation_list(sprite, list);

Streit Beschreibung
sprite Der Sprite-Index der Spine-Skelettanimation, von der die Liste abgerufen werden soll.
list Die ID der DS-Liste, die mit den Animationsnamen gefüllt werden soll.


Kehrt zurück:

N/A


Beschreibung

Mit dieser Funktion können Sie ein (vordefiniertes) ds_list mit allen Namen der Animationen, die als Teil des Skelettanimations-Sprites enthalten sind. Die Namen sind Strings und können dann in den anderen Animationsfunktionen für diese Sprite-Typen verwendet werden.

WICHTIG! Diese Funktion ist mit der Testlizenz des Produkts nicht verfügbar.


Beispiel:

var list = ds_list_create();
skeleton_animation_list(sprite_index, list);
for (var i = 1; i < ds_list_size(list); i++;)
   {
   skeleton_animation_mix(list[| 0], list[| i], 0.5);
   }
ds_list_destroy(list);

Der obige Code erstellt ein ds_list aller Animationsnamen für das Sprite, das von der Instanz verwendet wird. Dann durchläuft es diese und legt den Mischwert für alle mit der ersten Animation auf 0,5 fest.
