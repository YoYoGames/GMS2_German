skeleton_animation_get_frame


Syntax:

skeleton_animation_get_frame(track);

Streit Beschreibung
track Die Animationsspur, um den Rahmen zu erhalten.


Kehrt zurück:

Integer


Beschreibung

Diese Funktion gibt die Bildnummer der Animation auf der angegebenen Animationsspur zurück. Die Funktion gibt -1 zurück, wenn der angegebenen Spur keine Animation zugewiesen wurde.

WICHTIG! Diese Funktion ist mit der Testlizenz des Produkts nicht verfügbar.


Beispiel:

var frame = skeleton_animation_get_frame(0);
if frame > 0 && frame < 2
    {
    var box = skeleton_get_minmax();
    var ww = (box[2] - box[0]) / 2;
    var hh = (box[3] - box[1]) / 2;
    part_particles_create(global.p_sys, box[0] + ww, box[1] + hh, global.Stars, 10);
    }

Der obige Code prüft das aktuelle Bild der Animation, die Spur 0 zugeordnet ist, und platzt dann einige Partikel von einem Punkt in der Mitte der gesamten Begrenzungsbox für das Sprite.
