skeleton_animation_set


Syntax:

skeleton_animation_set(animname);

Streit Beschreibung
animname Der Name (eine Zeichenfolge) des zu verwendenden Animationssatzes.


Kehrt zurück:

N/A


Beschreibung

Sobald Sie Ihrer Instanz ein Skeleton-Animations-Sprite zugewiesen haben, können Sie mit dieser Funktion festlegen, welcher Animations-Set zu einem bestimmten Zeitpunkt verwendet werden soll. Wenn Sie Ihr Sprite (in Spine) erstellt haben, haben Sie mehrere Animationssets für verschiedene Zustände definiert und jedes von ihnen entsprechend benannt. Anhand dieser Namen können Sie GameMaker Studio 2 mitteilen, welchen Animationssatz Sie in Ihrem Spiel verwenden möchten. Wenn dein Sprite zum Beispiel für ein Plattformspiel gedacht ist, kannst du einen "Sprung" -, einen "Lauf" - und einen "Leerlauf" -Satz haben, die alle in dem einen Sprite enthalten sind, und du kannst zwischen ihnen in deinem Spiel wechseln Funktion.

Beachten Sie, dass das Setzen einer neuen Animation auf diese Weise das Zurücksetzen von image_index auf 0, startet den neuen Animationssatz vom Anfang seiner Animationsschleife. Wenn dies nicht das gewünschte Verhalten ist, können Sie einen "Misch" -Wert mit verwenden skeleton_animation_mix und wenn Sie eine neue Animation einstellen, wird diese nahtlos mit dem vorherigen Satz interpoliert. Sie sollten außerdem sicherstellen, dass zu Beginn jeder Animation ein Keyframe vorhanden ist, um zu verhindern, dass Daten von früheren Animationen übernommen werden, es sei denn, Sie planen separate Animationsspuren.

WICHTIG! Diese Funktion ist mit der Testlizenz des Produkts nicht verfügbar.


Beispiel:

if keyboard_check_pressed(vk_space)
   {
   if skeleton_animation_get() != "jump"
      {
      skeleton_animation_set("jump");
      }
   }

Der obige Code ändert das verwendete Animationssatz zu dem "Sprung" -Eintrag, wenn die Leertaste gedrückt wird, aber nur, wenn der aktuelle Satz nicht bereits "springen" ist.
