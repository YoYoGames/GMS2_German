sprite_exists


Syntax:

sprite_exists(index);

Streit Beschreibung
index Der Index des zu überprüfenden Sprites.


Kehrt zurück

N/A


Beschreibung

Diese Funktion gibt zurück, ob ein Sprite mit dem angegebenen Index existiert oder nicht. Bitte beachten Sie, dass der für die Überprüfung verwendete Wert zuvor initialisiert sein muss, da sonst ein Fehler auftreten kann, der zum Schließen von GameMaker Studio 2: Studio führt.


Beispiel:

if sprite_exists(spr_array[0])
   {
   sprite_index = spr_array[0];
   }
else
   {
   sprite_index = spr_BaseSprite;
   }

Der obige Code überprüft ein Array, um zu sehen, ob es einen gültigen Sprite-Index enthält, und wenn dies der Fall ist, weist es das Sprite der Instanz zu, wenn nicht, weist es ein Sprite aus den enthaltenen Ressourcen zu.
