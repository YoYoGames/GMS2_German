sprite_save_strip


Syntax:

sprite_save_strip(ind, filename);

Streit Beschreibung
ind Der Index des Sprites zum Ändern der Transparenzkarte von.
filename Der Dateiname für den gespeicherten Sprite-Strip.


Kehrt zurück

N/A


Beschreibung

Diese Funktion erstellt ein Streifenbild aller Unterbilder von einem Sprite. Sie können den Namen der zu speichernden Datei angeben, aber der tatsächliche Dateityp muss lauten *.png.

HINWEIS: Abhängig von der ausgewählten Zielplattform sind Sie eingeschränkt, wo Sie Dateien speichern und laden können. Weitere Informationen finden Sie unter Dateiverarbeitung.


Beispiel:

var spr, i;
spr = sprite_create_from_surface(0, 0, 32, 32, true, true, 16, 16);
for (i = 1; i < 8; i +=1)
   {
   sprite_add_from_surface(spr, i, 0, 32, 32, true, true, 16, 16);
   }
sprite_save_strip(spr, "Custom_Player_Sprite.png");
sprite_delete(spr);

Der obige Code erstellt ein Sprite aus der Oberfläche, die in der Variablen "surf" indiziert ist, und weist seinen Index der lokalen Variablen "spr_Custom" zu und verwendet dann a for Schleife, um sich über die Oberfläche zu bewegen und verschiedene Abschnitte aufzunehmen, die als Teilbilder in das Sprite eingefügt werden. Dieses neue Sprite wird dann als Png-Strip gespeichert, bevor es aus dem Speicher entfernt wird.
