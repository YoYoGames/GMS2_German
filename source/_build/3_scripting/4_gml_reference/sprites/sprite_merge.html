sprite_merge


Syntax:

sprite_merge(ind1, ind2);

Streit Beschreibung
ind1 Der Index des zu vermischenden Sprites.
ind2 Der Index des Sprites, mit dem ind1 verbunden werden soll.


Kehrt zurück

N/A


Beschreibung

Diese Funktion mischt das in Argument1 ("ind2") indizierte Sprite mit dem in Argument0 ("ind1") indizierten Sprite zusammen. Die Bilder selbst werden NICHT zusammengefügt, sondern die Bildindizes werden zusammengefügt, wobei die Unterbilder des Sprites "ind2" an die des Sprites "ind1" angehängt werden, dh sie werden am Ende hinzugefügt. Wenn die Sprites unterschiedliche Größen haben, werden die angehängten Sprites auf die Bildgröße für "ind1" gestreckt.

Diese Änderung ist dauerhaft. Von dem Moment an, an dem Sie diese Funktion verwenden, bis das Spiel geschlossen oder das Sprite gelöscht wird, wird das Sprite, in das ein Teil eingefügt wird, geändert. Das Sprite, mit dem es zusammengefügt wird, bleibt jedoch gleich.

HINWEIS: Weder das Image-Asset, in das ein Bild eingefügt wird, noch das Image-Asset, das zusammengefügt werden soll, kann eine der ursprünglichen Spielressourcen sein. Du musst benutzen sprite_duplicate Erstellen Sie zuerst zwei neue Ressourcen (wenn Sie die enthaltenen Spielelemente zusammenführen), und führen Sie diese dann zusammen.
HINWEIS: Diese Funktion ist nur für Bitmap- Sprites nützlich und funktioniert nicht mit SWF- oder JSON-Sprites (Spine).


Beispiel:

var tspr;
tspr[0] = spr_add(working_directory + "Sprite Assets\explode1.png", 16, true, true, 0, 0);
tspr[1] = sprite_duplicate(spr_Explosion);
sprite_merge(tspr[0], tspr[1]);
sprite_index = t_spr[0];
sprite_delete(tspr[1]);

Mit dem obigen Code wird ein Sprite in eine lokale Variable geladen und dann mit einem anderen Sprite zusammengeführt, das dupliziert wurde. Schließlich wird das zusammengeführte Sprite der Instanz zugewiesen sprite_index während das geladene Sprite entfernt wird, um den verwendeten Speicher freizugeben. Beachten Sie, dass Sie irgendwann nach dieser Operation, wie bei den Destroy- oder Room End-Ereignissen, auch verwenden müssen sprite_delete um das zusammengeführte Sprite aus dem Speicher zu entfernen.
