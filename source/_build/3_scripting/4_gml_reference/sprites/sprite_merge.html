sprite_merge


Syntax:

sprite_merge(ind1, ind2);

Streit Beschreibung
ind1 Der Index des zu mischenden Sprites.
ind2 Der Index des Sprites, mit dem ind1 zusammengeführt werden soll.


Kehrt zurück

N/A


Beschreibung

Diese Funktion wird das in argument1 ("ind2") indexierte Sprite mit dem in argument0 ("ind1") indexierten Sprite zusammenführen. Die Bilder selbst werden NICHT miteinander verschmolzen, sondern die Bildindizes werden zusammengeführt, wobei die Unterbilder von Sprite "ind2" an diejenigen des Sprites "ind1" angehängt werden, dh: sie werden am Ende hinzugefügt. Wenn die Sprites unterschiedliche Größen haben, werden die angehängten Sprites entsprechend der Bildgröße für "ind1" gestreckt.

Diese Änderung ist dauerhaft, und von dem Moment an, an dem Sie diese Funktion verwenden, bis das Spiel geschlossen oder das Sprite gelöscht wird, wird das Sprite, in das es eingefügt wird, geändert, das Sprite, mit dem es verschmolzen wird, bleibt jedoch gleich.

HINWEIS: Weder das Bild-Asset, mit dem es zusammengeführt wird, noch das Bild-Asset, das zum Zusammenführen vorgesehen ist, kann eine der ursprünglichen Spiel-Ressourcen sein. Sie müssen verwenden sprite_duplicate Erstellen Sie zunächst zwei neue Ressourcen (wenn Sie eingeschlossene Spielelemente zusammenführen) und fügen Sie diese dann zusammen.
HINWEIS: Diese Funktion ist nur für Bitmap- Sprites und nicht für SWF- oder JSON (Spine) -Sprites geeignet.


Beispiel:

var tspr;
tspr[0] = spr_add(working_directory + "\Sprite Assets\explode1.png", 16, true, true, 0, 0);
tspr[1] = sprite_duplicate(spr_Explosion);
sprite_merge(tspr[0], tspr[1]);
sprite_index = t_spr[0];
sprite_delete(tspr[1]);

Der obige Code lädt ein Sprite in eine lokale Variable und führt es dann mit einem anderen Sprite zusammen, das dupliziert wurde. Abschließend wird das zusammengefügte Sprite der Instanz zugewiesen sprite_index während das geladene Sprite entfernt wird, um den von ihm verwendeten Speicher freizugeben. Beachten Sie, dass Sie zu einem bestimmten Zeitpunkt nach diesem Vorgang, wie in den Destroy- oder Room-End-Ereignissen, ebenfalls verwenden müssen sprite_delete um das zusammengeführte Sprite auch aus dem Speicher zu entfernen.
