sprite_set_bbox


Beschreibung

Mit dieser Funktion können Sie die Begrenzungsrahmenwerte für ein Sprite festlegen. Sie geben den zu verwendenden Sprite-Index und dann die Werte links, oben, rechts und unten für die Begrenzungsrahmenpositionen an. Die Positionen sind absolute Werte, wobei die Position (0, 0) der oberen linken Ecke des Sprites entspricht, unabhängig vom Versatz des Sprites, etwaigen "leeren" Pixeln, die das Sprite möglicherweise hat, oder wo es im Sprite gezeichnet wird Zimmer.

ANMERKUNG: Diese Funktion wirkt sich auf die Sprite- Ressource aus, sodass alle weiteren Instanzen mit diesem Sprite denselben Begrenzungsrahmen haben.


Syntax:

sprite_set_bbox(ind, left, top, right, bottom);

Streit Beschreibung
ind Der Index des Sprites, für den der Begrenzungsrahmen gesetzt werden soll.
left Die linke Seite des Begrenzungsrahmens
top Die Oberseite des Begrenzungsrahmens.
right Die rechte Seite des Begrenzungsrahmens
bottom Der Boden des Begrenzungsrahmens.


Kehrt zurück

N/A


Beispiel:

if sprite_get_bbox_mode(sprite_index) == 2
    {
    left = irandom(sprite_width / 2);
    right = irandom((sprite_width / 2) + irandom(sprite_width / 2));
    top = irandom(sprite_height / 2);
    bottom = irandom((sprite_height / 2) + irandom(sprite_height / 2));
    sprite_set_bbox(sprite_index, left, top, right, bottom);
    }

Der obige Code überprüft den Begrenzungsrahmenmodus des Sprites, das dem zugewiesen ist sprite_index und wenn es eingestellt ist manual Dann wird der Begrenzungsrahmen geändert.
