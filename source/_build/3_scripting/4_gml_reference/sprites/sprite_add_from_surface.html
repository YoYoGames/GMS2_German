sprite_add_from_surface


Syntax:

sprite_add_from_surface(index, surface, x, y, w, h, removeback, smooth);

Streit Beschreibung
index Der Index des Sprites, zu dem das neue Bild hinzugefügt werden soll.
surface Der Index der Oberfläche, von der das Bild kommt.
x Die x-Position, aus der kopiert werden soll.
y Die y-Position, aus der kopiert werden soll.
w Die Breite des zu kopierenden Bereichs (von der x-Position).
h Die Höhe des zu kopierenden Bereichs (von der y-Position).
removeback Gibt an, ob alle Pixel mit der Hintergrundfarbe (linkes unteres Pixel) transparent gemacht werden sollen.
smooth Gibt an, ob die Kanten geglättet werden sollen.


Kehrt zurück:

N/A


Beschreibung

Diese Funktion funktioniert genauso wie sprite_create_from_surface Nur statt eines neuen Sprites aus dem Bereich der indizierten Oberfläche, die Sie auswählen, wird der definierte Bereich der Oberfläche als neues Unterbild zu einem zuvor erstellten Sprite hinzugefügt.

WICHTIG! Diese Funktion ist mit der Testlizenz des Produkts nicht verfügbar.


Beispiel:

spr_custom = sprite_create_from_surface(surf, 0, 0, 32, 32, true, true, 16, 16);
var i;
for (i = 1; i < 8; i +=1)
   {
   sprite_add_from_surface(spr_Custom, surf, i, 0, 32, 32, true, true, 16, 16);
   }

Der obige Code erstellt ein Sprite aus der Oberfläche, die in der Variablen "surf" indiziert ist, weist seinen Index der Variablen "spr_Custom" zu und verwendet dann a for Schleife, um sich über die Oberfläche zu bewegen und verschiedene Abschnitte aufzunehmen, die als Teilbilder in das Sprite eingefügt werden.
