Sprites


Sprites sind im Allgemeinen die visuellen Darstellungen von Objekten innerhalb des Spiels, das Sie erstellen, obwohl sie für viele andere Zwecke verwendet werden können. Ein Sprite ist entweder ein einzelnes Bild oder eine Reihe von Bildern, die, wenn sie nacheinander abgespielt werden, wie eine Loop-Animation aussehen. Sprites können in vielen Formaten erstellt werden, wobei das "traditionelle" Sprite aus Frames von Pixelbildern besteht, die in GameMaker Studio 2 importiert werden *.png Dateien. Sie können jedoch auch Vektor - Sprites importieren, die in der *.swf Format oder Skeleton Animation Sprites, die als gespeichert wurden *.json Dateien mit begleitenden *.atlas und *.png Dateien (weitere Informationen zu diesen Arten von Bilddateien und zum Import in GameMaker Studio 2 Sie im Abschnitt Importieren von Nicht-Bitmap-Sprites ).

Da Sprites ein integraler Bestandteil jedes Spiels sind, verfügt GameMaker Studio 2 über eine große Anzahl von Funktionen, die dazu dienen, Informationen über bestimmte Sprites zu erhalten oder diese in Ihrem Spiel zu ändern oder sogar Ihre eigenen Sprites in eine Anwendung zu laden aus einer Quellbilddatei.

Die folgenden Funktionen können verwendet werden, um Informationen zu einem Sprite-Asset abzurufen. Bitte beachten Sie, dass diese Funktionen die Informationen aus dem Basis- Sprite (dh dem Sprite-Asset) und nicht aus dem Sprite erhalten, wie es in einer Instanz verwendet wird. Sie können beispielsweise die Instanzvariable verwenden sprite_width um die Breite des Sprites zu ermitteln, das eine Instanz verwendet, und diese Breite berücksichtigt alle Skalierungen, die möglicherweise angewendet wurden. Wenn Sie jedoch die Breite eines Sprites normal kennen müssen, ohne dass Transformationen angewendet werden, müssen Sie die Option verwenden sprite_get_width() Funktion unten. Weitere Informationen über das Sprite sehen Instanzvariablen im Abschnitt Instanzvariablen.

Das bedeutet auch, dass Sie vor dem Verweisen darauf achten müssen, ob ein Sprite vorhanden ist, oder ob vor dem Ändern darauf verwiesen wird.
