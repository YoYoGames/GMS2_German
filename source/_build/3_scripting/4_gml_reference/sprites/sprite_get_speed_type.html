sprite_get_speed_type


Syntax:

sprite_get_speed_type(index)

Streit Beschreibung
index Der Index des Sprites, um den Animationstyp zu erhalten


Kehrt zurück:

Constant (see above)


Beschreibung

Mit dieser Funktion können Sie den Sprite-Animationstyp abrufen, der für die Sprite-Ressource im Sprite-Editor definiert wurde. Der zurückgegebene Wert ist eine der folgenden Konstanten:

Konstante Beschreibung
spritespeed_framespersecond Das Sprite wurde mit Animation in Frames pro Sekunde definiert.
spritespeed_framespergameframe Das Sprite wurde in Animation in Frames pro Spielrahmen definiert.


Die folgenden zwei Beispiele veranschaulichen den Unterschied, den dies beim Berechnen von Animationsgeschwindigkeiten für das Sprite machen kann:

  • Wenn Sie ein Sprite haben, das 1 Bild pro Sekunde zeichnet und die Bildgeschwindigkeit auf 0,5 setzt, wird es mit 0,5 Bildern pro Sekunde gezeichnet. Wenn die Frame-Rate Ihres Spiels 60 Frames pro Sekunde beträgt, zeichnet das Sprite 1 Frame pro 120 Spiel-Frames.

  • Wenn Sie einen Sprit haben, die 1 Bild pro Spiel Rahmen zieht und stellen Sie die Bildgeschwindigkeit auf 0,5 wird es 0,5 Bilder pro Spiel Rahmen ziehen. Wenn die Frame-Rate Ihres Spiels 60 Frames pro Sekunde beträgt, zeichnet das Sprite 30 Frames pro 60 Spiel-Frames.

Sie können herausfinden, welcher Geschwindigkeitswert für die Sprite-Animation die Funktion verwendet hat sprite_get_speed und Sie können die Animationsgeschwindigkeit und -art mit der Funktion einstellen sprite_set_speed.


Beispiel:

s_speed = sprite_get_speed(sprite_index);
s_type = sprite_get_speed_type(sprite_index);

Der obige Code erhält die Sprite-Geschwindigkeit und den Sprite-Animationstyp und speichert sie in Instanzvariablen für die zukünftige Verwendung.
