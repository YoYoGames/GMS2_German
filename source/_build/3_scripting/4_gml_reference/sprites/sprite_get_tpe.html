sprite_get_tpe


Syntax:

sprite_get_tpe(sprite, index);

Streit Beschreibung
sprite Der Index des Sprites, um den Texturseiteneintrag von zu finden.
index Der Bildindex des Sprites.


Kehrt zurück

Real


Beschreibung

Diese Funktion gibt einen speziellen Wert zurück, der sich auf die Position des gegebenen Sprites und Unterbildes auf der von Ihrem Spiel verwendeten Texturseite bezieht. Auf diese Weise können Sie den Wert an Erweiterungen übergeben, die außerhalb des kompilierten Spiels verwendet werden. Dabei werden die GameMaker Studio 2 Zeichnungsfunktionen umgangen und das Sprite kann in DOM-Inhalten verwendet werden, die dann innerhalb des Fensters mit der Spielfläche gezeichnet werden können. Diese Funktion ist besonders für diejenigen interessant, die Schaltflächen und andere interaktive Medien außerhalb des GameMaker Studio 2 Canvas-Elements auf ihrer Host-Seite mithilfe der Funktion erstellen möchten clickable_add.

Hinweis: Diese Funktion ist nur für HTML5.


Beispiel:

home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");

Der obige Code erstellt eine DOM-Schaltfläche an der Position (32, 32) der Seite, auf der der Spielbereich läuft. Die Schaltfläche verwendet das Sprite, auf das von der Texturseite verwiesen wird, als "spr_MS_Home". Wenn Sie darauf klicken, öffnet die Schaltfläche ein neues Fenster für die angegebene URL und mit den definierten Eigenschaften für dieses Fenster.
