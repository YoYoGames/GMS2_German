sprite_delete


Syntax:

sprite_delete(index);

Streit Beschreibung
index Der Index des zu löschenden Sprites.


Kehrt zurück

N/A


Beschreibung

Diese Funktion löscht ein Sprite aus dem Spiel und gibt allen Speicher frei, der dafür reserviert wurde. Dies ist eine permanente Entfernung, und wenn das Asset, das du löschst, als Teil der Ressourcen des Spiels hinzugefügt wurde, wird das Spiel nicht neu gestartet (es sei denn, du schließtest die Anwendung zuerst). Dies ist eine sehr wichtige Funktion in den Momenten, in denen Sie Sprites aus externen Ressourcen erstellen und ändern müssen. Sie sollten immer dazu verwendet werden, solche Assets zu entfernen, die von einem Spiel nicht mehr benötigt werden, oder ein indiziertes Asset von einer Variablen zu löschen. Zuweisen eines anderen Inhalts zu dieser Variablen Dies kann nicht zum Löschen von Sprites verwendet werden, die als Teil des Ressourcenbaums im Spiel enthalten sind.

WICHTIG! Diese Funktion ist mit der Testlizenz des Produkts nicht verfügbar.


Beispiel:

var spr = sprite_create_from_surface(application_surface, 0, 0, 32, 32, false, false, 16, 16);
sprite_merge(spr_Player, spr);
sprite_delete(spr);

Der obige Code erstellt eine lokale Variable und speichert dann den Index des Sprites, das von der Anwendungsoberfläche erstellt wurde. Dieses Sprite wird dann mit dem in der Variablen "spr_Player" indizierten Objekt zusammengeführt, bevor es wieder aus dem Speicher entfernt wird.
