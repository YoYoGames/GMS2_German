sprite_set_cache_size


Syntax:

sprite_set_cache_size(ind, max);

Streit Beschreibung
ind Der Index des Sprites zum Ändern der Cachegröße von.
max Die maximale Anzahl der zwischengespeicherten Kopien des Sprites, die gespeichert werden können.


Kehrt zurück

N/A


Beschreibung

Beim Durchführen von Bildmischungen kann HTML5 dies nicht auf die gleiche Art und Weise tun, wie eine ausführbare Datei ausgeführt werden könnte. Daher muss GameMaker Studio 2 eine gemischte Kopie des Bildes vorübergehend speichern und laden. Diese Funktion legt fest, wie viele gemischte Kopien des gegebenen Sprites zwischengespeichert werden können, bevor alte überschrieben werden. Der Standardwert ist 4. Dies wird auf alle Unterbilder des Sprites angewendet.


Beispiel:

sprite_set_cache_size(sprite0, 2);

Dadurch wird der Sprite-Cache aller Unterbilder von sprite0 auf 2 Kopien gesetzt.
