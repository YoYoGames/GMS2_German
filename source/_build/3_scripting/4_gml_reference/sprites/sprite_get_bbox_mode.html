sprite_get_bbox_mode


Beschreibung

Diese Funktion gibt den aktuellen "Modus" für die Begrenzungsrahmenberechnungen zurück. Sie geben den Sprite-Index des Sprites zur Überprüfung an, und die Funktion gibt eine der unten gezeigten Konstanten zurück.


Syntax:

sprite_get_bbox_mode(ind);

Streit Beschreibung
ind Der Index des zu überprüfenden Sprites.


Kehrt zurück

Constant

Konstante Beschreibung
bboxmode_automatic Automatisch - Der Begrenzungsrahmen wird basierend auf der Toleranzeinstellung für das Sprite automatisch berechnet
bboxmode_fullimage Vollbild - Der Begrenzungsrahmen wird so eingestellt, dass er die volle Breite und Höhe des Sprites verwendet, unabhängig von der Toleranz und den "leeren" Pixeln
bboxmode_manual Manuell - Der Begrenzungsrahmen wurde manuell auf benutzerdefinierte Werte festgelegt (entweder im Sprite-Editor oder mithilfe der Funktion) sprite_set_bbox() )


Beispiel:

if sprite_get_bbox_mode(sprite_index) != 0
    {
    sprite_set_bbox_mode(sprite_index, bboxmode_automatic);
    }

Das obige prüft den Bbox-Modus für das aktuelle Sprite und falls nicht automatic dann wird es auf diesen Wert gesetzt.
