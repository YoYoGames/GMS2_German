sprite_duplikat


Syntax:

sprite_duplicate(index);

Streit Beschreibung
index Der Index des zu duplizierenden Sprites.


Kehrt zurück

Real (unique sprite ID value)


Beschreibung

Diese Funktion gibt den Index eines neu erstellten Sprites zurück, das ein Duplikat (Kopie) der einen Eingabe als "Index" -Argument ist. Wenn die Funktion fehlschlägt, wird -1 zurückgegeben. Diese Funktion muss verwendet werden, um alle Sprites aus den ursprünglichen Assets zu kopieren, bevor Transformationen an ihnen vorgenommen werden können. Wenn Sie beispielsweise das Begrenzungsrechteck für ein Sprite ändern oder sein Alpha von einem anderen Sprite aus setzen möchten, müssen Sie es zuerst duplizieren und dann den Vorgang für das duplizierte Sprite durchführen und verwenden. Ein dupliziertes Sprite wird beim Erstellen auf eine eigene einzigartige Texturseite platziert. Dies bedeutet, dass das Duplizieren mehrerer Sprites mehrere Texturseiten erzeugt und sich auf die Leistung auswirkt. Verwenden Sie diese Funktion daher nur bei Bedarf.

HINWEIS: Wenn Sie ein Sprite in GameMaker Studio 2 duplizieren, müssen Sie daran denken, es wieder zu entfernen (mit sprite_delete ) wenn es nicht mehr benötigt wird, besteht sonst die Gefahr eines Speicherlecks, das das Spiel verlangsamt und schließlich zum Absturz bringt.
WICHTIG! Diese Funktion ist mit der Testlizenz des Produkts nicht verfügbar.


Beispiel:

new_spr = sprite_duplicate(sprite_index)

Der obige Code dupliziert das Sprite, das momentan als Sprite-Index der Instanz verwendet wird, und speichert den Index für dieses neue Sprite in der Variablen "new_spr".
