sprite_set_speed


Syntax:

sprite_set_speed(index, speed, type)

Streit Beschreibung
index Der Index des Sprites, um den Animationstyp zu erhalten


Kehrt zurück:

Constant (see above)


Beschreibung

Mit dieser Funktion können Sie die Geschwindigkeit und den Typ der Basisanimation wie für die Sprite-Ressource festlegen. Dies wird normalerweise im Sprite-Editor festgelegt, aber es kann Momente geben, in denen Sie dies manuell für eine bestimmte Sprite-Ressource festlegen müssen. Sie müssen den zu setzenden Sprite-Index, die Animationsgeschwindigkeit und die Art der anzuwendenden Animationsgeschwindigkeit angeben. Der Animationsgeschwindigkeitstyp kann eine der folgenden Konstanten sein:

Konstante Beschreibung
spritespeed_framespersecond Das Sprite wurde mit Animation in Frames pro Sekunde definiert.
spritespeed_framespergameframe Das Sprite wurde in Animation in Frames pro Spielrahmen definiert.


Es ist sehr wichtig, den Geschwindigkeitstyp für die Berechnung der Animationen zu wählen, da zwischen den beiden Optionen ein großer Unterschied besteht. Die folgenden zwei Beispiele veranschaulichen diesen Unterschied:

  • Wenn Sie ein Sprite haben, das 1 Bild pro Sekunde zeichnet und die Bildgeschwindigkeit auf 0,5 setzt, wird es mit 0,5 Bildern pro Sekunde gezeichnet. Wenn die Frame-Rate Ihres Spiels 60 Frames pro Sekunde beträgt, zeichnet das Sprite 1 Frame pro 120 Spiel-Frames.

  • Wenn Sie einen Sprit haben, die 1 Bild pro Spiel Rahmen zieht und stellen Sie die Bildgeschwindigkeit auf 0,5 wird es 0,5 Bilder pro Spiel Rahmen ziehen. Wenn die Frame-Rate Ihres Spiels 60 Frames pro Sekunde beträgt, zeichnet das Sprite 30 Frames pro 60 Spiel-Frames.

Sie können herausfinden, welcher Geschwindigkeitswert für die Sprite-Animation die Funktion verwendet hat sprite_get_speed und Sie können die Art der Animationsgeschwindigkeit mit verwenden sprite_get_speed_type.

WICHTIG! Diese Funktion wirkt sich auf die Sprite- Ressource aus. Alle Änderungen, die Sie mit dieser Funktion vornehmen, wirken sich daher auf alle Instanzen aus, die das Sprite und alle zukünftigen Instanzen verwenden.


Beispiel:

sprite_set_speed(sprite_index, 10, spritespeed_framespersecond);

Der obige Code legt die Basisanimationsgeschwindigkeit für das der Instanz zugewiesene Sprite auf 10 Frames pro Sekunde fest.
