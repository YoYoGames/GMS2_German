network_send_raw

Syntax:

network_send_raw(socket, buffer, size);


Streit Beschreibung
socket Die ID des zu verwendenden Sockets.
buffer Die ID des Puffers zum Abrufen der Daten.
size Die Größe (in Byte) der Daten.


Rückgabe: Echt


Beschreibung

Mit dieser Funktion können Sie ein "rohes" Datenpaket über das Netzwerk senden. Die Funktion benötigt die Socket-ID, um eine Verbindung herzustellen. Anschließend müssen Sie die Puffer-ID angeben, die die zu sendenden Daten enthält (weitere Informationen zu Puffern finden Sie unter Referenz - Puffer ) und schließlich die Größe (in Byte) des Datenpakets. Die gesendeten Daten werden von GameMaker Studio 2 in keiner Weise formatiert und die empfangenden Geräte erhalten die Daten als Stream, was bedeutet, dass Sie selbst damit umgehen müssen. Die Funktion gibt die Anzahl der gesendeten Daten an oder eine Zahl kleiner als 0, wenn der Sendevorgang fehlgeschlagen ist.


Beispiel:

buff = buffer_load("player_save.dat");
network_send_raw(sock, buff, buffer_get_size(buff));

Die obige Information lädt zuvor gespeicherte Pufferdaten in den Speicher und gibt die Puffer-ID zurück, die in der Variablen "buff" gespeichert werden soll. Dieser vollständige Puffer wird dann als ein Rohdatenpaket über das Netzwerk gesendet, wobei der durch die Variable "Sock" identifizierte Socket verwendet wird.
