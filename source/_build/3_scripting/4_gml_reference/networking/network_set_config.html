Netzwerk_Set_config

Syntax:

network_set_config(config_value, setting);


Streit Beschreibung
config_value Die zu konfigurierende Konfigurationswertkonstante (unten aufgeführt).
setting Die Einstellung des Konfigurationswerts.


Rückgabe: Zeichenfolge


Beschreibung

Mit dieser Funktion können Sie verschiedene Netzwerkkonfigurationen einstellen. Diese Konfigurationen werden als Konstanten angegeben und die Einstellung hängt von der von Ihnen gewählten Konstante ab. Die folgende Tabelle listet die verfügbaren Konstanten und ihre Einstellungen auf:

Konstante Beschreibung Rahmen
network_config_connect_timeout Legen Sie einen Zeitlimitwert für die Verbindung fest A value in milliseconds
network_config_use_non_blocking_socket GameMaker Studio 2 Sie GameMaker Studio 2 auf, die Verbindung nicht zu blockieren. 0 = block, 1 = don't block



Die erste Option ( network_config_connect_timeout ) legt einfach den Timeout für die Verbindung zu einem Server fest, ändert jedoch nicht das Verbindungsverhalten, abgesehen von der Zeit, die Sie warten müssen. Es wird empfohlen, dass Sie dies nicht zu niedrig einstellen und es bei ungefähr 1000 für ein LAN-Only-Spiel oder 4000 oder so für Internet halten, oder das Spiel kann sich nicht zufällig verbinden. Wenn Sie einen Timeout-Wert für das Senden / Empfangen von Paketen festlegen möchten, verwenden Sie die Funktion network_set_timeout().

Die zweite Möglichkeit ( network_config_use_non_blocking_socket ) bedeutet, dass die Netzwerkverbindungsfunktionen alle a zurückgeben socket_id sofort, aber Sie können nicht an diesem Socket senden oder empfangen, bis Sie ein asynchrones Netzwerkereignis erhalten haben. Für das ausgelöste Ereignis wird der Schlüssel "type" auf gesetzt network_type_non_blocking_connect (Weitere Details finden Sie auf der Seite zum asynchronen Netzwerkereignis ). Dies ist eine globale Einstellung, da GameMaker Studio 2 keine Mischung aus Blockieren und Nicht-Blockieren in einer Anwendung unterstützt.


Beispiel:

network_set_config(network_config_connect_timeout, 1000);
network_set_config(network_config_use_non_blocking_socket, 1);

Der obige Code setzt das Timeout für die Netzwerkverbindung auf 1000 ms und weist GameMaker Studio 2 an, die Verbindung nicht zu blockieren.
