network_create_server

Syntax:

network_create_server(type, port, max_client);


Streit Beschreibung
type Der zu erstellende Servertyp (siehe die unten aufgeführten Konstanten).
port Der Port, den der Server verwenden wird.
max_client Die maximale Anzahl von Clients, die gleichzeitig verbunden werden können.


Rückgabe: Real


Beschreibung

Mit dieser Funktion können Sie einen neuen Netzwerkserver für Ihr Spiel erstellen, indem Sie eines der zulässigen Verbindungsprotokolle verwenden (siehe die unten aufgeführten Konstanten). Sie geben den Servertyp an, geben dann einen zu verwendenden Port an und schließlich die Anzahl der maximal zulässigen Verbindungen zum Server (beachten Sie, dass diese Anzahl von Ihnen abhängt, aber zu viele verbundene Clients die Sättigung erreichen.) Netzwerk oder die Geräte-CPU können nicht mit der Verarbeitung dieser Anzahl von Spielern umgehen. Die Funktion gibt eine eindeutige ID zurück, die in einer Variablen gespeichert und zur Identifizierung des Servers in allen weiteren Netzwerkfunktionen verwendet werden sollte, oder einen Wert unter 0, wenn die Verbindung fehlschlägt.

Konstante Beschreibung
network_socket_tcp Erstellen Sie einen Web-Socket mit TCP.
network_socket_udp Erstellen Sie einen Web-Socket mit UDP.
network_socket_bluetooth Erstellen Sie eine Bluetooth-Buchse ( derzeit nicht verfügbar! ).




Beispiel:

var port = 6510;
server = network_create_server(network_socket_tcp, port, 32);
while (server < 0 && port < 65535)
   {
   port++
   server = network_create_server(network_socket_tcp, port, 32);
   }

Mit dem obigen Code wird versucht, einen Server mit TCP über Port 6510 zu erstellen. Wenn dieser Port nicht verfügbar ist, durchläuft er die Ports, um einen Port zu finden.
