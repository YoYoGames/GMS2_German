base64_encode

Syntax:

base64_encode(string)


Streit Beschreibung
string Die zu codierende Zeichenfolge.


Rückgabe: Zeichenfolge


Beschreibung

Diese Funktion konvertiert eine Zeichenfolge in eine Base64-codierte Zeichenkette. Dies ist ein häufig verwendetes Codierungsschema, das häufig für alle Medien verwendet wird, die als Text über das Internet gespeichert oder übertragen werden müssen, und macht die Ausgabe für das menschliche Auge unlesbar.


Beispiel:

var str, file;
str = base64_encode(game_data);
file = file_text_open_write("save.txt");
file_text_write_string(file, str);
file_text_close(file);

Der obige Code wird die in "game_data" gespeicherte Zeichenfolge in eine base64-codierte Zeichenfolge konvertieren, die dann in einer externen Textdatei gespeichert wird.
