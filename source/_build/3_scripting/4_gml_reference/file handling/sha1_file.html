sha1_file


Beschreibung

In der Kryptographie ist SHA-1 eine kryptographische Hash-Funktion, die von der US-amerikanischen National Security Agency (US-amerikanische National Security Agency) entwickelt wurde. Sie wird in mehreren weit verbreiteten Anwendungen verwendet, in denen Git zum Überprüfen von Dateiänderungen verwendet wird. Die Protokolle TLS und SSL, PGP, SSH und S / MIME und IPsec. Diese Funktion nimmt eine Eingabedatei und gibt einen 160-Bit-Nachrichten-Digest im ASCII-Format zurück, der für diese Datei eindeutig ist und zu einem späteren Zeitpunkt für die Integritätsprüfung verwendet wird.


Syntax:

sha1_file(filename)


Streit Beschreibung
filename Die Datei, für die der sha1-Hash generiert wird.


Kehrt zurück:

String


Beispiel:

hash = sha1_file(working_directory + "game_data.ini")

Der obige Code generiert einen sha1-Hash für die angegebene Datei und speichert den zurückgegebenen Wert in der Variablen "Hash".
