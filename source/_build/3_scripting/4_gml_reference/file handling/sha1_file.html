sha1_datei

Syntax:

sha1_file(filename)


Streit Beschreibung
filename Die Datei, für die der sha1-Hash generiert werden soll.


Rückgabe: Zeichenfolge


Beschreibung

In der Kryptographie ist SHA-1 eine kryptografische Hash-Funktion, die von der National Security Agency der Vereinigten Staaten entwickelt wurde und in mehreren weitverbreiteten Anwendungen verwendet wird, dem Git zur Überprüfung von Dateiänderungen und den Protokollen TLS und SSL, PGP, SSH, S / MIME und IPsec. Diese Funktion nimmt eine Eingabedatei und gibt einen 160-Bit-Nachrichtenauszug im ASCII-Format zurück, der für diese Datei eindeutig ist und später für die Integritätsprüfung verwendet werden kann.


Beispiel:

hash = sha1_file(working_directory + "\game_data.ini")

Der obige Code erzeugt einen sha1-Hash für die angegebene Datei und speichert den zurückgegebenen Wert in der Variablen "hash".
