file_bin_open

Syntax:

file_bin_open(fname, mode);


Streit Beschreibung
fname Der Name der Datei, aus der gelesen werden soll.
mode der Indikator, was mit der Datei gemacht werden kann.


Rückgabe: Echt


Beschreibung

Diese Funktion öffnet die Binärdatei mit dem angegebenen Namen. Der Modus zeigt an, was mit der Datei gemacht werden kann:

  • 0 = Lesen
  • 1 = Schreiben
  • 2 = Lesen und Schreiben

Wenn die Datei nicht existiert, wird sie erstellt, und die Funktion gibt die ID der Datei zurück, die in den anderen Funktionen verwendet werden muss. Sie können mehrere Dateien gleichzeitig öffnen (max. 32), aber vergessen Sie nicht, sie zu schließen, wenn Sie fertig sind.

WARNUNG! Diese Funktion funktioniert möglicherweise nicht wie erwartet, weil GameMaker Studio 2 Sandboxing ist! Weitere Informationen finden Sie im Abschnitt zu Dateisystemgrenzen.

Hinweis: diese Funktionen funktionieren nicht, wenn das Zielmodul HTML5 ist.


Beispiel:

file = file_bin_open('myfile.bin', 2);

Dies würde eine Datei aus demselben Verzeichnis wie das Spiel öffnen und ihren Index der Variablen "Datei" zuweisen.
