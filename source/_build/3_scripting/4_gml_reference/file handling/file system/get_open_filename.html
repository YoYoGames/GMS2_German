get_open_filename

Syntax:

get_open_filename(filter, fname);


Streit Beschreibung
filter Der Dateityp-Filter und / oder Name.
fname Der vorgeschlagene Dateiname, der verwendet werden soll.


Rückgabe: Zeichenfolge.


Beschreibung

Diese Funktion öffnet einen Dialog und fordert den Player auf, einen Dateinamen mit dem angegebenen Filter zu öffnen. Der Filter hat die Form " Name1 | Maske1 | Name2 | Maske2 |... ", wobei die Maske die verschiedenen Optionen mit einem Semikolon zwischen ihnen enthalten kann und Sie können auch ein "*" verwenden, um eine beliebige Zeichenfolge zu bezeichnen. Zum Beispiel: " bitmaps | *.bmp; *. Wmf ", würde den Benutzer auffordern, den Namen einer Bitmap-Datei aus dem System auszuwählen, aber nur diejenigen, die die zwei angegebenen Erweiterungen haben ( bmp und png ). Diese Funktion öffnet nicht die Datei selbst! Es gibt nur eine Zeichenfolge mit dem vollständigen Namen und dem Pfad der Datei zurück. Wenn der Benutzer " Abbrechen " drückt, wird eine leere Zeichenfolge "" zurückgegeben.

Bei der Verwendung dieser Funktion ist es wichtig, dass Sie für die Dauer Ihres Spiels bestimmte Berechtigungen für diese Datei erhalten. Sobald Sie also den Dateipfad vom Benutzer erhalten, können Sie immer wieder auf ihn zugreifen, ohne nachfragen zu müssen. Da diese Funktion jedoch zum Lesen einer Datei dient, sind die gewährten Berechtigungen nur zum Lesen und Speichern durch das Betriebssystem zulässig. Wenn Sie die Datei lesen und in die Datei schreiben müssen, können Sie mit der Funktion eine Berechtigung erhalten get_save_filename.

HINWEIS: Dies ist eine Windows-Funktion.


Beispiel:

var file;
file = get_open_filename("text file|*.txt", "");
if file != ""
   {
   file_text_open_read(file);
   }

Der obige Code fordert den Benutzer auf, eine Textdatei zum Öffnen auszuwählen und dann den zurückgegebenen Dateinamen und Pfad zu überprüfen, und wenn er existiert, wird die Datei zum Lesen geöffnet.
