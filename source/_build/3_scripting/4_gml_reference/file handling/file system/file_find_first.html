file_find_first

Syntax:

file_find_first(mask, attr);


Streit Beschreibung
mask Die für die Suche zu verwendende Maske.
attr Das spezifische Dateiattribut, nach dem gesucht werden soll.


Rückgabe: String


Beschreibung

Diese Funktion gibt den Namen der ersten Datei zurück, die die Maske und die Attribute erfüllt. Wenn keine solche Datei vorhanden ist, wird eine leere Zeichenfolge zurückgegeben. Die Maske kann einen Pfad enthalten und Wildzeichen enthalten, z. B. 'C: \ temp \ *. Doc'. Die Attribute geben die zusätzlichen Dateien an, die Sie anzeigen möchten. Die normalen Dateien werden immer zurückgegeben, wenn sie der Maske entsprechen. Sie können die folgenden Konstanten addieren, um den gewünschten Dateityp anzuzeigen (wenn Sie keine Attribute hinzufügen möchten, verwenden Sie 0):

  • fa_readonly - schreibgeschützte Dateien
  • fa_hidden - versteckte Dateien
  • fa_sysfile - Systemdateien
  • fa_volumeid - Datenträger-ID-Dateien
  • fa_directory - Verzeichnisse
  • fa_archive - archivierte Dateien


WARNUNG! Diese Funktion funktioniert möglicherweise nicht wie erwartet, da GameMaker Studio 2 einer Sandbox ausgeführt wird! Weitere Informationen finden Sie im Abschnitt Dateisystemlimits.

HINWEIS: Diese Funktion wird auf dem HTML5-Ziel überhaupt nicht funktionieren, und die Filterflags funktionieren nur unter Windows (verwenden Sie 0 für alle anderen Plattformen).


Beispiel:

if directory_exists("\User Content")
   {
   file = file_find_first("\User Content\*.doc", fa_readonly);
   }

Dadurch wird geprüft, ob das angegebene Verzeichnis vorhanden ist. Wenn dies der Fall ist, gehen Sie dorthin und geben das erste "Nur Lesen" aus. doc Datei gefunden.
