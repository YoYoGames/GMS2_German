Dateiattribute

Syntax:

file_attributes(fname, attr);


Streit Beschreibung
fname Der Name der zu überprüfenden Datei.
attr Die Attribute, nach denen gesucht werden soll.


Rückgabe: Boolean.


Beschreibung

Sie können diese Funktion verwenden, um die Attribute einer bestimmten Datei zu überprüfen. Sie können die folgenden Konstanten addieren, um den gewünschten Dateityp zu sehen:

  • fa_readonly - schreibgeschützte Dateien
  • fa_hidden - versteckte Dateien
  • fa_sysfile - Systemdateien
  • fa_volumeid - Volume-ID-Dateien
  • fa_directory - Verzeichnisse
  • fa_archive - archivierte Dateien

HINWEIS: Dies ist eine Windows-Funktion.


Beispiel:

if !file_attributes(file, fa_hidden) file_delete(file);

Dies würde eine Datei prüfen, um zu sehen, ob sie versteckt ist oder nicht, und falls nicht, wird sie gelöscht.
