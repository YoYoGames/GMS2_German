Dateisystem-Funktionen

In GameMaker Studio 2 eine Reihe von Funktionen, mit denen Sie Dateien auf verschiedene Arten bearbeiten können:

  • Datei existiert
  • file_delete
  • Dateiname
  • Dateikopie
  • file_find_first
  • file_find_next
  • file_find_close
  • Dateiattribute

Die folgenden Funktionen können verwendet werden, um Dateinamen zu ändern:

HINWEIS: Diese Funktionen funktionieren nicht mit den eigentlichen Dateien! Sie behandeln nur die Strings, die sich auf die verschiedenen Pfadelemente beziehen, und sollten in Verbindung mit den anderen Dateifunktionen verwendet werden.
  • Dateiname_Name
  • Dateiname_Pfad
  • Dateiname_Verz
  • Dateiname_Drive
  • Dateiname_ext
  • Dateiname_change_ext

Schließlich gibt es zwei spezielle Funktionen zum Abrufen von Dateinamen, die zum Laden und Speichern von Dateien mit Dateitypfiltern verwendet werden können:

HINWEIS: Diese Funktionen sind NUR Windows
  • get_open_filename
  • get_open_filename_ext
  • get_save_filename
  • get_save_filename_ext
WARNUNG! Diese Funktion funktioniert möglicherweise nicht wie erwartet, weil GameMaker Studio 2 Sandboxing ist! Weitere Informationen finden Sie im Abschnitt zu Dateisystemgrenzen.
