get_save_filename_ext

Syntax:

get_save_filename_ext(filter, fname, directory, caption);


Streit Beschreibung
filter Der Dateityp-Filter und / oder Name.
fname Der vorgeschlagene Dateiname, der verwendet werden soll.
directory Das Verzeichnis, von dem aus die Suche gestartet werden soll.
caption Die Beschriftung für das Fenster.


Rückgabe: Zeichenfolge.


Beschreibung

Diese Funktion öffnet einen Dialog und fragt den Spieler nach einem Dateinamen zum Speichern mit dem gegebenen Filter. Der Filter hat die Form " Name1 | Maske1 | Name2 | Maske2 |... ", wobei die Maske die verschiedenen Optionen mit einem Semikolon zwischen ihnen enthalten kann und Sie können auch ein "*" verwenden, um eine beliebige Zeichenfolge zu bezeichnen. Zum Beispiel: " bitmaps | *.bmp; *. Wmf ", würde den Benutzer auffordern, den Namen einer Bitmap-Datei aus dem System auszuwählen, aber nur diejenigen, die die zwei angegebenen Erweiterungen haben ( bmp und png ). Diese Funktion speichert nicht die Datei selbst! Es gibt nur eine Zeichenfolge mit dem vollständigen Namen und dem Pfad der Datei zurück. Wenn der Benutzer " Abbrechen " drückt, wird eine leere Zeichenfolge "" zurückgegeben. Sie können auch ein Verzeichnis hinzufügen, von dem die Suche gestartet wird, und eine Beschriftung, die anstelle des Standardfensters im oberen Bereich des Fensters angezeigt wird.

Bei der Verwendung dieser Funktion ist es wichtig, dass Sie für die Dauer Ihres Spiels bestimmte Berechtigungen für diese Datei erhalten. Sobald Sie also den Dateipfad vom Benutzer erhalten, können Sie immer wieder darauf zugreifen, ohne etwas fragen zu müssen, und die Berechtigung gilt gleichermaßen für das Lesen und Schreiben.

HINWEIS: Dies ist eine Windows-Funktion.


Beispiel:

var file;
file = get_save_filename_ext("screenshot|*.png", "", working_directory, "Save a level here");
if file != ""
   {
   screen_save(file);
   }

Der obige Code fordert den Benutzer auf, eine Bilddateinamensdatei zum Speichern im Arbeitsverzeichnis auszuwählen und dann den zurückgegebenen Dateinamen und Pfad zu prüfen. Wenn es keine leere Zeichenkette ist, wird ein Screenshot gemacht und unter dem gegebenen Namen gespeichert.
