file_copy

Syntax:

file_copy(fname, newname);


Streit Beschreibung
fname Der Name der zu kopierenden Datei.
newname Der Name der neuen, kopierten Datei.


Rückgabe: N / A


Beschreibung

Diese Funktion kopiert die angegebene Datei mit dem von Ihnen gewählten neuen Namen. Wenn Sie eine Datei auf Android aus dem Bereich " Enthaltene Dateien" (der sich in Ihrem APK befindet) kopieren, wird diese zuerst in den Speicher kopiert. Beachten Sie, dass das Kopieren sehr großer Dateien aus dem APK dazu führen kann, dass den Geräten nicht genügend Speicherplatz zur Verfügung steht. Auf dem HTML5-Ziel verwenden wir die lokale HTML5- Speicher-API zum Speichern auf der Clientseite. Die Funktionen unterstützen keine binären Dateioperationen. Es wird also davon ausgegangen, dass Sie eine Textdatei kopieren.


WARNUNG! Diese Funktion funktioniert möglicherweise nicht wie erwartet, da GameMaker Studio 2 einer Sandbox ausgeführt wird! Weitere Informationen finden Sie im Abschnitt Dateisystemlimits.


Beispiel:

if file_exists("level1.txt")
   {
   file_copy("level1.txt", "level2.txt");
   }

Dies würde prüfen, ob eine Datei vorhanden ist und ob sie vorhanden ist, wird sie kopiert.
