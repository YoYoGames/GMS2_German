file_find_next

Syntax:

file_find_next();


Rückgabe: Zeichenfolge


Beschreibung

Diese Funktion gibt den Namen der nächsten Datei zurück, die die zuvor angegebene Maske und die Attribute erfüllt (definiert durch file_find_first ). Wenn keine solche Datei vorhanden ist, wird die leere Zeichenfolge zurückgegeben.


WARNUNG! Diese Funktion funktioniert möglicherweise nicht wie erwartet, weil GameMaker Studio 2 Sandboxing ist! Weitere Informationen finden Sie im Abschnitt zu Dateisystemgrenzen.

HINWEIS: Diese Funktion funktioniert auf allen C ++ Zielplattformen (Windows, Mac, iOS, Android und WinPhone), ABER die Filter-Flags funktionieren nur unter Windows und WinPhone.


Beispiel:

file = file_find_next();

Dadurch wird die nächste Datei zurückgegeben, die den zuvor angegebenen Attributen entspricht.
