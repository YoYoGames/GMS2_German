zip_unzip

Syntax:

zip_unzip(zip_file, target_directory)


Streit Beschreibung
zip_file Die zu öffnende Zip-Datei
target_directory Das Zielverzeichnis, in das die Dateien extrahiert werden sollen


Rückgabe: Echt


Beschreibung

Diese Funktion öffnet eine gespeicherte Zip-Datei und extrahiert ihren Inhalt in das angegebene Verzeichnis. Beachten Sie, dass, wenn Sie dann das aktuelle Laufwerk root verwendet wird keinen vollständigen Pfad zum ZIP - Verzeichnis liefern, und wenn Sie es in einem relativen Pfad zu dem Spiel Bündel Arbeitsverzeichnis platzieren mögen, dann sollten Sie die Verwendung working_directory Variable als Teil des Pfades (relative Pfade mit "." oder ".." funktionieren nicht wie erwartet, sollten also vermieden werden). Beachten Sie auch, dass die Zip- Datei entweder Teil des Spielpakets sein muss (dh eine enthaltene Datei ) oder mit http_get_file in den Speicherbereich heruntergeladen wurde.

Die Funktion gibt einen Wert zurück, der die Anzahl der extrahierten Dateien angibt, oder sie gibt 0 oder weniger zurück, wenn die Extraktion fehlgeschlagen ist.


Beispiel:

var num = zip_unzip("/downloads/level_data.zip", working_directory + "extracted/");
if num <= 0
   {
   show_debug_message("Extraction Failed!");
   }

Der obige Code öffnet die im Verzeichnis "downloads" gespeicherte Zip-Datei und entpackt deren Inhalt in das Verzeichnis "extrahiert" (erzeugt dieses Verzeichnis, falls es nicht bereits existiert) und prüft dann, ob die Extraktion korrekt ist und angezeigt wird eine Debug-Nachricht sollte es fehlschlagen.
