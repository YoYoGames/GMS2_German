file_text_open_append

Syntax:

file_text_open_append(fname);


Streit Beschreibung
fname Der Name der Datei, an die die Datei angehängt werden soll.


Rückgabe: Echt


Beschreibung

Diese Funktion öffnet die Textdatei mit dem angegebenen Dateinamen zum Schreiben (wenn die Datei nicht existiert, wird sie erstellt) und gibt die eindeutige ID der Datei zurück, die in einer Variablen gespeichert werden soll, wie sie für alle weiteren Aktionen verwendet werden soll mach mit dieser Datei. Die Position innerhalb der Datei, in die geschrieben werden soll, wird auf die letzte Textzeile der Datei gesetzt. Beachten Sie, dass die Funktion -1 zurückgibt, wenn die Datei nicht erstellt werden kann (z. B. wegen eines ungültigen Dateinamens).


HINWEIS: Sie können maximal 32 Dateien gleichzeitig geöffnet haben. Sie sollten auch immer Dateien schließen, wenn Sie fertig sind, da dies die Informationen schreibt und den mit der Datei verbundenen Speicher freigibt.

WARNUNG! Diese Funktion funktioniert möglicherweise nicht wie erwartet, weil GameMaker Studio 2 Sandboxing ist! Weitere Informationen finden Sie im Abschnitt zu Dateisystemgrenzen.


Beispiel:

file = file_text_open_append(working_directory + "save.txt");

Dies würde "save.txt" aus demselben Verzeichnis wie das Spiel öffnen und die Datei-ID in der Variablen "file" speichern.
