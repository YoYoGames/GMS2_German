file_text_open_write

Syntax:

file_text_open_write(fname);


Streit Beschreibung
fname Der Name der Datei, in die geschrieben werden soll.


Rückgabe: Real


Beschreibung

Diese Funktion öffnet die Textdatei mit dem angegebenen Dateinamen nur zum Schreiben (wenn die Datei nicht existiert, wird sie erstellt), und gibt die eindeutige ID der Datei zurück, die in einer Variablen gespeichert werden soll, da sie für alle weiteren Aktionen verwendet wird mit dieser Datei zu tun. Wenn die Datei nicht erstellt werden kann (beispielsweise aufgrund eines ungültigen Dateinamens), gibt die Funktion -1 zurück.


HINWEIS: Es können jeweils nur maximal 32 Dateien geöffnet sein. Sie sollten auch immer Dateien schließen, wenn Sie fertig sind, da dies die Informationen schreibt und den mit der Datei verknüpften Speicherplatz freigibt.

WARNUNG! Diese Funktion funktioniert möglicherweise nicht wie erwartet, da GameMaker Studio 2 einer Sandbox ausgeführt wird! Weitere Informationen finden Sie im Abschnitt Dateisystemlimits.


Beispiel:

var file;
file = file_text_open_write(working_directory + "level.txt");
file_text_write_string(file, level_data);
file_text_close(file);

Der obige Code öffnet die Datei "level.txt" zum Schreiben und schreibt dann die in der Variablen "level_data" gespeicherte Zeichenfolge, bevor die Datei wieder geschlossen wird.
