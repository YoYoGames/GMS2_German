file_text_open_write

Syntax:

file_text_open_write(fname);


Streit Beschreibung
fname Der Name der Datei, in die geschrieben werden soll.


Rückgabe: Echt


Beschreibung

Diese Funktion öffnet die Textdatei mit dem angegebenen Dateinamen nur zum Schreiben (wenn die Datei nicht existiert, wird sie erstellt) und gibt die eindeutige ID der Datei zurück, die in einer Variablen gespeichert werden soll, da sie für alle weiteren Aktionen verwendet wird mit dieser Datei zu tun haben. Beachten Sie, dass die Funktion -1 zurückgibt, wenn die Datei nicht erstellt werden kann (z. B. wegen eines ungültigen Dateinamens).


HINWEIS: Sie können maximal 32 Dateien gleichzeitig geöffnet haben. Sie sollten auch immer Dateien schließen, wenn Sie fertig sind, da dies die Informationen schreibt und den mit der Datei verbundenen Speicher freigibt.

WARNUNG! Diese Funktion funktioniert möglicherweise nicht wie erwartet, weil GameMaker Studio 2 Sandboxing ist! Weitere Informationen finden Sie im Abschnitt zu Dateisystemgrenzen.


Beispiel:

var file;
file = file_text_open_write(working_directory + "\level.txt");
file_text_write_string(file, level_data);
file_text_close(file);

Der obige Code öffnet die Datei "level.txt" zum Schreiben und schreibt dann die in der Variablen "level_data" gespeicherte Zeichenkette, bevor die Datei wieder geschlossen wird.
