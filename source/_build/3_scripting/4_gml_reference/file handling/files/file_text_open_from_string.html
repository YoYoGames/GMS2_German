file_text_open_from_string

Syntax:

file_text_open_from_string(string);


Streit Beschreibung
string Die Zeichenfolge, aus der die Datei erstellt werden soll.


Rückgabe: Echt


Beschreibung

Diese Funktion erstellt eine Textdatei aus einer Zeichenkette und öffnet sie zum Lesen, wobei die Datei "handle" zurückgegeben wird, die in allen weiteren Dateifunktionsaufrufen verwendet werden sollte, um aus dieser Datei zu lesen. Beachten Sie, dass diese Datei temporär und schreibgeschützt ist und daher in dem Moment, in dem sie geschlossen wird, aus dem Speicher entfernt wird.


HINWEIS: Sie können maximal 32 Dateien gleichzeitig geöffnet haben. Sie sollten die Dateien auch immer schließen, wenn Sie fertig sind, da hierdurch der mit der Datei verknüpfte Speicher freigegeben wird.

WARNUNG! Diese Funktion funktioniert möglicherweise nicht wie erwartet, weil GameMaker Studio 2 Sandboxing ist! Weitere Informationen finden Sie im Abschnitt zu Dateisystemgrenzen.


Beispiel:

file = file_text_open_from_string(reset_str);

Der obige Code nimmt den in der Variablen "reset_str" gespeicherten String und erstellt daraus eine schreibgeschützte Textdatei. Der "Handle" für diese Datei wird dann in der Variablen "file" für alle weiteren zu verwendenden Dateifunktionen gespeichert.
