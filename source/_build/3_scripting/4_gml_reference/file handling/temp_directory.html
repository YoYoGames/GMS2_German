temp_verzeichnis

Syntax:

temp_directory


Rückgabe: Zeichenfolge.


Beschreibung

Dies kann verwendet werden, um das temporäre Verzeichnis, das für Ihr Spiel erstellt wurde, jedes Mal zurückzugeben, wenn es ausgeführt wird (das Stammverzeichnis enthält nicht das endgültige "\"). In diesem Verzeichnis befinden sich Dateien, auf die während des Spielverlaufs zugegriffen werden kann. Sie werden jedoch beim Schließen des Spiels entfernt (zusammen mit allen darin enthaltenen Dateien).


WARNUNG! Diese Funktion funktioniert möglicherweise nicht wie erwartet, weil GameMaker Studio 2 Sandboxing ist! Weitere Informationen finden Sie im Abschnitt zu Dateisystemgrenzen.


Beispiel:

ini_open(temp_directory + "\temp_ini.ini");

Dies öffnet eine Ini-Datei im temporären Verzeichnis des Spiels (erstellt es, wenn es nicht bereits existiert).
