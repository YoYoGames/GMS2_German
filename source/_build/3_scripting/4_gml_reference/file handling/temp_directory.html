temp_directory


Beschreibung

Dies kann verwendet werden, um das für Ihr Spiel erstellte temporäre Verzeichnis bei jeder Ausführung zurückzugeben (das Stammverzeichnis enthält nicht das endgültige "\"). Dieses Verzeichnis enthält Dateien, auf die zugegriffen werden kann, während das Spiel ausgeführt wird. Es wird jedoch zusammen mit allen darin enthaltenen Dateien entfernt, wenn das Spiel geschlossen wird.

WARNUNG! Diese Funktion funktioniert möglicherweise nicht wie erwartet, da GameMaker Studio 2 einer Sandbox ausgeführt wird! Weitere Informationen finden Sie im Abschnitt Dateisystemlimits.


Syntax:

temp_directory


Kehrt zurück:

String


Beispiel:

ini_open(temp_directory + "\temp_ini.ini");

Dies öffnet eine Ini-Datei im temporären Verzeichnis des Spiels (erstellt sie, falls noch nicht vorhanden).
