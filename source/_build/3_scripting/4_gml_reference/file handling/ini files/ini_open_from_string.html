ini_open_from_string


Beschreibung

Diese Funktion erstellt eine Ini-Datei aus einem String und öffnet sie zum Lesen / Schreiben. Die Zeichenfolge sollte korrekt als Ini-Datei formatiert sein (dh: mit Abschnitten, Zeilenumbrüchen, Schlüsseln und Werten), andernfalls wird die Ini-Datei nicht korrekt erstellt. Beachten Sie, dass diese ini-Datei temporär ist und beim Schließen des Speichers aus dem Speicher entfernt wird. Dabei gehen alle Informationen verloren, die darin gespeichert wurden ini_close Die Funktion gibt eine Zeichenfolge der vollständigen Ini-Datei zurück, die dann auf einem Server oder auf einer Festplatte gespeichert werden kann.


Syntax:

ini_open_from_string(string);


Streit Beschreibung
string Die Zeichenfolge, die alle Ini-Informationen enthält.


Kehrt zurück:

N/A


Beispiel:

ini_open_from_string(str)
global.sound = ini_read_string("Options", "Sound", true);
ini_close();

Der obige Code legt ein temporäres ini fest, um die Informationen aus der Zeichenfolge "str" ​​zu speichern, und liest dann aus dem ini, bevor er sie wieder schließt.
