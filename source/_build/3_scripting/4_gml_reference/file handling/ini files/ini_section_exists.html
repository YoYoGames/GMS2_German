ini_section_exists


Beschreibung

Diese Funktion prüft, ob ein Abschnitt im aktuell geöffneten Ini vorhanden ist. Dies ist keine notwendige Überprüfung, um Fehler zu vermeiden. Wenn ein Abschnitt nicht vorhanden ist, wird beim Lesen eines nicht vorhandenen Abschnitts nur ein Standardwert zurückgegeben. Es kann jedoch nützlich sein, festzustellen, ob eine Ini-Datei bestimmte Daten gespeichert hat.


Syntax:

ini_section_exists(section);


Streit Beschreibung
section Der Abschnitt in der geöffneten INI-Datei, nach dem gesucht werden soll.


Kehrt zurück:

Boolean


Beispiel:

ini_open("savedata.ini");
if !ini_section_exists("save1")
   {
   global.savegame = "1";
   }
ini_close();

Dies setzt die Variable global.savegame auf "1", wenn "save1" NICHT gefunden wird. Mit einer for-Schleife könnte dies als eine grundlegende Methode zum Ermitteln der Anzahl der zu speichernden Spiele für ein Spiel verwendet werden (z. B. Erstellen eines Systems für mehrere gleichzeitige Speicherspiele für ein Spiel).
