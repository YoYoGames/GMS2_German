ini_section_exists

Syntax:

ini_section_exists(section);


Streit Beschreibung
section Der Abschnitt in der geöffneten INI-Datei, nach dem gesucht werden soll.


Rückgabe: Boolean


Beschreibung

Diese Funktion überprüft, ob ein Abschnitt in der aktuell geöffneten Ini existiert. Dies ist keine notwendige Prüfung, um Fehler zu vermeiden, da, wenn ein Abschnitt nicht existiert, das Lesen von einem nicht vorhandenen Abschnitt nur einen Standardwert zurückgibt, es kann jedoch nützlich sein zu sehen, ob eine Ini-Datei bestimmte Daten gespeichert hat.


Beispiel:

ini_open("savedata.ini");
if !ini_section_exists("save1")
   {
   global.savegame = '1';
   }
ini_close();

Dadurch wird die Variable global.savegame auf '1' gesetzt, wenn 'save1' NICHT gefunden wird. Mit einer for-Schleife könnte dies als eine grundlegende Methode verwendet werden, um herauszufinden, wie viele gespeicherte Spiele es für ein Spiel gibt (dh: ein System für mehrere simultane Speicherspiele für ein Spiel erstellen).
