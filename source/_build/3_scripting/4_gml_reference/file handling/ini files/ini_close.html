ini_close


Beschreibung

Diese Funktion sollte aufgerufen werden, sobald Sie mit dem Lesen oder Schreiben einer geöffneten INI-Datei fertig sind. Wenn Sie die Funktion nicht verwenden, nachdem Sie eine der Ini-Write-Funktionen verwendet haben, wird nichts auf die Festplatte geschrieben, da die Dateiinformationen im Speicher gehalten werden, bis diese Funktion aufgerufen wird, was das Schreiben erzwingt. Wenn Sie versuchen, ein Ini zu öffnen, ohne zuvor ein anderes (oder das gleiche) geschlossen zu haben, wird auch eine Fehlermeldung angezeigt.

Die Funktion gibt auch einen String mit der darin codierten INI-Datei zurück. Diese Zeichenfolge kann dann auf einem Server gespeichert und / oder zusammen mit der Funktion verwendet werden ini_open_from_string das ini neu erstellen.


Syntax:

ini_close();


Kehrt zurück:

String


Beispiel:

ini_open("savedata.ini");
score = ini_read_real("save1", "score", 0);
ini_close();

Dadurch wird 'savedata.ini' geöffnet und der Score auf den Wert unter "save1" mit dem Schlüssel "score" gesetzt. Anschließend wird die.ini wieder geschlossen. Wenn unter "save1", "score" kein Wert vorhanden ist oder keine Datei "Savedata.ini" vorhanden ist, wird der Score auf 0 gesetzt und eine neue ini_file erstellt.
