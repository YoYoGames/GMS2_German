ini_write_string

Syntax:

ini_write_string(section, key, value);


Streit Beschreibung
section Der Abschnitt der.ini zum Schreiben.
key Der Schlüssel innerhalb des relevanten Abschnitts der.ini, um zu schreiben.
value Die Zeichenfolge, die an das entsprechende Ziel geschrieben wird.


Rückkehr: N / A


Beschreibung

Sie können diese Funktion verwenden, um eine Zeichenfolge (Text) in eine Ini-Datendatei zu schreiben. Ini - Dateien werden in Abschnitte aufgeteilt und anschließend wird jeder Abschnitt in Schlüssel - Wert - Paare aufgeteilt. Eine typische Ini-Datei würde also ungefähr so ​​aussehen:


Beispiel:

ini_open("savedata.ini");
ini_write_string("Save", "Player", global.Name);
ini_close();

Der obige Code öffnet eine Ini-Datei zum Lesen und Schreiben, schreibt dann die in der globalen Variable "Name" gespeicherte Zeichenfolge in den Abschnitt "Speichern" mit dem Schlüssel "Player", bevor die Datei erneut geschlossen wird.
