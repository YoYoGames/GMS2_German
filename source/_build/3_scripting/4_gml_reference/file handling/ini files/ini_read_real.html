ini_read_real

Syntax:

ini_read_real(section, key, default);


Streit Beschreibung
section Der Abschnitt der.ini zum Lesen.
key Der Schlüssel innerhalb des relevanten Abschnitts der.ini, aus dem gelesen werden soll.
default Der Rückgabewert, wenn an der definierten Stelle kein Wert gefunden wird (oder die INI-Datei nicht existiert). Muss eine echte Nummer sein.


Rückgabe: Echt


Beschreibung

Sie können diese Funktion verwenden, um eine Nummer aus einer Ini-Datei zu lesen. Ini - Dateien werden in Abschnitte aufgeteilt und anschließend wird jeder Abschnitt in Schlüssel - Wert - Paare aufgeteilt. Eine typische Ini-Datei würde also ungefähr so ​​aussehen:


Beispiel:

ini_open( 'savedata.ini' );
score = ini_read_real( 'save1', 'score', 0 );
ini_close();

Dies öffnet 'savedata.ini' und setzt den Wert auf den Wert unter 'save1'> 'score' und schließt dann die.ini erneut. Sollte unter "save1"> "score" kein Wert vorhanden sein oder keine Datei "savedata.ini" vorhanden sein, wird der Score auf 0 gesetzt.
