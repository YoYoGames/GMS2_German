display_set_ui_visibility


Beschreibung

Diese Funktion kann verwendet werden, um die System-Benutzeroberfläche nur unter Android anzuzeigen oder auszublenden. Für die Funktion müssen Sie ein oder mehrere Systemflags als Ganzzahl angeben. Wenn Sie mehr als ein einzelnes Flag verwenden, müssen Sie diese mit dem bitweisen "oder" zusammenführen, wie im folgenden Beispiel gezeigt. Eine Liste der Android -Systemflags finden Sie hier.


Syntax:

display_set_ui_visibility(flags);

Streit Flaggen
flags Zu verwendende Flags des Android Systems (als ganzzahliger Wert).


Kehrt zurück:

N/A


Beispiel:

var flags = 1024|4096;
display_set_ui_visibility(flags);

Der obige Code verwendet die Android Systemflags "SYSTEM_UI_FLAG_IMMERSIVE_STICKY" und "SYSTEM_UI_FLAG_LAYOUT_FULLSCREEN", um die Sichtbarkeit der Anzeige festzulegen.
