display_set_ui_visibility


Syntax:

display_set_ui_visibility(flags);


Streit Flaggen
flags Die zu verwendenden Android SYSTEM-Flags (als Integer-Wert).


Rückkehr: N / A


Beschreibung

Diese Funktion kann verwendet werden, um die Systembenutzeroberfläche nur auf Android anzuzeigen oder auszublenden. Die Funktion erfordert, dass Sie ein oder mehrere Systemflags als Integer-Wert angeben. Wenn mehr als ein einzelnes Flag verwendet wird, müssen diese mit dem bitweisen "oder" zusammengeführt werden, wie im folgenden Beispiel gezeigt. Sie können eine Liste von finden Android System - Flags hier.


Beispiel:

var flags = 1024|4096;
display_set_ui_visibility(flags);

Der obige Code verwendet die Android System-Flags "SYSTEM_UI_FLAG_IMMERSIVE_STICKY" und "SYSTEM_UI_FLAG_LAYOUT_FULLSCREEN", um die Sichtbarkeit der Anzeige festzulegen.
