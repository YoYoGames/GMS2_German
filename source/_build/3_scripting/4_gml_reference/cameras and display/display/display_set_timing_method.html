display_set_timing_method


Beschreibung

Diese Funktion kann verwendet werden, um die Timing-Methode für das Rendern Ihres Spiels festzulegen. Die Methode kann eine der folgenden Konstanten sein:

Konstante Beschreibung
tm_sleep Verwenden Sie den Schlafbereich als Hauptmethode
tm_countvsyncs Verwenden Sie das vsync-Timing als Haupt-Timing-Methode (Standardeinstellung für alle unterstützten Plattformen).


Die vsync-Timing-Methode verwendet die Zielplattformenunterstützung für die vertikale Synchronisierung, um einen Anker für die Timing-Berechnungen der Spiele-Rendering bereitzustellen. Wenn Sie für den Modus den Ruhemodus einstellen, wird nur versucht, sicherzustellen, dass jeder Frame die korrekte Zeitdauer hat (z. B. 1/30) oder 1/60 Sekunde) durch Warten oder Schlafen. Im Allgemeinen liefert das Standard-vsync-Timing die glattesten Ergebnisse. Beachten Sie jedoch, dass bei Verwendung der vsync-Methode die Werte für den Schlafbereich immer noch relevant sind, auch wenn die Auswirkungen hierdurch geringer sind. Wir empfehlen, den Standardwert bei Verwendung dieser Methode beizubehalten.

Auf allen Plattformen außer PS4, Ubuntu und HTML5 verwendet GameMaker Studio 2 standardmäßig die vsync-Zeitmethode, während auf den nicht unterstützten Plattformen nur die Zeitspanne für die Zeitspanne für den Schlafbereich verwendet werden kann.

Wenn Sie die Schlafspanne einstellen möchten, können Sie dies mit der Funktion tun display_set_sleep_margin() und Sie können herausfinden, welche Timing-Methode aktuell mit der Funktion verwendet wird display_get_timing_method().


Syntax:

display_set_timing_method(method);

Streit Beschreibung
method Die zu verwendende Timing-Methode (siehe Liste der Konstanten oben)


Kehrt zurück:

N/A


Beispiel:

if display_get_timing_method() != tm_sleep
{
display_set_timing_method(tm_sleep);
if display_get_sleep_margin() != 20
    {
    display_set_sleep_margin(20);
    }
}

Der obige Code prüft die Timing-Methode und dann, wenn sie nicht eingestellt ist tm_sleep es wird darauf eingestellt und der Schlafbereich auf 20 eingestellt.
