display_set_gui_maximise


Beschreibung

Diese Funktion kann verwendet werden, um die GUI-Ebene zu maximieren und sie in Bezug auf die Bildschirmabmessungen anstelle der Standardposition und -abmessungen der Anwendungsoberfläche zu skalieren und zu verschieben. Standardmäßig ist die GUI-Ebene 1: 1 mit der Auflösung der Anwendungsoberfläche und wird auch an der (0,0) -Position der Oberfläche gezeichnet. Dies ist jedoch nicht immer das, was Sie möchten, und Sie können diese Funktion verwenden, um sie relativ zur absoluten (0,0) Position des Displays oder des Spielfensters zu zeichnen. Welche Argumente Sie für diese Funktion angeben, hängt von dem gewünschten Effekt auf der GUI-Schicht ab. Wenn Sie die Funktion ohne Argumente aufrufen, wird die GUI-Ebene so festgelegt, dass sie an der (0,0) -Position des Bildschirms oder des Spielfensters gezeichnet wird. Breite und Höhe werden so skaliert, dass sie in den gesamten Bereich passen:

display_set_gui_maximise();

Sie können jedoch den Skalierungsfaktor für die GUI-Ebene festlegen. Breite und Höhe werden um diesen Betrag skaliert. Denken Sie daran, dass die GUI-Ebene immer an die Größe des Displays oder des Spielfensters oder der Anwendungsoberfläche angepasst wird. Wenn Sie diesen Wert auf einen anderen Wert als 1 setzen, wird die Pixelanzahl im Wesentlichen nach Breite und Höhe skaliert. Wenn Ihre Anzeige 1024 x 768 ist und Sie den Skalierungsfaktor auf 0,5 skalieren, ist Ihre GUI-Ebene halb so groß wie die Anzeige, wodurch die Pixelgröße verdoppelt wird:

display_set_gui_maximise(0.5, 0.5);

Wenn Sie die Werte auf diese Weise festlegen, wird die Zeichenposition auch auf (0,0) des Bildschirms oder des Spielfensters gesetzt, sodass Sie Versatzwerte angeben können, um die Position (0,0) zu "verschieben":

var pos = application_get_position();
display_set_gui_maximise(0.5, 0.5, pos[0], pos[1]);

Zum Schluss können Sie die GUI-Schicht mit dieser Funktion zurücksetzen, indem Sie die Skalierungsfaktoren auf -1 setzen. Dadurch wird die GUI-Ebene wieder auf eine 1: 1-Skala gesetzt und die Zeichenposition auf die (0,0) -Position der Anwendungsoberfläche eingestellt, nicht auf die Anzeige oder das Fenster.

display_set_gui_maximise(-1, -1);


Syntax:

display_set_gui_maximise(xscale, yscale, xoffset, yoffset);

Streit Beschreibung
xscale <Optional> Der horizontale Skalierungsfaktor (verwenden Sie -1, um die Standardeinstellung wiederherzustellen).
yscale <Optional> Der vertikale Skalierungsfaktor (verwenden Sie -1, um die Standardeinstellung wiederherzustellen).
xoffset <Optional> Die x-Versatzposition für das Zeichnen.
yoffset <Optional> Die y-Versatzposition zum Zeichnen.


Kehrt zurück:

N/A


Beispiel:

display_set_gui_size(display_get_width() / 2, display_get_height() / 2);
display_set_gui_maximise(2, 2, 0, 0);

Der obige Code sperrt das Zeichnungs-GUI-Ereignis auf die angegebene Breite und Höhe und skaliert alle Komponenten, um die Anzeige unter Verwendung dieses Verhältnisses anzupassen.
