display_set_gui_maximise

Skaliert die GUI auf die angegebene Breite oder Höhe.

Syntax:

display_set_gui_maximise(xscale, yscale, xoffset, yoffset);


Streit Beschreibung
xscale <Optional> Der horizontale Skalierungsfaktor (verwenden Sie -1, um auf den Standardwert zurückzusetzen).
yscale <Optional> Der vertikale Skalierungsfaktor (verwenden Sie -1, um auf den Standardwert zurückzusetzen).
xoffset <Optional> Die x-Versatzposition für das Zeichnen.
yoffset <Optional> Die y-Versatzposition für das Zeichnen.


Rückkehr: N / A


Beschreibung

Diese Funktion kann verwendet werden, um die GUI-Ebene zu maximieren und sie so zu skalieren und zu versetzen, dass sie in Relation zu den Bildschirmdimensionen statt der Standardposition und -größe der Anwendungsoberfläche steht. Standardmäßig ist die GUI-Ebene 1: 1 mit der Auflösung der Anwendungsoberfläche und wird ebenfalls an der Position (0,0) der Oberfläche gezeichnet. Dies ist jedoch nicht immer das, was Sie wollen und Sie können diese Funktion verwenden, um es relativ zur absoluten (0,0) Position des Anzeige- oder Spielfensters zu zeichnen.

Welche Argumente Sie für diese Funktion angeben, hängt davon ab, welchen Effekt sie auf der GUI-Ebene haben soll. Wenn Sie einfach die Funktion ohne Argumente aufrufen, wird die GUI-Ebene an der Position (0,0) des Bildschirms oder Spielfensters angezeigt, wobei Breite und Höhe so skaliert werden, dass sie dem gesamten Bereich entsprechen.

display_set_gui_maximise();



Sie können jedoch den Skalierungsfaktor für die GUI-Ebene festlegen, und die Breite und Höhe werden um diesen Betrag skaliert. Denken Sie daran, dass die GUI-Ebene immer der Größe des Anzeige- oder Spielfensters oder der Anwendungsoberfläche entspricht. Wenn Sie also für diesen Wert einen anderen Wert als 1 festlegen, wird die Anzahl der Pixel entlang der Breite und Höhe skaliert. Wenn Ihr Bildschirm eine Auflösung von 1024 x 768 hat und Sie den Skalierungsfaktor auf 0,5 skalieren, ist Ihre GUI-Ebene halb so groß wie das Display und verdoppelt effektiv die Pixelgröße.

display_set_gui_maximise(0.5, 0.5);



Wenn Sie die Werte auf diese Weise einstellen, wird die Zeichenposition ebenfalls auf (0,0) des Anzeige- oder Spielfensters gesetzt, sodass Sie Offset-Werte bereitstellen können, um die (0,0) -Position "zu verschieben":

var pos = application_get_position();
display_set_gui_maximise(0.5, 0.5, pos[0], pos[1]);



Schließlich können Sie die GUI-Ebene mithilfe dieser Funktion zurücksetzen, indem Sie die Skalierungsfaktoren auf -1 setzen. Dies setzt die GUI-Ebene wieder auf eine 1: 1-Skalierung und setzt die Zeichenposition auf die Position (0,0) der Anwendungsoberfläche und nicht auf die Anzeige oder das Fenster.

display_set_gui_maximise(-1, -1);


Beispiel:

display_set_gui_size(display_get_width() / 2, display_get_height() / 2;
display_set_gui_maximise(2, 2, 0, 0);

Der obige Code sperrt das Draw-GUI-Ereignis auf die angegebene Breite und Höhe und skaliert alle Komponenten so, dass sie auf die Anzeige passen.
