Fenster_Zentrum

Zentriert die Region basierend auf dem sichtbaren oder Browserbereich.

Syntax:

window_center();


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie das Spielfenster in der Anzeige zentrieren, wenn das Zielmodul Windows, Ubuntu (Linux) oder MacOS ist, oder Sie können es im Browser zentrieren, wenn das Zielmodul HTML5 ist. Diese Funktion hat keine Auswirkung auf andere Geräte.

HINWEIS: Wenn Sie die Größe des Spielfensters geändert haben (z. B. durch Umschalten von Vollbild auf Fenster) und das neue Fenster zentrieren möchten, sollte diese Funktion mindestens einen Schritt später als die Fenstergröße in einem Alarm [0] aufgerufen werden ändern, sonst wird es nicht richtig funktionieren.


Beispiel:

if !window_get_fullscreen()
   {
   window_center();
   }

Der obige Code zentriert das Spielfenster innerhalb des Displays, wenn es nicht im Vollbildmodus ist.
