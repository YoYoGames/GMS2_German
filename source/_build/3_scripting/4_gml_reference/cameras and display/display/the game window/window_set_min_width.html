window_set_min_width

Legt eine minimale Breite für das Spielfenster fest.

Syntax:

window_set_min_width(width);


Streit Beschreibung
width Die minimale Breite in Pixeln für das Spielfenster.


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie eine Mindestfensterbreite für Ihr Spiel festlegen. Wenn Sie die Option zur Größenänderung des Fensters in den Spieloptionen für die Zielplattform aktivieren, kann der Player das Spielfenster auf die von Ihnen gewünschte Größe skalieren. Mit dieser Funktion können Sie jedoch die Mindestbreite auf die von Ihnen angegebene Größe begrenzen. Wenn Sie zum Standardverhalten zurückkehren möchten (dh: kein Minimum), verwenden Sie einen Wert von -1.

HINWEIS: Diese Funktion ist nur auf den Plattformen Windows, Mac OS und Linux verfügbar.


Beispiel:

window_set_min_width(640);
window_set_min_height(480);
window_set_max_width(1280);
window_set_max_height(960);

Der obige Code legt die minimale und maximale Breite und Höhe für das Spielfenster fest.
