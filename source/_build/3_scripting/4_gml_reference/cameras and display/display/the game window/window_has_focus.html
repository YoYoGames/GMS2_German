window_has_focus

Gibt zurück, wenn das Spielfenster scharf ist oder nicht.

Syntax:

window_has_focus();


Rückgabe: Boolean


Beschreibung

Mit dieser Funktion können Sie den Status des Fensters (oder Tabs) abfragen und wenn er den Fokus verliert, kehrt die Funktion zurück false Sonst wird es zurückkehren true. In den meisten Fällen können Sie einfach die os_is_paused Funktion, um dies zu testen, aber in einigen sehr speziellen Fällen (z. B. Spiele in Chrome Apps) wird diese Funktion nicht ausgelöst, in diesem Fall sollten Sie stattdessen diese Funktion verwenden.

HINWEIS: Diese Funktion ist nur für die Module HTML5, Windows, MacOS, Windows 8 (JS) und Tizen (JS) gültig.


Beispiel:

if !window_has_focus()
   {
   scr_PauseGame();
   }

Der obige Code überprüft, ob das Spielfenster scharf ist oder nicht und ob die Funktion zurückkehrt false Ein Skript wird aufgerufen.
