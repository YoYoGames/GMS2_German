Fenstergriff

Gibt die ID des Spielfensters zurück.

Syntax:

window_handle();


Rückgabe: Zeiger


Beschreibung

Mit dieser Funktion können Sie die interne << span class = "notranslate"> Windows-ID (den HWND, einen Zeiger ) HWND. Diese Funktion ist nur für Erweiterungsautoren nützlich, die die window handle Aufruf von Windows API's in DLL-Code (der zurückgegebene Zeiger sollte in einen String umgewandelt werden und dann in C ++ einfach in ein HWND umgewandelt werden). Die nachstehende Tabelle zeigt die unterstützten Plattformen und deren Ergebnisse:

Plattform Kehrt zurück
Windows Fenster HWND
macOS NSWindow class
Ubuntu (Linux) XWindow Griff
HTML5 Canvas ID


Beispiel:

win_id = windows_handle();

Der obige Code speichert die Spielfenster-ID in einer Variablen.
