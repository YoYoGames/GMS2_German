window_set_size

Legt die Größe des Fensters in Pixeln fest.

Syntax:

window_set_size( w, h );


Streit Beschreibung
w Die neue Breite des Fensters.
h Die neue Höhe des Fensters.


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie die Größe des Spielfensters ändern. Wenn Sie diese Funktion verwenden, um die Größe des Spielfensters zu ändern, wird der Inhalt des Fensters so skaliert, dass ein 1: 1-Verhältnis zur Hintergrundgrafik besteht, wodurch das Bild scharf und klar bleibt (obwohl skaliert, wenn das Fenster anders ist) als die Raum- oder Ansichtgröße). Wenn Sie beispielsweise ein Fenster mit 200 x 150 Pixeln haben und der Player das Fenster durch Ziehen der Rahmen dehnt, beträgt die endgültige Fenstergröße 400 x 300 Pixel. Die Zeichenfläche beträgt jedoch STILL 200 x 150, wodurch ein verschwommenes Bild entsteht da das Pixelverhältnis jetzt 1: 2 ist. Wenn Sie diese Funktion jedoch verwenden, um die Fenstergröße festzulegen, wird das 1: 1-Verhältnis beibehalten, obwohl das Bild auf die neue Fenstergröße skaliert wird.


Beispiel:

if mouse_check_button_pressed(mb_left)
   {
   global.scale += 1;
   window_set_size(200 * global.scale, 150 * global.scale);
   }

Der obige Code ändert die Größe des Fensters basierend auf dem Wert der Variable "global.scale", wenn die Maustaste gedrückt wird.
