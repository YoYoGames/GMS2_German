display_get_orientation

Diese Variable enthält je nach Ausrichtung des Geräts eine andere Konstante.

Syntax:

display_get_orientation()


Rückgabe: Konstante


Beschreibung

Diese Funktion gibt einen von zwei konstanten GameMaker Studio 2 muss Ihnen mitteilen, ob das Gerät, auf dem das Spiel ausgeführt wird, im Quer- oder Hochformat gehalten wird:

Konstante Beschreibung
display_landscape Das Gerät wird horizontal gehalten, dh: Die längste Kante ist von links nach rechts und die Menüschaltfläche ist rechts.
display_landscape_flipped Wie oben, nur jetzt ist die Menü-Taste auf der linken Seite.
display_portrait Das Gerät wird vertikal gehalten, dh: Die längste Kante ist von oben nach unten und die Menüschaltfläche ist unten.
display_portrait_flipped Wie oben, nur jetzt ist der Menü-Button oben.

Bitte beachten Sie, dass diese Funktion die Ausrichtung des Geräts im HTML5-Zielmodul möglicherweise nicht korrekt erkennt. Dies kann jedoch leicht durch Verwendung des folgenden Skripts nachgeahmt werden:

return (browser_width < browser_height);

Es wird zurückkehren true für Porträt und false für Landschaft.


Beispiel:

if display_get_orientation() = display_landscape
   {
   global.Config = 0;
   }
else
   {
   global.Config = 1;
   }

Der obige Code überprüft die Ausrichtung des Geräts und legt abhängig vom zurückgegebenen Wert eine globale Variable fest.
