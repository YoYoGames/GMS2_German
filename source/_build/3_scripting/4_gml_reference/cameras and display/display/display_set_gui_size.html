display_set_gui_größe

Skaliert die GUI auf die angegebene Breite oder Höhe.

Syntax:

display_set_gui_size(width, height);


Streit Beschreibung
width Die Breite der GUI (-1 entspricht 1: 1 mit der Anzeige oder Application_Surface).
height Die Höhe der GUI (-1 entspricht 1: 1 mit der Anzeige oder Application_Surface).


Rückkehr: N / A


Beschreibung

Diese Funktion betrifft nur Objekte, die im Draw-GUI-Ereignis gezeichnet wurden, und kann verwendet werden, um eine bestimmte Breite und Höhe für alle GUI-Komponenten in diesem Ereignis festzulegen, und egal, welche Größe der Anzeige oder des Fensters zur Anzeige der GUI verwendet wird skaliert, um richtig zu passen. Wohin die GUI positioniert ist (dh die Position (0,0)) hängt davon ab, ob Sie die Funktion verwendet haben display_set_gui_maximise() oder nicht. Wenn Sie diese Funktion nicht verwenden (die Standardeinstellung), wird die GUI-Ebene mit ihrem (0,0) -Punkt an derselben Position wie die Anwendungsoberfläche (der Bereich, in dem Ihr Spiel gezeichnet wird) und die Breite positioniert und die Höhe wird so gestreckt, dass sie auf diese Oberfläche passt, so dass die Einstellung einer GUI-Breite von 256 Pixeln mit dieser Funktion, wenn die Anwendungsoberfläche 512 Pixel groß ist, diese 256 Pixels dehnen würde, was die Pixel doppelt so groß macht. Dies ist ideal für Spiele, bei denen die Seitenverhältniskorrektur in den globalen Spieleinstellungen eingestellt ist.

Wenn du das benutzt hast display_set_gui_maximise() Funktion dann wird die (0,0) Position für die GUI-Ebene standardmäßig auf die obere linke Ecke des Spielfensters gesetzt oder angezeigt (wenn im Vollbild oder auf Geräten), mit dem von dieser Funktion eingestellten Skalierungsfaktor und der Breite und Höhe von diesem festgelegt. Sie können die GUI-Ebene jederzeit auf die Standardkonfiguration zurücksetzen, indem Sie diese Funktion mit -1 als Breite und Höhe aufrufen. Dadurch wird die Position (0, 0) auf die Anwendungsoberfläche gesetzt und die Breite und Höhe werden zurückgesetzt 1: 1 auch mit der Oberfläche.


Beispiel:

display_set_gui_size(768, 1024);

Der obige Code sperrt das Draw-GUI-Ereignis auf die angegebene Breite und Höhe und skaliert alle Komponenten so, dass sie entweder der Anwendungsoberfläche oder der Anzeige entsprechen.
