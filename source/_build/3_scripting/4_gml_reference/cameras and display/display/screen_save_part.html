screen_save_part

Speichert einen Screenshot eines definierten Bereichs des Spielfensters als.png unter einem bestimmten Dateinamen.

Syntax:

screen_save_part(fname,x,y,w,h);


Streit Beschreibung
fname Der Dateiname für den Screenshot zum Speichern
x Die X-Koordinate der oberen linken Ecke des zu erfassenden Bereichs des Bildschirms
y Die y-Koordinate der oberen linken Ecke des zu erfassenden Bereichs des Bildschirms
w Die Breite des zu erfassenden Bereichs des Bildschirms
h Die Höhe des zu erfassenden Bereichs des Bildschirms


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie einen Screenshot eines Teils des Spiels speichern, wie er gerade im Spielfenster oder Browser angezeigt wird. Die Auswahlkoordinaten basieren auf der absoluten Größe der Anzeige, also ist (0,0) immer die obere linke Ecke des Bildschirms. Unten ist ein Bild, das zeigt, wie diese Funktion funktioniert: Die Datei wird im Arbeitsverzeichnis des Spiels gespeichert.

Hinweis: diese Funktion funktioniert nicht für das HTML5-Ziel.


Beispiel:

if keyboard_check_pressed(ord("S"))
   {
   screen_save_part(working_directory + "\Screens\Screen_"+string(num)+".png", 0, 0, 100, 100)
   num += 1;
   }

Der obige Code erstellt einen Screenshot eines Teils des Spielbildschirms, der 100 Pixel groß ist, wenn der Benutzer die Taste "S" drückt. Dieser Screenshot wird in einem bestimmten Verzeichnis gespeichert und mit einer Variablen nummeriert, die dann inkrementiert wird.
