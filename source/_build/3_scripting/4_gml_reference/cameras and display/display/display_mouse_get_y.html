display_mouse_get_y

Gibt die Y-Koordinate der Maus relativ zur Anzeige zurück.

Syntax:

display_mouse_get_y();


Rückgabe: Echt


Beschreibung

Diese Funktion gibt die Mausposition auf dem Bildschirm zurück. Es sollte beachtet werden, dass diese Funktion nur ordnungsgemäß funktioniert, wenn sie auf dem Windows-Ziel verwendet wird. Es kann auch für HTML5 verwendet werden, gibt aber nur einen Wert relativ zur 0,0 des Canvas selbst zurück und gibt keinen Wert zurück, während sich die Maus außerhalb des Canvas befindet. Für andere Geräte wird 0 zurückgegeben, und Sie sollten das verwenden device_mouse_raw_x und device_mouse_raw_y funktioniert stattdessen.


Beispiel:

my_y = display_mouse_get_y();

Dies würde my_y auf die Y-Koordinate der Maus relativ zur Anzeige setzen.
