Ansicht_Kamera


Beschreibung

Dieses Array enthält die eindeutige Kamera-ID, die dem angegebenen Anzeige-Port zugewiesen ist, und kann auf eine neue Kamera eingestellt oder gelesen werden, um die aktuelle Kamera zu erhalten, und -1 zurückgeben, wenn keine Kamera zugewiesen ist. Sie können in einem Raum bis zu 8 Anzeige-Ports aktiv haben (Array-Werte 0 bis 7) und jeder Kamera eine Kamera zuordnen, indem Sie diese Variable einfach auf den Kamera-ID-Wert setzen (wie von den Funktionen zurückgegeben) camera_create() oder camera_create_view() ). Wenn Sie die Kamera über den Raum-Editor auf einen Anzeige-Port eingestellt haben, enthält diese Variable die Kamera-ID für diesen Port.


Syntax:

view_camera[0 ... 7];


Kehrt zurück:

Camera index or -1 if no camera is assigned


Beispiel:

if view_camera[0] == -1
   {
   view_camera[0] = camera_create_view(0, 0, room_width, room_height);
   }

Der obige Code überprüft, ob der Ansichtsport [0] mit einer Kamera verbunden ist, und wenn dies nicht der Fall ist, wird eine erstellt und auf sie gesetzt.
