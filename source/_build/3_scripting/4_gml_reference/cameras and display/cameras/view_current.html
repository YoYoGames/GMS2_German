view_current


Beschreibung

Diese schreibgeschützte Variable ist nur im Draw-Ereignis gültig und gibt den aktuellen Ansichtsport zurück, der gerendert wird. Der Rückgabewert wird während des Zeichenereignisses geändert, wenn Sie verschiedene Ansichten haben, da das Zeichenereignis einmal für jeden Anzeigeport nacheinander aufgerufen wird. Wenn Sie also zum Beispiel in Ihrem Spielzimmer den Anzeigeport [0] und den Anzeigeport [1] verwenden, wird das Zeichenereignis für ALLE Instanzen zweimal ausgeführt, einmal für jeden Port und mit dieser Variable können Sie überprüfen, ob es angezeigt wird Welche Ansicht wird gerade erstellt? Im Allgemeinen werden nur bestimmte Details für einen einzelnen Port gerendert, wenn mehrere View-Ports gleichzeitig im Raum sichtbar sind. Siehe den folgenden Beispielcode.


Syntax:

view_current;


Kehrt zurück:

Real (view port index from 0 to 7)


Beispiel:

if view_current == 0
   {
   var xx = camera_get_view_x(view_camera[0]);
   var yy = camera_get_view_y(view_camera[0]);
   draw_text(xx + 32, yy + 32, "Player 1");
   }
else
   {
   var xx = camera_get_view_x(view_camera[1]);
   var yy = camera_get_view_y(view_camera[1]);
   draw_text(xx + 32, yy + 32, "Player 2");
   }

Der obige Code überprüft, welche Ansicht gerade gezeichnet wird, und zeichnet dann basierend auf dem Rückgabewert einen anderen Text für jede Ansicht.
