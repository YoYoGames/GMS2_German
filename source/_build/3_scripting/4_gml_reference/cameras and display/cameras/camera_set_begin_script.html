camera_set_begin_script


Beschreibung

Mit dieser Funktion kann ein Skript festgelegt werden, das am Anfang jedes Spielfensters aufgerufen wird, das der Kamera als sichtbarer und aktiver Ansichtsport zugewiesen ist. Sie geben den eindeutigen Kamera-ID-Wert (wie vom anderen zurückgegeben) an camera_create Funktionen) und die ID des anzurufenden Skripts. Die Reihenfolge der an die Kameras angehängten Skripts und das tatsächliche Rendern der Kameraansicht ist wie folgt:

  • Die Kameras für alle sichtbaren und aktiven Ansichtsports haben ihr Update-Skript aufgerufen

  • Dann für jede Ansicht:
    • Das Startskript für die Kamera für diese Ansicht wird aufgerufen
    • Die Zeichenereignisse werden für diese Ansicht ausgeführt
    • Das Endskript für die Kamera wird aufgerufen
  • Gehe zur nächsten aktiven sichtbaren Ansicht und wiederhole...


Syntax:

camera_set_begin_script(camera_id, script)


Argumente:

Streit Beschreibung
camera_id Der eindeutige Kamera-ID-Wert, der beim Erstellen der Kamera zurückgegeben wurde.
script Das Skript zum Ausführen jedes Spielrahmens


Kehrt zurück:

N/A


Beispiel:

camera_set_begin_script(view_camera[0], beginCamera);

Der obige Code legt das Startskript für die Kamera fest, die dem Port [0] zugeordnet ist.
