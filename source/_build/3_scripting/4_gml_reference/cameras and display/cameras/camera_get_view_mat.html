camera_get_view_mat


Beschreibung

Diese Funktion kann verwendet werden, um die Ansichtsmatrix abzurufen. Die Funktion gibt den Matrix-ID-Wert zurück, der dann in anderen Matrixfunktionen oder zur Einstellung der Ansichtsmatrix einer anderen Kamera verwendet werden kann (mit camera_set_view_mat ). Beachten Sie, dass diese Funktion im Allgemeinen für Kameras verwendet wird, die mit erstellt wurden camera_create(), kann aber auch für diejenigen verwendet werden, die im Raumeditor erstellt wurden (oder mit camera_create_view ) - In diesem Fall ist die zurückgegebene Matrix erst gültig, nachdem die angegebene Kamera in einem Ansichtsport zum ersten Mal verwendet wurde.


Syntax:

camera_get_view_mat(camera_id)


Argumente:

Streit Beschreibung
camera_id Der eindeutige Kamera-ID-Wert wird beim Erstellen der Kamera zurückgegeben


Kehrt zurück:

Matrix ID (real)


Beispiel:

mat = camera_get_view_mat(camera_view[0]);

Der obige Code ruft die Ansichtsmatrix für die Kamera ab, die dem Ansichtsanschluss [0] zugewiesen ist.
