view_visible


Beschreibung

Diese Variable kann verwendet werden, um herauszufinden, ob ein bestimmter Ansichtsport gegenwärtig sichtbar ist oder nicht. Sie können diese Variable auch so einstellen, dass sie eine Ansicht wirksam ein- oder ausschaltet, indem Sie den Wert auf einstellen true (sichtbar) oder false (unsichtbar). Beachten Sie, dass, selbst wenn Sie einen Ansichtsport auf sichtbar gesetzt haben, wenn Ansichtsports nicht aktiviert sind (mit der Funktion view_enabled oder aktivieren Sie sie im Raum-Editor), dann werden sie nicht auf den Bildschirm gezeichnet.


Syntax:

view_visible[0 ... 7];


Kehrt zurück:

Boolean (true: enabled, false: disabled)


Beispiel:

if !view_visible[0]
   {
   view_visible[0] = true;
   }

Der obige Code überprüft, ob die Sicht [0] sichtbar ist oder nicht, und wenn dies nicht der Fall ist, wird sie als sichtbar festgelegt.
