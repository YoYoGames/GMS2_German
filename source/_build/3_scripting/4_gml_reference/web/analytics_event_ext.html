analytics_event_ext


Beschreibung

Diese Funktion sendet den angegebenen Text an den Analyseanbieter, den Sie über die HTML5-Spieloptionen eingerichtet haben. Diese Funktion kann verwendet werden, um eine benutzerdefinierte Analyse zu erstellen, um etwas außerhalb des Bereichs des verwendeten Anbieters zu verfolgen, und akzeptiert auch benutzerdefinierte Parameter / Wert-Paare, wobei der Parameter eine Zeichenfolge und der Wert eine reelle Zahl ist. Für Google Analytics können Sie nur ein zusätzliches Paar hinzufügen, während Flurry bis zu 7 akzeptiert.


Syntax:

analytics_event_ext(string, string_param, value);


Streit Beschreibung
string Eine Zeichenfolge, die an den Anbieter gesendet wird.
string_param[0 -9] Der zu sendende Parameter (eine Zeichenfolge).
value[0 - 9] Der Wert des Parameters (eine reelle Zahl).


Kehrt zurück:

N/A


Beispiel:

var num, time;
ini_open("Options.ini");
num = ini_read_real("Data", "Plays", 0);
num += 1;
time = current_time;
analytics_event_ext(GAME_NAME, "Plays", num, "Time", time);
ini_write_real("Data", "Plays", num);
ini_close();

Der obige Code wird Spielinformationen von einer INI-Datei sowie der aktuellen Zeit erhalten und diese Details dann an den Analyseanbieter senden.
