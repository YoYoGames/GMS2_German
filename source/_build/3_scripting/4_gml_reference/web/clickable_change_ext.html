clickable_change_ext

Syntax:

clickable_change_ext(index, tpe, x, y, alpha, scale)


Streit Beschreibung
index Der Index des anklickbaren Symbols, das geändert werden soll.
tpe Der Texturseiteneintrag für das zu verwendende Sprite (verwenden Sie sprite_get_tpe ).
x Die neue x-Position innerhalb des Fensters.
y Die neue y-Position innerhalb des Fensters.
scale Die Größe des Symbols (Standard 1).
alpha Das Bild-Alpha des Symbols (Standard 1).


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie das Sprite und die Position eines zuvor erstellten klickbaren Symbols ändern clickable_add. Beachten Sie, dass die Position auf dem Fenster und nicht auf der Canvas-Position (0,0) basiert und dass das Sprite direkt von der Textur-Seite referenziert werden muss (siehe: sprite_get_tpe ). Mit dieser Funktion können Sie auch den Maßstab des verwendeten Sprites ändern (als Multiplikator, so dass 1 der Standardwert ist, 0,5 wäre die Hälfte und 2 wären doppelt) und der Alpha-Wert von 0 (vollständig transparent) bis 1 (vollständig undurchsichtig) für das endgültige Symbol auf dem Bildschirm.


Beispiel:

switch (room)
   {
   case rm_Menu: clickable_change_ext(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 1), 32, 32, 2, 1); break;
   case rm_Game: clickable_change_ext(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 0), 200, 32, 1, 0.5); break;
   }

Der obige Code ändert den Bildindex und die Position des klickbaren Symbols, das in der Variable "global.Help" indiziert ist, abhängig vom aktuellen Raum, wobei auch die Symbolskalierung und Alpha geändert werden.
