clickable_add_ext

Syntax:

clickable_add_ext(x, y, tpe, url, target, params, scale, alpha);


Streit Beschreibung
x Die x-Position innerhalb des Fensters.
y Die y-Position innerhalb des Fensters.
tpe Der Textur-Seiteneintrag für das zu verwendende Sprite (verwenden Sie sprite_get_tpe ).
url Die URL (Website-Adresse), zu der verlinkt werden soll, oder das Skript, das ausgeführt werden soll.
target Dies ist der Zielbereich zum Öffnen der URL (siehe Beschreibung).
params Die verschiedenen Parameter für das Symbol (siehe Beschreibung).
scale Der Maßstab des Symbols (Standard 1).
alpha Das Bild-Alpha des Symbols (Standard 1).


Rückgabe: Real


Beschreibung

Mit dieser Funktion können Sie ein benutzerdefiniertes, anklickbares Symbol erstellen, das als "Schaltfläche" außerhalb des Spielbereichs, jedoch innerhalb des Browserfensters selbst fungiert. Die Funktion gibt einen ID-Wert zurück, der in einer Variablen gespeichert werden muss, um in allen weiteren Tastenfunktionen verwendet werden zu können, die auf das neue Symbol verweisen. Diese Funktion ist sehr nützlich, um zu verhindern, dass der Browser beim Klicken auf die Schaltfläche ein Popup-Fenster erstellt, da dies alles über GameMaker Studio 2 selbst erfolgt.

Diese Funktion funktioniert genauso wie die clickable_add Funktion, bei der Sie die Position relativ zu (0,0) des Browserfensters festlegen müssen, geben Sie den Eintrag für die Texturseite für das Sprite an, das das Schaltflächensymbol haben soll (siehe: sprite_get_tpe ), geben Sie entweder eine zu öffnende URL oder ein Skript (aus dem Ressourcenbaum von GameMaker Studio 2 ) zum Öffnen oder Ausführen an und legen Sie das zu verwendende Ziel und die zu verwendenden Parameter fest. Diese Funktion verfügt jedoch auch über zwei zusätzliche Optionen, mit denen Sie das Symbol Alpha von 0 (vollständig transparent) auf 1 (vollständig undurchsichtig) und die Skalierung des Symbols festlegen können. Hierbei handelt es sich um einen Multiplikatorwert, der auf das verwendete Sprite angewendet wird. Ein Wert von 1 ist also die Standardskala, während 0,5 zum Beispiel die halbe Skala und 2 die doppelte wäre.

Es ist auch erwähnenswert, dass, wenn das URL-Argument, das Sie übergeben, der Name eines Skripts ist (dh: es beginnt nicht mit "http: //") UND das Skript aufgerufen wird gmcallback_* Dann können Sie direkte Feeds in ein HTML5-Ereignis einbauen. Funktionsnamen von GameMaker Studio 2 werden nicht verschleiert, wenn sie mit beginnen gmcallback_.


Beispiel:

home_but = clickable_add_ext(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0", 2, 1);

Der obige Code erstellt ein anklickbares DOM-Symbol an der Position (32, 32) der Seite, auf der der Game Canvas ausgeführt wird. Das Symbol verwendet das von der Textur-Seite referenzierte Sprite als "spr_MS_Home". Wenn Sie darauf klicken, öffnet das Symbol ein neues Fenster für die angegebene URL und die für dieses Fenster definierten Eigenschaften. Die Schaltfläche wird außerdem auf das Doppelte der ursprünglichen Sprite-Größe und ein halbes Bild-Alpha von 1 skaliert.
