clickable_add_ext

Syntax:

clickable_add_ext(x, y, tpe, url, target, params, scale, alpha);


Streit Beschreibung
x Die x-Position innerhalb des Fensters.
y Die y-Position innerhalb des Fensters.
tpe Der Texturseiteneintrag für das zu verwendende Sprite (verwenden Sie sprite_get_tpe ).
url Die URL (Website-Adresse), auf die verlinkt werden soll, oder das Skript, das ausgeführt werden soll.
target Dies ist der Zielbereich zum Öffnen der URL (siehe Beschreibung).
params Die verschiedenen Parameter für das Symbol (siehe Beschreibung).
scale Die Größe des Symbols (Standard 1).
alpha Das Bild-Alpha des Symbols (Standard 1).


Rückgabe: Echt


Beschreibung

Mit dieser Funktion können Sie ein benutzerdefiniertes, anklickbares Symbol erstellen, das als "Schaltfläche" außerhalb des Spielbereichs, jedoch innerhalb des Browserfensters selbst fungiert. Die Funktion gibt einen ID-Wert zurück, der in einer Variablen gespeichert werden muss, um in allen weiteren Schaltflächenfunktionen verwendet zu werden, die auf das neue Symbol verweisen. Diese Funktion ist sehr nützlich, um zu verhindern, dass der Browser ein Popup-Fenster erstellt, wenn er auf die Schaltfläche klickt, da dies alles in GameMaker: Studio selbst geschieht.

Diese Funktion funktioniert genauso wie die clickable_add Funktion, wo Sie die Position relativ zum (0,0) des Browserfensters einstellen müssen, geben Sie den Texturseiteneintrag für das Sprite ein, das das Buttonsymbol haben soll (siehe: sprite_get_tpe ), geben Sie entweder eine zu öffnende URL oder ein Skript (aus dem Ressourcenbaum von GameMaker: Studio ) an, das geöffnet oder ausgeführt werden soll. Legen Sie dann das zu verwendende Ziel und die Parameter fest. Diese Funktion verfügt jedoch auch über zwei zusätzliche Optionen, mit denen Sie das Symbol Alpha von 0 (vollständig transparent) auf 1 (vollständig undurchsichtig) und die Skalierung des Symbols einstellen können. Dies ist ein Multiplikatorwert, der auf das verwendete Sprite angewendet wird. Also ist ein Wert von 1 die Standardskala, während (zum Beispiel) 0,5 eine halbe Skala wäre und 2 wäre doppelt.

Es ist auch erwähnenswert, dass das übergebene URL-Argument der Name eines Skripts ist (dh es beginnt nicht mit "http: //") UND das Skript wird aufgerufen gmcallback_* Dann können Sie direkte Feeds in ein html5-Event bekommen. GameMaker: Studio Funktionsnamen werden nicht verschleiert, wenn sie mit beginnen gmcallback_.


Beispiel:

home_but = clickable_add_ext(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0", 2, 1);

Der obige Code erstellt ein anklickbares DOM-Symbol an der Position (32, 32) der Seite, auf der das Spielfenster läuft. Das Symbol verwendet das Sprite, auf das von der Texturseite verwiesen wird, als "spr_MS_Home". Wenn Sie darauf klicken, öffnet das Symbol ein neues Fenster für die angegebene URL und mit den definierten Eigenschaften für dieses Fenster. Die Schaltfläche wird außerdem auf das Doppelte der ursprünglichen Sprite-Größe und auf ein halbes Bild Alpha von 1 skaliert.
