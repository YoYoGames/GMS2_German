url_open_ext

Syntax:

url_open_ext(url, target);


Streit Beschreibung
url Die URL (Website-Adresse), zu der verlinkt werden soll.
target Dies ist der Zielbereich zum Öffnen der URL.


Rückgabe: N / A


Beschreibung

Dadurch wird die angegebene URL im Browser des ausgewählten Zielgeräts oder, wenn Sie das HTML5-Modul verwenden, im aktuell geöffneten Browser geöffnet. Der von Ihnen angegebene "Ziel" -Parameter entspricht dem Standardwert für den JavaScript-Namen, wenn Sie den open() Sie sollten wissen, dass alle außer '_self' dazu führen können, dass der Browser blockiert oder der Benutzer gefragt wird, ob er dies zulassen möchte.

Gültige Ziele sind:

"_self": Öffnet das verknüpfte Dokument in dem Rahmen, auf den geklickt wurde (dies ist die Standardeinstellung).
"_blank": Öffnet das verknüpfte Dokument in einem neuen Fenster oder einer neuen Registerkarte (dies funktioniert nicht, wenn Popups vom Benutzer blockiert werden. In diesem Fall können Sie das verwenden clickable_* stattdessen Funktionen).
"_parent": Öffnet das verknüpfte Dokument im übergeordneten Frame.
"_top": Öffnet das verknüpfte Dokument im gesamten Hauptteil des Fensters.


Beispiel:

url_open_ext( 'http://yoyogames.com', '_blank');

Dies öffnet die YoYo Games Homepage in einem neuen Fenster.
