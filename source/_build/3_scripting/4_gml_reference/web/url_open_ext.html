url_open_ext

Syntax:

url_open_ext(url, target);


Streit Beschreibung
url Die URL (Website-Adresse), auf die verlinkt werden soll.
target Dies ist der Zielbereich, in dem die URL geöffnet werden soll.


Rückkehr: N / A


Beschreibung

Dadurch wird die angegebene URL im Browser des ausgewählten Zielgeräts oder, wenn Sie das HTML5-Modul verwenden, im derzeit geöffneten Browser geöffnet. Der von Ihnen angegebene Parameter "target" entspricht bei Verwendung des Befehls "name" dem Standard-JavaScript-Wert open() Methode, und Sie sollten sich bewusst sein, dass alle außer "_self" dazu führen können, dass der Browser blockiert, oder fragen Sie den Benutzer, ob er dies zulassen möchte.

Gültige Ziele sind:

"_self": Öffnet das verknüpfte Dokument in demselben Frame, in dem es angeklickt wurde (dies ist der Standardwert).
"_blank": Öffnet das verknüpfte Dokument in einem neuen Fenster oder einer neuen Registerkarte (dies funktioniert nicht, wenn Popups vom Benutzer blockiert werden. In diesem Fall können Sie das clickable_* funktioniert stattdessen).
"_parent": Öffnet das verknüpfte Dokument im übergeordneten Frame.
"_top": Öffnet das verknüpfte Dokument im gesamten Fensterbereich.

HINWEIS: Auf der Windows 8 (JS) Zielplattform, wird diese Funktion arbeiten, aber zusätzliche Parameter und der Standard - Browser mit der angegebenen URL gestartet werden ignoriert.


Beispiel:

url_open_ext( 'http://yoyogames.com', '_blank');

Dies öffnet die YoYo Games Homepage in einem neuen Fenster.
