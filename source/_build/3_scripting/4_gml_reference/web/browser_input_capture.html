browser_input_capture

Syntax:

browser_input_capture(enable)


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie einstellen, ob das Browserfenster alle Eingaben erfassen soll (stellen Sie es auf ein false ) oder ob das Spiel die Eingabe erfassen soll (auf true ). Beachten Sie, dass diese Funktion nur für das HTML5-Modul verwendet werden kann.


Beispiel:

browser_input_capture(true);

Der obige Code legt das Spiel fest, um alle Browser-Eingaben auf der HTML5-Zielplattform zu erfassen.
