url_open_full

Syntax:

url_open_full(url, target, options);


Streit Beschreibung
url Die URL (Website-Adresse), zu der verlinkt werden soll.
target Dies ist der Zielbereich zum Öffnen der URL (siehe Beschreibung).
options Standardbrowseroptionen (siehe Beschreibung).


Rückgabe: N / A


Beschreibung

Dadurch wird die angegebene URL im Browser des ausgewählten Zielgeräts oder, wenn Sie das HTML5-Modul verwenden, im aktuell geöffneten Browser geöffnet. Der von Ihnen angegebene "Ziel" -Parameter entspricht dem Standardwert für den JavaScript-Namen, wenn Sie den open() method (beachten Sie, dass alle außer '_self' dazu führen können, dass der Browser blockiert oder den Benutzer fragt, ob er dies zulassen möchte) und dass die "Optionen" mit dem JavaScript-Parameter "specs" identisch sind, um zu steuern, welche Eigenschaften das neue Fenster hat / tab sollte angezeigt werden (möglicherweise unterstützen nicht alle Browser alle Funktionen).

Gültige Ziele sind:

"_self": Öffnet das verknüpfte Dokument in dem Rahmen, auf den geklickt wurde (dies ist die Standardeinstellung).
"_blank": Öffnet das verknüpfte Dokument in einem neuen Fenster oder einer neuen Registerkarte (dies funktioniert nicht, wenn Popups vom Benutzer blockiert werden. In diesem Fall können Sie das verwenden clickable_* stattdessen Funktionen).
"_parent": Öffnet das verknüpfte Dokument im übergeordneten Frame.
"_top": Öffnet das verknüpfte Dokument im gesamten Hauptteil des Fensters.

Gültige Optionen sind:

"height = [px]": Die Höhe des Fensters mit einem Mindestwert von 100.
"width = [px]": Die Breite des Fensters mit einem Mindestwert von 100.
"left = [px]": Die linke Position des Fensters.
"top = [px]": Die oberste Position des Fensters (nur IE).
"location = [boolean]": Angabe, ob das Adressfeld angezeigt werden soll oder nicht (Standardeinstellung ist 1).
"menubar = [boolean]": Anzeige der Menüleiste (Standardeinstellung ist 1).
"resizable = [boolean]": Ob das Fenster in der Größe verändert werden kann oder nicht (Standardeinstellung ist 1).
"scrollbars = [boolean]": Angabe, ob Bildlaufleisten angezeigt werden sollen oder nicht (Standardeinstellung ist 1).
"status = [boolean]": Gibt an, ob eine Statusleiste hinzugefügt werden soll oder nicht (Standardeinstellung ist 1).
"titlebar = [boolean]": Gibt an, ob die Titelleiste angezeigt werden soll oder nicht. Dies wird ignoriert, es sei denn, die aufrufende Anwendung ist eine HTML-Anwendung oder ein vertrauenswürdiges Dialogfeld (der Standardwert ist 1).
"toolbar = [boolean]": Gibt an, ob die Browser- Symbolleiste angezeigt werden soll oder nicht (Standardeinstellung ist "Ja").


Beispiel:

url_open_full( 'http://yoyogames.com', '_blank', 'resizable=0, height=200, scrollbars=0');

Dies würde die YoYo Games Startseite in einem neuen Fenster öffnen, das nicht in der Größe geändert werden kann, eine Höhe von 200 Pixel hat und keine Bildlaufleisten enthält.
