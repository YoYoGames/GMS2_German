Die unten aufgeführten Funktionen sind allgemeine Web-Funktionen, die auf allen Zielplattformen verwendet werden können:

  • url_open
  • url_open_ext
  • url_open_full
  • url_get_domain

Neben den oben aufgeführten allgemeinen Funktionen verfügt das HTML5-Ziel über einige zusätzliche Webfunktionen, die speziell für die Spiele entwickelt wurden, auf die es ausgerichtet ist:

  • browser_height
  • browser_breite
  • browser_input_capture
  • webgl_enabled
  • klickbare_exists
  • klickbare_add
  • clickable_add_ext
  • clickable_change
  • clickable_change_ext
  • clickable_set_style
  • clickable_delete
  • analytik_event
  • analytics_event_ext
  • os_browser

Neben diesen Funktionen für das HTML5-Target-Modul gibt es auch eine spezielle JavaScript-Variable, die in den seltenen Fällen verwendet werden kann, in denen der Inhalt des html5game-Ordners an einer völlig anderen Stelle (URL) gespeichert ist. Normalerweise ist dieser Pfad in der JavaScript-Datei fest codiert und relativ zum Speicherort, von dem aus JavaScript aufgerufen wird. Sie können jedoch die Variable festlegen g_GameMakerHTML5Dir vor dem Aufruf des Spiels Skript und das Spiel wird von dem neu definierten Ort laufen. Beispielsweise:

<script>var g_GameMakerHTML5Dir = "http://mygamealternatedirectory.com/";</script>
