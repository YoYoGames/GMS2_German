iap_product_details

Fügen Sie Details zu einem einzelnen Produkt zu einer ds_map hinzu.

Syntax:

iap_product_details(product_id, ds_map);


Streit Beschreibung
purchase_id Die Produkt-ID-Zeichenfolge für den Kauf.
ds_map Die ds_map, die die Produktinformationen speichert.


Rückkehr: N / A


Beschreibung

Mit dieser Funktion können Sie ein zuvor erstelltes Objekt auffüllen ds_map mit einer Anzahl von Schlüssel / Wert-Paaren, die den gegebenen Kauf beschreiben. Die folgenden Daten werden in der Karte zurückgegeben:

  • " id " - Die Produkt-ID-Zeichenfolge.

  • " Titel " - Der Titel des Produkts.

  • " Beschreibung " - Die Beschreibung des Produkts.

  • " Preis " - Der Preis des Produkts, sofern verfügbar (nicht alle Geschäfte unterstützen dies).

  • " type " - Der Produkttyp, der entweder "Consumable" oder "Durable" ist, sofern verfügbar (nicht alle Shops unterstützen dies).

  • " verifiziert " - Wird entweder sein true oder false (zB: reelle Zahlen 0 oder 1), abhängig davon, ob Ihr Spiel Details zu diesem Produkt aus dem Store erhalten hat oder nicht, und glaubt daher, dass dieses Produkt zum Kauf verfügbar ist.

HINWEIS: Die ds_map used wird nicht von der Funktion erstellt, dh Sie müssen es zuvor erstellen und es aus dem Speicher entfernen, wenn es nicht mit der entsprechenden Funktion verwendet wird.


Beispiel:

var p_map = ds_map_create();
iap_product_details(ds_map_find_value(iap_data, "index"), p_map);
p_title = " + string(ds_map_find_value(p_map, "title")));
p_description = " + string(ds_map_find_value(p_map, "description")));
ds_map_destroy(p_map);

Der obige Code erstellt ein ds_map und füllen Sie sie dann mit den Produktdetails für das Produkt aus dem Special iap_data ds_map, die ausschließlich im IAP-Event erstellt wird. Der "Titel" und die "Beschreibung" werden dann zu den Variablen hinzugefügt, damit sie später auf dem Bildschirm gezeichnet werden können.
