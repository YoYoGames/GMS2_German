iap_consume

Löscht den Kaufstatus nicht verwalteter Käufe.

Syntax:

iap_consume(purchase_id);


Streit Beschreibung
purchase_id Die Produkt-ID-Zeichenfolge für den Kauf.


Rückkehr: N / A


Beschreibung

Diese Funktion kann verwendet werden, um den Kaufstatus von nicht verwalteten (verbrauchbaren) Käufen zu löschen. Die "purchase_id" ist die Produkt-ID-Zeichenfolge, die Sie beim Aktivieren des IAP mit angegeben haben iap_activate und sollte identisch mit dem sein, mit dem Sie das Produkt im Zielspeicher identifiziert haben. Diese Funktion löst ein IAP Event des Typs iap_ev_consume, die folgende zusätzliche Schlüssel enthalten wird:

  • " Produkt " - Die Produkt-ID für das konsumierte Produkt.

Anhand dieser Produkt-ID kann dann überprüft werden, welcher Einkauf das Ereignis ausgelöst hat, und Sie können entsprechend damit umgehen.

WARNUNG: Sie können ein Verbrauchsmaterial nicht mehrmals hintereinander ohne Verbrauch kaufen - Sie müssen zwischen folgenden Käufen einen Verbrauchsvorgang durchführen.


Beispiel:

if mouse_check_button_pressed(mb_left)
   {
   iap_consume("noads")
   }

Der obige Code löscht den Kaufstatus, wenn die Maustaste gedrückt wird.
