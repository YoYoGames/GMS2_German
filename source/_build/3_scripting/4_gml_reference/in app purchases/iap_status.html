iap_status

Ermöglicht Ihnen, den Status des In-App-Einkaufssystems zu verfolgen.

Syntax:

iap_status();


Rückgabe: Konstante


Beschreibung

Mit dieser Funktion können Sie den Status von GameMaker Studio 2 im App-Einkaufssystem verfolgen und eine der folgenden Konstanten zurückgeben:

Konstante Beschreibung
iap_status_uninitialised In App-Käufe wurden für dein Spiel nicht aktiviert.
iap_status_unavailable Für den Zielspeicher ist keine Verbindung verfügbar.
iap_status_loading Eine Verbindung zum Zielspeicher wird beendet und aktiviert.
iap_status_available Einkäufe wurden aktiviert und der Zielspeicher ist verfügbar.
iap_status_processing Es gibt momentan eine Kommunikation zwischen dem Spiel und dem Zielladen, zum Beispiel wenn (zum Beispiel) ein Kauf getätigt wird oder wenn ein Kauf wiederhergestellt wird, etc...
iap_status_restoring Auf dem iOS- Ziel kann dies zurückgegeben werden, während das Spiel versucht, Einkäufe wiederherzustellen.


Beispiel:

status = iap_status();

Der obige Code fordert den Speicherstatus für das aktuelle Spiel an und speichert ihn in einer Variablen zum Prüfen.
