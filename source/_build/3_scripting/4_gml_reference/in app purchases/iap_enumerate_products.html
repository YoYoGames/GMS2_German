iap_enumerate_products

Füllen Sie eine ds_list mit allen derzeit verfügbaren IAP-Produkten aus dem Zielspeicher aus.

Syntax:

iap_enumerate_products();


Rückkehr: N / A


Beschreibung

Diese Funktion ruft die Liste der gefundenen Produkt-IDs in App-Kauf ab. Auf den meisten Plattformen gibt dies einfach den Satz von Produkt-IDs zurück, die durch bereitgestellt werden iap_activate(), aber auf Plattformen, auf denen der Laden die Möglichkeit bietet, eine vollständige Liste verfügbarer Produkte für einen Titel abzurufen, enthält dieser Satz von Produkt-IDs alle verfügbaren Produkt-IDs für das Spiel. Die Funktion erfordert, dass Sie ein zuvor erstelltes Objekt bereitstellen ds_list (die Sie zuerst löschen sollten, wenn sie für andere Zwecke verwendet wurde), die durch den Funktionsaufruf gefüllt wird. Unabhängig davon, ob der Zielspeicher nicht verfügbar ist, erhält er die Produkt-IDs, die bei der Aktivierung von Käufen mit angegeben wurden iap_activate.


Beispiel:

var status = iap_status();
var iap_list = ds_list_create();
if status == iap_status_available
   {
   iap_enumerate_products(iap_list);
   }

Der obige Code überprüft den Status des Ziel-IAP-Speichers. Wenn dieser verfügbar ist, füllt er die erstellte Liste mit den verfügbaren Produkt-IDs für die zukünftige Verwendung auf.
