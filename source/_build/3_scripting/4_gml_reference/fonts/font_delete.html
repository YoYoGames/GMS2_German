font_delete


Beschreibung

Mit dieser Funktion können Sie eine Schriftart aus dem Spiel löschen. Dies ist eine dauerhafte Entfernung, und das Umkleiden oder Neustarten des Spiels bringt die entfernte Schriftart nicht zurück. Dazu müsste der Spieler das Spiel verlassen und auf diese Weise neu starten. Seien Sie also vorsichtig, wenn Sie diese Funktion verwenden. Im Allgemeinen wird es nur benötigt, um Speicher freizugeben, der von einer Schriftart verwendet wurde, die dem Spiel über die Funktionen hinzugefügt wurde font_add oder font_add_sprite.

WICHTIG! Diese Funktion ist mit der Testlizenz des Produkts nicht verfügbar.


Syntax:

font_delete(ind);

Streit Beschreibung
ind Der Index der zu löschenden Schriftart.


Kehrt zurück:

N/A


Beispiel:

font_delete(global.Font);

Der obige Code löscht die Schriftart, die in der globalen Variable "Font" indiziert ist.
