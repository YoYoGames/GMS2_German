font_add


Beschreibung

Mit dieser Funktion können Sie Ihrem Spiel aus den Zeichensätzen, die auf dem System installiert sind, eine Schriftart hinzufügen. Sie können die Größe der Schrift (in Punkten) sowie die Angabe, ob die Schrift fett oder kursiv sein soll, sowie den Zeichenbereich festlegen, der eingeschlossen werden soll (eine vollständige Tabelle der verfügbaren Zeichen und ihren UTF8-Wert finden Sie unter Schriftart.) Tabellen ). Die Funktion gibt einen Indexwert zurück, der in einer Variablen gespeichert werden sollte, da dies in allen weiteren Codes erforderlich ist, die auf diese Schriftart verweisen.

WICHTIG! Diese Funktion ist nicht mit der Testlizenz des Produkts verfügbar.


Für das HTML5-Zielmodul kann diese Funktion nur zum Hinzufügen von Webfonts verwendet werden und funktioniert nur, wenn WebGL nicht aktiviert ist. Wenn WebGL aktiviert ist, müssen Sie eine allgemeine Fontressource verwenden (z. B. using draw_set_font zusammen mit einer Schriftart aus dem Ressourcenbaum) oder Bitmap-Schriftarten (siehe font_add_sprite ).

WICHTIG! Bei der Ausrichtung auf HTML5 werden kursiver und fetter Text nur unterstützt, wenn das Schriftartenasset oder die Webschriftart selbst diese Zeichen enthält. Sie werden nicht aus der Basisschriftart synthetisiert.

Bei allen anderen Zielen (nicht Webzielen) können Sie diese Funktion verwenden, um eine Schriftart aus einer Datei hinzuzufügen. Die Datei muss mithilfe der Funktion " GameMaker Studio 2 Files" von GameMaker Studio 2 und muss ein *.ttf Format-Font-Datei, nützlich zum Hinzufügen von nicht standardmäßigen Schriftarten wie Asiatisch oder Arabisch.

WARNUNG!: Wenn Sie eine Schrift einfügen *.ttf mit einem Spiel formatieren, muss es für die Verteilung mit dem Spiel lizenziert werden.


Wenn Sie eine Schriftart aus einer Datei auf diese Weise laden, wird die Größe der Schriftart in Pixel angegeben, und der erste und der letzte Wert werden ignoriert. Das bedeutet, dass alle Glyphen in der Schriftart hinzugefügt werden. Auf diese Weise hinzugefügte Schriftarten werden einer eigenen Textur-Seite zugewiesen. Sie sollten daher bei der Verwendung dieser Funktion vorsichtig sein, da die Anzahl der Textur-Swaps beim Zeichnen erhöht wird. Es ist auch erwähnenswert, dass Schriftarten beim Zeichnen etwas größer erscheinen, da Glyphen Teile enthalten können, die außerhalb des für die Schrift definierten Begrenzungsrahmens gezeichnet werden. Sie sollten sich auch bewusst sein, dass es eine Beschränkung auf 200 Zeichen gibt, die in einem einzelnen Frame für eine bestimmte Schriftart gerendert werden können.

HINWEIS: Wenn Sie eine Schrift in GameMaker Studio 2: Studio laden, müssen Sie sie unbedingt wieder entfernen (mit font_delete ) wenn es nicht mehr benötigt wird, andernfalls besteht die Gefahr eines Speicherverlusts, der das Spiel verlangsamt und letztendlich zum Absturz bringt.


Syntax:

font_add(name, size, bold, italic, first, last);


Streit Beschreibung
name Der Name der hinzuzufügenden Schriftart (z. B. 'Arial') oder der Dateipfad, falls die Schriftart enthalten ist *.ttf Datei.
size Die Größe der Schriftart - Punkte für Webfonts, Pixel für *.ttf Schriftarten.
bold Ob die Schriftart fett ist (true) oder nicht (false).
italics Gibt an, ob die Schriftart kursiv ist (true) oder nicht (false).
first Das erste Zeichen, das eingefügt werden soll (wenn Sie sich nicht sicher sind, gehen Sie auf 32).
last Das letzte Zeichen, das eingefügt werden soll (wenn Sie sich nicht sicher sind, gehen Sie für 128).


Kehrt zurück:

Real


Beispiel:

newfont = font_add( 'Arial', 24, true, true, 32, 128);

Dadurch wird eine neue Schriftart mit einer Größe von 24 Pixeln erstellt, die fett und kursiv "Arial" verwendet wird. Der Index für diese neue Schriftart wird in der Variablen "newfont" gespeichert. Der Schriftbereich umfasst Groß- und Kleinbuchstaben, Zahlen und alle gängigen Satzzeichen.
