font_add


Beschreibung

Diese Funktion kann verwendet werden, um eine Schriftart aus den Schriftarten hinzuzufügen, die auf dem System installiert sind, auf dem sie ausgeführt wird. Sie können die Größe der Schriftart (in Punkten) sowie die Schriftart fett oder kursiv definieren und den Bereich der zu berücksichtigenden Zeichen definieren (eine vollständige Tabelle der verfügbaren Zeichen und ihren UTF8-Wert finden Sie unter Schriftart Tabellen ). Die Funktion gibt einen Indexwert zurück, der in einer Variablen gespeichert werden sollte, da dies in allen weiteren Codes, die sich auf diese Schriftart beziehen, benötigt wird.

WICHTIG! Diese Funktion ist mit der Testlizenz des Produkts nicht verfügbar.


Für das HTML5-Zielmodul kann diese Funktion nur zum Hinzufügen von Webfonts verwendet werden, funktioniert jedoch nur, wenn WebGL nicht aktiviert ist. Wenn WebGL aktiviert ist, müssen Sie eine enthaltene Schriftartdatei verwenden. Die folgende Tabelle zeigt die Schriftarten, die standardmäßig in allen Browsern verwendet werden und für die Verwendung ohne Probleme verfügbar sein sollten. Jede andere Schriftart kann auf dem Computer oder Gerät, auf dem das Spiel ausgeführt wird, vorhanden sein oder nicht. Daher sollte die Verwendung dieser Funktion auf diese Standardschriftarten beschränkt sein:

Bei allen anderen (nicht JavaScript-) Zielen können Sie mit dieser Funktion eine Schriftart aus einer Datei hinzufügen. Die Datei muss mit der Funktion " Included Files" von GameMaker Studio 2 in das GameMaker Studio 2 werden und muss ein *.ttf Format Schriftdatei, nützlich für das Hinzufügen von nicht-Standard-Schriftarten wie asiatische oder arabische.

WARNUNG!: Wenn Sie eine Schriftart einbeziehen *.ttf Format mit einem Spiel, muss es für die Verteilung mit dem Spiel lizenziert werden.


Wenn Sie auf diese Weise eine Schriftart aus einer Datei laden, wird die Größe der Schriftart in Pixeln angegeben. Der erste und der letzte Wert werden ignoriert, dh alle Glyphen in der Schriftart werden hinzugefügt. Fonts, die auf diese Weise hinzugefügt wurden, werden ihrer eigenen Textur-Seite zugewiesen. Bei der Verwendung dieser Funktion sollte daher Vorsicht walten, da die Anzahl der Textur-Swaps beim Zeichnen erhöht wird. Es ist auch erwähnenswert, dass Schriftarten beim Zeichnen etwas größer erscheinen, da Glyphen Teile haben können, die außerhalb des für die Schriftart definierten Begrenzungsrahmens gezeichnet werden. Sie sollten sich darüber im Klaren sein, dass es eine Begrenzung von ungefähr 200 Zeichen gibt, die in einem einzelnen Bild für eine bestimmte Schriftart gerendert werden können.

HINWEIS: Wenn Sie eine Schriftart in GameMaker Studio 2: Studio laden, müssen Sie daran denken, sie erneut zu entfernen (mit font_delete ) wenn es nicht mehr benötigt wird, besteht sonst die Gefahr eines Speicherlecks, das das Spiel verlangsamt und schließlich zum Absturz bringt.


Syntax:

font_add(name, size, bold, italic, first, last);


Streit Beschreibung
name Der Name der hinzuzufügenden Schriftart (z. B. "Arial") oder der Dateipfad, wenn die Schriftart enthalten ist *.ttf Datei.
size Die Größe der Schriftart - Punkte für Web Fonts, Pixel für *.ttf Schriftarten.
bold Ob die Schriftart fett (wahr) oder nicht (falsch) ist.
italics Ob die Schriftart kursiv (wahr) oder nicht (falsch) ist.
first Das erste Zeichen, das enthalten sein soll (wenn du unsicher bist, gehe auf 32).
last Das letzte Zeichen, das eingeschlossen werden soll (wenn Sie sich nicht sicher sind, wählen Sie 128).


Kehrt zurück:

Real


Beispiel:

newfont = font_add( 'Arial', 24, true, true, 32, 128);

Dies erzeugt eine neue Schriftart mit einer Größe von 24pt, verwendet "Arial", fett und kursiv, und der Index für diese neue Schriftart wird in der Variable "newfont" gespeichert. Der Schriftartenbereich umfasst Groß- und Kleinbuchstaben, Zahlen und alle gängigen Interpunktionszeichen.
