font_replace_sprite_ext


Beschreibung

Diese Funktion ersetzt eine Sprite-Schriftart, die Sie zuvor mit der Funktion erstellt haben font_add_sprite_ext. Es verwendet einen "Sprite-Strip", wobei jedes Teilbild ein einzelnes Symbol oder ein einzelner Buchstabe ist und die Bilder basierend auf dem Argument "string_map" der Funktion auf Buchstaben abgebildet werden. Dieses Argument ist eine Zeichenfolge, mit der Sie GameMaker Studio 2 mitteilen können, in welcher Reihenfolge die Teilbilder der Sprite-Schriftart angeordnet sind, und diese automatisch beim Schreiben von Text zuordnen. So zum Beispiel, wenn Sie einen String-Karte „AaBbCcDdEeFfGgHh“ haben, Ihre Sprite Schriftart auf diese Weise haben die Teilbilder bestellt müssen.

Sie können den Abstand für die Schriftart auch proportional festlegen ( true ) oder nicht ( false ), wobei eine proportionale Schriftart basierend auf der tatsächlichen Breite der Buchstaben beabstandet ist (so nimmt beispielsweise der Buchstabe "i" weniger Platz ein als der Buchstabe "w"), während eine nichtproportionale Schriftart basierend auf der Unterteilung beabstandet ist. Bildbreite, wie eine monospaced Schriftart. Schließlich können Sie den Abstand definieren, der beim Schreiben zwischen den einzelnen Buchstaben verbleibt. Dies kann ein beliebiger ganzzahliger Wert sein, wobei 0 kein Leerzeichen ist (die Buchstaben werden "berühren", wenn sie proportional sind). Die Funktion gibt einen Indexwert zurück, der in einer Variablen gespeichert werden sollte, da dies in allen weiteren Codes, die sich auf diese Schriftart beziehen, benötigt wird.

HINWEIS: Für diese Funktion müssen Sie zuvor eine Sprite-Schriftart mit verwenden font_add_sprite(), da es das ersetzen wird und nicht für Schriftarten verwendet werden kann, die als Teil der Spielressourcen enthalten sind.
WICHTIG! Diese Funktion ist mit der Testlizenz des Produkts nicht verfügbar.


Syntax:

font_replace_sprite_ext(font, spr, string_map, prop, sep);

Streit Beschreibung
font Die zu ersetzende Schriftart.
spr Das Sprite zum Hinzufügen einer Schriftart basierend auf.
string_map Zeichenfolge, aus der die Sprite-Teilbildreihenfolge ausgewählt wird.
prop Als proportionale Schriftart festlegen oder nicht.
sep Der Platz zwischen jedem Buchstaben zu verlassen.


Kehrt zurück:

N/A


Beispiel:

font_replace_sprite_ext(global.Font, spr_CalcFont, "0123456789+-*/=", true, 2);

Der obige Code ersetzt ein Sprite-Font-Asset, das in der Variablen "global.Font" mit Sprite-Images indiziert ist, die in der Variablen "spr_CalcFont" indiziert sind.
