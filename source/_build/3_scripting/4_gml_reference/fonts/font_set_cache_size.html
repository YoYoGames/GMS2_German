font_set_cache_size


Beschreibung

Beim Durchführen von Bildüberblendung und -färbung kann HTML5 dies nicht auf die gleiche Art und Weise tun, wie eine ausführbare Datei ausgeführt werden könnte. Daher muss GameMaker Studio 2 vorübergehend eine gemischte Kopie der Bilder speichern und bei Bedarf laden. Diese Funktion legt fest, wie viele gemischte Kopien der gegebenen Schriftart zwischengespeichert werden können, bevor alte überschrieben werden. Der Standardwert ist 4.

HINWEIS: Für Sprite-Fonts sollten Sie die entsprechende Funktion für Sprites verwenden, sprite_set_cache_size.


Syntax:

font_set_cache_size(ind, max);


Streit Beschreibung
ind Der Index der Schriftart zum Ändern der Cachegröße von.
max Die maximale Anzahl der zwischengespeicherten Kopien der Schriftart, die gespeichert werden können.


Kehrt zurück:

N/A


Beispiel:

font_set_cache_size(fnt_MainMenu, 2);

Dadurch wird der Font-Cache der Schriftart, die in der Variablen "fnt_MainMenu" indiziert ist, auf 2 Kopien gesetzt.
