font_replace_sprite


Beschreibung

Mit dieser Funktion können Sie einen "Sprite-Streifen" verwenden, um einen zuvor erstellten Sprite-Zeichensatz zu ersetzen, wobei jedes Unterbild ein einzelnes Symbol oder ein einzelner Buchstabe wäre. Der Index ist die UTF8-Zuordnungsnummer für das erste zu verwendende Zeichen. Die zu verwendenden Bereiche sind die gleichen wie bei einer normalen Schriftart, beginnend mit "!" Denken Sie jedoch daran, dass Sie nur das aufnehmen sollten, was Sie brauchen und dass Sie das verwenden sollten ord Funktion, um den korrekten UTF8-Wert für den Anfangsbuchstaben zu erhalten.

Sie können den Abstand der Schrift auch proportional einstellen ( true ) oder nicht ( false ), wenn ein proportionaler Zeichensatz basierend auf der tatsächlichen Breite der Buchstaben beabstandet ist (so nimmt der Buchstabe "i" beispielsweise weniger Platz als der Buchstabe "w"), während ein nichtproportionaler Zeichensatz basierend auf dem Unterabstand angeordnet wird. Bildbreite, wie eine Monospace-Schrift. Schließlich können Sie das Leerzeichen definieren, das beim Schreiben zwischen den einzelnen Buchstaben bleiben soll. Dies kann ein beliebiger ganzzahliger Wert sein, wobei 0 kein Leerzeichen ist (die Buchstaben berühren sich, wenn sie proportional sind).

Es ist auch erwähnenswert, dass es keine Leerzeichen in der Zeichentabelle gibt. Wenn Sie beispielsweise '.' und '0', aber nicht '/', müssen Sie entweder '/' einschließen oder ein leeres Unterbild zwischen den beiden verwendeten Zeichen lassen. WICHTIG! Diese Funktion ist nicht mit der Testlizenz des Produkts verfügbar.


Syntax:

font_replace_sprite(ind, spr, first, prop, sep);


Streit Beschreibung
ind Der Index der Schriftart, die ersetzt werden soll (dh alle Verweise auf diese Ressource nach dieser Funktion beziehen sich auf die neue Schriftart, nicht auf das Original).
spr Das Sprite, zu dem eine Schrift hinzugefügt werden soll.
first Der Index des ersten verfügbaren Zeichens.
prop Gibt an, ob Leerzeichen um jeden Buchstaben entfernt werden sollen (bei False wird der erstellte Zeichensatz monospaced, andernfalls wie bei den meisten regulären Zeichensätzen).
sep Das Leerzeichen zwischen jedem Buchstaben (0) bedeutet, dass die rechte Seite eines Buchstabens die linke Seite des nächsten Buchstaben berührt.


Kehrt zurück:

N/A


Beispiel:

font_replace_sprite(global.Font, working_directory + "Sprites\spr_Font.png", ord("!"), true, 2);

Der obige Code ersetzt das in der globalen Variablen "Font" indizierte Font-Asset durch ein neues Font-Asset aus einer externen Bilddatei.
