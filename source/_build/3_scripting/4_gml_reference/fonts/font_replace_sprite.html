font_replace_sprite


Beschreibung

Mit dieser Funktion können Sie eine Sprite-Leiste verwenden, um eine zuvor erstellte Sprite-Schriftart zu ersetzen, wobei jedes Teilbild ein einzelnes Symbol oder ein Buchstabe ist. Der Index ist die UTF8-Kartennummer für das erste zu verwendende Zeichen und die zu verwendenden Bereiche sind die gleichen wie für eine normale Schriftart, beginnend mit "!" Aber denken Sie daran, dass Sie nur angeben sollten, was Sie brauchen und dass Sie das verwenden sollten ord Funktion, um den richtigen UTF8-Wert für den Anfangsbuchstaben zu erhalten.

Sie können den Abstand für die Schriftart auch proportional festlegen ( true ) oder nicht ( false ), wobei eine proportionale Schriftart basierend auf der tatsächlichen Breite der Buchstaben beabstandet ist (so nimmt beispielsweise der Buchstabe "i" weniger Platz ein als der Buchstabe "w"), während eine nichtproportionale Schriftart basierend auf der Unterteilung beabstandet ist. Bildbreite, wie eine monospaced Schriftart. Schließlich können Sie den Abstand definieren, der beim Schreiben zwischen den einzelnen Buchstaben verbleibt. Dies kann ein beliebiger ganzzahliger Wert sein, wobei 0 kein Leerzeichen ist (die Buchstaben werden "berühren", wenn sie proportional sind).

Beachten Sie auch, dass in der Zeichentabelle keine Leerzeichen enthalten sein dürfen, wenn Sie beispielsweise '.' und '0', aber nicht '/', müssen Sie entweder '/' eingeben oder ein leeres Unterbild zwischen den beiden verwendeten Zeichen lassen. WICHTIG! Diese Funktion ist mit der Testlizenz des Produkts nicht verfügbar.


Syntax:

font_replace_sprite(ind, spr, first, prop, sep);


Streit Beschreibung
ind Der Index der Schriftart, die ersetzt werden soll (dh alle Referenzen auf diese Ressource hinter dieser Funktion beziehen sich auf die neue Schriftart und nicht auf das Original).
spr Das Sprite zum Hinzufügen einer Schriftart basierend auf.
first Der Index des ersten verfügbaren Zeichens.
prop Ob der Leerraum um jeden Buchstaben herum entfernt werden soll (falls false, wird die erzeugte Schriftart monospaced sein, ansonsten ist sie wie die meisten normalen Schriften).
sep Der Abstand zwischen jedem Buchstaben, 0 bedeutet das Recht eines Buchstabens, berührt den linken des nächsten Buchstabens.


Kehrt zurück:

N/A


Beispiel:

font_replace_sprite(global.Font, working_directory + "\Sprites\spr_Font.png", ord("!"), true, 2);

Der obige Code ersetzt das in der globalen Variable "Font" indizierte Schriftarten-Asset durch ein neues Font-Asset aus einer externen Image-Datei.
