font_add_sprite


Beschreibung

Mit dieser Funktion können Sie einen "Sprite-Strip" verwenden, um ein neues Font-Asset zu erstellen, wobei jedes Teilbild ein einzelnes Symbol oder ein einzelner Buchstabe ist. Der Index ist die UTF8-Kartennummer für das erste zu verwendende Zeichen und die zu verwendenden Bereiche sind die gleichen wie für eine normale Schriftart, beginnend mit "!" Aber denken Sie daran, dass Sie nur angeben sollten, was Sie brauchen und dass Sie das verwenden sollten ord Funktion, um den richtigen UTF8-Wert für den Anfangsbuchstaben zu erhalten. Die erstellte Sprite-Schriftart wird dann die Teilbilder auf ein einzelnes Glyph aus der Schriftart festgelegt, wobei die "erste" Glyphe den image_index von 0 haben wird. Sie können "space" als beliebiges Zeichen definieren, z. B. als einzelnes Zeichen Linie die Größe, die der Raum sein soll, und alle Räume im Text werden auf dieser Breite gerendert (aber das gewählte Bild wird nie gerendert werden), oder wenn Sie kein Sprite für Platz dann die Breite der breitesten liefern Zeichen in der Sprite-Schriftart werden stattdessen verwendet.

Sie können den Abstand für die Schriftart auch proportional festlegen ( true ) oder nicht ( false ), wobei eine proportionale Schriftart basierend auf der tatsächlichen Breite der Buchstaben beabstandet ist (so nimmt beispielsweise der Buchstabe "i" weniger Platz ein als der Buchstabe "w"), während eine nichtproportionale Schriftart basierend auf der Unterteilung beabstandet ist. Bildbreite, wie eine monospaced Schriftart. Schließlich können Sie den Abstand definieren, der beim Schreiben zwischen den einzelnen Buchstaben verbleibt. Dies kann ein beliebiger ganzzahliger Wert sein, wobei 0 kein Leerzeichen ist (die Buchstaben werden "berühren", wenn sie proportional sind). Die Funktion gibt einen Indexwert zurück, der in einer Variablen gespeichert werden sollte, da dies in allen weiteren Codes, die sich auf diese Schriftart beziehen, benötigt wird.

Beachten Sie auch, dass in der Zeichentabelle keine Leerzeichen enthalten sein dürfen, wenn Sie beispielsweise '.' und '0', aber nicht '/', müssen Sie entweder '/' eingeben oder ein leeres Unterbild zwischen den beiden verwendeten Zeichen lassen.

Das Sprite selbst kann ein Sprite-Asset aus dem Ressourcenbaum oder ein aus einer Datei geladenes sein sprite_add zuerst, um den Index des geladenen Sprites zu erhalten (welches dann in der Funktion verwendet wird). Weitere Informationen über die zu verwendende Zeichenmappe beim Erstellen von Sprite-Schriftarten finden Sie auf der Seite zu Schriftarttabellen. Es ist wichtig zu beachten, dass für die Funktionen zur Schriftausrichtung (wie draw_set_halign() ), um wie erwartet zu arbeiten, wenn eine Sprite-Schriftart verwendet wird, sollte der Ursprung des Sprites in der oberen linken Ecke liegen. Wenn Sie andere Werte verwenden, müssen Sie den Ursprungsoffset berücksichtigen, wenn Sie Text mit der Schriftart zeichnen.

HINWEIS: Wenn Sie eine Schriftart in GameMaker Studio 2 laden, müssen Sie daran denken, sie erneut zu entfernen (mit font_delete ) wenn es nicht mehr benötigt wird, besteht sonst die Gefahr eines Speicherlecks, das das Spiel verlangsamt und schließlich zum Absturz bringt.
WICHTIG! Diese Funktion ist mit der Testlizenz des Produkts nicht verfügbar.


Syntax:

font_add_sprite(spr, first, prop, sep);


Streit Beschreibung
spr Das Sprite zum Hinzufügen einer Schriftart basierend auf.
first Der Index des ersten verfügbaren Zeichens.
prop Als proportionale Schriftart festlegen oder nicht.
sep Der Platz zwischen jedem Buchstaben zu verlassen.


Kehrt zurück:

Real


Beispiel:

global.Font = font_add_sprite(spr_HUD_Font, ord("!"), true, 2);

Der obige Code erstellt ein neues Schriftarten-Asset aus einem Sprite-Asset und speichert den Index der neuen Schriftart in der Variablen "global.Font" zur späteren Verwendung.
