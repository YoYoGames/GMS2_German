Letzte Windows


Die Liste der zuletzt geöffneten Fenster kann über das Windows- Menü geöffnet werden und zeigt eine kurze Liste (standardmäßig sind es maximal 10 Einträge) der Fenster, die Sie zuvor geöffnet haben. Zu Beginn mit dieser Liste wird leer sein, aber wenn Sie verschiedene Fenster in der IDE öffnen (wie ein Sprite oder ein Skript oder ein Objekt, etc...) wird die Liste aufgefüllt. Sollten Sie zu einem dieser zuvor geöffneten Fenster zurückkehren müssen, klicken Sie einfach mit der linken Maustaste auf dem Element in der Liste und der Arbeitsbereich wird in das Fenster verschoben. Wenn Sie mit der rechten Maustaste klicken auf einem Gegenstand, dann öffnen Sie das folgende Menü:

Hier können Sie auswählen, ob Sie einen Artikel (oder Artikel, wie Sie ihn verwenden können) löschen + oder + um mehrere auszuwählen) aus der Liste oder Sie können sie an den Anfang der Liste verschieben lassen. Das RMB-Menü bietet außerdem eine zusätzliche Option, mit der Sie die Liste vollständig löschen können. Beachten Sie, dass die Liste der zuletzt verwendeten Windows an die Seiten der IDE angedockt werden kann, und Sie können auch bestimmte Einstellungen anpassen, um das Verhalten zu ändern.
