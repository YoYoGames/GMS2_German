Audiogruppen

Die Audio Group Manager ist von der zur Verfügung stehenden Werkzeuge Menü in der IDE. Hier können Sie hinzufügen, löschen und Audio - Gruppen umbenennen sowie setzen ihre Plattform Exportoptionen. GameMaker Studio 2 können Sie jede Audio-Ressource (Soundeffekte und Musik) verschiedenen Audio-Gruppen zuweisen, um die Anzahl der gleichzeitig gespielten Sounds zu optimieren, und Ihnen die weitere Kontrolle darüber geben, welche Plattformen sie verwenden werden exportiert. Um eine Audiogruppe zu definieren, müssen Sie klicken Klicken Sie auf die Schaltfläche Neu hinzufügen, um eine neue Gruppe zu erstellen, die Sie dann benennen können. Um die Gruppe zu ändern, klicken Sie auf Wählen Sie im Drop-down-Menü den Ordner aus, den Sie bearbeiten möchten.

WICHTIG! Audiogruppen sind mit der Testlizenz des Produkts nicht verfügbar.


Um einen Sound zu einer Audiogruppe hinzuzufügen, müssen Sie den Sound-Editor für diesen Sound öffnen und dann die Gruppe aus der Option auswählen, die Sie dort finden. Sobald Sie einer benutzerdefinierten Audiogruppe einen Sound hinzugefügt haben, können Sie in diesem Fenster auswählen, auf welcher Plattform der Sound exportiert werden soll. Es kann sein, dass Sie standardmäßig alle Ihre Sounds in höchster Qualität haben, aber für HTML5 (zum Beispiel) möchten Sie eine niedrigere Qualität der Sounddatei verwenden, um weniger Speicherplatz zu verbrauchen. In diesem Fall fügen Sie die Dateien mit der niedrigeren Qualität zu GameMaker Studio 2 und weisen sie dann einer neuen Audiogruppe zu. Sie würden dann diese Audiogruppe aus dem Dropdown-Menü auswählen und festlegen, dass nur in HTML5 exportiert werden soll, und den HTML5-Export aus den Exportoptionen für Audiogruppen mit höherer Qualität entfernen. Es ist wichtig zu beachten, dass Sie die Exportoptionen für die Audiogruppe "Standard" nicht ändern können und dass sie beim Erstellen eines endgültigen Spielpakets immer auf alle verfügbaren Plattformen exportiert werden.

Sobald Sie Audiogruppen definiert und ihnen Sounds zugewiesen haben, können Sie sie in der Liste auf der linken Seite sehen, wenn Sie die Gruppe auswählen. Es wird immer eine "Standard" -Audiogruppe geben und alle Sounds, die sich in dieser Gruppe befinden, sind immer im Spielpaket für alle Plattformen enthalten und werden beim Start alle in den Speicher geladen (es sei denn, sie werden als "gestreamt" gekennzeichnet) in den Sound Editor-Eigenschaften), aber beim Erstellen einer benutzerdefinierten Audiogruppe werden die hinzugefügten Dateien erst beim Aufruf der Funktion in den Speicher geladen audio_group_load(). Auf diese Weise können Sie steuern, welche Audiodaten zu jeder Zeit im Speicher vorhanden sind. Beachten Sie auch, dass für alle Audio-Gruppenfunktionen, müssen Sie die Audio-Gruppen - ID - Wert liefern. Dies ist einfach der Name, den Sie der Audiogruppe gegeben haben.

Audiogruppen sind auch mit den Konfigurationseinstellungen verknüpft. Wenn Sie eine Konfiguration auswählen, können Sie das Fenster "Audiogruppen" öffnen und die Exportziele für diese Konfiguration auf der rechten Seite auswählen. Wenn Sie dann die Konfigurationen ändern, werden diese Ausgabeziele geändert. Beachten Sie, dass Sie keine Soundressourcen für verschiedene Gruppen pro Konfiguration festlegen können, sondern nur das Exportziel für die angegebene Gruppe.
