Spieleinstellungen


Die Spieloptionen sind in zwei Abschnitte unterteilt, Hauptoptionen und Plattformeinstellungen. Details zu den Hauptoptionen finden Sie in den folgenden Abschnitten:

Im Fenster " Allgemeine Optionen" können Sie bestimmte Einstellungen für das Spiel festlegen, unabhängig von der Plattform, auf der es ausgeführt wird. Das Fenster hat folgende Abschnitte:

  • DnD™ als Standard-Skripttyp verwenden: Wenn diese Option aktiviert ist, werden alle neuen Skripts und Objektereignisse mit dem DnD™ -Editor und nicht mit dem Code-Editor geöffnet.

  • Game-Frames pro Sekunde: Dieser Wert legt die Anzahl der Frames pro Sekunde fest, die GameMaker Studio 2 versuchen sollte, Ihre Spiele beim Laufen aufrechtzuerhalten. Dieser Wert gibt an, wie oft die Spielschleife in einer Sekunde ausgeführt wird (eine einzelne Spielschleife wird als einzelner "Frame" betrachtet) und ist auf einen Standardwert von 30 eingestellt, obwohl 60 häufig verwendet wird. Es gibt viele Argumente für die ideale Raumgeschwindigkeit, aber die tatsächliche Antwort hängt von der Art des Spiels, der Größe des Raums, der Anzahl der Instanzen und der Intensität des Codes ab. Wenn Sie beispielsweise einen großen Raum und eine Raumgeschwindigkeit von 30 haben, kann das Spiel "abgehackt" erscheinen, je größer der Bereich ist, den Sie betrachten, desto leichter ist es für das Auge, niedrige Raumgeschwindigkeiten wahrzunehmen, jedoch das Spiel Wenn Sie die Raumgeschwindigkeit auf 100 einstellen, kann das Spiel "verzögern", da die Anzahl der Bilder pro Sekunde aufgrund der Menge der durchgeführten Verarbeitung sinkt. Achten Sie also auf diesen Wert und versuchen Sie, ihn immer an die Anforderungen Ihres Spiels anzupassen.

  • Standard-Zeichnungsfarbe: Die Standardfarbe für das Zeichnen von Text oder Foren (und anderen Dingen), wenn keine Farbe mithilfe von Code festgelegt wurde.

  • Projektbilder generieren: Wenn Sie auf diese Option klicken, wird das Projektbildgenerator- Tool geöffnet.

  • Projektstartdatum: Das Datum, an dem das Projekt initiiert wurde.

  • Projektnutzungszeit: Die Zeit, die seit dem Initiieren des Projekts vergangen ist.

  • Projekt-GUID: Dies ist die eindeutige Kennung, die für jedes Projekt in GameMaker Studio 2. Wenn Sie dies ändern möchten, klicken Sie auf die Schaltfläche Create New GUID (Neue GUID erstellen), um eine neue zu erstellen.

  • Generieren von Mipmaps für separate Textur-Seiten: Mit dieser Option erstellt GameMaker Studio 2 Mipmaps für die Sprites, die die Option Separate Texture Page in den Textur-Einstellungen des Sprite-Editors markiert haben. Das Aktivieren von Mipmapping bedeutet, dass die ursprüngliche Texturseite für das Sprite in mehreren Auflösungen innerhalb der Texturdatei skaliert und gefiltert werden soll. Dies ist normalerweise deaktiviert, aber wenn Sie Ansichten verwenden, um große Bereiche des Spielzimmers zu vergrößern oder zu verkleinern, oder wenn Sie mit einem 3D-Spiel arbeiten, möchten Sie dies möglicherweise aktivieren und dann die Mipmapping-Funktionen verwenden, um die Einstellungen anzupassen Spiel behandelt die verschiedenen Mipmap-Ebenen. Wenn Sie Sprites nicht als einzelne Textur-Seiten kennzeichnen, können Sie das Mipmapping weiterhin auf Texturbasis für jede Textur im Texture Group Manager aktivieren. Diese Option ist standardmäßig deaktiviert.

  • Steam ID: Wenn Sie ein Spiel erstellen, das über Steam verteilt werden soll, müssen Sie hier die Steam App-ID angeben. Vergewissern Sie sich, dass Sie den Pfad zum Steam Works-SDK in den Plattformeinstellungen hinzugefügt haben, wenn Sie diese Option verwenden, und stellen Sie sicher, dass im Steam Works-Dashboard alles ordnungsgemäß eingerichtet ist.

  • Anonyme Spielestatistiken zulassen: Aktivieren oder deaktivieren Sie das Senden von GameMaker Studio 2 Spielanalysedaten an YoYo Games, sowohl von der IDE als auch von den von Ihnen erstellten Spielen. Wenn diese Option aktiviert ist, werden anonyme Daten über den PC gesendet, auf dem das Programm / Spiel ausgeführt wird (z. B. Größe des Monitors, CPU, GPU, Speicher usw.), und die endgültigen ausführbaren Dateien senden auch Daten darüber, wie das Spiel heißt. Diese Daten werden gesammelt, damit YoYo Games ihre Dienste besser auf die Bedürfnisse der Benutzer YoYo Games kann. In Zukunft möchte YoYo Games diese Daten den Entwicklern zur Verfügung stellen (ähnlich wie Steam mit seiner Steam Software Survey ). Weitere Informationen finden Sie in Teil 6 des Abschnitts Informationen, die wir in den Datenschutzbestimmungen von YoYo Games sammeln. Diese Option ist nur für Benutzer mit einer kostenpflichtigen Lizenz (Desktop, Mobile, HTML5, UWP oder Console) verfügbar. Benutzer mit der Testlizenz senden immer anonyme statistische Daten.

  • Quellcodeverwaltung aktivieren: Wenn Sie möchten, dass das Projekt unter SCM (Source Control Management) ist, müssen Sie dies überprüfen. Es ist standardmäßig (siehe ab hier für weitere Informationen über die Quellcodeverwaltung).

  • Autor: Der Name des Spielautors (oder des Verlegers oder Studios).

  • Projekt zuletzt geändert: In diesem Abschnitt wird lediglich der Datums- / Zeitstempel für das letzte Mal angezeigt, an dem das Projekt geändert wurde (dies ist nicht bearbeitbar).

GameMaker Studio 2 erfordert die Installation bestimmter Add-Ons als Erweiterungen für Ihr Projekt, wenn Sie erweiterte Funktionen wie Leaderboards oder In-App-Käufe auf bestimmten Plattformen verwenden möchten. Durch Klicken auf die Schaltfläche " Download " wird die Erweiterung heruntergeladen und im aktuellen Projekt installiert.

Im Ressourcenbaum stehen Ihnen auch andere Spieloptionen zur Verfügung, abhängig von den Zielplattformen, die für die Lizenz, die Sie besitzen, verfügbar sind. Möglicherweise sind nicht alle verfügbar oder sichtbar. Es gibt folgende Plattformoptionen:

  1. Windows
  2. Mac OS
  3. Ubuntu
  4. Windows UWP
  5. HTML5
  6. Amazon Fire
  7. Android
  8. iOS
HINWEIS: Die verschiedenen Konsolenspieloptionen werden hier nicht aufgeführt, da die verschiedenen Unternehmen unterschiedliche gesetzliche Auflagen erfüllen. Wenn Sie über eine Konsolenlizenz verfügen, können Sie diese Informationen vom YoYo Games Helpdesk erhalten. Sie müssen sich jedoch zunächst mit Ihrer Konsolen-ID-E-Mail an YoYo Games wenden und die Berechtigung zum Anzeigen der verschiedenen Abschnitte anfordern.
